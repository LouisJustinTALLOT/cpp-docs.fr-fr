# <a name="file-and-database-classes"></a>Classes de fichiers et de bases de données

Ces classes vous permettent de stocker des informations dans une base de données ou un fichier sur disque. Il existe trois ensembles de classes de base de données, OLE DB, ODBC et DAO, qui offrent des fonctionnalités similaires. Le groupe de OLE DB est implémenté à l’aide de OLE DB et fonctionne avec les modèles de consommateur OLE DB, le groupe DAO est implémenté à l’aide de l’objet d’accès aux données et le groupe ODBC est implémenté à l’aide de Open Database Connectivity. Il existe également un ensemble de classes pour manipuler les fichiers standard, les flux actifs et les flux HTML.

Les catégories de classes suivantes prennent en charge la persistance des données.

- [Classes d'E/S de fichier](file-i-o-classes.md)

- [Classes OLE DB](ole-db-classes.md)

- [Classes DAO](dao-classes.md)

- [Classes ODBC](odbc-classes.md)
