# Classes de fichiers et de bases de donn&#233;es
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Ces classes vous permettent de stocker des informations dans une base de données ou un fichier de disque.  Il existe trois jeux de classes de base de données — OLE DB, ODBC, et DAO — qui fournissent des fonctionnalités semblables.  Le groupe OLE DB est implémenté à l'aide de OLE DB et fonctionne avec les modèles de consommateur OLE DB, le groupe de DAO est implémenté en utilisant l'objet d'accès aux données, et le groupe ODBC est implémenté par l'Open Database Connectivity.  Il existe également un jeu de classes permettant de manipuler les fichiers standard, les flux de données actifs, et le flux de données HTML.  
  
 Les catégories suivantes de classes prennent en charge la persistance de données.

-   [Classes de base de données OLE](../mfc/ole-db-classes.md)  
  
-   [Classes DAO](../mfc/dao-classes.md)  
  
-   [Classes ODBC](../mfc/odbc-classes.md)
