# <a name="scrolling-arranging-sorting-and-finding-in-list-controls"></a>Défilement, organisation, tri et recherche dans les contrôles de liste

Les contrôles de liste ([CListCtrl](../mfc/reference/clistctrl-class.md)) peuvent faire l’aller défiler par défaut. Pour plus d’informations, consultez [Scroll position](/windows/win32/Controls/using-list-view-controls) dans le SDK Windows et la fonction de membre [Scroll](../mfc/reference/clistctrl-class.md#scroll) .

Vous pouvez appeler des `CListCtrl` fonctions membres pour organiser des éléments de liste dans le contrôle, trier des éléments et Rechercher des éléments particuliers. Pour plus d’informations, [consultez Utilisation des contrôles ListView](/windows/win32/Controls/using-list-view-controls) dans le SDK Windows et les membres [CListCtrl](../mfc/reference/clistctrl-class.md) [arrange](../mfc/reference/clistctrl-class.md#arrange), [SortItems](../mfc/reference/clistctrl-class.md#sortitems)et [FindItem](../mfc/reference/clistctrl-class.md#finditem).
