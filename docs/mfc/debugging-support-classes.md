# Classes de prise en charge du d&#233;bogage
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC fournit les classes suivantes pour vous aider à régler des problèmes d'allocation de mémoire dynamique.  
  
 [CDumpContext](../mfc/reference/cdumpcontext-class.md)  
 Fournit une destination pour les vidages de diagnostic.  
  
 [CMemoryState](../mfc/reference/cmemorystate-structure.md)  
 Structure qui fournit les instantanés de l'utilisation de la mémoire.  Également utilisé pour comparer des instantanés antérieures et ultérieures de mémoire.
