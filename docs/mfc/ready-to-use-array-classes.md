# <a name="ready-to-use-array-classes"></a>Classes de tableaux prêtes à être utilisées

Les classes suivantes sont des classes de tableau prêtes à l’emploi.

[CByteArray](../mfc/reference/cbytearray-class.md)<br/>
Stocke des éléments de type **Byte** dans un tableau.

[CDWordArray](../mfc/reference/cdwordarray-class.md)<br/>
Stocke des éléments de type **DWORD** dans un tableau.

[CObArray](../mfc/reference/cobarray-class.md)<br/>
Stocke des pointeurs vers des objets de classe `CObject` ou vers des objets de classes dérivées de `CObject` dans un tableau.

[CPtrArray](../mfc/reference/cptrarray-class.md)<br/>
Stocke des pointeurs vers **`void`** (pointeurs génériques) dans un tableau.

[CUIntArray](../mfc/reference/cuintarray-class.md)<br/>
Stocke les éléments de type **uint** dans un tableau.

[CWordArray](../mfc/reference/cwordarray-class.md)<br/>
Stocke des éléments de type **Word** dans un tableau.

[CStringArray](../mfc/reference/cstringarray-class.md)<br/>
Stocke `CString` des objets dans un tableau.
