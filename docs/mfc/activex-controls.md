# <a name="activex-controls"></a>Contrôles ActiveX

Dans Visual C++, vous pouvez créer des contrôles ActiveX à l’aide de MFC ou ATL.

>[!IMPORTANT]
> ActiveX est une technologie héritée qui ne doit pas être utilisée pour tout nouveau développement. De nombreuses fonctionnalités de contrôles ActiveX peuvent être effectuées de façon plus simple et plus sûre avec des technologies modernes comme HTML5 et JavaScript, les extensions de navigateur moderne ou WebAssembly modules. Pour plus d’informations, consultez [un saut dans le passé, partie 2 : Saying goodbye pour ActiveX, VBScript, attachEvent...](https://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/) et [messagerie Native](/microsoft-edge/extensions/guides/native-messaging) et [les extensions Microsoft Edge](/microsoft-edge/extensions) et [WebAssembly](https://webassembly.org/).
