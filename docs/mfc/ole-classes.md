# <a name="ole-classes"></a>Classes OLE
Les classes OLE fonctionnent avec les autres classes application framework pour accéder facilement à l’API ActiveX, en donnant à vos programmes un moyen simple de fournir la puissance des contrôles ActiveX à vos utilisateurs. ActiveX, vous pouvez :  
  
-   Créer des documents composés, permettant aux utilisateurs de créer et modifier des documents contenant les données créées par plusieurs applications, y compris le texte, les graphiques, les feuilles de calcul, audio, ou autres types de données.  
  
-   Créer des objets OLE qui peuvent être incorporés dans des documents composés.  
  
-   Utilisez le glisser-déplacer OLE pour copier des données entre applications.  
  
-   Automation permet de contrôler un programme avec un autre.  
  
-   Créer des contrôles ActiveX et les conteneurs de contrôles ActiveX (anciennement contrôles OLE et les conteneurs de contrôles OLE, respectivement).
