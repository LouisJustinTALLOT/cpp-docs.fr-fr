# <a name="dialog-data-exchange-and-validation"></a>Échange et validation de données de boîtes de dialogue

L’échange de données de boîtes de dialogue (DDX) est un moyen simple d’initialiser les contrôles dans votre boîte de dialogue et de collecter des données entrées par l’utilisateur. La validation des données de boîte de dialogue (DDV) est un moyen simple de valider la saisie de données dans une boîte de dialogue. Pour tirer parti de DDX et DDV dans vos boîtes de dialogue, utilisez l' [Assistant Ajout de variable membre](../ide/add-member-variable-wizard.md) pour créer les membres de données et définir leurs types de données et spécifier des règles de validation.
