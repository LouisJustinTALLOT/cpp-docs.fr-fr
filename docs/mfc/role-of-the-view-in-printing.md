# <a name="role-of-the-view-in-printing"></a>Rôle de la vue dans l'impression

Votre affichage joue également deux rôles importants dans l’impression du document associé.

La vue :

- Utilise le même code [OnDraw](../mfc/reference/cview-class.md#ondraw) pour dessiner sur l’imprimante que pour dessiner à l’écran.

- Gère la Division du document en pages à imprimer.

Pour plus d’informations sur l’impression et sur le rôle de la vue dans l’impression, consultez [impression et aperçu avant impression](../mfc/printing-and-print-preview.md).
