# Conteneurs
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Une application conteneur est une application qui peut incorporer des éléments liés ou incorporés dans ses propres documents.  Les documents traités par une application conteneur doivent pouvoir stocker et afficher les composants de document composites OLE ainsi que les données créées par l'application elle\-même.  Une application doit également autoriser les utilisateurs à insérer de nouveaux éléments des éléments existants.
