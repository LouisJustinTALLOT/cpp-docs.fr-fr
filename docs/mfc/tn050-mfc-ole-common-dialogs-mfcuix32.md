# <a name="tn050-mfcole-common-dialogs-mfcuix32"></a>TN050 : boîtes de dialogue communes MFC/OLE (MFCUIx32)

Cette note a initialement abordé certains problèmes et l’avenir des boîtes de dialogue communes OLE fournies et utilisées par MFC. Les boîtes de dialogue communes OLE sont désormais fournies en tant que composant intégré au système (OLEDLG.DLL) et sont entièrement documentées dans la documentation du produit.
