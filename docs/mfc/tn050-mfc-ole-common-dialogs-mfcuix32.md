# <a name="tn050-mfcole-common-dialogs-mfcuix32"></a>TN050 : boîtes de dialogue communes MFC/OLE (MFCUIx32)

Cette note couvert à l’origine des problèmes et l’avenir du OLE boîtes de dialogue courantes fournies et utilisées par MFC. Les boîtes de dialogue communes OLE sont désormais disponibles en tant que composant intégré au système (OLEDLG. DLL) et est entièrement décrite dans la documentation du produit.
