# <a name="clipboard-when-to-use-each-clipboard-mechanism"></a>Presse-papiers : quand utiliser chaque mécanisme de Presse-papiers

Suivez ces instructions dans le presse-papiers :

- Utilisez le mécanisme OLE Clipboard Now pour activer les nouvelles fonctionnalités à l’avenir. Tandis que l’API du presse-papiers standard est conservée, le mécanisme OLE est l’avenir du transfert de données.

- Utilisez le mécanisme de presse-papiers OLE si vous écrivez une application OLE ou si vous souhaitez utiliser l’une des fonctionnalités OLE, telles que le glisser-déplacer.

- Utilisez le mécanisme de presse-papiers OLE si vous fournissez des formats OLE.
