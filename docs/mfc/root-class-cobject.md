# Classe racine&#160;: CObject
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La plupart des classes à la bibliothèque Microsoft Foundation Class \(MFC\) sont dérivées d'une seule classe de base pour la racine de la hiérarchie de classes.  `CObject` fournit plusieurs fonctions utiles pour toutes les classes dérivées de celui\-ci, à la surcharge faible.  Pour plus d'informations sur `CObject` et ses fonctions, consultez [Utilisation CObject](../mfc/using-cobject.md).  
  
 [CObject](../mfc/reference/cobject-class.md)  
 La classe de base finale de la plupart des classes de MFC.  Prend en charge des données à sérialiser et l'obtention des informations d'exécution sur une classe.  
  
 [CRuntimeClass](../mfc/reference/cruntimeclass-structure.md)  
 Structure utilisée pour déterminer la classe spécifique d'un objet au moment de l'exécution.
