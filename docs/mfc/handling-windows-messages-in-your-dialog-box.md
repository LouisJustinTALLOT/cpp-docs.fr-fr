# <a name="handling-windows-messages-in-your-dialog-box"></a>Traitement des messages Windows dans votre boîte de dialogue

Les boîtes de dialogue sont des fenêtres et peuvent donc traiter des messages Windows si vous fournissez les fonctions de gestionnaire appropriées. Lorsque vous créez votre classe de boîte de dialogue à l’aide de l' [Assistant classe](reference/mfc-class-wizard.md), l’Assistant ajoute une table des messages vide à la classe. Utilisez l’Assistant pour mapper les messages Windows ou les commandes que votre classe doit gérer.

Pour plus d’informations, consultez [mappage des messages Windows à votre classe de boîte de dialogue](../mfc/mapping-windows-messages-to-your-class.md) .
