# <a name="window-dialog-and-control-classes"></a>Classes de fenêtre, de boîte de dialogue et de contrôle

La classe `CWnd` et ses classes dérivées encapsulent une `HWND`, un handle vers une fenêtre Windows. `CWnd` peut être utilisé seul ou comme base pour dériver de nouvelles classes. Les classes dérivées fournies par la bibliothèque de classes représentent différents genres de fenêtres.

[CWnd](../mfc/reference/cwnd-class.md)<br/>
Classe de base pour toutes les fenêtres. Vous pouvez utiliser l’une des classes dérivées de `CWnd` ou dériver vos propres classes directement à partir de celle-ci.
