# Classes de fen&#234;tre, de bo&#238;te de dialogue et de contr&#244;le
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La classe `CWnd` et ses classes dérivées encapsulent `HWND`, un descripteur de la fenêtre Windows.  `CWnd` peut être utilisée seule ou comme base pour dériver de nouvelles classes.  Les classes dérivées fournies par la bibliothèque de classes représentent des types de fenêtres.  
  
 [CWnd](../mfc/reference/cwnd-class.md)  
 La classe de base pour toutes les fenêtres.  Vous pouvez utiliser l'une des classes dérivées de `CWnd` ou dériver vos propres classes à partir de celui\-ci.
