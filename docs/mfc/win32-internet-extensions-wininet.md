# <a name="win32-internet-extensions-wininet"></a>Extension Internet Win32 (WinInet)

Une application cliente Internet est un programme qui accède à des informations à partir d’une source de données réseau (serveur) à l’aide de protocoles Internet tels que Gopher, FTP ou HTTP. Une application cliente Internet peut accéder à un serveur pour récupérer des données telles que les cartes météo, les cotations boursières ou les titres des journaux, par exemple. Le client Internet peut accéder au serveur via un réseau externe (Internet) ou un réseau interne (parfois appelé intranet).

MFC comprend les extensions Internet Win32, ou WinInet, pour la création d’une application cliente Internet. MFC encapsule ces extensions Internet dans un ensemble de classes standard et faciles à utiliser. Vous pouvez écrire une application cliente WinInet en appelant les fonctions Win32 directement ou à l’aide des classes WinInet MFC.

Le composant Microsoft Win32 Internet Functions (WinInet) vous aide à faire de l’Internet une partie intégrante de toute application. Les nouvelles fonctions, contenues dans WININET.DLL, simplifient l’accès à Internet à l’aide du protocole HTTP (Hypertext Transfer Protocol), FTP (protocole FTP) et Gopher.
