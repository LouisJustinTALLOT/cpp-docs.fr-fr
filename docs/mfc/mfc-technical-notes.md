# <a name="mfc-technical-notes"></a>Notes techniques MFC
Une note technique est un document écrit pour les programmeurs par les programmeurs.  
  
 Chaque note technique décrit un problème ou une fonctionnalité qui dépasse le cadre du reste de la documentation de MFC. Les notes techniques fournies reflètent les demandes d’informations provenant d’utilisateurs, ainsi que des informations spécifiques que les développeurs MFC anticiper les utilisateurs avancés.  
  
 Il existe deux façons de naviguer dans les notes techniques :  
  
-   [Notes techniques par numéro](../mfc/technical-notes-by-number.md)  
  
-   [Notes techniques par catégorie](../mfc/technical-notes-by-category.md)
