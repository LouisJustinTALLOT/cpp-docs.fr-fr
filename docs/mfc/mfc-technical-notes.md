# <a name="mfc-technical-notes"></a>Notes techniques MFC

Une note technique est un document écrit pour les programmeurs par les programmeurs.

Chaque note technique décrit un problème ou une fonctionnalité qui dépasse le cadre du reste de la documentation de MFC. Les notes techniques fournies reflètent les demandes d’informations des utilisateurs, ainsi que des informations spécialisées que les développeurs MFC anticiper les utilisateurs expérimentés souhaiteront.

Il existe deux façons de parcourir les notes techniques :

- [Notes techniques par numéro](../mfc/technical-notes-by-number.md)

- [Notes techniques par catégorie](../mfc/technical-notes-by-category.md)
