# <a name="mfc-technical-notes"></a>Notes techniques MFC

Une note technique est un document écrit pour les programmeurs par les programmeurs.

Chaque note technique décrit un problème ou une fonctionnalité qui dépasse le cadre du reste de la documentation MFC. Les notes techniques fournies reflètent les demandes d’informations de la part des utilisateurs, ainsi que les informations spécialisées que les développeurs de MFC anticipent pour les utilisateurs expérimentés.

Il existe deux façons de parcourir les notes techniques :

- [Notes techniques par numéro](technical-notes-by-number.md)

- [Notes techniques par catégorie](technical-notes-by-category.md)
