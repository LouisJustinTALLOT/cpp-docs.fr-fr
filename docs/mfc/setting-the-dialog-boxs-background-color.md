# <a name="setting-the-dialog-boxs-background-color"></a>Définition de la couleur d'arrière-plan de la boîte de dialogue

Vous pouvez définir la couleur d’arrière-plan de vos boîtes de dialogue en gérant WM_CTLCOLOR messages pour la fenêtre de boîte de dialogue. La couleur que vous définissez est utilisée uniquement pour la boîte de dialogue spécifiée.

Pour obtenir un exemple, consultez le [blog codexpert](https://codexpert.ro/blog/2013/03/13/painting-the-dialog-backround/) .
