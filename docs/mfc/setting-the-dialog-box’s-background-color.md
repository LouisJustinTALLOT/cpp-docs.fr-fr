# D&#233;finition de la couleur d&#39;arri&#232;re-plan de la bo&#238;te de dialogue
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Définissez la couleur d'arrière\-plan des boîtes de dialogue en gérant les messages `WM_CTLCOLOR` pour la fenêtre de boîte de dialogue.  La couleur que vous définissez est utilisé uniquement pour la boîte de dialogue spécifiée.
