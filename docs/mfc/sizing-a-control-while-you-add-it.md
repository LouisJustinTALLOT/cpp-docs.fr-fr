# Sizing a Control While You Add It
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Pour dimensionner un contrôle lorsque vous l'ajoutez  
  
1.  Sélectionnez un contrôle dans la [fenêtre Boîte à outils](../Topic/Toolbox.md).  
  
2.  Placez votre curseur \(qui s'affiche sous forme de croix\) à l'emplacement où vous souhaitez voir apparaître le coin supérieur gauche du nouveau contrôle dans votre boîte de dialogue.  
  
3.  Cliquez et maintenez le bouton de la souris enfoncé pour ancrer le coin supérieur gauche de votre contrôle dans la boîte de dialogue, puis faites glisser le curseur et vers la droite et vers le bas jusqu'à ce que le contrôle atteigne la taille souhaitée.  
  
    > [!NOTE]
    >  Vous pouvez ancrer n'importe lequel des quatre coins du contrôle que vous dessinez.  Cette procédure utilisait le coin supérieur gauche comme exemple.  
  
4.  Relâchez le bouton de la souris.  Le contrôle est placé dans la boîte de dialogue avec la taille spécifiée.  
  
    > [!TIP]
    >  Vous pouvez redimensionner le contrôle après l'avoir déplacée dans la boîte de dialogue grâce au mouvement des poignées de redimensionnement situées sur les bords du contrôle.  Pour plus d'informations, consultez [Dimensionnement de contrôles individuels](../mfc/sizing-individual-controls.md).  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.* Pour obtenir des informations sur l'ajout de fichiers de ressources aux projets managés, l'accès aux ressources, l'affichage de ressources statiques et l'assignation de chaînes de ressources aux propriétés, et ce manuellement, consultez [Procédure pas à pas : localisation des applications Windows Forms](http://msdn.microsoft.com/fr-fr/9a96220d-a19b-4de0-9f48-01e5d82679e5) et [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
