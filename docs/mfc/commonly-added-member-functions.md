# Fonctions membres couramment ajout&#233;es
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Si la boîte de dialogue contient les boutons poussoir autres que OK ou une annulation, vous devez écrire des fonctions membres responsables de messages dans la classe de la boîte de dialogue pour répondre aux messages de notification contrôle qu'elles produisent.  Pour obtenir un exemple, consultez l'exemple de programme d'[Dessin à la main levée](../top/visual-cpp-samples.md).  Vous pouvez également traiter les messages de notification contrôle d'autres contrôles dans la boîte de dialogue.
