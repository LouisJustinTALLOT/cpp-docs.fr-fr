# <a name="template-classes-for-arrays-lists-and-maps"></a>Classes de modèle pour les tableaux, listes et tables

Ces classes de collection sont des modèles dont les paramètres déterminent les types des objets stockés dans les agrégats. Les `CArray` `CMap` classes, et `CList` utilisent des fonctions d’assistance globales qui doivent généralement être personnalisées. Pour plus d’informations sur ces fonctions d’assistance, consultez la rubrique [assistance de classe de collection](../mfc/reference/collection-class-helpers.md). Les classes de pointeurs typés sont des wrappers pour d’autres classes de la bibliothèque de classes. À l’aide de ces wrappers, vous inscrivez la vérification de type du compilateur pour vous aider à éviter les erreurs. Pour plus d’informations sur l’utilisation de ces classes, consultez [Collections](../mfc/collections.md).

Ces classes fournissent des modèles que vous pouvez utiliser pour créer des tableaux, des listes et des mappages à l’aide de n’importe quel type de votre choix.

[CArray](../mfc/reference/carray-class.md)<br/>
Classe de modèle pour la création de tableaux de types arbitraires.

[CList](../mfc/reference/clist-class.md)<br/>
Classe de modèle pour la création de listes de types arbitraires.

[CMap](../mfc/reference/cmap-class.md)<br/>
Classe de modèle pour la création de mappages avec des types de clé et de valeur arbitraires.

[CTypedPtrArray](../mfc/reference/ctypedptrarray-class.md)<br/>
Classe de modèle pour les tableaux de type sécurisé des pointeurs.

[CTypedPtrList](../mfc/reference/ctypedptrlist-class.md)<br/>
Classe de modèle pour les listes de types sécurisés de pointeurs.

[CTypedPtrMap](../mfc/reference/ctypedptrmap-class.md)<br/>
Classe de modèle pour les mappages de type sécurisé avec des pointeurs.
