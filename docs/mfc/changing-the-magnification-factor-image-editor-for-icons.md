# Changing the Magnification Factor (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Par défaut, l'Éditeur d'images affiche dans le volet gauche la taille réelle de la ressource et dans le volet droit, la ressource agrandie 6 fois.  Le facteur d'agrandissement \(indiqué dans la barre d'état en bas de l'espace de travail\) correspond au rapport entre la taille réelle de l'image et la taille affichée.  Le facteur par défaut est 6 dans une plage comprise entre 1 et 10.  
  
### Pour changer le facteur d'agrandissement  
  
1.  Sélectionnez le volet de l'**Éditeur d'images** dont vous souhaitez changer le facteur d'agrandissement.  
  
2.  Dans la [barre d'outils Éditeur d'images](../mfc/toolbar-image-editor-for-icons.md), cliquez sur la flèche située à droite de l'[outil Loupe](../mfc/toolbar-image-editor-for-icons.md) et sélectionnez le facteur d'agrandissement dans le sous\-menu : **1X**, **2X**, **6X** ou **8X**.  
  
    > [!NOTE]
    >  Pour sélectionner un facteur d'agrandissement autre que ceux qui sont proposés dans l'outil **Loupe**, utilisez les touches accélérateur.
