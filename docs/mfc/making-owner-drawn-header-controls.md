# Rendre des contr&#244;les header owner-drawn
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Vous pouvez définir des éléments individuels d'un contrôle header \([CHeaderCtrl](../mfc/reference/cheaderctrl-class.md)\) pour être des éléments dessinés par le propriétaire.  Pour plus d'informations, consultez [Contrôles headers dessinés par le propriétaire](http://msdn.microsoft.com/library/windows/desktop/bb775238) dans [!INCLUDE[winSDK](../atl/includes/winsdk_md.md)].
