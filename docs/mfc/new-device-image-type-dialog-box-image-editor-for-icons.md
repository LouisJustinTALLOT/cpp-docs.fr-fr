# Nouveau type d’image de &lt;appareil&gt;, bo&#238;te de dialogue (&#201;diteur d’images pour les ic&#244;nes)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Vous permet de créer une nouvelle image de périphérique d'un type spécifié.  Pour ouvrir la boîte de dialogue **Nouveau type d'image \<périphérique\>**, cliquez sur **Nouveau type d'image** dans le menu **Image**.  Pour plus d'informations sur les tailles d'icône que Windows utilise, consultez [Icônes](_win32_Icons_cpp) dans la documentation du Kit de développement logiciel \(SDK\) Windows.  
  
 **Type d'image cible**  
 Répertorie les types d'images disponibles.  Sélectionnez le type d'image que vous souhaitez ouvrir :  
  
||||  
|-|-|-|  
|-   16 x 16, 16 couleurs|-   48 x 48, 16 couleurs|-   96 x 96, 16 couleurs|  
|-   16 x 16, 256 couleurs|-   48 x 48, 256 couleurs|-   96 x 96, 256 couleurs|  
|-   16 x 16, Monochrome|-   48 x 48, Monochrome|-   96 x 96, Monochrome|  
|-   32 x 32, 16 couleurs|-   64 x 64, 16 couleurs||  
|-   32 x 32, 256 couleurs|-   64 x 64, 256 couleurs||  
|-   32 x 32, Monochrome|-   64 x 64, Monochrome||  
  
> [!NOTE]
>  Toutes les images existantes ne sont pas affichées dans la liste.  
  
 **Personnalisé**  
 Ouvre la [boîte de dialogue Type d'image personnalisée](../mfc/custom-image-dialog-box-image-editor-for-icons.md) dans laquelle vous pouvez créer une nouvelle image avec une taille et un nombre de couleurs personnalisés.  
  
## Configuration requise  
 Aucun
