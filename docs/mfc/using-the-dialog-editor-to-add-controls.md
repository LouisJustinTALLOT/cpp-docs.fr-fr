# <a name="using-the-dialog-editor-to-add-controls"></a>Utilisation de l'Éditeur de boîtes de dialogue pour ajouter des contrôles

Lorsque vous créez votre ressource de modèle de boîte de dialogue avec l' [éditeur de boîtes](../windows/dialog-editor.md)de dialogue, vous faites glisser des contrôles à partir d’une palette de contrôles et déposez-les dans la boîte de dialogue. Cela ajoute les spécifications pour ce type de contrôle à la ressource de modèle de boîte de dialogue. Lorsque vous construisez un objet de boîte de dialogue et appelez sa `Create` `DoModal` fonction membre, le Framework crée un contrôle Windows et le place dans la fenêtre de dialogue à l’écran.

À la place [, vous pouvez créer des contrôles manuellement](../mfc/adding-controls-by-hand.md) si vous le souhaitez. C’est plus de travail.
