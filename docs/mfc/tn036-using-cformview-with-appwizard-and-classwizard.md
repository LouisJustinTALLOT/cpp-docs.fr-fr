# <a name="tn036-using-cformview-with-appwizard-and-classwizard"></a>TN036 : Utilisation de CFormView avec AppWizard et ClassWizard

Cette note technique a décrit comment modifier une application générée par AppWizard afin qu’elle utilise un `CFormView` au lieu de la valeur par défaut `CView` comme classe de vue principale. Cette version de Visual C++ est prise en charge directement.
