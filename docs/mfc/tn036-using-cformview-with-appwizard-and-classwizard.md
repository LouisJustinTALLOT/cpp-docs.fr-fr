# TN036&#160;: utilisation de CFormView avec AppWizard et ClassWizard
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Cette note technique décrit comment modifier une application générée par AppWizard afin qu'elle utilise `CFormView` au lieu de `CView` par défaut comme classe d'affichage principale.  Cela est prise en charge directement avec cette version de Visual C\+\+.
