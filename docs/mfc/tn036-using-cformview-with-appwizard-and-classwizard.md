# <a name="tn036-using-cformview-with-appwizard-and-classwizard"></a>TN036 : utilisation de CFormView avec AppWizard et ClassWizard

Cette note technique a décrit comment modifier une application générée par AppWizard afin qu’elle utilise un `CFormView` au lieu de la `CView` par défaut comme classe de vue principale. Cela est pris en charge directement avec cette version C++de Visual.
