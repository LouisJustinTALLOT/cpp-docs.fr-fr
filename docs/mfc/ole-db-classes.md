# Classes OLE&#160;DB
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La prise en charge OLE DB de MFC inclut actuellement la classe [COLEDBRecordView](../mfc/reference/coledbrecordview-class.md).  `COleDBRecordView` affiche les enregistrements de base de données dans les contrôles, via un mode formulaire directement connectée à un objet de [CRowset](../data/oledb/crowset-class.md).  Pour plus d'informations sur les modèles de consommateur OLE DB, consultez [Liste de modèles du consommateur OLE DB](../data/oledb/ole-db-consumer-templates-reference.md).
