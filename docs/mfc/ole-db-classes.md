# <a name="ole-db-classes"></a>Classes OLE DB

La prise en charge OLE DB dans MFC est actuellement constituée de la classe [COLEDBRecordView](../mfc/reference/coledbrecordview-class.md). `COleDBRecordView` affiche les enregistrements de base de données dans des contrôles, via un mode formulaire directement connecté à un objet [CRowset](../data/oledb/crowset-class.md) . Pour plus d’informations sur les modèles de consommateur OLE DB, consultez [liste des modèles de consommateur OLE DB](../data/oledb/ole-db-consumer-templates-reference.md).
