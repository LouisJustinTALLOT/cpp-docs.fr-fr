# <a name="ole-db-classes"></a>Classes OLE DB

La prise en charge OLE DB dans MFC se compose actuellement de la classe [COLEDBRecordView](../mfc/reference/coledbrecordview-class.md). `COleDBRecordView` Affiche les enregistrements de base de données dans des contrôles, via une vue de formulaire directement connecté à un [CRowset](../data/oledb/crowset-class.md) objet. Pour plus d’informations sur les modèles du consommateur OLE DB, consultez [liste de modèles consommateur OLE DB](../data/oledb/ole-db-consumer-templates-reference.md).
