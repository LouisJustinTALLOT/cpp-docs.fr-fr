# <a name="using-your-old-toolbars"></a>Utilisation de vos anciennes barres d'outils

Si vous avez utilisé des versions précédentes de Visual C++ pour créer des barres d’outils personnalisées, la nouvelle implémentation de la classe [CToolBar](../mfc/reference/ctoolbar-class.md) peut provoquer des problèmes. Afin que vous n’ayez pas à abandonner vos anciennes barres d’outils pour utiliser les nouvelles fonctionnalités, l’ancienne implémentation est toujours prise en charge.

L’exemple DOCKTOOL n’utilise pas les anciennes barres d’outils, mais uniquement les nouvelles barres d’outils de style.

Vous ne pouvez pas modifier les anciennes barres d’outils avec l’éditeur de ressources de barre d’outils.
