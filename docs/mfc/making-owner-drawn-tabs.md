# <a name="making-owner-drawn-tabs"></a>Rendre des onglets owner-drawn

Vous pouvez définir des éléments individuels d’un contrôle onglet ([CTabCtrl](../mfc/reference/ctabctrl-class.md)) pour qu’ils soient des éléments owner-drawn. Pour plus d’informations, consultez les [onglets owner-drawn](/windows/win32/Controls/tab-controls) dans le SDK Windows.
