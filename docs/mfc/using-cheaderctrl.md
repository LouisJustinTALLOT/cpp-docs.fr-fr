# <a name="using-cheaderctrl"></a>Utilisation de CHeaderCtrl

Utilisez un contrôle d’en-tête, représenté par la classe [CHeaderCtrl](../mfc/reference/cheaderctrl-class.md), pour afficher des en-têtes de colonne pour une liste de colonnes. Par exemple, un contrôle d’en-tête serait utile pour la mise en œuvre des commandes de colonnes dans une feuille de calcul.

Le contrôle de l’en-tête est généralement divisé en parties, appelées « éléments d’en-tête », chacun portant un titre pour la colonne de texte ou de nombres associée. Selon les styles que vous définissez, vous pouvez fournir un certain nombre de moyens directs pour les utilisateurs de manipuler les éléments d’en-tête.

> [!NOTE]
> [CListCtrl](../mfc/reference/clistctrl-class.md) fournit un contrôle d’en-tête intégré, `CListCtrl` et [CListView](../mfc/reference/clistview-class.md) encapsule dans une classe MFC. En général, pensez `CHeaderCtrl` à utiliser pour étiqueter les listes que vous avez l’intention de dessiner vous-même.
