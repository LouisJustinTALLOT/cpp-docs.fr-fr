# Classes de documents actifs
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les documents actifs peuvent être affichés dans la fenêtre cliente entière d'un navigateur Web, tel qu'Internet Explorer 5.5, ou dans un conteneur actif, comme le Classeur Microsoft Office, qui prend en charge les documents actifs.  
  
 [CDocObjectServer](../mfc/reference/cdocobjectserver-class.md)  
 Mappe les interfaces de document actifs et initialise et active un document objet actif.  
  
 [CDocObjectServerItem](../mfc/reference/cdocobjectserveritem-class.md)  
 Implémente les verbes de serveur OLE , en particulier pour les serveurs documents Active.  
  
 [COleDocObjectItem](../mfc/reference/coledocobjectitem-class.md)  
 Implémente la relation contenant\-contenu de document actif.
