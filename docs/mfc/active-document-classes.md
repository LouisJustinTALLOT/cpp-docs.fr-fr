# <a name="active-document-classes"></a>Classes de documents actifs
Documents actifs peuvent être affichés dans la fenêtre cliente entière d’un navigateur Web, tel que Internet Explorer 5.5, ou dans un conteneur actif, telles que le classeur Microsoft Office, qui prend en charge les documents actifs.  
  
 [CDocObjectServer](../mfc/reference/cdocobjectserver-class.md)  
 Mappe les interfaces de document actif et initialise et Active un objet de document actif.  
  
 [CDocObjectServerItem](../mfc/reference/cdocobjectserveritem-class.md)  
 Implémente les verbes de serveur OLE spécialement pour les serveurs de documents actifs.  
  
 [COleDocObjectItem](../mfc/reference/coledocobjectitem-class.md)  
 Implémente la relation contenant-contenu de document actif.
