# <a name="using-cdatetimectrl"></a>Utilisation de CDateTimeCtrl

Le contrôle de sélecteur de date et d’heure ([CDateTimeCtrl](../mfc/reference/cdatetimectrl-class.md)) implémente une méthode intuitive et reconnaissable pour l’entrée ou la sélection d’une date spécifique. L’interface principale du contrôle est similaire à une zone de liste déroulante. Toutefois, si l’utilisateur développe le contrôle, un contrôle Month Calendar s’affiche (par défaut), ce qui permet à l’utilisateur de spécifier une date particulière. Quand vous choisissez une date, le contrôle Month Calendar disparaît automatiquement.

> [!NOTE]
> Pour utiliser les `CDateTimePicker` classes et `CMonthCalCtrl` dans votre projet, vous devez inclure AFXDTCTL. H, généralement dans le STDAFX de votre projet. Fichier H.
