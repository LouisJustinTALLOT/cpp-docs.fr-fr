# <a name="using-cdatetimectrl"></a>Utilisation de CDateTimeCtrl

Le contrôle du cueilleur de dates et d’heures ([CDateTimeCtrl](../mfc/reference/cdatetimectrl-class.md)) met en œuvre une méthode intuitive et reconnaissable d’entrée ou de sélection d’une date spécifique. L’interface principale du contrôle est similaire en fonctionnalité à une boîte combo. Toutefois, si l’utilisateur élargit le contrôle, un contrôle de calendrier de mois apparaît (par défaut), permettant à l’utilisateur de spécifier une date particulière. Lorsqu’une date est choisie, le contrôle du calendrier du mois disparaît automatiquement.

> [!NOTE]
> Pour utiliser `CDateTimePicker` les `CMonthCalCtrl` classes et les classes de votre projet, vous devez inclure AFXDTCTL. H, habituellement dans le STDAFX de votre projet. Fichier H.
