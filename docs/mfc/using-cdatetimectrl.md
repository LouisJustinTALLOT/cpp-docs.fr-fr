# Utilisation de CDateTimeCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le contrôle Date Time Picker \([CDateTimeCtrl](../mfc/reference/cdatetimectrl-class.md)\) implémente une méthode intuitive et reconnaissable pour écrire ou sélectionne une date précise.  L'interface principale du contrôle est similaire dans son fonctionnement à celle d'une zone de liste déroulante.  Toutefois, si l'utilisateur développe le contrôle, un contrôle calendrier du mois s'affiche \(par défaut\), ce qui permet à l'utilisateur de spécifier une date particulière.  Lorsque la date est sélectionnée, le contrôle calendrier du mois disparaît automatiquement.  
  
> [!NOTE]
>  Pour utiliser à la fois les classes **CDateTimePicker** et `CMonthCalCtrl` dans votre projet, vous devez inclure AFXDTCTL.H, en général dans le fichier STDAFX.H de votre projet.
