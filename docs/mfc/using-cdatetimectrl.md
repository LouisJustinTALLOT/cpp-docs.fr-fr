# <a name="using-cdatetimectrl"></a>Utilisation de CDateTimeCtrl

Le contrôle de sélecteur de date et d’heure ([CDateTimeCtrl](../mfc/reference/cdatetimectrl-class.md)) implémente une méthode intuitive et reconnaissable entrer ou sélectionner une date spécifique. L’interface principale du contrôle est des fonctionnalités similaires à une zone de liste déroulante. Toutefois, si l’utilisateur développe le contrôle, un contrôle month calendar s’affiche (par défaut), ce qui permet à l’utilisateur de spécifier une date particulière. Quand une date est sélectionnée, le contrôle month calendar disparaît automatiquement.

> [!NOTE]
>  Pour utiliser à la fois le `CDateTimePicker` et `CMonthCalCtrl` classes dans votre projet, vous devez inclure AFXDTCTL. H, généralement dans votre fichier STDAFX. Fichier de H.
