# Utilisation de CTabCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Un "contrôle de tableau" est analogue aux diviseurs d'un carnet de notes ou aux dossiers étiquetés dans un classeur de fichiers.  Utilisez le contrôle onglet, représenté par la classe [CTabCtrl](../mfc/reference/ctabctrl-class.md), pour afficher plusieurs pages des informations ou les contrôles à un utilisateur, un par un, dans un format qui suggère une relation d'égal ou logique entre chaque page.  
  
 Pour plus d'informations sur les contrôles onglet, consultez [Contrôle onglet](http://msdn.microsoft.com/library/windows/desktop/bb760550) dans [!INCLUDE[winSDK](../atl/includes/winsdk_md.md)].
