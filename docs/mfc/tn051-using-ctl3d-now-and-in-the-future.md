# <a name="tn051-using-ctl3d-now-and-in-the-future"></a>TN051 : utilisation de CTL3D maintenant et dans le futur

Cette note technique, qui a déjà abordé CTL3D et MFC, est désormais obsolète. L’effet 3D pour les contrôles est automatiquement implémenté par MFC.
