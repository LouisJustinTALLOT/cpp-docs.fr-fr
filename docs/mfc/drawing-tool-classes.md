# <a name="drawing-tool-classes"></a>Classes d'outil de dessin

Ces classes encapsulent les outils de dessin utilisés pour dessiner sur un contexte de périphérique.

[CGdiObject](reference/cgdiobject-class.md)<br/>
Classe de base pour les outils de dessin GDI.

[CBrush](reference/cbrush-class.md)<br/>
Encapsule un pinceau GDI qui peut être sélectionné en tant que pinceau actuel dans un contexte de périphérique. Les pinceaux sont utilisés pour remplir les intérieurs des objets qui sont dessinés.

[CPen](reference/cpen-class.md)<br/>
Encapsule un stylet GDI qui peut être sélectionné comme stylet actuel dans un contexte de périphérique. Les stylets sont utilisés pour dessiner les lignes de bordure des objets.

[CFont](reference/cfont-class.md)<br/>
Encapsule une police GDI qui peut être sélectionnée comme police actuelle dans un contexte de périphérique.

[CBitmap](reference/cbitmap-class.md)<br/>
Encapsule une bitmap GDI, en fournissant une interface pour la manipulation des bitmaps.

[CPalette](reference/cpalette-class.md)<br/>
Encapsule une palette de couleurs GDI pour une utilisation en tant qu’interface entre l’application et un périphérique de sortie de couleur tel qu’un affichage.

[CRectTracker](reference/crecttracker-class.md)<br/>
Affiche et gère l’interface utilisateur pour le redimensionnement et le déplacement d’objets rectangulaires.
