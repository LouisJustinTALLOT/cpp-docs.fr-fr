# <a name="trackers-implementing-trackers-in-your-ole-application"></a>Dispositifs de suivi : implémentation de dispositifs de suivi dans votre application OLE

Les dispositifs de suivi fournissent une interface graphique qui permet aux utilisateurs d’interagir avec les éléments du client OLE. À l’aide de différents styles de suivi, les éléments de client OLE peuvent être affichés avec des bordures hachurées, des poignées de redimensionnement ou divers autres effets visuels. Cet article aborde les points suivants :

- [Comment implémenter le suivi dans votre code](../mfc/how-to-implement-tracking-in-your-code.md).

- [Bandes de caoutchouc et pistes](../mfc/rubber-banding-and-trackers.md).

L’article aborde également l’utilisation de styles avec des dispositifs de suivi. En outre, il fait plusieurs références à l’exemple OLE MFC [OCLIENT](../overview/visual-cpp-samples.md).
