# <a name="tn037-multithreaded-mfc-21-applications"></a>TN037 : applications MFC 2.1 multithread

Initialement, cette note technique décrit les limitations de programmes multithreads avec MFC 2.1, à l’origine fournies avec Visual C++ 1.0 pour Windows NT. Prend en charge de MFC 3.0 multithreading directement et documenté. Consultez cette référence pour plus d’informations.
