# TN037&#160;: applications MFC 2.1 multithread
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Cette note technique décrivait initialement les limitations des applications multithread avec MFC 2.1, initialement données avec Visual C\+\+ 1.0 pour Windows NT.  MFC 3,0 prend en charge le multithreading directement et est documenté.  Consultez cette référence pour plus d'informations.
