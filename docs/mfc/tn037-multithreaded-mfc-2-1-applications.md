# <a name="tn037-multithreaded-mfc-21-applications"></a>TN037 : applications MFC 2.1 multithread

Cette note technique a initialement décrit les limitations des programmes multithread avec MFC 2,1, initialement fournie avec Visual C++ 1,0 pour Windows NT. MFC 3,0 prend en charge le multithreading directement et est documenté. Pour plus d’informations, consultez cette référence.
