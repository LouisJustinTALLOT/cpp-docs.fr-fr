# Dessin et impression de classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Dans windows, toute sortie graphique est dessinée sur la zone de dessin virtuelle appelée un contexte \(DC\) de périphérique.  MFC fournit des classes pour encapsuler les différents types de contrôleurs de domaine, ainsi que des encapsulations des outils de dessin Windows comme bitmap, les pinceaux, les palettes et les stylets.
