# <a name="drawing-and-printing-classes"></a>Dessin et impression de classes

Dans Windows, toutes les sorties graphiques sont dessinées sur une zone de dessin virtuelle appelée contexte de périphérique (DC). MFC fournit des classes pour encapsuler les différents types de contrôleurs de schéma, ainsi que des encapsulations pour les outils de dessin Windows, tels que les bitmaps, les pinceaux, les palettes et les stylets.
