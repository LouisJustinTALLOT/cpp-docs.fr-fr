# <a name="class-library-overview"></a>Vue d'ensemble de la bibliothèque de classes

Cette vue d’ensemble catégorise et décrit les classes de la version 9.0 de la Microsoft Foundation Class Library (MFC). Les classes de MFC, prises ensemble, constituent un cadre d’application — le cadre d’une application écrite pour l’API Windows. Votre tâche de programmation est de remplir le code qui est spécifique à votre application.

Pour un aperçu du cadre, voir [Utiliser les classes pour écrire des applications pour Windows](../mfc/using-the-classes-to-write-applications-for-windows.md). Certaines des classes énumérées ci-dessus sont des classes polyvalentes qui peuvent être utilisées en dehors du cadre et fournir des abstractions utiles telles que les collections, les exceptions, les fichiers et les cordes.

Pour voir l’héritage d’une classe, utilisez le [tableau de la hiérarchie des classes](../mfc/hierarchy-chart.md).

En plus des classes énumérées dans cette vue d’ensemble, la Bibliothèque MFC contient un certain nombre de fonctions globales, de variables globales et de macros. Il ya une vue d’ensemble et une liste détaillée de ces dans le sujet [MFC Macros et Globals](../mfc/reference/mfc-macros-and-globals.md), qui suit la référence alphabétique aux classes MFC.
