# Comment le Framework effectue des recherches dans les tables des messages
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le framework recherche dans la table de table des messages des correspondances avec les messages entrants.  Une fois que vous entrez une entrée de la table des messages pour chaque message où vous souhaitez qu'une classe gère et écrive les gestionnaires de correspondance, le framework  appelle les gestionnaires automatiquement.  Les rubriques suivantes expliquent la table des messages :  
  
-   [Où trouver des tables de messages](../mfc/where-to-find-message-maps.md)  
  
-   [Tables des messages dérivées](../mfc/derived-message-maps.md)  
  
-   [Chaînes de mappage des messages, ID de commande, ou ID de contrôle pour un gestionnaire](../mfc/handlers-for-message-map-ranges.md)
