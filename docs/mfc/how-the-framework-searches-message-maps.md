# <a name="how-the-framework-searches-message-maps"></a>Comment le Framework effectue des recherches dans les tables des messages

L’infrastructure recherche les correspondances avec les messages entrants dans la table de la table des messages. Une fois que vous avez écrit une entrée de table des messages pour chaque message que vous souhaitez qu’une classe gère et écrive les gestionnaires correspondants, le Framework appelle automatiquement vos gestionnaires. Les rubriques suivantes décrivent la recherche de table des messages :

- [Où trouver des tables de messages](where-to-find-message-maps.md)

- [Tables des messages dérivées](derived-message-maps.md)

- [Mappage de plages de messages, d’ID de commande ou d’ID de contrôle à un gestionnaire](handlers-for-message-map-ranges.md)
