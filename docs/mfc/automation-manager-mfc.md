# <a name="automation-manager-mfc"></a>Gestionnaire d'Automation (MFC)
AUTMGR32. EXE doit être copié dans le répertoire système Windows de chaque ordinateur qui doit fournir des objets Automation à distance. Il est en général, C:\WINDOWS\SYSTEM32.  
  
 Si vous souhaitez activer des rappels à partir du serveur au client, ce fichier exécutable doit également être copié dans le répertoire système de chaque ordinateur client.  
  
 Lorsque le Gestionnaire d’Automation est en cours d’exécution, il affiche une petite fenêtre sur le serveur qui contient des informations d’état. Si vous souhaitez masquer, consultez l’article Q138067 dans la Base de connaissances Microsoft.
