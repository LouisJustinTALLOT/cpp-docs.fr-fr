# Gestionnaire d&#39;Automation (MFC)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

AUTMGR32.EXE doit être copié dans le répertoire système Windows de chaque ordinateur qui prédit des objets d'automatisation à distance.  Pour Windows 95 et Windows 98, ce répertoire est généralement C:\\WINDOWS\\SYSTEM.  Pour Windows NT et Windows 2000, il s'agit généralement C:\\WINNT\\system32.  
  
 Si vous souhaitez activer les rappels du serveur au client, l'exécutable doit également être copié dans le répertoire système de chaque ordinateur client.  
  
 Lorsque Gestionnaire d'automatisation exécute, elle affiche une petite fenêtre sur l'ordinateur serveur qui contient les brèves informations d'état.  Si vous le souhaitez masquer, consultez l'article Q138067 dans la Base de connaissances Microsoft.
