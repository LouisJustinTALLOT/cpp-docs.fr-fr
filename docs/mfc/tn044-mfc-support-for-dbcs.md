# <a name="tn044-mfc-support-for-dbcs"></a>TN044 : prise en charge de DBCS par MFC

Cette note technique a décrit la prise en charge dans MFC pour les « jeux de caractères codés sur deux octets » ou DBCS. Ces informations, ainsi que des informations sur la prise en charge de MFC pour UNICODE, sont désormais disponibles dans la référence de la *bibliothèque de classes*.
