# <a name="tn044-mfc-support-for-dbcs"></a>TN044 : prise en charge de DBCS par MFC

Cette note technique décrit la prise en charge dans MFC pour « jeux de double-byte character » ou DBCS. Ces informations ainsi que des informations sur MFC prend en charge pour UNICODE est désormais disponible dans le *Class Library Reference*.
