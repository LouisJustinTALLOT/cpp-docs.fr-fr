# <a name="ole-mfc"></a>OLE (MFC)
Implémentation de la fonctionnalité OLE dans votre programme affecte votre interface utilisateur de plusieurs façons :  
  
-   Édition visuelle (place activation) affiche l’interface utilisateur d’un autre programme dans les fenêtres de votre programme et modifie les menus de votre programme avec des éléments à partir de l’autre programme.  
  
-   Glisser -déplacer permet aux utilisateurs de faire glisser des objets dans et entre les fenêtres et entre des programmes.  
  
-   Dispositifs de suivi fournissent des signaux visuels à l’état des objets lors de l’édition visual et glisser -déplacer.
