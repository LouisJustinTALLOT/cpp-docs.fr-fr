# <a name="using-cmonthcalctrl"></a>Utilisation de CMonthCalCtrl

Le contrôle du calendrier du mois ([CMonthCalCtrl](../mfc/reference/cmonthcalctrl-class.md)) implémente une interface utilisateur de type calendrier. Cela fournit à l’utilisateur une méthode très intuitive et reconnaissable d’entrer ou de sélectionner une date. Le contrôle fournit également à l’application les moyens d’obtenir et de définir les informations de date dans le contrôle à l’aide de types de données existants. Par défaut, le contrôle du calendrier des mois affiche le jour et le mois en cours. Cependant, l’utilisateur est en mesure de faire défiler vers les mois précédents et les prochains et de sélectionner un mois et/ou une année spécifiques.

> [!NOTE]
> Pour utiliser `CMonthCalCtrl` la classe de votre projet, vous devez inclure AFXDTCTL. H, généralement en *stdafx.h*.
