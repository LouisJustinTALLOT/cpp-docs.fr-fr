# <a name="application-framework"></a>Infrastructure de l'application

Le cœur de la bibliothèque Microsoft Foundation classes (MFC) est une encapsulation d’une grande partie de l’API Windows sous forme de C++. Les classes de bibliothèque représentent windows, les boîtes de dialogue, les contextes de périphérique, GDI courantes objets tels que des pinceaux, des contrôles et des stylets et autres éléments Windows standards. Ces classes fournissent une pratique interface de fonction membre C++ pour les structures de Windows qui ils encapsulent. Pour plus d’informations sur l’utilisation de ces classes, consultez [rubriques d’objet de fenêtre](../mfc/window-objects.md).

Mais la bibliothèque MFC fournit également une couche supplémentaire de fonctionnalités d’application basée sur l’encapsulation C++ de l’API Windows. Cette couche est une infrastructure d’application de travail pour Windows qui fournit la majeure partie de l’interface utilisateur commune attendu des programmes pour Windows, y compris les barres d’outils, barres d’état, l’impression, aperçu avant impression, de base de données prise en charge et ActiveX. [Utilisation des Classes pour écrire des Applications pour Windows](../mfc/using-the-classes-to-write-applications-for-windows.md) explique l’infrastructure en détail.
