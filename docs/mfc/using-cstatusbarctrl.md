# Utilisation de CStatusBarCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Utilisez le contrôle de la barre d'état\([CStatusBarCtrl](../mfc/reference/cstatusbarctrl-class.md)\) pour créer un point de contrôle qui reflète différents types d'informations d'état concernant l'application.  La fenêtre d'état peut être divisé selon les sections qui affichent plusieurs types d'informations.
