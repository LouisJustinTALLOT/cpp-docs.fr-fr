# <a name="internet-and-networking-classes"></a>Internet et classes réseau

Ces classes vous permettent d’échanger des informations avec un autre ordinateur à l’aide d’un Socket de Windows ou l’Internet Win32 (WinInet). Il existe également un ensemble de classes pour la manipulation de Windows Sockets.

Les catégories suivantes de classes prend en charge la connectivité.

- [Windows Sockets, classes](../mfc/windows-sockets-classes.md)

- [Internet Win32, classes](../mfc/win32-internet-classes.md)
