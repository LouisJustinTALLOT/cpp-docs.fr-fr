Ces classes vous permettent d’échanger des informations avec un autre ordinateur à l’aide d’un socket Windows ou d’Internet Win32 (WinInet). Il existe également un ensemble de classes pour la manipulation de Windows Sockets.

Les catégories de classes suivantes prennent en charge la connectivité.

- [Windows Sockets, classes (C++)](windows-sockets-classes.md)

- [Classes Internet Win32](win32-internet-classes.md)
