# <a name="adding-an-mfc-message-handler"></a>Ajout d'un gestionnaire de messages MFC

Vous pouvez utiliser l' [Assistant classe](mfc-class-wizard.md) ou la fenêtre **Propriétés** de l' **affichage de classes** pour ajouter un gestionnaire de messages (une fonction membre qui gère les messages Windows) à une classe et mapper des messages Windows au gestionnaire de messages. Vous pouvez également ajouter [un gestionnaire d’événements pour n’importe quel contrôle de boîte de dialogue](../../windows/adding-event-handlers-for-dialog-box-controls.md).

En utilisant l' **Assistant classe** ou la fenêtre **propriétés** (dans **affichage de classes**) pour définir les fonctions de gestion des messages et des événements, vous pouvez mettre à jour automatiquement la table de distribution des messages (ou la table des messages) et votre fichier d’en-tête de classe.

> [!NOTE]
>  Vous pouvez ajouter un gestionnaire de messages à une classe ATL à l’aide de l’Assistant classe. Toutefois, certains résultats peuvent varier. Pour plus d’informations, consultez la rubrique ATL [Ajout d’un gestionnaire de messages ATL](../../atl/adding-an-atl-message-handler.md).
