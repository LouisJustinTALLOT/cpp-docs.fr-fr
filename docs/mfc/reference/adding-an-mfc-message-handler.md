# Ajout d&#39;un gestionnaire de messages MFC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Vous pouvez [utiliser la fenêtre Propriétés pour ajouter un gestionnaire de messages](../../mfc/reference/mapping-messages-to-functions.md) \(fonction membre traitant les messages Windows\) à une classe et mapper les messages Windows à celui\-ci.  Vous pouvez également ajouter [un gestionnaire d'événements pour n'importe quel contrôle de boîte de dialogue](../../mfc/adding-event-handlers-for-dialog-box-controls.md).  
  
 En utilisant la fenêtre Propriétés pour définir des fonctions de gestion des messages \(et des événements\), vous pouvez mettre à jour automatiquement la table de répartition des messages \(ou table des messages\) et le fichier d'en\-tête de votre classe.  
  
> [!NOTE]
>  Vous pouvez ajouter un gestionnaire de messages à une classe ATL à l'aide de la fenêtre Propriétés ; cependant, il se peut que certains résultats diffèrent.  Pour plus d'informations, consultez la rubrique ATL [Ajout d'un gestionnaire de messages ATL](../../atl/adding-an-atl-message-handler.md).
