# <a name="adding-an-mfc-message-handler"></a>Ajout d'un gestionnaire de messages MFC

Vous pouvez [utiliser la fenêtre Propriétés pour ajouter un gestionnaire de messages](../../mfc/reference/mapping-messages-to-functions.md) (une fonction membre qui gère les messages Windows) à une classe et mapper les messages Windows au Gestionnaire de messages. Vous pouvez également ajouter [un gestionnaire d’événements pour n’importe quel contrôle de boîte de dialogue](../../windows/adding-event-handlers-for-dialog-box-controls.md).

À l’aide de la fenêtre Propriétés pour définir des fonctions de gestion de message et d’événements, vous pouvez automatiquement mettre à jour la table de distribution des messages (ou de la table des messages) et votre fichier d’en-tête de classe.

> [!NOTE]
>  Vous pouvez ajouter un gestionnaire de messages à une classe ATL à l’aide de la fenêtre Propriétés. Toutefois, certains résultats peuvent varier. Pour plus d’informations, consultez la rubrique ATL [Ajout d’un gestionnaire de messages ATL](../../atl/adding-an-atl-message-handler.md).
