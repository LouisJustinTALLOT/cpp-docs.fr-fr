# <a name="adding-an-mfc-message-handler"></a>Ajout d'un gestionnaire de messages MFC

Vous pouvez utiliser le [Class Wizard](mfc-class-wizard.md) ou la fenêtre **Propriétés** dans **CLass View** pour ajouter un gestionnaire de message (une fonction membre qui gère les messages Windows) à une classe et cartographier les messages Windows au gestionnaire de messages. Vous pouvez également ajouter [un gestionnaire d’événements pour n’importe quel contrôle de boîte de dialogue](../../windows/adding-event-handlers-for-dialog-box-controls.md).

En utilisant la fenêtre **Class Wizard** ou **Properties** (dans **Class View**) pour définir les fonctions de traitement des messages et des événements, vous pouvez mettre à jour automatiquement la table d’expédition de messages (ou carte de message) et votre fichier d’en-tête de classe.

> [!NOTE]
> Vous pouvez ajouter un gestionnaire de message à une classe ATL à l’aide de l’assistant de classe; cependant, certains résultats peuvent varier. Pour plus d’informations, voir le sujet ATL [Ajout d’un message ATL Handler](../../atl/adding-an-atl-message-handler.md).
