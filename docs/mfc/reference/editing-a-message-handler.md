# <a name="editing-a-message-handler"></a>Modification d'un gestionnaire de messages
Une fois que vous avez défini un gestionnaire de messages, vous pouvez atteindre la définition de la fonction membre pour ajouter ou modifier le code.  
  
 Pour accéder à une définition de fonction membre à partir de la [éditeur de boîte de dialogue](../../windows/dialog-editor.md), double-cliquez sur un contrôle pour lequel un gestionnaire est déjà défini. Vous accédez au fichier dans lequel le Gestionnaire de messages du contrôle sélectionné est défini.  
  
 Pour accéder à une définition de fonction membre à partir de [affichage de classes](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), double-cliquez sur le nom de fonction dans l’affichage de classes.
