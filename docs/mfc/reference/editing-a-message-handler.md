# <a name="editing-a-message-handler"></a>Modification d'un gestionnaire de messages

Une fois que vous avez défini un gestionnaire de messages, vous pouvez accéder à la définition de la fonction membre pour ajouter ou modifier du code.

Pour accéder à une définition de fonction membre à partir de l' [éditeur de boîtes de dialogue](../../windows/dialog-editor.md), double-cliquez sur un contrôle pour lequel un gestionnaire est déjà défini. Cela vous permet d’accéder au fichier dans lequel le gestionnaire de messages du contrôle sélectionné est défini.

Pour accéder à une définition de fonction membre à partir de [affichage de classes](/visualstudio/ide/viewing-the-structure-of-code), double-cliquez sur le nom de la fonction dans affichage de classes.
