# Modification d&#39;un gestionnaire de messages
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Lorsque vous avez défini un gestionnaire de messages, vous pouvez accéder à la définition de la fonction membre afin d'ajouter ou de modifier le code.  
  
 Pour accéder à une définition de fonction membre à partir de l'[Éditeur de boîtes de dialogue](../../mfc/dialog-editor.md), double\-cliquez sur un contrôle pour lequel un gestionnaire est déjà spécifié.  Vous accédez ainsi au fichier dans lequel est défini le gestionnaire de messages du contrôle sélectionné.  
  
 Pour accéder à une définition de fonction membre à partir de l'[affichage de classes](http://msdn.microsoft.com/fr-fr/8d7430a9-3e33-454c-a9e1-a85e3d2db925), double\-cliquez sur le nom de la fonction.
