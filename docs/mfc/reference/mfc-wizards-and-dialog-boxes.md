# <a name="mfc-wizards-and-dialog-boxes"></a>MFC, assistants et boîtes de dialogue

Les assistants MFC (Microsoft Foundation Class) génèrent du code réutilisable pour différents genres de composants et d’objets. Vous pouvez exécuter les assistants en ouvrant le menu contextuel d’un projet dans **Explorateur de solutions** et en choisissant **Ajouter**, **classe**.
