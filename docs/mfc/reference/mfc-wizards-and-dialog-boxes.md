# <a name="mfc-wizards-and-dialog-boxes"></a>MFC, assistants et boîtes de dialogue
Les Assistants de Microsoft Foundation classes (MFC) génèrent un code réutilisable pour les différents types de composants et les objets. Vous pouvez exécuter les Assistants en ouvrant le menu contextuel pour un projet dans **l’Explorateur de solutions** et en choisissant **ajouter**, **classe**.
