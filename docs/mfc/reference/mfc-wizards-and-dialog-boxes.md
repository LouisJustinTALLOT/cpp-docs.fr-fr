# MFC, assistants et bo&#238;tes de dialogue
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Les assistants de Microsoft Foundation Class \(MFC\) génèrent le code passe\-partout pour différents types de composants et d'objets.  Vous pouvez exécuter les assistants en ouvrant le menu de raccourci pour un projet dans **Explorateur de solutions** et en choisissant **Ajouter**, **Classe**.
