# <a name="adding-an-mfc-class"></a>Ajout d'une classe MFC

Pour ajouter des classes dérivées des classes de bibliothèque microsoft Foundation Class (MFC) à votre projet, utilisez le bouton Ajouter la **classe** dans [Class Wizard](mfc-class-wizard.md). Spécifier le nom de la nouvelle classe, sélectionnez la classe de base et sélectionnez l’ID de la boîte de dialogue avec laquelle il est associé (le cas échéant). L’assistant crée un fichier d’en-tête et un fichier de mise en œuvre et les ajoute à votre projet.

> [!NOTE]
> Vous pouvez ajouter des classes MFC à une application ATL COM si vous avez initialement [créé l’application avec le support MFC](../../atl/reference/mfc-support-in-atl-projects.md). Vous pouvez également ajouter des classes MFC aux projets Win32 qui ont le soutien de MFC.

### <a name="to-add-an-mfc-class-to-your-project"></a>Pour ajouter une classe MFC à votre projet

De Class View, cliquez à droite sur le nom du projet. Cliquez sur **Class Wizard,** puis cliquez sur **Ajouter classe** pour ouvrir la boîte de dialogue De classe [Add MFC.](mfc-add-class-wizard.md) Utilisez **Class Wizard** pour ajouter des gestionnaires de messages ou de nouvelles méthodes.
