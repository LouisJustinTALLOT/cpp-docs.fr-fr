# <a name="adding-an-mfc-class"></a>Ajout d'une classe MFC

Pour ajouter des classes dérivées de classes de bibliothèque MFC (Microsoft Foundation Class) à votre projet, utilisez le bouton **Ajouter une classe** dans l' [Assistant classe](mfc-class-wizard.md). Spécifiez le nom de la nouvelle classe, sélectionnez la classe de base, puis sélectionnez l’ID de la boîte de dialogue à laquelle elle est associée (le cas échéant). L’Assistant crée un fichier d’en-tête et un fichier d’implémentation, puis les ajoute à votre projet.

> [!NOTE]
>  Vous pouvez ajouter des classes MFC à une application COM ATL si vous avez initialement [créé l’application avec la prise en charge MFC](../../atl/reference/mfc-support-in-atl-projects.md). Vous pouvez également ajouter des classes MFC aux projets Win32 qui prennent en charge MFC.

### <a name="to-add-an-mfc-class-to-your-project"></a>Pour ajouter une classe MFC à votre projet

Dans Affichage de classes, cliquez avec le bouton droit sur le nom du projet. Cliquez sur **Assistant classe** , puis sur **Ajouter une classe** pour ouvrir la boîte de dialogue Ajouter une [classe MFC](mfc-add-class-wizard.md) . Utilisez l' **Assistant classe** pour ajouter des gestionnaires de messages ou de nouvelles méthodes.
