 Les macros suivantes peuvent être utilisées pour gérer les événements de DHTML dans [CDHtmlDialog](../../mfc/reference/cdhtmldialog-class.md)\- classes dérivées.

## URL Mappage d'événements \(macros\)  
 Les macros suivantes peuvent être utilisées pour gérer les événements de DHTML dans [CMultiPageDHtmlDialog](../../mfc/reference/cmultipagedhtmldialog-class.md)\- classes dérivées.  
  
|||  
|-|-|  
|[BEGIN\_DHTML\_URL\_EVENT\_MAP](../Topic/BEGIN_DHTML_URL_EVENT_MAP.md)|Marque le début de la page multiple du mappage d'événement de DHTML et d'URL.|

|[END\_DHTML\_URL\_EVENT\_MAP](../Topic/END_DHTML_URL_EVENT_MAP.md)|Marque la fin de la page multiple du mappage d'événement de DHTML et d'URL.|
