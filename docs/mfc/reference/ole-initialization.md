# Initialisation d&#39;OLE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Avant qu'une application puisse utiliser OLE de services système, elle doit initialiser les DLL système OLE et vous assurer que les DLL ont la version correcte.  La fonction **AfxOleInit** initialise les DLL système OLE.
