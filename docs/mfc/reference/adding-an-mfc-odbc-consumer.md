# <a name="adding-an-mfc-odbc-consumer"></a>Ajout d'un consommateur ODBC MFC

Un consommateur ODBC MFC se compose d’une classe de Recordset ODBC et de liaisons de données nécessaires pour accéder à une source de données.

### <a name="to-add-an-mfc-odbc-consumer"></a>Pour ajouter un consommateur ODBC MFC

1. Dans Affichage de classes, cliquez avec le bouton droit sur le projet. Dans le menu contextuel, cliquez sur **Ajouter** , puis sur **Ajouter une classe**.

1. Dans le dossier Visual C++, double-cliquez sur l’icône **consommateur ODBC MFC** ou sélectionnez-la et cliquez sur **ouvrir**.

   L’Assistant consommateur ODBC MFC s’ouvre.

1. Définissez les paramètres comme décrit dans l' [Assistant consommateur ODBC MFC](../../mfc/reference/mfc-odbc-consumer-wizard.md).

1. Cliquez sur **Terminer** pour fermer l’Assistant, ce qui entraîne l’insertion du code de consommateur ODBC nouvellement créé dans votre projet.
