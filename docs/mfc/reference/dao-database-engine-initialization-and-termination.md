# Initialisation et terminaison du moteur de base de donn&#233;es DAO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

En utilisant des objets DAO de MFC, le moteur de base de données DAO doit d'abord être initialisé puis complet avant que votre application ou DLL ne se termine  Deux fonctions, `AfxDaoInit` et `AfxDaoTerm`, effectuent ces tâches.  
  
### Initialisation et terminaison du moteur de base de données DAO  
  
|||  
|-|-|  
|[AfxDaoInit](../Topic/AfxDaoInit.md)|Initialise le moteur de base de données DAO.|  
|[AfxDaoTerm](../Topic/AfxDaoTerm.md)|Arrête le moteur de base de données DAO.|
