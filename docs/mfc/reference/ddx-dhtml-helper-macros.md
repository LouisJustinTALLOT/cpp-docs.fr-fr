# DDX_DHtml Helper Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Les macros d'assistance de DDX\_DHtml permettent de faciliter l'accès aux propriétés utilisées en général des contrôles sur une page HTML.  
  
### Macros d'échange de données  
  
|||  
|-|-|  
|[DDX\_DHtml\_ElementValue](../Topic/DDX_DHtml_ElementValue.md)|Définit ou récupère la propriété de la valeur du contrôle sélectionné.|  
|[DDX\_DHtml\_ElementInnerText](../Topic/DDX_DHtml_ElementInnerText.md)|Définit ou extrait le texte entre le début et les balises de début de l'élément actuel.|  
|[DDX\_DHtml\_ElementInnerHtml](../Topic/DDX_DHtml_ElementInnerHtml.md)|Définit ou récupère le HTML entre le début et les balises de début de l'élément actuel.|  
|[DDX\_DHtml\_Anchor\_Href](../Topic/DDX_DHtml_Anchor_Href.md)|Définit ou extrait l'URL ou le point d'ancrage de destination.|  
|[DDX\_DHtml\_Anchor\_Target](../Topic/DDX_DHtml_Anchor_Target.md)|Définit ou récupère la fenêtre ou le frame cible.|  
|[DDX\_DHtml\_Img\_Src](../Topic/DDX_DHtml_Img_Src.md)|Définit ou extrait le nom d'une image ou un clip vidéo dans le document.|  
|[DDX\_DHtml\_Frame\_Src](../Topic/DDX_DHtml_Frame_Src.md)|Définit ou extrait l'URL du frame associé.|  
|[DDX\_DHtml\_IFrame\_Src](../Topic/DDX_DHtml_IFrame_Src.md)|Définit ou extrait l'URL du frame associé.|  
  
## Configuration requise  
 **Header:** afxdhtml.h
