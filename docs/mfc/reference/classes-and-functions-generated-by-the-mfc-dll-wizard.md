# <a name="classes-and-functions-generated-by-the-mfc-dll-wizard"></a>Classes et fonctions générées par l'Assistant DLL MFC

Le code généré par le MFC DLL Wizard dépend du type de DLL que vous créez et des options que vous avez sélectionnées. Le MFC DLL Wizard génère le même code pour les deux formes de DLL MFC réguliers.

|Type de DLL|Option|Classes|Fonctions|
|-----------------|------------|-------------|---------------|
|[Extension](../../build/extension-dlls-overview.md)|None|None|`DllMain`|
|[Normal](../../build/regular-dlls-dynamically-linked-to-mfc.md)|None|Catégorie d’application dérivée de`CWinApp`|None|
|[Normal](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Automatisation|Catégorie d’application dérivée de`CWinApp`|`DllGetClassObject` `DllCanUnloadNow` `DllRegisterServer`|
|[Extension](../../build/extension-dlls-overview.md)|Sockets de fenêtre|None|`DllMain`|
|[Normal](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Sockets de fenêtre|Catégorie d’application dérivée de`CWinApp`|`InitInstance`contient l’appel à`AfxSocketInit`|
