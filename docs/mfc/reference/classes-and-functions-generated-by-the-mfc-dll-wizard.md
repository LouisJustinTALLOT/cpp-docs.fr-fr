# <a name="classes-and-functions-generated-by-the-mfc-dll-wizard"></a>Classes et fonctions générées par l'Assistant DLL MFC

Le code généré par l’Assistant DLL MFC dépend du type de DLL que vous créez et des options que vous avez sélectionnées. L’Assistant DLL MFC génère le même code pour les deux formes de DLL MFC normales.

|Type de DLL|Option|Classes|Fonctions|
|-----------------|------------|-------------|---------------|
|[Extension](../../build/extension-dlls-overview.md)|None|None|`DllMain`|
|[Normal](../../build/regular-dlls-dynamically-linked-to-mfc.md)|None|Classe d’application dérivée de `CWinApp`|None|
|[Normal](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Automatisation|Classe d’application dérivée de `CWinApp`|`DllGetClassObject` `DllCanUnloadNow` `DllRegisterServer`|
|[Extension](../../build/extension-dlls-overview.md)|Sockets Windows|None|`DllMain`|
|[Normal](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Sockets Windows|Classe d’application dérivée de `CWinApp`|`InitInstance` contient un appel à `AfxSocketInit`|
