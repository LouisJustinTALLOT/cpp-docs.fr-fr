# <a name="cmfctabctrl-class"></a>CMFCTabCtrl Class

La `CMFCTabCtrl` classe fournit des fonctionnalités pour un contrôle de l’onglet. Le contrôle onglet affiche une fenêtre ancrable avec des onglets plats ou tridimensionnels en haut ou en bas. Les onglets peuvent afficher un texte et une image et peuvent changer de couleur en cas d'activation.

## <a name="syntax"></a>Syntaxe

```
class CMFCTabCtrl : public CMFCBaseTabCtrl
```

## <a name="remarks"></a>Notes

La `CMFCTabCtrl` classe prend en charge :

- Styles de contrôle d’onglets qui incluent 3D, plat, et plat avec une barre horizontale partagée de rouleau.

- Onglets situés en haut ou en bas de la fenêtre.

- Onglets qui affichent du texte, des images ou du texte et des images.

- Onglets qui changent de couleur lorsque l’onglet est actif.

- Modifications de la taille de la bordure pour les onglets réglables.

- Fenêtres tabbed amovibles.

La `CMFCTabCtrl` classe peut être utilisée avec une boîte de dialogue, mais est destinée aux applications qui utilisent des barres de contrôle d’amarrage comme Microsoft Excel et Visual Studio. Pour plus d’informations, voir [CDockablePane Class](../../mfc/reference/cdockablepane-class.md).

Suivez ces étapes pour ajouter un contrôle resizable et d’amarrage de l’onglet dans votre application :

1. Créez une instance de [la classe CTabbedPane](../../mfc/reference/ctabbedpane-class.md).

1. Appelez [CDockablePane::Créer](../../mfc/reference/cdockablepane-class.md#create).

1. Utilisez [CBaseTabbedPane::AddTab](../../mfc/reference/cbasetabbedpane-class.md#addtab) ou [CMFCBaseTabCtrl::InsertTab](../../mfc/reference/cmfcbasetabctrl-class.md#inserttab) pour ajouter de nouveaux onglets.

1. Appelez [CBasePane::EnableDocking](../../mfc/reference/cbasepane-class.md#enabledocking) de sorte que le contrôle actuel de l’onglet d’amarrage peut s’amarrer à la fenêtre du cadre principal.

1. Appelez [CFrameWndEx::DockPane](../../mfc/reference/cframewndex-class.md#dockpane) pour amarrer la fenêtre tabbed au cadre principal.

Pour un exemple de la façon de créer une fenêtre tabbed comme une barre de contrôle d’amarrage, voir [CTabbedPane Classe](../../mfc/reference/ctabbedpane-class.md). Pour `CMFCTabCtrl` utiliser comme un contrôle non-docking, créer un `CMFCTabCtrl` objet, puis appeler [CMFCTabCtrl::Créer](#create).

## <a name="cmfctabctrlsetimagelist"></a><a name="setimagelist"></a>CMFCTabCtrl::SetImageList

Spécifie une liste d’images.

```
virtual BOOL SetImageList(
    UINT uiID,
    int cx=15,
    COLORREF clrTransp=RGB(255, 0, 255));

virtual BOOL SetImageList(HIMAGELIST hImageList);
```

### <a name="parameters"></a>Paramètres

*uiID*<br/>
[dans] L’ID d’une ressource bitmap qui contient la liste d’images.

*Cx*<br/>
[dans] La largeur de chaque image, en pixels. La valeur par défaut est 15.

*clrTransp*<br/>
[dans] La couleur transparente de l’image. Les parties de l’image qui sont cette couleur seront transparentes. La valeur par défaut est la couleur magenta, RGB(255,0,255).

*hImageList*<br/>
[dans] Une poignée à une liste d’images préchargée.

### <a name="return-value"></a>Valeur de retour

VRAI si cette méthode est réussie. FALSE si le contrôle de l’onglet est créé en utilisant un style plat ou si la première surcharge de méthode ne peut pas charger la bitmap qui est spécifiée par le paramètre *uiID.*

### <a name="remarks"></a>Notes

Utilisez cette méthode pour définir une liste d’images pour le contrôle de l’onglet. Les images de la liste d’images sont affichées à côté de l’étiquette de l’onglet. Cette méthode recalcule la hauteur de l’onglet de sorte que l’onglet est dimensionné pour contenir à la fois l’image et le texte.

Utilisez la méthode [CMFCBaseTabCtrl::AddTab](../../mfc/reference/cmfcbasetabctrl-class.md#addtab) qui est héritée par le contrôle de l’onglet pour spécifier l’index de l’image à afficher.

## <a name="cmfctabctrlsynchronizescrollbar"></a><a name="synchronizescrollbar"></a>CMFCTabCtrl::SynchronizeScrollBar

```

```

### <a name="parameters"></a>Paramètres

*pScrollInfo (en anglais)*<br/>
[out] Pointeur vers une structure [SCROLLINFO](/windows/win32/api/winuser/ns-winuser-scrollinfo) ou NULL. Lorsque cette méthode revient, et si ce paramètre n’est pas NULL, la structure contient tous les paramètres de la barre de défilement. La valeur par défaut est NULL.
