# <a name="cmfcpropertygridctrl-class"></a>Classe CMFCPropertyGridCtrl

Pour plus d’informations, consultez le code source situé dans le **VC\\atlmfc\\src\\mfc** dossier de votre installation de Visual Studio.

Prend en charge un contrôle de grille de propriétés modifiable qui peut afficher des propriétés par ordre alphabétique ou hiérarchique.

## <a name="syntax"></a>Syntaxe

```
class CMFCPropertyGridCtrl : public CWnd
```

## <a name="remarks"></a>Notes

Le `CMFCPropertyGridCtrl` classe affiche un contrôle de grille de propriété qui contient les propriétés modifiables dérivées de la [CMFCPropertyGridProperty](../../mfc/reference/cmfcpropertygridproperty-class.md) classe. Chaque propriété peut représenter un type et il peut contenir des sous-éléments. Le contrôle de grille de propriétés prend en charge une zone redimensionnable en bas qui peut afficher la description d’une propriété sélectionnée.

Pour utiliser un contrôle de grille de propriétés, vous devez construire un `CMFCPropertyGridCtrl` de l’objet, puis appelez le [CMFCPropertyGridCtrl::Create](#create) (méthode). Utilisez le [CMFCPropertyGridCtrl::AddProperty](#addproperty) méthode pour ajouter des propriétés à la liste.

## <a name="selection-properties"></a>Propriétés de la sélection

Au lieu de représentant une valeur, un élément de propriété peut démarrer une boîte de dialogue qui permet à l’utilisateur sélectionner une couleur, un fichier ou une police.

Le tableau suivant répertorie les quatre types de propriété de sélection :

|Classe|Description|
|-----------|-----------------|
|[CMFCPropertyGridProperty, classe](../../mfc/reference/cmfcpropertygridproperty-class.md)|Une propriété à usage général qui est utilisée pour spécifier la valeur des chaînes, valeurs booléennes, dates et ainsi de suite.|

##  <a name="hittest"></a>  CMFCPropertyGridCtrl::HitTest

Récupère un pointeur vers l’objet de propriété qui correspond à un élément de contrôle de grille de propriétés si un point spécifié se trouve dans l’élément. Cette méthode indique également la zone dans le contrôle de grille de propriétés qui contient le point.

```
CMFCPropertyGridProperty* HitTest(
    CPoint pt,
    CMFCPropertyGridProperty::ClickArea* pnArea=NULL,
    BOOL bPropsOnly=FALSE) const;
```

### <a name="parameters"></a>Paramètres

*pt*<br/>
[in] Un point, en coordonnées clientes.

*pnArea*<br/>
[in, out] Un pointeur vers un `ClickArea` variable. Lorsque cette méthode est retournée, la variable indique la *zone de propriété* qui contient le point spécifié. Pour plus d’informations sur une zone de propriété, consultez la section Notes.

*bPropsOnly*<br/>
[in] TRUE pour tester uniquement la zone de propriété ; FALSE pour tester le *zone description* si le point spécifié ne se trouve pas dans la zone de propriété. La valeur par défaut est FALSE. Pour plus d’informations sur la zone de description, consultez la section Notes.

### <a name="return-value"></a>Valeur de retour

Si le *bPropsOnly* paramètre a la valeur TRUE et le point spécifié se trouve dans une zone de propriété, la valeur de retour est un pointeur vers l’objet de propriété correspondant. En outre, le *pnArea* est affectée à la partie qui contient le point spécifié. Sinon, la valeur de retour est NULL et la *pnArea* paramètre n’est pas modifié.

Si le *bPropsOnly* paramètre est FALSE, la valeur de retour est toujours NULL. Toutefois, si le point spécifié se trouve dans la zone description, le *pnArea* paramètre est défini sur `CMFCPropertyGridProperty::ClickDescription`.

### <a name="remarks"></a>Notes

Le terme *zone de propriété* fait référence à n’importe laquelle d'entre le nom, la valeur, ou développez les zones de zone d’un élément de contrôle de grille de propriétés. Le *zone description* est la zone en bas d’un contrôle de grille de propriété. Lorsque vous cliquez sur un élément de contrôle de grille de propriétés, la zone de description affiche une description de la propriété correspondante.

Cette méthode définit la valeur de la variable que la *pnArea* paramètre pointe vers. Le tableau suivant répertorie les valeurs possibles et les zones correspondantes.

|Value|Domaine|
|-----------|----------|
|`ClickArea::ClickExpandBox`|Propriété développez contrôle de zone.|
|`ClickArea::ClickName`|Nom de propriété.|
|`ClickArea::ClickValue`|Valeur de la propriété.|
|`CMFCPropertyGridProperty::ClickDescription`|Zone de description de contrôle de grille de propriété.|

Le nom de catégorie est affiché sur une ligne qui dirige un ensemble de valeurs et les propriétés associées.
