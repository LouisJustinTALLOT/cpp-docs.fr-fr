# Objet Globals d&#39;analyse des URL Internet
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Lorsqu'un client envoie une requête au serveur Web, vous pouvez utiliser l'un des globals de l'URL pour extraire des informations sur le client.  
  
### Objet Globals d'analyse des URL Internet  
  
|||  
|-|-|  
|[AfxParseURL](../Topic/AfxParseURL.md)|Analyse une chaîne d'URL et retourne le type de service et ses composants.|  
|[AfxParseURLEx](../Topic/AfxParseURLEx.md)|Analyse une chaîne d'URL et retourne le type de service et ses composants, et précise le nom d'utilisateur et le mot de passe.|
