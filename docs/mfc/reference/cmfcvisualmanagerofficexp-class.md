# CMFCVisualManagerOfficeXP Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

`CMFCVisualManagerOfficeXP` donne à une application une apparence de Microsoft Office XP.  
  
## Syntaxe  
  
```  
class CMFCVisualManagerOfficeXP : public CMFCVisualManager  
```  
  
## Membres  
 Tous les membres de cette classe sont des fonctions virtuelles qui sont dérivées de l'ancêtre de cette classe, [CMFCVisualManager Class](../../mfc/reference/cmfcvisualmanager-class.md).  
  
## Notes  
 Vous utilisez la classe d' `CMFCVisualManagerOfficeXP` pour modifier l'apparence visuelle de votre application pour ressembler à celle de Microsoft Office XP.
