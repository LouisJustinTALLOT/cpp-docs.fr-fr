# <a name="cmfcvisualmanagerofficexp-class"></a>CMFCVisualManagerOfficeXP, classe

`CMFCVisualManagerOfficeXP`Donne à une application une apparence Microsoft Office XP.

## <a name="syntax"></a>Syntaxe

```
class CMFCVisualManagerOfficeXP : public CMFCVisualManager
```

## <a name="members"></a>Membres

Tous les membres de cette classe sont des fonctions virtuelles dérivées de l’ancêtre de cette classe, la [classe CMFCVisualManager](../../mfc/reference/cmfcvisualmanager-class.md).

## <a name="remarks"></a>Notes

Vous utilisez la `CMFCVisualManagerOfficeXP` classe pour modifier l’apparence visuelle de votre application pour qu’elle ressemble à celle de Microsoft Office XP.
