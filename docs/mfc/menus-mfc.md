# <a name="menus-mfc"></a>Menus (MFC)

MFC fournit deux éléments pour vous aider à travailler avec les menus :

- Classe [CMenu](../mfc/reference/cmenu-class.md) pour la manipulation des menus de votre programme en cours d’exécution. Utilisez la documentation pour `CMenu` et l’exemple pour apprendre à utiliser `CMenu` efficacement.

- Un mécanisme de mise à jour des menus et boutons de barre d’outils : l’activation ou de les désactiver à la volée en fonction des conditions du programme.

Visual C++ fournit également un [éditeur de menus](../windows/menu-editor.md) pour créer et modifier des ressources de menu de votre programme.
