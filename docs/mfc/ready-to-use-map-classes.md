# <a name="ready-to-use-map-classes"></a>Classes de mappage prêtes à être utilisées

Les classes suivantes sont des classes de mappage de prêts à l’emploi.

[CMapPtrToPtr](../mfc/reference/cmapptrtoptr-class.md)<br/>
Utilise **void** pointeurs en tant que clés pour rechercher d’autres **void** pointeurs.

[CMapPtrToWord](../mfc/reference/cmapptrtoword-class.md)<br/>
Utilise **void** pointeurs en tant que clés pour rechercher des données de type **WORD**.

[CMapStringToOb](../mfc/reference/cmapstringtoob-class.md)<br/>
Utilise `CString` objets comme clés de recherche `CObject` pointeurs.

[CMapStringToPtr](../mfc/reference/cmapstringtoptr-class.md)<br/>
Utilise `CString` objets comme clés de recherche **void** pointeurs.

[CMapStringToString](../mfc/reference/cmapstringtostring-class.md)<br/>
Utilise `CString` objets comme clés pour rechercher d’autres `CString` objets.

[CMapWordToOb](../mfc/reference/cmapwordtoob-class.md)<br/>
Utilise les données de type **WORD** trouver `CObject` pointeurs.

[CMapWordToPtr](../mfc/reference/cmapwordtoptr-class.md)<br/>
Utilise les données de type **WORD** trouver **void** pointeurs.
