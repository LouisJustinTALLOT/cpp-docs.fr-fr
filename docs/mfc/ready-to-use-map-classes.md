# <a name="ready-to-use-map-classes"></a>Classes de mappage prêtes à être utilisées

Les classes suivantes sont des classes de mappage prêtes à l’emploi.

[CMapPtrToPtr](../mfc/reference/cmapptrtoptr-class.md)<br/>
Utilise **`void`** des pointeurs en tant que clés pour la recherche d’autres **`void`** pointeurs.

[CMapPtrToWord](../mfc/reference/cmapptrtoword-class.md)<br/>
Utilise des **`void`** pointeurs en tant que clés pour rechercher des données de type **Word**.

[CMapStringToOb](../mfc/reference/cmapstringtoob-class.md)<br/>
Utilise `CString` des objets comme clés pour rechercher des `CObject` pointeurs.

[CMapStringToPtr](../mfc/reference/cmapstringtoptr-class.md)<br/>
Utilise `CString` des objets comme clés pour rechercher des **`void`** pointeurs.

[CMapStringToString](../mfc/reference/cmapstringtostring-class.md)<br/>
Utilise `CString` des objets comme clés pour rechercher d’autres `CString` objets.

[CMapWordToOb](../mfc/reference/cmapwordtoob-class.md)<br/>
Utilise des données de type **Word** pour rechercher des `CObject` pointeurs.

[CMapWordToPtr](../mfc/reference/cmapwordtoptr-class.md)<br/>
Utilise des données de type **Word** pour rechercher des **`void`** pointeurs.
