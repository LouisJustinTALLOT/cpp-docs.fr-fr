# Utilisation de CComboBoxEx
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le contrôle de la zone de liste déroulante \(combo box\) étendue est une extension du contrôle zone de liste déroulante standard qui fournit une prise en charge native pour les images d'élément.  Ces images peuvent être utilisées pour indiquer l'état de chaque élément dans la zone de liste déroulante, comme les éléments sélectionnés et désélectionnés.  Pour que les images d'élément soient facilement accessibles, le contrôle fournit une prise en charge de la liste des images.  
  
 Utilisez ce contrôle pour fournir des fonctionnalités d'une zone de liste déroulante sans devoir effectuer manuellement les graphiques d'élément.
