# <a name="using-ccomboboxex"></a>Utilisation de CComboBoxEx

Le contrôle de zone de liste déroulante étendue est une extension du contrôle de zone de liste déroulante standard qui fournit la prise en charge native pour les images d’élément. Ces images peuvent être utilisées pour indiquer l’état des éléments individuels dans la zone de liste déroulante, tels que les éléments actuellement sélectionnés et désélectionnés. Pour faciliter l’accès aux images d’élément, le contrôle fournit la prise en charge de la liste d’images.

Utilisez ce contrôle pour fournir les fonctionnalités d’une zone de liste déroulante sans avoir à dessiner manuellement les graphiques d’éléments.
