# <a name="servers-user-interface-issues"></a>Serveurs : problèmes d'interface utilisateur

Une application serveur a un certain nombre de fonctionnalités qui doivent être ajoutées à l’interface utilisateur pour fournir des éléments OLE aux applications conteneur. Pour plus d’informations sur les menus et les ressources supplémentaires qui doivent être ajoutées à une application serveur, consultez [menus et ressources : ajouts de serveurs](../mfc/menus-and-resources-server-additions.md).
