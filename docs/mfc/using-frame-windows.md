# <a name="using-frame-windows"></a>Utilisation de fenêtres frame

Infrastructure MFC crée des fenêtres frame de document, vues et vos documents, dans le cadre de son implémentation des commandes nouveau et ouvrir dans le menu fichier. Étant donné que l’infrastructure effectue l’essentiel du travail de fenêtre frame pour vous, vous ne jouez qu’un rôle limité dans la création, à l’aide et la destruction de ces fenêtres. Vous pouvez, toutefois, explicitement créer vos propres fenêtres frame et fenêtres enfants à des fins spéciales.
