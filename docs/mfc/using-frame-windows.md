# <a name="using-frame-windows"></a>Utilisation de fenêtres frame

L’infrastructure MFC crée des fenêtres frame de document, ainsi que leurs vues et documents, dans le cadre de son implémentation des commandes nouveau et ouvrir dans le menu fichier. Étant donné que le Framework fait la plupart du travail de fenêtre frame pour vous, vous jouez uniquement un petit rôle dans la création, l’utilisation et la destruction de ces fenêtres. Toutefois, vous pouvez créer explicitement vos propres fenêtres frames et fenêtres enfants à des fins spéciales.
