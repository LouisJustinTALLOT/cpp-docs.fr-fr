# <a name="tn049-mfcole-mbcs-to-unicode-translation-layer-mfcans32"></a>TN049 : MBCS MFC/OLE vers couche de traduction Unicode (MFCANS32)

Cette note a initialement décrit comment MFCANS32.DLL fournit des interfaces ANSI dans le monde Unicode principalement de l’OLE 32 bits. Cette DLL n’est plus utilisée par MFC.
