# Windows Sockets
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Cette famille d'articles aborde l'implémentation MFC des Sockets Windows.  MFC fournit deux classes pour prendre en charge les applications réseau de programmation avec l'API de Sockets Windows.  La classe [CAsyncSocket](../mfc/reference/casyncsocket-class.md) inclut l'API de Sockets Windows une par une, ce qui permet aux programmeurs avancés de réseau de bénéficier du maximum de puissance et de polyvalence  La classe [CSocket](../mfc/reference/csocket-class.md) fournit une interface simplifiée pour sérialiser des données vers et à partir d'un objet de [CArchive](../mfc/reference/carchive-class.md).
