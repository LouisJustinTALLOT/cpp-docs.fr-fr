# Cr&#233;ation de fen&#234;tres
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

L'infrastructure crée automatiquement la plupart des fenêtres que vous avez besoin dans un programme\-cadre.  [Création de documents\/vue](../mfc/document-view-creation.md) montre comment l'infrastructure crée les fenêtres cadres associées aux documents et des vues.  Mais à des fins spéciaux vous pouvez créer vos propres windows — et vos propres fenêtres enfants windows cadres ou de vues — outre les fenêtres fournies par l'infrastructure.
