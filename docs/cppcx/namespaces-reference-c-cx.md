# R&#233;f&#233;rence aux espaces de noms (C++/CX)
Les articles de cette section de la documentation décrivent les espaces de noms qui prennent en charge le compilateur pour [!INCLUDE[cppwrt](../cppcx/includes/cppwrt-md.md)] \([!INCLUDE[cppwrt_short](../cppcx/includes/cppwrt-short-md.md)]\).  
  
## Espaces de noms fournis par le compilateur  
 Pour simplifier le codage des programmes qui ciblent le [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)], le compilateur [!INCLUDE[cppwrt_short](../cppcx/includes/cppwrt-short-md.md)] et ses fichiers d'en\-tête de prise en charge fournissent les espaces de noms qui définissent une large gamme de types. Les espaces de noms définissent les types numériques intégrés, les chaînes, tableaux et collections, les exceptions Visual C\+\+ qui représentent les erreurs [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)] et les améliorations propres aux langages pour les types [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)] standard.
