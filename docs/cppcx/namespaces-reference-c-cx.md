# <a name="namespaces-reference-ccx"></a>Référence aux espaces de noms (C++/CX)

Les Articles de cette section de la documentation décrivent les espaces de noms qui prennent en charge le compilateur pour C++/CX.

## <a name="compiler-supplied-namespaces"></a>Espaces de noms fournis par le compilateur

Pour simplifier le codage des programmes qui ciblent le Windows Runtime, le compilateur C++/CX et ses fichiers d’en-tête de prise en charge fournissent des espaces de noms qui définissent un large éventail de types. Les espaces de noms définissent les types numériques intégrés. chaînes, tableaux et collections ; Exceptions C++ qui représentent Windows Runtime Erreurs ; et les améliorations propres aux langages pour les types de Windows Runtime standard.
