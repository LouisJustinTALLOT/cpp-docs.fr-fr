## <a name="compiler-supplied-namespaces"></a>Espaces de noms fournis par le compilateur

Pour simplifier le codage des programmes qui ciblent le Windows Runtime, C++le compilateur/CX et ses fichiers d’en-tête de prise en charge fournissent des espaces de noms qui définissent un large éventail de types. Les espaces de noms définissent les types numériques intégrés. chaînes, tableaux et collections ; C++ les exceptions qui représentent Windows Runtime Erreurs ; et les améliorations propres aux langages pour les types de Windows Runtime standard.
