# SizeT::SizeT (constructeur)
Initialise une nouvelle instance de SizeT avec la valeur spécifiée.  
  
## Syntaxe  
  
```cpp  
SizeT( uint32 value1 );   SizeT( void* value2 );  
```  
  
## Paramètres  
 value1  
 Valeur 32 bits non signée.  
  
 value2  
 Pointeur vers une valeur 32 bits non signée.
