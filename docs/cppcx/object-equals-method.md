# Object::Equals, m&#233;thode
Détermine si l'objet spécifié est identique à l'objet actuel.  
  
## Syntaxe  
  
```cpp  
  
bool Equals(  
    Object^ obj  
)  
```  
  
## Paramètres  
 obj  
 Objet à comparer.  
  
## Valeur de retour  
 `true` si les objets sont identiques ; sinon, `false`.
