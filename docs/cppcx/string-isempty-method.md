# String::IsEmpty, m&#233;thode
Indique si l'objet String actuel est vide.  
  
## Syntaxe  
  
```cpp  
  
bool IsEmpty()  
```  
  
## Valeur de retour  
 `true` si l'objet String actuel est `null` ou une chaîne vide \(L ""\) ; sinon, `false`.
