# Guid::operator(), op&#233;rateur
Convertit implicitement un GUID de [structure GUID](http://msdn.microsoft.com/library/windows/desktop/aa373931\(v=vs.85\).aspx) en Platform::Guid.  
  
## Syntaxe  
  
```cpp  
Platform::Guid operator()  
```  
  
## Valeur de retour  
 Structure de GUID.
