Remarque : Visual C++ ne prend pas en charge les optimisations guidées par profil pour la plateforme Windows universelle. Si vous essayez de générer un projet avec ces options dans l'environnement IDE, une erreur de build se produit. Les applications console ne sont pas prises en charge.
