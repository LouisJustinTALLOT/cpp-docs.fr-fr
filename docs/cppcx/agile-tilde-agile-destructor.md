# Agile::~Agile, destructeur
Détruit l'instance actuelle de la classe Agile.  
  
## Syntaxe  
  
```cpp  
  
~Agile();  
```  
  
## Notes  
 Ce destructeur libère aussi l'objet représenté par l'objet Agile actuel.
