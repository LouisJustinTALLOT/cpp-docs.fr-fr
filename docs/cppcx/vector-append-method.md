# Vector::Append, m&#233;thode
Insère l'élément spécifié après le dernier élément du Vector actif.  
  
## Syntaxe  
  
```  
  
virtual void Append(  
   T item  
);  
```  
  
#### Paramètres  
 `index`  
 Élément à insérer dans le Vector. Le type de `item` est défini par le nom de type *T*.  
  
## Configuration requise  
 **En\-tête :** collection.h  
  
 **Espace de noms :** Platform::Collections
