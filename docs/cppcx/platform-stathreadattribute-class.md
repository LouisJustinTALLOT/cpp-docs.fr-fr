# Platform::STAThreadAttribute (classe)
Indique que le modèle de thread d'une application est un modèle STA \(Single\-Threaded Apartment\).  
  
## Syntaxe  
  
```  
public ref class STAThreadAttribute sealed : Attribute  
```  
  
## Membres  
  
### Constructeurs publics  
  
|Nom|Description|  
|---------|-----------------|  
|[STAThreadAttribute \(constructeur\)](../cppcx/stathreadattribute-constructor-1.md)|Initialise une nouvelle instance de la classe.|  
  
### M&\#233;thodes publiques  
 L’attribut STAThreadAttribute hérite de [Platform::Object, classe](../cppcx/platform-object-class.md). STAThreadAttribute surcharge ou possède également les membres suivants :  
  
|Nom|Description|  
|---------|-----------------|  
|[STAThreadAttribute::Equals](../cppcx/stathreadattribute-equals.md)|Détermine si l'objet spécifié est identique à l'objet actuel.|  
|[STAThreadAttribute::GetHashCode](../cppcx/stathreadattribute-gethashcode.md)|Retourne le code de hachage de cette instance.|  
|[STAThreadAttribute::ToString](../cppcx/stathreadattribute-tostring.md)|Retourne une chaîne qui représente l'objet actuel.|
