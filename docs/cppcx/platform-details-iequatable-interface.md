# Platform::Details::IEquatable (interface)
Cette interface n’est destinée qu’à une utilisation interne, et non à des fins de développement.  
  
## Syntaxe  
  
```cpp  
  
interface class IEquatable;  
```  
  
## Notes  
 Cette interface est fournie uniquement par souci d’exhaustivité, car elle peut être inspectée avec les navigateurs.
