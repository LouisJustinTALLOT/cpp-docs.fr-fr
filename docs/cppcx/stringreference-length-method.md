# StringReference::Length, m&#233;thode
Retourne le nombre de caractères de la chaîne.  
  
## Syntaxe  
  
```cpp  
unsigned int Length() const  
```  
  
## Valeur de retour  
 Entier non signé qui Spécifie le nombre de caractères de la chaîne.
