# String::Begin, m&#233;thode
Retourne un pointeur au début de la chaîne actuelle.  
  
## Syntaxe  
  
```cpp  
  
char16* Begin()  
```  
  
## Valeur de retour  
 Pointeur au début de la chaîne actuelle.
