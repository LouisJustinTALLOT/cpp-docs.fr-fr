# Platform::Metadata::FlagsAttribute (attribut)
Indique qu'une énumération peut être traitée comme un champ de bits, c'est\-à\-dire un ensemble d'indicateurs.  
  
## Syntaxe  
  
```cpp  
  
public ref class Flags abstract : Attribute  
```
