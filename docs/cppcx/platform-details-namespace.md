# <a name="platformdetails-namespace"></a>Platform::Details (espace de noms)
Cet espace de noms n'est destiné qu'à une utilisation interne et non à des fins de développement.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
  
namespace Platform {  
   namespace Details {  
}}  
```
