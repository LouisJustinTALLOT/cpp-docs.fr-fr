# STAThreadAttribute::Equals
Détermine si l'objet spécifié est identique à l'objet actuel.  
  
## Syntaxe  
  
```cpp  
public:virtual override bool Equals(  Object^ obj)  
```  
  
## Paramètres  
 obj  
 Objet à comparer.  
  
## Valeur de retour  
 `true` si les objets sont égaux ; sinon `false`.
