# IBox::Value, propri&#233;t&#233;
Retourne la valeur qui a été enregistrée à l'origine dans cet objet.  
  
## Syntaxe  
  
```cpp  
property T Value {T get();}  
```  
  
```csharp  
  
```  
  
#### Paramètres  
 `T`  
 Type de la valeur boxed.  
  
## Valeur de propriété\/valeur de retour  
 Retourne la valeur qui a été enregistrée à l'origine dans cet objet.
