# Platform::MTAThreadAttribute (classe)
Indique que le modèle de thread d'une application est un modèle MTA \(MultiThreaded Apartment\).  
  
## Syntaxe  
  
```  
public ref class MTAThreadAttribute sealed : Attribute  
```  
  
## Membres  
  
### Constructeurs publics  
  
|Nom|Description|  
|---------|-----------------|  
|Constructeur [MTAThreadAttribute, constructeur](../cppcx/mtathreadattribute-constructor-1.md)|Initialise une nouvelle instance de la classe.|  
  
### M&\#233;thodes publiques  
 L’attribut MTAThreadAttribute hérite de [Platform::Object, classe](../cppcx/platform-object-class.md). MTAThreadAttribute surcharge ou possède également les membres suivants :  
  
|Nom|Description|  
|---------|-----------------|  
|[MTAThreadAttribute::Equals](../cppcx/mtathreadattribute-equals.md)|Détermine si l'objet spécifié est identique à l'objet actuel.|  
|[MTAThreadAttribute::GetHashCode](../cppcx/mtathreadattribute-gethashcode.md)|Retourne le code de hachage de cette instance.|  
|[MTAThreadAttribute::ToString](../cppcx/mtathreadattribute-tostring.md)|Retourne une chaîne qui représente l'objet actuel.|
