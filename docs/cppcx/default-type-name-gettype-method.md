# default::(type_name)::GetType, m&#233;thode
Retourne un Platform::Type^ qui représente le type actif.  
  
## Syntaxe  
  
```cpp  
  
Platform::Type^ GetType()  
```  
  
## Valeur de retour  
 Objet [Platform::Type](../cppcx/platform-type-class.md)^ qui représente l'objet actif.
