# IntPtr::ToInt32 (m&#233;thode)
Convertit la valeur IntPtr actuelle en un entier 32 bits.  
  
## Syntaxe  
  
```cpp  
int32 IntPtr::ToInt32();  
```  
  
## Valeur de retour  
 Entier 32 bits.
