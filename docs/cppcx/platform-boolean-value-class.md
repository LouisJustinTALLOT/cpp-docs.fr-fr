# Classe de valeur Platform::Boolean
Représente une valeur booléenne. Équivalent de `bool`.  
  
## Syntaxe  
  
```cpp  
public value struct Boolean  
```  
  
## Membres  
 Boolean contient les méthodes Equals\(\), GetHashCode\(\), et ToString\(\) dérivées de la [Platform::Object, classe](../cppcx/platform-object-class.md), et la méthode GetTypeCode\(\) dérivée de la [Platform::Type \(classe\)](../cppcx/platform-type-class.md).
