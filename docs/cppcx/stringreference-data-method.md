# StringReference::Data, m&#233;thode
Retourne le contenu de ce `StringReference` sous forme de tableau de valeurs char16.  
  
## Syntaxe  
  
```cpp  
const ::default::char16 * Data() const  
```  
  
## Valeur de retour  
 Tableau de caractères de texte UNICODE char16.
