# Map::Size, m&#233;thode
Retourne le nombre d’éléments [Windows::Foundation::Collections::IKeyValuePair\<K,V\>](http://msdn.microsoft.com/library/windows/apps/br226031.aspx) contenus dans le Map.  
  
## Syntaxe  
  
```  
  
virtual property unsigned int Size;  
```  
  
## Valeur de retour  
 Nombre d'éléments du Map.  
  
## Configuration requise  
 **En\-tête :** collection.h  
  
 **Espace de noms :** Platform::Collections
