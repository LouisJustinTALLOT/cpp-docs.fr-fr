# <a name="wrl-reference"></a>Référence WRL

Cette section contient des informations de référence sur C++ la bibliothèque de modèles Windows Runtime (WRL).

> [!NOTE]
> La bibliothèque C++ de modèles Windows Runtime définit les fonctionnalités qui prennent C++ en charge l’infrastructure de la bibliothèque de modèles Windows Runtime et n’est pas destinée à être utilisée directement à partir de votre code. Cette fonctionnalité est indiquée dans cette documentation.

## <a name="in-this-section"></a>Dans cette section

[Microsoft::WRL, espace de noms](microsoft-wrl-namespace.md)<br/>
Définit les types fondamentaux qui composent la C++ bibliothèque de modèles Windows Runtime.

[Microsoft::WRL::Wrappers, espace de noms](microsoft-wrl-wrappers-namespace.md)<br/>
Définit des types de wrappers RAII (Resource Acquisition Is Initialization) qui simplifient la gestion de la durée de vie des objets, des chaînes et des handles.

[Microsoft::WRL::Wrappers::HandleTraits, espace de noms](microsoft-wrl-wrappers-handletraits-namespace.md)<br/>
Décrit les caractéristiques des types de ressources basés sur les descripteurs courants.

[Windows::Foundation, espace de noms](windows-foundation-namespace.md)<br/>
Active les fonctionnalités fondamentales de Windows Runtime, telles que la création d’objets et de fabriques.
