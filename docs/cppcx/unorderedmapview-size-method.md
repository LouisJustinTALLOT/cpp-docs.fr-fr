# UnorderedMapView::Size, m&#233;thode
Retourne le nombre d’éléments [Windows::Foundation::Collections::IKeyValuePair\<K,V\>](http://msdn.microsoft.com/library/windows/apps/br226031.aspx) de l’objet UnorderedMapView.  
  
## Syntaxe  
  
```cpp  
  
virtual property unsigned int Size;  
```  
  
## Valeur de retour  
 Nombre d'éléments contenus dans l'objet UnorderedMapView actif.  
  
## Configuration requise  
 **En\-tête :** collection.h  
  
 **Espace de noms :** Platform::Collections
