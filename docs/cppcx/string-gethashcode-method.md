# String::GetHashCode, m&#233;thode
Retourne le code de hachage de cette instance.  
  
## Syntaxe  
  
```cpp  
  
virtual override int GetHashCode()  
```  
  
## Valeur de retour  
 Code de hachage de cette instance.
