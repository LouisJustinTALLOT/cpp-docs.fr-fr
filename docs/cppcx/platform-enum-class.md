# <a name="platformenum-class"></a>Platform::Enum, classe

Classe value qui représente un ensemble de constantes nommées.

## <a name="syntax"></a>Syntaxe

```cpp
public class Enum
```

### <a name="members"></a>Membres

La classe Enum hérite des méthodes Equals(), GetHashCode() et ToString() de la [Platform::Object Class](../cppcx/platform-object-class.md).

### <a name="remarks"></a>Notes

Utilisez le mot clé [classe enum publique](../extensions/enum-class-cpp-component-extensions.md) pour créer des énumérations. N'utilisez pas le type Platform::Enum explicitement. Pour plus d'informations, consultez [Énumérations](../cppcx/enums-c-cx.md).

### <a name="requirements"></a>Configuration requise

**Prise en charge minimale du client :** Windows 8

**Serveur pris en charge minimale :** Windows Server 2012

**Espace de noms :** Plateforme

**Métadonnées :** platform.winmd
