# <a name="safeint-library"></a>Bibliothèque SafeInt

**SafeInt** est une bibliothèque portable qui peut être utilisée avec MSVC, GCC ou Clang pour éviter les dépassements d’entiers pouvant survenir lorsque l’application exécute des opérations mathématiques. La dernière version de cette bibliothèque se trouve dans [https://github.com/dcleblanc/SafeInt](https://github.com/dcleblanc/SafeInt).
