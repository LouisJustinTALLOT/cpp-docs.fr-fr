# <a name="iob"></a>_iob

Tableau de structures de contrôle stdio.

## <a name="syntax"></a>Syntaxe

```
FILE _iob[_IOB_ENTRIES];
```

## <a name="remarks"></a>Notes

`IOB_ENTRIES` est défini en tant que 20 dans stdio.h.
