# <a name="exit_success-exit_failure"></a>EXIT_SUCCESS, EXIT_FAILURE

## <a name="required-header"></a>En-tête requis

```c
#include <stdlib.h>
```

## <a name="remarks"></a>Notes

Il s’agit d’arguments pour les fonctions [exit](reference/exit-exit-exit.md) et [_exit](reference/exit-exit-exit.md), et les valeurs de retour pour les fonctions [atexit](reference/atexit.md) et [_onexit](reference/onexit-onexit-m.md).

|Constante|Valeur définie|
|-|-|
|EXIT_SUCCESS|0|
|EXIT_FAILURE|1|
