# <a name="randmax"></a>RAND_MAX

## <a name="syntax"></a>Syntaxe

```
#include <stdlib.h>
```

## <a name="remarks"></a>Remarques

La constante `RAND_MAX` est la valeur maximale qui peut être retournée par la fonction `rand`. `RAND_MAX` est défini comme la valeur 0x7fff.
