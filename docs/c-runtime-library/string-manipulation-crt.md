# Manipulation de cha&#238;nes (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Ces routines traitent les caractères codés sur un octet et terminés par null, les caractères élargis, ainsi que les chaînes de caractères codées sur plusieurs octets.  Utilisez les routines de manipulation de mémoire tampon, décrites dans [Manipulation de mémoire tampon](../c-runtime-library/buffer-manipulation.md), pour travailler avec les tableaux de caractères qui ne se terminent pas avec un caractère Null.
