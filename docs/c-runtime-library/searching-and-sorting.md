# <a name="searching-and-sorting"></a>Recherche et tri

Utilisez les fonctions suivantes pour la recherche et le tri.

## <a name="searching-and-sorting-functions"></a>Fonctions de recherche et de tri

|Fonction|Rechercher ou trier|
|--------------|--------------------|
|[bsearch](../c-runtime-library/reference/bsearch.md)|Recherche binaire|
|[bsearch_s](../c-runtime-library/reference/bsearch-s.md)|Version plus sécurisée de **bsearch**|
|[_lfind](../c-runtime-library/reference/lfind.md)|Recherche linéaire pour une valeur donnée|
|[_lfind_s](../c-runtime-library/reference/lfind-s.md)|Version plus sécurisée de **_lfind**|
|[_lsearch](../c-runtime-library/reference/lsearch.md)|Recherche linéaire pour une valeur donnée, ajoutée au tableau si introuvable|
|[_lsearch_s](../c-runtime-library/reference/lsearch-s.md)|Version plus sécurisée de **_lsearch**|
|[qsort](../c-runtime-library/reference/qsort.md)|Tri rapide|
|[qsort_s](../c-runtime-library/reference/qsort-s.md)|Version plus sécurisée de **qsort**|
