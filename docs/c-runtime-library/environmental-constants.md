# Constantes d&#39;environnement
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Syntaxe  
  
```  
#include <stdlib.h>  
```  
  
## Notes  
 Cette constante définit la longueur d'environnement pour les chaînes.  
  
|Constante|Signification|  
|---------------|-------------------|  
|`_MAX_ENV`|Taille maximale de la plage de chaîne environnementale.|
