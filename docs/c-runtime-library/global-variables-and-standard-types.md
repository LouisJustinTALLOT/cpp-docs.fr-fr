# Variables globales et types standard
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La bibliothèque Runtime de Microsoft contient des définitions pour [variables globales](../c-runtime-library/global-variables.md), [indicateurs de contrôle](../c-runtime-library/control-flags.md), et [types standard](../c-runtime-library/standard-types.md) utilisé par les routines de bibliothèque.  Accédez à ces variables, indicateurs, et types en les déclarant dans votre programme ou en incluant les fichiers d'en\-tête appropriés.
