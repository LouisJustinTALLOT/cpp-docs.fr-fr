# FOPEN_MAX, _SYS_OPEN
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Syntaxe  
  
```  
  
#include <stdio.h>  
  
```  
  
## Notes  
 Il s'agit du nombre maximal de fichiers qui peuvent être ouverts simultanément.  `FOPEN_MAX` is le nom compatible avec ANSI.  `_SYS_OPEN` est fourni pour la compatibilité avec le code existant.
