# <a name="windows-platforms-crt"></a>Plateformes Windows (CRT)
Les bibliothèques Runtime C pour [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] prennent en charge [!INCLUDE[win8](../build/reference/includes/win8_md.md)], [!INCLUDE[winserver8](../build/reference/includes/winserver8_md.md)], [!INCLUDE[win7](../build/includes/win7_md.md)], [!INCLUDE[winsvr08](../build/reference/includes/winsvr08_md.md)], Windows Vista, [!INCLUDE[winxp](../build/includes/winxp_md.md)] Service Pack 3 (SP3) pour x86, [!INCLUDE[winxp](../build/includes/winxp_md.md)] Service Pack 2 (SP2) pour x64 et [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] Service Pack 2 (SP2) pour x86 et x64. Tous ces systèmes d’exploitation prennent en charge l’interface de programmation d’applications (API) Windows et fournissent la prise en charge d’Unicode. En outre, toute application Win32 peut utiliser un jeu de caractères multioctets (MBCS).  
  
> [!NOTE]
>  [!INCLUDE[vs_dev11_long](../build/includes/vs_dev11_long_md.md)] ne prend pas en charge le développement [!INCLUDE[winxp](../build/includes/winxp_md.md)] et [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)]. Vous devez installer [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] pour ajouter la prise en charge de [!INCLUDE[winxp](../build/includes/winxp_md.md)] et [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)].
