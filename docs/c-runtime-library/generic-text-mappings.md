# <a name="generic-text-mappings"></a>Mappages de texte générique

Pour simplifier l’écriture de code pour les marchés internationaux, les mappages de texte générique sont définis dans TCHAR.H pour ce qui suit :

- [Types de données](../c-runtime-library/data-type-mappings.md)

- [Constantes et variables globales](../c-runtime-library/constant-and-global-variable-mappings.md)

- [Mappages de routine](../c-runtime-library/routine-mappings.md)

Pour plus d’informations, consultez [Utilisation de mappages de texte générique](../c-runtime-library/using-generic-text-mappings.md). Les mappages de texte générique sont des extensions Microsoft non compatibles ANSI.
