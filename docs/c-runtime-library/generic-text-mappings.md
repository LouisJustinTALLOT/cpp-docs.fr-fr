# Mappages de texte g&#233;n&#233;rique
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Pour simplifier le code d'écriture pour les marchés internationaux, les mappages de texte générique sont définis dans TCHAR.H pour :  
  
-   [Types de données](../c-runtime-library/data-type-mappings.md)  
  
-   [Variables et constantes globales](../c-runtime-library/constant-and-global-variable-mappings.md)  
  
-   [Mappages de routine](../c-runtime-library/routine-mappings.md)  
  
 Pour plus d'informations, consultez [Mappages de texte générique dans Tchar.h](../c-runtime-library/using-generic-text-mappings.md).  Les mappings de texte générique sont des extensions  Microsoft qui ne sont pas compatibles pour ANSI.
