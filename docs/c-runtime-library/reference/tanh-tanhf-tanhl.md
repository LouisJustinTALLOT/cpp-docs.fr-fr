# <a name="tanh-tanhf-tanhl"></a>tanh, tanhf, tanhl

Calcule la tangente hyperbolique.

## <a name="syntax"></a>Syntaxe

```C
double tanh( double x );
float tanhf( float x );
long double tanhl( long double x );
```

```cpp
float tanh( float x );  // C++ only
long double tanh( long double x );  // C++ only
```

### <a name="parameters"></a>Paramètres

*x*<br/>
Angle en radians.

## <a name="return-value"></a>Valeur retournée

Les fonctions **Tanh** retournent la tangente hyperbolique de *x*. Aucun retour d'erreur.

|Entrée|Exception SEH|**Supertherr** Titre|
|-----------|-------------------|-------------------------|
|± QNAN,IND|Aucun|_DOMAIN|

## <a name="remarks"></a>Notes 

C++ autorisant la surcharge, vous pouvez appeler des surcharges de **Tanh** qui acceptent et retournent des valeurs **float** ou **long** **double** . Dans un programme C, **Tanh** prend toujours et retourne **double**.

Par défaut, l’état global de cette fonction est limité à l’application. Pour modifier cette valeur, consultez [état global dans le CRT](../global-state.md).

|-------------|---------------------|-|

Pour plus d'informations sur la compatibilité, voir [Compatibilité](../../c-runtime-library/compatibility.md).

## <a name="example"></a>Exemple

```C
// crt_tanh.c
// This program displays the tangent of pi / 4
// and the hyperbolic tangent of the result.
// Compile by using: cl crt_tanh.c

#include <math.h>
#include <stdio.h>

int main( void )
{
   double pi = 3.1415926535;
   double x, y;

   x = tan( pi / 4 );
   y = tanh( x );
   printf( "tan( %f ) = %f\n", pi/4, x );
   printf( "tanh( %f ) = %f\n", x, y );
}
```

```Output
tan( 0.785398 ) = 1.000000
tanh( 1.000000 ) = 0.761594
```
