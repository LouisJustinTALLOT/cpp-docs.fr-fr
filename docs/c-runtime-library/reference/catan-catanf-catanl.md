# catan, catanf, catanl
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Récupère l’arc tangente d’un nombre complexe par morceaux de branche en dehors de l’intervalle \[−1 ; \+ 1\] sur l’axe imaginaire.  
  
## Syntaxe  
  
```  
_Dcomplex catan(   
   _Dcomplex z   
);  
_Fcomplex catan(   
   _Fcomplex z   
);  // C++ only  
_Lcomplex catan(   
  _Lcomplex z   
);  // C++ only  
_Fcomplex catanf(   
   _Fcomplex z   
);  
_Lcomplex catanl(   
   _Lcomplex z   
);  
```  
  
#### Paramètres  
 `z`  
 Un nombre complexe qui représente l’angle, en radians.  
  
## Valeur de retour  
 L’arc tangente de `z`, en radians. Le résultat est illimité sur l’axe imaginaire et dans l’intervalle \[−π\/2 ; \+ π\/2\] sur l’axe réel.  
  
## Notes  
 C\+\+ autorisant la surcharge, vous pouvez appeler des surcharges de `catan` qui acceptent et retournent des valeurs `_Fcomplex` et `_Lcomplex`. Dans un programme C, `catan` accepte et retourne toujours un `_Dcomplex` valeur.  
  
## Configuration requise

 Pour plus d'informations sur la compatibilité, voir [Compatibilité](../../c-runtime-library/compatibility.md) dans l'introduction.
