Cette fonction POSIX est déconseillée. Utilisez plutôt [_spawnve](spawnve-wspawnve.md), qui est conforme à la norme ISO C++.

> Cette API ne peut pas être utilisée dans les applications qui s’exécutent dans le Windows Runtime.
