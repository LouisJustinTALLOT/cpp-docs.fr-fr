# <a name="bitand"></a>bitand

Une alternative à l’opérateur &.

## <a name="syntax"></a>Syntaxe

```C

#define bitand &
```

## <a name="remarks"></a>Notes

La macro génère l'opérateur

## <a name="example"></a>Exemple

```cpp
// iso646_bitand.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 1, b = 2, result;

   result = a & b;
   cout << result << endl;

   result= a bitand b;
   cout << result << endl;
}
```

```Output
0
0
```
