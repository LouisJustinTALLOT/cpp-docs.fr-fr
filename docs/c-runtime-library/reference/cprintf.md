# <a name="cprintf"></a>cprintf

Cette fonction POSIX est déconseillée. Utilisez plutôt [_cprintf](cprintf-cprintf-l-cwprintf-cwprintf-l.md) ou sa version sécurisée [_cprintf_s](cprintf-s-cprintf-s-l-cwprintf-s-cwprintf-s-l.md), qui sont conformes à la spécification ISO C++.

> [!IMPORTANT]
> Cette API ne peut pas être utilisée dans les applications qui s’exécutent dans le Windows Runtime. Pour plus d’informations, consultez [Fonctions CRT non prises en charge dans les applications de la plateforme Windows universelle](../../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md).
