# <a name="internationalization"></a>Internationalisation

La bibliothèque Microsoft Runtime fournit de nombreuses routines que vous pouvez utiliser pour personnaliser votre application pour les marchés internationaux, par exemple :
- [routines liées aux paramètres régionaux](../c-runtime-library/locale.md)
- routines à caractères larges
- routines de caractères multioctets
- routines de texte générique

Pour des raisons pratiques, la plupart des routines liées aux paramètres régionaux sont également classées par ce qu’elles font.

Les routines multioctets et les routines à caractères larges sont décrites avec leurs équivalents de caractères codés sur un octet, lorsqu’ils existent.

Les alternatives d’opérateur ISO646 sont également incluses.
