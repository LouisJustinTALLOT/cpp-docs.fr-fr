# <a name="null-crt"></a>NULL (CRT)

**NULL** est la valeur de pointeur null utilisée avec de nombreuses fonctions et opérations de pointeur. Il équivaut à 0. **NULL** est définie dans les fichiers d’en-tête suivants : CRTDBG.H, LOCALE.H, STDDEF.H, STDIO.H, STDLIB.H, STRING.H, TCHAR.H, TIME.H et WCHAR.H.
