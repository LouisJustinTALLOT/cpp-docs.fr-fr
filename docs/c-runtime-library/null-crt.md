# NULL (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**NULL** est la valeur du pointeur null utilisée avec de nombreuses opérations du pointeur et fonctions.  Est équivalent à 0.  **NULL** est défini dans les fichiers d'en\-tête suivants : CRTDBG.H, LOCALE.H, STDDEF.H, STDIO.H, STDLIB.H, STRING.H, TCHAR.H, TIME.H et. WCHAR.H.
