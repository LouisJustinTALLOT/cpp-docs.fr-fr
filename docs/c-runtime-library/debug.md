# _DEBUG
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le compilateur définit `_DEBUG` lorsque vous spécifiez l'option \/MTd ou \/MDd.  Ces options spécifient les versions de débogage de la bibliothèque Runtime C.  
  
 Pour plus d'informations, consultez [CRT Debugging Techniques](../Topic/CRT%20Debugging%20Techniques.md).
