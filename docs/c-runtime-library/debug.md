# <a name="debug"></a>_DEBUG
Le compilateur définit `_DEBUG` quand vous spécifiez l’option /MTd ou /MDd. Ces options spécifient les versions Debug de la bibliothèque Runtime C.  
  
 Pour plus d’informations, consultez [Techniques de débogage CRT](/visualstudio/debugger/crt-debugging-techniques).
