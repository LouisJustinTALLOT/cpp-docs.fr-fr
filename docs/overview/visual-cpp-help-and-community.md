# <a name="microsoft-cc-help-and-community"></a>Aide et communauté Microsoft C/C++

Voici comment obtenir de l’aide et des informations sur l’écriture de code C++ et l’utilisation des outils de développement Visual Studio.

## <a name="samples"></a>Exemples

- [Exemples de code de développeur](/samples): contient un exemple de code téléchargeable de Microsoft et des contributeurs de la communauté.

## <a name="product-documentation"></a>Documentation du produit

- [C++ dans Visual Studio](visual-cpp-in-visual-studio.md): contient une documentation de référence et conceptuelle sur Visual C++.

- [Centre de développement Windows](https://developer.microsoft.com/windows/): contient des informations sur l’utilisation de C++ et d’autres langages pour développer des applications pour Windows 10.

### <a name="online-and-offline-documentation"></a>Documentation en ligne et hors connexion

Vous pouvez afficher le contenu de développement Microsoft en ligne. Ce contenu est mis à jour régulièrement.

Vous pouvez également télécharger et afficher le contenu localement dans la Visionneuse de l’aide hors connexion. La documentation hors connexion est organisée par ouvrages de contenu associé, lesquels sont également mis à jour périodiquement. Vous pouvez télécharger les ouvrages qui vous intéressent, une fois qu’ils sont disponibles. Pour plus d’informations, consultez [Visionneuse d’aide Microsoft](/visualstudio/ide/microsoft-help-viewer).

De nombreuses sections de la documentation sont également disponibles au format PDF. Ces sections comportent un lien **Télécharger au format PDF** sous la table des matières sur docs.Microsoft.com.

## <a name="related-sites"></a>Sites connexes

- [Blog de l’équipe c++](https://devblogs.microsoft.com/cppblog/): contient des publications sur divers sujets par les experts de l’équipe produit Microsoft c++.

- [Channel 9](https://channel9.msdn.com/): contient des interviews vidéo et des conférences. Utilisez la zone de recherche dans la page d’accueil de Channel 9 pour rechercher du contenu C++.

- [Site Web Visual Studio](https://visualstudio.microsoft.com/): contient des articles et des informations sur Visual Studio et les outils de développement associés. Ce site héberge également des téléchargements Visual Studio et des fichiers redistribuables.

- [Microsoft docs Q&a](/answers/topics/c%2B%2B.html): forums Microsoft officiels où vous pouvez poser des questions sur C++ et Visual Studio et obtenir des réponses de Microsoft et des experts de la communauté.

- [Communauté des développeurs Visual Studio c++](https://aka.ms/vsfeedback/browsecpp): forums pour signaler des problèmes dans le compilateur et les outils C++ ou l’IDE, ou pour faire des suggestions de produit. Vous pouvez voter pour vos idées favorites ou vos rapports de bogues, ou en ajouter un nouveau. Ce site est analysé par l’équipe de développement.
