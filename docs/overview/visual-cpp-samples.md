# <a name="visual-studio-c-samples"></a>Échantillons Visual Studio CMD

Des échantillons pour Visual Studio CMD sont disponibles sur le web. Microsoft a produit de nombreux échantillons de CMD qui démontrent différentes fonctionnalités à travers de multiples technologies. Voici quelques-uns des endroits pour trouver des échantillons supplémentaires:

- [Échantillons de Microsoft Docs - C](https://docs.microsoft.com/samples/browse/?term=c%2B%2B)

- [Échantillons de fenêtres sur GitHub](https://microsoft.github.io/windows/)

- [Échantillons de code Windows Dev Center](https://developer.microsoft.com/windows/samples)

- [Archives d’échantillons CodePlex](https://archive.codeplex.com/)

- [Exemples de code ADO](/office/client-developer/access/desktop-database-reference/ado-code-examples-in-microsoft-visual-c)

- [Exemples de développement de matériel pour Windows](https://docs.microsoft.com/samples/browse/?products=windows-wdk)

## <a name="archived-c-samples-on-github"></a>Échantillons archivés de CMD sur GitHub

Visual Studio a inclus le code d’échantillon de CMD dans les versions précédentes. Le code de l’échantillon a été installé avec Visual Studio, soit disponible en téléchargement séparé. De nombreux articles dans notre documentation se réfèrent à ces échantillons. Ils ne sont plus installés par Visual Studio. Au lieu de cela, un référentiel est disponible sur GitHub. Les tableaux ci-dessous ont des descriptions pour chaque échantillon, et des liens vers l’annuaire de l’échantillon dans le référentiel.
