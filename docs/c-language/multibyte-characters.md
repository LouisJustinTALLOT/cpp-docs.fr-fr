# <a name="multibyte-characters"></a>Caractères multioctets
**ANSI 2.2.1.2** États de décalage pour les caractères multioctets  
  
 Les caractères multioctets sont utilisés par certaines implémentations, y compris Microsoft C, pour représenter des caractères de langue étrangère qui ne sont pas représentés dans le jeu de caractères de base. Toutefois, Microsoft C ne prend en charge aucun encodage dépendant de l'état. Par conséquent, aucun état de décalage n'est présent. Consultez [Caractères multioctets et larges](../c-language/multibyte-and-wide-characters.md) pour plus d'informations.
