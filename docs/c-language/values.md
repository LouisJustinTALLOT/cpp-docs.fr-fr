# <a name="values"></a>Valeurs

**ANSI 3.1.2.5** Représentations et jeux de valeurs des différents types de nombres à virgule flottante

Le **`float`** type contient 32 bits : 1 pour le signe, 8 pour l’exposant et 23 pour la mantisse. Sa plage est +/- 3.4E38 avec au moins 7 chiffres de précision.

Le **`double`** type contient 64 bits : 1 pour le signe, 11 pour l’exposant et 52 pour la mantisse. Sa plage est +/- 1.7E308 avec au moins 15 chiffres de précision.

Le **`long double`** type est distinct, mais a la même représentation que le type **`double`** dans le compilateur Microsoft C.
