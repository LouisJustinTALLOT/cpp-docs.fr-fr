# <a name="significant-characters-without-external-linkage"></a>Caractères significatifs sans liaison externe
**ANSI 3.1.2** Nombre de caractères significatifs sans liaison externe  
  
 Les identificateurs sont significatifs jusqu'à 247 caractères. Le compilateur ne restreint pas le nombre de caractères que vous pouvez utiliser dans un identificateur. Il ignore simplement les caractères au delà de la limite.
