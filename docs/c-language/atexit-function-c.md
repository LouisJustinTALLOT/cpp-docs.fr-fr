# atexit, fonction (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.10.4.3** L'état retourné par la fonction `atexit` si la valeur de l'argument est autre que zéro, EXIT\_SUCCESS, ou EXIT\_FAILURE  
  
 La fonction `atexit` retourne zéro en cas de réussite ou une valeur différente de zéro en cas d'échec.
