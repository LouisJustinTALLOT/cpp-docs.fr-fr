# <a name="range-of-char-values"></a>Plage de valeurs char

**3.2.1.1 ANSI** Si un « plain » **`char`** a la même plage de valeurs qu’un **`signed char`** ou un**`unsigned char`**

Toutes les valeurs de caractères signées sont comprises entre -128 et 127. Toutes les valeurs de caractères non signées sont comprises entre 0 et 255.

L' [`/J`](../build/reference/j-default-char-type-is-unsigned.md) option de compilateur modifie le type par défaut de **`char`** de **`signed char`** à **`unsigned char`** .
