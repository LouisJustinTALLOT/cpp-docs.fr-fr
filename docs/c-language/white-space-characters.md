# Espace blanc, caract&#232;res
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les caractères espace, de tabulation, de saut de ligne, de retour chariot, de saut de page et de tabulation verticale sont appelés « espaces blancs » car ils servent le même objectif que les espaces entre les mots et les lignes dans une page imprimée : ils facilitent la lecture.  Les jetons sont délimités \(limités\) par des espaces blancs et par d'autres jetons, tels que les opérateurs et la ponctuation.  Lors de l'analyse du code, le compilateur C ignore les espaces blancs à moins que vous les utilisiez comme séparateurs ou comme composants de constantes caractères ou de littéraux de chaîne.  Utilisez les espaces blancs pour rendre un programme plus lisible.  Notez que le compilateur traite également les commentaires comme des espaces blancs.
