# <a name="ansi-conformance"></a>Conformité ANSI

Microsoft C est conforme à la norme relative au langage C telle qu'elle est établie dans l'édition 9899:1990 de la norme C ANSI.

Les extensions Microsoft de la norme C ANSI sont notées dans le texte et la syntaxe de cette documentation ainsi que dans la documentation en ligne. Étant donné que les extensions ne font pas partie de la norme C ANSI, leur utilisation peut restreindre la portabilité des programmes entre les systèmes. Par défaut, les extensions Microsoft sont activées. Pour les désactiver, utilisez l'option du compilateur /Za. Avec /Za, tout le code non ANSI génère des erreurs ou des avertissements.
