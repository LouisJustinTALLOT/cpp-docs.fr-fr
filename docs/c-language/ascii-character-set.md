# <a name="ascii-character-set"></a>Jeu de caractères ASCII
**ANSI 2.2.1** Membres des jeux de caractères source et d'exécution  
  
 Le jeu de caractères source est le jeu de caractères reconnus qui peuvent figurer dans les fichiers sources. Pour Microsoft C, le jeu de caractères source est le jeu de caractères ASCII standard.  
  
> [!NOTE]
>  **Avertissement** Puisque les pilotes de clavier et de console peuvent remapper le jeu de caractères, les programmes prévus pour une distribution internationale doivent vérifier le code de pays/région.
