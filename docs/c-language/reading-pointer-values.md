# Valeurs des pointeurs de lecture
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.6.2** L'entrée pour la conversion **%p** dans la fonction `fscanf`  
  
 Lorsque le caractère de format **%p** est spécifié, la fonction `fscanf` convertit des pointeurs de valeurs ASCII hexadécimales vers l'adresse correcte.
