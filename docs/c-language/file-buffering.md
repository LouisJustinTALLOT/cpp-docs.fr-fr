# Mise en m&#233;moire tampon de fichiers
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** Caractéristiques du fichier tampon  
  
 Les fichiers disque accessibles via les fonctions E\/S standard sont entièrement mis en mémoire tampon.  Par défaut, la mémoire tampon contient 512 octets.
