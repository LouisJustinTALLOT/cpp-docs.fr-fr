# Caract&#232;res significatifs avec liaison externe
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.1.2** Nombre de caractères significatifs avec liaison externe  
  
 Les identificateurs déclarés `extern` dans les programmes compilés avec Microsoft C sont significatifs jusqu'à 247 caractères.  Vous pouvez modifier cette valeur par défaut à un nombre plus petit à l'aide de l'option \/H \(limiter la longueur des noms externes\).
