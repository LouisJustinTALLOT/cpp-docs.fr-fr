# <a name="bits-per-character"></a>Bits par caractère
**ANSI 2.2.4.2.1** Nombre de bits dans un caractère  
  
 Le nombre de bits dans un caractère est représenté par la constante de manifeste **CHAR_BIT**. Le fichier LIMITS.H définit **CHAR_BIT** avec la valeur 8.
