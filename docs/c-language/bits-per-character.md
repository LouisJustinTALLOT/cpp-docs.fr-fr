# Bits par caract&#232;re
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 2.2.4.2.1** Nombre de bits dans un caractère  
  
 Le nombre de bits dans un caractère est représenté par la constante de manifeste **CHAR\_BIT**.  Le fichier LIMITS.H définit **CHAR\_BIT** avec la valeur 8.
