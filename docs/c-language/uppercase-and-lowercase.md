# <a name="uppercase-and-lowercase"></a>Majuscules et minuscules

**ANSI 3.1.2** Si les distinctions de casse sont significatives

Microsoft C traite les identificateurs d'une unité de compilation comme respectant la casse.

L'éditeur de liens Microsoft respecte la casse. Vous devez spécifier tous les identificateurs de manière cohérente selon le cas.
