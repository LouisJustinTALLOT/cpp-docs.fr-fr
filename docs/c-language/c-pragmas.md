# <a name="c-pragmas"></a>Pragmas C

**Spécifique à Microsoft**

Un *pragma* indique au compilateur d’effectuer une action particulière au moment de la compilation. Les pragmas varient d'un compilateur à l'autre. Par exemple, vous pouvez utiliser le pragma **optimize** pour définir les optimisations à effectuer sur votre programme. Les pragmas Microsoft C sont les suivants :

|||||
|-|-|-|-|
|**alloc_text**|**data_seg**|**inline_recursion**|**setlocale**|
|**auto_inline**|**fonctionnalités**|**essence**|**tres**|
|**check_stack**|**hdrstop**|**message**||
|**code_seg**|**include_alias**|**requêtes**||
|**Commentaire**|**inline_depth**|**presse**||

Consultez [Directives pragma et mot clé _Pragma](../preprocessor/pragma-directives-and-the-pragma-keyword.md) pour obtenir une description des pragmas du compilateur Microsoft C.

**FIN spécifique à Microsoft**
