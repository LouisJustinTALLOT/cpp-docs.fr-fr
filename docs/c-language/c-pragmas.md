# Pragmas C
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Section spécifique à Microsoft**  
  
 Un pragma indique au compilateur d'effectuer une action particulière au moment de la compilation.  Les pragmas varient d'un compilateur à l'autre.  Par exemple, vous pouvez utiliser le pragma **optimize** pour définir les optimisations à effectuer sur votre programme.  Les pragmas Microsoft C sont les suivants :  
  
|||||  
|-|-|-|-|  
|**alloc\_text**|**data\_seg**|**inline\_recursion**|**setlocale**|  
|**auto\_inline**|**function**|**intrinsic**|**warning**|  
|**check\_stack**|**hdrstop**|**message**||  
|**code\_seg**|**include\_alias**|**optimize**||  
|**comment**|**inline\_depth**|**pack**||  
  
 Consultez [Directives pragma et mot clé \_Pragma](../preprocessor/pragma-directives-and-the-pragma-keyword.md) dans la *Référence du préprocesseur* pour obtenir une description des pragmas du compilateur Microsoft C.  
  
 **FIN de la section spécifique à Microsoft**
