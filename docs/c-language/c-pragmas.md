# <a name="c-pragmas"></a>Pragmas C

**Spécifique à Microsoft**

Un *pragma* indique au compilateur d’effectuer une action particulière au moment de la compilation. Les pragmas varient d'un compilateur à l'autre. Par exemple, vous pouvez utiliser le `optimize` pragma pour définir les optimisations à effectuer sur votre programme. Les pragmas Microsoft C sont les suivants :
