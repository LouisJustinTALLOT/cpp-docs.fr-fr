# <a name="type-char"></a>char, type

Le **`char`** type est utilisé pour stocker la valeur entière d’un membre du jeu de caractères représentable. Cette valeur est le code ASCII correspondant au caractère spécifié.

**Spécifique à Microsoft**

Les valeurs de caractère de type **`unsigned char`** ont une plage de 0 à 0xFF hexadécimal. Un **`signed char`** a une plage de 0x80 à 0x7F. En valeur décimale, ces plages s’étendent de 0 à 255 et de -128 à +127, respectivement. L’option du compilateur/J remplace la valeur par défaut par **`signed`** **`unsigned`** .

**FIN spécifique à Microsoft**
