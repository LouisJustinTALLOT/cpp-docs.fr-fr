# clock, fonction (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.12.2.1** L'ère pour la fonction `clock`  
  
 L'ère de la fonction `clock` commence \(avec une valeur de 0\) au démarrage du programme C à exécuter.  Elle retourne des heures exprimées dans 1**HORLOGES\_PAR\_SEC** \(qui équivaut à 1\/1000 pour Microsoft C\).
