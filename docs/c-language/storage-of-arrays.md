# <a name="storage-of-arrays"></a>Stockage de tableaux

Le stockage associé à un type de tableau est le stockage requis pour tous ses éléments. Les éléments d'un tableau sont stockés dans des emplacements de mémoire contigus et en augmentation, du premier élément au dernier.
