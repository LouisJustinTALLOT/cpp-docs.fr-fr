# <a name="constants-in-primary-expressions"></a>Constantes dans les expressions primaires

Un opérande constant a la valeur et le type de la valeur constante qu’il représente. Une constante caractère a le type `int`. Une constante entière a `int`un type, `unsigned int` **long**, `unsigned long` ou, selon la taille de l’entier et la manière dont la valeur est spécifiée. Reportez-vous à l'article [Constantes](../c-language/c-constants.md) pour plus d'informations.
