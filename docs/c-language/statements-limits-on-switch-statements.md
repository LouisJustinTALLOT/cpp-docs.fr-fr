# <a name="statements-limits-on-switch-statements"></a>Instructions : limites dans des instructions switch
**ANSI 3.6.4.2** Nombre maximal de valeurs **case** dans une instruction `switch`  
  
 Microsoft C ne limite pas le nombre de valeurs **case** dans une instruction **switch**. Le nombre est limité uniquement par la mémoire disponible.
