# <a name="statements-limits-on-switch-statements"></a>Instructions : limites dans des instructions switch

**3.6.4.2 ANSI** Nombre maximal de **`case`** valeurs dans une **`switch`** instruction

Microsoft C ne limite pas le nombre de **`case`** valeurs dans une **`switch`** instruction. Le nombre est limité uniquement par la mémoire disponible.
