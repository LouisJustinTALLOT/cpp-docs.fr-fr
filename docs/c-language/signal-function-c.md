# signal, fonction (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.7.1.1** Ensemble des signaux pour la fonction **signal**  
  
 Le premier argument passé à **signal** doit être l'une des constantes symboliques décrites dans la *Référence de la bibliothèque Runtime* pour la fonction **signal**.  Les informations contenues dans la *Référence de la bibliothèque Runtime* répertorient également la prise en charge du mode d'opération de chaque signal.  Les constantes sont également définies dans SIGNAL.H.
