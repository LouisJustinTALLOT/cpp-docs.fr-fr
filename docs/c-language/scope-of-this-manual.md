# <a name="scope-of-this-manual"></a>Portée de ce manuel

C est un langage flexible qui vous permet de choisir vous-mêmes de nombreuses décisions de programmation. Conformément à cette philosophie, C applique peu de restrictions dans des domaines tels que la conversion de type. Bien que cette fonctionnalité du langage puisse faciliter votre travail de programmation, vous devez bien connaître le langage pour comprendre comment les programmes se comporteront. Ce manuel fournit des informations sur les composants de langage C et les fonctionnalités de l’implémentation Microsoft. La syntaxe du langage C est tirée du langage ANSI X3.159-1989, *American National Standard for Information Systems – Langage de programmation - C* (ci-après appelée la norme C ANSI), bien qu’elle ne fasse pas partie de la norme C ANSI. Le [Résumé de syntaxe du langage C](../c-language/c-language-syntax-summary.md) fournit la syntaxe et une description sur la façon de lire et d’utiliser les définitions de syntaxe.

Ce manuel ne traite pas la programmation en C++. Pour plus d’informations sur le langage C++, consultez [Référence du langage C++](../cpp/cpp-language-reference.md).
