# <a name="remainders"></a>Restes

**ANSI 3.3.5** Signe du reste de la division d'entier

Le signe du reste est identique au signe du dividende. Par exemple,

```
50 / -6 == -8
50 % -6 ==  2
-50 /  6 == -8
-50 %  6 == -2
```
