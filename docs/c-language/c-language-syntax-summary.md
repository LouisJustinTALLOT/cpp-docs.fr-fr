# R&#233;sum&#233; de la syntaxe du langage C
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Cette section donne la description complète du langage C et des fonctionnalités de langage C spécifiques de Microsoft.  Vous pouvez utiliser la notation de syntaxe de cette section pour déterminer la syntaxe exacte pour tout composant de langage.  L'explication pour la syntaxe apparaît dans la section de ce manuel où un sujet est abordé.  
  
> [!NOTE]
>  Ce résumé de syntaxe ne fait pas partie de la norme C ANSI, mais est inclus à titre d'information uniquement.  La syntaxe spécifique de Microsoft est désignée dans les commentaires suivant la syntaxe.
