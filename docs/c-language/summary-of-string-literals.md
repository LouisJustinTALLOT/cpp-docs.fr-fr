# <a name="summary-of-string-literals"></a>Résumé des littéraux de chaîne
*string-literal*:  
 **'** *s-char-sequence* opt **'**  
  
 **L'** *s-char-sequence* opt **'**  
  
 *s-char-sequence*:  
 *s-char*  
  
 *s-char-sequence s-char*  
  
 *s-char* :  
 tout membre du jeu de caractères de code source sauf le guillemet double (''), la barre oblique inverse (\\) ou la séquence d'échappement (*escape-sequence*) de caractère de saut de ligne
