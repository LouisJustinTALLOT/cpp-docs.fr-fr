# Parcours de la structures de classes (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Vous pouvez parcourir la structure de classe d'un projet, notamment les fonctions, les héritages et les interfaces, à l'aide de l'[affichage de classes](http://msdn.microsoft.com/fr-fr/8d7430a9-3e33-454c-a9e1-a85e3d2db925) ou de l'[Explorateur d'objets](http://msdn.microsoft.com/fr-fr/f89acfc5-1152-413d-9f56-3dc16e3f0470).  
  
 Vous pouvez accéder à une déclaration de fonction en double\-cliquant sur le nom de la fonction dans l'affichage de classes.  Pour accéder à une définition de classe, double\-cliquez sur le nom de la classe dans l'affichage de classes.
