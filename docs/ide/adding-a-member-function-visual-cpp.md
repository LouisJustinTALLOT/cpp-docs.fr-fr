# <a name="adding-a-member-function-visual-c"></a>Ajout d’une fonction membre (Visual C++)
Dans **l’affichage de classes**, vous pouvez ajouter une fonction membre à n’importe quelle classe. Dans ce cas, une déclaration est ajoutée au fichier d’en-tête et un corps de fonction membre stub est ajouté au fichier d’implémentation de la classe, que vous pouvez ensuite modifier.  
  
### <a name="to-add-a-member-function-to-a-class"></a>Pour ajouter une fonction membre à une classe  
  
1.  Dans **l’affichage de classes**, développez le nœud du projet afin d’afficher ses classes. (Pour ouvrir **l’affichage de classes**, dans la barre de menus, choisissez **Afficher**, **Affichage de classes**.)  
  
2.  Ouvrez le menu contextuel de la classe à laquelle vous souhaitez ajouter une fonction membre, puis choisissez **Ajouter**, **Ajouter une fonction**.  
  
3.  Fournissez les détails appropriés sur la fonction membre. Pour plus d’informations, consultez [Assistant Ajout de fonction membre](../ide/add-member-function-wizard.md).  
  
4.  Cliquez sur le bouton **Terminer** pour générer le code de la fonction membre.
