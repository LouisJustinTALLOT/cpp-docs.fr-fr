# Assistant personnalis&#233;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

L'Assistant personnalisé Visual C\+\+ est un type d'Assistant particulier, conçu pour vous aider à créer vos propres Assistants.  Une fois configuré, votre Assistant génère les fichiers de départ dont vous avez besoin pour vos projets.  
  
## Vue d'ensemble  
 Cette page de l'Assistant décrit les paramètres d'application en cours du projet Assistant personnalisé que vous créez.  Par défaut, l'Assistant crée un projet comme suit :  
  
-   Le nom convivial de l'Assistant est défini sur le nom du projet que vous avez indiqué dans **Nouveau projet, boîte de dialogue**.  
  
-   L'Assistant a une interface utilisateur.  
  
-   L'interface utilisateur de l'Assistant possède une page.  
  
 Vous pouvez modifier ces paramètres par défaut dans la page [Paramètres de l'application](../ide/application-settings-custom-wizard.md).
