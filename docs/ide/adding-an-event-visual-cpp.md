# Ajout d&#39;un &#233;v&#233;nement (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Dans l'Affichage de classes, vous ne pouvez ajouter un événement à l'aide de l'[Assistant Ajout d'événement](../ide/add-event-wizard.md) qu'à la classe de contrôle de votre projet de [contrôle ActiveX MFC](../mfc/reference/creating-an-mfc-activex-control.md).  Si vous souhaitez ajouter un événement à un autre type de projet, utilisez le bouton **Événements** de la [fenêtre Propriétés](../Topic/Properties%20Window.md).  
  
### Pour ajouter un événement à votre projet de contrôle ActiveX MFC  
  
1.  Dans l'Affichage de classes, développez le nœud du projet afin d'afficher ses classes.  
  
2.  Cliquez avec le bouton droit sur la classe de contrôle du projet.  
  
3.  Dans le menu contextuel, cliquez sur **Ajouter** et sur **Ajouter un événement** afin d'afficher l'Assistant Ajout d'événement.  
  
4.  Indiquez les informations sur l'événement dans les zones appropriées de l'Assistant.  
  
5.  Cliquez sur **Terminer** pour ajouter l'événement au projet.
