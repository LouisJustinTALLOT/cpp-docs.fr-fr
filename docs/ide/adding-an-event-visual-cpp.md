# <a name="adding-an-event-visual-c"></a>Ajout d'un événement (Visual C++)
À partir de l’affichage de classes, vous pouvez ajouter un événement à l’aide de la [Assistant Ajout d’événement](../ide/add-event-wizard.md) uniquement à la classe de contrôle dans votre [contrôle ActiveX MFC](../mfc/reference/creating-an-mfc-activex-control.md) projet. Si vous souhaitez ajouter un événement à un autre type de projet, utilisez le **événements** situé dans le [fenêtre Propriétés](/visualstudio/ide/reference/properties-window).  
  
### <a name="to-add-an-event-to-your-mfc-activex-control-project"></a>Pour ajouter un événement à votre projet de contrôle ActiveX MFC  
  
1.  Dans Affichage de classes, développez le nœud du projet pour afficher les classes dans le projet.  
  
2.  Avec le bouton droit de la classe du projet contrôle.  
  
3.  Dans le menu contextuel, cliquez sur **ajouter**, puis cliquez sur **ajouter un événement** pour afficher l’Assistant Ajout d’événement.  
  
4.  Fournissez les informations d’événement dans les zones appropriées de l’Assistant.  
  
5.  Cliquez sur **Terminer** pour ajouter l’événement au projet.
