# <a name="application-settings-makefile-project-wizard"></a>Paramètres de l'application, Assistant Projet Makefile
Fournissez les informations pour le projet. Les options que vous spécifiez sont répercutées dans le fichier [page de propriétés](../ide/working-with-project-properties.md). Lorsque vous créez un projet, vous pouvez afficher et modifier chacune des options suivantes dans le **Nmake** page de page de propriétés du projet.  
  
 **Ligne de commande build**  
 Spécifie la ligne de commande à exécuter lorsque l’utilisateur sélectionne **générer** à partir de la **générer** menu. Affiché dans le **ligne de commande Build** champ sur la **Nmake** page de page de propriétés du projet.  
  
 **Sortie**  
 Spécifie le nom du fichier qui contiendra la sortie de la ligne de commande. Par défaut, cette option est basée sur le nom du projet. Affiché dans le **sortie** champ sur la **Nmake** page de page de propriétés du projet.  
  
 **Commandes de nettoyage**  
 Spécifie la ligne de commande à exécuter lorsque l’utilisateur sélectionne **Clean** à partir de la **générer** menu. Affiché dans le **ligne de commande Clean** champ sur la **Nmake** page de page de propriétés du projet.  
  
 **Ligne de commande Rebuild**  
 Spécifie la ligne de commande à exécuter lorsque l’utilisateur sélectionne **reconstruire** à partir de la **générer** menu. Affiché dans le **tous de ligne de commande Rebuild** champ sur la **Nmake** page de page de propriétés du projet.
