# <a name="hlsl-property-pages-advanced"></a>Pages de propriétés HLSL : Avancé

Pour configurer les propriétés suivantes du compilateur HLSL (fxc.exe), utilisez sa page de propriétés **Avancé**. Pour plus d’informations sur l’accès à la page de propriétés **Avancé** dans le dossier HLSL, consultez [Utilisation des propriétés de projet](../ide/working-with-project-properties.md).

## <a name="uielement-list"></a>Liste des éléments d’interface

- **Suppression de la bannière de démarrage**

   **Oui (/nologo)** pour supprimer l’affichage de la bannière de démarrage et du message d’information ; sinon, **Non**. Par défaut, la valeur est **Oui (/nologo)**.

- **Considérer les avertissements comme des erreurs**

   **Oui (/WX)** pour considérer tous les avertissements du compilateur comme des erreurs ; sinon, **Non**.
