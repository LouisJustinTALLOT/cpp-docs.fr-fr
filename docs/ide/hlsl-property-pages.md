# Pages de propri&#233;t&#233;s HLSL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Vous pouvez utiliser les pages de propriétés du compilateur de HLSL \(fxc.exe\) pour configurer la façon dont les fichiers shaders de HLSL sont générés.  Les propriétés sont divisées en trois catégories :  
  
-   [Pages de propriétés HLSL : Général](../ide/hlsl-property-pages-general.md)  
  
     Fournit des propriétés pour spécifier des options fréquemment utilisées pour le compilateur de HLSL.  
  
-   [Pages de propriétés HLSL : Avancé](../ide/hlsl-property-pages-advanced.md)  
  
     Fournit des propriétés pour spécifier des options supplémentaires pour le compilateur de HLSL.  
  
-   [Pages de propriétés HLSL : fichiers de sortie](../ide/hlsl-property-pages-output-files.md)  
  
     Fournit des propriétés pour spécifier la façon dont les fichiers sont sortis par le compilateur de HLSL.  
  
 Vous pouvez spécifier des arguments de ligne de commande au compilateur de HLSL à l'aide de la propriété **Options supplémentaires** de la page de propriétés **Ligne de commande** ; cela inclut les arguments qui ne peuvent pas être configurés à l'aide de autres propriétés des pages de propriétés de HLSL.  Pour plus d'informations sur le compilateur de HLSL, consultez [Outil de compilateur d'effets](http://go.microsoft.com/fwlink/p/?LinkID=258285&clcid=0x409)
