# <a name="managed-resources-property-page"></a>Ressources managées (page de propriétés)
Active les paramètres pour le compilateur de ressources.  
  
 Le **ressources gérées** page de propriétés contient les propriétés suivantes :  
  
 **Nom logique de la ressource**  
 Spécifie le *nom logique* du fichier .resources intermédiaire généré. Le nom logique est le nom utilisé pour charger la ressource. Si aucun nom logique n’est spécifié, le nom de fichier de ressources (.resx) est utilisé comme nom logique.  
  
 **Nom du fichier de sortie**  
 Spécifie le nom du fichier de sortie final que le fichier de ressources (.resx) contribue à.  
  
 **Ressources localisées par défaut**  
 Spécifie si le fichier .resx donné contribue aux ressources par défaut ou à une DLL satellite.  
  
 Pour plus d’informations sur l’accès à la **ressources gérées** page de propriétés, consultez [utilisation des propriétés de projet](../ide/working-with-project-properties.md).
