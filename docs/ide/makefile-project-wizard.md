# Projet Makefile (Assistant)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Utilisez cet Assistant pour créer une ligne de commande que le moteur de génération exécutera.  Une fois que vous avez créé le projet, vous pouvez afficher et modifier les propriétés du projet dans la [page de propriétés](../ide/working-with-project-properties.md) du projet.  
  
## Vue d'ensemble  
 Cette page de l'Assistant décrit les paramètres en cours pour le projet que vous créez.  Par défaut, ce projet est configuré pour générer les configurations Debug et Release de *NomProj*.exe.  
  
 Pour changer ces paramètres par défaut, cliquez sur l'onglet [Paramètres de l'application](../ide/application-settings-makefile-project-wizard.md) dans la colonne de gauche de l'Assistant et apportez les modifications voulues.
