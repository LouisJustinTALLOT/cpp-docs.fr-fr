# <a name="makefile-project-wizard"></a>Projet Makefile (Assistant)
Utilisez cet Assistant pour créer une ligne de commande pour le moteur de génération à exécuter. Lorsque vous créez un projet, vous pouvez afficher et modifier les propriétés du projet dans le fichier [page de propriétés](../ide/working-with-project-properties.md).  
  
## <a name="overview"></a>Vue d'ensemble  
 Cette page de l’Assistant décrit les paramètres du projet en cours pour le projet que vous créez. Par défaut, le projet est configuré pour générer les versions debug et release des configurations de *NomProj*.exe.  
  
 Pour modifier ces valeurs par défaut, cliquez sur le [paramètres de l’Application](../ide/application-settings-makefile-project-wizard.md) onglet dans la colonne de gauche de l’Assistant et apportez les modifications souhaitées.
