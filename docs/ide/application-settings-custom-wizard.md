# Param&#232;tres de l&#39;application, Assistant personnalis&#233;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Utilisez cette page de l'Assistant personnalisé pour créer un Assistant.  
  
 **Nom convivial de l'Assistant**  
 Définit le nom de l'Assistant que vous créez.  Ce nom est affiché dans le fichier ReadMe.txt et possède le titre du fichier HTML.  
  
 **Interface utilisateur**  
 Valeur par défaut.  Spécifie que l'Assistant personnalisé affiche une interface utilisateur.  Si vous souhaitez que l'utilisateur sélectionne des options spécifiant le code que votre Assistant va générer, l'Assistant doit avoir une interface utilisateur.  L'Assistant n'a pas besoin d'une interface utilisateur si son objectif est de générer des exemples de code sans option utilisateur.  
  
 **Nombre de pages**  
 Définit le nombre de pages dont votre Assistant a besoin.  Cette option s'applique uniquement à un Assistant doté d'une interface utilisateur.
