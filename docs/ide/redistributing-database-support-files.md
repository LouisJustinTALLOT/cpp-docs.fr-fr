# <a name="redistributing-database-support-files"></a>Redistribution de fichiers de prise en charge de base de données
Vous pouvez redistribuer les fichiers de prise en charge pour les objets DAO (Data Access) et pour les technologies de base de données dans le SDK Microsoft Data Access.  
  
## <a name="installing-dao-support-files"></a>Installation des fichiers de prise en charge DAO  
 Pour obtenir la version la plus récente de DAO, consultez [Article 239114 : comment obtenir le dernier service pack pour le moteur de base de données Microsoft Jet 4.0](http://go.microsoft.com/fwlink/p/?linkid=198014) sur le site Web de Support technique de Microsoft.  
  
## <a name="installing-microsoft-data-access-sdk-support-files"></a>L’installation des fichiers de support technique de Microsoft Data Access SDK  
 Utiliser Mdac_typ.exe pour installer la prise en charge d’ODBC, OLE DB, ActiveX Data Objects (ADO) et Services de données à distance (RDS). Mdac_typ.exe se trouve dans le... Dossier \WCU\MDAC28\ sur le support d’installation de Visual Studio. Vous pouvez également télécharger Mdac_typ.exe à partir de la [Microsoft Download Center](http://go.microsoft.com/fwlink/p/?linkid=198015) site Web.  
  
 Pour plus d’informations sur la [MSDN](http://go.microsoft.com/fwlink/p/?linkid=198016) site Web, recherchez « Redistribution MDAC 2.8 SP1 ». Si vous utilisez des projets d’installation de Visual Studio pour déployer votre application, consultez [déploiement et dépendances](http://msdn.microsoft.com/en-us/49e9b84d-bd6a-4388-b9ac-46ea79cf0733).
