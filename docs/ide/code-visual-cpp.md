# <a name="ltcodegt-visual-c"></a>&lt;code&gt; (Visual C++)
Le \<code > balise vous donne un moyen d’indiquer une ou plusieurs lignes en tant que code.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
<code>content</code>  
```  
  
#### <a name="parameters"></a>Paramètres  
 `content`  
 Texte à marquer comme étant du code.  
  
## <a name="remarks"></a>Notes  
 Utilisez [ \<c >](../ide/c-visual-cpp.md) pour indiquer une partie du texte doit être marquée en tant que code.  
  
 Compilez avec [/doc](../build/reference/doc-process-documentation-comments-c-cpp.md) pour placer les commentaires de documentation dans un fichier en vue de les traiter.
