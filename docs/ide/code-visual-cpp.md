# &lt;code&gt; (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La balise d' \<code\> vous fournit un moyen d'indiquer un ou plusieurs lignes en tant que code.  
  
## Syntaxe  
  
```  
<code>content</code>  
```  
  
#### Paramètres  
 `content`  
 Texte que vous souhaitez marquer comme étant du code.  
  
## Notes  
 L'utilisation [\<c\>](../ide/c-visual-cpp.md) d'indiquer une partie du texte doit être marquée comme code.  
  
 Compilez avec [\/doc](../build/reference/doc-process-documentation-comments-c-cpp.md) pour traiter les commentaires de documentation et les placer dans un fichier.
