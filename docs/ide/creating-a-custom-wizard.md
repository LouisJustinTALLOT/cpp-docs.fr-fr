# Cr&#233;ation d’un Assistant personnalis&#233;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

L'Assistant personnalisé Visual C\+\+ est l'outil à utiliser lorsque vous avez besoin de créer un Assistant personnalisé.  La solution la plus simple pour le créer consiste à utiliser l'[Assistant personnalisé](../ide/custom-wizard.md).  
  
> [!TIP]
>  Le modèle d'Assistant personnalisé est pas disponible dans les éditions Express.  
  
### Pour créer un Assistant à l'aide de l'Assistant personnalisé  
  
1.  Suivez les instructions de la rubrique d'aide [Création d'un projet à l'aide d'un Assistant Application Visual C\+\+](../ide/creating-desktop-projects-by-using-application-wizards.md).  
  
2.  Dans la boîte de dialogue **Nouveau projet**, sélectionnez l'icône de **Assistant personnalisé** dans le volet modèles **Visual C\+\+ &#124; Général** pour ouvrir l'Assistant personnalisé.  
  
3.  Définissez vos [paramètres d'application](../ide/application-settings-custom-wizard.md).  
  
4.  Cliquez sur **Terminer** pour fermer l'Assistant et ouvrir votre nouveau projet dans l'environnement de développement.  
  
 Une fois votre projet créé, tous les nouveaux fichiers sont affichés dans l'**Explorateur de solutions**.  Pour plus d'informations sur les fichiers créés pour l'Assistant personnalisé, consultez [Fichiers créés pour votre Assistant](../ide/files-created-for-your-wizard.md).
