# <a name="concurrency-runtime-best-practices"></a>Meilleures pratiques sur le runtime d'accès concurrentiel

Cette section décrit les pratiques recommandées qui peuvent vous aider à utiliser efficacement le runtime d'accès concurrentiel. Ces meilleures pratiques s’appliquent à la [bibliothèque de modèles parallèles](../../parallel/concrt/parallel-patterns-library-ppl.md) (PPL), à la [bibliothèque d’agents asynchrones](../../parallel/concrt/asynchronous-agents-library.md)et à la [Planificateur de tâches](../../parallel/concrt/task-scheduler-concurrency-runtime.md).
