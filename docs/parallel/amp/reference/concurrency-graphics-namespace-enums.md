##  <a name="a-nameaddressmodea--addressmode-enumeration"></a><a name="address_mode"></a>address_mode, énumération  
 Type d’énumération permet de désigner les modes d’adresse pris en charge pour l’échantillonnage de texture.  
  
```  
enum address_mode;  
```  
  
##  <a name="a-namefiltermodea--filtermode-enumeration"></a><a name="filter_mode"></a>filter_mode, énumération

```  
enum filter_mode;  
```
