# <a name="concurrencygraphicsdirect3d-namespace"></a>Concurrency::graphics::direct3d, espace de noms

Fournit les méthodes [get_texture](concurrency-graphics-direct3d-namespace-functions.md#get_texture) et [make_texture](concurrency-graphics-direct3d-namespace-functions.md#make_texture) .

## <a name="syntax"></a>Syntaxe

```cpp
namespace direct3d;
```

## <a name="members"></a>Membres

### <a name="functions"></a>Fonctions

|Nom<br /><br /> Description|
|--------------------------|
|[get_sampler](concurrency-graphics-direct3d-namespace-functions.md#get_sampler)<br /><br /> Obtient l’interface d’état de l’échantillonneur Direct3D sur la vue d’accélérateur spécifiée qui représente l’objet d’échantillonnage spécifié.|
|[get_texture](concurrency-graphics-direct3d-namespace-functions.md#get_texture)<br /><br /> Obtient l’interface de texture Direct3D sous-jacente à l’objet de [texture](texture-class.md) spécifié.|
|[make_sampler](concurrency-graphics-direct3d-namespace-functions.md#make_sampler)<br /><br /> Créez un échantillonneur à partir d’un pointeur d’interface d’état d’échantillonnage Direct3D.|
|[make_texture](concurrency-graphics-direct3d-namespace-functions.md#make_texture)<br /><br /> Crée un objet de [texture](texture-class.md) à l’aide des paramètres spécifiés.|
|[msad4](concurrency-graphics-direct3d-namespace-functions.md#msad4)<br /><br /> Compare une valeur de référence de 4 octets et une valeur source de 8 octets et accumule un vecteur de 4 sommes.|

**Espace de noms :** Concurrency :: Graphics
