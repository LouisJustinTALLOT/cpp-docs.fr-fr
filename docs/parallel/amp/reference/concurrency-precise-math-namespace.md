# <a name="concurrencyprecisemath-namespace"></a>Concurrency::precise_math, espace de noms

Fonctions dans le `precise_math` espace de noms sont conformes C99. À la fois simple précision et double précision les versions de chaque fonction sont incluses. Par exemple, `acos` est la version double précision et `acosf` est la version simple précision. Ces fonctions, y compris les fonctions simple précision, requièrent une prise en charge double précision étendue sur l’accélérateur. Vous pouvez utiliser la [accelerator::supports_double_precision](accelerator-class.md#supports_double_precision) pour déterminer si vous pouvez exécuter ces fonctions sur un accélérateur spécifique.

## <a name="syntax"></a>Syntaxe

```cpp
namespace precise_math;
```
