# C.1 Notation
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Les règles de syntaxe incluent le nom d'un non terminal, suivie d'un signe deux\-points, suivi des alternatives de remplacement sur des lignes distinctes.  
  
 Le terme syntaxe d'expressionopt indique que le terme est facultatif dans un remplacement.  
  
 *Le terme*syntaxe d'expressionoptseq équivaut *à terme\-seqopt*avec les règles supplémentaires suivantes :  
  
 *terme\-seq* :  
  
 *terme*  
  
 *terme terme\-seq*  
  
 *terme\-seq* , *terme*
