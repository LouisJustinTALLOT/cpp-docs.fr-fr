# <a name="ompgetthreadnum"></a>omp_get_thread_num
Retourne le nombre de threads d’exécution du thread au sein de son équipe de thread.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
int omp_get_thread_num( );  
```
