-   le **omp_get_max_threads** (fonction).  
  
-   le **omp_get_thread_num** (fonction).  
  
-   le **omp_get_num_procs** (fonction).  
  
-   le **omp_in_parallel** (fonction).  
  
-   le **omp_set_dynamic** (fonction).  
  
-   le **omp_get_dynamic** (fonction).
