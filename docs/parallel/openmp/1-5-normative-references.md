# 1.5 Normative References
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

-   ISO\/CEI 9899:1999, *technologie de l'information \- langages de programmation \- C.* Cette spécification des API d'OpenMP fait référence au ISO\/CEI 9899:1999 comme C99.  
  
-   ISO\/CEI 9899:1990, *technologie de l'information \- langages de programmation \- C.* Cette spécification des API d'OpenMP fait référence au ISO\/CEI 9899:1990 comme C90.  
  
-   ISO\/CEI 14882:1998, *technologie de l'information \- langages de programmation \- C\+\+*.  Cette spécification des API d'OpenMP fait référence au ISO\/CEI 14882:1998 en C\+\+.  
  
 Lorsque cette spécification des API d'OpenMP fait référence à C, la référence est réalisée à la langue de base prise en charge par l'implémentation.
