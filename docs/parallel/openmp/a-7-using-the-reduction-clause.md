# A.7   Using the reduction Clause
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

L'exemple suivant illustre la clause reduction \([section 2.7.2.6](../../parallel/openmp/2-7-2-6-reduction.md) Ã  la page 28\) :  
  
```  
#pragma omp parallel for private(i) shared(x, y, n) \  
                         reduction(+: a, b)  
    for (i=0; i<n; i++) {  
        a = a + x[i];  
        b = b + y[i];  
    }  
```
