# <a name="openmp-in-visual-c"></a>OpenMP dans Visual C++

L’interface du programme d’application C et C++ OpenMP vous permet d’écrire des applications qui utilisent efficacement plusieurs processeurs. Visual C++ prend en charge la norme OpenMP 2,0.

## <a name="in-this-section"></a>Dans cette section

Fournit des liens vers des constructions utilisées dans l’API OpenMP.

[API C et C++](../../parallel/openmp/openmp-c-and-cpp-application-program-interface.md)<br/>
Présente l’API C et C++ OpenMP, comme indiqué dans la spécification de la version 2,0 du tableau de révision de l’architecture OpenMP.
