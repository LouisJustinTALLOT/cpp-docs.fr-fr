# Composant de registre ATL (Inscription)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ATL Registrar fournit l'accès optimisé à la base de registres via une interface personnalisée.  Le registre est libre de threads et permet la liaison statique du code pour les clients C\+\+.  
  
> [!NOTE]
>  Code source pour ATL Registrar se trouve dans atlmfc\\include\\atliface.h.
