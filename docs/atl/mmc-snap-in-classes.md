# <a name="mmc-snap-in-classes"></a>Classes de Snap-In MMC

Les classes suivantes prennent en charge le développement de composants logiciels enfichables Microsoft Management Console (MMC) :

- [CSnapInItemImpl](../atl/reference/csnapinitemimpl-class.md) Implémente un objet de nœud de composant logiciel enfichable, tel que l’ajout d’éléments de menu et de barres d’outils, et le transfert de commandes pour le nœud de composant logiciel enfichable vers la fonction de gestionnaire appropriée.

- [CSnapInPropertyPageImpl](../atl/reference/csnapinpropertypageimpl-class.md) Implémente un objet de page de propriétés de composant logiciel enfichable.
