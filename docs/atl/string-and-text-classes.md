# String and Text Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Ces classes fournissent la prise en charge des chaînes et des conversions de chaîne de texte.  
  
-   [CA2AEX](../atl/reference/ca2aex-class.md) cette classe est utilisé par les macros de conversion de chaînes `CA2TEX` et `CT2AEX`, et le typedef **CA2A**.  
  
-   [CA2CAEX](../atl/reference/ca2caex-class.md) cette classe est utilisé par les macros de conversion de chaînes `CA2CTEX` et `CT2CAEX`, et le typedef **CA2CA**.  
  
-   [CA2WEX](../atl/reference/ca2wex-class.md) cette classe est utilisé par les macros de conversion de chaînes `CA2TEX`, `CA2CTEX`, `CT2WEX`, et `CT2CWEX`, et le typedef **CA2W**.  
  
-   [CW2AEX](../atl/reference/cw2aex-class.md) cette classe est utilisé par les macros de conversion de chaînes `CT2AEX`, `CW2TEX`, `CW2CTEX`, et `CT2CAEX`, et le typedef **CW2A**.  
  
-   [CW2CWEX](../atl/reference/cw2cwex-class.md) cette classe est utilisé par les macros de conversion de chaînes `CW2CTEX` et `CT2CWEX`, et le typedef **CW2CW**.  
  
-   [CW2WEX](../atl/reference/cw2wex-class.md) cette classe est utilisé par les macros de conversion de chaînes `CW2TEX` et `CT2WEX`, et le typedef `CW2W`.  
  
-   [CComBSTR](../atl/reference/ccombstr-class.md) cette classe est un wrapper pour `BSTR`S.  
  
-   [\_U\_STRINGorID](../atl/reference/u-stringorid-class.md) Cette classe d'adaptateur d'argument permet des noms de ressources \(`LPCTSTR`s\) ou des identificateurs de ressource \(**uint**s\) à passer à une fonction sans que l'appelant de convertir l'ID d'une chaîne à l'aide de la macro de **MAKEINTRESOURCE** .
