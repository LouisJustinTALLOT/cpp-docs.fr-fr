- Quelles sont les [tables des messages](../atl/message-maps-atl.md) et comment les utiliser.

- Syntaxe pour les [fonctions de gestionnaire de messages](../atl/message-handler-functions.md).

- Comment [implémenter une fenêtre avec CWindowImpl](../atl/implementing-a-window-with-cwindowimpl.md).
