# Inserting a Composite Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La boîte de dialogue d' **Add Class** vous permet d'insérer un objet ATL à un projet.  Accéder à cette boîte de dialogue en cliquant avec le bouton droit sur le nom du projet dans l'explorateur de solutions, en pointant sur **Ajouter**, puis en cliquant sur **Add Class**.  
  
 Dans la boîte de dialogue d' **Add Class** , choisissez **Contrôle ATL**.  Cela commence [L'Assistant Contrôle ATL](../atl/reference/atl-control-wizard.md).  Pour créer un contrôle composite, sélectionnez l'onglet **Options** , puis cliquez sur la case à cocher de **Composite control** .  
  
 Une page HTML par défaut est créée pour visualiser le contrôle.
