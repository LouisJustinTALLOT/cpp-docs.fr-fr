# <a name="inserting-a-composite-control"></a>Insertion d’un contrôle composite

La boîte de dialogue **Ajouter une classe** vous permet d’insérer un objet ATL dans un projet. Pour accéder à cette boîte de dialogue, cliquez avec le bouton droit sur le nom du projet dans Explorateur de solutions, pointez sur **Ajouter**, puis cliquez sur **Ajouter une classe**.

Dans la boîte de dialogue **Ajouter une classe** , choisissez **contrôle ATL**. Cette opération démarre l' [Assistant contrôle ATL](../atl/reference/atl-control-wizard.md). Pour créer un contrôle composite, sélectionnez l’onglet **options** , puis activez la case à cocher **contrôle composite** .

Une page HTML par défaut est créée pour l’affichage du contrôle.
