# <a name="inserting-a-composite-control"></a>Insertion d’un contrôle Composite

Le **ajouter une classe** boîte de dialogue vous permet d’insérer un objet ATL dans un projet. Accéder à cette boîte de dialogue en cliquant sur le nom du projet dans l’Explorateur de solutions, en pointant sur **ajouter**, puis en cliquant sur **ajouter une classe**.

Dans le **ajouter une classe** boîte de dialogue, sélectionnez **contrôle ATL**. Ceci démarrera le [Assistant contrôle ATL](../atl/reference/atl-control-wizard.md). Pour créer un contrôle composite, sélectionnez le **Options** onglet, puis cliquez sur le **contrôle Composite** case à cocher.

Une page de code HTML par défaut est créée pour le contrôle d’affichage.
