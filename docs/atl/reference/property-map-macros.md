# Property Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ces macros définissent des mappages de propriétés et des entrées.  
  
|||  
|-|-|  
|[BEGIN\_PROP\_MAP](../Topic/BEGIN_PROP_MAP.md)|Marque le début du mappage de propriétés ATL.|  
|[PROP\_DATA\_ENTRY](../Topic/PROP_DATA_ENTRY.md)|Indique l'étendue, ou des dimensions, d'un contrôle ActiveX.|  
|[PROP\_ENTRY\_TYPE](../Topic/PROP_ENTRY_TYPE.md)|Écrit une description de la propriété, une propriété DISPID, et une page de propriétés CLSID dans le mappage de propriétés.|  
|[PROP\_ENTRY\_TYPE\_EX](../Topic/PROP_ENTRY_TYPE_EX.md)|Entre dans une description de la propriété, une propriété DISPID, une page de propriétés CLSID, et un `IDispatch` IID dans le mappage de propriétés.|  
|[PROP\_PAGE](../Topic/PROP_PAGE.md)|Écrit une page de propriétés CLSID dans le mappage de propriétés.|  
|[END\_PROP\_MAP](../Topic/END_PROP_MAP.md)|Marque la fin du mappage de propriétés ATL.|
