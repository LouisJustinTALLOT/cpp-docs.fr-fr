# Snap-In Object Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ces macros fournissent la prise en charge des extensions de composant logiciel enfichable.  
  
|||  
|-|-|  
|[BEGIN\_EXTENSION\_SNAPIN\_NODEINFO\_MAP](../Topic/BEGIN_EXTENSION_SNAPIN_NODEINFO_MAP.md)|Marque le début du mappage de classes de données d'extensions de composants logiciels enfichables pour un objet enfichable.|  
|[BEGIN\_SNAPINTOOLBARID\_MAP](../Topic/BEGIN_SNAPINTOOLBARID_MAP.md)|Marque le début du mappage de barre d'outils pour un objet enfichable.|  
|[END\_EXTENSION\_SNAPIN\_NODEINFO\_MAP](../Topic/END_EXTENSION_SNAPIN_NODEINFO_MAP.md)|Marque la fin du mappage de classes de données d'extensions de composants logiciels enfichables pour un objet enfichable.|  
|[END\_SNAPINTOOLBARID\_MAP](../Topic/END_SNAPINTOOLBARID_MAP.md)|Marque la fin du mappage de barre d'outils pour un objet enfichable.|  
|[EXTENSION\_SNAPIN\_DATACLASS](../Topic/EXTENSION_SNAPIN_DATACLASS.md)|Crée une donnée membre de la classe de données de l'extension de composant logiciel enfichable.|  
|[EXTENSION\_SNAPIN\_NODEINFO\_ENTRY](../Topic/EXTENSION_SNAPIN_NODEINFO_ENTRY.md)|Écrit une classe de données d'extension de composant logiciel enfichable dans le mappage de classes de données d'extensions de composants logiciels enfichables de l'objet enfichable.|  
|[SNAPINMENUID](../Topic/SNAPINMENUID.md)|Déclare l'ID du menu contextuel utilisé par l'objet enfichable.|  
|[SNAPINTOOLBARID\_ENTRY](../Topic/SNAPINTOOLBARID_ENTRY.md)|Entre dans une barre d'outils dans le mappage de barre d'outils de l'objet enfichable.|
