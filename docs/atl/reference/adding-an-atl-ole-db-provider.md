# <a name="adding-an-atl-ole-db-provider"></a>Ajout d’un fournisseur ATL OLE DB
Utilisez cet Assistant pour ajouter un fournisseur OLE DB ATL à un projet. Un fournisseur OLE DB ATL se compose d’une source de données, de session, de commande et de classes de l’ensemble de lignes. Le projet doit avoir été créé en tant qu’une application COM ATL.  
  
### <a name="to-add-an-atl-ole-db-provider-to-your-project"></a>Pour ajouter un fournisseur OLE DB ATL à votre projet  
  
1.  Dans Affichage de classes, cliquez sur le projet. Dans le menu contextuel, cliquez sur **ajouter** puis cliquez sur **ajouter une classe**.  
  
2.  Dans le dossier Visual C++, double-cliquez sur le **fournisseur OLE DB ATL** icône ou sélectionnez-la et cliquez sur **ouvrir**.  
  
     L’Assistant fournisseur OLE DB ATL s’ouvre.  
  
3.  Définissez les paramètres comme décrit dans [Assistant fournisseur OLE DB ATL](../../atl/reference/atl-ole-db-provider-wizard.md).  
  
4.  Cliquez sur **Terminer** pour fermer l’Assistant, qui insère le code du fournisseur OLE DB nouvellement créé dans votre projet.
