# Marshaling Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ces fonctions fournissent la prise en charge de marshaling et de convertir des informations de marshaling dans des pointeurs d'interface.  
  
> [!IMPORTANT]
>  Les fonctions répertoriées dans le tableau suivant ne peuvent pas être utilisées dans les applications qui s'exécutent dans [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
|||  
|-|-|  
|[AtlFreeMarshalStream](../Topic/AtlFreeMarshalStream.md)|Libère les marshaler de données et le pointeur d' `IStream` .|  
|[AtlMarshalPtrInProc](../Topic/AtlMarshalPtrInProc.md)|Crée un objet de flux et marshale le pointeur d'interface spécifié.|  
|[AtlUnmarshalPtr](../Topic/AtlUnmarshalPtr.md)|Convertit les données du marshaling d'un flux vers un pointeur d'interface.|
