# Window Class Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ces macros définissent les utilitaires de classe de fenêtre.  
  
|||  
|-|-|  
|[DECLARE\_WND\_CLASS](../Topic/DECLARE_WND_CLASS.md)|Vous permet de spécifier le nom d'une nouvelle classe de fenêtre.|  
|[DECLARE\_WND\_SUPERCLASS](../Topic/DECLARE_WND_SUPERCLASS.md)|Vous permet de spécifier le nom d'une classe de fenêtre existante sur laquelle une nouvelle classe de fenêtre est basée.|  
|[DECLARE\_WND\_CLASS\_EX](../Topic/DECLARE_WND_CLASS_EX.md)|Permet de spécifier des paramètres d'une classe.|
