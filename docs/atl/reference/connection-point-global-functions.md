# Connection Point Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ces fonctions fournissent la prise en charge des points de connexion et des cartes de récepteur.  
  
> [!IMPORTANT]
>  Les fonctions répertoriées dans le tableau suivant ne peuvent pas être utilisées dans les applications qui s'exécutent dans [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
|||  
|-|-|  
|[AtlAdvise](../Topic/AtlAdvise.md)|Crée une connexion le récepteur entre le point de connexion d'un objet et d'un client.|  
|[AtlUnadvise](../Topic/AtlUnadvise.md)|Termine la connexion établie par `AtlAdvise`.|  
|[AtlAdviseSinkMap](../Topic/AtlAdviseSinkMap.md)|Informe ou des entrées d'unadvises dans une table de récepteurs d'événements.|
