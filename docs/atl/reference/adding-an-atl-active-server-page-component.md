# Ajout d&#39;un composant Active Server Page ATL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Pour ajouter un objet ATL \(Active Template Library\) à votre projet, ce dernier doit avoir été créé en tant qu'application ATL COM ou MFC intégrant la prise en charge ATL.  Vous pouvez utiliser l'[Assistant Projet ATL](../../atl/reference/atl-project-wizard.md) pour créer une application ATL, sélectionner **Ajouter la prise en charge ATL aux MFC** dans la [boîte de dialogue Ajouter une classe](../../ide/add-class-dialog-box.md) ou [ajouter un objet ATL à votre application MFC](../../mfc/reference/adding-atl-support-to-your-mfc-project.md) afin d'implémenter la prise en charge ATL pour une application MFC.  
  
 Les composants Active Server Pages font partie de l'architecture des services IIS \(Internet Information Server\), fournissant les fonctionnalités de développement Web avancées suivantes :  
  
-   Vous pouvez incorporer des composants ASP dans vos pages HTML pour créer du contenu dynamique, indépendant du navigateur.  
  
-   Vous pouvez utiliser des pages ASP pour fournir une connectivité de base de données basée sur les normes.  
  
-   Vous pouvez utiliser les fonctions de gestion des erreurs ASP pour vos applications Web.  
  
### Pour ajouter un composant Active Server Page ATL à votre projet  
  
1.  Dans l'**Explorateur de solutions** ou dans l'[Affichage de classes](http://msdn.microsoft.com/fr-fr/8d7430a9-3e33-454c-a9e1-a85e3d2db925), cliquez avec le bouton droit sur le nom du projet auquel vous souhaitez ajouter le composant Active Server Page ATL.  
  
2.  Dans le menu contextuel, cliquez sur **Ajouter**, puis sur **Ajouter une classe**.  
  
3.  Dans le volet Modèles de la boîte de dialogue [Ajouter une classe](../../ide/add-class-dialog-box.md), cliquez sur **Composant ASP ATL**, puis cliquez sur **Ouvrir** pour afficher l'[Assistant Composant Active Server Page ATL](../../atl/reference/atl-active-server-page-component-wizard.md).
