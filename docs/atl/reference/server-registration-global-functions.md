# Server Registration Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ces fonctions fournissent la prise en charge d'enregistrer et d'annuler l'enregistrement des objets serveur dans la table d'objets.  
  
> [!IMPORTANT]
>  Les fonctions répertoriées dans le tableau suivant ne peuvent pas être utilisées dans les applications qui s'exécutent dans [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
|||  
|-|-|  
|[AtlComModuleRegisterServer](../Topic/AtlComModuleRegisterServer.md)|Cette fonction est appelée pour enregistrer chaque objet dans la table d'objets.|  
|[AtlComModuleUnregisterServer](../Topic/AtlComModuleUnregisterServer.md)|Cette fonction est appelée pour annuler l'inscription de chaque objet dans la table d'objets.|  
|[AtlComModuleRegisterClassObjects](../Topic/AtlComModuleRegisterClassObjects.md)|Cette fonction est appelée pour stocker des objets de classe.|  
|[AtlComModuleRevokeClassObjects](../Topic/AtlComModuleRevokeClassObjects.md)|Cette fonction est appelée pour révoquer des objets de classe d'un module COM.|  
|[AtlComModuleGetClassObject](../Topic/AtlComModuleGetClassObject.md)|Cette fonction est appelée pour obtenir l'objet de classe.|
