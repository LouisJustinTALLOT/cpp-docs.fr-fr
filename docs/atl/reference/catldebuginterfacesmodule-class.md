# <a name="catldebuginterfacesmodule-class"></a>CAtlDebugInterfacesModule (classe)
Cette classe prend en charge pour les interfaces de débogage.  
  
## <a name="syntax"></a>Syntaxe  
  
```
class CAtlDebugInterfacesModule
```  
  
## <a name="remarks"></a>Remarques  
 `CAtlDebugInterfacesModule`Fournit la prise en charge requis pour les interfaces de débogage. Il est inclus dans tout projet qui définit le symbole [_ATL_DEBUG_QI](http://msdn.microsoft.com/library/4f1feddf-1469-42ad-a2f7-5a816d1c0bc5).
