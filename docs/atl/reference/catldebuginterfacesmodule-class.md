# CAtlDebugInterfacesModule Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Cette classe fournit la prise en charge des interfaces de débogage.  
  
## Syntaxe  
  
```  
  
class CAtlDebugInterfacesModule  
  
```  
  
## Notes  
 `CAtlDebugInterfacesModule` fournit la prise en charge nécessaire pour les interfaces de débogage.  Il est inclus dans un projet qui définit le symbole [\_ATL\_DEBUG\_QI](../Topic/_ATL_DEBUG_QI.md).  
  
## Configuration requise  
 **Header:** atlbase.h
