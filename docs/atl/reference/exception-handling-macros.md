# Exception Handling Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ces macros fournissent la prise en charge de la gestion des exceptions.  
  
|||  
|-|-|  
|[\_ATLCATCH](../Topic/_ATLCATCH.md)|Instructions pour gérer les erreurs qui se produisent dans `_ATLTRY`associé.|  
|[\_ATLCATCHALL](../Topic/_ATLCATCHALL.md)|Instructions pour gérer les erreurs qui se produisent dans `_ATLTRY`associé.|  
|[\_ATLTRY](../Topic/_ATLTRY.md)|Marque une section protégée de code où une erreur peut éventuellement se produire.|
