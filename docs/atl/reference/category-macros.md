# Category Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ces macros définissent des cartes de catégorie.  
  
|||  
|-|-|  
|[BEGIN\_CATEGORY\_MAP](../Topic/BEGIN_CATEGORY_MAP.md)|Marque le début du mappage de catégorie.|  
|[END\_CATEGORY\_MAP](../Topic/END_CATEGORY_MAP.md)|Marque la fin du mappage de catégorie.|  
|[IMPLEMENTED\_CATEGORY](../Topic/IMPLEMENTED_CATEGORY.md)|Indique les catégories qui sont implémentées par l'objet COM.|  
|[REQUIRED\_CATEGORY](../Topic/REQUIRED_CATEGORY.md)|Indique les catégories nécessaires du conteneur par l'objet COM.|
