# <a name="creating-an-atl-project"></a>Création d’un projet ATL

Le moyen le plus simple de créer un projet ATL est d’utiliser l’Assistant Projet ATL, situé dans le dossier **Projets Win32** de la boîte de dialogue **Nouveau projet**.

## <a name="to-create-an-atl-project-using-the-atl-project-wizard"></a>Pour créer un projet ATL à l’aide de l’Assistant Projet ATL

1. Dans Visual Studio, sélectionnez **Fichier > Nouveau > Projet** dans le menu principal.

1. Sélectionnez l’icône **Projet ATL** dans le volet **Modèles** pour ouvrir l’**Assistant Projet ATL**.

1. Définissez les paramètres de votre application dans la page [Paramètres d’application](../../atl/reference/application-settings-atl-project-wizard.md) de l’**Assistant Projet ATL**.

   > [!NOTE]
   > Ignorez cette étape afin que l'Assistant conserve les paramètres par défaut.

1. Cliquez sur **Terminer** pour fermer l’Assistant et ouvrir votre nouveau projet dans l’environnement de développement.

Une fois votre projet créé, vous pouvez visualiser les fichiers créés dans l’**Explorateur de solutions**. Pour plus d'informations sur les fichiers créés par l'Assistant pour votre projet, consultez le fichier Readme.txt généré pour le projet. Pour plus d’informations sur les types de fichiers, consultez [Types de fichiers créés pour les projets Visual Studio C++](../../build/reference/file-types-created-for-visual-cpp-projects.md). Pour plus d’informations sur les configurations du nouveau projet ATL et savoir comment les modifier, consultez [Configurations de projet ATL par défaut](../../atl/reference/default-atl-project-configurations.md).
