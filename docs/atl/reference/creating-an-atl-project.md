# Cr&#233;ation d&#39;un projet ATL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

L'utilisation de l'Assistant Projet ATL, situé dans le dossier Projets Win32 de la boîte de dialogue **Nouveau projet**, constitue le moyen le plus rapide de créer un projet ATL.  
  
### Pour créer un projet ATL à l'aide de l'Assistant Projet ATL  
  
1.  Suivez les instructions fournies dans la rubrique d'aide [Création d'un projet à l'aide d'un Assistant Application Visual C\+\+](../../ide/creating-desktop-projects-by-using-application-wizards.md).  
  
2.  Dans le volet Modèles, sélectionnez l'icône **Projet ATL** pour démarrer l'Assistant Projet ATL.  
  
3.  Définissez les paramètres de votre application en utilisant la page [Paramètres de l'application](../../atl/reference/application-settings-atl-project-wizard.md) de l'`ATL Project Wizard`.  
  
    > [!NOTE]
    >  Ignorez cette étape afin que l'Assistant conserve les paramètres par défaut.  
  
4.  Cliquez sur **Terminer** pour fermer l'Assistant et ouvrir votre nouveau projet dans l'environnement de développement.  
  
 Une fois votre projet créé, vous pouvez visualiser les fichiers créés dans l'**Explorateur de solutions**.  Pour plus d'informations sur les fichiers créés par l'Assistant pour votre projet, consultez le fichier Readme.txt généré pour le projet.  Pour plus d'informations sur les types de fichiers, consultez [Types de fichiers créés pour les projets Visual C\+\+](../../ide/file-types-created-for-visual-cpp-projects.md).  Pour plus d'informations sur les configurations du nouveau projet ATL et sur leur modification, consultez [Configurations des projets ATL par défaut](../../atl/reference/default-atl-project-configurations.md).
