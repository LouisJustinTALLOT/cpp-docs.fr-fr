# <a name="adding-an-atl-com-10-component"></a>Ajout d’un composant ATL COM + 1.0
Cet Assistant ajoute un objet à votre projet qui prend en charge les services COM + 1.0, notamment les transactions.  
  
### <a name="to-add-an-atl-com-10-component-to-your-project"></a>Pour ajouter un composant ATL COM + 1.0 à votre projet  
  
1.  Dans le **l’Explorateur de solutions** ou [affichage de classes](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), cliquez sur le nom du projet auquel vous souhaitez ajouter le composant ATL COM + 1.0.  
  
2.  Dans le menu contextuel, cliquez sur **ajouter**, puis cliquez sur **ajouter une classe**.  
  
3.  Dans le [ajouter une classe](../../ide/add-class-dialog-box.md) boîte de dialogue, dans le volet Modèles, cliquez sur **composant ATL COM + 1.0**, puis cliquez sur **ouvrir** pour afficher le [Assistant composant ATL COM + 1.0 ](../../atl/reference/atl-com-plus-1-0-component-wizard.md).
