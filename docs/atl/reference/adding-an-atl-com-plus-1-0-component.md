# Ajout d&#39;un composant ATL COM+&#160;1.0
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Cet Assistant ajoute un objet à votre projet prenant en charge les services COM\+ 1.0, notamment les transactions.  
  
### Pour ajouter un composant ATL COM\+ 1.0 à votre projet  
  
1.  Dans l'**Explorateur de solutions** ou dans l'[affichage de classes](http://msdn.microsoft.com/fr-fr/8d7430a9-3e33-454c-a9e1-a85e3d2db925), cliquez avec le bouton droit sur le nom du projet auquel vous souhaitez ajouter le composant ATL COM\+ 1.0.  
  
2.  Dans le menu contextuel, cliquez sur **Ajouter**, puis sur **Ajouter une classe**.  
  
3.  Dans le volet Modèles de la boîte de dialogue [Ajouter une classe](../../ide/add-class-dialog-box.md), cliquez sur **Composant ATL COM\+ 1.0**, puis sur **Ouvrir** pour afficher l'[Assistant Composant ATL COM\+ 1.0](../../atl/reference/atl-com-plus-1-0-component-wizard.md).
