# Object Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ces macros définissent des tables d'objets et des entrées.  
  
|||  
|-|-|  
|[DECLARE\_OBJECT\_DESCRIPTION](../Topic/DECLARE_OBJECT_DESCRIPTION.md)|Vous permet de spécifier la description textuelle d'un objet de classe, qui est entrée dans la table d'objets.|  
|[OBJECT\_ENTRY\_AUTO](../Topic/OBJECT_ENTRY_AUTO.md)|Écrit un objet ATL dans la table d'objets, met à jour le Registre, et crée une instance de l'objet.|  
|[OBJECT\_ENTRY\_NON\_CREATEABLE\_EX\_AUTO](../Topic/OBJECT_ENTRY_NON_CREATEABLE_EX_AUTO.md)|Vous permet de spécifier que l'objet doit être stocké et initialisé, mais il ne doit pas être être créés de l'extérieur via `CoCreateInstance`.|
