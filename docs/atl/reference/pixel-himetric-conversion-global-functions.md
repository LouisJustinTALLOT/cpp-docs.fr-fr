# Pixel/HIMETRIC Conversion Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ces fonctions fournissent la prise en charge de convertir vers et à partir de le pixel et des unités HIMETRIC.  
  
> [!IMPORTANT]
>  Les fonctions répertoriées dans le tableau suivant ne peuvent pas être utilisées dans les applications qui s'exécutent dans [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
|||  
|-|-|  
|[AtlHiMetricToPixel](../Topic/AtlHiMetricToPixel.md)|Unités HIMETRIC de convertis \(chaque unité est de 0,01 millimètres\) aux pixels.|  
|[AtlPixelToHiMetric](../Topic/AtlPixelToHiMetric.md)|Pixels de convertis en unités HIMETRIC \(chaque unité est de 0,01 millimètres\).|
