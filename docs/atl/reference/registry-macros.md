# Registry Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ces macros définissent les fonctionnalités utiles de bibliothèque de types et de Registre.  
  
|||  
|-|-|  
|[\_ATL\_STATIC\_REGISTRY](../Topic/_ATL_STATIC_REGISTRY.md)|Indique que le code d'inscription pour que votre objet soit dans l'objet pour éviter une dépendance sur ATL.DLL.|  
|[DECLARE\_LIBID](../Topic/DECLARE_LIBID.md)|Offre un moyen pour qu'ATL obtenir *le libid* de la bibliothèque de types.|  
|[DECLARE\_NO\_REGISTRY](../Topic/DECLARE_NO_REGISTRY.md)|Éviter l'inscription par défaut ATL.|  
|[DECLARE\_REGISTRY](../Topic/DECLARE_REGISTRY.md)|Écrit ou supprime l'entrée d'objet principal dans la base de registres.|  
|[DECLARE\_REGISTRY\_APPID\_RESOURCEID](../Topic/DECLARE_REGISTRY_APPID_RESOURCEID.md)|Spécifie les informations requises pour enregistrer automatiquement l' *appid*.|  
|[DECLARE\_REGISTRY\_RESOURCE](../Topic/DECLARE_REGISTRY_RESOURCE.md)|Recherche la ressource nommée et exécute le script de Registre dans celui\-ci.|  
|[DECLARE\_REGISTRY\_RESOURCEID](../Topic/DECLARE_REGISTRY_RESOURCEID.md)|Recherche la ressource identifiée par un numéro d'ID et exécute le script de Registre dans celui\-ci.|
