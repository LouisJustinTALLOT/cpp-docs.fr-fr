# <a name="strings-atl-property-page-wizard"></a>Chaînes, Assistant Page de propriétés ATL
Fournit le texte associé à la page de propriétés.  
  
 **Titre**  
 Définit le texte qui apparaît sous l’onglet de la page de propriétés.  
  
 **Chaîne du document**  
 Définit une chaîne de texte décrivant la page. Cette chaîne peut être affichée dans la boîte de dialogue de feuille de propriétés. Le frame de propriété peut utiliser la description dans une info-bulle état ligne ou l’outil. Actuellement, le frame de propriété standard n’utilise pas cette chaîne.  
  
 **Fichier d’aide**  
 Définit le nom du fichier d’aide qui décrit l’utilisation de la page de propriétés. Ce nom ne doit pas inclure un chemin d’accès. Lorsque l’utilisateur appuie sur **aide**, le frame ouvre le fichier d’aide dans le répertoire spécifié dans la valeur de la clé HelpDir dans les entrées de Registre des pages de propriété sous son CLSID.
