# Cha&#238;nes, Assistant Page de propri&#233;t&#233;s ATL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fournit le texte associé à la page de propriétés.  
  
 **Titre**  
 Définit le texte qui s'affiche sous l'onglet de la page de propriétés.  
  
 **Chaîne Doc**  
 Définit une chaîne de texte décrivant la page.  Cette chaîne peut s'afficher dans la boîte de dialogue de la feuille de propriétés.  Le frame de propriété peut indiquer la description dans une ligne d'état ou une info\-bulle.  Le frame de propriété standard n'utilise pas cette chaîne actuellement.  
  
 **Fichier d'aide**  
 Définit le nom du fichier d'aide décrivant la manière d'utiliser la page de propriétés.  Ce nom ne doit pas comprendre de chemin d'accès.  Si l'utilisateur clique sur **Aide**, le frame ouvre le fichier d'aide dans le répertoire spécifié dans la valeur de la clé HelpDir des entrées du Registre de la page de propriétés, sous son identificateur CLSID.
