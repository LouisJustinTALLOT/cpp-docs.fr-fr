# <a name="interface-pointers-classes"></a>Classes de pointeurs d’interface

Les classes suivantes gèrent un pointeur d’interface donné :

- [CComPtr](../atl/reference/ccomptr-class.md) Effectue un décompte de références automatique.

- [CComQIPtr](../atl/reference/ccomqiptr-class.md) Semblable à `CComPtr` , mais effectue également une interrogation automatique des interfaces.

- [CInterfaceArray](../atl/reference/cinterfacearray-class.md) Fournit des méthodes utiles lors de la construction d’un tableau de pointeurs d’interface COM.

- [CInterfaceList](../atl/reference/cinterfacelist-class.md) Fournit des méthodes utiles lors de la construction d’une liste de pointeurs d’interface COM.

- [CComGITPtr](../atl/reference/ccomgitptr-class.md) Fournit des méthodes pour traiter les pointeurs d’interface et la table GIT (Global Interface Table).
