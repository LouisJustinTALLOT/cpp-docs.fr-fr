# Interface Pointers Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les classes suivantes traitent un pointeur d'interface donné :  
  
-   [CComPtr](../atl/reference/ccomptr-class.md) exécute le décompte de références automatique.  
  
-   [CComQIPtr](../atl/reference/ccomqiptr-class.md) semblable à `CComPtr`, mais exécute également interroger automatique des interfaces.  
  
-   [CInterfaceArray](../atl/reference/cinterfacearray-class.md) fournit des méthodes utiles lorsque vous construisez un tableau de pointeurs d'interface COM.  
  
-   [CInterfaceList](../atl/reference/cinterfacelist-class.md) fournit des méthodes utiles lorsque vous construisez une liste des pointeurs d'interface COM.  
  
-   [CComGITPtr](../atl/reference/ccomgitptr-class.md) fournit des méthodes pour traiter les pointeurs d'interface et le tableau global interface \(GIT\).
