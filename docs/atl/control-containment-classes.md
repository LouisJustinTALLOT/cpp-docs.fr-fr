# <a name="control-containment-classes"></a>Classes de relation contenant-contenu de contrôle

Les classes suivantes fournissent la prise en charge de la relation contenant-contenu pour l’hébergement des contrôles :

- [CAxWindow](../atl/reference/caxwindow-class.md) Fournit des méthodes pour manipuler une fenêtre qui héberge un contrôle ActiveX.

- [CAxWindow2T](../atl/reference/caxwindow2t-class.md) Fournit des méthodes pour manipuler une fenêtre qui héberge un contrôle ActiveX et prend également en charge l’hébergement de contrôles ActiveX sous licence.

- [IAxWinAmbientDispatch](../atl/reference/iaxwinambientdispatch-interface.md) Appelez les méthodes sur cette interface pour définir les propriétés ambiantes disponibles pour un contrôle hébergé.

- [IAxWinHostWindow](../atl/reference/iaxwinhostwindow-interface.md) Appelez les méthodes sur cette interface pour créer et/ou attacher un contrôle à un objet hôte, ou pour obtenir une interface à partir d’un contrôle hébergé.
