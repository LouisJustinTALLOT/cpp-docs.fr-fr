# Class Factories Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les classes suivantes implémentent ou prennent en charge une fabrique de classe :  
  
-   [CComClassFactory](../atl/reference/ccomclassfactory-class.md) fournit une fabrique de classe par défaut pour la création d'objets.  
  
-   Création d'objets de contrôles de[CComClassFactory2](../atl/reference/ccomclassfactory2-class.md) via une licence.  
  
-   [CComClassFactoryAutoThread](../atl/reference/ccomclassfactoryautothread-class.md) permet des objets à créer dans plusieurs des apartments regroupés par thread.  
  
-   [CComClassFactorySingleton](../atl/reference/ccomclassfactorysingleton-class.md) crée un objet unique.  
  
-   [CComCoClass](../atl/reference/ccomcoclass-class.md) définit la fabrique de classe pour l'objet.
