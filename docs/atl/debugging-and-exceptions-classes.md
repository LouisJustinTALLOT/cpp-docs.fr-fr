# Debugging and Exceptions Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Ces classes fournissent la prise en charge de la gestion des exceptions et de le débogage.  
  
-   [CAtlDebugInterfacesModule](../atl/reference/catldebuginterfacesmodule-class.md) cette classe fournit la prise en charge des interfaces de débogage.  
  
-   [CAtlException](../atl/reference/catlexception-class.md) cette classe définit une exception ATL.
