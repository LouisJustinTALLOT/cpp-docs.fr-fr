# <a name="scope-of-atl"></a>Portée d’ATL

ATL vous permet de créer facilement des objets COM, des serveurs Automation et des contrôles ActiveX. ATL offre une prise en charge intégrée de la plupart des interfaces COM fondamentales.

ATL est fourni en tant que code source que vous incluez dans votre application. ATL rend également une DLL disponible (atl90.dll), qui contient le code qui peut être partagé entre les composants. Toutefois, cette DLL n’est pas nécessaire.
