# <a name="scope-of-atl"></a>Portée d’ATL

ATL vous permet de créer facilement des objets COM, des serveurs Automation et des contrôles ActiveX. ATL fournit une prise en charge intégrée pour la plupart des interfaces COM fondamentaux.

ATL est livré en tant que code source que vous incluez dans votre application. ATL met également à votre disposition une DLL (atl90.dll), qui contient le code qui peut être partagé entre plusieurs composants. Toutefois, cette DLL n’est pas nécessaire.
