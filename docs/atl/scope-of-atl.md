# Scope of ATL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ATL permet de créer facilement des contrôles d'objets COM, de serveurs Automation, et ActiveX.  ATL fournit la prise en charge intégrée pour plusieurs interfaces COM fondamentales.  
  
 ATL est fourni comme code source que vous incluez dans votre application.  ATL rend également une DLL disponible \(atl90.dll\), qui contient le code qui peut être partagé par plusieurs composants.  Toutefois, cette DLL n'est pas nécessaire.
