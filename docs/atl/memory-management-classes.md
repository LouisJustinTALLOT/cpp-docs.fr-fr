# Memory Management Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Ces classes fournissent la prise en charge des pointeurs de tas, les pointeurs intelligents, et d'autres routines d'allocation de mémoire.  
  
-   [CAutoPtr](../atl/reference/cautoptr-class.md) cette classe représente un objet de pointeur intelligent.  
  
-   [CAutoPtrArray](../atl/reference/cautoptrarray-class.md) cette classe fournit des méthodes utiles lorsque vous construisez un tableau de pointeurs intelligents.  
  
-   [CAutoPtrList](../atl/reference/cautoptrlist-class.md) cette classe fournit des méthodes utiles lorsque vous construisez une liste de pointeurs intelligents.  
  
-   [CAutoVectorPtr](../atl/reference/cautovectorptr-class.md) cette classe représente un objet de pointeur intelligent à nouveau vectorielles et des opérateurs delete.  
  
-   [CComAllocator](../atl/reference/ccomallocator-class.md) cette classe fournit des méthodes pour la gestion de la mémoire à des routines de mémoire COM.  
  
-   [CComGITPtr](../atl/reference/ccomgitptr-class.md) cette classe fournit des méthodes pour traiter les pointeurs d'interface et le tableau global interface \(GIT\).  
  
-   [CComHeap](../atl/reference/ccomheap-class.md) cette classe implémente [IAtlMemMgr](../atl/reference/iatlmemmgr-class.md) à l'aide de les fonctions d'allocation de mémoire COM.  
  
-   Classe intelligente du pointeur de[CComHeapPtr](../atl/reference/ccomheapptr-class.md) Un pour gérer des pointeurs de tas.  
  
-   Classe intelligente du pointeur de[CComPtr](../atl/reference/ccomptr-class.md) Un pour gérer des pointeurs d'interface COM.  
  
-   [CComPtrBase](../atl/reference/ccomptrbase-class.md) cette classe constitue une base pour les classes intelligentes de pointeur à l'aide de les routines de mémoire COM.  
  
-   Classe intelligente du pointeur de[CComQIPtr](../atl/reference/ccomqiptr-class.md) Un pour gérer des pointeurs d'interface COM.  
  
-   [CCRTAllocator](../atl/reference/ccrtallocator-class.md) cette classe fournit des méthodes pour la gestion de la mémoire à des routines de mémoire CRT.  
  
-   [CCRTHeap](../atl/reference/ccrtheap-class.md) cette classe implémente [IAtlMemMgr](../atl/reference/iatlmemmgr-class.md) à l'aide de les fonctions de tas CRT.  
  
-   [CGlobalHeap](../atl/reference/cglobalheap-class.md) cette classe implémente [IAtlMemMgr](../atl/reference/iatlmemmgr-class.md) à l'aide de les fonctions globales du tas Win32.  
  
-   [CHandle](../atl/reference/chandle-class.md) cette classe fournit des méthodes pour la création et l'utilisation d'un objet de handle.  
  
-   Classe intelligente du pointeur de[CHeapPtr](../atl/reference/cheapptr-class.md) Un pour gérer des pointeurs de tas.  
  
-   [CHeapPtrBase](../atl/reference/cheapptrbase-class.md) cette classe forme la base de plusieurs classes intelligentes de pointeur du tas.  
  
-   [CHeapPtrList](../atl/reference/cheapptrlist-class.md) cette classe fournit des méthodes utiles lorsque vous construisez une liste des pointeurs de tas.  
  
-   [CLocalHeap](../atl/reference/clocalheap-class.md) cette classe implémente [IAtlMemMgr](../atl/reference/iatlmemmgr-class.md) à l'aide de les fonctions locales de tas Win32.  
  
-   [CWin32Heap](../atl/reference/cwin32heap-class.md) cette classe implémente [IAtlMemMgr](../atl/reference/iatlmemmgr-class.md) à l'aide de les fonctions d'allocation du tas Win32.  
  
-   [IAtlMemMgr](../atl/reference/iatlmemmgr-class.md) cette classe représente l'interface à un gestionnaire de mémoire.
