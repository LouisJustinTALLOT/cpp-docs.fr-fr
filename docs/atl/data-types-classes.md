# Data Types Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les classes suivantes encapsulent des types de données C\+\+ :  
  
-   [CComBSTR](../atl/reference/ccombstr-class.md) encapsule le type de données d' `BSTR` .  
  
-   [CComVariant](../atl/reference/ccomvariant-class.md) encapsule le type de données de **variant** .  
  
-   [CComCurrency](../atl/reference/ccomcurrency-class.md) inclut des méthodes et des opérateurs pour créer et gérer un objet d' `CURRENCY` .
