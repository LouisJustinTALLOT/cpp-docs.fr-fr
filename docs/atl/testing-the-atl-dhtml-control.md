# <a name="testing-the-atl-dhtml-control"></a>Test du contrôle ATL DHTML
Une fois que vous avez créé votre projet, vous pouvez générer et tester l’exemple de contrôle. Avant cela, utilisez l’affichage de classes et Explorateur de solutions pour examiner le projet. Les éléments de votre projet sont décrits en détail dans [identifier les éléments du projet de contrôle DHTML](../atl/identifying-the-elements-of-the-dhtml-control-project.md).  
  
#### <a name="to-build-and-test-the-atl-dhtml-control"></a>Pour générer et tester le contrôle ATL DHTML  
  
1.  Générez le projet. À partir de la **générer** menu, cliquez sur **générer la Solution**.  
  
2.  Lorsque la build est terminée, ouvrez le conteneur de Test. Consultez [test des propriétés et des événements avec le conteneur de Test](../mfc/testing-properties-and-events-with-test-container.md) pour plus d’informations sur la façon d’accéder au conteneur de Test.  
  
3.  Dans le conteneur de Test, à partir de la **modifier** menu, cliquez sur **insérer un nouveau contrôle**.  
  
4.  Dans le **insérer un contrôle** boîte de dialogue, sélectionnez votre contrôle à partir de la zone de liste. N’oubliez pas, son nom est basé sur le nom court que vous avez indiquée dans l’Assistant contrôle ATL. Cliquez sur **OK**.  
  
5.  Examinez le contrôle. Notez qu’il possède une barre de défilement. Utilisez les poignées du contrôle pour redimensionner le contrôle pour activer la barre de défilement.  
  
6.  Testez les boutons du contrôle. La couleur d’arrière-plan modifie la couleur indiquée par le bouton.  
  
7.  Fermez le conteneur de Test.  
  
 Ensuite, essayez de [la modification du contrôle DHTML](../atl/modifying-the-atl-dhtml-control.md).
