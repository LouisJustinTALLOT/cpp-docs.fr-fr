# <a name="active-template-library-atl-tutorial"></a>Didacticiel ATL (Active Template Library)

ATL est conçu pour simplifier le processus de création de contrôles efficace, flexibles et légers. Ce didacticiel vous guide tout au long de la création d’un contrôle ActiveX, démonstration de nombreux principes de base de COM et ATL.

En suivant ce didacticiel, vous allez apprendre à ajouter un contrôle à un projet ATL qui dessine un cercle et un polygone plein. Vous allez ensuite ajouter une propriété pour indiquer le nombre de côtés du polygone et créer le code de dessin pour la mise à jour le contrôle lorsque la propriété change. Le contrôle sera ensuite être affiché sur une page Web à l’aide de VBScript pour répondre aux événements.

Le didacticiel est divisé en sept étapes. Vous devez effectuer chaque étape dans l’ordre comme étapes ultérieures dépendent des tâches terminées précédemment. Avant de commencer, vous devez confirmer que vous disposez des privilèges requis pour inscrire un composant ActiveX sur votre ordinateur. Cela est généralement uniquement un problème si vous exécutez Visual Studio .NET via une connexion de Services Terminal Server.
