# <a name="active-template-library-atl-tutorial"></a>Didacticiel ATL (Active Template Library)

ATL est conçu pour simplifier le processus de création de contrôles efficaces, flexibles et légers. Ce didacticiel vous guide tout au long de la création d’un contrôle ActiveX, illustrant de nombreuses notions de base sur ATL et COM.

En suivant ce didacticiel, vous allez apprendre à ajouter un contrôle à un projet ATL qui dessine un cercle et un polygone plein. Vous allez ensuite ajouter une propriété pour indiquer le nombre de côtés du polygone et créer du code de dessin pour mettre à jour le contrôle lorsque la propriété change. Le contrôle s’affichera ensuite sur une page Web à l’aide de VBScript pour lui permettre de répondre aux événements.

Ce didacticiel est divisé en sept étapes. Vous devez effectuer chaque étape dans l’ordre, car les étapes ultérieures dépendent des tâches effectuées précédemment. Avant de commencer, vous devez vous assurer que vous disposez des privilèges nécessaires pour inscrire un composant ActiveX sur votre ordinateur particulier. Ce n’est généralement qu’un problème si vous exécutez Visual Studio .NET sur une connexion aux services Terminal Server.
