# <a name="site-information-classes"></a>Classes d’informations de site

Les classes suivantes permettent à un objet de communiquer avec son site :

- [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) Récupère et définit un pointeur vers le site d’un objet. Utilisé pour les objets qui ne sont pas des contrôles.

- [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) Récupère et définit un pointeur vers le site d’un objet. Utilisé pour les contrôles.
