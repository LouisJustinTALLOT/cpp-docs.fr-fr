# <a name="site-information-classes"></a>Classes d’informations de site

Les classes suivantes permettent à un objet communiquer avec son site :

- [IObjectWithSiteImpl de prendre facilement](../atl/reference/iobjectwithsiteimpl-class.md) récupère et définit un pointeur vers site d’un objet. Utilisé pour les objets qui ne sont pas des contrôles.

- [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) récupère et définit un pointeur vers site d’un objet. Utilisé pour les contrôles.
