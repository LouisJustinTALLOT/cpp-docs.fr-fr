# Site Information Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les classes suivantes permettent à un objet pour communiquer avec son site :  
  
-   [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) extrait et définit un pointeur vers le site d'un objet.  Utilisé pour les objets qui ne sont pas des contrôles.  
  
-   [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) extrait et définit un pointeur vers le site d'un objet.  Utilisé pour les contrôles.
