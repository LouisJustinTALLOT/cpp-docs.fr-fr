# <a name="error-information-classes"></a>Classes d’informations sur l’erreur

La classe suivante indique comment les informations sur l’erreur sont gérées :

- [ISupportErrorInfoImpl](../atl/reference/isupporterrorinfoimpl-class.md) Détermine si l’objet prend en charge l’interface [IErrorInfo](/windows/win32/api/oaidl/nn-oaidl-ierrorinfo) . `IErrorInfo` permet de propager les informations d’erreur au client.
