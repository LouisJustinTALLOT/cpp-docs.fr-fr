# <a name="data-transfer-classes"></a>Classes de transfert de données

Les classes suivantes prennent en charge différents types de transfert de données :

- [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) prend en charge de transfert de données uniforme à l’aide de formats standards pour récupérer et définir des données. Traite les données des notifications de modifications par la gestion des connexions pour les récepteurs de notifications.

- [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md) permet un moniker asynchrone envoyer et recevoir des informations sur le transfert asynchrone des données vers et à partir de votre objet.
