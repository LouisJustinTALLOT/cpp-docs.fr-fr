# Data Transfer Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les différents types de transfert de données en charge suivant de classes :  
  
-   Uniform Data Transfer de prend en charge[IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) à l'aide de les formats standard pour récupérer et définir des données.  Traite des notifications de modification des données en gérant les connexions aux récepteurs de notifications.  
  
-   [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md) permet à un moniker asynchrone pour envoyer et recevoir des informations sur la migration de données asynchrone à partir de votre objet.
