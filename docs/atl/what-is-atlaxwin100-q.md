# What Is AtlAxWin100?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

`AtlAxWin100` est le nom d'une classe de fenêtre qui aide fournissent les fonctionnalités du contrôle hébergement ATL.  Lorsque vous créez une instance de cette classe, la procédure de fenêtre utilisera automatiquement l'API de contrôle hébergement pour créer un objet hôte associé à la fenêtre et pour le charger avec le contrôle que vous spécifiez comme titre de la fenêtre.
