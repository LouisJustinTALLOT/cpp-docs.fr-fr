# <a name="introduction-to-atl"></a>Introduction à ATL

ATL est le Active Template Library, un ensemble de classes C++ basées sur des modèles, avec lequel vous pouvez créer facilement des objets COM (Component Object Model) petits et rapides. Il offre une prise en charge spéciale des fonctionnalités COM clés, notamment : les implémentations de l’action [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown), [IClassFactory](/windows/win32/api/unknwnbase/nn-unknwnbase-iclassfactory), [IClassFactory2](/windows/win32/api/ocidl/nn-ocidl-iclassfactory2)et les `IDispatch` interfaces doubles, les interfaces d’énumérateur com standard, les points de connexion, les interfaces détachables et les contrôles ActiveX.

Le code ATL peut être utilisé pour créer des objets à thread unique, des objets de modèle Apartment, des objets de modèle à thread libre, ou des objets de modèle cloisonné et à thread libre.
