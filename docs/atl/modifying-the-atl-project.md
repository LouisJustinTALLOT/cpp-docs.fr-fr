# <a name="modifying-the-atl-project"></a>Modification du projet ATL
À ce stade, votre projet de contrôle composite implémente les objets nécessaires pour votre contrôle composite. L’étape suivante consiste à ajouter les contrôles que le contrôle composite contenir et de gérer les événements nécessaires.  
  
 Pour ajouter des contrôles ActiveX ou Windows supplémentaires, ajoutez un nouveau script de ressources, puis utilisez l’éditeur de boîte de dialogue. Pour plus d’informations sur l’ajout de contrôles (et les tâches associées), consultez [boîte de dialogue Éditeur](../windows/dialog-editor.md).  
  
 Pour gérer les événements nécessaires à partir des contrôles ActiveX, consultez [Ajout de fonctionnalités au contrôle Composite](../atl/adding-functionality-to-the-composite-control.md).
