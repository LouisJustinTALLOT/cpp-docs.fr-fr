# Modifying the ATL Project
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

À ce stade, votre projet de contrôle composite implémente les objets nécessaires à votre contrôle composite.  L'étape suivante consiste à ajouter les contrôles que le contrôle composite contient et gérera les événements nécessaires.  
  
 Pour ajouter ActiveX supplémentaire ou des contrôles Windows, ajoutez un nouveau script de ressources puis utilisez l'éditeur de boîtes de dialogue.  Pour plus d'informations sur l'ajout de contrôles \(et les tâches associées\), consultez l' [Éditeur de boîtes de dialogue](../mfc/dialog-editor.md).  
  
 Pour gérer tous les événements nécessaires des contrôles ActiveX, consultez l' [Ajouter des fonctionnalités au contrôle composite](../atl/adding-functionality-to-the-composite-control.md).
