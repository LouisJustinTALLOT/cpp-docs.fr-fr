# Persistence Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La persistance suivante d'objet d'implémentation de classes :  
  
-   [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) permet à un client pour charger et enregistrer des propriétés de l'objet à un conteneur des propriétés.  
  
-   [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md) permet à un client pour charger et enregistrer les données persistantes d'un objet dans un flux.  
  
-   [IPersistStorageImpl](../atl/reference/ipersiststorageimpl-class.md) permet à un client pour charger et enregistrer les données persistantes d'un objet en mémoire.
