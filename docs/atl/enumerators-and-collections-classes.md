# Enumerators and Collections Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les classes suivantes fournissent la prise en charge des collections et des énumérations COM :  
  
-   [CComEnum](../atl/reference/ccomenum-class.md) définit un objet énumérateur COM selon un tableau.  
  
-   [CComEnumImpl](../atl/reference/ccomenumimpl-class.md) fournit l'implémentation pour une interface d'énumérateur COM dans lequel les éléments sont énumérés sont stockés dans un tableau.  
  
-   [CComEnumOnSTL](../atl/reference/ccomenumonstl-class.md) définit un objet énumérateur COM selon une collection STL.  
  
-   [IEnumOnSTLImpl](../atl/reference/ienumonstlimpl-class.md) fournit l'implémentation pour une interface d'énumérateur COM dans lequel les éléments sont énumérés sont stockés dans un conteneur STL\- compatible.  
  
-   [ICollectionOnSTLImpl](../atl/reference/icollectiononstlimpl-class.md) fournit l'implémentation pour **Nombre**, **Élément**, les propriétés et d' `_NewEnum` d'une interface de la collection.
