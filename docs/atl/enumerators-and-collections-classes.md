# <a name="enumerators-and-collections-classes"></a>Énumérateurs et classes de collections

Les classes suivantes fournissent la prise en charge des énumérations et des collections COM :

- [CComEnum](../atl/reference/ccomenum-class.md) Définit un objet énumérateur COM basé sur un tableau.

- [CComEnumImpl](../atl/reference/ccomenumimpl-class.md) Fournit l’implémentation pour une interface d’énumérateur COM où les éléments qui sont énumérés sont stockés dans un tableau.

- [CComEnumOnSTL](../atl/reference/ccomenumonstl-class.md) Définit un objet énumérateur COM basé sur une collection de bibliothèques standard C++.

- [IEnumOnSTLImpl](../atl/reference/ienumonstlimpl-class.md) Fournit l’implémentation pour une interface d’énumérateur COM où les éléments qui sont énumérés sont stockés dans un conteneur compatible avec la bibliothèque standard C++.

- [ICollectionOnSTLImpl](../atl/reference/icollectiononstlimpl-class.md) Fournit l’implémentation pour les `Count` `Item` Propriétés, et `_NewEnum` d’une interface de collection.
