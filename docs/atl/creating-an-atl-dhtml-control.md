# Creating an ATL DHTML Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

L'Assistant Contrôle ATL automatiser le processus de création d'un contrôle DHTML.  Il génère les fichiers de ressources nécessaires, y compris un fichier HTML qui contient l'exemple de code.  
  
#### Pour créer un contrôle ATL DHTML  
  
1.  Suivez les étapes de [créer un projet ATL](../atl/reference/creating-an-atl-project.md).  
  
2.  Dans l'Affichage de classes, cliquez avec le bouton droit sur le nœud de projet, pointez sur **Ajouter**, puis cliquez sur **Add Class** le menu contextuel.  Dans la boîte de dialogue d' **Add Class** , double\-cliquez sur [L'Assistant Contrôle ATL](../atl/reference/atl-control-wizard.md).  Dans l'Assistant Contrôle ATL, cliquez sur l'onglet **Options** et sélectionnez **DHTML control**.  
  
 Vous pouvez maintenant [testez le contrôle par défaut](../atl/testing-the-atl-dhtml-control.md).
