# <a name="creating-an-atl-dhtml-control"></a>Création d’un contrôle ATL DHTML
L’Assistant contrôle ATL automatise le processus de création d’un contrôle DHTML. Il génère des fichiers de ressources nécessaires, y compris un fichier HTML qui contient des exemples de code.  
  
#### <a name="to-create-an-atl-dhtml-control"></a>Pour créer un contrôle ATL DHTML  
  
1.  Suivez les étapes de [création d’un projet ATL](../atl/reference/creating-an-atl-project.md).  
  
2.  Dans l’affichage de classes, cliquez sur le nœud du projet, pointez sur **ajouter**, puis cliquez sur **ajouter une classe** dans le menu contextuel. Dans le **ajouter une classe** boîte de dialogue, double-cliquez sur le [Assistant contrôle ATL](../atl/reference/atl-control-wizard.md). Dans l’Assistant contrôle ATL, cliquez sur le **Options** onglet et sélectionnez **contrôle DHTML**.  
  
 Vous pouvez désormais [tester le contrôle par défaut](../atl/testing-the-atl-dhtml-control.md).
