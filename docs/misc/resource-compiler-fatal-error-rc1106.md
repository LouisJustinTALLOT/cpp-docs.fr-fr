# Erreur irr&#233;cup&#233;rable&#160;RC1106 du compilateur de ressources
option switch: non valide  
  
 L’option n’est pas une option de compilateur de ressources valide.  
  
 Les options valides sont les suivantes :  
  
 \/r  
  
 Émettre un fichier .res  
  
 \/v  
  
 Commentaires \(imprimer les messages d’avancement\)  
  
 \/d  
  
 Définir un symbole  
  
 \/fo  
  
 Renommer un fichier .res  
  
 \/l  
  
 ID de langue par défaut au format hexadécimal  
  
 \/I  
  
 Ajouter un chemin pour les recherches **INCLUDE**  
  
 \/x  
  
 Ignorer la variable d’environnement **INCLUDE**  
  
 \/c  
  
 Définir une page de code utilisée par la conversion NLS
