# Wizard file &lt;name&gt; is missing a required value.
Le fichier .vsz de l'Assistant ne possède pas le format correct.  Cette erreur se produit si un fichier .vsz est endommagé ou qu'il provient d'un produit différent de Visual Studio.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que l'Assistant est un Assistant Visual Studio et réessayez.  
  
     \- ou \-  
  
     Réinstallez Visual Studio pour remplacer le fichier endommagé.
