# MSBuild Error MSB3110
**MSB3110 : L'assembly '\<assembly\>' a une identité incompatible '\<IdentitéAssembly\>', nom de fichier attendu : '\<fichier\>'.**  
  
 Cette erreur est générée lorsque le nom de fichier de l'assembly obtenu à partir de l'identité de l'assembly ne correspond pas au nom de fichier réel.  
  
### Pour corriger cette erreur  
  
-   Corrigez le nom de fichier de l'assembly afin qu'il corresponde à l'identité de l'assembly.
