# Argument is missing in the pattern.
Cette erreur se produit généralement quand une syntaxe incomplète a été utilisée pour une expression régulière entrée dans la chaîne recherchée.  
  
### Pour corriger cette erreur  
  
1.  Entrez à nouveau la chaîne.  Pour plus d'informations sur la syntaxe, consultez [Expressions régulières](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md).
