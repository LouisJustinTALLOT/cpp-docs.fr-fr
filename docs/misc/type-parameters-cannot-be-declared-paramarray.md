# Les param&#232;tres &lt;type&gt; ne peuvent pas &#234;tre d&#233;clar&#233;s &#39;ParamArray&#39;
Une définition d’un délégué, événement ou opérateur déclare un paramètre [ParamArray](../Topic/ParamArray%20\(Visual%20Basic\).md).  
  
 Les paramètres `ParamArray` sont uniquement autorisés sur les paramètres `Declare`, `Function`, `Property` et `Sub`.  
  
 **ID d’erreur :** BC33009  
  
### Pour corriger cette erreur  
  
-   Supprimez le mot clé `ParamArray` de la liste de paramètres.  
  
-   Si vous définissez un opérateur, vous pouvez peut\-être obtenir la fonctionnalité `ParamArray` avec une série de surcharges.  
  
-   Si vous définissez un délégué ou un événement, vous devez revoir toute la logique de cette partie de votre application. Vous ne pouvez pas utiliser les paramètres [Optional](../Topic/Optional%20\(Visual%20Basic\).md) ou `ParamArray`, ni les versions surchargées, sur des paramètres de délégué ou d’événement.
