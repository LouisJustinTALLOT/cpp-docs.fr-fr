# D&#233;pannage des exceptions&#160;: System.ServiceModel.Dispatcher.NavigatorInvalidBodyAccessException
Une exception <xref:System.ServiceModel.Dispatcher.NavigatorInvalidBodyAccessException> est renvoyée lorsqu'il est demandé à un <xref:System.Xml.XPath.XPathNavigator> d'examiner le corps d'un message qui n'est pas mis en mémoire tampon.
