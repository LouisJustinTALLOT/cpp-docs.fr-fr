# Erreur MSBuild MSB4141
**MSB4141: MSBuildToolsPath n'est pas spécifié pour le ToolsVersion défini à "&lt ; x.x &gt ;".**  
  
 Un ensemble d'outils personnalisé est défini mais aucune valeur n'est spécifiée pour `MSBuildToolsPath`.  
  
### Pour corriger cette erreur  
  
-   Spécifiez une valeur valide pour `MSBuildToolsPath` lorsque vous définissez un ensemble d'outils personnalisé dans le Registre ou le fichier de configuration [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)].
