# D&#233;pannage des exceptions&#160;: System.IdentityModel.Selectors.IdentityValidationException
Une exception <xref:System.IdentityModel.Selectors.IdentityValidationException> est renvoyée si le certificat du destinataire n'est pas valide. Cela peut se produire si le certificat ne s'enchaîne pas avec succès au certificat racine.
