# L’argument &lt;nom_argument&gt; ne peut pas &#234;tre une cha&#238;ne vide ou avoir la valeur Nothing
L’argument exige une valeur autre qu’une chaîne vide \(""\) ou `Nothing`. Cette erreur peut être le résultat d’un calcul incorrect de la valeur.  
  
### Pour corriger cette erreur  
  
-   Fournissez une valeur valide pour l’argument.
