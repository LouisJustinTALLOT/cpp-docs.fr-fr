# Erreur du compilateur CS1630
Option 'option' non valide pour \/errorreport ; doit avoir la valeur prompt, send, queue ou none  
  
 L’option de ligne de commande [\/errorreport](../Topic/-errorreport%20\(C%23%20Compiler%20Options\).md) doit être suivie de **prompt**, **send**, **queue** ou **none**, qui spécifient l’action qui doit être effectuée quand une erreur interne du compilateur se produit.
