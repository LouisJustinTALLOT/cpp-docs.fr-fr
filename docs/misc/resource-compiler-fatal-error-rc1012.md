# Erreur irr&#233;cup&#233;rable RC1012 du compilateur de ressources 
parenthèses non équilibrées : 'caractère' manquant\(e\)  
  
 Les parenthèses au niveau d’une directive de préprocesseur ne sont pas équilibrées.
