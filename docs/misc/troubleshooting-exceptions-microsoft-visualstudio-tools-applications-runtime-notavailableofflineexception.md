# D&#233;pannage des exceptions&#160;: Microsoft.VisualStudio.Tools.Applications.Runtime.NotAvailableOfflineException
Exception levée lorsqu'une tentative est faite pour appeler un assembly, mais que l'assembly n'est pas disponible dans le cache ou sur le réseau.
