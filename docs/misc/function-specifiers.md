# Sp&#233;cificateurs de fonction
Vous pouvez utiliser les mots clés [inline](../misc/inline-inline-forceinline.md) et [virtual](../cpp/virtual-specifier.md) comme spécificateurs dans les déclarations de fonction. Cette utilisation de **virtual** diffère de son utilisation dans le spécificateur de classe de base d'une définition de classe.
