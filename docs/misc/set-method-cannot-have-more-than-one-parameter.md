# La m&#233;thode &#39;Set&#39; ne peut pas comporter plusieurs param&#232;tres
Plusieurs paramètres sont définis pour des procédures `Set` de propriété.  
  
 **ID d’erreur :** BC31063  
  
### Pour corriger cette erreur  
  
-   Supprimez les paramètres en trop des procédures `Set` de propriété.
