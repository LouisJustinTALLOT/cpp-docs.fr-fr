# Object with program ID &lt;number&gt; cannot be created.
La chaîne de classe que vous essayez d'utiliser n'est pas valide.  Ce message indique une erreur du système d'exploitation.  
  
### Pour corriger cette erreur  
  
1.  Réinstallez Visual Studio ou l'application concernée et recommencez.
