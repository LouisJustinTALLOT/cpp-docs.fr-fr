# Some Unicode characters in this file will not be saved.
Cette erreur se produit généralement lorsqu'un fichier contient des caractères qui ne peuvent être enregistrés qu'en appliquant une page de code Unicode.  
  
### Pour enregistrer le fichier avec des caractères Unicode  
  
1.  Cliquez sur **Annuler**.  
  
2.  Dans le menu **Fichier**, sélectionnez **Options d'enregistrement avancées**.  
  
3.  Dans la boîte de dialogue **Options d'enregistrement avancées**, choisissez l'encodage approprié à votre fichier.
