# D&#233;pannage des exceptions&#160;: System.Web.Caching.DatabaseNotEnabledForNotificationException
Exception levée lorsqu'une base de données Microsoft SQL Server ne peut pas prendre en charge les dépendances associées à la classe <xref:System.Web.Caching.SqlCacheDependency>.
