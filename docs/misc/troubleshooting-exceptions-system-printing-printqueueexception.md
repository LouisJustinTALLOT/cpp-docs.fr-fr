# D&#233;pannage des exceptions&#160;: System.Printing.PrintQueueException
Une exception <xref:System.Printing.PrintQueueException> est renvoyée lorsqu'une condition d'erreur empêche l'accès ou la création d'une <xref:System.Printing.PrintQueue>.
