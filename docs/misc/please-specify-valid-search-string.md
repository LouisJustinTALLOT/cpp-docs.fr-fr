# Please specify valid search string.
Cette erreur se produit généralement lors d'une tentative de recherche sans chaîne de recherche spécifiée dans **Rechercher**, dans la boîte de dialogue **Rechercher un symbole**.  
  
### Pour corriger cette erreur  
  
1.  Tapez le texte correct dans la zone de texte **Rechercher** et choisissez à nouveau **Rechercher**.
