# D&#233;pannage des exceptions&#160;: System.ServiceModel.MsmqException
Une exception <xref:System.ServiceModel.MsmqException> encapsule les erreurs retournées par Message Queuing. Cette exception est renvoyée par le transport et le canal d'intégration Message Queuing.
