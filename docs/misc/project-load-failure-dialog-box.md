# &#201;chec du chargement du projet, bo&#238;te de dialogue
Cette boîte de dialogue apparaît lorsqu'un un fichier projet ne peut pas être chargé.  
  
 **Corrigez les erreurs spécifiées et rechargez le projet.**  
 Donne des informations sur l'erreur.  Utilisez les informations de cette boîte de dialogue pour localiser et résoudre l'erreur dans le fichier projet.  Enregistrez votre projet, puis rechargez\-le dans l'IDE.  
  
 **Ne plus afficher cette boîte de dialogue pendant cette session**  
 Lorsqu'elle est sélectionnée, la boîte de dialogue **Échec du chargement du projet** n'apparaît de nouveau qu'à la prochaine ouverture de l'IDE.
