# CA0057
Les violations ne peuvent pas être mappées sur l'ensemble actuel de cibles et de règles  
  
 Cette erreur se produit si FxCopCmd.exe est appelé manuellement avec un fichier projet ou un rapport XML importé qui contient des violations qui ne peuvent pas être mappées sur l'ensemble actuel de cibles et de règles.  
  
 Se produit uniquement si l'analyse est effectuée à l'aide de la version de ligne de commande de l'outil.
