# Erreur du compilateur CS1542
'dll' ne peut pas être ajouté à cet assembly, car il s’agit déjà d’un assembly ; utilisez plutôt l’option '\/R'  
  
 Le fichier référencé avec l’option du compilateur [\/addmodule](../Topic/-addmodule%20\(C%23%20Compiler%20Options\).md) n’a pas été généré avec [\/target:module](../Topic/-target:module%20\(C%23%20Compiler%20Options\).md) ; utilisez [\/reference](../Topic/-reference%20\(C%23%20Compiler%20Options\).md) pour faire référence au fichier dans cette compilation.
