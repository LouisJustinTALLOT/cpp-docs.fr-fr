# The application definition file is already opened in an incompatible editor
**Le fichier de définition d'application est déjà ouvert dans un éditeur incompatible.  Fermez l'autre éditeur et rechargez la page des propriétés de projet.**  
  
 Cette erreur se produit lorsque vous essayez d'ouvrir le fichier de définition d'application et qu'il est déjà ouvert dans un autre éditeur.  
  
### Pour corriger cette erreur  
  
1.  Fermez l'autre éditeur.  
  
2.  Ouvrez le Concepteur de projets.  
  
3.  Dans le menu **Projet**, cliquez sur **Propriétés** pour rouvrir le Concepteur de projets.
