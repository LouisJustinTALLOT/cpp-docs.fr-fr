# CA0062
La règle '*RuleId*' référencée dans l'ensemble de règles '*RuleSetName*' est introuvable.  
  
 Cette erreur indique que la règle spécifiée n'a pas pu être trouvée.  
  
 Cette erreur se produit généralement parce qu'un fichier ruleset a été modifié à la main.  Pour éviter cette erreur, utilisez l'éditeur d'ensemble de règles de [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] pour définir les règles.  Pour plus d'informations, consultez [Création d'ensembles de règles personnalisés](../Topic/Creating%20Custom%20Code%20Analysis%20Rule%20Sets.md).  
  
 Pour résoudre ce problème, assurez\-vous que tous les ID de contrôle qui sont spécifiés dans le fichier de ruleset sont valides. si vous utilisez des fichiers non\-standard de règle, faites et les chemins d'accès appropriées de l'indicateur de règle sont spécifiés dans le fichier de ruleset.
