# D&#233;pannage des exceptions&#160;: System.ServiceModel.ProtocolException
Une exception <xref:System.ServiceModel.ProtocolException> se produit sur le client lorsque la communication avec le tiers distant est impossible à cause de protocoles de transfert de données incompatibles.  
  
## Notes  
 L'<xref:System.ServiceModel.ProtocolException> peut résulter de liaisons incompatibles entre des points de terminaison, de l'utilisation d'une adresse de point de terminaison distant incorrecte ou d'une configuration incorrecte similaire.
