# S&#233;lectionnez plusieurs plateformes. Si vous voulez apporter des modifications &#224; une seule plateforme, quittez la bo&#238;te de dialogue Plateformes multiples et s&#233;lectionnez cette configuration dans la liste d&#233;roulante des plateformes.
Cette erreur se produit généralement quand une seule plateforme a été sélectionnée dans la boîte de dialogue **Plateformes multiples**.  
  
### Pour corriger cette erreur  
  
1.  Sélectionnez d’autres plateformes.  
  
     \- ou \-  
  
2.  Si vous voulez mettre à jour une seule plateforme, choisissez Annuler et sélectionnez la plateforme appropriée dans la liste des plateformes.
