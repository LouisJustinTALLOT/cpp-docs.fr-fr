# D&#233;pannage des exceptions&#160;: System.RankException
Une exception <xref:System.RankException> est levée lorsqu'un tableau ayant un nombre incorrect de dimensions est passé à une méthode.  
  
## Conseils associés  
 **Assurez\-vous que votre tableau dispose du nombre de dimensions requis.**  
 Pour plus d’informations, les utilisateurs de Visual Basic peuvent consulter [Array Dimensions in Visual Basic](../Topic/Array%20Dimensions%20in%20Visual%20Basic.md).  
  
 Pour plus d’informations, les utilisateurs de C\# peuvent consulter [Tableaux](../Topic/Arrays%20\(C%23%20Programming%20Guide\).md).  
  
## Remarques  
 Compte tenu que les indices de tableau commencent à 0, le plus petit indice disponible pour chaque dimension est toujours 0.
