# D&#233;pannage des exceptions&#160;: System.Workflow.Activities.Rules.RuleSetValidationException
Une exception <xref:System.Workflow.Activities.Rules.RuleSetValidationException> est levée lorsque le traitement ne peut pas se poursuivre car un jeu de règles ne peut pas être validé. L'exception est renvoyée par le constructeur <xref:System.Workflow.Activities.Rules.RuleEngine> si le <xref:System.Workflow.Activities.Rules.RuleSet> utilisé comporte des erreurs de validation.
