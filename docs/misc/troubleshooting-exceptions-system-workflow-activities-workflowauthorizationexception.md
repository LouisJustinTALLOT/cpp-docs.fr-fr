# D&#233;pannage des exceptions&#160;: System.Workflow.Activities.WorkflowAuthorizationException
Une exception <xref:System.Workflow.Activities.WorkflowAuthorizationException> est levée lorsque la validation du rôle échoue en raison d'une identité spécifiée qui n'est pas contenue dans <xref:System.Workflow.Activities.WorkflowRoleCollection>.
