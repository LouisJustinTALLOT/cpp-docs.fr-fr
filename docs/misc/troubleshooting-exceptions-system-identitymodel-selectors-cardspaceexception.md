# D&#233;pannage des exceptions&#160;: System.IdentityModel.Selectors.CardSpaceException
Une exception <xref:System.IdentityModel.Selectors.CardSpaceException> est renvoyée lorsqu'une ou plusieurs exceptions se sont produites au niveau du service CardSpace. La cause de l'erreur se trouve dans le journal des événements.  
  
## Notes  
 Les conditions à l'origine de cette exception comprennent la survenue d'une erreur pendant le chargement d'un magasin ou d'un registre.
