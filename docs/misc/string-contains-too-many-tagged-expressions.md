# String contains too many tagged expressions.
Cette erreur se produit généralement lorsque plus de neuf expressions avec balises ont été spécifiées dans la chaîne recherchée.  
  
### Pour corriger cette erreur  
  
1.  Réduisez le nombre d'expressions avec balises spécifiées.
