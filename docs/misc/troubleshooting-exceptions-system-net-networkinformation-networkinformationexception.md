# D&#233;pannage des exceptions&#160;: System.Net.NetworkInformation.NetworkInformationException
Exception levée lorsqu'une erreur se produit pendant la récupération d'informations réseau.  
  
## Notes  
 Les types de l'espace de noms <xref:System.Net.NetworkInformation> lèvent cette exception lorsqu'un appel à une fonction Win32 échoue. La propriété <xref:System.Net.NetworkInformation.NetworkInformationException.ErrorCode%2A> contient le résultat retourné par la fonction.
