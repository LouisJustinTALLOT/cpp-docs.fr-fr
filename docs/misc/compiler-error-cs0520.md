# Erreur du compilateur CS0520
Le type prédéfini 'nom' n’est pas déclaré correctement  
  
 Un ou plusieurs fichiers requis par le compilateur sont introuvables. Réinstallez le Common Language Runtime \(CLR\) .NET Framework.
