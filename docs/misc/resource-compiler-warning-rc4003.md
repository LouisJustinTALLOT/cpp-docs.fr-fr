# Avertissement RC4003 du compilateur de ressources
nombre de paramètres réels insuffisants pour la macro 'identificateur'  
  
 Le nombre de paramètres réels spécifiés avec l’identificateur donné est inférieur au nombre de paramètres formels spécifiés dans la définition de macro de l’identificateur.  
  
 Quand un paramètre formel est référencé dans la définition et que le paramètre réel correspondant n’a pas été fourni, du texte vide est inséré dans l’expansion macro.
