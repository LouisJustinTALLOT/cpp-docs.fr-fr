# Cannot find main menu commandbar in resources.
Cette erreur se produit lorsque l'environnement de développement intégré ne parvient pas à trouver la barre de commandes MenuBar au démarrage.  La .dll a été supprimée, déplacée ou endommagée.  
  
### Pour corriger cette erreur  
  
-   Utilisez **Ajout\/Suppression de programmes** pour réparer ou réinstaller le produit.
