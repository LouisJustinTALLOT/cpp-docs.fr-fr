# D&#233;pannage des exceptions&#160;: Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException
Une exception <xref:Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException> est levée lorsque votre code essaie d'accéder à un contrôle absent du document. Le contrôle a pu être supprimé par l'utilisateur final ou par un autre code. Il n'y a aucune façon de vérifier la présence du contrôle avant de tenter d'y accéder ; si le contrôle a été supprimé, la solution ne fonctionnera pas.
