# This project cannot be added to the solution because it already exists in the solution in a hidden Solution Folder.
Un projet ne peut exister dans une solution qu'à un seul emplacement.  
  
### Pour corriger cette erreur  
  
1.  Pour afficher les Dossiers Solution masqués, sélectionnez la solution ou de Dossier Solution qui contient un Dossier Solution masqué, puis dans le menu **Projet**, sélectionnez **Afficher les dossiers**.
