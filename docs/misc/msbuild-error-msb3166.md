# MSBuild Error MSB3166
**MSB3166 : Impossible de trouver le fichier requis '\<fichier\>' pour l'élément '\<package\>'.**  
  
 Cette erreur est générée lorsqu'un CLUF ou un fichier de contrôle externe est manquant.  Assurez\-vous que le fichier est présent sur le disque.  
  
### Pour corriger cette erreur  
  
-   Réinstallez le Kit de développement Visual Studio SDK  
  
     \- ou \-  
  
-   Copiez les fichiers requis vers l'emplacement approprié.
