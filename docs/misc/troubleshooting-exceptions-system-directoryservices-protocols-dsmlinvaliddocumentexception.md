# D&#233;pannage des exceptions&#160;: System.DirectoryServices.Protocols.DsmlInvalidDocumentException
Exception levée lorsqu'un document de demande ou de réponse DSML \(Directory Services Markup Language\) n'est pas un document XML correctement construit, ou lorsqu'il ne peut pas être validé avec le schéma DSMLV2.
