# Erreur RC2126 du compilateur de ressources
chaîne de menu attendue  
  
 Chaque instruction **MENUITEM** et **POPUP** doit contenir un champ de *texte*. Ce champ correspond à une chaîne placée entre guillemets doubles qui spécifie le nom de l’élément de menu ou le menu contextuel. Une instruction**MENUITEM SEPARATOR** ne nécessite pas de chaîne entre guillemets.
