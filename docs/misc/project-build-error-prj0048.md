# Erreur de g&#233;n&#233;ration de projet PRJ0048
Impossible de démarrer une génération parce qu'une génération est déjà en cours.  
  
 Vous ne pouvez pas commencer une build si une autre build est en cours. Attendez que la build en cours se termine ou annulez\-la.
