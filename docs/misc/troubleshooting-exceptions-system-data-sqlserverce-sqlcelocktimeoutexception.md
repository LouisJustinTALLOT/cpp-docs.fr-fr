# D&#233;pannage des exceptions&#160;: System.Data.SqlServerCe.SqlCeLockTimeoutException
Une exception `System.Data.SqlServerCe.SqlCeLockTimeoutException` est levée si le délai d'attente du verrouillage a été atteint et SQL Server Compact a dépassé le délai d'attente pour l'obtention d'un verrou. Le délai d'attente par défaut est de 2 000 millisecondes. Pour plus d’informations, consultez [Bibliothèque de classes SQL Server Compact 3.5](http://go.microsoft.com/fwlink/?LinkID=102595).
