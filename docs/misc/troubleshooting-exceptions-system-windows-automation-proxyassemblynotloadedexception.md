# D&#233;pannage des exceptions&#160;: System.Windows.Automation.ProxyAssemblyNotLoadedException
Une exception <xref:System.Windows.Automation.ProxyAssemblyNotLoadedException> contient les informations concernant une exception renvoyée lorsqu'un problème se produit lors du chargement d'un assembly contenant des fournisseurs côté client.
