# D&#233;pannage des exceptions&#160;: System.Web.Caching.TableNotEnabledForNotificationException
Exception levée lorsqu'une classe <xref:System.Web.Caching.SqlCacheDependency> est utilisée avec une table de base de données qui n'est pas activée pour les notifications de modifications.
