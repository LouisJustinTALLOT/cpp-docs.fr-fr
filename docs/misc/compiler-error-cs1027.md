# Erreur du compilateur CS1027
directive \#endif attendue  
  
 Aucune `#endif` [directive de préprocesseur](../Topic/C%23%20Preprocessor%20Directives.md) correspondante n’a été trouvée pour une directive `#if` spécifiée. Ou : le compilateur a trouvé une directive `#endregion` alors qu’aucune directive `#region` correspondante ne se trouve dans un bloc `#if`.  
  
 L’exemple suivant génère l’erreur CS1027 :  
  
```  
// CS1027.cs #if true   // CS1027, uncomment next line to resolve // #endif namespace x { public class clx { public static void Main() { } } }  
```
