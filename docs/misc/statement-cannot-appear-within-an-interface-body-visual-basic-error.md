# L’instruction ne peut pas appara&#238;tre dans le corps d’une interface (erreur Visual Basic)
Une construction de langage inattendue a été rencontrée. Une construction `End Interface` est peut\-être manquante et tout code source après ce point est en dehors de l’interface.  
  
 **ID d’erreur :** BC30604  
  
### Pour corriger cette erreur  
  
1.  Vérifiez la syntaxe du code utilisée dans la définition de l’interface.  
  
2.  Vérifiez que la définition de l’interface se termine par une construction `End Interface`.
