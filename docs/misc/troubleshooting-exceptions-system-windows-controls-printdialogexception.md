# D&#233;pannage des exceptions&#160;: System.Windows.Controls.PrintDialogException
L'exception <xref:System.Windows.Controls.PrintDialogException> est levée lorsqu'une condition d'erreur se produit pendant l'ouverture ou l'utilisation de <xref:System.Windows.Controls.PrintDialog>, ou pendant l'accès de ce dernier.
