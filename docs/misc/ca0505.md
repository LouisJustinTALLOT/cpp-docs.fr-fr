# CA0505
La propriété déconseillée sera ignorée car la propriété prioritaire a été définie.  
  
 L'avertissement CA505 est généré lorsque les propriétés d'analyse du code déconseillées et prioritaires spécifiées contiennent les valeurs d'un fichier de projet ou de configuration de build.  Pour résoudre ce problème, supprimez la propriété déconseillée.
