# Erreur irr&#233;cup&#233;rable RC1049 du compilateur de ressources
argument numérique non valide 'string'  
  
 Le compilateur de ressources attendait un argument numérique, mais a reçu la chaîne donnée.  
  
 Cette erreur peut être due à l’utilisation d’un nombre hexadécimal sans le préfixe \\x nécessaire ou à un nombre à virgule flottante incorrect.
