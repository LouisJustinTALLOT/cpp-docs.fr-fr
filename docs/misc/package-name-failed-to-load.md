# Package &lt;name&gt; failed to load.
Le package n'a pas pu être ouvert.  Cette erreur se produit généralement lorsque le package n'a pas été inscrit correctement, qu'une DLL est absente du package, qu'une DLL est endommagée, ou qu'une DLL requise dans un autre composant par le package est absente.  
  
### Pour corriger cette erreur  
  
1.  Réinstallez le produit qui fournit le package.
