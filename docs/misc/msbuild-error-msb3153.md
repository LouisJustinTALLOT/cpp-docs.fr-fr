# MSBuild Error MSB3153
**MSB3153 : La validation XML n'a pas réussi pour l'élément '\<package\>' situé dans '\<dossier\>'.**  
  
 Cet avertissement est généré lorsque le manifeste \(package.xml\) ne réussit pas la validation XML.  Les problèmes spécifiques sont répertoriés dans un message d'erreur ultérieur \([MSBuild Error MSB3159](../misc/msbuild-error-msb3159.md) ou [MSBuild Error MSB3160](../misc/msbuild-error-msb3160.md)\).  
  
### Pour corriger cette erreur  
  
-   Résolvez les problèmes de validation du manifeste répertoriés dans les messages d'erreur ultérieurs.
