# Erreur de g&#233;n&#233;ration de projet PRJ0051
Impossible de copier 'fichier' dans le répertoire cible.  Vérifiez que le fichier de destination n’est pas en cours d’utilisation.  
  
 Le système de génération n’a pas pu copier un fichier.  
  
 Vérifiez que le fichier n’est pas ouvert sur votre ordinateur ou qu’il n’est pas utilisé par un autre programme.
