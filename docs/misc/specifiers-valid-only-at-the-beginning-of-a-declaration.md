# Les sp&#233;cificateurs sont valides uniquement au d&#233;but d’une d&#233;claration
Les mots clés spécifiant l’accessibilité et d’autres désignations doivent apparaître avant le mot clé de déclaration, tel que `Dim` ou `Function`.  
  
 **ID d’erreur :** BC30181  
  
### Pour corriger cette erreur  
  
-   Déplacez tous les spécificateurs entre la liste d’attributs \(le cas échéant\) et le mot clé de déclaration.
