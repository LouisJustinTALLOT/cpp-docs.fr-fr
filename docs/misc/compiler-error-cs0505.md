# Erreur du compilateur CS0505
'membre1' : substitution impossible, car 'membre2' n’est pas une fonction  
  
 Une déclaration de classe a tenté de substituer un élément qui n’est pas une méthode dans une classe de base. Les substitutions doivent correspondre au type de membre. Si la présence d’une méthode de même nom qu’une méthode contenue dans une classe de base est souhaitée, utilisez [new](../Topic/new%20\(C%23%20Reference\).md) \(et non [override](../Topic/override%20\(C%23%20Reference\).md)\) dans la déclaration de méthode de la classe de base.  
  
 L’exemple suivant génère l’erreur CS0505 :  
  
```  
// CS0505.cs // compile with: /target:library public class clx { public int i; } public class cly : clx { public override int i() { return 0; }   // CS0505 }  
```
