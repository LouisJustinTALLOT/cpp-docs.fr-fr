# Add-in &lt;name&gt; caused an access violation. Remove it from the Add-in Manager?
Ce complément est arrêté et provoque peut\-être des problèmes dans l'environnement de développement intégré.  
  
### Pour corriger cette erreur  
  
1.  Choisissez Oui pour supprimer le complément de la liste des compléments disponibles.  
  
     ou  
  
     Choisissez Non.  
  
    > [!NOTE]
    >  Contactez le créateur du complément pour lui faire part du problème.
