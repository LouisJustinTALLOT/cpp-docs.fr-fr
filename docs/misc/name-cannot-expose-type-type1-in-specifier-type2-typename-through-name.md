# &#39;&lt;nom&gt;&#39; ne peut pas exposer le type &lt;type1&gt; dans &lt;sp&#233;cificateur&gt; &lt;type2&gt; &#39;&lt;nom_type&gt;&#39; via &lt;nom&gt;
Vous avez tenté d’exposer un type privé en dehors de la classe publique `MyClass`.  
  
 **ID d’erreur :** BC30508  
  
### Pour corriger cette erreur  
  
-   Déclarez le type `Public`.
