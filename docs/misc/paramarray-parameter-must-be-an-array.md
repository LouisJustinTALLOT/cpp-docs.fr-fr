# Le param&#232;tre ParamArray doit &#234;tre un tableau
Un argument `ParamArray` doit être déclaré comme type de tableau unidimensionnel. La déclaration n’inclut pas les parenthèses après le nom d’argument `ParamArray`.  
  
 **ID d’erreur :** BC30050  
  
### Pour corriger cette erreur  
  
-   Ajoutez une paire de parenthèses après le nom d’argument `ParamArray`
