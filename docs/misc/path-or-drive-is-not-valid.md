# Path or drive is not valid.
Cette erreur se produit généralement quand vous appuyez sur ENTRÉE alors que la zone Parcourir de la boîte de dialogue Regarder dans est vide.  
  
### Pour corriger cette erreur  
  
1.  Tapez un chemin d'accès ou un répertoire, puis appuyez sur ENTRÉE.
