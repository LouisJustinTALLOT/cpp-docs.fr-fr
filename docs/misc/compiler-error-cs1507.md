# Erreur du compilateur CS1507
Impossible de lier le fichier de ressources 'fichier' lors de la génération d’un module  
  
 [\/linkresource](../Topic/-linkresource%20\(C%23%20Compiler%20Options\).md) a été utilisé dans la même compilation que [\/target:module](../Topic/-target:module%20\(C%23%20Compiler%20Options\).md), ce qui n’est pas autorisé. Par exemple, les options suivantes génèrent l’erreur CS1507 :  
  
```  
csc /linkresource:rf.resource /target:module in.cs  
```  
  
 Cependant, l’incorporation de ressources \([\/resource](../Topic/-resource%20\(C%23%20Compiler%20Options\).md)\) est autorisée.
