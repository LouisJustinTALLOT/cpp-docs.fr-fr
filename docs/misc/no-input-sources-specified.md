# Aucune source d&#39;entr&#233;e sp&#233;cifi&#233;e
Lors de la compilation, aucune source d’entrée n’a été spécifiée.  
  
 **ID d’erreur :** BC2008  
  
### Pour corriger cette erreur  
  
-   Vérifiez que vous avez inclus un fichier de code source.
