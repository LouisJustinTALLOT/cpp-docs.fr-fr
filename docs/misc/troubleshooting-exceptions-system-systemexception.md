# D&#233;pannage des exceptions&#160;: System.SystemException
Définit la classe de base pour les exceptions prédéfinies de l'espace de noms `System`.  
  
## Notes  
 Cette classe est fournie comme un moyen de faire la différence entre des exceptions définies par le système et des exceptions définies par les applications.
