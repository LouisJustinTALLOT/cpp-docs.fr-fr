# MSBuild Error MSB3484
**MSB3484 : La cible de signature '\<fichier\>' est introuvable.**  
  
 Cette erreur est générée lorsque le fichier que vous avez essayé de signer est introuvable.  
  
### Pour corriger cette erreur  
  
-   Assurez\-vous que vous avez spécifié un fichier valide pour la signature.
