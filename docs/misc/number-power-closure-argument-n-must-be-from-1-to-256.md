# &lt;number&gt; power closure argument &#39;n&#39; must be from 1 to 256.
Cette erreur se produit généralement quand le nombre dépasse la plage admise, qui est comprise entre 1 et 256.  
  
### Pour corriger cette erreur  
  
1.  Entrez un nombre compris entre 1 et 256, puis recommencez.  Pour plus d'informations sur la syntaxe, consultez [Expressions régulières](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md).
