# Erreur du compilateur CS0026
Le mot clé 'this' n'est pas valide dans un initialiseur de propriété statique, de méthode statique ou de champ statique  
  
 Le mot clé [this](../Topic/this%20\(C%23%20Reference\).md) fait référence à un objet qui est une instance d’un type. Dans la mesure où les méthodes statiques sont indépendantes de toute instance de la classe conteneur, le mot clé « this » est sans signification et n’est donc pas autorisé. Pour plus d’informations, consultez [Classes statiques et membres de classe statique](../Topic/Static%20Classes%20and%20Static%20Class%20Members%20\(C%23%20Programming%20Guide\).md) et [Objets](../Topic/Objects%20\(C%23%20Programming%20Guide\).md).  
  
## Exemple  
 L’exemple suivant génère l’erreur CS0026 :  
  
```  
// CS0026.cs public class A { public static int i = 0; public static void Main() { // CS0026 this.i = this.i + 1; // Try the following line instead: // i = i + 1; } }  
```
