# Switch &lt;name&gt; does not allow a value.
Cette erreur se produit généralement lorsqu'une valeur est utilisée avec un commutateur qui ne nécessite pas de valeur.  
  
### Pour corriger cette erreur  
  
1.  Entrez à nouveau la commande, en omettant la valeur pour le commutateur.
