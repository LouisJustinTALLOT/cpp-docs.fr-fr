# An error occurred trying to create the application events file. &#39;&lt;error&gt;&#39;
Cette erreur se produit lorsque vous créez une application de navigateur WPF et que Visual Studio ne peut pas générer la file d'événements d'application \(Application.xaml.vb ou App.xaml.cs\).  
  
### Pour corriger cette erreur  
  
-   Enregistrez et fermez la solution.
