# Utilitaire en ligne de commande VsRegEx
L’utilitaire VsRegEx a été remplacé par le [Utilitaire de CreateExpInstance](../Topic/CreateExpInstance%20Utility.md) dans Visual Studio 2010 et versions ultérieures.  
  
## Remarques  
 Dans Visual Studio 2008, les informations de configuration sur l’instance expérimentale de l’IDE étaient stockées dans le Registre système. Dans Visual Studio 2010 et versions ultérieures, elles sont stockées dans le système de fichiers.
