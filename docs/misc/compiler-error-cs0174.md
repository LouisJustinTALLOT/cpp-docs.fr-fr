# Erreur du compilateur CS0174
Une classe de base est requise pour une référence 'base'  
  
 Cette erreur se produit uniquement quand le Common Language Runtime du .NET Framework compile le code source pour la classe `System.Object`, qui est la seule classe qui ne dispose d’aucune classe de base.
