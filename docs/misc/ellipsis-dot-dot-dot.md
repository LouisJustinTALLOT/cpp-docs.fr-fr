# Points de suspension (...)
L'opérateur ellipse \(`...`\) a de nombreux usages dans C et C\+\+, selon le contexte.  Initialement utilisé dans C comme un déclarateur abstrait pour les listes de paramètres de fonction variables, il est également utilisé dans la programmation de macro de préprocesseur sous la forme de macros variadiques.  En C\+\+, l'ellipse `...` est utilisée dans les blocs `catch` de gestion des exceptions, et en C\+\+ 11, elle est utilisée pour les modèles variadiques.
