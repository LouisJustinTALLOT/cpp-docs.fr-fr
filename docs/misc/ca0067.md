# CA0067
Répertoire introuvable.  
  
 La valeur de l'option **\/directory** dans la ligne de commande FxCopCmd est introuvable.  
  
 Cet avertissement peut se produire si vous utilisez la propriété **CodeAnalysisDependentAssemblyPaths** de [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] pour spécifier les chemins de recherche supplémentaires d'assembly de référence et un de ces chemins d'accès n'existe pas.  
  
-   Si le message d'avertissement CA0067 apparaît avec des erreurs [CA0058](../misc/ca0058.md) ou des avertissements [CA0060](../misc/ca0060.md), la résolution des autres erreurs permet généralement de résoudre CA0067.  
  
-   Si le message d'avertissement CA0067 s'affiche sans autres erreurs ou avertissements, vous pouvez normalement ignorer l'avertissement parce que le répertoire n'était pas nécessaire.
