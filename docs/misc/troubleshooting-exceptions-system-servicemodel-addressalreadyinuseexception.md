# D&#233;pannage des exceptions&#160;: System.ServiceModel.AddressAlreadyInUseException
Une exception <xref:System.ServiceModel.AddressAlreadyInUseException> est renvoyée lorsqu'une adresse n'est pas disponible car elle est déjà en cours d'utilisation.
