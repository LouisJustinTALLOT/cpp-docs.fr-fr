# D&#233;pannage des exceptions&#160;: System.Windows.Xps.XpsPackagingException
Une exception <xref:System.Windows.Xps.XpsPackagingException> est renvoyée lors de la lecture, l'écriture, l'enregistrement d'un <xref:System.Windows.Xps.Packaging.XpsDocument> ou pendant l'accès de quelque autre manière que ce soit à ce dernier.
