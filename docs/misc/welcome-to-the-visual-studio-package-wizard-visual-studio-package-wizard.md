# Bienvenue dans l’Assistant Visual Studio Integration Package, Visual Studio Integration Package
Le modèle [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] Package permet de créer une implémentation de base d’un VSPackage que vous pouvez développer plus tard. Vous pouvez également utiliser le modèle [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] Package pour ajouter des composants facultatifs à un VSPackage.  
  
 **Annuler**  
 Annule la création du VSPackage.  
  
 **Précédent**  
 Revient à l’étape précédente.  
  
 **Suivant**  
 Passe à l’étape suivante.
