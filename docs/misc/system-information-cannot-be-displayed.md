# System information cannot be displayed.
Les informations concernant le système d'exploitation ne peuvent pas être affichées, probablement parce qu'une valeur de Registre est absente ou que l'exécution de msinfo32.exe a échoué.  
  
### Pour corriger cette erreur  
  
1.  Réinstallez Visual Studio.
