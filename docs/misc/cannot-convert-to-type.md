# Impossible de convertir vers le type &#39;&lt;type&gt;&#39;
Une conversion non valide a été tentée.  
  
 **ID d’erreur :** BC30748  
  
### Pour corriger cette erreur  
  
-   Choisissez un type de conversion compatible.
