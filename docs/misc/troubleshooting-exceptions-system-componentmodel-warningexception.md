# D&#233;pannage des exceptions&#160;: System.ComponentModel.WarningException
Spécifie une exception gérée en tant qu'avertissement plutôt qu'erreur.  
  
## Notes  
 Si cette exception n'est pas interceptée, elle se présente à l'utilisateur comme un message d'avertissement.
