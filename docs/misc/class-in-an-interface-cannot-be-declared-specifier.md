# Une classe dans une interface ne peut pas &#234;tre d&#233;clar&#233;e &#39;&lt;sp&#233;cificateur&gt;&#39;
Un modificateur non valide a été spécifié avec une classe déclarée dans une interface.  
  
 **ID d’erreur :** BC31070  
  
### Pour corriger cette erreur  
  
-   Supprimez les modificateurs tels que `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend` ou `Overridable`.
