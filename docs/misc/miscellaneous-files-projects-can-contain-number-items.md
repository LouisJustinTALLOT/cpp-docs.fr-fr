# Miscellaneous Files projects can contain &lt;number&gt; items.
Le projet Fichiers divers peut contenir jusqu'à 256 éléments.  Cette erreur se produit généralement lorsqu'un nombre inférieur à 1 ou supérieur à 256 a été entré dans la zone **Le projet des fichiers divers est enregistré en dernier \<nombre\>** dans le volet Documents des options Environnement de la boîte de dialogue Options.  
  
### Pour corriger cette erreur  
  
1.  Entrez un nombre compris entre 1 et 256 dans la zone de texte.
