# Application did not register a class factory for class ID &lt;number&gt;.
Un objet COM qui implémente l'interface IClassFactory pour l'application n'a pas été inscrit.  Il s'agit d'une erreur du système d'exploitation.  
  
### Pour corriger cette erreur  
  
1.  Réinstallez Visual Studio ou l'application concernée et recommencez.
