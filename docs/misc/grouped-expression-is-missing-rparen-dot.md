# Grouped expression is missing &#39;)&#39;.
Cette erreur se produit généralement lorsqu'une parenthèse ouvrante `(` a été spécifiée, mais que la parenthèse fermante `)` a été omise dans une chaîne de recherche.  
  
### Pour corriger cette erreur  
  
1.  Pour rechercher le caractère littéral `(`, utilisez `\(`.  
  
2.  Pour grouper une expression, entrez la parenthèse `)` manquante.
