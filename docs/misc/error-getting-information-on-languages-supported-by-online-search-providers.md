# Error getting information on languages supported by online search providers.
Cette erreur se produit généralement lorsque vous essayez de sélectionner une langue dans la liste **Langue de la rubrique en ligne** située sur [Help, Environment, Options Dialog Box](http://msdn.microsoft.com/fr-fr/a2a5dff5-70a5-4f14-bf4c-66793894de6a), mais ne pouvez pas accéder aux services en ligne.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que vous disposez d'une connexion Internet active.  
  
2.  Si vous utilisez un serveur proxy, vérifiez que le serveur proxy est configuré correctement.  
  
3.  Sélectionnez à nouveau la liste déroulante **Langue de la rubrique en ligne**.
