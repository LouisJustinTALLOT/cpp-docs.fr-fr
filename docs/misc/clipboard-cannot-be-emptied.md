# Clipboard cannot be emptied.
Cette erreur se produit généralement lorsqu'un élément a été copié mais n'a pas été placé dans le Presse\-papiers car le Presse\-papiers contient déjà des informations qui ne peuvent pas être supprimées.  
  
### Pour corriger cette erreur  
  
1.  Fermez les programmes susceptibles d'utiliser le Presse\-papiers et recommencez.
