# Visual Studio requires Windows 2000 or above.
Visual Studio nécessite Windows NT 4.0, version 1381 avec Service Pack 3 ou plus, ou Windows 95, version 708 ou version ultérieure.  
  
### Pour corriger cette erreur  
  
1.  Mettez à niveau votre version actuelle de Windows NT ou de Windows vers les versions indiquées ci\-dessus.
