# L’argument &#39;&lt;nom_argument&gt;&#39; a la valeur Nothing
Une expression contient un argument null.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez l’orthographe des arguments dans l’expression. Un nom de variable mal orthographié peut créer implicitement une variable numérique initialisée à zéro.  
  
2.  Vérifiez les opérations précédentes sur les variables de l’expression, surtout celles passées dans la procédure en tant qu’arguments à partir d’autres procédures.
