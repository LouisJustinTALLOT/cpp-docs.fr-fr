# Menu captions cannot contain separators.
Cette erreur se produit généralement lorsqu'un tiret \( \- \) est utilisé dans le nom d'une légende de menu.  Le tiret est utilisé pour la séparation des menus.  
  
### Pour corriger cette erreur  
  
1.  Entrez un nom de légende qui ne contient pas de tiret.
