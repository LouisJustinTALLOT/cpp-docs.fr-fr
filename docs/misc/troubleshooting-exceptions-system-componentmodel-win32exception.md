# D&#233;pannage des exceptions&#160;: System.ComponentModel.Win32Exception
Exception levée pour un code d'erreur Win32.  
  
## Notes  
 À l'affichage, les représentations numériques des codes d'erreur Win32 sont converties en message système. Utilisez la propriété <xref:System.ComponentModel.Win32Exception.NativeErrorCode%2A> pour accéder à la représentation numérique du code d'erreur.
