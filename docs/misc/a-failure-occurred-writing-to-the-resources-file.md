# A failure occurred writing to the resources file
Le système de projet n'a pas pu écrire dans le fichier de sortie .resources.  Dans le cadre de la préparation des ressources, le système de projet transforme les fichiers XML ayant une extension .resx en fichiers .resources binaires pour le Gestionnaire de ressources .NET.  
  
 Une raison spécifique au système d'exploitation \(le cas échéant\) est ajoutée à la fin de ce message.  Le nom de fichier du fichier .resx est également ajouté.  
  
 Cette erreur entraîne l'échec du processus de génération.
