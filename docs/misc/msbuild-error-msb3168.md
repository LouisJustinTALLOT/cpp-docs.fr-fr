# MSBuild Error MSB3168
**MSB3168 : L'élément en double '\<package\>' sera ignoré.**  
  
 Cet avertissement se produit si vous avez spécifié que le programme d'amorçage doit installer deux packages de programme d'amorçage identiques.  Dans ce cas, le programme d'amorçage n'installe qu'une seule instance.  
  
### Pour corriger cette erreur  
  
-   Pour plus d'informations sur la cause d'une erreur MSBuild et les résolutions possibles, consultez [Additional Resources](../Topic/Additional%20MSBuild%20Resources.md).
