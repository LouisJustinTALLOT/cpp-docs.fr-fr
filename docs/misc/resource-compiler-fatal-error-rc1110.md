# Erreur irr&#233;cup&#233;rable RC1110 du compilateur de ressources
impossible d’ouvrir nom\_fichier  
  
 Le compilateur de ressources n’a pas pu ouvrir le fichier de script de ressources spécifié. Vérifiez que le fichier existe.
