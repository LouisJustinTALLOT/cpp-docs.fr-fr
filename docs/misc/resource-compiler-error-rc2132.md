# Erreur RC2132 du compilateur de ressources 
Mot clé VALUE, BLOCK ou END attendu  
  
 Un bloc d’une ressource **VERSION** ne se termine pas correctement ou un nouveau bloc ne commence pas correctement. En conséquence, vous obtenez une instruction **VERSION** qui n’est pas un bloc valide.
