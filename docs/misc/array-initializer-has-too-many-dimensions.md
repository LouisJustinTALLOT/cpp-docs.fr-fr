# L’initialiseur de tableau utilise trop de dimensions
Vous avez fourni trop de dimensions dans l’initialiseur de votre tableau.  
  
 **ID d’erreur :** BC30566  
  
### Pour corriger cette erreur  
  
-   Vérifiez votre initialiseur de tableau pour déterminer le nombre de dimensions nécessaire.  
  
-   Supprimez le ou les arguments en trop.
