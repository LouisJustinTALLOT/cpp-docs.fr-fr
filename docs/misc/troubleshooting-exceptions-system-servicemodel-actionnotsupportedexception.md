# D&#233;pannage des exceptions&#160;: System.ServiceModel.ActionNotSupportedException
Une exception <xref:System.ServiceModel.ActionNotSupportedException> est généralement levée sur le client lorsque l'action liée à l'opération appelée ne correspond à aucune action d'opérations sur le serveur.
