# Error getting Codezone Community sites.
Cette erreur se produit lorsqu'une connexion au service Web en ligne est introuvable.  Les services Web renseignent la page **En ligne**, sous **Aide**, **Environnement**, de la boîte de dialogue **Options**.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que vous disposez d'une connexion Internet active.  
  
     \- ou \-  
  
     Si vous vous connectez à Internet via un serveur proxy, vérifiez les paramètres du serveur proxy.  
  
2.  Essayez à nouveau d'accéder à la page En ligne.
