# VSPackages g&#233;r&#233;s
Les rubriques suivantes expliquent comment créer un VSPackage.  Un VSPackage est un module de logiciels qui étend l'environnement de développement intégré \(IDE\) de \(IDE\) [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] en fournissant des \(UI\) éléments de l'interface utilisateur, des services, des projets, des éditeurs, et les concepteurs.  Pour plus d'informations, consultez [Packages VS](../Topic/VSPackages.md).
