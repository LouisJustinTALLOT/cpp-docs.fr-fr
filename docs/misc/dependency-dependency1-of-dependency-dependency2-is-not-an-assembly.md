# La d&#233;pendance &#39;d&#233;pendance1&#39; de la d&#233;pendance &#39; d&#233;pendance2&#39; n&#39;est pas un assembly
Le système de projet a déterminé qu’une dépendance \(dépendance1\) d’un assembly \(dépendance2\) n’est pas un assembly .NET.  
  
 **Pour corriger cette erreur**  
  
-   Réinstallez [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] \(si l’assembly a été fourni avec [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] ou le .NET Framework\).  
  
     ou  
  
-   Réinstallez le contrôle tiers approprié.  
  
     Cette erreur n’empêche pas la création du projet. Toutefois, une erreur d’exécution est possible.
