# Erreur du compilateur CS0061
Accessibilité incohérente : l’interface de base 'interface 1' est moins accessible que l’interface 'interface 2'  
  
 Une construction [public](../Topic/public%20\(C%23%20Reference\).md) doit retourner un objet accessible publiquement.  
  
 L’accessibilité d’interface ne peut pas être restreinte dans une interface dérivée. Pour plus d’informations, consultez [Interfaces](../Topic/Interfaces%20\(C%23%20Programming%20Guide\).md) et [Modificateurs d'accès](../Topic/Access%20Modifiers%20\(C%23%20Programming%20Guide\).md).  
  
 L’exemple suivant génère l’erreur CS0061.  
  
```  
// CS0061.cs // compile with: /target:library internal interface A {} public interface AA : A {}  // CS0061 // OK public interface B {} internal interface BB : B {} internal interface C {} internal interface CC : C {}  
```
