# Erreur du compilateur CS0585
Erreur interne du compilateur : étape 'étape'  
  
 Essayez de déterminer si le compilateur échoue en raison de son incapacité à analyser une syntaxe inattendue. Si ce n’est pas le cas, essayez [Nous contacter](../Topic/Talk%20to%20Us.md).
