# MSBuild Error MSB3189
**MSB3189 : L'emplacement de mise à jour de cette application est un chemin d'accès local.**  
  
 Cet avertissement s'applique uniquement aux manifestes de déploiement.  
  
### Pour corriger cette erreur  
  
-   Vérifiez que l'emplacement de mise à jour est une URL valide.
