# The application cannot undo.
Cette erreur se produit généralement lorsqu'une opération d'annulation a été tentée avec un élément qui possède une annulation liée à un autre fichier.  Si une action a eu lieu depuis la modification liée, Visual Studio ne peut pas annuler la modification.  
  
### Pour corriger cette erreur  
  
1.  Annulez manuellement la modification.
