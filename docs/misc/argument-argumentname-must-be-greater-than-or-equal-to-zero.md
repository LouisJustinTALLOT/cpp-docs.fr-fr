# L’argument &lt;nom_argument&gt; doit &#234;tre sup&#233;rieur ou &#233;gal &#224; z&#233;ro
Une valeur d’argument est inférieure à zéro. Les valeurs d’argument doivent être supérieures ou égales à zéro.  
  
### Pour corriger cette erreur  
  
1.  Assurez\-vous que la valeur de l’argument est supérieure ou égale à zéro.  
  
2.  Si l’argument est fourni par un calcul, vérifiez les valeurs traitées par le calcul.
