# &#39;&lt;nom_type&gt;&#39; est un espace de noms et ne peut pas &#234;tre utilis&#233; en tant qu’expression
Un nom de type se produit là où une expression est requise. Une expression doit se composer de variables, de constantes, de littéraux, de propriétés et d’appels de procédure `Function`.  
  
 **ID d’erreur :** BC30112  
  
### Pour corriger cette erreur  
  
-   Supprimez le nom de type et construisez l’expression à l’aide d’éléments valides.
