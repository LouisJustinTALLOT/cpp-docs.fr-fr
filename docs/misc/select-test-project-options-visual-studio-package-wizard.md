# S&#233;lectionner les options de projet de test, Assistant de package Visual Studio
Cette étape vous permet de :  
  
-   Générer un projet de test d’intégration pour votre VSPackage.  
  
-   Générer un projet de test unitaire pour votre VSPackage.  
  
 **Projet de test d'intégration**  
 Crée un projet de test d’intégration. Les tests d’intégration s’exécutent dans une instance expérimentale de Visual Studio et utilisent le modèle Automation pour tester l’intégration de votre package dans Visual Studio.  
  
 **Projet de test unitaire**  
 Crée un projet de test unitaire. Les tests unitaires sont de petites instances MSTest que vous pouvez utiliser pour tester des classes et des membres dans votre package.  
  
 **Annuler**  
 Annule et arrête la nouvelle commande de projet.  
  
 **Précédent**  
 Passe à l’étape précédente.  
  
 **Terminer**  
 Crée le VSPackage.
