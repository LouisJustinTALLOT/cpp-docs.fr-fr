# Erreur RC2018 du compilateur de ressources
caractère inconnu 'nombre\_hexadécimal'  
  
 Le caractère ASCII correspondant au nombre hexadécimal donné se trouve dans le fichier source, mais il s’agit d’un caractère non conforme.  
  
 Cette erreur peut être due à l’endommagement du fichier source.
