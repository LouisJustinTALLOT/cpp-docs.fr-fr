# D&#233;pannage des exceptions&#160;: System.Data.SqlServerCe.SqlCeException
Une exception `System.Data.SqlServerCe.SqlCeException` est levée lorsque le fournisseur sous\-jacent retourne un avertissement ou une erreur à partir d'une source de données SQL Server Compact. Pour plus d’informations sur la cause de cette erreur, consultez [Erreurs SQL Server Compact Edition](http://msdn2.microsoft.com/library/ms171849.aspx) sur le site web MSDN.  
  
## Remarques  
 Un `System.Data.SqlServerCe.SqlCeException` contient toujours au moins une instance de `System.Data.SqlServerCe.SqlCeError`. La classe ne peut pas être héritée.  
  
 Pour plus d’informations, consultez [Bibliothèque de classes SQL Server Compact 3.5](http://go.microsoft.com/fwlink/?LinkID=102595).
