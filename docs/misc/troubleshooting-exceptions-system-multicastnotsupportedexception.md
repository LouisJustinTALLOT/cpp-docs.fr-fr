# D&#233;pannage des exceptions&#160;: System.MulticastNotSupportedException
Exception levée lors d'une tentative de combinaison de deux instances dont le type délégué ne peut pas être combiné. Une exception n'est pas levée si l'une des opérandes est une référence nulle \(`Nothing` en Visual Basic\).
