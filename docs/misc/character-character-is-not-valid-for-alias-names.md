# Character &lt;character&gt; is not valid for alias names.
Cette erreur se produit généralement lorsqu'un caractère incorrect a été entré dans un nom d'alias.  Les noms d'alias ne peuvent pas contenir les caractères suivants : espace blanc, \=, \[, \], \/, \- ou ".  
  
### Pour corriger cette erreur  
  
1.  Entrez à nouveau le nom d'alias en supprimant le caractère incorrect.
