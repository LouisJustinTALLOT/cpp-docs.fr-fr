# D&#233;pannage des exceptions&#160;: Microsoft.VisualStudio.Tools.Applications.Runtime.NonCacheableTypeException
Exception levée quand une tentative de mise en cache d’un type est effectuée, alors qu’il ne s’agit pas d’un type <xref:System.Data.DataSet>, <xref:System.Data.DataTable> ou d’un type qui répond aux exigences spécifiées dans [Caching Data](http://msdn.microsoft.com/fr-fr/e60dc731-2cb0-4de6-a861-0f0d131d780a).
