# MSBuild Error MSB3175
**MSB3175 : Valeur non valide pour '\<fichier\>' de l'élément '\<TâcheGénération\>'.**  
  
 Cet avertissement est généré lorsque le processus de génération ne reconnaît pas la valeur de diverses propriétés d'éléments de tâche basées sur l'énumération, telles que `AssemblyType`, `DependencyType`, `FileType` ou `TargetZone`.  S'applique aux manifestes d'application et de déploiement.
