# MSBuild Error MSB3176
**MSB3176 : La version minimale requise que vous avez spécifiée est ultérieure à celle de la version actuelle publiée.  Spécifiez une version antérieure ou égale à la version actuelle publiée.**  
  
 Cette erreur est générée lorsque la version d'application minimale requise est ultérieure à la version actuelle du manifeste de déploiement.  
  
### Pour corriger cette erreur  
  
-   Spécifiez une version minimale requise antérieure ou égale à la version actuelle publiée.
