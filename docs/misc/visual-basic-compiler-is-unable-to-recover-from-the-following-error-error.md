# Le compilateur Visual Basic n’est pas en mesure de r&#233;cup&#233;rer &#224; partir de l’erreur suivante&#160;: &lt;erreur&gt;
Le compilateur [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] n’est pas en mesure de récupérer à partir de l’erreur suivante : \<`error`\>.  
  
 Certaines fonctionnalités de l’IDE ont été désactivées.  
  
 Une erreur interne s’est produite dans le compilateur [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)].  
  
 **ID d’erreur :** BC31021  
  
### Pour corriger cette erreur  
  
1.  Enregistrez votre travail et redémarrez [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
2.  Si l'erreur se produit à nouveau, réinstallez [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
3.  Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
