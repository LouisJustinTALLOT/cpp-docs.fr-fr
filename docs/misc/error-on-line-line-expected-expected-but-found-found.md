# Error on line &lt;line&gt;. Expected &#39;expected&#39; but found &#39;found&#39;
Il s'agit d'une erreur d'analyse XML générale lors de la lecture du fichier projet.  Cette erreur peut se produire pour les fichiers projet \(.vbproj\/.csproj\), ainsi que pour les fichiers de paramètres utilisateur \(.vbproj.user\/.csproj.user\).  Le système de projet fournit davantage d'informations dans les paramètres `expected` et `found`.  
  
 La cause la plus probable de cette erreur est la modification manuelle du fichier projet \(ou utilisateur\).  
  
 **Pour corriger cette erreur**  
  
-   Modifiez de nouveau le fichier projet et remettez\-le en état.  
  
     Selon la gravité de l'erreur, vous pouvez être dans l'incapacité d'ouvrir le projet.
