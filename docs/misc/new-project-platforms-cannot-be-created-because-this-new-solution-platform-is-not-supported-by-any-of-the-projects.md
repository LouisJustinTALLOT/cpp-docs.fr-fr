# New project platforms cannot be created because this new solution platform is not supported by any of the projects.
Cette erreur se produit lorsque vous sélectionnez **Créer des plateformes de projet**, mais qu'aucun des projets ne prend en charge la nouvelle plateforme de solution.  
  
### Pour corriger cette erreur  
  
-   Désélectionnez **Créer des plateformes de projet**, puis cliquez sur **OK**.  
  
     \- ou \-  
  
     Désélectionnez **Créer des plateformes de projet** et sélectionnez un autre nom de plateforme dans la liste déroulante, puis cliquez sur **OK**.
