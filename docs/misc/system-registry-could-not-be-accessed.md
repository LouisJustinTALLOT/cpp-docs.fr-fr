# System registry could not be accessed.
Cette erreur se produit généralement lorsque le système d'exploitation ne trouve pas le fichier du Registre, que ce fichier est endommagé ou qu'une tentative de lecture ou d'écriture dans ce fichier a échoué.  
  
### Pour corriger cette erreur  
  
1.  Réinstallez Visual Studio.
