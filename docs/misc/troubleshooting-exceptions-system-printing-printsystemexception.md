# D&#233;pannage des exceptions&#160;: System.Printing.PrintSystemException
Une exception <xref:System.Printing.PrintSystemException> se produit lorsqu'une condition d'erreur empêche l'accès ou la création d'un <xref:System.Printing.PrintSystemObject>.
