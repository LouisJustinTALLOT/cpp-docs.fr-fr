# A failure occurred while attempting to start the compilation
Une erreur inconnue a été rencontrée en émettant la requête **Démarrer la build** au compilateur [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] ou [!INCLUDE[csprcs](../ide/includes/csprcs_md.md)].  
  
 **Pour corriger cette erreur**  
  
-   Redémarrez [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
     Cette erreur entraîne l'échec du processus de génération.
