# &lt;name&gt; is not valid editor for this file type.
Cette erreur se produit généralement lorsqu'une valeur incorrecte a été entrée pour l'argument \/e:*editorname* de la fenêtre **Commande** ou de la zone de texte **Rechercher\/Commande**.  
  
### Pour corriger cette erreur  
  
1.  Consultez la documentation pour obtenir la syntaxe correcte pour \/e:*editorname* et entrez à nouveau la commande.
