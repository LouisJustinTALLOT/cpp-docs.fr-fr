# D&#233;pannage des exceptions&#160;: System.Net.Mail.SmtpFailedRecipientsException
Exception levée lorsqu'un courrier électronique, envoyé à l'aide d'un <xref:System.Net.Mail.SmtpClient>, ne peut pas être remis à tous les destinataires.
