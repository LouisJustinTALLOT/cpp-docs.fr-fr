# D&#233;pannage des exceptions&#160;: System.Reflection.CustomAttributeFormatException
Exception levée lorsque le format binaire d'un attribut personnalisé est non valide.  
  
## Notes  
 Ce peut être un résultat d'un bogue dans un outil utilisé pour créer le type de données qui lève l'erreur.
