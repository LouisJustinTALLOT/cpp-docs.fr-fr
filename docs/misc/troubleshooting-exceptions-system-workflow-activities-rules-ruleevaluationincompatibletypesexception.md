# D&#233;pannage des exceptions&#160;: System.Workflow.Activities.Rules.RuleEvaluationIncompatibleTypesException
Une <xref:System.Workflow.Activities.Rules.RuleEvaluationIncompatibleTypesException> est renvoyée lorsqu'une tentative est faite pour appliquer un opérateur aux opérandes incompatibles lors de l'évaluation d'une règle.
