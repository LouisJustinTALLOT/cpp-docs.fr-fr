# D&#233;pannage des exceptions&#160;: System.ServiceModel.Persistence.InstanceLockException
L'exception <xref:System.ServiceModel.Persistence.InstanceLockException> est prévue pour être utilisée par les classes qui héritent de <xref:System.ServiceModel.Persistence.LockingPersistenceProvider> lorsque l'opération ne peut pas être exécutée en raison de l'état du verrou d'instance.
