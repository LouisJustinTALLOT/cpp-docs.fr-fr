# Les instructions de d&#233;claration de m&#233;thode doivent figurer en premier sur une ligne logique.
Une instruction `Function` ou `Sub` suit un séparateur d’instruction deux\-points \(:\).`Function` ou `Sub` doit être la seule instruction sur sa ligne source.  
  
 **ID d’erreur :** BC32009  
  
### Pour corriger cette erreur  
  
-   Séparez les multiples instructions en les plaçant sur des lignes différentes.
