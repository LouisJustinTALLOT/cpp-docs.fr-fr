# D&#233;pannage des exceptions&#160;: Microsoft.VisualStudio.Tools.Applications.Runtime.ItemAlreadyInCacheException
Exception levée lorsqu'une tentative est faite pour ajouter un élément au cache du document, mais que l'élément existe déjà dans le cache.
