# D&#233;pannage des exceptions&#160;: System.Printing.PrintCommitAttributesException
Une exception <xref:System.Printing.PrintCommitAttributesException> est levée lorsqu'une condition d'erreur empêche la validation d'attributs par un <xref:System.Printing.PrintSystemObject> sur l'ordinateur, l'imprimante ou le périphérique réel représenté par l'objet.
