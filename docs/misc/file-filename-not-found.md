# Fichier &#39;&lt; nom_fichier&gt;&#39; introuvable
Un nom de fichier est incorrect ou le fichier n’existe pas.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que le nom de fichier est correct et qu’il existe.
