# D&#233;pannage des exceptions&#160;: System.Net.NetworkInformation.PingException
Exception levée lorsqu'une méthode <xref:System.Net.NetworkInformation.Ping.Send%2A> ou <xref:System.Net.NetworkInformation.Ping.SendAsync%2A> appelle une méthode levant une exception. C'est le cas, par exemple, lorsqu'un serveur ne prend pas ping en charge ou un pare\-feu bloque l'opération.
