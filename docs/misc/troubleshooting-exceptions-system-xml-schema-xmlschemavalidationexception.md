# D&#233;pannage des exceptions&#160;: System.Xml.Schema.XmlSchemaValidationException
Exception levée lorsque des erreurs et des avertissements de validation du langage XSD \(XML Schema Definition\) sont trouvés dans un document XML en cours de validation.
