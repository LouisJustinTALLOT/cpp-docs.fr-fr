# Erreur du compilateur CS0236
Un initialiseur de champ ne peut pas faire référence au champ, à la méthode ou à la propriété non statique 'champ'  
  
 Les champs d’instance ne peuvent pas être utilisés pour initialiser d’autres champs d’instance en dehors d’une méthode. Si vous tentez d’initialiser une variable en dehors d’une méthode, envisagez d’effectuer l’initialisation dans le constructeur de classe. Pour plus d'informations, consultez [Méthodes](../Topic/Methods%20\(C%23%20Programming%20Guide\).md).  
  
 L’exemple suivant génère l’erreur CS0236 :  
  
```  
// CS0236.cs public class MyClass { public int i = 5; public int j = i;  // CS0236 public int k;      // initialize in constructor MyClass() { k = i; } public static void Main() { } }  
```
