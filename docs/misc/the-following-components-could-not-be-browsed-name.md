# The following components could not be browsed: &lt;name&gt;
Cette erreur se produit généralement lorsque vous essayez d'ajouter un type de fichier qui ne prend pas en charge l'exploration d'objets dans la boîte de dialogue du Sélecteur de composants, tel que les fichiers .htm et .txt.  L'Explorateur d'objets prend en charge les types de fichiers .bsc, .olb, .tlb, .dll, .exe et .ocx.  
  
### Pour corriger cette erreur  
  
1.  Supprimez le type de fichier non pris en charge de la liste des composants sélectionnés et sélectionnez un autre fichier.
