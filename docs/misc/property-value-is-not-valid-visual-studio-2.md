# Property value is not valid. (Visual Studio) (2)
Cette erreur se produit généralement lorsqu'une valeur non valide est entrée pour une propriété.  Ceci peut être le cas si vous spécifiez une valeur entière au format incorrect, ou une valeur trop grande pour un entier court.  
  
### Pour corriger cette erreur  
  
1.  Entrez une valeur valide pour la propriété.  
  
    > [!TIP]
    >  Recherchez dans la documentation en ligne les valeurs valides pour la propriété.
