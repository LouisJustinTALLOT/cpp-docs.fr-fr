# D&#233;pannage des exceptions&#160;: System.Data.InvalidExpressionException
Une exception <xref:System.Data.InvalidExpressionException> est levée lorsque vous tentez d'ajouter à <xref:System.Data.DataColumn> un <xref:System.Data.DataColumn.Expression%2A> contenant un <xref:System.Data.DataColumnCollection> non valide.  
  
## Conseils associés  
 **Vérifiez la syntaxe de votre expression.**  
 La propriété <xref:System.Data.DataColumn.Expression%2A> permet de calculer la valeur d'une colonne ou de créer une colonne de regroupement.
