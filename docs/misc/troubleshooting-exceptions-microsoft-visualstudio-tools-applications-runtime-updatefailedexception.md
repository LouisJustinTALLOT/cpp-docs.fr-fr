# D&#233;pannage des exceptions&#160;: Microsoft.VisualStudio.Tools.Applications.Runtime.UpdateFailedException
Exception levée lorsqu'un manifeste de déploiement fait référence à un manifeste d'application dont la version est différente de celle du manifeste d'application du document, mais que le manifeste d'application du document ne peut pas être mis à jour.
