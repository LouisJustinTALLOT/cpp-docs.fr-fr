# L’argument &#39;&lt;nom_argument&gt;&#39; doit &#234;tre compris entre -32&#160;768 et 65&#160;535.
La valeur d’un argument n’est pas comprise dans la plage autorisée de \-32 768 \- 65 535.  
  
### Pour corriger cette erreur  
  
1.  Assurez\-vous que la valeur de l’argument est comprise dans la plage valide.  
  
2.  Si l’argument est fourni par un calcul, vérifiez les valeurs traitées lors du calcul.
