# Top-level menu names cannot contain separators.
Cette erreur se produit généralement lors d'une tentative de glisser\-déplacer de texte contenant un caractère de séparation tel qu'un tiret \(\-\) dans un nom de menu.  Les menus de niveau supérieur ne peuvent pas contenir ce caractère, car il est réservé à la séparation des ensembles d'éléments de menu.  
  
### Pour corriger cette erreur  
  
1.  Entrez le texte sans utiliser le caractère de séparation.
