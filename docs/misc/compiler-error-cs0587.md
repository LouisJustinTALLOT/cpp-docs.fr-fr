# Erreur du compilateur CS0587
Erreur interne du compilateur : étape 'stage'  
  
 Essayez de déterminer si le compilateur échoue en raison de son incapacité à analyser une syntaxe inattendue. Si ce n’est pas le cas, essayez de [Nous contacter](../Topic/Talk%20to%20Us.md).
