# The project file is missing the &#39;subsection&#39; sub-section of the &#39;section&#39; section
Le paramètre SubSection \(sous\-section\) correspond à Include et section \(section\) correspond à Files.  
  
 La cause la plus probable de cette erreur est la modification manuelle du fichier projet.  
  
 **Pour corriger cette erreur**  
  
-   Activez le mode **Afficher tous les fichiers** \(en cliquant sur le bouton correspondant dans la barre d'outils de l'Explorateur de solutions\) et ajoutez les fichiers appropriés.  
  
     Si vous obtenez cette erreur, le projet s'ouvrira, mais aucun fichier ne sera ajouté au projet.
