# Erreur du compilateur CS2012
Impossible d’ouvrir 'file' en écriture  
  
 Quand vous utilisez l’option du compilateur [\/bugreport:](../Topic/-bugreport%20\(C%23%20Compiler%20Options\).md)`file`, le fichier ne s’ouvre pas en écriture. Assurez\-vous d’avoir spécifié un nom de fichier valide et que ce fichier n’est pas en lecture seule.
