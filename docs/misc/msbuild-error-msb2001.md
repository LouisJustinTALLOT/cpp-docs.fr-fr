# MSBuild Error MSB2001
**L'élément \<{0}\> ne contient pas l'attribut requis "{1}".**  
  
 Le projet converti a été modifié, endommagé, ou n'a pas été créé par [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
### Pour corriger cette erreur  
  
-   Vérifiez si le fichier projet a été modifié ou endommagé.  S'il a été modifié ou endommagé, ouvrez le projet dans la version de [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] avec laquelle il a été créé, enregistrez\-le, puis essayez à nouveau de le convertir.
