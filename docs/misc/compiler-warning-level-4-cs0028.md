# Avertissement du compilateur (niveau&#160;4) CS0028
'déclaration\_fonction' possède une signature erronée pour un point d’entrée  
  
 La déclaration de méthode pour `Main` n’est pas valide : elle a été déclarée avec une signature non valide.`Main` doit être déclarée comme static et doit retourner [int](../Topic/int%20\(C%23%20Reference\).md) ou [void](../Topic/void%20\(C%23%20Reference\).md). Pour plus d'informations, consultez [Main\(\) et arguments de ligne de commande](../Topic/Main\(\)%20and%20Command-Line%20Arguments%20\(C%23%20Programming%20Guide\).md).  
  
 L’exemple suivant génère l’erreur CS0028 :  
  
```  
// CS0028.cs // compile with: /W:4 /warnaserror public class a { public static double Main(int i)   // CS0028 // Try the following line instead: // public static void Main() { } }  
```
