# Le membre &#39;&lt;nom_membre&gt;&#39; est introuvable dans la classe &#39;&lt;nom_classe&gt;&#39;
Le membre '\<nom\_membre\>' est introuvable dans la classe '\<nom\_classe\>'. Cette condition est généralement le résultat d'un 'Microsoft.VisualBasic.dll' incompatible.  
  
 Un membre défini est introuvable.  
  
 **ID d’erreur :** BC31097  
  
### Pour corriger cette erreur  
  
1.  Recompilez le programme pour voir si l'erreur se produit à nouveau.  
  
2.  Le cas échéant, enregistrez votre travail et redémarrez [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
3.  Si l'erreur se produit à nouveau, réinstallez [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)].  
  
4.  Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
