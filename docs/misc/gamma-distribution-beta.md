# gamma_distribution::beta
Retourne le paramètre de distribution bêta.  
  
## Syntaxe  
  
```  
RealType beta() const;  
```  
  
## Notes  
 Cette fonction membre retourne la valeur stockée `beta`.

 **En\-tête :** \<random\>  
  
 **Espace de noms :** std
