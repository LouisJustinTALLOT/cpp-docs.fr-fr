# CA0502
Langage non pris en charge  
  
 Cette erreur se produit lorsque l'utilisateur tente d'analyser un projet ASP.NET qui contient une ou plusieurs pages ou classes écrites dans un langage autre que C\# ou Visual Basic.  Pour corriger cette erreur, supprimez le fichier du projet ou désactivez l'analyse de code.
