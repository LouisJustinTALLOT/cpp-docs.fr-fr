# Wizard file &lt;name&gt; format is not recognized.
Impossible d'ouvrir l'Assistant.  Cette erreur est généralement provoquée par un fichier d'Assistant endommagé ou un Assistant qui ne correspond pas au numéro de version de Visual Studio.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que le fichier de l'Assistant possède le même numéro de version que Visual Studio dans VSWIZARD, par exemple `VSWIZARD 7.0`.  
  
     \- ou \-  
  
     Réinstallez Visual Studio pour réparer le fichier endommagé.
