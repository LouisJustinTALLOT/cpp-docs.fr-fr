# D&#233;pannage des exceptions&#160;: System.IdentityModel.Selectors.UntrustedRecipientException
Une exception <xref:System.IdentityModel.Selectors.UntrustedRecipientException> est levée lorsqu'un utilisateur passe en revue les informations d'une entité qui demande un jeton, et décide de ne pas approuver celle\-ci. L'utilisateur clique sur **Désactiver Windows CardSpace** dans l'interface utilisateur CardSpace.
