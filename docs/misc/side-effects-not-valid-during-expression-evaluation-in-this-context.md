# Effets secondaires non valides pendant l’&#233;valuation de l’expression dans ce contexte
Les instructions qui modifient l’état d’un objet ne sont pas autorisées dans ce contexte.  
  
 **ID d’erreur :** BC30700  
  
### Pour corriger cette erreur  
  
-   Modifiez votre code afin que les instructions soient évaluées sans modifier l’état d’autres éléments.
