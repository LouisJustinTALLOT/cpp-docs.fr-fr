# Erreur du compilateur CS0205
Impossible d’appeler un membre de base abstrait : 'méthode'  
  
 Vous ne pouvez pas appeler une méthode [abstraite](../Topic/abstract%20\(C%23%20Reference\).md), car elle n’a pas de corps de méthode. Pour plus d'informations, consultez [Classes abstract et sealed et membres de classe](../Topic/Abstract%20and%20Sealed%20Classes%20and%20Class%20Members%20\(C%23%20Programming%20Guide\).md).  
  
 L’exemple suivant génère l’erreur CS0205 :  
  
```  
// CS0205.cs abstract public class MyClass { abstract public void M(); } public class MyClass2 : MyClass { public override void M() { base.M();   // CS0205, delete this line } public static void Main() { } }  
```
