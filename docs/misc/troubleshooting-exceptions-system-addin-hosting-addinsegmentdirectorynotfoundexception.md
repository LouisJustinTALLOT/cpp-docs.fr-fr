# D&#233;pannage des exceptions&#160;: System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException
L'exception <xref:System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException> est renvoyée lorsqu'un répertoire de segment est manquant dans la structure de répertoires du pipeline.  
  
## Notes  
 Lorsqu'un complément est activé, son pipeline vers l'hôte est construit. Le système de complément nécessite que les segments de pipeline soient placés dans une structure de répertoires spécifiée avec les noms de répertoire indiqués, comme décrit dans [Spécifications du développement de pipelines](http://msdn.microsoft.com/fr-fr/ef9fa986-e80b-43e1-868b-247f4c1d9da5).
