# &#39;Shared&#39; ne peut pas &#234;tre combin&#233; avec &#39;&lt;sp&#233;cificateur&gt;&#39; pour une d&#233;claration de propri&#233;t&#233;
Vous avez tenté d’associer le modificateur `Shared` à un spécificateur qui n’est pas valide dans une telle combinaison pour une déclaration de propriété.  
  
 **ID d’erreur :** BC30502  
  
### Pour corriger cette erreur  
  
-   Supprimez le spécificateur.
