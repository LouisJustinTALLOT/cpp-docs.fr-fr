# An invalid cookie was passed to UnregisterExtenderProvider().
Vous avez spécifié un cookie qui n'est plus valide.  
  
### Pour corriger cette erreur  
  
1.  Retournez la même valeur de cookie que celle que vous avez obtenue en appelant RegisterExtenderProvider\(\).
