# Failed to generate resource files
Une erreur générale s'est produite au niveau du système de projet pour lire un fichier .resx dans le cadre du processus de création de fichiers .resources binaires que le Gestionnaire de ressources .NET peut comprendre.  
  
 Cette erreur est généralement causée par un fichier .resx incorrect.  
  
 Cet avertissement n'entraîne pas l'échec de la génération.  Le fichier .resx pour lequel cette erreur est affichée ne sera toutefois pas inclus en tant que ressource de manifeste d'assembly dans le projet.
