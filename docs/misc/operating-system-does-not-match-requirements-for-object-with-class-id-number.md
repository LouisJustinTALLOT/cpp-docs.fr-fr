# Operating system does not match requirements for object with class ID &lt;number&gt;.
Cette erreur se produit généralement lorsqu'un système d'exploitation incompatible ou une version de système d'exploitation incompatible ont été détectés.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez la configuration requise pour l'application.  
  
2.  Installez un système d'exploitation compatible et recommencez.
