# Erreur RC2022 du compilateur de ressources
'nombre' : trop grand pour un caractère  
  
 Le nombre octal situé après une barre oblique inverse \(**\\**\) dans une constante caractère ou chaîne était trop grand pour être représenté sous forme de caractère.
