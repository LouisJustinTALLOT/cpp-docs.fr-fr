# D&#233;pannage des exceptions&#160;: System.Web.Security.MembershipCreateUserException
Exception levée lorsqu'un fournisseur d'appartenances ne parvient pas à créer un utilisateur.  
  
## Notes  
 La propriété <xref:System.Web.Security.MembershipCreateUserException.StatusCode%2A> indique la raison de l'exception.
