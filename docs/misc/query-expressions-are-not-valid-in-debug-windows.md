# Les expressions de requ&#234;te ne sont pas valides dans les fen&#234;tres de d&#233;bogage
Vous avez entré une requête LINQ dans la fenêtre **Espion** ou **Exécution** pendant une session de débogage. Les requêtes ne sont pas disponibles dans les fenêtres de débogage.  
  
 **ID d’erreur :** BC36609
