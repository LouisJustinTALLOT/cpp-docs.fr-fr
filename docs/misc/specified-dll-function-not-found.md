# Fonction DLL sp&#233;cifi&#233;e introuvable
La bibliothèque de liens dynamiques \(DLL\) d’une référence de bibliothèque utilisateur a été trouvée, mais la fonction DLL spécifiée est introuvable dans la DLL.  
  
### Pour corriger cette erreur  
  
1.  Spécifiez un ordinal valide dans la déclaration de la fonction.  
  
2.  Vérifiez que le nom de la DLL et l’alias sont corrects.
