# D&#233;pannage des exceptions&#160;: System.IO.PathTooLongException
Une exception <xref:System.IO.PathTooLongException> est levée lorsqu'un chemin d'accès ou un nom de fichier dépasse la longueur maximale définie par le système.  
  
## Conseils associés  
 **Assurez\-vous que le chemin d'accès n'est pas plus long que la valeur maximale définie par le système.**  
 Sur les plateformes Windows, les chemins d'accès et les noms de fichiers ne doivent pas dépasser respectivement 248 et 260 caractères.
