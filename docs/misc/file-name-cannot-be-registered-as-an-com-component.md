# File &lt;name&gt; cannot be registered as an COM component.
Le fichier ne peut pas être ajouté aux références du projet.  Cette erreur se produit lorsqu'un fichier est endommagé, ne contient pas de bibliothèques de types, utilise un type de fichier incompatible, ou que le système ne possède pas assez de ressources disponibles pour terminer l'opération.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que le fichier contient une bibliothèque de types valide et recommencez.  
  
     \- ou \-  
  
     Réinstallez le produit qui a installé le fichier.  
  
    > [!NOTE]
    >  Le produit qui a installé le fichier peut ne pas être Visual Studio.
