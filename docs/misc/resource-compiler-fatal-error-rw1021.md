# Erreur irr&#233;cup&#233;rable RW1021 du compilateur de ressources
Erreur d’E\/S lors de la lecture du fichier  
  
 Le compilateur de ressources n’a pas pu lire un fichier. Vérifiez que le lecteur contenant le fichier est accessible et que le fichier est valide.
