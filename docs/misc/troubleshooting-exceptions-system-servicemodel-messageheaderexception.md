# D&#233;pannage des exceptions&#160;: System.ServiceModel.MessageHeaderException
Une exception <xref:System.ServiceModel.MessageHeaderException> est renvoyée lorsque les attentes concernant les en\-têtes d'un message SOAP ne sont pas satisfaites pendant le traitement du message.
