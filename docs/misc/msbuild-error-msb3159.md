# MSBuild Error MSB3159
**MSB3159 : Erreur de validation Xml dans le fichier '\<fichier\>' : '\<erreur\>'**  
  
 Cette erreur est générée lorsque le produit ou le fichier package.xml n'est pas conforme au schéma XML.  La dernière partie de cette erreur est un message d'erreur spécifique.  
  
### Pour corriger cette erreur  
  
-   Mettez à jour le fichier XML pour résoudre l'erreur de validation.
