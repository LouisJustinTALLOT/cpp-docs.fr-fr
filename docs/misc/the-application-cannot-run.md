# The application cannot run.
Visual Studio n'a pas suffisamment de mémoire pour s'exécuter.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que l'ordinateur satisfait à la configuration minimale requise pour exécuter Visual Studio.  
  
2.  Fermez d'autres programmes exécutés sur l'ordinateur pour libérer de la mémoire.  
  
     \- ou \-  
  
     Réinstallez Visual Studio.
