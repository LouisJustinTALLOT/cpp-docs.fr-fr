# Erreur RC2014 du compilateur de ressources
une commande de préprocesseur doit commencer comme premier espace autre que blanc  
  
 Des caractères autres que des espaces blancs ont été détectés avant le symbole dièse \(**\#**\) d’une directive de préprocesseur sur la même ligne.
