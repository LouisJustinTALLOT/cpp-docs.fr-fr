# This program requires an application for class ID &lt;number&gt;.
Visual Studio n'a pas pu trouver l'application que vous essayez d'utiliser.  Ce message indique une erreur du système d'exploitation.  
  
### Pour corriger cette erreur  
  
1.  Réinstallez l'application concernée et recommencez.  
  
     \- ou \-  
  
     Réinstallez Visual Studio et réessayez.
