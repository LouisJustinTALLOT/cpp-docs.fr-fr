# Look In list must contain valid files or folders, separated by semicolons.
Cette erreur se produit généralement quand le répertoire ou les fichiers répertoriés dans la liste **Regarder dans** contiennent des caractères qui ne sont pas valides.  
  
### Pour corriger cette erreur  
  
1.  Utilisez la boîte de dialogue **Regarder dans** pour rechercher le répertoire ou le nom de fichier qui vous intéresse au lieu de taper le chemin ou le nom de fichier correspondant.  Pour accéder à la boîte de dialogue **Regarder dans**, choisissez le bouton de sélection \(...\) situé en regard de la liste **Regarder dans**.
