# Window menu can contain &lt;number&gt; items.
La liste des fenêtres du menu Fenêtre ne peut afficher que jusqu'à vingt\-quatre noms de fenêtres.  Cette erreur se produit généralement lorsqu'un nombre inférieur à 1 ou supérieur à 24 a été entré dans la zone de texte **Le menu Fenêtre contient \<nombre\> éléments**, dans le volet Général des options Environnement de la boîte de dialogue Options.  
  
### Pour corriger cette erreur  
  
1.  Entrez un nombre compris entre 1 et 24 dans la zone de texte.
