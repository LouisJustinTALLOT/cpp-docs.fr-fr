# Erreur RC2146 du compilateur de ressources
VIRGULE manquante dans l’instruction LANGAGE  
  
 La virgule séparant la langue principale et la langue secondaire est manquante.  
  
 L’instruction **LANGUAGE** doit utiliser la syntaxe suivante :  
  
 **LANGUAGE** *ID\_langue\_principale*,*ID\_langue\_secondaire*
