# &#39;&lt;nom&gt;&#39; est ambigu entre les d&#233;clarations dans les modules &#39;&lt;nom_module1&gt;&#39; et &#39;&lt;nom_module2&gt;&#39;
Vous avez fourni un nom ambigu qui entre par conséquent en conflit avec un autre nom. Le compilateur [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] ne possède aucune règle de résolution de conflit ; vous devez lever vous\-même toute ambiguïté sur les noms.  
  
 **ID d’erreur :** BC30562  
  
### Pour corriger cette erreur  
  
-   Utilisez un nom qualifié complet.
