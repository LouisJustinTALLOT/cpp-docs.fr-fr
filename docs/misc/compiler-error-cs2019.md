# Erreur du compilateur CS2019
Type de cible non valide pour \/target : vous devez spécifier 'exe', 'winexe', 'library' ou 'module'  
  
 L’option [\/target](../Topic/-target%20\(C%23%20Compiler%20Options\).md) du compilateur a été utilisée, mais un paramètre non valide a été passé. Pour résoudre cette erreur, recompilez le programme en utilisant la forme de l’option **\/target** appropriée à votre fichier de sortie.  
  
 L’exemple suivant génère l’erreur CS2017 :  
  
```  
// CS2019.cs // compile with: /target:libra // CS2019 expected class MyClass { }  
```
