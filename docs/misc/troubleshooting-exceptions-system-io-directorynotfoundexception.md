# D&#233;pannage des exceptions&#160;: System.IO.DirectoryNotFoundException
Une exception <xref:System.IO.DirectoryNotFoundException> est levée lorsqu'une partie du chemin d'accès à un fichier ou d'un répertoire est introuvable.  
  
## Conseils associés  
 **Vérifiez que le répertoire apparaît dans l'emplacement spécifié.**  
 Vérifiez que le répertoire existe dans l'emplacement spécifié.  
  
 **Lorsque vous utilisez des chemins d'accès relatifs, assurez\-vous que le répertoire actif est correct.**  
 Le chemin d'accès peut être incorrect si vous utilisez un répertoire actif incorrect.
