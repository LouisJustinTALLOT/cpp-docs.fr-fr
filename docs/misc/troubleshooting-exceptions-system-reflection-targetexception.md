# D&#233;pannage des exceptions&#160;: System.Reflection.TargetException
Exception levée lors d'une tentative d'appel à une cible non valide.  
  
## Notes  
 Une <xref:System.Reflection.TargetException> est levée lors d'une tentative d'appel à une méthode non statique sur un objet nul. Les raisons possibles sont que l'appelant n'a pas accès au membre, que la cible ne définit pas le membre, ou d'autres circonstances semblables.
