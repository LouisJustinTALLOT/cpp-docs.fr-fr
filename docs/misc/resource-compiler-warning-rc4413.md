# Avertissement RC4413 du compilateur de ressources
impossible de déterminer le type de fichier pour 'nomfichier' : ASCII 8 bits pris par défaut  
  
 Le compilateur de ressources ne connaît pas le type de fichier spécifié. Il a été compilé en tant que fichier ASCII 8 bits.
