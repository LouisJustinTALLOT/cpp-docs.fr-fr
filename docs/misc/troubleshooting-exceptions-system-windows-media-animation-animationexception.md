# D&#233;pannage des exceptions&#160;: System.Windows.Media.Animation.AnimationException
Une exception <xref:System.Windows.Media.Animation.AnimationException> est levée lorsqu'une erreur se produit pendant l'animation d'une propriété.  
  
## Notes  
 Cette classe n'est pas utilisée en cas d'erreurs de temporisation, telles que la création d'une animation avec une <xref:System.Windows.Media.Animation.Timeline.Duration%2A> négative ou avec des paramètres <xref:System.Windows.Media.Animation.Timeline.AccelerationRatio%2A> et <xref:System.Windows.Media.Animation.Timeline.DecelerationRatio%2A> non valides.
