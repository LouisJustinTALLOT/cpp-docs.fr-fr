# Tagged expression is missing &#39;}&#39;.
Cette erreur se produit généralement lorsqu'une accolade ouvrante `{` a été spécifiée, mais que l'accolade fermante `}` a été omise dans une chaîne de recherche.  
  
### Pour corriger cette erreur  
  
1.  Pour rechercher le caractère littéral `{`, utilisez `\{`.  
  
2.  Pour délimiter l'expression, entrez l'accolade `}` manquante.
