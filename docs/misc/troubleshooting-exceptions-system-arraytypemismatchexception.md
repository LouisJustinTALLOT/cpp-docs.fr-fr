# D&#233;pannage des exceptions&#160;: System.ArrayTypeMismatchException
Une exception <xref:System.ArrayTypeMismatchException> est levée lorsque le système ne parvient pas à convertir l'élément en type déclaré pour le tableau.  
  
## Conseils associés  
 **Vérifiez que le type d'objet est convertible type de tableau.**  
 Une conversion doit être possible entre les deux types.  
  
 Pour plus d’informations, les utilisateurs de Visual Basic peuvent consulter [Tableaux](../Topic/Arrays%20in%20Visual%20Basic.md).
