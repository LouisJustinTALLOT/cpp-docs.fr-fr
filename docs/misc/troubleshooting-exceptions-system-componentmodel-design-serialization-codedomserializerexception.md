# D&#233;pannage des exceptions&#160;: System.ComponentModel.Design.Serialization.CodeDomSerializerException
Exception levée lorsque le sérialiseur CodeDOM rencontre une erreur.  
  
## Notes  
 Cette exception autorise le sérialiseur à retourner des informations sur l'emplacement et le type de l’erreur.
