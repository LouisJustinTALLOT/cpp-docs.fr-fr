# Syntax error in pattern.
Cette erreur se produit généralement quand une chaîne recherchée comprend deux ou plusieurs expressions régulières mais que l'une au moins de ces expressions régulières contient une erreur de syntaxe.  
  
### Pour corriger cette erreur  
  
1.  Entrez à nouveau la chaîne.  Pour plus d'informations sur la syntaxe, consultez [Expressions régulières](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md).
