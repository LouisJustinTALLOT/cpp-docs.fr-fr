# Erreur du compilateur&#160;CS1013
Nombre non valide  
  
 Le compilateur a détecté un nombre dont le format est incorrect. Pour plus d’informations sur les types intégraux, consultez [Tableau des types intégraux](../Topic/Integral%20Types%20Table%20\(C%23%20Reference\).md).  
  
## Exemple  
 L’exemple suivant génère l’erreur CS1013 :  
  
```  
// CS1013.cs class Sample { static void Main() { int i = 0x;   // CS1013 } }  
```
