# D&#233;pannage des exceptions&#160;: Microsoft.Office.Tools.ControlNameAlreadyExistsException
Une exception <xref:Microsoft.Office.Tools.ControlNameAlreadyExistsException> est levée lorsque vous essayez de créer par programme un contrôle portant le même nom qu'un autre contrôle de votre projet.  
  
## Conseils associés  
 **Utilisez un nom différent pour l'un des contrôles.**  
 Les noms de contrôles doivent être uniques.
