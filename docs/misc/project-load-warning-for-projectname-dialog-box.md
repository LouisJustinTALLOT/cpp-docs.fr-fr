# Avertissement sur le chargement du projet &lt;nom_projet&gt;, bo&#238;te de dialogue
Cette boîte de dialogue apparaîtra si les paramètres, pour les projets ouverts dans l'IDE, sont chargés incorrectement.  
  
 **Veuillez corriger les erreurs spécifiées et recharger le projet**  
 Fournit des informations sur la complication comme le chemin d'accès ou le nom de fichier non valide.  Pour résoudre les erreurs, supprimez ou remplacez toutes références non valides des pages de propriétés de votre projet.  Enregistrez votre projet, puis rechargez\-le dans l'IDE.  
  
 **Ne plus afficher cette boîte de dialogue pendant cette session**  
 Lorsque sélectionné, l'avertissement de charge de projet pour la boîte de dialogue de \<Projectname\> n'apparaît plus jusqu'à ce que la prochaine ouverture de l'IDE.
