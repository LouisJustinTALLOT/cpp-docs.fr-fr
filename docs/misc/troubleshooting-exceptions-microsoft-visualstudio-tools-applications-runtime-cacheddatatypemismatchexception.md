# D&#233;pannage des exceptions&#160;: Microsoft.VisualStudio.Tools.Applications.Runtime.CachedDataTypeMismatchException
Exception levée lorsque l'état sérialisé XML de l'objet dans le cache ne correspond pas aux informations de type se trouvant dans le manifeste des données en mémoire cache.
