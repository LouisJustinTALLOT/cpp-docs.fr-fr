# MSBuild Error MSB1018
**Le niveau de commentaires n'est pas valide.**  
  
 Le niveau de commentaires spécifié n'est pas l'un des niveaux de commentaires disponibles.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez l'orthographe du niveau de commentaires.  Les niveaux de commentaires disponibles sont : q\[uiet\], m\[inimal\], n\[ormal\], d\[etailed\], et diag\[nostic\], par exemple, `/verbosity:quiet`, `/verbosity:q` ou  `/v:q`
