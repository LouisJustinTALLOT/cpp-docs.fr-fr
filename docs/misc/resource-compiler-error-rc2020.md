# Erreur RC2020 du compilateur de ressources
chiffre non conforme pour la base  
  
 Le `digit` spécifié n’est pas un chiffre valide pour la base spécifiée par `radix`. Le chiffre ou la base est peut\-être incorrect\(e\).  
  
 Les chiffres en notation octale doivent être nombres de 0 à 9, et les chiffres en notation hexadécimale doivent être des nombres de 0 à 9 ou des lettres de A à E.
