# There was an error trying to open or create the application definition file for this project. &#39;&lt;error&gt;&#39;
Cette erreur se produit lorsque vous essayez d'ouvrir le fichier de définition d'application alors qu'il n'est pas disponible.  Il a peut\-être été déplacé ou supprimé.  
  
### Pour corriger cette erreur  
  
-   Recréez le fichier de définition d'application.
