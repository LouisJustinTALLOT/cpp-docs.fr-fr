# Erreur de g&#233;n&#233;ration de projet&#160;PRJ0012
Les ressources système sont peut\-être dangereusement basses. Impossible de créer le thread nécessaire au lancement d’une génération.  
  
 Si les ressources système sont insuffisantes :  
  
-   Fermez certaines des applications en cours d’exécution.  
  
-   Redémarrez Visual Studio pour libérer des ressources.  
  
-   Redémarrez.
