# MSBuild Error MSB3114
**MSB3114 : Impossible de trouver le fichier '\<fichier\>' référencé par l'assembly '\<assembly\>'.**  
  
 Cette erreur est générée lorsqu'une référence qui ne peut pas être résolue figure dans un manifeste existant.  
  
### Pour corriger cette erreur  
  
-   Vérifiez vos manifestes pour détecter tout conflit dans les références de fichier.
