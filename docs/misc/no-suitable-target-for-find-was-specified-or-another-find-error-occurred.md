# No suitable target for find was specified or another find error occurred.
Cette erreur se produit généralement lors d'une tentative de recherche avec la zone de recherche ayant pour valeur Dans le document actif alors qu'aucun fichier n'est ouvert.  
  
### Pour corriger cette erreur  
  
1.  Ouvrez un fichier et recommencez la recherche ou modifiez la zone de recherche.
