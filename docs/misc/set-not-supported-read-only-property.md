# Set non pris en charge (propri&#233;t&#233; en lecture seule)
Vous avez essayé de définir ou modifier une propriété qui est en lecture seule.  
  
### Pour corriger cette erreur  
  
1.  Supprimez la référence à la propriété de votre code.  
  
2.  Modifiez la référence pour qu’elle renvoie uniquement la valeur de la propriété au moment de l’exécution.
