# Avertissement RC4009 du compilateur de ressources
chaîne trop grande, caractères de fin tronqués  
  
 Une chaîne était trop grande pour tenir dans une mémoire tampon. Les caractères de fin ont été tronqués et la chaîne restante a été utilisée.
