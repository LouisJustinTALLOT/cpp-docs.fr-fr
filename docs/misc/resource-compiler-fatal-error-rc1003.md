# Erreur irr&#233;cup&#233;rable RC1003 du compilateur de ressources
le nombre d’erreurs est supérieur à une valeur ; arrêt de la compilation  
  
 Le nombre d’erreurs est trop élevé pour poursuivre la compilation. Résolvez des erreurs, puis recompilez.
