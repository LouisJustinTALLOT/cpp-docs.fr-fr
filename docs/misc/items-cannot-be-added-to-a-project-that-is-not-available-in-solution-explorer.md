# Impossible d’ajouter des &#233;l&#233;ments &#224; un projet non disponible dans l’Explorateur de solutions.
Cette erreur se produit généralement quand un projet ayant été spécifié n’est pas ouvert dans l’Explorateur de solutions. Visual Studio ne peut pas ajouter d’éléments à un projet qui n’est pas disponible dans l’Explorateur de solutions.  
  
### Pour corriger cette erreur  
  
1.  Ajoutez le projet spécifié à la solution actuelle ou ouvrez la solution contenant le projet spécifié.
