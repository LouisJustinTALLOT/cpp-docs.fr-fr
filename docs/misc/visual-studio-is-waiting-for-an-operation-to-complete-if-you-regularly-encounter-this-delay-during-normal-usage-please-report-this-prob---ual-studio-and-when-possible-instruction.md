# Visual Studio is waiting for an operation to complete. If you regularly encounter this delay during normal usage please report this problem to Microsoft. Please include a description of the work you’re doing in Visual Studio and when possible instruction
L'application serveur est occupée et ne peut pas terminer la demande en cours.  
  
 Cette erreur peut également être provoquée par un logiciel antivirus qui bloque des processus devenv.exe.  Plusieurs fonctionnalités du produit utilisent des scripts, et ces scripts peuvent être bloqués par un logiciel antivirus.  Pour des informations connexes, consultez « PROBLÈME : L'environnement Visual IDE ne s'ouvre pas au démarrage ou vous recevez un message « Impossible de démarrer l'application » disponible à l'adresse suivante [http:\/\/support.microsoft.com\/default.aspx?scid\=kb;en\-us;306905&Product\=vsnet](http://support.microsoft.com/default.aspx?scid=kb;en-us;306905&Product=vsnet).  
  
### Pour corriger des erreurs liées à l'application serveur  
  
1.  Choisissez Basculer vers pour ouvrir l'application et essayer de résoudre le problème.  
  
     \- ou \-  
  
     Choisissez Réessayer pour attendre que l'application serveur ait fini de traiter la demande.  
  
     \- ou \-  
  
     Choisissez Annuler pour mettre fin à la demande.  
  
### Pour corriger les erreurs liées à un antivirus  
  
-   Dans votre logiciel antivirus, spécifiez que le logiciel doit autoriser l'exécution des scripts provenant de devenv.exe.  Pour des instructions détaillées, consultez la documentation produit de l'antivirus.
