# File name &lt;name&gt; cannot contain character device names.
Cette erreur se produit généralement lorsqu'un nom de fichier a été entré et que celui\-ci contient un nom de périphérique.  Parmi les noms de périphériques, on trouve CON, PRN, AUX, CLOCK$, NUL, LPT1, LPT2, COM1, COM2, COM3 et COM4.  
  
### Pour corriger cette erreur  
  
1.  Entrez un nom de fichier différent.
