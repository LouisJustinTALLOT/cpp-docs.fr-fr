# Erreur du compilateur CS0221
La valeur de constante 'value' ne peut pas être convertie en 'type' \(utilisez la syntaxe 'unchecked' pour la remplacer\)  
  
 Une opération d’assignation qui entraînerait une perte de données a été détectée par [checked](../Topic/checked%20\(C%23%20Reference\).md), qui est activé par défaut. Corrigez l’assignation ou utilisez [unchecked](../Topic/unchecked%20\(C%23%20Reference\).md) pour résoudre cette erreur. Pour plus d'informations, consultez [Checked et unchecked](../Topic/Checked%20and%20Unchecked%20\(C%23%20Reference\).md).  
  
 L’exemple suivant génère l’erreur CS0221 :  
  
```  
// CS0221.cs public class MyClass { public static void Main() { // unchecked // { int a = (int)0xFFFFFFFF;   // CS0221 a++; // } } }  
```
