# Internal error processing pattern.
Cette erreur se produit généralement quand une expression régulière a été incluse dans une chaîne recherchée alors que la syntaxe de l'expression régulière contient des erreurs.  
  
### Pour corriger cette erreur  
  
1.  Entrez à nouveau la chaîne recherchée en utilisant une syntaxe correcte dans l'expression régulière.  Pour plus d'informations, consultez [Expressions régulières](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md).
