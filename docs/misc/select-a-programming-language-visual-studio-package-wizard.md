# s&#233;lectionner un langage de programmation, Assistant Package Visual Studio Package
Cette étape vous permet de :  
  
-   sélectionner le langage de programmation que vous voulez utiliser pour créer votre VSPackage ;  
  
-   générer ou utiliser un fichier de clé existant pour signer votre assembly VSPackage.  
  
 **Visual C\#**  
 Crée votre VSPackage en C\#.  
  
 **Visual Basic**  
 Crée votre VSPackage en Visual Basic.  
  
 **Visual C\+\+**  
 Crée votre VSPackage en Visual C\+\+.  
  
## Obtenir les clés pour la signature de l'assembly  
 **Générer un nouveau fichier de clé pour la signature de l'assembly**  
 Permet à l’Assistant de générer un fichier de clé. Le fichier de clé généré s’appelle key.snk.  
  
 **Utilisez le fichier de clé suivant pour signer l’assembly :**  
 Permet de taper le chemin d’un fichier de clé existant \(sinon, cliquez sur **Parcourir** pour obtenir le fichier\). L’Assistant effectue une copie du fichier et le nomme key.snk.  
  
 **Annuler**  
 Annule la création du VSPackage.  
  
 **Précédent**  
 Revient à la page précédente de l’Assistant.  
  
 **Suivant**  
 Accède à la page suivante de l’Assistant.  
  
 **Terminer**  
 Crée le VSPackage.
