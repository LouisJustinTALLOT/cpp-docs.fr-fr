# Erreur du compilateur CS0742
Un corps de requête doit terminer par une clause select ou une clause group  
  
 Une expression de requête doit se terminer par une clause `select` ou une clause `group` sans continuation.  
  
### Pour corriger cette erreur  
  
1.  Ajoutez une [clause select](../Topic/select%20clause%20\(C%23%20Reference\).md) ou une [clause group](../Topic/group%20clause%20\(C%23%20Reference\).md) à la requête.  
  
## Exemple  
 Le code suivant génère l’erreur CS0742 :  
  
```  
// cs0742.cs using System.Linq; public class Test { public static int Main() { int[] array = { 1, 2, 3 }; var query = from num in array; // CS0742 return 1; } }  
```  
  
 Si la clause `group` utilise le mot clé [into](../Topic/into%20\(C%23%20Reference\).md) pour stocker les résultats du regroupement dans un identificateur temporaire, elle ne peut pas être la dernière clause dans une requête. Une clause `select` ou une deuxième clause `group` est toujours exigée.
