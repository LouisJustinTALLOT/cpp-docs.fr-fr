# Erreur de l’&#233;valuateur d’expression CXX0003
erreur interne dans l’évaluateur d’expression  
  
 L’évaluateur d’expression a rencontré une erreur interne.  
  
 Notez les circonstances de l’erreur et signalez\-la à Microsoft Corporation. Vous pouvez signaler l’erreur en ligne à l’adresse [http:\/\/support.microsoft.com\/support\/visualc\/report\/](http://support.microsoft.com/support/visualc/report/) ou vous pouvez appeler les Services de support technique Microsoft. Pour plus d’informations, cliquez sur **Support technique** dans le menu **Aide** de Visual C\+\+.  
  
 Cette erreur est identique à l’erreur CAN0003.
