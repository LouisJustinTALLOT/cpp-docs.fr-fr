# Erreur irr&#233;cup&#233;rable RC1107 du compilateur de ressources
utilisation non valide ; utilisez RC \/? pour l'aide  
  
 Vous avez spécifié une option de compilateur de ressources non valide. Les options valides sont \/r, \/v, \/d, \/fo, \/l, \/i, \/x et \/c.
