# MSBuild Error MSB3124
**MSB3124 : Une association de fichier a déjà été créée pour l'extension '\<NomExtension\>'.**  
  
 Cette erreur se produit lorsqu'une extension d'association de fichier en double est rencontrée.  
  
### Pour corriger cette erreur  
  
-   Supprimez les attributs `extension` de l'élément fileAssociation \(voir [ClickOnce Deployment Manifest](../Topic/ClickOnce%20Deployment%20Manifest.md)\) qui ne sont pas uniques.  Chaque attribut d'extension de l'élément \<fileAssociation\> répertorié doit être unique.
