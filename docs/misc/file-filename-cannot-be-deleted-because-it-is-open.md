# Impossible de supprimer le fichier &#39;&lt;nom_fichier&gt;&#39;, car il est ouvert
L’utilisateur a tenté de supprimer un fichier ouvert.  
  
### Pour corriger cette erreur  
  
-   Fermez le fichier avant de le supprimer.
