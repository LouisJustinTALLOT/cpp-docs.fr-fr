# Le nom de fichier n’existe pas, n’est pas valide ou contient des param&#232;tres impossibles &#224; valider. Voulez-vous modifier votre entr&#233;e&#160;?
Cette erreur se produit généralement quand un chemin ou un nom de fichier incorrect est entré dans la boîte de dialogue **Ajouter un programme**.  
  
### Pour corriger cette erreur  
  
1.  Choisissez **Parcourir** pour accéder au fichier approprié.  
  
     \- ou \-  
  
2.  Tapez à nouveau le chemin et le nom de fichier.
