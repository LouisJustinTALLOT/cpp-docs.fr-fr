# Erreur MSBuild MSB8020
Les outils de build pour *\<plateforme\>* \(ensemble d’outils de plateforme \= '*\<version\>*'\) sont introuvables. Pour effectuer la génération avec les outils de build *\<version\>*, installez les outils de build pour *\<plateforme\>*.  Vous pouvez également mettre à niveau les outils Visual Studio actuels en sélectionnant le menu Projet ou en cliquant avec le bouton droit sur la solution, puis en sélectionnant « Recibler la solution ».
