# L’argument &#39;&lt;nom_argument&gt;&#39; doit &#234;tre sup&#233;rieur ou &#233;gal &#224; -1
La valeur d’un argument est inférieure à \-1.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que la valeur de l’argument est comprise dans la plage valide.  
  
2.  Si l’argument est fourni par un calcul, vérifiez les valeurs traitées lors du calcul.
