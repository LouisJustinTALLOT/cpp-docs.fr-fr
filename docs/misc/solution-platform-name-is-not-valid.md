# Solution platform name is not valid.
Les noms des plateformes de solution ne peuvent pas contenir les caractères suivants :  
  
-   dièse \(\#\)  
  
-   pourcentage \(%\)  
  
-   et commercial \(&\)  
  
-   astérisque \(\*\)  
  
-   barre verticale \(&#124;\)  
  
-   barre oblique inverse \(\\\)  
  
-   deux\-points \(:\)  
  
-   guillemet double \("\)  
  
-   inférieur à \(\<\)  
  
-   supérieur à \(\>\)  
  
-   point \(.\)  
  
-   point d'interrogation \(?\)  
  
-   barre oblique \(\/\)  
  
-   espace à gauche ou à droite \(' '\)  
  
-   noms réservés à Windows ou à MS\-DOS \(nul, aux, con, com1, lpt1, etc.\)  
  
### Pour corriger cette erreur  
  
1.  Entrez un nom de plateforme de solution qui ne contient pas les caractères mentionnés ci\-dessus.
