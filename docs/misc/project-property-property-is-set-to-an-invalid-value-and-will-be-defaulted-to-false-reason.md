# Project property &#39;property&#39; is set to an invalid value and will be defaulted to FALSE. &lt;reason&gt;
Une valeur persistante d'une propriété `Boolean` n'est pas l'une des valeurs suivantes :  
  
-   1 ou 0 ;  
  
-   On ou Off ;  
  
-   true ou false.  
  
 La cause la plus probable de cette erreur est la modification manuelle du fichier projet.  
  
 **Pour corriger cette erreur**  
  
-   Modifiez de nouveau le fichier projet et remettez\-le en état.  
  
     La valeur de la propriété affectée aura la valeur "false".
