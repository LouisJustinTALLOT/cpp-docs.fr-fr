# D&#233;pannage des exceptions&#160;: System.Drawing.Printing.InvalidPrinterException
Une exception <xref:System.Drawing.Printing.InvalidPrinterException> est levée lors d'une tentative d'accès à une imprimante à l'aide de paramètres d'impression non valides.  
  
## Conseils associés  
 **Assurez\-vous que l'imprimante existe.**  
 La cause la plus courante de l'invalidité de paramètres d'impression est la référence à une imprimante inexistante. Pour plus d'informations, consultez <xref:System.Drawing.Printing>.  
  
 **Assurez\-vous qu'une imprimante par défaut a été installée.**  
 Si aucune imprimante n'a été spécifiée, assurez\-vous qu'une imprimante par défaut a été installée. Pour plus d'informations, consultez <xref:System.Drawing.Printing.PrintDocument.PrinterSettings%2A>
