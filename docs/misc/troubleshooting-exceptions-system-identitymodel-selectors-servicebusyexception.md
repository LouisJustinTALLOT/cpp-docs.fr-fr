# D&#233;pannage des exceptions&#160;: System.IdentityModel.Selectors.ServiceBusyException
Une exception <xref:System.IdentityModel.Selectors.ServiceBusyException> est renvoyée pour indiquer que le service CardSpace est occupé à traiter d'autres demandes. CardSpace ne met pas en file d'attente les demandes et ne peut en traiter qu'une seule à la fois.  
  
 Déterminez si une autre instance de CardSpace est exécutée. Le cas échéant, terminez cette instance en arrêtant le processus icardagt.exe dans le Gestionnaire de tâches.
