# D&#233;pannage des exceptions&#160;: System.Workflow.Runtime.WorkflowOwnershipException
Une exception <xref:System.Workflow.Runtime.WorkflowOwnershipException> est renvoyée lorsque le moteur d'exécution du workflow essaie de charger une instance du workflow actuellement chargée par une autre instance du moteur d'exécution du workflow. En outre, cette exception est levée lorsque le moteur d'exécution du workflow essaie d'enregistrer un workflow après le délai d'attente de propriété spécifié alors que le chargement du workflow a expiré.
