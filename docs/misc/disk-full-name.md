# Disk full: &lt;name&gt;
Le lecteur contenant le programme ne possède pas suffisamment d'espace libre pour enregistrer les fichiers programme, tels que le toolbox.tbd, lors de la fermeture du programme.  
  
### Pour corriger cette erreur  
  
1.  Supprimez des fichiers du lecteur et réessayez ensuite de fermer le programme.
