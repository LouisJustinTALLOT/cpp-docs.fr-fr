# Erreur de g&#233;n&#233;ration de projet PRJ0038
Répertoire racine IIS introuvable. IIS doit être installé pour que l’outil de déploiement web fonctionne.  
  
 Le déploiement web nécessite qu’IIS soit installé sur l’ordinateur.
