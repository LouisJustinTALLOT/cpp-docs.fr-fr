# Erreur du compilateur CS1541
Option de référence non valide : 'symbole' \-\- ne peut pas faire référence à des répertoires  
  
 Le compilateur a détecté une tentative de spécification d’un répertoire au lieu d’un fichier. Par exemple, quand vous utilisez l’option du compilateur [\/reference](../Topic/-reference%20\(C%23%20Compiler%20Options\).md), vous devez spécifier un fichier. Vous ne pouvez pas spécifier un répertoire.  
  
 Par exemple, si vous passez `/reference:c:\` au compilateur, l’erreur CS1541 est générée.
