# Erreur du compilateur CS0180
'membre' ne peut pas être à la fois extern et abstract  
  
 Les mots clés [abstract](../Topic/abstract%20\(C%23%20Reference\).md) et [extern](../Topic/extern%20\(C%23%20Reference\).md) s’excluent mutuellement. Le mot clé `extern` signifie que le membre est défini en dehors du fichier, et **abstract** signifie que l’implémentation est fournie dans une classe dérivée. Pour plus d’informations, consultez [Méthodes](../Topic/Methods%20\(C%23%20Programming%20Guide\).md).  
  
 L’exemple suivant génère l’erreur CS0180 :  
  
```  
// CS0180.cs namespace MyNamespace { public class MyClass { public extern abstract int Foo(int a);   // CS0180 public static void Main() { } } }  
```
