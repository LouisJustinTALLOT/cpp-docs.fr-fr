# Component is not correctly registered.
Cette erreur se produit généralement lorsque le Registre est endommagé, que le système n'arrive pas à accéder au registre ou qu'un élément a été écrit dans le Registre dans un format incorrect.  
  
### Pour corriger cette erreur  
  
1.  Réinstallez le composant.
