# File &lt;filename&gt; cannot be created.
Cette erreur se produit généralement lors d'une tentative de création d'un rapport Web de commentaires du code alors que l'utilisateur en cours ne possède pas d'accès en écriture.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que vous disposez d'un accès en écriture.  
  
2.  Vérifiez que l'espace disponible est suffisant pour enregistrer les pages.
