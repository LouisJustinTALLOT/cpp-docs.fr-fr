# &#39;Compare&#39; attendu
Une instruction `Option` spécifie `Text` ou `Binary`, mais pas `Compare`.  
  
 **ID d’erreur :** BC30208  
  
### Pour corriger cette erreur  
  
-   Ajoutez le mot clé `Compare` à l’instruction `Option` ; par exemple, `Option Compare Binary`.
