# La d&#233;finition de la m&#233;thode &lt;m&#233;thode&gt; n’est pas accessible dans ce contexte
Une fonction d’agrégation qui n’existe pas a été incluse avec l’opérateur `Into`.  
  
 **ID d’erreur :** BC36594  
  
### Pour corriger cette erreur  
  
-   Vérifiez l’orthographe du nom de la fonction d’agrégation et apportez les corrections nécessaires.  
  
-   Ajoutez la méthode comme fonction d’agrégation personnalisée \(méthode d’extension\). Pour plus d’informations sur la création d’une fonction d’agrégation personnalisée, consultez [Aggregate Clause](../Topic/Aggregate%20Clause%20\(Visual%20Basic\).md).
