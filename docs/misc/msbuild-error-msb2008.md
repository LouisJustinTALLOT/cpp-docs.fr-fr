# MSBuild Error MSB2008
**Ce système de projet Visual Studio ne peut pas convertir les projets "{0}".  Il ne permet de convertir que les projets clients C\# et VB.**  
  
 Seuls les projets [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] et [!INCLUDE[csprcs](../ide/includes/csprcs_md.md)] valides peuvent être convertis à l'aide de [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)].  
  
### Pour corriger cette erreur  
  
-   Si le projet est un projet [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] ou [!INCLUDE[csprcs](../ide/includes/csprcs_md.md)], vérifiez si le fichier projet a été modifié ou endommagé.  S'il a été modifié ou endommagé, ouvrez le projet dans la version de [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] avec laquelle il a été créé, enregistrez\-le, puis essayez à nouveau de le convertir.  
  
-   S'il ne s'agit pas d'un projet [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] ou [!INCLUDE[csprcs](../ide/includes/csprcs_md.md)], utilisez l'outil approprié pour le convertir.
