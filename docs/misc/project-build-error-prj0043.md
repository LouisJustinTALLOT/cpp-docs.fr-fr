# Erreur de g&#233;n&#233;ration de projet PRJ0043
La propriété 'Sorties' de la règle de génération personnalisée 'règle' assignée au fichier 'fichier' n’est pas valide. Cette propriété contient 'chaîne' qui prend la valeur 'valeur'.  
  
 La propriété Sorties a été évaluée comme une chaîne vide. Les règles de génération personnalisées ont besoin de la sortie de l’action de génération.
