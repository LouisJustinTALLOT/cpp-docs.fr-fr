# D&#233;pannage des exceptions&#160;: System.ServiceModel.PoisonMessageException
Une exception <xref:System.ServiceModel.PoisonMessageException> est renvoyée lorsqu'un message est considéré comme "incohérent". Un message est considéré comme incohérent si plusieurs tentatives de remise du message sont répétées.
