# CA0055
Impossible de charger le fichier\<CheminAccès\>  
  
 L'outil d'analyse du code n'a pas réussi à charger le fichier cible spécifié pour l'analyse.  Cela se produit si le fichier est introuvable ou si l'accès au fichier a été refusé.  En général, cela se produit si l'utilisateur a choisi de substituer la propriété d'assembly d'entrée dans [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)], ou a appelé fxcopcmd.exe manuellement.
