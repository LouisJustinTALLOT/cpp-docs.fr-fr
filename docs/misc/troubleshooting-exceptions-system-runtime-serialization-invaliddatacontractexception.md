# D&#233;pannage des exceptions&#160;: System.Runtime.Serialization.InvalidDataContractException
Une exception <xref:System.Runtime.Serialization.InvalidDataContractException> est renvoyée lorsque le <xref:System.Runtime.Serialization.DataContractSerializer> ou le <xref:System.Runtime.Serialization.NetDataContractSerializer> rencontre un contrat de données non valide pendant la sérialisation et la désérialisation.
