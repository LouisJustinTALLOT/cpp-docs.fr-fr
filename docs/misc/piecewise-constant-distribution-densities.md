# piecewise_constant_distribution::densities
Retourne les densités stockées pour la distribution.  
  
## Syntaxe  
  
```  
std::vector<double> densities() const;  
```  
  
## Notes  
 La fonction membre retourne les densités stockées, calculées à partir des paramètres de poids.  
  
## Configuration requise  
 **En\-tête :** \<random\>  
  
 **Espace de noms :** std
