# File name or default path is too long.
Cette erreur se produit généralement lors d'une tentative de création d'un rapport Web de commentaires du code alors que le nom de fichier ou le chemin d'accès spécifié dans la zone Enregistrer les pages Web dans \(boîte de dialogue Générer les pages Web de commentaires\) est trop long.  
  
### Pour corriger cette erreur  
  
1.  Spécifiez un nom de fichier ou un chemin d'accès comptant moins de 260 caractères.
