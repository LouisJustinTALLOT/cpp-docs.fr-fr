# Command does not accept switches.
Cette erreur se produit généralement lorsqu'un commutateur a été spécifié avec une commande qui n'accepte que des arguments.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez la syntaxe requise pour la commande.  Pour plus d'informations, consultez [Commandes Visual Studio utilisant des arguments](../Topic/Visual%20Studio%20Commands.md).
