# Evaluation stack overflow. The pattern may be too complex.
Cette erreur se produit généralement quand une expression régulière complexe a été utilisée dans une chaîne recherchée qui ne peut pas être traitée par l'environnement de développement intégré \(IDE, Integrated Development Environment\).  
  
### Pour corriger cette erreur  
  
1.  Réduisez le nombre de clôtures et de clauses, puis recommencez.
