# D&#233;pannage des exceptions&#160;: System.Transactions.TransactionInDoubtException
Exception levée lors d'une tentative d'opération sur une transaction incertaine.  
  
## Notes  
 Une transaction est considérée comme incertaine lorsqu'un gestionnaire de ressources inscrit ne peut pas être contacté.
