# Property value is not valid. (Visual Studio) (1)
Cette erreur se produit généralement lorsqu'une valeur non valide est entrée pour une propriété.  Par exemple, si la propriété est une valeur booléenne dont les valeurs valides sont TRUE et FALSE, la valeur MAYBE n'est pas valide pour la propriété.  
  
### Pour corriger cette erreur  
  
1.  Entrez une valeur valide pour la propriété.  
  
    > [!TIP]
    >  Recherchez dans la documentation en ligne les valeurs valides pour la propriété.
