# &lt;adresse&gt; n’est pas une adresse valide de fichier distant
\<adresse\> n’est pas une adresse valide de fichier distant. Une adresse valide doit inclure un protocole, un chemin et un nom de fichier.  
  
 Une adresse non valide a été spécifiée pour une opération de chargement ou de téléchargement. Les adresses doivent inclure un protocole, tel que `ftp` ou `http`, un chemin valide et le nom de fichier. Cette erreur peut être le résultat d’une analyse incorrecte des chemins de fichier.  
  
### Pour corriger cette erreur  
  
-   Vérifiez l’adresse que vous avez spécifiée pour déterminer l’emplacement de l’erreur et indiquez les informations nécessaires.
