# D&#233;pannage des exceptions&#160;: System.InvalidProgramException
Exception levée lorsqu'un programme contient du langage MSIL \(Microsoft Intermediate Language\) ou des métadonnées non valides. Elle indique généralement une erreur dans le compilateur qui a généré le programme.
