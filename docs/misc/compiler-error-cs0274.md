# Erreur du compilateur CS0274
Impossible de spécifier des modificateurs d’accessibilité pour les accesseurs de la propriété ou de l’indexeur 'property\/indexer'  
  
 Cette erreur se produit quand vous déclarez une propriété ou un indexeur avec des modificateurs d’accès sur ses deux accesseurs. Pour résoudre cette erreur, utilisez un modificateur d’accès sur un seul des deux accesseurs. Pour plus d'informations, consultez [Accessibilité des accesseurs](../Topic/Restricting%20Accessor%20Accessibility%20\(C%23%20Programming%20Guide\).md).  
  
 L’exemple suivant génère l’erreur CS0274 :  
  
```  
// CS0274.cs public class MyClass { public int Property   // CS0274 { public get { return 0; } protected set { } } }  
```
