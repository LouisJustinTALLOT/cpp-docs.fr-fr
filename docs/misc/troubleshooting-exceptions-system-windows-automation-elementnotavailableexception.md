# D&#233;pannage des exceptions&#160;: System.Windows.Automation.ElementNotAvailableException
Une exception <xref:System.Windows.Automation.ElementNotAvailableException> contient les informations relatives à l'exception qui est levée lors d'une tentative d'accès à un <xref:System.Windows.Automation.AutomationElement> d'interface utilisateur correspondant à une partie de l'interface utilisateur qui n'est plus disponible.  
  
## Notes  
 Cette exception peut être renvoyée si l'élément se trouve dans une boîte de dialogue qui a été fermée ou dans une application qui a été fermée ou arrêtée.
