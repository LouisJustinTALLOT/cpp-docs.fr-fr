# Cet outil n&#233;cessite un titre. Si vous ne voulez pas enregistrer cet outil, cliquez sur Supprimer dans la bo&#238;te de dialogue Outils externes.
Cette erreur se produit généralement quand un nouvel outil a été ajouté à la boîte de dialogue Outils externes mais qu'aucun titre n'a été spécifié.  
  
### Pour corriger cette erreur  
  
1.  Si vous avez l'intention d'enregistrer l'outil, cliquez sur **OK** et entrez un titre pour l'outil dans la boîte de dialogue **Outils externes**.  
  
2.  Si vous n'avez pas l'intention d'enregistrer l'outil, cliquez sur **OK**, puis sur **Supprimer** dans la boîte de dialogue **Outils externes**.
