# Erreur irr&#233;cup&#233;rable RC1014 du compilateur de ressources
fichiers Include trop nombreux : profondeur \= 'level'  
  
 La profondeur d’imbrication des directives `#include` est trop importante.  
  
 Utilisez des directives imbriquées pour inclure des fichiers ouverts. Le fichier source contenant la directive est considéré comme l’un des fichiers.
