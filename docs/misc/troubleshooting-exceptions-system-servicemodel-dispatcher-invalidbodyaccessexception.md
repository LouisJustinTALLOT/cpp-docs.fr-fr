# D&#233;pannage des exceptions&#160;: System.ServiceModel.Dispatcher.InvalidBodyAccessException
Une exception <xref:System.ServiceModel.Dispatcher.InvalidBodyAccessException> représente la classe de base abstraite pour les exceptions renvoyées si une tentative est faite pour accéder au corps d'un message alors que cela n'est pas permis.
