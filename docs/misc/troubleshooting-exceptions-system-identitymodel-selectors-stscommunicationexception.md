# D&#233;pannage des exceptions&#160;: System.IdentityModel.Selectors.StsCommunicationException
Une exception <xref:System.IdentityModel.Selectors.StsCommunicationException> est levée en cas de problème lors de la communication avec le service d'émission de jeton de sécurité.  
  
 ce problème peut résulter des conditions suivantes :  
  
-   Panne réseau  
  
-   Point de terminaison MEX inaccessible  
  
-   Le service d'émission de jeton de sécurité ne possède pas d'identité de point de terminaison valide
