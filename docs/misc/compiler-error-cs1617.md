# Erreur du compilateur CS1617
Option 'option' non valide pour \/langversion ; doit avoir la valeur ISO\-2, ISO\-2 ou Default  
  
 Cette erreur se produit si vous avez utilisé le paramètre de projet ou le commutateur de ligne de commande [\/langversion](../Topic/-langversion%20\(C%23%20Compiler%20Options\).md) sans spécifier d’option de langue valide. Pour résoudre cette erreur, vérifiez la syntaxe de ligne de commande ou le paramètre de projet et remplacez\-le par l’une des options répertoriées.  
  
 Par exemple, si vous compilez avec `csc /langversion:ISO`, l’erreur CS1617 est générée.
