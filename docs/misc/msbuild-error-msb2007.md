# MSBuild Error MSB2007
**Visual Studio ne peut pas trouver le fichier projet "{0}".**  
  
 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] n'a pas pu trouver le fichier projet spécifié.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez l'orthographe du nom du fichier projet.
