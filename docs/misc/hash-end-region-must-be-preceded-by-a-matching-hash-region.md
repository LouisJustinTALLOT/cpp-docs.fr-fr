# &#39;#End Region&#39; doit &#234;tre pr&#233;c&#233;d&#233; d’un &#39;#Region&#39; correspondant
`#Region` vous permet de spécifier un bloc de code que vous pouvez développer ou réduire lors de l’utilisation de la fonctionnalité mode Plan de l’éditeur de code [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]. Le début et la fin des instructions `#Region` doivent figurer dans le même bloc de code.  
  
 **ID d’erreur :** BC30680  
  
### Pour corriger cette erreur  
  
-   Insérez `#Region` à l’emplacement approprié avant l’instruction `#End` `Region` correspondante.
