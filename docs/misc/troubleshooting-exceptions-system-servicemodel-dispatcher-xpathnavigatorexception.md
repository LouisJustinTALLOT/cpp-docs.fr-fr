# D&#233;pannage des exceptions&#160;: System.ServiceModel.Dispatcher.XPathNavigatorException
Une exception <xref:System.ServiceModel.Dispatcher.XPathNavigatorException> est renvoyée lorsque le quota de nœuds dont l'inspection est autorisée par un <xref:System.Xml.XPath.XPathNavigator> est dépassé.
