# Erreur du compilateur CS1007
Accesseur de propriété déjà défini  
  
 Lorsque vous déclarez une [propriété](../Topic/Using%20Properties%20\(C%23%20Programming%20Guide\).md), vous devez également déclarer ses méthodes d’accesseur. Toutefois, une propriété ne peut pas avoir plusieurs méthodes d’accesseur `get` ni plusieurs méthodes d’accesseur `set`.  
  
## Exemple  
 L’exemple suivant génère l’erreur CS1007 :  
  
```  
// CS1007.cs public class clx { public int MyProperty { get { return 0; } get   // CS1007, this is the second get method { return 0; } } public static void Main() {} }  
```
