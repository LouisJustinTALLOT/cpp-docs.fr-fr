# D&#233;pannage des exceptions&#160;: System.Windows.Automation.NoClickablePointException
Une exception <xref:System.Windows.Automation.NoClickablePointException> contient les informations relatives à l'exception renvoyée lorsque <xref:System.Windows.Automation.AutomationElement.GetClickablePoint%2A> est appelé sur un <xref:System.Windows.Automation.AutomationElement> d'interface utilisateur qui ne comporte pas de point interactif.  
  
## Notes  
 Cette exception peut être levée lorsque la fenêtre d'application est réduite ou que l'élément est hors écran.
