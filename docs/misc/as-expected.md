# &#39;As&#39; attendu
Une déclaration ne fournit pas de clause `As` pour un ou plusieurs des éléments déclarés.  
  
 **ID d’erreur :** BC30197  
  
### Pour corriger cette erreur  
  
-   Fournissez la clause `As` en fonction des besoins ou définissez `Option Strict Off`.
