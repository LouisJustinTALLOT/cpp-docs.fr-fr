# D&#233;pannage des exceptions&#160;: System.Threading.SemaphoreFullException
Exception levée lorsque la surcharge de la méthode <xref:System.Threading.Semaphore.Release%2A> est appelée alors que le compteur du sémaphore est déjà à la valeur maximale.
