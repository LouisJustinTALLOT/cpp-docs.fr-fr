# Erreur de g&#233;n&#233;ration de projet PRJ0011
Les ressources système sont peut\-être dangereusement basses. Impossible de créer un événement requis au lancement d’une génération.  
  
 Si les ressources système sont insuffisantes :  
  
-   Fermez certaines des applications en cours d’exécution.  
  
-   Redémarrez Visual Studio pour libérer des ressources.  
  
-   Redémarrez.
