# Unable to Open Linked File
Un fichier a été ajouté au projet sous la forme d'un fichier lié.  Lorsque vous avez ouvert le projet, toutefois, le système de projet a été incapable de localiser le fichier.  
  
 **Pour corriger cette erreur**  
  
-   Supprimez le fichier lié dans l'Explorateur de solutions.  
  
     \- ou \-  
  
-   Replacez le fichier à l'emplacement où le système de projet s'attend à le trouver.
