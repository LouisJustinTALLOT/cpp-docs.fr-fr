# Espace blanc obligatoire manquant
Un littéral XML a été spécifié sans l’espace de séparation exigé. L’espace entre les attributs consécutifs d’un littéral d’élément XML constitue un exemple d’espace de séparation exigé.  
  
 **ID d’erreur :** BC31173  
  
### Pour corriger cette erreur  
  
-   Ajoutez le séparateur d’espace obligatoire.
