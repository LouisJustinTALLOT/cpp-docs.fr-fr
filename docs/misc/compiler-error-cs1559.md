# Erreur du compilateur CS1559
Impossible d’utiliser 'object' pour la méthode Main, car il est importé  
  
 Une classe non valide a été spécifiée pour l’option du compilateur [\/main](../Topic/-main%20\(C%23%20Compiler%20Options\).md) ; la classe ne peut pas être utilisée comme emplacement pour la méthode [Main](../Topic/Main\(\)%20and%20Command-Line%20Arguments%20\(C%23%20Programming%20Guide\).md).
