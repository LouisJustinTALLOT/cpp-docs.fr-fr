# Switch &lt;name&gt; specified with conflicting values.
Cette erreur se produit lorsque vous entrez deux fois un commutateur dans la fenêtre Commande, et que la valeur spécifiée pour le commutateur est différente à chaque instance.  
  
### Pour corriger cette erreur  
  
-   Ré\-entrez la commande, mais utilisez chaque commutateur une seule fois.
