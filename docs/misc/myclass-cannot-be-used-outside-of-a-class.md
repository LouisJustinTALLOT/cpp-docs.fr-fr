# &#39;MyClass&#39; ne peut pas &#234;tre utilis&#233; &#224; l’ext&#233;rieur d’une classe
Vous avez utilisé le mot clé `MyClass` en dehors d’une structure de classe.  
  
 **ID d’erreur :** BC30470  
  
### Pour corriger cette erreur  
  
-   Déplacez la déclaration `MyClass` vers votre structure de classe.
