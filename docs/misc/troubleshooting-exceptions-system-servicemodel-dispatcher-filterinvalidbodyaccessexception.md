# D&#233;pannage des exceptions&#160;: System.ServiceModel.Dispatcher.FilterInvalidBodyAccessException
Une exception <xref:System.ServiceModel.Dispatcher.FilterInvalidBodyAccessException> est renvoyée lorsqu'un filtre ou une table de filtres essaie d'accéder au corps d'un message qui n'est pas placé en mémoire tampon.
