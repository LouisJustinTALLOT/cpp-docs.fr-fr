# Erreur irr&#233;cup&#233;rable RC1005 du compilateur de ressources
fichier introuvable : nom de fichier  
  
 Le fichier spécifié sur la ligne de commande du compilateur de ressources est introuvable. Vérifiez que le fichier a été déplacé vers un autre répertoire et que le nom de fichier ou le chemin est correct.  
  
 Les fichiers sont recherchés à l’aide de la variable d’environnement **INCLUDE** ou du paramètre Répertoires Visual C\+\+.
