# D&#233;pannage des exceptions&#160;: System.IO.FileFormatException
Une exception <xref:System.IO.FileFormatException> est renvoyée lorsqu'un fichier d'entrée ou un flux de données supposé se conformer à une spécification de format de fichier spécifique est incorrect.
