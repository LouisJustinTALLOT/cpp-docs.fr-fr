# Command requires &lt;number&gt; arguments.
Cette erreur se produit généralement lorsqu'un nombre trop faible ou trop important d'arguments a été spécifié avec une commande.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez dans la documentation la syntaxe à respecter pour la commande et recommencez.
