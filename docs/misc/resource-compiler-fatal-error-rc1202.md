# Erreur irr&#233;cup&#233;rable RC1202 du compilateur de ressources
commutateur non valide \- ID de langue par défaut manquant après \/l  
  
 L’option de spécification de la langue par défaut \(\/ l\) n’a pas été suivie d’un ID de langue hexadécimal.
