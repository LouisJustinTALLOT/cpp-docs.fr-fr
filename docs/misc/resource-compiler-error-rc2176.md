# Erreur RC2176 du compilateur de ressources
**ancien DIB dans le nom de fichier. Passez\-le par le biais de SDKPAINT.**  
  
 Une bitmap indépendante du périphérique \(DIB\) à l’ancien format a été détectée dans le fichier spécifié. Vous devez la convertir au format actuel. Pour cela, vous pouvez utiliser l’application SDKPAINT fournie dans le Kit de développement logiciel \(SDK\) Windows 3.0 ou une application équivalente.
