# Impossible de cr&#233;er une interface runtime&#160;.NET&#160;: &lt;error&gt;
Une erreur interne s’est produite lors de la création d’une interface, ou un appel de méthode sur une interface a échoué.  
  
 **ID d’erreur :** BC31024  
  
### Pour corriger cette erreur  
  
1.  Enregistrez votre travail et redémarrez [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
2.  Si l’erreur se produit à nouveau, réinstallez [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
3.  Si l’erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
