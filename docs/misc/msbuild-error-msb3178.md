# MSBuild Error MSB3178
**MSB3178 : L'assembly '\<assembly\>' n'est pas spécifié correctement comme fichier.**  
  
 Cet avertissement est généré pendant la génération du manifeste d'application lorsque le processus de génération détecte qu'une référence de fichier est en réalité un assembly \(managé ou natif\).  
  
### Pour corriger cette erreur  
  
-   Pour plus d'informations sur la cause d'une erreur MSBuild et les résolutions possibles, consultez [Additional Resources](../Topic/Additional%20MSBuild%20Resources.md).
