# Unable to read the resource information from the resx file
Le système de projet n'a pas pu lire un fichier .resx.  
  
 Une raison spécifique au système d'exploitation \(le cas échéant\) est ajoutée à la fin de ce message.  Le nom de fichier du fichier qui n'a pas pu être lu est également ajouté.  
  
 Parmi les causes possibles de cette erreur, citons notamment l'absence des autorisations requises pour lire les fichiers ou une perte de la communication avec le serveur Web pour les projets Web.  
  
 Cette erreur entraîne l'échec du processus de génération.
