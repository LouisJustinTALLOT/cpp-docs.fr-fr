# Erreur du compilateur CS1604
Impossible d’assigner 'variable', car il est en lecture seule  
  
 Une assignation a été faite à une variable en lecture seule. Pour éviter cette erreur, n’assignez pas cette variable ou n’essayez pas de la modifier dans ce contexte.
