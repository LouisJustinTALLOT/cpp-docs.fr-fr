# &#39;&lt;rsetstmt&gt;&#39; n’est pas d&#233;clar&#233;.
'\<rsetstmt\>' n’est pas déclaré. Les instructions RSet ne sont plus prises en charge ; utilisez Microsoft.VisualBasic.RSet à la place.  
  
 Plusieurs fonctions qui étaient intrinsèques à [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] dans les versions précédentes ont été déplacées vers l’espace de noms <xref:Microsoft.VisualBasic?displayProperty=fullName>. Leurs fonctionnalités sont ainsi plus largement accessibles à tous les langages de programmation.  
  
 **ID d’erreur :** BC30821  
  
### Pour corriger cette erreur  
  
-   Utilisez plutôt la fonction `RSet` dans l’espace de noms `Microsoft.VisualBasic`.
