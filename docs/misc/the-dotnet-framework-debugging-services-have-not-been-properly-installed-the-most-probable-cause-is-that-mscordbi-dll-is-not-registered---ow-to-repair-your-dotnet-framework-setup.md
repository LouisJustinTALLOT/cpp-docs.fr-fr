# Les services de d&#233;bogage .NET Framework n’ont pas &#233;t&#233; correctement install&#233;s. La cause la plus probable est que mscordbi.dll n’est pas correctement enregistr&#233;. Cliquez sur l’aide pour plus d’informations sur la fa&#231;on de r&#233;parer votre installation de .NET Framework.
Cette erreur se produit généralement quand Visual Studio ne trouve pas le fichier mscordbi.dll, qui est installé avec les services de débogage du Common Language Runtime.  
  
### Pour corriger cette erreur  
  
-   Réparez l’installation de .NET Framework sur votre ordinateur.
