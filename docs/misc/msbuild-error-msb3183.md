# MSBuild Error MSB3183
**MSB3183 : La référence '\<assembly\>' est un assembly d'interopérabilité nécessitant une confiance totale.**  
  
 Cet avertissement est généré lorsqu'un assembly d'interopérabilité figure parmi les références lors de la génération du manifeste pour une application d'un niveau de confiance partiel.
