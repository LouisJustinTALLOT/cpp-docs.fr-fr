# MSBuild Error MSB3123
**MSB3123 : Le nombre d'associations de fichiers dépasse la limite de \<limite\>.**  
  
 Seul un nombre fixe d'extensions de nom de fichier associés est autorisé pour chaque fichier.  Cette erreur se produit lorsque vous tentez d'associer plus d'extensions de nom de fichier que le nombre autorisé pour le système d'exploitation cible.  
  
### Pour corriger cette erreur  
  
-   Limitez le nombre d'associations de fichier au nombre spécifié.
