# Erreur du compilateur CS1565
Options spécifiées en conflit : fichier de ressources Win32 ; icône Win32  
  
 Il n’est pas permis de spécifier l’option de compilateur [\/win32res](../Topic/-win32res%20\(C%23%20Compiler%20Options\).md) et l’option de compilateur [\/win32ico](../Topic/-win32icon%20\(C%23%20Compiler%20Options\).md) dans une même compilation.
