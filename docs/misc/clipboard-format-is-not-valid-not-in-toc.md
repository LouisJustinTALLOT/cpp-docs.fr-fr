# Clipboard format is not valid.
Le Presse\-papiers n'accepte que les données ayant un certain format spécifié.  Cette erreur se produit généralement lorsque vous essayez de placer des données d'un composant dans le Presse\-papiers, mais que les données ont un format incompatible avec le Presse\-papiers.  
  
### Pour corriger cette erreur  
  
1.  Consultez la documentation correspondant au composant pour déterminer s'il est possible d'utiliser le Presse\-papiers pour transférer des données à partir du composant.  
  
     \- ou \-  
  
2.  Sélectionnez un autre élément et recommencez.
