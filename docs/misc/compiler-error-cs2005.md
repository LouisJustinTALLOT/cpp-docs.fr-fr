# Erreur du compilateur CS2005
Spécification de fichier manquante pour l’option 'option'  
  
 Une [option du compilateur](../Topic/C%23%20Compiler%20Options.md) n’a été spécifiée que partiellement.  
  
 Par exemple, lorsque vous utilisez [\/recurse](../Topic/-recurse%20\(C%23%20Compiler%20Options\).md), vous devez spécifier le fichier à rechercher : **\/recurse:***nom\_fichier***.cs**.  
  
## Exemple  
 L’exemple suivant génère l’erreur CS2005.  
  
```  
// CS2005.cs // compile with: /recurse: // CS2005 expected class x { public static void Main() {} }  
```
