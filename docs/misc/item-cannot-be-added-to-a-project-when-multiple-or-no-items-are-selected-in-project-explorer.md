# Item cannot be added to a project when multiple or no items are selected in Project Explorer.
Cette erreur se produit généralement quand aucun élément n'a été sélectionné dans l'Explorateur de solutions ou que plusieurs éléments ont été sélectionnés.  
  
### Pour corriger cette erreur  
  
1.  Ne sélectionnez qu'un seul élément dans l'Explorateur de solutions.
