# Erreur irr&#233;cup&#233;rable RC1206 du compilateur de ressources
La page de codes spécifiée à la ligne de commande n’existe pas dans le Registre  
  
 L’option de spécification de page de codes \(\/c\) a été suivie d’une page de codes dont l’emplacement n’est pas spécifié dans le Registre. Exécutez SETUP pour installer la page de codes spécifiée.
