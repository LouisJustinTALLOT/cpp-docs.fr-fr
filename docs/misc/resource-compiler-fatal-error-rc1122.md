# Erreur irr&#233;cup&#233;rable RC1122 du compilateur de ressources
Erreur d’E\/S lors de l’écriture du fichier  
  
 Le compilateur de ressources n’a pas pu écrire dans un fichier.  
  
### Pour corriger en vérifiant les causes possibles suivantes  
  
1.  Espace disque insuffisant. L’espace libre doit être égal à environ deux fois la taille du fichier exécutable que vous créez.  
  
2.  Volume en lecture seule.  
  
3.  Secteur défectueux.  
  
4.  Violation de partage.
