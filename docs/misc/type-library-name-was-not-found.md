# Type library &lt;name&gt; was not found.
Visual Studio ne peut pas s'exécuter sans les bibliothèques de types DTE.olb et vsext.tlb.  Cette erreur se produit généralement lorsque ces deux fichiers ont été supprimés ou désinstallés.  
  
### Pour corriger cette erreur  
  
1.  Réinstallez Visual Studio.
