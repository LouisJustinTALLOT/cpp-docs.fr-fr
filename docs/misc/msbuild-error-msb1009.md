# MSBuild Error MSB1009
**Le fichier projet n'existe pas.**  
  
 Ce fichier projet ne figure pas dans le répertoire spécifié ou, si aucun répertoire n'a été spécifié, il ne figure pas dans le répertoire de travail actuel.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez l'orthographe du nom de fichier et du chemin d'accès.
