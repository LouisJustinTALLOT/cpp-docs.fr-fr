# Add-in &lt;name&gt; cannot be loaded. Remove it from the Add-in Manager?
Le complément sélectionné est introuvable.  Cette erreur se produit généralement lorsqu'une DLL ou un autre composant requis du complément est manquant.  
  
### Pour corriger cette erreur  
  
1.  Réinstallez le complément.
