# Erreur du compilateur CS0185
'type' n’est pas un type référence requis par l’instruction lock  
  
 L’instruction [lock](../Topic/lock%20Statement%20\(C%23%20Reference\).md) ne peut évaluer que des types référence. Pour plus d’informations, consultez [Synchronisation des threads](../Topic/Thread%20Synchronization%20\(C%23%20and%20Visual%20Basic\).md) et [Types référence](../Topic/Reference%20Types%20\(C%23%20Reference\).md).  
  
## Exemple  
 L’exemple suivant génère l’erreur CS0185 :  
  
```  
// CS0185.cs public class MainClass { public static void Main () { lock (1)   // CS0185 // try the following lines instead // MainClass x = new MainClass(); // lock(x) { } } }  
```
