# D&#233;pannage des exceptions&#160;: System.IdentityModel.Tokens.SecurityTokenValidationException
Une exception <xref:System.IdentityModel.Tokens.SecurityTokenValidationException> est renvoyée lorsqu'un jeton de sécurité reçu n'est pas valide. L'exception est généralement renvoyée par la méthode <xref:System.IdentityModel.Selectors.SecurityTokenAuthenticator.ValidateToken%2A>.
