# MSBuild Error MSB2006
**Le fichier projet ne contient pas l'élément racine \<{0}\>.**  
  
 Le nom de l'élément racine n'est pas orthographié correctement ou n'est pas reconnu par [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)].  
  
### Pour corriger cette erreur  
  
-   Vérifiez l'orthographe du nom de l'élément.  
  
-   Vérifiez si le fichier projet a été modifié ou endommagé.  S'il a été modifié ou endommagé, ouvrez le projet dans la version de [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] avec laquelle il a été créé, enregistrez\-le, puis essayez à nouveau de le convertir.
