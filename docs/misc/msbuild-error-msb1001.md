# MSBuild Error MSB1001
**Commutateur inconnu.**  
  
 Un commutateur inconnu a été spécifié sur la ligne de commande.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez l'orthographe du commutateur.  Pour afficher la liste des commutateurs valides, tapez `msbuild /help`.
