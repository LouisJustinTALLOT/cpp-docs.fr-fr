# Erreur du compilateur CS0170
Utilisation d’un champ potentiellement non assigné 'field'  
  
 Un champ dans une structure a été utilisé sans être au préalable initialisé. Pour résoudre ce problème, déterminez d’abord le champ qui a été initialisé, puis initialisez\-le avant d’essayer d’y accéder. Pour plus d’informations sur l’initialisation de structures, consultez [Structures](../Topic/Structs%20\(C%23%20Programming%20Guide\).md) et [Utilisation de structs](../Topic/Using%20Structs%20\(C%23%20Programming%20Guide\).md).  
  
 L’exemple suivant génère l’erreur CS0170 :  
  
```  
// CS0170.cs public struct error { public int i; } public class MyClass { public static void Main() { error e; // uncomment the next line to resolve this error // e.i = 0; System.Console.WriteLine( e.i );   // CS0170 because //e.i was never assigned } }  
```
