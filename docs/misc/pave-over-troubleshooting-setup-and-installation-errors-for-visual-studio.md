# Informations suppl&#233;mentaires sur les erreurs d&#39;installation et de configuration
Utilisez les ressources suivantes pour obtenir de l'aide concernant le dépannage des problèmes de configuration et d'installation de [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
-   Forum [Installation et configuration de Visual Studio](http://go.microsoft.com/fwlink/?LinkID=151190).  Examinez les questions et les réponses des autres dans la communauté Visual Studio.  Si vous ne trouvez pas ce dont vous avez besoin, posez vos propres questions.
