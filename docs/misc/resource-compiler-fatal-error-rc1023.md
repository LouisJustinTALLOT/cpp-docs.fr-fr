# Erreur irr&#233;cup&#233;rable RC1023 du compilateur de ressources
impossible d’ouvrir le fichier source 'nom\_fichier'  
  
 Le fichier spécifié n’existe pas, n’a pas pu être ouvert ou est introuvable.  
  
 Vérifiez que les paramètres d’environnement sont valides et que le chemin du fichier est spécifié et correct.  
  
 Si cette erreur s’affiche sans message d’erreur, cela signifie que le compilateur de ressources n’a plus assez de descripteurs de fichiers.
