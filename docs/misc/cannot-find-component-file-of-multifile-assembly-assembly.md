# Cannot find component &#39;file&#39; of multifile assembly &#39;assembly&#39;
Un fichier d'un assembly multifichier n'a pas pu être trouvé.  Un assembly multifichier fait référence à un ou plusieurs fichiers, comme les modules ou les fichiers de ressources.  
  
 L'assembly en cours de copie peut être une référence directe \(consultez le nœud **References** dans l'Explorateur de solutions\) ou il peut s'agir d'un assembly qui constitue une dépendance d'une référence directe.  
  
 **Pour corriger cette erreur**  
  
-   Réinstallez le composant.  
  
     Il se peut que le projet soit généré, mais qu'il ne s'exécute pas.
