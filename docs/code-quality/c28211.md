# <a name="c28211"></a>C28211

> AVERTISSEMENT C28211 : nom de contexte statique attendu pour SAL_context

Cet avertissement indique que l’opérande de l’annotation de `_Static_context_` doit être le nom d’un contexte défini par l’outil. Cet avertissement ne doit jamais se produire normalement.
