# <a name="c28232"></a>C28232

> AVERTISSEMENT C28232 : \_pré\_, \_\_de publication ou \_Deref\_ non appliqué à une annotation

Cet avertissement indique qu’un opérateur `_Pre_`, `_Post_`ou `_Deref_` apparaît dans une expression d’annotation sans une annotation fonctionnelle ultérieure ; le modificateur a été ignoré, mais cela indique une annotation mal codée.
