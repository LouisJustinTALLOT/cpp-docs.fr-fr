# <a name="c28204"></a>C28204

> AVERTISSEMENT C28204 : \<fonction >: seule une de cette surcharge et celle à \<nom de fichier > (\<ligne >) sont annotées pour \<paramName >: les deux ou aucune des deux doivent être annotées.

Cet avertissement est signalé en cas d’erreur dans les annotations.
