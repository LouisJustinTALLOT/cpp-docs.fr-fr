# <a name="c28219"></a>C28219

> AVERTISSEMENT C28219 : membre de l’énumération attendu pour l’annotation le paramètre dans l’annotation

Un paramètre d’une annotation est supposé être un membre du **`enum`** type nommé et un autre symbole a été rencontré ; utilisez un membre de ce **`enum`** type. Cela indique généralement une macro d’annotation mal codée.
