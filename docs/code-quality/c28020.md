# <a name="c28020"></a>C28020

> AVERTISSEMENT C28020 : l’expression \<expr> n’est pas vraie pour cet appel

Cet avertissement est signalé lorsque l' \_ expression satisfaite \_ n’est pas remplie. Souvent, cela indique un paramètre incorrect.

Si cela se produit sur une déclaration de fonction, les annotations indiquent une condition impossible.
