# <a name="c28209"></a>C28209

> AVERTISSEMENT C28209 : la déclaration pour le symbole a une déclaration en conflit

Cet avertissement indique une déclaration d’annotation mal construite. Cet avertissement ne doit jamais se produire normalement.
