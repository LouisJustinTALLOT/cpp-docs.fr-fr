# <a name="c28224"></a>C28224

> AVERTISSEMENT C28224 : l’annotation requiert des paramètres

Cet avertissement indique que l’annotation nommée est utilisée sans paramètres et qu’au moins un paramètre est requis. Cela ne doit pas être possible si les macros standard sont utilisées pour les annotations ; Cet avertissement n’est pas censé être affiché dans un usage normal.
