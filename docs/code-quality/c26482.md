# <a name="c26482-no_dynamic_array_indexing"></a>C26482 NO_DYNAMIC_ARRAY_INDEXING

Indexez uniquement les tableaux à l’aide d’expressions constantes.

## <a name="see-also"></a>Voir aussi

[Limites de C++ Core Guidelines. 2](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-bounds)

## <a name="example"></a>Exemple

```cpp
int getSomeIndex();

void function(int* p, int count)
{
    p[getSomeIndex()] = 0; // C26482, Only index into arrays using constant expressions
}
```
