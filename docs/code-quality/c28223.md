# <a name="c28223"></a>C28223

> AVERTISSEMENT C28223 : jeton/identificateur attendu introuvable pour l’annotation, paramètre

Cet avertissement indique qu’un paramètre d’une annotation est supposé être un identificateur et qu’un autre symbole a été rencontré. Cela indique généralement une macro d’annotation mal codée. L’utilisation de C ou C++ de mots clés à cette position provoquera cette erreur.
