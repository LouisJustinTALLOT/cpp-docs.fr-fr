# <a name="c28223"></a>C28223

> AVERTISSEMENT C28223 : jeton/identificateur attendu introuvable pour l’annotation, paramètre

Cet avertissement indique qu’un paramètre d’une annotation est supposé être un identificateur et qu’un autre symbole a été rencontré. Cela indique généralement une macro d’annotation mal codée. L’utilisation de mots clés C ou C++ à cette position provoquera cette erreur.
