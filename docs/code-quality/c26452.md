# <a name="arithmetic-overflow-left-shift-count-is-negative-or-greater-than-or-equal-to-the-operand-size-which-is-undefined-behavior"></a>Débordement arithmétique : Le nombre de quarts de travail de gauche est négatif ou supérieur ou égal à la taille de l’opéra qui est un comportement indéfini

Cet avertissement indique que le nombre de quarts est négatif ou supérieur ou égal au nombre de bits de l’opéra décalé, ce qui entraîne un comportement indéfini.
Remarque : C4293 est un contrôle similaire dans le compilateur Microsoft C.

## <a name="example"></a>Exemple

```cpp
unsigned __int64 combine(unsigned lo, unsigned hi)
{
  return (hi << 32) | lo; // C26252 here
}
```

Pour corriger cet avertissement, utilisez le code suivant :

```cpp
unsigned __int64 combine(unsigned lo, unsigned hi)
{
  return ((unsigned __int64)hi << 32) | lo; // OK
}
```
