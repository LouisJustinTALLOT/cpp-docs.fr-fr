# <a name="c28283"></a>C28283

> AVERTISSEMENT C28283 : pour le symbole, la spécification de taille spécifiée n’est pas encore prise en charge

L’avertissement indique que l’annotation « Sentinel » de la spécification de taille a reçu une valeur différente de zéro. Fondamentalement, l’appelant a tenté de dire que la chaîne se termine par un caractère autre que le zéro binaire.
