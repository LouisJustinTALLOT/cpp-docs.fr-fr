# <a name="c28160"></a>C28160

> AVERTISSEMENT C28160 : annotation d’erreur

Cet avertissement est signalé lorsqu’une annotation de `__drv_error` a été rencontrée. Cette annotation est utilisée pour signaler les pratiques de codage qui doivent être corrigées et peut être utilisée avec une annotation `__drv_when` pour indiquer des combinaisons de paramètres spécifiques.
