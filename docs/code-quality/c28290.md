# <a name="c28290"></a>C28290

> AVERTISSEMENT C28290 : l’annotation pour la fonction contient plus d’externes que le nombre réel de paramètres

L’outil d’analyse du code signale cet avertissement lorsque l’annotation pour la fonction contient plus d’externes que le nombre réel de paramètres.
