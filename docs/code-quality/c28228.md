# <a name="c28228"></a>C28228

> AVERTISSEMENT C28228 : paramètre d’annotation : impossible d’utiliser le type dans les annotations

Cet avertissement indique qu’un paramètre est de type non pris en charge. Les annotations ne peuvent utiliser qu’un ensemble limité de types comme paramètres. Cela ne doit pas être possible si les macros standard sont utilisées pour les annotations ; Cet avertissement n’est pas censé être affiché dans un usage classique.
