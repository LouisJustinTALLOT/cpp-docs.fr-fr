# <a name="c28305"></a>C28305

> AVERTISSEMENT C28305 : une erreur s’est produite lors de l’analyse d' \<jeton > a été découvert.

Cet avertissement est signalé lorsque l’expression contenant le jeton spécifié est incorrecte.
