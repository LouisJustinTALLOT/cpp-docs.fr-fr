# <a name="c28241"></a>C28241

> AVERTISSEMENT C28241 : l’annotation pour la fonction sur le paramètre n’est pas reconnue

Un nom d’annotation non reconnu a été utilisé. Cela ne doit pas être possible si les macros standard sont utilisées pour les annotations ; Cet avertissement n’est pas censé être affiché dans un usage classique.
