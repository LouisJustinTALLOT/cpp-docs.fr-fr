# <a name="arithmetic-overflow-operator-operation-produces-a-negative-unsigned-result-at-compile-time"></a>Débordement arithmétique : l’opération « % d’opérateur » produit un résultat négatif non signé au moment de la compilation

  Cet avertissement indique que l’opération de soustraction produit un résultat négatif qui a été évalué dans un contexte non signé. Cela peut entraîner des débordements imprévus.

## <a name="example"></a>Exemple

```cpp
unsigned int negativeunsigned()
{
    const unsigned int x = 1u - 2u; // C26454 reported here
    return x;
}
```

Pour corriger cet avertissement, utilisez le code suivant :

```cpp
unsigned int negativeunsigned()
{
    const unsigned int x = 4294967295; // OK
    return x;
}
```
