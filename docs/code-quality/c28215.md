# <a name="c28215"></a>C28215

> AVERTISSEMENT C28215 : le typefix ne peut pas être appliqué à un paramètre qui a déjà un typefix

L’application `__typefix` d’une annotation à un paramètre qui a déjà cette annotation est une erreur. Les `__typefix` annotations sont utilisées uniquement dans quelques cas spéciaux et cet avertissement n’est pas censé être affiché dans un usage normal.
