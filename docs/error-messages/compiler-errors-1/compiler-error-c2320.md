# <a name="compiler-error-c2320"></a>Erreur du compilateur C2320

> ' : 'attendu pour suivre le spécificateur d’accès'*spécificateur*'

Le mot clé **`public`** , **`protected`** ou **`private`** doit être suivi d’un signe deux-points.
