# <a name="fatal-error-c1603"></a>Erreur irrécupérable C1603

cible de branche assembly inline dépasse de 'nombre' octets

La distance calculée entre une instruction JCXZ ou JECXZ et l’étiquette cible spécifiée était supérieure à 128 octets. Mettre à jour votre code afin que l’étiquette est plus proche de l’instruction.
