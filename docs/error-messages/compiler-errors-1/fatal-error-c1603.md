# <a name="fatal-error-c1603"></a>Erreur irrécupérable C1603

cible de branche d’assembly inline en dehors de la plage par’nombre’octets

La distance calculée entre une instruction JCXZ ou JECXZ et son étiquette cible spécifiée est supérieure à 128 octets. Mettez à jour votre code afin que l’étiquette soit plus proche de l’instruction.
