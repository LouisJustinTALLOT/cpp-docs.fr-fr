# Erreur du compilateur C2172
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : le paramètre réel n’est pas un pointeur : paramètre number  
  
 Le paramètre `number` n’est pas un pointeur. La fonction attend un pointeur.
