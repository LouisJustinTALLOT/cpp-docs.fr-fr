# Erreur irr&#233;cup&#233;rable C1003
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

le nombre d’erreurs est supérieur à une valeur ; arrêt de la compilation  
  
 Les erreurs du programme sont trop nombreuses pour permettre sa récupération. Le compilateur doit terminer son exécution.
