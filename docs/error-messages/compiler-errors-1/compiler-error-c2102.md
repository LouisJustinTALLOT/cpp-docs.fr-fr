# Erreur du compilateur C2102
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'&' nécessite une l\-value  
  
 L’opérateur d’adresse \(`&`\) doit avoir une l\-value comme opérande.
