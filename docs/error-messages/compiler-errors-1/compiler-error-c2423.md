# Erreur du compilateur C2423
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'nombre' : échelle non conforme  
  
 Le code assembleur inline utilise un numéro autre que 1, 2, 4 ou 8 pour échelonner un registre.  
  
 L'exemple suivant génère l'erreur C2423 :  
  
```  
// C2423.cpp  
// processor: x86  
int main() {  
   _asm {  
      lea EAX, [EAX*3]   // C2423  
      lea EAX, [EAX+EAX*2]   // OK  
   }  
}  
```
