# <a name="compiler-error-c2163"></a>Erreur du compilateur C2163

'fonction' : non disponible comme fonction intrinsèque

Un `intrinsic` `function` pragma ou répertorie une fonction non disponible sous une forme intrinsèque. Par exemple, certaines fonctions intrinsèques ne sont pas disponibles lors de la compilation d’un programme qui utilise la programmation/CLR.
