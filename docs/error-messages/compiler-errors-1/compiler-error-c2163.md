# Erreur du compilateur C2163
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : non disponible comme fonction intrinsèque  
  
 Un pragma `intrinsic` ou `function` inclut dans sa liste une fonction non disponible en forme intrinsèque.  Certaines formes intrinsèques ne sont par exemple pas disponibles lors de la compilation d'un programme qui utilise la programmation \/clr.
