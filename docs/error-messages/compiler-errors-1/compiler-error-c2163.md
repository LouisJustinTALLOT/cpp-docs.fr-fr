# <a name="compiler-error-c2163"></a>Erreur du compilateur C2163

'fonction' : non disponible comme fonction intrinsèque

Un `intrinsic` ou `function` pragma répertorie une fonction non disponible au format intrinsèque. Par exemple, certaines formes intrinsèques ne sont pas disponibles lors de la compilation d’un programme qui utilise la programmation /clr.
