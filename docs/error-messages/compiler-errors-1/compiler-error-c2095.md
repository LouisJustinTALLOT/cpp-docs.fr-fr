# <a name="compiler-error-c2095"></a>Erreur du compilateur C2095
'fonction' : le paramètre réel a 'void 'comme type : paramètre 'nombre'  
  
 Le paramètre passé à la fonction est de type `void`, ce qui n’est pas autorisé. Utilisez un pointeur vers void ( `void *`) à la place.  
  
 Le `number` indique le paramètre `void`.
