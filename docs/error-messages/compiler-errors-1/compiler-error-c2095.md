# <a name="compiler-error-c2095"></a>Erreur du compilateur C2095

'fonction' : le paramètre réel a le type 'void' : paramètre 'nombre'

Le paramètre passé à la fonction est de type `void`, ce qui n’est pas autorisé. Utiliser un pointeur vers void ( `void *`) à la place.

Le `number` indique le paramètre `void`.
