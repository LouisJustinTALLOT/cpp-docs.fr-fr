# <a name="compiler-error-c2095"></a>Erreur du compilateur C2095

'fonction' : le paramètre réel a le type’void' : paramètre’Number'

Le paramètre passé à la fonction est de type **`void`** , ce qui n’est pas autorisé. Utilisez à la place un pointeur vers void ( `void *` ).

`number`Indique le paramètre **`void`** .
