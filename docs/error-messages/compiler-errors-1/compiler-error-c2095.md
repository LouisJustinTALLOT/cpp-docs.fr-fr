# Erreur du compilateur C2095
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : le paramètre actuel a 'void' comme type : paramètre 'numéro'  
  
 Le paramètre passé à la fonction est du type `void`, ce qui n'est pas autorisé.  Utilisez plutôt un pointeur vers void \(`void *`\).  
  
 Le `number` indique le paramètre `void`.
