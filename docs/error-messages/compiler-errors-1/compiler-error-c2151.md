# <a name="compiler-error-c2151"></a>Erreur du compilateur C2151
plusieurs attributs de langue  
  
 Une fonction a plusieurs mots clés ( `__cdecl`, `__stdcall`, ou `__fastcall`) en spécifiant une convention d’appel.
