# <a name="compiler-error-c2151"></a>Erreur du compilateur C2151

plus d’un attribut de langage

Une fonction a plus d’un mot clé ( `__cdecl`, `__stdcall`, ou `__fastcall`) en spécifiant une convention d’appel.
