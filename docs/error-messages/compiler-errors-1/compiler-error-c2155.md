# <a name="compiler-error-c2155"></a>Erreur du compilateur C2155
' ?' : gauche opérande attendu arithmétique non valide ou le type de pointeur  
  
 Une expression sur le côté gauche de `?` ne peut pas être comparée à zéro. Vous devez utiliser une expression arithmétique ou pointeur qui peut être comparée à zéro.
