# Erreur du compilateur C2155
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'?' : opérande gauche non valide, type arithmétique ou pointeur attendu  
  
 Une expression du côté gauche du `?` ne peut pas être comparée à zéro.  Vous devez utiliser une expression arithmétique ou de pointeur pouvant être comparée à zéro.
