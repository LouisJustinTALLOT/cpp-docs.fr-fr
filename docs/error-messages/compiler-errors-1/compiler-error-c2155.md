# <a name="compiler-error-c2155"></a>Erreur du compilateur C2155

' ? ' : opérande gauche non valide, type arithmétique ou pointeur ATTENDU

Une expression sur le côté gauche de `?` ne peut pas être comparée à zéro. Vous devez utiliser une expression arithmétique ou de pointeur qui peut être comparée à zéro.
