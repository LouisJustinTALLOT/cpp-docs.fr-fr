# <a name="compiler-error-c2224"></a>Erreur du compilateur C2224

gauche de '.identifier' doit avoir un type struct/union

L’opérande à gauche de l’opérateur point (.) n’est pas une classe, une structure ou une union.

Cette erreur peut être provoquée par un opérande gauche est une variable non définie (par conséquent, tapez `int`).
