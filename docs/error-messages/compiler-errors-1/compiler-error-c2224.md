# Erreur du compilateur C2224
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la partie gauche de '.identificateur' doit avoir un type struct\/union  
  
 L'opérande à gauche du point \(.\) n'est pas une classe, une structure ou une union.  
  
 Cette erreur peut être provoquée par un opérande gauche qui est une variable indéfinie \(par conséquent, de type `int`\).
