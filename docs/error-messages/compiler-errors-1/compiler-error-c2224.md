# <a name="compiler-error-c2224"></a>Erreur du compilateur C2224

la partie gauche de'. identifier’doit avoir un type struct/union

L’opérande à gauche du point (.) n’est pas une classe, une structure ou une Union.

Cette erreur peut être causée par un opérande gauche qui est une variable non définie (par conséquent, tapez `int`).
