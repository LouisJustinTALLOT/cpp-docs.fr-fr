# <a name="compiler-error-c2224"></a>Erreur du compilateur C2224
partie gauche de '.identifier' doit avoir un type struct/union  
  
 L’opérande à gauche de l’opérateur point (.) n’est pas une classe, structure ou union.  
  
 Cette erreur peut résulter d’un opérande de gauche est une variable indéfinie (par conséquent, tapez `int`).
