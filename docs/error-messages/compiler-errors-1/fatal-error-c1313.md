# Erreur irr&#233;cup&#233;rable C1313
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

limite du compilateur : les blocs type ne doivent pas être imbriqués au\-delà de 'nombre' niveaux de profondeur.  
  
 Les blocs de gestion des exceptions \(ou de gestion structurée des exceptions\) ont été imbriqués trop profondément.  Simplifiez votre code.
