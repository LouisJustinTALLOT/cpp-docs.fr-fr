# <a name="fatal-error-c1313"></a>Erreur irrécupérable C1313

limite du compilateur : les blocs de type ne peuvent pas être imbriqués plus près que les niveaux de nombre

Les blocs de gestion des exceptions (ou gestion structurée des exceptions) étaient trop profondément imbriqués.  Simplifiez votre code.
