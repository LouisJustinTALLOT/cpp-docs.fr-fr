# Erreur du compilateur C2408
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Type non conforme sur l’opérateur PTR dans 'context'  
  
 Le premier paramètre de l’opérateur `PTR` n’est pas une spécification de type conforme.
