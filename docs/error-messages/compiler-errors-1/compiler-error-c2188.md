# Erreur du compilateur C2188
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'nombre' : trop grand pour un caractère large  
  
 Le nombre dépasse la limite de taille pour le type de caractère large. Choisissez un type plus grand.
