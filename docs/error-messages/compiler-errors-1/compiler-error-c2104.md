# <a name="compiler-error-c2104"></a>Erreur du compilateur C2104

' &' sur un champ de bits ignoré

Vous ne pouvez pas prendre l’adresse d’un champ de bits.

L’exemple suivant génère l’erreur C2104 :

```
// C2104.cpp
struct X {
   int sb : 1;
};

int main() {
   X x;
   &x.sb;   // C2104
   x.sb;   // OK
}
```
