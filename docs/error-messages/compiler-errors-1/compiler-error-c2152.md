# Erreur du compilateur C2152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : pointeurs vers des fonctions avec des attributs différents  
  
 Un pointeur vers une fonction avec une convention d'appel \(`__cdecl`, `__stdcall` ou `__fastcall`\) est assigné à un pointeur vers une fonction avec une autre convention d'appel.
