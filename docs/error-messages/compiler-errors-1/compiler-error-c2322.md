# Erreur du compilateur C2322
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Identificateur' : adresse de dllimport 'dllimport' non static  
  
 Une valeur non static est donnée comme adresse d’une fonction déclarée avec `dllimport`.
