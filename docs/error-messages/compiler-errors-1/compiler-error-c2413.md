# Erreur du compilateur C2413
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'jeton' : taille d'alignement non conforme  
  
 La taille utilisée avec la directive `ALIGN` est manquante ou en dehors de la plage valide.
