# Erreur irr&#233;cup&#233;rable C1075
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

le jeton gauche n’a pas de correspondance à la fin du fichier  
  
 Le compilateur attendait un *jeton* correspondant avant d’atteindre la fin du fichier.  
  
 Cette erreur peut être provoquée par une parenthèse ou une accolade sans correspondance \(ou tout autre caractère apparié\).
