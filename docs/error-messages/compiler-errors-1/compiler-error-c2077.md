# <a name="compiler-error-c2077"></a>Erreur du compilateur C2077

initialiseur de champ non scalaire’identificateur'

Vous avez essayé d’initialiser un champ de bits avec un non scalaire (struct, Union, tableau ou classe). Utilisez un entier ou une valeur à virgule flottante.
