# Erreur du compilateur C2077
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

initialiseur de champ non scalaire 'identificateur'  
  
 Vous avez tenté d'initialiser un champ de bits avec une variable non scalaire \(struct, union, tableau ou classe\).  Utilisez une valeur entière ou à virgule flottante.
