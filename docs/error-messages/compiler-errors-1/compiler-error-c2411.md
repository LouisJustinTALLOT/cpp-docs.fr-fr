# Erreur du compilateur C2411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : membre struct\/union non conforme dans 'contexte'  
  
### Pour résoudre le problème en vérifiant les causes possibles suivantes  
  
1.  L'identificateur n'est pas un membre d'une structure ou d'une union visible dans ce contexte.  
  
2.  L'identificateur n'est pas un membre de la structure ou de l'union spécifiée avec l'opérateur de sélection des membres \(.\).
