# <a name="compiler-error-c2411"></a>Erreur du compilateur C2411

'identificateur' : membre struct/union non conforme dans’Context'

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes

1. L’identificateur n’est pas un membre d’une structure ou d’une Union visible dans ce contexte.

1. L’identificateur n’est pas un membre de la structure ou de l’Union spécifiée avec l’opérateur de sélection de membres (.).
