# <a name="compiler-error-c2411"></a>Erreur du compilateur C2411
'identificateur' : membre struct/union non conforme dans 'contexte'  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes  
  
1.  L’identificateur n’est pas un membre d’une structure visible ou d’une union dans ce contexte.  
  
2.  L’identificateur n’est pas un membre de la structure ou l’union spécifiée avec l’opérateur de sélection de membres (.).
