# Erreur du compilateur C2173
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : le paramètre réel n’est pas un pointeur : paramètre 'numéro1', liste de paramètres 'numéro2'  
  
 Le paramètre `number1` passé à la liste de paramètres `number2` n’est pas un pointeur. La fonction attend un pointeur.
