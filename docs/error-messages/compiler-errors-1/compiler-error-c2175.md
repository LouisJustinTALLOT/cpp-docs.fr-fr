# <a name="compiler-error-c2175"></a>Erreur du compilateur C2175
'locale' : variable locale non valide  
  
 Les paramètres régionaux spécifiés ne sont pas valide. Consultez [chaînes de langues et pays/région](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) dans les *Run-Time Library Reference* pour les paramètres régionaux pris en charge.
