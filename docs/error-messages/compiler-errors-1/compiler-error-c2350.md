# Erreur du compilateur C2350
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' n’est pas un membre static  
  
 Les membres non static d’une classe ou d’une structure ne peuvent pas être définis.
