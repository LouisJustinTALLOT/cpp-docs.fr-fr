# Erreur du compilateur C2402
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : le registre doit être un index de 'contexte'  
  
 Le Registre utilisé dans un opérande de mémoire indirect doit être un Registre d’index dans ce contexte.
