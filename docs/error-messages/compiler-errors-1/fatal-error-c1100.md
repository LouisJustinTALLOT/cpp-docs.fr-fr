# <a name="fatal-error-c1100"></a>Erreur irrécupérable C1100

Impossible d’initialiser OLE : message d’erreur système

Le compilateur ne peut pas initialiser la bibliothèque COM (Component Object Model). Consultez [CoInitialize](/windows/win32/api/objbase/nf-objbase-coinitialize).
