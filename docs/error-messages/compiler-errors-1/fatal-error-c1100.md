# Erreur irr&#233;cup&#233;rable C1100
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossible d'initialiser OLE : message d'erreur système  
  
 Le compilateur ne peut pas initialiser la bibliothèque COM \(Component Object Model\).  Consultez [CoInitialize](http://msdn.microsoft.com/library/windows/desktop/ms678543).
