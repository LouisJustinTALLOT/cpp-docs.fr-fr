# Erreur du compilateur C2168
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : nombre de paramètres réels de fonction intrinsèque insuffisant  
  
 Une référence à une fonction `intrinsic` n’a pas assez de paramètres.
