# Erreur irr&#233;cup&#233;rable C1506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossible de récupérer l'erreur relative à la portée du bloc  
  
 Le bloc est trop grand pour être compilé.  
  
### Pour résoudre le problème en vérifiant les causes possibles suivantes  
  
1.  Accolades non appariées  
  
2.  Fonction ou classe anormalement grande
