# <a name="fatal-error-c1506"></a>Erreur irrécupérable C1506

impossible de récupérer l'erreur relative à la portée du bloc

Le bloc était trop grand pour être compilé.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes

1. Accolades incompatibles

1. Fonction ou classe inhabituellement volumineuse
