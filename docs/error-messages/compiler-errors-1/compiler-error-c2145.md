# Erreur du compilateur C2145
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erreur de syntaxe : absence de 'jeton' avant l’identificateur  
  
 Le compilateur attendait `token` et a trouvé un identificateur à la place.  
  
 Cette erreur peut être due à un point\-virgule manquant après la dernière déclaration d’un bloc.
