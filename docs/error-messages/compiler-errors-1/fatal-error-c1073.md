# Erreur irr&#233;cup&#233;rable C1073
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erreur interne liée à la compilation incrémentielle \(fichier compilateur 'NomFichier', ligne 'numéro'\)  
  
 Recompilez le fichier sans utiliser la compilation incrémentielle.
