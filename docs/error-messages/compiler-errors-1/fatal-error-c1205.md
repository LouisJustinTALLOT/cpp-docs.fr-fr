# Erreur irr&#233;cup&#233;rable C1205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Les génériques ne sont pas pris en charge par la version installée du runtime  
  
 La version du Common Language Runtime utilisée par le compilateur n’est pas une version du runtime prise en charge par le compilateur actuel.  Par exemple, la fonctionnalité de génériques exige un runtime qui correspond au compilateur.  
  
 Vous devrez peut\-être modifier la spécification du chemin.
