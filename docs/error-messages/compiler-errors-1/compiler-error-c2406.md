# <a name="compiler-error-c2406"></a>Erreur du compilateur C2406
'identifier' : nom non défini dans 'context'  
  
 Un identificateur non défini est utilisé avec le `SIZE`, `LENGTH`, ou un opérateur de sélection de membres (.).
