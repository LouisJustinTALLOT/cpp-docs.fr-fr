# <a name="compiler-error-c2405"></a>Erreur du compilateur C2405

Référence de préversion Short non conforme avec décalage

Les références anticipées ne doivent faire référence qu’à une étiquette. Un décalage supplémentaire ne peut pas être utilisé.
