# Erreur du compilateur C2405
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

référence au type short avec offset non conforme  
  
 Les références au type short doivent se rapporter à une étiquette uniquement.  Un offset supplémentaire n'est pas autorisé.
