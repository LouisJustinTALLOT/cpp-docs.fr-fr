# Erreur irr&#233;cup&#233;rable C1113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Échec de \#using sur 'fichier'  
  
 La directive [\#using](../../preprocessor/hash-using-directive-cpp.md) ne peut accepter qu'un fichier au format MSIL \(Microsoft Intermediate Language\).  L'option du compilateur [\/clr](../../build/reference/clr-common-language-runtime-compilation.md) permet de créer un fichier de sortie MSIL.  Les autres langages [!INCLUDE[vsprvs](../../assembler/masm/includes/vsprvs_md.md)] produisent également des fichiers MSIL.
