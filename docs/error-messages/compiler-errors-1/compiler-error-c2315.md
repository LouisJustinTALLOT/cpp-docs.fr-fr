# Erreur du compilateur C2315
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type1' : référence interceptée par 'type2' à la ligne 'numéro'  
  
 Le type d’exception est géré par un gestionnaire précédent. La référence de la deuxième interception a le type de la première.
