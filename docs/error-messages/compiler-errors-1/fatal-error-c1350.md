# <a name="fatal-error-c1350"></a>Erreur irrécupérable C1350

Erreur lors du chargement de la dll 'dll' : dll introuvable

Impossible de trouver la DLL qui prend en charge l’opération tentée.

Cela indique un problème avec votre installation, vous devez réinstaller la DLL à partir de votre CD du produit.
