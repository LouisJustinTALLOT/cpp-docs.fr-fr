# Erreur irr&#233;cup&#233;rable C1350
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erreur lors du chargement de la dll 'dll' : dll non trouvée  
  
 La DLL qui prend en charge l'opération n'a pas été trouvée.  
  
 Ceci indique un problème dans votre installation, vous devez réinstaller la DLL à partir du CD du produit.
