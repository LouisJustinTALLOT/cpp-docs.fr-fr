# <a name="fatal-error-c1099"></a>Erreur irrécupérable C1099

Fin de la compilation du moteur Modifier & Continuer

Modifier &amp; Continuer a chargé un fichier d’en-tête précompilé en préparation de la compilation des modifications du code, mais des actions ultérieures (telles que des modifications de code avant l’instruction `#include` d’en-tête précompilé ou l’arrêt du débogueur) ont empêché Modifier &amp; Continuer de terminer la compilation de ce processus. Aucune action n’est nécessaire pour corriger cette erreur.
