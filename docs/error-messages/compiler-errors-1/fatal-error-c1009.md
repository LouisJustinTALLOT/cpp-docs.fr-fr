# Erreur irr&#233;cup&#233;rable C1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

limite du compilateur : macros imbriquées trop profondément  
  
 Le compilateur a tenté de développer trop de macros en même temps.  Le compilateur est limité à 256 niveaux d'imbrication de macros.  Fractionnez les macros imbriquées en macros plus simples.
