# <a name="fatal-error-c1009"></a>Erreur irrécupérable C1009

limite du compilateur : macros imbriquées trop profondément

Le compilateur a essayé de développer trop de macros en même temps. Le compilateur a une limite de 256 niveaux de macros imbriquées. Fractionnez les macros imbriquées en macros plus simples.
