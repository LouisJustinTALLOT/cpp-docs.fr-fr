# Erreur du compilateur C2256
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

utilisation non conforme du spécificateur friend sur 'fonction'  
  
 Un destructeur ou constructeur ne peut pas être spécifié comme [friend](../../cpp/friend-cpp.md).  
  
 L'exemple suivant génère l'erreur C2256 :  
  
```  
// C2256.cpp  
// compile with: /c  
class C {  
public:  
   friend ~C();   // C2256  
   ~C();   // OK  
};  
```
