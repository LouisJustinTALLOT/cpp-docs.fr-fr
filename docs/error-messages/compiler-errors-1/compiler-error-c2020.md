# <a name="compiler-error-c2020"></a>Erreur du compilateur C2020
'membre' : redéfinition du membre 'classe'  
  
 Un membre hérité d’une classe de base ou une structure est redéfini. Les membres hérités ne peut pas être redéfinis, sauf si déclarée en tant que `virtual` dans la classe de base.
