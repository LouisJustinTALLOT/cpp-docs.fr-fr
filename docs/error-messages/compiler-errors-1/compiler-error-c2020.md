# Erreur du compilateur C2020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'membre' : redéfinition du membre 'classe'  
  
 Un membre hérité d'une classe de base ou d'une structure est redéfini.  Les membres hérités ne peuvent pas être redéfinis sauf s'ils sont déclarés comme `virtual` dans la classe de base.
