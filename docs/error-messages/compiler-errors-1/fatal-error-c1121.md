# Erreur irr&#233;cup&#233;rable C1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

échec de l'appel à CryptoAPI  
  
 Le compilateur a lancé un appel à CryptoAPI et l'appel a échoué.  Réinstallez Visual Studio et éventuellement le système d'exploitation.
