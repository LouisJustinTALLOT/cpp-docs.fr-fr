# Erreur du compilateur C2426
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'jeton' : opérateur non conforme dans 'contexte'  
  
 Le jeton ne peut pas servir d'opérateur dans ce contexte.  Les opérateurs d'index, par exemple, ne peuvent être imbriqués.
