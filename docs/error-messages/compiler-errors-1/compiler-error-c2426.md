# <a name="compiler-error-c2426"></a>Erreur du compilateur C2426
'jeton' : opérateur non conforme dans 'contexte'  
  
 Le jeton ne peut pas être utilisé comme opérateur dans ce contexte. Opérateurs d’index, par exemple, ne peut pas être imbriquées.
