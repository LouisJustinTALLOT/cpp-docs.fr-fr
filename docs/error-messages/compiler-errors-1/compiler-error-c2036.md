# Erreur du compilateur C2036
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : taille inconnue  
  
 Une opération sur `identifier` requiert la taille de l'objet de données, qui ne peut être déterminée.  
  
## Exemple  
 L'exemple suivant génère l'erreur C2036.  
  
```  
// C2036.c  
// a C program  
struct A* pA;  
struct B { int i; } *pB;  
int main() {  
   pA++;   // C2036, size of A not known  
   ((char*)pA)++;   // OK  
  
   pB++;   // OK  
}  
```  
  
## Exemple  
 L'exemple suivant génère l'erreur C2036.  
  
```  
// C2036_2.cpp  
// a C++ program  
struct A* pA;  
int main() {  
   pA++;   // C2036, size of A not known  
   ((char*&)pA)++;   // OK, if sizeof(A) == sizeof(char)  
}  
```
