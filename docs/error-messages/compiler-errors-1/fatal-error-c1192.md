# <a name="fatal-error-c1192"></a>Erreur irrécupérable C1192

\#échec de l’utilisation de’file'

Seul un fichier au format MSIL (Microsoft Intermediate Language) peut être passé à une directive [#using](../../preprocessor/hash-using-directive-cpp.md) . L’option du compilateur [/CLR](../../build/reference/clr-common-language-runtime-compilation.md) vous permet de créer un fichier de sortie MSIL. D’autres langages Visual Studio produisent également des fichiers MSIL.
