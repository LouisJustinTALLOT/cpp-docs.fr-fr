# <a name="compiler-error-c2044"></a>Erreur du compilateur C2044

instruction continue non conforme

Une instruction [continue](../../cpp/continue-statement-cpp.md) est valide uniquement dans **`do`** une **`for`** instruction, ou **`while`** .
