# Erreur du compilateur C2122
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : paramètre de prototype non conforme dans une liste de noms  
  
 Le paramètre n'est pas un type conforme.  C ANSI ne prend pas en charge les types définis par l'utilisateur.
