# Erreur irr&#233;cup&#233;rable C1601
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opcode d'assembly inline non pris en charge  
  
 Cette erreur indique un problème de compatibilité dans les fichiers .exe du compilateur pouvant résulter d'une installation incomplète.  Par exemple, il se peut que vous ayez installé un service pack mais pas le Processor Pack.  Installez tous les produits requis.
