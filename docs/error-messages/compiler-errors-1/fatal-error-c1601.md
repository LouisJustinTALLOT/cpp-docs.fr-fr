# <a name="fatal-error-c1601"></a>Erreur irrécupérable C1601
opcode d'assembly inline non pris en charge  
  
 Cette erreur indique une incompatibilité dans les fichiers .exe du compilateur, qui pourrait avoir lieu en raison d’une installation incomplète. Par exemple, vous avez peut-être installé un service pack, mais pas le Pack de processeur. Installez tous les produits requis.
