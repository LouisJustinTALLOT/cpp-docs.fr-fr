# <a name="fatal-error-c1206"></a>Erreur irrécupérable C1206

Les données par appdomain ne sont pas prises en charge par la version installée du runtime

Certaines fonctionnalités, telles que les données de domaine par application, sont uniquement prises en charge par le Common Language Runtime qui prend en charge la fonctionnalité.

L’erreur C1206 indique que la version la plus récente du runtime n’est pas installée sur votre ordinateur. Installez la version du Common Language Runtime prévue pour votre compilateur.

Pour plus d'informations, voir [appdomain](../../cpp/appdomain.md) .
