# <a name="compiler-error-c2109"></a>Erreur du compilateur C2109

indice requiert de type tableau ou pointeur

L’indice a été utilisé sur une variable qui n’était pas un tableau.

L’exemple suivant génère C2109 :

```
// C2109.cpp
int main() {
   int a, b[10] = {0};
   a[0] = 1;   // C2109
   b[0] = 1;   // OK
}
```
