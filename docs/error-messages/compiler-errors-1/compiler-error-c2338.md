# Erreur du compilateur C2338
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' Atl Attribute Provider  
  
 L’attribut fait référence à un identificateur non déclaré. Cette erreur peut être associée à l’[erreur ATL2010 d’ATL Provider](http://msdn.microsoft.com/fr-fr/27a25924-b52b-4971-80d3-c3b3e9ffbe41).
