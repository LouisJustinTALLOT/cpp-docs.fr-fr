# <a name="compiler-error-c2170"></a>Erreur du compilateur C2170

'identifier' : non déclaré comme fonction, ne peut pas être intrinsèque

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes

1. `intrinsic`Le pragma est utilisé pour un élément autre qu’une fonction.

1. `intrinsic`Le pragma est utilisé pour une fonction sans forme intrinsèque.
