# <a name="compiler-error-c2223"></a>Erreur du compilateur C2223
la partie gauche de '-> identificateur' doit pointer vers struct/union  
  
 L’opérande à gauche de `->` n’est pas un pointeur vers une classe, structure ou union.  
  
 Cette erreur peut résulter d’un opérande de gauche est une variable indéfinie (par conséquent, tapez `int`).
