# <a name="compiler-error-c2223"></a>Erreur du compilateur C2223

la partie gauche de'->identifier’doit pointer vers struct/union

L’opérande à gauche de `->` n’est pas un pointeur vers une classe, structure ou union.

Cette erreur peut être causée par un opérande gauche qui est une variable non définie (par conséquent, type **`int`** ).
