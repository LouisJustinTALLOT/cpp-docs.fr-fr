# <a name="compiler-error-c2219"></a>Erreur du compilateur C2219

erreur de syntaxe : le qualificateur de type doit être après' * '

Le qualificateur de type ( **`const`** ou **`volatile`** ) apparaît là où il n’est pas autorisé.
