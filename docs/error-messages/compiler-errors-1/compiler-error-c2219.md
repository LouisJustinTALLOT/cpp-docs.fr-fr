# Erreur du compilateur C2219
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erreur de syntaxe : le qualificateur de type doit figurer après '\*'  
  
 Le qualificateur de type \(`const` ou `volatile`\) apparaît à un endroit non autorisé.
