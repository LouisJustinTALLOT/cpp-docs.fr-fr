# <a name="compiler-error-c2487"></a>Erreur du compilateur C2487

'identificateur' : un membre de la classe d’interface dll ne peut pas être déclaré avec une interface dll

Vous pouvez déclarer une classe entière, ou certains membres d’une classe d’interface non DLL, avec l’interface DLL. Vous ne pouvez pas déclarer une classe avec une interface DLL, puis déclarer un membre de cette classe avec une interface DLL.
