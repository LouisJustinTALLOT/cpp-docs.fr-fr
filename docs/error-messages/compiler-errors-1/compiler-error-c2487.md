# Erreur du compilateur C2487
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : le membre d'une classe d'interface dll ne peut pas être déclaré avec une interface dll  
  
 Vous pouvez déclarer une classe complète ou certains membres d'une classe d'interface non DLL avec une interface DLL.  Vous ne pouvez pas déclarer une classe avec l'interface DLL, puis déclarer un membre de cette classe avec l'interface DLL.
