# <a name="compiler-error-c2487"></a>Erreur du compilateur C2487
'identificateur' : membre d’une classe d’interface dll ne peut pas être déclaré avec une interface dll  
  
 Vous pouvez déclarer une classe complète ou certains membres d’une classe d’interface non DLL, avec une interface DLL. Vous ne peut pas déclarer une classe avec une interface DLL, puis de déclarer un membre de cette classe avec une interface DLL.
