# Erreur du compilateur C2022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'nombre' : trop grand pour un caractère  
  
 Le nombre octal suivant une barre oblique inverse \(\\\) dans une constante caractère ou chaîne est trop grand pour représenter un caractère.
