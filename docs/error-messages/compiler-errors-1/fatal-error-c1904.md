# <a name="fatal-error-c1904"></a>Erreur irrécupérable C1904
interaction de fournisseurs incorrecte : 'fichier'  
  
 Cette erreur indique l’échec d’un fournisseur d’attributs.  
  
 Notez les circonstances de l’erreur, essayez d’isoler le problème et de créer un cas de test reproductible, puis contactez [nous contacter](/visualstudio/ide/talk-to-us).
