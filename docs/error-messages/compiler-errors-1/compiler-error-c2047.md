# <a name="compiler-error-c2047"></a>Erreur du compilateur C2047

instruction default non conforme

Le mot clé **`default`** ne peut apparaître que dans une **`switch`** instruction.

L’exemple suivant génère l’erreur C2047 :

```cpp
// C2047.cpp
int main() {
   int i = 0;
   default:   // C2047
   switch(i) {
      case 0:
      break;
   }
}
```

Solution possible :

```cpp
// C2047b.cpp
int main() {
   int i = 0;
   switch(i) {
      case 0:
      break;
      default:
      break;
   }
}
```
