# <a name="fatal-error-c1007"></a>Erreur irrécupérable C1007

indicateur de chaîne non reconnu dans l’option

L’option de ligne de commande contient une chaîne non valide. Recherchez les erreurs dans la ligne de commande **CL** et la variable d’environnement.
