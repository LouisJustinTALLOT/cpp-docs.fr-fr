# <a name="compiler-error-c2444"></a>Erreur du compilateur C2444

'identificateur' : prototype ANSI utilisé, 'type' rencontré, '{' ou ';' attendu

Le prototype de fonction est suivi d’un type.

Cette erreur peut être due à un point-virgule ou une accolade manquant(e).
