# <a name="fatal-error-c1309"></a>Erreur irrécupérable C1309

Versions incompatibles de C2. DLL et PGODB\<version >. DLL

L’ensemble d’outils que vous utilisez pour créer et utiliser [optimisations guidées par profil](../../build/profile-guided-optimizations.md) contient des composants incompatibles.  Si vous ne pouvez pas résoudre cette erreur manuellement, réinstallez Visual C++.
