# Erreur irr&#233;cup&#233;rable C1309
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Versions incompatibles de C2.DLL et de PGODB\< version \>.DLL  
  
 L'ensemble d'outils que vous utilisez pour générer et utiliser des [optimisations guidées par profil](../../build/reference/profile-guided-optimizations.md) contient des composants incompatibles.  Si vous ne pouvez pas résoudre cette erreur manuellement, réinstallez Visual C\+\+.
