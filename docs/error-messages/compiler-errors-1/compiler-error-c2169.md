# Erreur du compilateur C2169
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : fonction intrinsèque, ne peut pas être définie  
  
 Une définition de fonction apparaît pour une fonction déjà déclarée `intrinsic`.
