# Erreur du compilateur C2302
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la gauche de '.~identifier' doit être de type classe\/struct\/union  
  
 L’expression située à gauche de l’opérateur point \(.\) n’est pas une classe, une structure ou une union.
