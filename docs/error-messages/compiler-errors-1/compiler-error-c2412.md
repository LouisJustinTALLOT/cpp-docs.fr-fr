# Erreur du compilateur C2412
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'étiquette' : redéfinition d'une étiquette \(sans distinction minuscules\/majuscules\)  
  
 L'étiquette est définie plusieurs fois dans la fonction en cours.  Modifiez l'orthographe de l'étiquette et ses références.
