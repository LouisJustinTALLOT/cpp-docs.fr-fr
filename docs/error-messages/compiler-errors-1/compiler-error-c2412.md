# <a name="compiler-error-c2412"></a>Erreur du compilateur C2412

'label' : étiquette non sensible à la casse redéfinie

L’étiquette est définie plus d’une fois dans la fonction active. Modifiez l’orthographe de l’étiquette et ses références.
