# <a name="fatal-error-c1312"></a>Erreur irrécupérable C1312

Trop de branches conditionnelles dans la fonction. Simplifiez ou refactorisez le code source.

Le code est trop complexe pour garantir au compilateur un traitement sans être à court de mémoire de pile.  Simplifiez votre code.
