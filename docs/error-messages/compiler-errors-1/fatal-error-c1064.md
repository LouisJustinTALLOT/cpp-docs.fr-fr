# Erreur irr&#233;cup&#233;rable C1064
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

limite du compilateur : débordement de jetons dans la mémoire tampon interne  
  
 Un identificateur dépasse la limite de la mémoire tampon interne utilisée pour les identificateurs.  Raccourcissez le nom.
