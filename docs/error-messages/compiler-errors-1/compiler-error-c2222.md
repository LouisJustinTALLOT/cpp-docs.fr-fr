# Compiler Error C2222
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

type inattendu 'type' : une classe ou un membre de base était attendu  
  
 La liste d'initialiseurs peut initialiser uniquement des membres ou des classes de base d'un seul type.  Pour résoudre cette erreur, vérifiez que seuls des membres ou des classes de base de ce type sont initialisés dans la liste d'initialiseurs.
