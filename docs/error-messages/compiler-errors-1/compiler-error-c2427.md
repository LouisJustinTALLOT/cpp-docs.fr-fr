# Erreur du compilateur C2427
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'classe' : impossible de définir une classe dans cette portée  
  
 Une tentative de définition d'une classe imbriquée a été effectuée, mais celle\-ci est un membre d'une classe de base, et non la classe conteneur de niveau supérieur.  
  
 L'exemple suivant génère l'erreur C2427 :  
  
```  
// C2427.cpp  
// compile with: /c  
template <class T>   
struct S {  
   struct Inner;   
};   
  
struct Y : S<int> {};   
  
struct Y::Inner {};   // C2427  
  
// OK  
template<typename T>  
struct S<T>::Inner {};  
```
