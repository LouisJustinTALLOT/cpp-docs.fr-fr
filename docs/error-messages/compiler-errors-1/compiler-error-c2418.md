# <a name="compiler-error-c2418"></a>Erreur du compilateur C2418

Impossible de supprimer le fichier browser : nom_fichier

Le compilateur n’a pas pu supprimer le fichier browser.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes

1. Le fichier est en lecture seule.

1. Le fichier est en cours d’utilisation par un autre processus.
