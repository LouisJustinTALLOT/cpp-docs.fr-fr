# <a name="compiler-error-c2418"></a>Erreur du compilateur C2418
Impossible de supprimer le fichier browser : nom de fichier  
  
 Le compilateur n’a pas pu supprimer le fichier du navigateur.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes  
  
1.  Fichier est en lecture seule.  
  
2.  Fichier est en cours d’utilisation par un autre processus.
