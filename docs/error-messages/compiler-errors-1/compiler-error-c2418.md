# Erreur du compilateur C2418
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossible de supprimer le fichier browser : 'NomFichier'  
  
 Le compilateur n'a pas pu supprimer le fichier browser.  
  
### Pour résoudre le problème en vérifiant les causes possibles suivantes  
  
1.  Fichier en lecture seule.  
  
2.  Fichier utilisé par un autre processus.
