# Erreur irr&#233;cup&#233;rable C1077
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

limite du compilateur : impossible d’avoir plus de 'nombre' options de ligne de commande  
  
 Le nombre d’options de ligne de commande dépasse la limite interne.  
  
 Il y a trop de symboles définis avec [\/D](../../build/reference/d-preprocessor-definitions.md). \(Placez plutôt les définitions dans un fichier d’en\-tête.\)
