# Erreur du compilateur C2112
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\-' : le retrait d'un pointeur nécessite un opérande de type intégral ou de type pointeur  
  
 Tentative de soustraction de pointeurs pointant vers des types différents.
