# Erreur irr&#233;cup&#233;rable C1093
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'appel de l'API 'nom de fonction' a échoué 'emplacement de l'appel' : 'texte du runtime'  
  
 Un appel à une fonction .NET a échoué.  ***texte du runtime*** peut être fourni ou non par le runtime COM.  
  
 Pour plus d'informations sur les messages d'erreur système, consultez le fichier système winerror.h, ainsi que [FormatMessage](http://msdn.microsoft.com/library/windows/desktop/ms679351).
