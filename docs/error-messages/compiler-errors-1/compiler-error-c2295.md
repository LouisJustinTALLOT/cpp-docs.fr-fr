# Erreur du compilateur C2295
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

séquence d’échappement 'character' : non conforme dans une définition de macro  
  
 Une définition de macro ne peut pas contenir une séquence d’échappement avec le caractère spécifié.
