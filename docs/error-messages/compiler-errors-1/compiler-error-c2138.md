# Erreur du compilateur C2138
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

il n'est pas conforme de définir une énumération sans membre  
  
 Une énumération doit avoir au moins un membre quand l'option \/Za \(désactiver les extensions Microsoft\) est activée.
