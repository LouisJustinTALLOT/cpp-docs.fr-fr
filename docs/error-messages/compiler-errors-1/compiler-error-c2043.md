# <a name="compiler-error-c2043"></a>Erreur du compilateur C2043

instruction break non conforme

Un [saut](../../cpp/break-statement-cpp.md) est autorisé uniquement dans une **`do`** **`for`** instruction,, **`while`** ou **`switch`** .
