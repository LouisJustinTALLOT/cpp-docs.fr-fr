# Erreur du compilateur C2154
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type' : seul le type énumération est autorisé en tant qu'argument pour un trait de type intrinsèque du compilateur '\_\_underlying\_type'  
  
 Vous pouvez obtenir uniquement le type sous\-jacent d'un type énumération.  
  
 Pour plus d'informations, voir [Compiler Support for Type Traits](../../windows/compiler-support-for-type-traits-cpp-component-extensions.md).
