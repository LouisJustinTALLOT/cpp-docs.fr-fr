# Erreur irr&#233;cup&#233;rable C1051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

le fichier 'FichierPdb' de la base de données du programme a un format obsolète, supprimez\-le et recompilez\-le  
  
 Le compilateur ne peut pas mettre à jour le fichier de base de données du programme, qui est d'une version plus ancienne.  Supprimez le fichier et recompilez votre programme avec **\/Zi** ou **\/ZI**.  Pour plus d’informations, consultez [\/Z7, \/Zi, \/ZI \(Format des informations de débogage\)](../../build/reference/z7-zi-zi-debug-information-format.md)
