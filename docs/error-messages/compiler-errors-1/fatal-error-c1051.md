# <a name="fatal-error-c1051"></a>Erreur irrécupérable C1051

le fichier de base de données du programme, 'PdbFile', a un format obsolète, supprimez-le et recompilez-le

Le compilateur ne peut pas mettre à jour le fichier de base de données du programme, qui a un numéro de version antérieur. Supprimez le fichier et recompilez votre programme avec **/Zi** ou **/Zi**. Pour plus d’informations, consultez [/Z7,/Zi,/ZI (format des informations de débogage)](../../build/reference/z7-zi-zi-debug-information-format.md) .
