# <a name="fatal-error-c1085"></a>Erreur irrécupérable C1085

Impossible d’écrire le fichier filetype : 'fichier' : message

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes

1. Lecteur est en lecture seule.

1. Le lecteur est plein.

1. Violation de partage.

1. Si le message indique « numéro de fichier incorrect », le fichier était peut-être en cours de fermeture au premier plan alors que la compilation se déroulait en arrière-plan.
