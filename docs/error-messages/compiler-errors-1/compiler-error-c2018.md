# Erreur du compilateur C2018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

caractère inconnu 'NombreHexa'  
  
 Le fichier source contient un caractère ASCII inattendu, désigné par son numéro hexadécimal.  Pour corriger l'erreur, supprimez le caractère.
