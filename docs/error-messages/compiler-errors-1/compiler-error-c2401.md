# Erreur du compilateur C2401
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : le registre doit être une base de 'contexte'  
  
 Le Registre utilisé dans un opérande de mémoire indirect doit être un Registre de base dans ce contexte.
