# Erreur du compilateur C2089
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : 'clé\-classe' trop grand  
  
 La structure ou l'union spécifiée dépasse la limite de 4 Go.
