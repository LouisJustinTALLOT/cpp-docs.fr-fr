# <a name="compiler-error-c2174"></a>Erreur du compilateur C2174

'fonction' : le paramètre réel a le type’void' : paramètre number1, liste de paramètres nombre2

Le paramètre `number1` passé à la liste de paramètres `number2` est un paramètre `void`. Les paramètres ne peuvent pas avoir de type `void`. Utilisez `void*` à la place.
