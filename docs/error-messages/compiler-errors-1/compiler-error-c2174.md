# Erreur du compilateur C2174
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : le paramètre actuel a 'void' comme type : paramètre 'numéro1', liste de paramètres 'numéro2'  
  
 Le paramètre `number1` passé à la liste de paramètres `number2` est un paramètre `void`.  Les paramètres ne peuvent pas être de type `void`.  Utilisez plutôt `void*`.
