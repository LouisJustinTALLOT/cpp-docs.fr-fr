# <a name="compiler-error-c2174"></a>Erreur du compilateur C2174

'fonction' : le paramètre réel a le type’void' : paramètre number1, liste de paramètres nombre2

`number1`Le paramètre passé à la liste de paramètres `number2` est un **`void`** paramètre. Les paramètres ne peuvent pas avoir de type **`void`** . Utilisez à la **`void*`** place.
