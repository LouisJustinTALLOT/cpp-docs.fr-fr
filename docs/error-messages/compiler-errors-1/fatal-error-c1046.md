# <a name="fatal-error-c1046"></a>Erreur irrécupérable C1046

limite du compilateur : structure imbriquée trop profondément

La structure, l’Union ou la classe a dépassé la limite d’imbrication, qui est de 15 niveaux. Réécrivez la définition pour réduire le niveau d’imbrication. Fractionnez la structure, l’Union ou la classe en deux ou plusieurs parties à l’aide **`typedef`** de pour définir une ou plusieurs des structures imbriquées.
