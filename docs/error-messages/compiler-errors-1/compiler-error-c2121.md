# <a name="compiler-error-c2121"></a>Erreur du compilateur C2121

' # ' : caractère non valide : peut-être le résultat d’une expansion macro

Un caractère # non valide a peut-être été inséré par une macro incorrecte qui utilise l’opérateur de collage de jeton (# #) au lieu de l’opérateur de chaîne (#).
