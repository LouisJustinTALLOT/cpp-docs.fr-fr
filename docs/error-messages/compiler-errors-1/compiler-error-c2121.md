# Erreur du compilateur C2121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\#' : caractère non valide : peut résulter d'une expansion macro  
  
 Un caractère \# non valide a peut\-être été inséré par une macro incorrecte utilisant l'opérateur de collage de jeton \(\#\#\) plutôt que l'opérateur de transformation en chaîne \(\#\).
