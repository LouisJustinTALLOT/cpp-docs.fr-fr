# <a name="fatal-error-c1033"></a>Erreur irrécupérable C1033

Impossible d’ouvrir le fichier PDB de base de données du programme

Cette erreur peut être due à une erreur disque, à un verrou temporaire créé par un programme antivirus, à une instance précédente du débogueur qui n’a pas été complètement arrêté, ou à une génération parallèle mspdbsrv.exe des processus qui tentent d’accéder au même fichier, entre autres causes possibles.
