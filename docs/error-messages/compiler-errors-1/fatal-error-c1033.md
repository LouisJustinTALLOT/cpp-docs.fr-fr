# <a name="fatal-error-c1033"></a>Erreur irrécupérable C1033

Impossible d’ouvrir le fichier pdb de base de données de programme

Cette erreur peut être provoquée par une erreur disque, un verrou temporaire créé par un antivirus, une instance de débogueur précédente n'a pas complètement arrêtés, ou les processus de mspdbsrv.exe génération parallèle qui tentent d’accéder au même fichier, entre autres les causes possibles.
