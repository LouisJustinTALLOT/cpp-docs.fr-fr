# Erreur du compilateur C2091
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

fonction retournée par la fonction  
  
 Une fonction ne peut pas retourner une fonction.  Retournez plutôt un pointeur vers une fonction.
