# Erreur du compilateur C2308
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

concaténation de chaînes incompatibles  
  
 Des chaînes à caractères larges et non larges ont été spécifiées pour la concaténation.  Il n'est pas possible de concaténer une chaîne de caractères larges et une chaîne de caractères non larges.
