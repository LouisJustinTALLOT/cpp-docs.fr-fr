# <a name="compiler-error-c2308"></a>Erreur du compilateur C2308

concaténation de chaînes incompatibles

Des chaînes de caractères larges et non larges ont été spécifiées pour la concaténation. Vous ne pouvez pas concaténer une chaîne de caractères larges et une chaîne de caractères non larges.
