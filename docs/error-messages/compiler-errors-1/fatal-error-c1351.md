# <a name="fatal-error-c1351"></a>Erreur irrécupérable C1351

> erreur lors du chargement de la dll'*dll*' : version non compatible

La version incorrecte d’une DLL a été trouvée.

Cela indique un problème avec votre installation et vous devez réinstaller Visual Studio.
