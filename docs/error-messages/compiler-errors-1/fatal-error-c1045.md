# <a name="fatal-error-c1045"></a>Erreur irrécupérable C1045

limite du compilateur : spécifications de la liaison imbriquées trop profondément

Les externes imbriqués dépassent la limite du compilateur. Les externes imbriqués sont autorisés avec le type de liaison externe, tel que **`extern`** « C++ ». Réduisez le nombre d’externes imbriqués pour résoudre l’erreur.
