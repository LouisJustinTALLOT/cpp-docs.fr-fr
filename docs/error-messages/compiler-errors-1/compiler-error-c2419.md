# Erreur du compilateur C2419
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

modulo par zéro dans 'contexte'  
  
 Le paramètre situé à droite de l’opérateur `MOD` est égal à zéro dans ce contexte.
