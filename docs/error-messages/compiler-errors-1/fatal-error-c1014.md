# <a name="fatal-error-c1014"></a>Erreur irrécupérable C1014

fichiers Include trop nombreux : profondeur = niveau

L’imbrication de directives `#include` est trop profonde. Des directives imbriquées peuvent inclure des fichiers ouverts. Le fichier source contenant la directive compte comme un seul fichier.
