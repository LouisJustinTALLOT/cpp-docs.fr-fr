# Erreur du compilateur C3134
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'valeur' : la valeur de l'argument d'attribut 'attribut' n'a pas un type valide 'type'  
  
 Une erreur de syntaxe a été détectée lorsqu'une valeur a été assignée à un attribut.
