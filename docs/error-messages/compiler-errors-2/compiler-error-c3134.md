# <a name="compiler-error-c3134"></a>Erreur du compilateur C3134

'valeur' : valeur de l’argument d’attribut 'attribut' n’a pas de type valid 'type'

Une erreur de syntaxe a été détectée lorsqu’une valeur a été attribuée à un attribut.
