# Erreur du compilateur C2855
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

option de ligne de commande 'option' non cohérente avec l'en\-tête précompilé  
  
 L'option donnée de la ligne de commande est différente de l'option utilisée pour créer l'en\-tête précompilé.  Régénérez l'en\-tête précompilé avec l'option de ligne de commande donnée.
