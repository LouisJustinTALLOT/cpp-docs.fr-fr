# Compiler Error C3456
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\[source\_annotation\_attribute\] non autorisé dans une déclaration de classe managée ou WinRT  
  
 source\_annotation\_attribute permet de définir des attributs personnalisés qui doivent être utilisés par l'analyse du code.  
  
 Ces attributs personnalisés ne sont utiles que s'il existe un plug\-in correspondant qui comprend l'attribut.
