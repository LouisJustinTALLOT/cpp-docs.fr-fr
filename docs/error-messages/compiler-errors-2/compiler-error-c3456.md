# <a name="compiler-error-c3456"></a>Erreur du compilateur C3456

[source_annotation_attribute] non autorisé dans une déclaration de classe managée ou WinRT

source_annotation_attribute permet de définir des attributs personnalisés qui doivent être utilisés par l'analyse du code.

Ces attributs personnalisés ne sont utiles que s'il existe un plug-in correspondant qui comprend l'attribut.
