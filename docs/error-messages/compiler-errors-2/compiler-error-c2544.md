# Erreur du compilateur C2544
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

attendu '\)' pour l'opérateur '\(\)'  
  
 L'opérateur d'appel de fonction est dépourvu de parenthèse gauche.  
  
 Cette erreur est peut\-être due à une expansion macro.
