# <a name="compiler-error-c2544"></a>Erreur du compilateur C2544
attendu ')' pour l’opérateur '()'  
  
 Il manque une parenthèse gauche de l’opérateur d’appel de fonction.  
  
 Cette erreur peut être dû à une expansion macro.
