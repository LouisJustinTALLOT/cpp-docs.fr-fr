# <a name="compiler-error-c3815"></a>Erreur du compilateur C3815

le type de retour de la méthode’get_accessor’doit correspondre au type du dernier paramètre d’un accesseur Set

Lors de la déclaration de propriétés, la valeur de retour de la `get_accessor` méthode doit correspondre au dernier paramètre de la déclaration de la méthode d’accesseur Set.

C3815 est accessible uniquement à l’aide de l’option de compilateur obsolète **/clr : oldSyntax**.
