# Erreur du compilateur C3157
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

L'attribut ParamArray ne peut être appliqué qu'au dernier paramètre  
  
 L'attribut [ParamArray](https://msdn.microsoft.com/en-us/library/system.paramarrayattribute.aspx) a été appliqué au paramètre incorrect.
