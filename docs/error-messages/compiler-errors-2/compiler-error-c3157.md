# <a name="compiler-error-c3157"></a>Erreur du compilateur C3157

L’attribut ParamArray ne peut être appliqué qu’au dernier paramètre

L' <xref:System.ParamArrayAttribute> attribut a été appliqué au mauvais paramètre.
