# <a name="compiler-error-c2823"></a>Erreur du compilateur C2823  
  
> un modèle typedef non conforme  
  
Les modèles ne sont pas autorisés dans `typedef` définitions.  
  
## <a name="example"></a>Exemple  
  
L’exemple suivant génère C2823 et illustre une façon de pour résoudre ce problème :  
  
```cpp  
// C2823.cpp  
template<class T>  
typedef struct x {  
   T i;   // C2823 can't use T, specify data type and delete template  
   int i;   // OK  
} x1;  
```
