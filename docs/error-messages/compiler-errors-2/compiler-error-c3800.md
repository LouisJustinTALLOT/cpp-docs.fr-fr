# <a name="compiler-error-c3800"></a>Erreur du compilateur C3800
'déclaration' : ne peut pas mélanger les propriétés et événements  
  
 Vous ne pouvez pas déclarer un constructeur qui soit à la fois une propriété et un événement.  
  
 C3800 est uniquement accessible à l’aide de l’option du compilateur obsolètes **/CLR : oldSyntax**.
