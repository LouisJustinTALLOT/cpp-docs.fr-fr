# <a name="compiler-error-c3800"></a>Erreur du compilateur C3800

'Déclaration' : impossible de mélanger les propriétés et les événements

Vous ne pouvez pas déclarer une construction comme étant à la fois une propriété et un événement.

C3800 est accessible uniquement à l’aide de l’option de compilateur obsolète **/clr : oldSyntax**.
