# <a name="compiler-error-c3159"></a>Erreur du compilateur C3159

'pointeur' : le tableau de pointeurs vers le type valeur ne peut pas être déclaré

Un tableau de pointeurs vers un type valeur ne peut pas être déclaré.

C3159 est accessible uniquement à l’aide de l’option de compilateur obsolète **/clr : oldSyntax**.
