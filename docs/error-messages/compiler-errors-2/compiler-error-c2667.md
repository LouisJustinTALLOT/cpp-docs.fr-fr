# <a name="compiler-error-c2667"></a>Erreur du compilateur C2667

'fonction' : aucune des surcharges de nombre n’a une meilleure conversion

Un appel de fonction surchargé est ambigu et ne peut pas être résolu.

La conversion nécessaire pour faire correspondre les paramètres réels dans l’appel de fonction à l’une des fonctions surchargées doit être strictement meilleure que les conversions requises par toutes les autres fonctions surchargées.
