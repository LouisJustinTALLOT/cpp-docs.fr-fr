# Erreur du compilateur C2830
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

seuls les paramètres de positionnement de 'operator new' peuvent avoir des valeurs par défaut  
  
 Les paramètres formels standard de [operator new](../Topic/operator%20new%20\(%3Cnew%3E\).md) ne peuvent pas avoir des valeurs par défaut.  Seuls les paramètres de positionnement définis par l'opérateur peuvent spécifier des valeurs par défaut.
