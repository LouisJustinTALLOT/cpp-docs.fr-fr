# <a name="compiler-error-c3500"></a>Erreur du compilateur C3500

ProgID non valide 'progid'

Un progid non valide a été spécifié avec la `#import` instruction. Vérifier le Registre Windows pour vous assurer que vous spécifiez valide progid.
