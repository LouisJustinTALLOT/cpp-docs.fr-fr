# <a name="compiler-error-c3500"></a>Erreur du compilateur C3500

ProgID’ProgID’non valide

Un ProgID non valide a été spécifié avec l' `#import` instruction. Vérifiez le Registre Windows pour vous assurer que vous spécifiez un ProgID valide.
