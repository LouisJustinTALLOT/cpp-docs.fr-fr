# Erreur du compilateur C3500
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ProgID non valide 'ProgId'  
  
 Un progid non valide a été spécifié avec l'instruction `#import`.  Vérifiez le Registre Windows pour vous assurer que vous spécifiez bien un progid valide.
