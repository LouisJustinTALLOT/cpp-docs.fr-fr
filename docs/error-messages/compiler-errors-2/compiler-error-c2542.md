# Erreur du compilateur C2542
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : l'objet de classe n'a pas de constructeur pour l'initialisation  
  
 Aucun constructeur n'a de liste de paramètres correspondant à l'initialisation.  
  
 Cette erreur peut être provoquée par des paramètres incorrects lors de l'initialisation.
