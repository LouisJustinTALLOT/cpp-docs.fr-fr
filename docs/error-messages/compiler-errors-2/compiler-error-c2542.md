# <a name="compiler-error-c2542"></a>Erreur du compilateur C2542

'identificateur' : l’objet de classe n’a aucun constructeur pour l’initialisation

Il n’existe aucun constructeur avec une liste de paramètres qui correspond à l’initialisation.

Cette erreur peut être due à des paramètres incorrects lors de l’initialisation.
