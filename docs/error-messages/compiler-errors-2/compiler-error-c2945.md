# <a name="compiler-error-c2945"></a>Erreur du compilateur C2945

l'instanciation explicite ne fait pas référence à une spécialisation de classe de modèle

Vous ne pouvez pas instancier explicitement un élément qui n’est pas basé sur un modèle.
