# Erreur du compilateur C2568
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur1' : impossible de résoudre la surcharge de fonction  
  
 Le compilateur ne peut pas déterminer quelle est la fonction surchargée à appeler.  Les paramètres en cours passés à la fonction doivent subir un cast pour correspondre aux paramètres formels de l'une des fonctions surchargées, mais aucune des correspondances trouvées n'est suffisamment univoque.
