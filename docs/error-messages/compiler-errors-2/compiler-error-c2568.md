# <a name="compiler-error-c2568"></a>Erreur du compilateur C2568

'identificateur1 ' : impossible de résoudre la surcharge de fonction

Le compilateur ne peut pas déterminer la fonction surchargée à appeler. Les paramètres réels passés à la fonction doivent être castés pour correspondre aux paramètres formels de l’une des fonctions surchargées, mais aucune correspondance n’est manifestement meilleure que toutes les autres.
