# <a name="compiler-error-c2651"></a>Erreur du compilateur C2651
'type de données' : gauche de 'operator' doit être une classe, struct ou une union  
  
 Pour utiliser un paramètre de modèle comme s’il s’agit d’une classe, spécialiser le modèle de classe avec une classe au lieu d’un type intégral.
