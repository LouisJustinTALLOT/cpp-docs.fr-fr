# Erreur du compilateur C2651
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type de données' : l'élément à gauche de ' ::' doit être une classe, un struct ou une union  
  
 Pour utiliser un paramètre modèle comme s'il s'agissait d'une classe, spécialisez le modèle de classe avec une classe au lieu d'un type intégral.
