# <a name="compiler-error-c2651"></a>Erreur du compilateur C2651

'Data type' : la partie gauche de’operator’doit être une classe, un struct ou une Union

Pour utiliser un paramètre de modèle comme s’il s’agissait d’une classe, Spécialisez le modèle de classe avec une classe au lieu d’un type intégral.
