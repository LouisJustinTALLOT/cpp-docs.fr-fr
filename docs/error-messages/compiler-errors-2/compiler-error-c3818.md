# <a name="compiler-error-c3818"></a>Erreur du compilateur C3818

la déclaration de propriété de tableau’property1 'ne doit pas surcharger une propriété d’index’Property2 '

Une surcharge n’est pas possible pour les propriétés lorsque l’une d’elles est un indexeur et que l’autre est une propriété de tableau.

C3818 est accessible uniquement à l’aide de l’option de compilateur obsolète **/clr : oldSyntax**.
