# Erreur du compilateur C2656
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : fonction non autorisée comme champ de bits  
  
 Une fonction est déclarée comme membre d'un champ de bits.  
  
 Cette erreur peut être provoquée par une erreur de syntaxe dans la liste d'initialiseurs d'un constructeur.
