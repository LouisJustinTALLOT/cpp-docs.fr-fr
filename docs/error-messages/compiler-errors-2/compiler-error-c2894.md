# <a name="compiler-error-c2894"></a>Erreur du compilateur C2894
les modèles ne peut pas être déclarés pour que la liaison 'C'  
  
 Cette erreur peut être provoquée par un modèle est défini à l’intérieur d’un `extern` bloc de « C ».  
  
 L’exemple suivant génère l’erreur C2894 :  
  
```  
// C2894.cpp  
extern "C" {  
   template<class T> class stack {};   // C2894 fail  
  
   template<class T> void f(const T &aT) {}   // C2894  
}  
```  
  
 L’exemple suivant génère l’erreur C2894 :  
  
```  
// C2894b.cpp  
// compile with: /c  
extern "C" template<class T> void f(const T &aT) {}   // C2894  
  
template<class T> void f2(const T &aT) {}   // OK  
```
