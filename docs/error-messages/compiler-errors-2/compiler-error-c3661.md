# <a name="compiler-error-c3661"></a>Erreur du compilateur C3661

la liste de substitution explicite n’a pas trouvé de méthode à substituer

Une substitution explicite spécifiait un ou plusieurs noms de types.  Toutefois, il n’existait aucune fonction avec la signature nécessaire dans le ou les types qui correspondaient à la signature de la fonction de substitution.  Si vous tentez de remplacer en fonction du nom de type, il doit y avoir une ou plusieurs fonctions virtuelles dans le ou les types spécifiés qui correspondent à la signature de la fonction de substitution.

Pour plus d’informations, consultez [substitutions explicites](../../extensions/explicit-overrides-cpp-component-extensions.md).
