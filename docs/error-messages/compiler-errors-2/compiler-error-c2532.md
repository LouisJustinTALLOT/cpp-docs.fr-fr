# <a name="compiler-error-c2532"></a>Erreur du compilateur C2532

'identificateur' : modificateur non conforme pour la référence

La référence a été modifiée. Les références ne peuvent pas être modifiées pour faire référence à un autre objet. Utilisez plutôt un pointeur.
