# Erreur du compilateur C2532
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : modificateur non conforme pour la référence  
  
 La référence a été modifiée.  Il n'est pas possible de modifier les références pour désigner un autre objet.  Utilisez un pointeur à la place.
