# <a name="compiler-error-c2616"></a>Erreur du compilateur C2616

'conversion' : Impossible de convertir implicitement une non-lvalue 'type1' en un 'type2' qui n’est pas const

Impossible d’initialiser une référence à partir d’une non-lvalue.

Il s’agit d’une erreur sous compatibilité ANSI ([/Za](../../build/reference/za-ze-disable-language-extensions.md)) et un avertissement sous extensions Microsoft (**/Ze**).
