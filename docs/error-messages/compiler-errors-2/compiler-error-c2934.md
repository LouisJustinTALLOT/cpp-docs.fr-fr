# <a name="compiler-error-c2934"></a>Erreur du compilateur C2934

'class' : type-class-id redéfini comme 'item' imbriqué de 'identifier'

Vous ne pouvez pas utiliser une classe générique ou de modèle en tant qu’élément imbriqué.
