# Erreur du compilateur C2958
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'élément délimiteur gauche de 'emplacement' n'a pas été équilibré correctement  
  
 Un délimiteur est mal équilibré.  
  
 Cette erreur peut être provoquée par une parenthèse non appariée dans une déclaration de modèle de fonction.
