# <a name="compiler-error-c2958"></a>Erreur du compilateur C2958

le délimiteur gauche trouvé à l’emplacement’location’n’a pas été correctement mis en correspondance

Le délimiteur n’est pas correctement mis en correspondance.

Cette erreur peut être causée par des parenthèses incompatibles dans une déclaration de modèle de fonction.
