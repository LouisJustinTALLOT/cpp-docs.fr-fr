# <a name="compiler-error-c2962"></a>Erreur du compilateur C2962

erreur de syntaxe : 'token' : une définition de membre de classe de modèle doit se terminer par '}'

Le jeton a provoqué une erreur de syntaxe dans une déclaration de modèle.

Cette erreur peut être provoquée par des délimiteurs non appariés
