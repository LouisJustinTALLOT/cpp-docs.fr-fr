# Erreur du compilateur C2817
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

le type de retour de 'operator delete' doit être 'void'  
  
 Une fonction [operator delete](../Topic/operator%20delete%20\(%3Cnew%3E\).md) surchargée ne peut pas retourner une valeur.
