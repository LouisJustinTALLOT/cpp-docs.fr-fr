# <a name="compiler-error-c2817"></a>Erreur du compilateur C2817
type de retour de 'operator delete' doit être 'void'  
  
Une procédure surchargée [opérateur delete](../../standard-library/new-operators.md#op_delete) fonction ne peut pas retourner une valeur.
