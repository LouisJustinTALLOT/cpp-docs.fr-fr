# Erreur du compilateur C2773
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#import et \#using disponibles seulement avec le compilateur C\+\+  
  
 Le compilateur en langage C ne reconnaît pas la directive de préprocesseur `#import`.  Compilez la source en C\+\+.  Utilisez [\/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md), si nécessaire.
