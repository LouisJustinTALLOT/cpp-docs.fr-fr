# <a name="compiler-error-c2773"></a>Erreur du compilateur C2773

\#importer et #using disponibles uniquement dans le compilateur C++

Le compilateur C ne reconnaît pas la `#import` directive de préprocesseur. Compilez la source en C++. Utilisez [/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) si nécessaire.
