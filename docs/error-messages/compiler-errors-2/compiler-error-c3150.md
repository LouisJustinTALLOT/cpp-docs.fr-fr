# <a name="compiler-error-c3150"></a>Erreur du compilateur C3150

'element' : 'attribute' peut uniquement être appliqué à une classe, interface, tableau ou un pointeur

`__gc` peut uniquement être utilisé sur une classe, une interface ou un tableau.

C3150 est uniquement accessible à l’aide de l’option de compilateur obsolète **/CLR : oldSyntax**.
