# <a name="compiler-error-c3150"></a>Erreur du compilateur C3150

'Element' : 'Attribute’ne peut s’appliquer qu’à une classe, une interface, un tableau ou un pointeur

**`__gc`** peut uniquement être utilisé sur une classe, une interface ou un tableau.

C3150 est accessible uniquement à l’aide de l’option de compilateur obsolète **/clr : oldSyntax**.
