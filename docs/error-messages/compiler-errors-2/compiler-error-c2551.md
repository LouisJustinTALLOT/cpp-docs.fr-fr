# Erreur du compilateur C2551
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

le type 'void \*' nécessite un cast explicite  
  
 Un pointeur `void` est assigné à un pointeur non void par conversion implicite. Vous devez utiliser un cast explicite.
