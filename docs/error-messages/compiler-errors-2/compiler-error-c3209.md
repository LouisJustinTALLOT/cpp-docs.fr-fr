# Compiler Error C3209
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'classe' : la classe générique doit être une classe managée ou WinRT  
  
 Une classe générique doit être une classe managée ou une classe Windows Runtime.  
  
 L'exemple suivant génère l'erreur C3209 et montre comment la corriger :  
  
```  
// C3209.cpp  
// compile with: /clr  
generic <class T>  
class C {};   // C3209  
  
// OK - ref class can be generic  
generic <class T>  
ref class D {};  
```
