# <a name="compiler-error-c3187"></a>Erreur du compilateur C3187
' __func\_\_' : est uniquement disponible dans le corps d’une fonction  
  
 La macro prédéfinie n'est pas disponible en dehors du corps d'une fonction.  
  
 Pour corriger cette erreur, déplacez la macro dans le corps d'une fonction.
