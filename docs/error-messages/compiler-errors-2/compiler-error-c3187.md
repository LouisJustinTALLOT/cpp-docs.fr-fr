# Erreur du compilateur C3187
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\_\_func\_\_' : est uniquement disponible dans le corps d'une fonction  
  
 La macro prédéfinie n'est pas disponible en dehors du corps d'une fonction.  
  
 Pour corriger cette erreur, déplacez la macro dans le corps d'une fonction.
