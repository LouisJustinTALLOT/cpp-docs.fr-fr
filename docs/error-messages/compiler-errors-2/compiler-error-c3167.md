# <a name="compiler-error-c3167"></a>Erreur du compilateur C3167

Impossible d’initialiser .NET Framework : Assurez-vous qu’il est installé

Le .NET Framework n’est pas installé sur cet ordinateur. Installez le .NET Framework.
