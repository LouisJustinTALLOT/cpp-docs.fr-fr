# <a name="compiler-error-c3167"></a>Erreur du compilateur C3167
Impossible d’initialiser le .NET Framework : Vérifiez qu’il est installé  
  
 Le .NET Framework n’est pas installé sur cet ordinateur ; installer le .NET Framework.
