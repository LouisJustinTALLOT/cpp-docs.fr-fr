# Erreur du compilateur C2545
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'opérateur' : impossible de trouver l'opérateur surchargé  
  
 L'opérateur ne peut être utilisé avec les opérandes fournis.  Vous devez fournir un opérateur surchargé avec les opérandes requis.  
  
 Cette erreur peut être générée si les opérandes possèdent un type incorrect.  
  
 Vous pouvez y remédier en définissant un opérateur de conversion ou un constructeur qui prend un paramètre unique.
