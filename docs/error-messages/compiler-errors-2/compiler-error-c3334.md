# <a name="compiler-error-c3334"></a>Erreur du compilateur C3334

#import de bibliothèque de types endommagée impossible

Le fichier de bibliothèque indiqué dans une instruction `#import` est endommagé et ne peut pas être utilisé par le compilateur.  Cette altération peut être causée par une génération incorrecte du module, un disque endommagé ou la modification du fichier binaire.

Pour plus d’informations, consultez [#import directive](../../preprocessor/hash-import-directive-cpp.md).
