# Erreur du compilateur C2994
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

classe sans nom dans une liste de paramètres de modèle  
  
 Vous ne pouvez pas utiliser le mot clé `class` comme argument template sans spécifier un nom de classe.
