# Erreur du compilateur C3135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'propriété' : une propriété ne peut avoir de type 'const' ou 'volatile'  
  
 Les mots clés [const](../../cpp/const-cpp.md) et [volatile](../../cpp/volatile-cpp.md) ne sont pas autorisés sur les propriétés.
