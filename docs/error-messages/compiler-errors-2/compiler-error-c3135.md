# <a name="compiler-error-c3135"></a>Erreur du compilateur C3135

'Property' : une propriété ne peut pas avoir un type’const’ou’volatile'

Les mots clés [const](../../cpp/const-cpp.md) et [volatile](../../cpp/volatile-cpp.md) ne sont pas autorisés sur les propriétés.
