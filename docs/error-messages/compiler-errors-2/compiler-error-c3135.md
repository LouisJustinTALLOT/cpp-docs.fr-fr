# <a name="compiler-error-c3135"></a>Erreur du compilateur C3135
'propriété' : une propriété ne peut pas avoir 'const' ou 'volatile' de type  
  
 Le [const](../../cpp/const-cpp.md) et [volatile](../../cpp/volatile-cpp.md) mots clés ne sont pas autorisés sur les propriétés.
