# Erreur du compilateur C2730
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class' : ne peut pas être une classe de base d’elle\-même  
  
 Les classes de base récursives ne sont pas valides. Spécifiez une autre classe en tant que classe de base.
