# <a name="compiler-error-c2795"></a>Erreur du compilateur C2795

'Super :: function’n’est pas une fonction membre

Ce message d’erreur apparaît chaque fois que vous essayez d’utiliser le [Super](../../cpp/super.md) pour accéder à un membre autre qu’une fonction membre.
