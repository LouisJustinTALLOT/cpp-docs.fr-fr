# <a name="compiler-error-c2795"></a>Erreur du compilateur C2795

'super::Function ' n’est pas une fonction membre

Ce message d’erreur s’affiche chaque fois que vous essayez d’utiliser [super](../../cpp/super.md) pour accéder à un membre autre qu’une fonction membre.
