# Erreur du compilateur C2795
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'super::function' n'est pas une fonction membre  
  
 Ce message d'erreur apparaît chaque fois que vous tentez d'utiliser [super](../../cpp/super.md) pour accéder à un membre autre qu'une fonction membre.
