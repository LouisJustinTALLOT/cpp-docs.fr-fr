# <a name="compiler-error-c3173"></a>Erreur du compilateur C3173

incompatibilité de version dans la fusion IDL

Cette erreur se produit lorsqu’un fichier objet contient un IDL incorporé qui a été généré avec une version antérieure du compilateur. Le compilateur encode un numéro de version pour s’assurer que le compilateur utilisé pour générer le contenu IDL incorporé dans les fichiers. obj est également le même que celui utilisé pour fusionner l’IDL incorporé.

Mettez à jour C++ votre installation visuelle afin que tous les outils proviennent de la dernière version publiée.
