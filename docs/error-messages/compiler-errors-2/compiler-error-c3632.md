# <a name="compiler-error-c3632"></a>Erreur du compilateur C3632

'Event' : style d’événement non conforme pour la construction

les déclarations de [__event](../../cpp/event.md) ne sont pas valides dans toutes les constructions.

C3632 est accessible uniquement à l’aide de l’option de compilateur obsolète **/clr : oldSyntax**.
