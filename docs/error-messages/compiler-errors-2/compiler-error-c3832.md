# <a name="compiler-error-c3832"></a>Erreur du compilateur C3832

'bibliothèque de types' : bibliothèque de types semble avoir été développée pour des pointeurs 32 bits ; Veuillez modifier le qualificateur 'ptrsize'

Information explicite fournie avec le `ptrsize` attribut de la [#import](../../preprocessor/hash-import-directive-cpp.md) directive ne correspond pas avec ce que le compilateur a trouvé dans la bibliothèque de types.
