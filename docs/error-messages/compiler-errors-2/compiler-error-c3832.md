# <a name="compiler-error-c3832"></a>Erreur du compilateur C3832

'bibliothèque de types' : la bibliothèque de types semble avoir été créée pour les pointeurs 32 bits ; Modifiez le qualificateur’ptrsize'

Les informations explicites fournies avec l' `ptrsize` attribut de la directive [#import](../../preprocessor/hash-import-directive-cpp.md) ne sont pas conformes à ce que le compilateur a trouvé dans la bibliothèque de types.
