# <a name="compiler-error-c3820"></a>Erreur du compilateur C3820

> '*Name*' : les initialiseurs doivent être managés

## <a name="remarks"></a>Notes

L’initialiseur *nom* d’un type managé doit également être managé. Pour corriger cette erreur, spécifiez une méthode d’initialiseur managée.
