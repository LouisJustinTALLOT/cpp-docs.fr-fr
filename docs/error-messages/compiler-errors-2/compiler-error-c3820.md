# Erreur du compilateur C3820
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'name' : les initialiseurs doivent être managés  
  
 L’initialiseur *nom* d’un type managé doit également être managé.  
  
### Pour corriger cette erreur  
  
1.  Spécifiez une méthode d’initialiseur managée.
