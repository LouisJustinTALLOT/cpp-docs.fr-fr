# Erreur du compilateur C3333
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'bibliothèque de types' : \#import de bibliothèque de types endommagée impossible  
  
 La bibliothèque de types spécifiée dans l’instruction `#import` ne peut pas être lue par le compilateur. Si possible, régénérez la bibliothèque de types ou demandez une nouvelle bibliothèque de types auprès de votre fournisseur. Vous pouvez utiliser la visionneuse OLE fournie avec Visual C\+\+ pour examiner le fichier de bibliothèque de types et identifier le problème.
