# <a name="compiler-error-c2829"></a>Erreur du compilateur C2829

'operator opérateur' ne peut pas avoir une liste de paramètres de variable

Seuls deux opérateurs peuvent prendre des listes de paramètres de variable :

- Appel de fonction `()`

- `new`
