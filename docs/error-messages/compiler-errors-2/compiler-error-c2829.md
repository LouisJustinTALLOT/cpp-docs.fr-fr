# Erreur du compilateur C2829
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator opérateur' ne peut pas comporter de liste de paramètres de variables  
  
 Seuls deux opérateurs peuvent comporter des listes de paramètres de variables :  
  
-   Appel de fonction `()`  
  
-   `new`
