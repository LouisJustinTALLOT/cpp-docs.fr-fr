# <a name="compiler-error-c2829"></a>Erreur du compilateur C2829

'operator opérateur’ne peut pas avoir une liste de paramètres de variables

Seuls deux opérateurs peuvent prendre des listes de paramètres de variables :

- Appel de fonction `()`

- **`new`**
