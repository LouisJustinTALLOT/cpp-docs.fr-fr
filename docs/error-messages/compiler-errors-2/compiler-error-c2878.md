# <a name="compiler-error-c2878"></a>Erreur du compilateur C2878
'nom' : un espace de noms ou une classe de ce nom n’existe pas  
  
 Vous avez apportées à la référence à un espace de noms ou une classe qui n’est pas défini.  
  
 L’exemple suivant génère l’erreur C2878 :  
  
```  
// C2878.cpp  
// compile with: /c  
namespace A {}  
namespace B = C;   // C2878 namespace C doesn't exist  
namespace B = A;   
```
