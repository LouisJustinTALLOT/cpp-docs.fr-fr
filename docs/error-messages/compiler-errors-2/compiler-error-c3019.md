# <a name="compiler-error-c3019"></a>Erreur du compilateur C3019

incrément 'd’instruction for' OpenMP a forme incorrecte

La partie incrément d’une OpenMP `for` boucle doit utiliser la variable d’index à la fois sur la gauche et droite de l’opérateur.

L’exemple suivant génère l’erreur C3019 :

```
// C3019.cpp
// compile with: /openmp
int main()
{
   int i = 0, j = 1, n = 3;

   #pragma omp parallel
   {
      #pragma omp for
      for (i = 0; i < 10; i = j + n)   // C3019
      // Try the following line instead:
      // for (i = 0; i < 10; i++)
         j *= 2;
   }
}
```
