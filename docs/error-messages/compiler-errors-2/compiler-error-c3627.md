# Erreur du compilateur C3627
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Seul un type valeur peut être converti  
  
 Seules les classes valeur peuvent être converties.  
  
 L'exemple suivant génère l'erreur C3627 :  
  
```  
// C3627.cpp  
// compile with: /clr:oldSyntax  
#using <mscorlib.dll>  
  
__value class vA {  
};  
  
__gc class A {  
};  
  
int main()  
{  
   A* a;  
   __box(a);   // C3627  
  
   // box a value type  
   vA va;  
   __box(va);  
}  
```
