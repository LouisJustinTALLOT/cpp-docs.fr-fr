# <a name="compiler-error-c3356"></a>Erreur du compilateur C3356

'attribute' : impossible d’appeler un attribut multicast avec un nom complet

Un attribut qui est traité par plusieurs processus, par exemple, le compilateur et le fournisseur ATL, n’a pas été spécifié correctement.
