# <a name="compiler-error-c3631"></a>Erreur du compilateur C3631

'fonction' : impossible de surcharger les événements managés ou WinRT

Un événement managé ou WinRT ne peut pas être surchargé.

C3631 est accessible uniquement à l’aide de l’option de compilateur obsolète **/clr : oldSyntax**.
