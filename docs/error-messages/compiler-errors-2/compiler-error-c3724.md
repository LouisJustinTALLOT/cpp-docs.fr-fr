# Erreur du compilateur C3724
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

doit \#inclure \<windows.h\> pour utiliser la fonction multi\-threading sur les événements  
  
 Le fichier windows.h est requis si vous utilisez la fonction multithread avec les événements.  Pour remédier à cette erreur, ajoutez `#include <windows.h>` au début du fichier dans lequel sont définies les sources d'événements et les récepteurs d'événements.  
  
```  
// C3724.cpp  
// uncomment the following line to resolve  
// #include <windows.h>  
  
[event_source(native), threading(apartment)]  
class CEventSrc {  
public:  
   __event void event1();   // C3724  
};  
  
[event_receiver(native)]  
class CEventRec {  
public:  
   void handler1() {  
   }  
  
   void HookEvents(CEventSrc* pSrc) {  
      __hook(CEventSrc::event1, pSrc, CEventRec::handler1);  
   }  
  
   void UnhookEvents(CEventSrc* pSrc) {  
      __unhook(CEventSrc::event1, pSrc, CEventRec::handler1);  
   }  
};  
  
int main() {  
}  
```
