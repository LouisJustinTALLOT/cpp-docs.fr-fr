# Erreur du compilateur C2526
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur1' : une fonction de liaison C ne peut pas retourner de classe C\+\+ 'identificateur2'  
  
 Une fonction définie avec la liaison C ne peut pas retourner de type défini par l'utilisateur.
