# <a name="compiler-error-c2654"></a>Erreur du compilateur C2654

'identificateur' : tentative d’accès à un membre en dehors d’une fonction membre

Un membre est accessible dans une déclaration. Les données membres sont accessibles uniquement dans les fonctions membres.

Cette erreur peut se produire lorsque vous tentez d’initialiser des variables dans une déclaration. Utilisez un constructeur à cet effet.
