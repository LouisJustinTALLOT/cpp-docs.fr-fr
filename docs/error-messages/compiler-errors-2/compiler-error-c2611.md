# <a name="compiler-error-c2611"></a>Erreur du compilateur C2611
'jeton' : non conforme après ' ~' (identificateur attendu)  
  
 Le jeton n’est pas un identificateur.  
  
 L’exemple suivant génère C2611 :  
  
```  
// C2611.cpp  
// compile with: /c  
class C {  
   C::~operator int();   // C2611  
   ~C();   // OK  
};  
```
