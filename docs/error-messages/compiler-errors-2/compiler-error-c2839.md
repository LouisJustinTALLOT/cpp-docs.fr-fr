# Erreur du compilateur C2839
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

type de retour 'type' invalide pour l'opérateur surchargé 'operateur \-\>'  
  
 L'opérateur `->` doit retourner une classe, une structure ou une union ou encore une référence à l'une d'elles.
