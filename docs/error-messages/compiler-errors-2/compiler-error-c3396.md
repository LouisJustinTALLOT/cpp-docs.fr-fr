# <a name="compiler-error-c3396"></a>Erreur du compilateur C3396

'attribut' : attribut personnalisé introuvable dans 'espace_noms'

L’erreur C3396 indique que vous n’utilisez pas un common language runtime à jour.  Réinstallez votre common language runtime pour y remédier.
