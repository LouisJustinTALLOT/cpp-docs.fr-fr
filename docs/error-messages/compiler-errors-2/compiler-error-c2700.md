# <a name="compiler-error-c2700"></a>Erreur du compilateur C2700

'identificateur' : ne peut pas être levé(e) (utilisez /W4 pour plus d’informations)

L’objet ne peut pas être levé. Pour obtenir davantage d’informations de diagnostic, effectuez une compilation à l’aide de [/W4](../../build/reference/compiler-option-warning-level.md) .
