# <a name="compiler-error-c3130"></a>Erreur du compilateur C3130

Erreur interne du compilateur : échec de l’écriture du bloc de code injecté dans PDB

Cette erreur se produit si le compilateur n’a pas pu écrire un bloc de code injecté dans le fichier. pdb. La raison la plus courante de l’échec est l’absence d’espace disque.
