# Erreur du compilateur C3130
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erreur interne du compilateur : échec de l'écriture du bloc de code injecté dans PDB  
  
 Cette erreur se produit si le compilateur ne réussit pas à écrire un bloc de code injecté dans le fichier .pdb.  La raison la plus courante de cette erreur est un manque d'espace disque.
