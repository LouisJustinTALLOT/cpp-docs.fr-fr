# <a name="compiler-error-c3248"></a>Erreur du compilateur C3248
'fonction1' : fonction déclarée comme '__sealed' ne peut pas être substituée par 'fonction2'  
  
Une classe dérivée a essayé de substituer un **__sealed** méthode virtuelle.  
  
C3248 est uniquement accessible à l’aide de l’option du compilateur obsolètes **/CLR : oldSyntax**.
