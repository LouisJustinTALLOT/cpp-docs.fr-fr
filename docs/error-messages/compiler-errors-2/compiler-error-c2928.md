# <a name="compiler-error-c2928"></a>Erreur du compilateur C2928

> instanciation explicite ; '*identifier*'n’est pas une fonction ni une donnée membre static de la classe de modèle'*Class-Name*'

Vous ne pouvez pas explicitement instancier un membre de *Class-Name* qui n’est pas une fonction ou une **`static`** variable.
