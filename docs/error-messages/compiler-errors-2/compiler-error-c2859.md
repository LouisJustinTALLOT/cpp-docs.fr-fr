# <a name="compiler-error-c2859"></a>Erreur du compilateur C2859

filename n’est pas le fichier de type qui a été utilisé lors de la création de cet en-tête précompilé, recréez l’en-tête précompilé.

La base de données de projet et les fichiers d’en-tête précompilés doivent être créés ensemble pour garantir la cohérence des informations. Régénérez le projet pour recréer l’en-tête précompilé.

Pour plus d’informations sur les en-têtes précompilés, consultez [/y (en-têtes précompilés)](../../build/reference/y-precompiled-headers.md).
