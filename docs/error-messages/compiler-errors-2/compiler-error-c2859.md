# Erreur du compilateur C2859
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'NomFichier' ne correspond pas au fichier de type utilisé lors de la création de cet en\-tête précompilé, recréez l'en\-tête précompilé.  
  
 La base de données du projet et les fichiers d'en\-tête précompilé doivent être créés ensemble pour garantir la cohérence des informations.  Régénérez le projet pour créer à nouveau l'en\-tête précompilé.  
  
 Pour plus d'informations sur les en\-têtes précompilés, consultez [\/Y \(En\-têtes précompilés\)](../../build/reference/y-precompiled-headers.md).
