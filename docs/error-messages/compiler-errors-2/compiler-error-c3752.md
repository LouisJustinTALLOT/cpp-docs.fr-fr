# Erreur du compilateur C3752
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'classe d'attributs' : impossible de classer l'attribut ; 'MotClé' ne doit pas être utilisé dans ce contexte
