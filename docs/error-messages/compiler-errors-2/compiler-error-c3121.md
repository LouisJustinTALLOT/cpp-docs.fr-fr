# Erreur du compilateur C3121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossible de changer GUID pour la classe 'nom de classe'  
  
 Vous avez essayé de remplacer l'ID de classe par [\_\_declspec\(uuid\)](../../cpp/uuid-cpp.md).  
  
 Par exemple, le code suivant génère C3121 :  
  
```  
// C3121.cpp  
[emitidl];  
[module(name="MyLibrary")];  
  
[coclass, uuid="00000000-0000-0000-0000-111111111111"]  
class __declspec(uuid("00000000-0000-0000-0000-111111111112")) A   // C3121  
{  
};  
int main()  
{  
}  
```
