# <a name="compiler-error-c3501"></a>Erreur du compilateur C3501

aucune TypeLib inscrite pour ProgID’ProgID'

L’ID de classe d’un ProgID donné n’a pas de bibliothèque de types associée. Par conséquent, vous ne pouvez pas transmettre ce ProgID à l’instruction `#import`.
