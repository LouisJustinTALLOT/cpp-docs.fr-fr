# Erreur du compilateur C3501
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

aucune typelib inscrite pour ProgID 'progid'  
  
 L'ID de classe pour un progid donné ne possède pas de bibliothèque de types associée.  Vous ne pouvez donc pas passer ce progid à l'instruction `#import`.
