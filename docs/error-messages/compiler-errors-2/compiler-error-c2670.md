# Erreur du compilateur C2670
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : le modèle de fonction ne peut pas convertir le paramètre numéro à partir du type 'type'  
  
 Impossible de convertir un paramètre dans le type requis.  
  
 Cette erreur peut être résolue en créant une conversion explicite.
