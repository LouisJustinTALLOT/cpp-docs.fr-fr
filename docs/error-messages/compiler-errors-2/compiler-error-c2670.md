# <a name="compiler-error-c2670"></a>Erreur du compilateur C2670

'identificateur' : le modèle de fonction ne peut pas convertir le numéro de paramètre du type’type'

Un paramètre n’a pas pu être converti en type requis.

Cette erreur peut être résolue si vous créez une conversion explicite.
