# <a name="compiler-error-c3812"></a>Erreur du compilateur C3812

'property' doit être le premier jeton dans une déclaration de propriété

Lorsque vous déclarez une propriété, le `__property` mot clé doit être le premier jeton sur la ligne.

C3812 est uniquement accessible à l’aide de l’option de compilateur obsolète **/CLR : oldSyntax**.
