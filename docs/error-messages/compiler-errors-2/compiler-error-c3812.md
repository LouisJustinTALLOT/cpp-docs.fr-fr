# <a name="compiler-error-c3812"></a>Erreur du compilateur C3812

'Property’doit être le premier jeton dans une déclaration de propriété

Lors de la déclaration d’une propriété, le `__property` mot clé doit être le premier jeton de la ligne.

C3812 est accessible uniquement à l’aide de l’option de compilateur obsolète **/clr : oldSyntax**.
