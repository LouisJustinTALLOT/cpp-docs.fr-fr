# <a name="compiler-error-c3322"></a>Erreur du compilateur C3322
'propriété' : propriété non valide pour l’attribut 'attribut'  
  
 Une propriété (ou un paramètre) non valide ou non reconnue a été passée à un attribut. Vérifiez la documentation de l’attribut.
