# Erreur du compilateur C2543
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

attendu '\]' pour l'opérateur '\[\]'  
  
 L'opérateur en indice est dépourvu de crochet gauche.  
  
 Cette erreur est peut\-être due à une expansion macro.
