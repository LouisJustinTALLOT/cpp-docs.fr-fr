# <a name="compiler-error-c2721"></a>Erreur du compilateur C2721

'specifier' : spécificateur de classe de stockage non conforme entre le type et le mot clé Operator

Les conversions de type définies par l’utilisateur s’appliquent à toutes les classes de stockage, vous ne pouvez donc pas spécifier une classe de stockage dans une conversion de type.
