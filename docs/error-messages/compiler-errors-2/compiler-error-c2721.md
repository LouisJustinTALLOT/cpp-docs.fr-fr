# Erreur du compilateur C2721
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'spécificateur' : spécificateur de stockage de classe non conforme entre le type et le mot clé operator  
  
 Comme les conversions de types définis par l'utilisateur s'appliquent à toutes les classes de stockage, vous ne pouvez pas spécifier une classe de stockage dans une conversion de type.
