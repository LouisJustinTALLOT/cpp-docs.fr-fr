# <a name="compiler-error-c2721"></a>Erreur du compilateur C2721

'spécificateur' : spécificateur de classe de stockage non conforme entre le mot clé operator

Conversions de type défini par l’utilisateur s’appliquent à toutes les classes de stockage, vous ne pouvez pas spécifier une classe de stockage dans une conversion de type.
