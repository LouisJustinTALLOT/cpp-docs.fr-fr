# Erreur du compilateur C2996
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : définition de modèle de fonction récurrente  
  
 Une définition de fonction essaie d’instancier sa classe basée sur un modèle racine. Les instanciations récurrentes d’un modèle ne sont pas autorisées.
