# <a name="compiler-error-c2517"></a>Erreur du compilateur C2517

'identificateur' : la droite de' :: 'n’est pas défini

L’identificateur à droite de l’opérateur de résolution de portée (`::`) doit être un membre défini de la classe, de la structure ou de l’Union sur la gauche. Si aucune classe, structure ou Union n’est nommée, l’identificateur à droite doit être déclaré avec une portée globale.
