# <a name="compiler-error-c3762"></a>Erreur du compilateur C3762
Impossible de traiter l’attribut 'attribute'  
  
 Un attribut défini par l’utilisateur qui hérite de `System.Security.Permissions.SecurityAttribute` est utilisé pour définir un attribut de sécurité. Ce type d’attribut ne peut pas être utilisé dans le même assembly dans lequel il est défini.
