# <a name="compiler-error-c3805"></a>Erreur du compilateur C3805

'jeton' : jeton inattendu, attendu '}' ou un identificateur

Lorsque vous définissez une propriété, un jeton non valide a été rencontré. Supprimez le jeton non valide.
