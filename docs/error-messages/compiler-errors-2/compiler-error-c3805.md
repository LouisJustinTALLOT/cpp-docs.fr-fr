# <a name="compiler-error-c3805"></a>Erreur du compilateur C3805

'Token' : jeton inattendu, '} 'ou identificateur attendu

Lors de la définition d’une propriété, un jeton non valide a été rencontré. Supprimez le jeton non valide.
