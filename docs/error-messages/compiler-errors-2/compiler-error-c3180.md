# Erreur du compilateur C3180
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'nom de type' : le nom dépasse la limite métadonnées de 'limite' caractères  
  
 Le compilateur a tronqué le nom d'un type managé dans une métadonnée.  La troncation rendra le type inutilisable avec la directive `#using` \(ou de son équivalent dans un autre langage\).  
  
 Le nom du type comprend toute qualification d'espace de noms.
