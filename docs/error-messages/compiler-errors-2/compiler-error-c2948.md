# Erreur du compilateur C2948
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

instanciation explicite ; spécificateur de classe de stockage 'spécificateur' non autorisé sur une spécialisation  
  
 Vous ne pouvez pas utiliser des spécificateurs de classe de stockage \(tels que `extern`\) dans une spécialisation d'une classe de modèle qui a été préalablement explicitement instanciée.
