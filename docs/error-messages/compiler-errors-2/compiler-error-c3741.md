# Erreur du compilateur C3741
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'classe' : doit être une coclasse quand le paramètre 'layout\_dependent' de event\_receiver a la valeur true  
  
 Lorsque `layout_dependent=true`pour une classe [event\_receiver](../../windows/event-receiver.md), la classe doit alors également posséder l'attribut [coclass](../../windows/coclass.md).  
  
 L'exemple suivant génère l'avertissement C3741 :  
  
```  
// C3741.cpp  
// compile with: /c  
// C3741 expected  
#define _ATL_ATTRIBUTES 1  
#include <atlbase.h>  
#include <atlcom.h>  
[module(name="xx")];  
  
[object, uuid("00000000-0000-0000-0000-000000000001")]  
__interface I{ HRESULT f(); };  
  
// Delete the following line to resolve.  
[ event_receiver(com, layout_dependent=true)]  
  
// class or struct must be declared with coclass  
// Uncomment the following line to resolve.  
// [ event_receiver(com, layout_dependent=true), coclass, uuid("00000000-0000-0000-0000-000000000002")]  
struct R : I {  
   HRESULT f(){ return 0; }  
   R(){}  
   R(I* a){ __hook(I, a); }  
};  
```
