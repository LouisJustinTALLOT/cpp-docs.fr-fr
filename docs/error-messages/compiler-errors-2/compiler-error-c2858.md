# Erreur du compilateur C2858
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

option de ligne de commande '\/Yc \(\/FdNomFichier\)' non cohérente avec l'en\-tête précompilé, qui utilisait '\/FdNomFichier'  
  
 La base de données du programme spécifiée par l'option Utiliser un en\-tête précompilé \([\/Yu](../../build/reference/yu-use-precompiled-header-file.md)\) n'est pas celle qui est spécifiée par l'option Créer un en\-tête précompilé \([\/Yc](../../build/reference/yc-create-precompiled-header-file.md)\) précédente.
