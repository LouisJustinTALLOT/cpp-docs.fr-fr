# <a name="compiler-error-c2858"></a>Erreur du compilateur C2858

option de ligne de commande'/YC (/Fdfilename) 'non cohérente avec l’en-tête précompilé, qui utilisait'/Fdfilename'

La base de données du programme spécifiée par l’option utiliser l’en-tête précompilé ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) n’est pas celle spécifiée par l’option créer un en-tête précompilé ([/Yc](../../build/reference/yc-create-precompiled-header-file.md)) précédente.
