# Avertissement du compilateur (niveau&#160;1) C4045
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'array' : dépassement des limites d’index du tableau  
  
 Le nombre d’initialiseurs du tableau est trop élevé. Les initialiseurs redondants vont être ignorés.  
  
 Assurez\-vous que la taille et la quantité des éléments du tableau correspondent à celles des initialiseurs.
