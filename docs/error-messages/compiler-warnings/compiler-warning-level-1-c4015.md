# <a name="compiler-warning-level-1-c4015"></a>Compilateur avertissement (niveau 1) C4015

'identificateur' : type de champ de bits doit être intégraux

Le champ de bits n’est pas déclaré comme type d’entier. Le compilateur suppose que le type de base du champ de bits est non signé. Champs de bits doivent être déclarés en tant que types d’entier non signé.
