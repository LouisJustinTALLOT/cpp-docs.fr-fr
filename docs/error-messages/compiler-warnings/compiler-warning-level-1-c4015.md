# Avertissement du compilateur (niveau 1) C4015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : un champ de bits doit avoir integral comme type  
  
 Le champ de bits n'est pas déclaré comme type entier.  Le compilateur suppose que le type de base du champ de bits est non signé.  Les champs de bits doivent être déclarés comme type entier non signé.
