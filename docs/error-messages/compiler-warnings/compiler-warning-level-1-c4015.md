# <a name="compiler-warning-level-1-c4015"></a>Avertissement du compilateur (niveau 1) C4015

'identificateur' : le type de champ de bits doit être intégral

Le champ de bits n’est pas déclaré en tant que type entier. Le compilateur suppose que le type de base du champ de bits est non signé. Les champs de bits doivent être déclarés en tant que types entiers non signés.
