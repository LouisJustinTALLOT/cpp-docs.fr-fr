# Avertissement du compilateur (niveau 1) C4022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : incompatibilité de pointeur pour le paramètre actuel 'numéro'  
  
 Le type pointeur du paramètre réel est différent du type pointeur du paramètre formel correspondant.  Le paramètre réel est passé sans modification.
