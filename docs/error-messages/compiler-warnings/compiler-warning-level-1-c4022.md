# <a name="compiler-warning-level-1-c4022"></a>Avertissement du compilateur (niveau 1) C4022

'fonction' : incompatibilité de pointeur pour le paramètre réel’nombre'

Le type de pointeur du paramètre réel diffère du type pointeur du paramètre formel correspondant. Le paramètre réel est passé sans modification.
