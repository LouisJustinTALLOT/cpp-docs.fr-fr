# <a name="compiler-warning-level-1-c4034"></a>Avertissement du compilateur (niveau 1) C4034

sizeof retourne 0

L' **`sizeof`** opérateur est appliqué à un opérande de taille zéro (une structure, une Union, une classe ou un type énuméré vide **`void`** ).
