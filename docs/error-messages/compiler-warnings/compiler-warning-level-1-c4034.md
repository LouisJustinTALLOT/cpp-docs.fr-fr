# Avertissement du compilateur (niveau 1) C4034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sizeof retourne 0  
  
 L'opérateur `sizeof` est appliqué à un opérande d'indice zéro \(structure, union, classe ou type énuméré vide, ou bien type `void`\).
