# <a name="compiler-warning-level-1-c4821"></a>Avertissement du compilateur (niveau 1) C4821

> Impossible de déterminer le type d’encodage Unicode, enregistrez le fichier avec la signature (BOM)

Le compilateur n’a pas pu déterminer le type d’encodage d’un fichier. Pour résoudre cet avertissement, enregistrez le fichier avec un marqueur d’ordre d’octet. Pour plus d’informations, consultez [gérer les fichiers avec encodage](/sql/ssms/solution/manage-files-with-encoding) .
