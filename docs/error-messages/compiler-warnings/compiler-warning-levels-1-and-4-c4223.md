# Avertissement du compilateur (niveaux 1 et 4) C4223
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

extension non standard utilisée : tableau de valeurs non lvalue converti en pointeur  
  
 En C standard, vous ne pouvez pas convertir un tableau non lvalue vers un pointeur.  Avec les extensions Microsoft par défaut \([\/Ze](../../build/reference/za-ze-disable-language-extensions.md)\), vous pouvez le faire.
