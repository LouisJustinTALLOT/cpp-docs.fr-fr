# <a name="compiler-warning-levels-1-and-4-c4223"></a>Avertissement du compilateur (niveaux 1 et 4) C4223

extension non standard utilisée : tableau non lvalue converti en pointeur

En C standard, vous ne pouvez pas convertir un tableau non lvalue en pointeur. Avec les extensions Microsoft par défaut ([/Ze](../../build/reference/za-ze-disable-language-extensions.md)), vous pouvez.
