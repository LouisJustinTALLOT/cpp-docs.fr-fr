# Avertissement du compilateur (niveau 1) C4411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : le symbole se résout en registre de déplacement  
  
 L'identificateur est un symbole local qui se résout en un registre de déplacement et peut donc être utilisé sur un opérande avec un autre symbole.
