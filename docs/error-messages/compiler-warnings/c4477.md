# <a name="compiler-warning-level-1-c4477"></a>Compilateur Warning (level 1) C4477

> '*fonction*' : chaîne de format '*chaîne*'requiert un argument de type'*type*', mais l’argument variadique *nombre* a le type '*type*'

Le compilateur a détecté une incompatibilité entre le type d’argument requis pour satisfaire l’espace réservé dans une chaîne de format et le type d’argument fourni. Utilisation correcte de la famille printf et scanf des fonctions variadiques, vous devez fournir les arguments de types spécifiés par la chaîne de format. Une incompatibilité signifie généralement qu'il existe un bogue dans votre code.

Pour plus d’informations sur les arguments associés à des espaces réservés de chaîne de format famille printf, consultez [syntaxe de spécification de Format : fonctions printf et wprintf](../../c-runtime-library/format-specification-syntax-printf-and-wprintf-functions.md). Consultez la documentation pour plus d’informations spécifiques à la fonction *fonction*.

Cet avertissement est une nouveauté dans Visual Studio 2015.

## <a name="example"></a>Exemple

Cet exemple montre deux avertissements C4477 pour la première fonction printf_s, lorsque deux arguments sont identifiés comme étant de type incorrect et montre également comment résoudre les problèmes.

```cpp
// C4477p.cpp
// compile with: cl /c /W4 C4477p.cpp
#include <stdio.h>

void print_func(int a, float f)
{
    printf_s("This takes int and double arguments %d and %f.\n", f, a); // C4477, expected int then double
    printf_s("This takes int and double arguments %d and %f.\n", a, f); // fix: types in correct order
}
```

Dans le premier message d’erreur, le compilateur détecte qu’un **double** argument est fourni lorsqu’une **int** est attendu. Un **float** est automatiquement promue en **double** dans la liste d’arguments variadiques. Le deuxième message d’erreur indique que le compilateur attendu un **double** argument, mais un **int** a été fourni. Le compilateur ne peut pas savoir que vous avez échangé l’ordre des arguments. Vous devez examiner la chaîne de format pour déterminer l’ordre de l’argument réel et de fournir les types d’arguments corrects.
