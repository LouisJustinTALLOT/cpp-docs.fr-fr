# <a name="compiler-warning-level-1-c4175"></a>Avertissement du compilateur (niveau 1) C4175

\#pragma Component (Browser, on) : les informations du navigateur doivent être initialement spécifiées sur la ligne de commande

Pour utiliser pragma [component](../../preprocessor/component.md) , vous devez générer des informations de consultation pendant la compilation ([/FR](../../build/reference/fr-fr-create-dot-sbr-file.md)).
