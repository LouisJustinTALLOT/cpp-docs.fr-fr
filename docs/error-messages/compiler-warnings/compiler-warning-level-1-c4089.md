# <a name="compiler-warning-level-1-c4089"></a>Compilateur avertissement (niveau 1) C4089
'fonction' : types différents dans le paramètre actuel 'nombre', paramètre formel 'nombre'  
  
 Les paramètres formels et réels correspondants ont des types différents. Le paramètre réel est transmis sans modification. La fonction convertit le paramètre réel au type spécifié dans la définition de fonction.
