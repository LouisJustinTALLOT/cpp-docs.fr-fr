# <a name="compiler-warning-level-1-c4089"></a>Avertissement du compilateur (niveau 1) C4089

'Function' : types différents dans le paramètre réel’Number', paramètre formel’Number'

Les paramètres formels et réels correspondants ont des types différents. Le paramètre réel est passé sans modification. La fonction effectue un cast du paramètre réel en type spécifié dans la définition de fonction.
