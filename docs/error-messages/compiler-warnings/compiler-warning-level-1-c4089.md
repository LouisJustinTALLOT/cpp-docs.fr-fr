# Avertissement du compilateur (niveau 1) C4089
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : types différents pour le paramètre actuel 'nombre' et le paramètre formel 'numéro'  
  
 Les paramètres formel et réel correspondants sont de types différents.  Le paramètre réel est passé sans modification.  La fonction effectue un cast du paramètre actuel en type spécifié dans la définition de la fonction.
