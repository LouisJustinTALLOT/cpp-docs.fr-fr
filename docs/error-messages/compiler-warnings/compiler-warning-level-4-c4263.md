# Avertissement du compilateur (niveau 4) C4263
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : une fonction membre ne se substitue à aucune fonction membre virtuelle de classe de base  
  
 Une définition de fonction de classe a le même nom qu'une fonction virtuelle dans une classe de base mais pas le même nombre ou type d'arguments.  Ceci a pour effet de masquer la fonction virtuelle dans la classe de base.  
  
 Cet avertissement est désactivé par défaut.  Pour plus d'informations, consultez [Avertissements du compilateur désactivés par défaut](../../preprocessor/compiler-warnings-that-are-off-by-default.md).  
  
 L'exemple suivant génère l'erreur C4263 :  
  
```  
// C4263.cpp  
// compile with: /W4  
#pragma warning(default:4263)  
#pragma warning(default:4264)  
class B {  
public:  
   virtual void func();  
};  
  
class D : public B {  
   void func(int);   // C4263  
};  
  
int main() {  
}  
```
