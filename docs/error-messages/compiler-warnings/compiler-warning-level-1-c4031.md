# Avertissement du compilateur (niveau 1) C4031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

seconde liste de paramètres formels plus longue que la première liste  
  
 Une fonction est redéclarée avec des paramètres formels différents.  Le compilateur utilise les paramètres formels donnés dans la première déclaration.
