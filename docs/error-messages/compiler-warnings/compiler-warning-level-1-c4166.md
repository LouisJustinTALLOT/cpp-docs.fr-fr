# Avertissement du compilateur (niveau&#160;1) C4166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**convention d'appel non conforme pour un constructeur\/destructeur**  
  
 Les constructeurs et les destructeurs ne peuvent pas avoir de conventions d’appel autres que celles par défaut de la plateforme \(sauf si vous spécifiez explicitement **\_\_clrcall**\).
