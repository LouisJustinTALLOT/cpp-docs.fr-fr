# <a name="compiler-warning-level-1-c4166"></a>Avertissement du compilateur (niveau 1) C4166

**convention d'appel non conforme pour un constructeur/destructeur**

Les constructeurs et les destructeurs ne peuvent pas avoir de conventions d’appel autres que celles par défaut de la plateforme (sauf si vous spécifiez explicitement **__clrcall**).
