# <a name="compiler-warning-level-2-c4948"></a>Avertissement du compilateur (niveau 2) C4948

le type de retour de’accessor’ne correspond pas au dernier type de paramètre de la méthode setter correspondante

Le compilateur a détecté une incompatibilité entre le type de données obtenu et défini pour une propriété indexée.

C4948 est accessible uniquement à l’aide de l’option de compilateur obsolète **/clr : oldSyntax**.
