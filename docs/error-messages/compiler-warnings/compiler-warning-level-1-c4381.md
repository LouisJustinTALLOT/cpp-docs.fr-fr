# <a name="compiler-warning-level-1-c4381"></a>Avertissement du compilateur (niveau 1) C4381
'fonction1' : méthode d’interface ne sera pas implémentée par une méthode non publique 'fonction2'  
  
 Une classe doit implémenter toutes les fonctions dans une interface. Une classe peut répondre à cette condition si une de ses classes de base implémente la fonction. Toutefois, la fonction doit être implémentée comme une fonction publique.
