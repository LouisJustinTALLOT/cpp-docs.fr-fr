# Avertissement du compilateur (niveau 1) C4381
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction1' : la méthode d'interface ne sera pas implémentée par une méthode non publique 'fonction2'  
  
 Une classe doit implémenter toutes les fonctions dans une interface.  Une classe peut répondre à cette condition si l'une de ses classes de base implémente la fonction.  Toutefois, la fonction doit être implémentée en tant que fonction publique.
