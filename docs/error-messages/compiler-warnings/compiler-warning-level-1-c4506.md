# <a name="compiler-warning-level-1-c4506"></a>Avertissement du compilateur (niveau 1) C4506

aucune définition pour la fonction inline’fonction'

La fonction donnée a été déclarée et marquée pour l’incorporation, mais elle n’a pas été définie.

Le compilateur n’a pas inséré la fonction.

Assurez-vous que les fonctions externes à Inline sont déclarées avec le **`extern`** mot clé.
