# Avertissement du compilateur (niveau 1) C4506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

aucune définition pour la fonction inline 'fonction'  
  
 La fonction donnée a été déclarée et marquée comme fonctionnalité inline mais n'a pas été définie.  
  
 Le compilateur n'a pas inclus la fonction inline.  
  
 Vérifiez que les fonctions externes à inclure en ligne sont déclarées avec le mot clé `extern`.
