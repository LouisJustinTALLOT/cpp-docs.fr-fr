# Avertissement du compilateur (niveau 4) C4515
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'EspaceNoms' : l'espace de noms s'utilise lui\-même  
  
 Un espace de noms est appelé de façon récursive.  
  
 L'exemple suivant génère l'erreur C4515 :  
  
```  
// C4515.cpp  
// compile with: /W4  
namespace A  
{  
   using namespace A; // C4515  
}  
  
int main()  
{  
}  
```
