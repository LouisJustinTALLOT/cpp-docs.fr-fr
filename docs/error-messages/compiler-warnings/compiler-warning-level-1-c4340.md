# <a name="compiler-warning-level-1-c4340"></a>Avertissement du compilateur (niveau 1) C4340

'valeur' : valeur passée d’une valeur positive à une valeur négative

La **`enum`** valeur est supérieure à la plus grande **`enum`** valeur positive incluse dans une valeur négative.
