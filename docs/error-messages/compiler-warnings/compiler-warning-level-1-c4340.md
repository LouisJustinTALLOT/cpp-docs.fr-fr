# Avertissement du compilateur (niveau 1) C4340
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'valeur' : la valeur est passée d'une valeur positive à une valeur négative  
  
 La valeur `enum` est supérieure à la plus grande valeur positive `enum`, elle est donc passée à une valeur négative.
