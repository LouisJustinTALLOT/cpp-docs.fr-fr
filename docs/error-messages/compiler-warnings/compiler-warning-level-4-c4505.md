# Avertissement du compilateur (niveau 4) C4505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : la fonction locale non référencée a été supprimée  
  
 La fonction donnée est locale et non référencée dans le corps du module ; la fonction est donc du code mort.  
  
 Le compilateur n'a pas généré de code pour cette fonction inutile.
