# <a name="compiler-warning-level-4-c4505"></a>Avertissement du compilateur (niveau 4) C4505

'fonction' : la fonction locale non référencée a été supprimée

La fonction donnée est locale et n’est pas référencée dans le corps du module ; par conséquent, la fonction est du code mort.

Le compilateur n’a pas généré de code pour cette fonction morte.
