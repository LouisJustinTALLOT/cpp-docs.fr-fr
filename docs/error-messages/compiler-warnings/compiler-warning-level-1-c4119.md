# <a name="compiler-warning-level-1-c4119"></a>Avertissement du compilateur (niveau 1) C4119

bases différentes 'base1' et 'base2' spécifiées

Deux pointeurs based sont incompatibles, car ils ont des bases différentes. Le compilateur ne peut pas effectuer une conversion entre eux.
