# <a name="compiler-warning-level-3-c4133"></a>Avertissement du compilateur (niveau 3) C4133

'type' : types incompatibles-de’type1 'à’type2 '

Cet avertissement peut être provoqué par une tentative de soustraction de deux pointeurs de types différents.

Pour éviter cet avertissement, fournissez un cast de type approprié.
