# <a name="compiler-warning-level-1-c4685"></a>Avertissement du compilateur (niveau 1) C4685

'> >' attendu, '>>' trouvé lors de l'analyse des paramètres du modèle

Une définition de modèle n’a pas été terminée correctement.
