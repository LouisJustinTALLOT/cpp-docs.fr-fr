# Avertissement du compilateur (niveau 1) C4264
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction\_virtuelle' : aucune substitution disponible pour la fonction membre virtuelle à partir de la base 'classe' ; la fonction est masquée  
  
 C4264 est toujours généré après [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md).  
  
 Cet avertissement est désactivé par défaut.  Pour plus d'informations, consultez [Avertissements du compilateur désactivés par défaut](../../preprocessor/compiler-warnings-that-are-off-by-default.md).
