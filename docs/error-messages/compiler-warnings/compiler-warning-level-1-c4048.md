# <a name="compiler-warning-level-1-c4048"></a>Du compilateur (niveau 1) d’avertissement C4048
autre déclaré indices de tableau : 'identificateur1' et 'identificateur2'  
  
 Une expression comprend des pointeurs vers des tableaux de taille différente. Les pointeurs sont utilisés sans conversion.  
  
 Cet avertissement peut être résolu si vous convertir explicitement les tableaux de type identique ou équivalent.
