# Avertissement du compilateur (niveau 1) C4048
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

indices de tableau différents : 'identificateur1' et 'identificateur2'  
  
 Une expression comprend des pointeurs vers des tableaux de taille différente.  Les pointeurs sont utilisés sans conversion.  
  
 Cet avertissement peut être résolu en effectuant un cast explicite des tableaux en un type identique ou équivalent.
