# Avertissement du compilateur (niveau&#160;1) C4900
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

incompatibilité de langage intermédiaire entre 'tool1' version 'version1' et 'tool2' version 'version2'  
  
 Les langages intermédiaires utilisés dans *tool1* et *tool2* ne concordent pas. Vérifiez si la version la plus actuelle de chaque outil a bien été installée.
