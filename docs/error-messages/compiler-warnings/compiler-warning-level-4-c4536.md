# Avertissement du compilateur (niveau 4) C4536
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'nom de type' : le nom de type dépasse la limite métadonnées de 'limite' caractères  
  
 Un nom de type serait tronqué dans les métadonnées s'il s'agissait d'un type managé.  Pour plus d'informations, consultez [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md).  
  
 Cet avertissement est désactivé par défaut.  Pour plus d'informations, consultez [Avertissements du compilateur désactivés par défaut](../../preprocessor/compiler-warnings-that-are-off-by-default.md).
