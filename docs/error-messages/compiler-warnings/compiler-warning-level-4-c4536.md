# <a name="compiler-warning-level-4-c4536"></a>Avertissement du compilateur (niveau 4) C4536

'type name' : nom de type dépasse la limite métadonnées de 'limite' caractères

Un nom de type serait tronqué dans les métadonnées s’il s’agissait d’un type managé. Consultez [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md) pour plus d’informations.

Cet avertissement est désactivé par défaut. Consultez [Avertissements du compilateur désactivés par défaut](../../preprocessor/compiler-warnings-that-are-off-by-default.md) pour plus d'informations.
