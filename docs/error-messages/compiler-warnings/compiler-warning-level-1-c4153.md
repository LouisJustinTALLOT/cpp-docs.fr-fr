# <a name="compiler-warning-level-1-c4153"></a>Avertissement du compilateur (niveau 1) C4153

conversion pointeur fonction/donnée dans l'expression

Un pointeur de fonction est converti en ou à partir d’un pointeur de données. Cette conversion est autorisée sous les extensions Microsoft (/Ze) mais pas sous ANSI C.
