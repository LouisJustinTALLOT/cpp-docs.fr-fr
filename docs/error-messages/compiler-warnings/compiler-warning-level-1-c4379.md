# Avertissement du compilateur (niveau 1) C4379
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La version 'version' du Common Language Runtime n'est pas prise en charge par ce compilateur.L'utilisation de cette version peut provoquer des résultats inattendus.  
  
 Vous disposez d'une version antérieure du Common Language Runtime sur votre ordinateur, mais pas de la version actuelle.  Pour remédier à l'erreur C4379, installez la version du Common Language Runtime qui a été fournie avec votre compilateur.
