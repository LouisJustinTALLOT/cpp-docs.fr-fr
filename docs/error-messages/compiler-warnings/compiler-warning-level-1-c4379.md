# <a name="compiler-warning-level-1-c4379"></a>Avertissement du compilateur (niveau 1) C4379

La version de la version de la common language runtime n’est pas prise en charge par ce compilateur. L’utilisation de cette version peut provoquer des résultats inattendus.

Vous disposez d’une version précédente du common language runtime sur votre ordinateur, mais pas de la version actuelle. Pour résoudre c4379, installez la version du common language runtime fournie avec votre compilateur.
