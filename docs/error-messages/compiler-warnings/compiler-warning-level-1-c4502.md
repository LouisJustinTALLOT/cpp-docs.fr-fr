# <a name="compiler-warning-level-1-c4502"></a>Avertissement du compilateur (niveau 1) C4502

'Linking Specification’nécessite l’utilisation du mot clé’extern’et doit précéder tous les autres spécificateurs

Une liaison a été spécifiée sans le **`extern`** mot clé. La liaison n’est pas pertinente pour les types non-extern.

Le compilateur a pris le **`extern`** mot clé.
