# Avertissement du compilateur (niveau 1) C4502
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'linkage specification' nécessite l'utilisation du mot clé 'extern' et doit précéder tous les autres spécificateurs  
  
 Un lien a été spécifié sans le mot clé `extern`.  Le lien ne concerne pas les types non externes.  
  
 Le compilateur a supposé que le mot clé `extern` était présent.
