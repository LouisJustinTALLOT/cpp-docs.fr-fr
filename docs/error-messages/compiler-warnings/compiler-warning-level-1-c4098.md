# <a name="compiler-warning-level-1-c4098"></a>Avertissement du compilateur (niveau 1) C4098

'fonction' : fonction void retournant une valeur

Une fonction déclarée avec le type de retour [void](../../cpp/void-cpp.md) a une **`return`** instruction qui retourne une valeur. Le compilateur suppose que la fonction retourne une valeur de type **`int`** .
