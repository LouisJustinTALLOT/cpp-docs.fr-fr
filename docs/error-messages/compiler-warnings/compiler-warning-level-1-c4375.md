# <a name="compiler-warning-level-1-c4375"></a>Avertissement du compilateur (niveau 1) C4375

la méthode non publique’méthode2 'ne se substitue pas à’méthode1 '

Type qui implémente une méthode de substitution définie par un autre type, mais la substitution n’est pas publique. Par conséquent, la méthode ne remplace pas la méthode de type de base.
