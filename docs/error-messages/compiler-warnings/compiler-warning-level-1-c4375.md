# Avertissement du compilateur (niveau 1) C4375
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la méthode non publique 'méthode2' ne substitue pas 'méthode1'  
  
 Un type qui implémente un autre type a défini une méthode override, mais celle\-ci n'était pas publique.  Par conséquent, cette méthode ne substitue pas la méthode du type de base.
