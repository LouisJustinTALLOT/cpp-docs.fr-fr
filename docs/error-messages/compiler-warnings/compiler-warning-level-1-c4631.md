# Avertissement du compilateur (niveau&#160;1) C4631
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

MSXML ou XPath non disponible, les commentaires de document XML ne seront pas traités. raison  
  
 Votre installation du Common Language Runtime n’a pas les fichiers nécessaires pour prendre en charge les commentaires de document de traitement. Réinstallez le Common Language Runtime.
