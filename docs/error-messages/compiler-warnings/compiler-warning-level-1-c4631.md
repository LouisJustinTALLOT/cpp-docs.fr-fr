# <a name="compiler-warning-level-1-c4631"></a>Avertissement du compilateur (niveau 1) C4631

MSXML ou XPath non disponible, les commentaires de document XML ne seront pas traités. reason

Votre installation du Common Language Runtime n’a pas les fichiers nécessaires pour prendre en charge les commentaires de document de traitement. Réinstallez le Common Language Runtime.
