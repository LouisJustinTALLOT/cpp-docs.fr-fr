# <a name="compiler-warning-level-1-c4403"></a>Avertissement du compilateur (niveau 1) C4403

opérateur PTR non conforme

Un opérateur PTR est utilisé de façon inappropriée dans le code assembleur inline.

Le compilateur a ignoré l’opérateur PTR.
