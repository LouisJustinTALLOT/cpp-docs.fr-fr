# Avertissement du compilateur (niveau 1) C4403
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opérateur PTR non conforme  
  
 Un opérateur PTR est utilisé de façon inappropriée dans du code assembleur inline.  
  
 Le compilateur a ignoré l'opérateur PTR.
