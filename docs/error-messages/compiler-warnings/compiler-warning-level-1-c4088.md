# Avertissement du compilateur (niveau 1) C4088
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction : incompatibilité de pointeur dans le paramètre actuel 'numéro', paramètre formel 'numéro'  
  
 Les paramètres formel et réel correspondants ont un niveau d'indirection différent.  Le paramètre réel est passé sans modification.  La fonction appelée interprète sa valeur comme un pointeur.
