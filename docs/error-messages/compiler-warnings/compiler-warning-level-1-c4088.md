# <a name="compiler-warning-level-1-c4088"></a>Avertissement du compilateur (niveau 1) C4088

'fonction' : incompatibilité de pointeur dans le paramètre réel’Number', paramètre formel’Number'

Les paramètres formels et réels correspondants ont un niveau d’indirection différent. Le paramètre réel est passé sans modification. La fonction appelée interprète sa valeur comme un pointeur.
