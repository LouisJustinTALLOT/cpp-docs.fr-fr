# Avertissement du compilateur (niveau&#160;1) C4811
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

valeur de pragma conform\(forScope, show\) \=\= value  
  
 Cet avertissement s’affiche quand vous utilisez l’option **show** du pragma [conform](../../preprocessor/conform.md).*value* est la valeur conforme actuelle.
