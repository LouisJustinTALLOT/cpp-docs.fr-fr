# <a name="compiler-warning-level-1-c4406"></a>Avertissement du compilateur (niveau 1) C4406

opérande sur directive ignoré

La directive ne prend pas d’opérandes, mais un opérande a été spécifié.

Le compilateur a ignoré l’opérande ou les opérandes donnés.
