# Avertissement du compilateur (niveau 1) C4406
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opérande sur directive ignoré  
  
 La directive ne prend pas d'opérandes, mais un opérande a été spécifié.  
  
 Le compilateur a ignoré le ou les opérandes donnés.
