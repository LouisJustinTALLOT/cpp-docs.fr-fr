# Avertissement du compilateur (niveau&#160;1) C4120
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

incompatibilité basée\/non basée  
  
 Le compilateur ne peut pas effectuer de conversion entre les deux pointeurs, car l’un d’eux est basé et l’autre non.
