# <a name="compiler-warning-level-1-c4143"></a>Avertissement du compilateur (niveau 1) C4143

pragma 'same_seg' non pris en charge ; utilisez l'allocation __based

**#pragma same_seg** n’est plus pris en charge. Utilisez plutôt le mot clé [__based](../../cpp/based-pointers-cpp.md) .
