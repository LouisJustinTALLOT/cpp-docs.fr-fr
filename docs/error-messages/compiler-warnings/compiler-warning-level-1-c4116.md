# <a name="compiler-warning-level-1-c4116"></a>Avertissement du compilateur (niveau 1) C4116

définition de type sans nom entre parenthèses

Une structure, une Union ou un type énuméré sans nom est défini dans une expression entre parenthèses. La définition du type n’a pas de sens.

Dans un appel de fonction C, la définition a une portée globale. Dans un C++ appel de fonction, la définition a la même portée que la fonction appelée.
