# Avertissement du compilateur C4961
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Aucune donnée de profil n’a été fusionnée dans 'fichier .pgd', les optimisations guidées par profil sont désactivées  
  
 Les optimisations guidées par profil ne peuvent pas être effectuées en raison de l’absence de données de profil \(fichiers .pgc\).
