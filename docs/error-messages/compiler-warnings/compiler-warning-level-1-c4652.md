# <a name="compiler-warning-level-1-c4652"></a>Avertissement du compilateur (niveau 1) C4652

option du compilateur 'option' non cohérente avec l’en-tête précompilé ; option de ligne de commande en cours se substituera à celle définie dans l’en-tête précompilé

L’option de ligne de commande donnée est différent de celui donné lors de la création de l’en-tête précompilé (.pch). L’option spécifiée dans la ligne de commande actuelle a été utilisée.

Cet avertissement peut être évité par la régénération de l’en-tête précompilé avec l’option de ligne de commande donnée.
