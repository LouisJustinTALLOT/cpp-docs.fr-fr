# <a name="compiler-warning-level-1-c4652"></a>Avertissement du compilateur (niveau 1) C4652

option du compilateur’option’non cohérente avec l’en-tête précompilé ; l’option de ligne de commande actuelle remplace celle qui est définie dans l’en-tête précompilé

L’option de ligne de commande donnée diffère de celle donnée lorsque l’en-tête précompilé (. pch) a été créé. L’option spécifiée dans la ligne de commande actuelle a été utilisée.

Cet avertissement peut être évité en régénérant l’en-tête précompilé avec l’option de ligne de commande donnée.
