# Avertissement du compilateur (niveau 3) C4013
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' non définie ; extern retournant int pris par défaut  
  
 Le compilateur a rencontré un appel à une fonction non définie.  
  
### Pour résoudre le problème en vérifiant les causes possibles suivantes  
  
1.  Orthographe incorrecte du nom de fonction  
  
2.  Fonctions externes non prototypées comme `extern`
