# <a name="compiler-warning-level-3-c4013"></a>Compilateur avertissement (niveau 3) C4013

'fonction' non définie ; extern retournant int

Le compilateur a rencontré un appel à une fonction non définie.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes

1. Orthographe incorrecte du nom de fonction

1. Fonctions externes non prototypées comme `extern`
