# <a name="compiler-warning-level-3-c4511"></a>Avertissement du compilateur (niveau 3) C4511

'class' : constructeur de copie n’a pas pu être généré.

Le compilateur n’a pas pu générer un constructeur de copie par défaut pour une classe ; une classe de base peut avoir un constructeur de copie privé.
