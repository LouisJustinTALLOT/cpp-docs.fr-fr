# <a name="compiler-warning-level-3-c4511"></a>Avertissement du compilateur (niveau 3) C4511

'classe' : impossible de générer le constructeur de copie

Le compilateur n’a pas pu générer de constructeur de copie par défaut pour une classe ; une classe de base peut avoir un constructeur de copie privé.
