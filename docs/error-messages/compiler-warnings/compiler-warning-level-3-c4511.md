# Avertissement du compilateur (niveau 3) C4511
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'classe' : le constructeur de copie n'a pas pu être généré  
  
 Le compilateur n'a pas pu générer un constructeur de copie par défaut pour une classe ; une classe de base pourrait avoir un constructeur de copie privé.
