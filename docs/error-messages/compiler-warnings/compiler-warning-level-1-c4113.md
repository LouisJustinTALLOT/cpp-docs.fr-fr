# <a name="compiler-warning-level-1-c4113"></a>Avertissement du compilateur (niveau 1) C4113

'identificateur1 'diffère de’identificateur2 'dans les listes de paramètres

Un pointeur de fonction est assigné à un autre pointeur de fonction, mais les listes de paramètres formels des fonctions ne sont pas d’accord. L’assignation est compilée sans modification.
