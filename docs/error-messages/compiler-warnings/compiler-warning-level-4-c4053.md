# <a name="compiler-warning-level-4-c4053"></a>Avertissement du compilateur (niveau 4) C4053
un opérande void pour '?:'  
  
 L’opérateur `?:` reçoit une expression de type `void`. La valeur de l’opérande `void` n’est pas définie.
