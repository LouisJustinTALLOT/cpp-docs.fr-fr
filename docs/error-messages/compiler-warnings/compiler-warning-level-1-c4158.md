# <a name="compiler-warning-level-1-c4158"></a>Avertissement du compilateur (niveau 1) C4158
#pragma pointers_to_members(full_generality, inheritance) pris par défaut  
  
 A **#pragma pointers_to_members (***unique &#124; plusieurs &#124; virtuel***)** a été émis sans être accompagnée de **#pragma pointers_to_ Members(full_generality)**.
