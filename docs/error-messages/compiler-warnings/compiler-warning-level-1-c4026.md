# <a name="compiler-warning-level-1-c4026"></a>Avertissement du compilateur (niveau 1) C4026

fonction déclarée avec une liste de paramètres formels

La déclaration de fonction a des paramètres formels, mais pas la définition de fonction. Les appels ultérieurs à cette fonction supposent que la fonction n’accepte aucun paramètre.
