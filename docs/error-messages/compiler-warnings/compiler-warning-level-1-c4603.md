# Avertissement du compilateur (niveau&#160;1) C4603
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\<identificateur\>' : la macro n’est pas définie ou la définition est différente après utilisation de l’en\-tête précompilé  
  
 La macro spécifiée par l’espace réservé *identificateur* est différente ou n’est plus définie après l’utilisation de l’en\-tête précompilé.
