# Avertissement du compilateur (niveau 4) C4206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**extension non standard utilisée : unité de traduction vide**  
  
 Le fichier était vide après passage par le préprocesseur.  
  
 Cette extension peut empêcher le portage de votre code vers d'autres compilateurs.  Elle génère une erreur sous compatibilité ANSI \([\/Za](../../build/reference/za-ze-disable-language-extensions.md)\) et ne s'applique qu'au code source C.
