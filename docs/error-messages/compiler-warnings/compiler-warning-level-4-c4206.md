# <a name="compiler-warning-level-4-c4206"></a>Avertissement du compilateur (niveau 4) C4206

**extension non standard utilisée : unité de traduction vide**

Le fichier était vide après le prétraitement.

Cette extension peut empêcher votre code d’être portable vers d’autres compilateurs. Elle génère une erreur sous compatibilité ANSI ([/za](../../build/reference/za-ze-disable-language-extensions.md)) et s’applique uniquement au code source C.
