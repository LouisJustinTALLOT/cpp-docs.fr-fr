# Avertissement du compilateur (niveau 1) C4679
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Le compilateur a rencontré une construction qu'il ne peut pas prendre en charge, qui ne peut pas être importée des métadonnées.  
  
 **'**   
 ***membre* ' : impossible d'importer le membre**  
  
 N'essayez pas d'utiliser cette construction.
