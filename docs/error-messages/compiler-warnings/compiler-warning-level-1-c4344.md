# Avertissement du compilateur (niveau 1) C4344
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

changement de comportement : l'utilisation d'arguments template explicites entraîne l'appel de 'fonction'  
  
 Un appel à une fonction utilisant des arguments template explicites appelle une autre fonction que celle qu'il appellerait si des arguments explicites n'avaient pas été spécifiés
