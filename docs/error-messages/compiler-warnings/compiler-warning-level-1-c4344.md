# <a name="compiler-warning-level-1-c4344"></a>Avertissement du compilateur (niveau 1) C4344

changement de comportement : l’utilisation d’arguments template explicites entraîne l’appel de’Function'

Un appel à une fonction à l’aide d’arguments template explicites appelle une fonction différente de celle qui serait si des arguments explicites n’avaient pas été spécifiés
