# Avertissement du compilateur (niveau 4) C4639
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erreur MSXML, les commentaires de document XML ne seront pas traités. raison  
  
 Cet avertissement peut se produire pour différentes raisons.  
  
 Pour résoudre cet avertissement :  
  
-   Recompilez.  
  
-   Réinstallez MSXML en réinstallant le Common Language Runtime.  
  
-   Modifiez ou supprimez les commentaires de documentation entraînant l'avertissement et recompilez.  
  
 Lorsque l'erreur C4639 est générée, le traitement de tout autre commentaire XML est désactivé, et le fichier .xdc n'est pas généré.
