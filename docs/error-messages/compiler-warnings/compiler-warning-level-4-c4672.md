# <a name="compiler-warning-level-4-c4672"></a>Avertissement du compilateur (niveau 4) C4672
'identifier1' : ambigu. Déjà rencontré comme 'identifier2'  
  
 L’objet spécifié devant être levé dans un bloc **try** est ambigu. L’ambiguïté de l’objet ne peut pas être enlevée si ce dernier est levé.
