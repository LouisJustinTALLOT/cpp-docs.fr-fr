# Avertissement du compilateur (niveau 1) C4402
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

doit utiliser l'opérateur PTR  
  
 Un type est utilisé sur un opérande sans l'opérateur PTR lors d'une référence ou d'un cast vers un type en code assembleur inline.  
  
 Le compilateur suppose que l'opérateur PTR est présent.
