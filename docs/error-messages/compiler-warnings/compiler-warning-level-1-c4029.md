# <a name="compiler-warning-level-1-c4029"></a>Avertissement du compilateur (niveau 1) C4029

liste de paramètres formels déclarée différente de la définition

Les types de paramètres formels dans la déclaration de fonction ne correspondent pas à ceux de la définition de fonction. Le compilateur utilise la liste de paramètres de la définition.
