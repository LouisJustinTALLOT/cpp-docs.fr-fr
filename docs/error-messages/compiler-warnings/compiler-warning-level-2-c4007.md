# Avertissement du compilateur (niveau 2) C4007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : doit être 'attribut'  
  
 Un attribut requis pour une fonction n'est pas mentionné explicitement.  Par exemple, la fonction **main** doit avoir l'attribut `__cdecl`.  Le compilateur force l'attribut.
