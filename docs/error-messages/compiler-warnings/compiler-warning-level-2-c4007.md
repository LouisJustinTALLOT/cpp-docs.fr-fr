# <a name="compiler-warning-level-2-c4007"></a>Avertissement du compilateur (niveau 2) C4007

'fonction' : doit être’Attribute'

Un attribut requis pour une fonction n’est pas explicitement indiqué. Par exemple, la fonction **main** doit avoir l' **`__cdecl`** attribut. Le compilateur force l’attribut.
