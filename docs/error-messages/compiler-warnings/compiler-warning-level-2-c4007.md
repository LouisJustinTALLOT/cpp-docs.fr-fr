# <a name="compiler-warning-level-2-c4007"></a>Compilateur avertissement (niveau 2) C4007

'fonction' : doit être 'attribut'

Un attribut requis pour une fonction n’est pas défini explicitement. Par exemple, la fonction **principal** doit avoir le `__cdecl` attribut. Le compilateur force l’attribut.
