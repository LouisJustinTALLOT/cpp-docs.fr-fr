# Avertissement du compilateur (niveau 1) C4024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : types différents pour le paramètre formel et actuel 'numéro'  
  
 Les paramètres formel et réel correspondants sont de types différents.  Le compilateur passe le paramètre réel sans modification.  La fonction réceptrice convertit le type de paramètre en type attendu.
