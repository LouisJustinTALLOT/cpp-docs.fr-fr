# <a name="compiler-warning-level-1-c4024"></a>Compilateur avertissement (niveau 1) C4024
'fonction' : types différents pour le paramètre formel et réel 'nombre'  
  
 Les paramètres formels et réels correspondants ont des types différents. Le compilateur passe le paramètre réel sans modification. La fonction de réception convertit le type de paramètre pour le type attendu.
