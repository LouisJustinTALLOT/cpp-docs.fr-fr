# <a name="compiler-warning-level-3-c4543"></a>Avertissement du compilateur (niveau 3) C4543

Texte injecté supprimé par l’attribut’no_injected_text'

L’attribut [no_injected_text](../../windows/attributes/no-injected-text.md) apparaissait dans le code source, ce qui signifie que le compilateur empêchera les attributs d’injecter du code.

Cet avertissement est destiné à vous rappeler que les attributs ne seront pas en mesure d’injecter du code.
