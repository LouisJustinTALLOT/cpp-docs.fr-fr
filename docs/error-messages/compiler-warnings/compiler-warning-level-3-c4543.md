# <a name="compiler-warning-level-3-c4543"></a>Avertissement du compilateur (niveau 3) C4543

Texte injecté supprimé par l’attribut 'no_injected_text'

Le [no_injected_text](../../windows/no-injected-text.md) attribut est apparu dans le code source, ce qui signifie que le compilateur empêchera les attributs d’injecter du code.

Cet avertissement consiste à vous rappeler que les attributs ne seront pas en mesure d’injecter du code.
