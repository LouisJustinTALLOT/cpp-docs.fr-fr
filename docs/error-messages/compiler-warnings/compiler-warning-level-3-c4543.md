# Avertissement du compilateur (niveau 3) C4543
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Texte injecté supprimé par l'attribut 'no\_injected\_text'  
  
 L'attribut [no\_injected\_text](../../windows/no-injected-text.md) est apparu dans le code source, ce qui signifie que le compilateur empêchera les attributs d'injecter du code.  
  
 Cet avertissement vous rappelle que les attributs ne pourront pas injecter de code.
