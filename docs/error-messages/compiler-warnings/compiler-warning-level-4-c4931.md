# Avertissement du compilateur (niveau 4) C4931
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

bibliothèque de types présumée construite pour des pointeurs 'nombre' bits  
  
 L'information explicite n'a pas été fournie avec l'attribut **ptrsize** de la directive [\#import](../../preprocessor/hash-import-directive-cpp.md) ; le compilateur a conclu que la taille du pointeur de la bibliothèque de types est ***nombre***.  
  
 Cet avertissement est désactivé par défaut.  Pour plus d'informations, consultez [Avertissements du compilateur désactivés par défaut](../../preprocessor/compiler-warnings-that-are-off-by-default.md).
