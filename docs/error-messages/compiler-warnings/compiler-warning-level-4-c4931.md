# <a name="compiler-warning-level-4-c4931"></a>Avertissement du compilateur (niveau 4) C4931

> Nous supposons que la bibliothèque de types a été générée pour les pointeurs de *nombre* bits

Les informations explicites n’ont pas été fournies avec l’attribut **ptrsize** de la directive [#import](../../preprocessor/hash-import-directive-cpp.md) ; le compilateur a conclu que la taille du pointeur de la bibliothèque de types est *Number*.

Cet avertissement est désactivé par défaut. Consultez [Avertissements du compilateur désactivés par défaut](../../preprocessor/compiler-warnings-that-are-off-by-default.md) pour plus d'informations.
