# &#201;valuateur d&#39;expression, erreur CXX0010
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

absence de crochet ouvrant  
  
 L'expression contient des crochets non appariés.  Entrez à nouveau l'expression avec des crochets correctement appariés.  
  
 Erreur identique à CAN0010.
