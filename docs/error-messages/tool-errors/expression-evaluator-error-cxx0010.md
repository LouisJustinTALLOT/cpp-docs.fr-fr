# <a name="expression-evaluator-error-cxx0010"></a>Évaluateur d'expression, erreur CXX0010

crochet gauche manquant

L’expression contient des crochets non appariés. Tapez à nouveau l’expression avec des crochets à charge équilibrée.

Cette erreur est identique à CAN0010.
