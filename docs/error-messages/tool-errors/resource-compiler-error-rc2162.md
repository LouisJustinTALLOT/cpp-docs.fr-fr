# <a name="resource-compiler-error-rc2162"></a>Erreur RC2162 du compilateur de ressources 

paramètre formel de macro ATTENDU

Le jeton qui suit un opérateur de chaîne ( **#** ) n’est pas un nom de paramètre formel.
