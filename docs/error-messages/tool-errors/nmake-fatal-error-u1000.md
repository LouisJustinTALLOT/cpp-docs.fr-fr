# Erreur irr&#233;cup&#233;rable NMAKE U1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erreur de syntaxe : '\)' absent de l'appel de la macro  
  
 Une parenthèse ouvrante **\(** apparaît sans parenthèse fermante **\)** dans un appel de macro.  La forme correcte est **$\(***nom***\)** ; `$` *n* est admis pour les noms composés d'un seul caractère.
