# Erreur irr&#233;cup&#233;rable CVTRES CVT1103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossible de lire nomfichier  
  
 Une erreur irrécupérable s'est produite lorsque CVTRES a essayé de lire le fichier donné.  
  
 Cette erreur est peut\-être due au fait que le fichier est endommagé.
