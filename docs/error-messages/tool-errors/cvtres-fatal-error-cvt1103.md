# <a name="cvtres-fatal-error-cvt1103"></a>Erreur irrécupérable CVTRES CVT1103

Impossible de lire le nom de fichier

Une erreur irrécupérable s’est produite lorsque CVTRES a tenté de lire le fichier donné.

Cette erreur peut être due à l’endommagement d’un fichier.
