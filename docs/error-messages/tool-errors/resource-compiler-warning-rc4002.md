# Avertissement RC4002 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

trop de paramètres réels pour la macro 'identificateur'  
  
 Le nombre de paramètres réels spécifiés à l'aide de l'identificateur donné est supérieur au nombre de paramètres formels spécifié dans la définition de macro de l'identificateur.  
  
 Les paramètres réels supplémentaires ont été collectés mais ignorés durant l'expansion de la macro.
