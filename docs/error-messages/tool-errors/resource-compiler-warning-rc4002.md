# <a name="resource-compiler-warning-rc4002"></a>Avertissement RC4002 du compilateur de ressources 

trop de paramètres réels pour la macro 'identifier'

Le nombre de paramètres réels spécifiés avec l’identificateur donné est supérieur au nombre de paramètres formels donné dans la définition de macro de l’identificateur.

Les paramètres réels supplémentaires ont été collectés mais ignorés pendant l’expansion de la macro.
