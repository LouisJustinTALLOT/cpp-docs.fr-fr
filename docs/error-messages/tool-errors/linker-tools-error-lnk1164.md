# <a name="linker-tools-error-lnk1164"></a>Erreur des outils Éditeur de liens LNK1164

alignement de section (nombre) supérieur à la valeur/ALIGN

La taille d’alignement pour la section donnée dans le fichier objet dépasse la valeur spécifiée avec l’option [/align](../../build/reference/align-section-alignment.md) . La valeur **/align** doit être une puissance de 2 et doit être égale ou supérieure à l’alignement de section spécifié dans le fichier objet.

Recompilez avec un plus petit alignement de section ou augmentez la valeur **/align** .
