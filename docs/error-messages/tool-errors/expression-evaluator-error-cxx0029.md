# &#201;valuateur d&#39;expression, erreur CXX0029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

n'est pas un pointeur struct  
  
 L'opérateur sélection de membre \(**\-\>**\) fut appliqué à une expression qui ne pointe pas vers une structure.  
  
 Vérifiez que l'expression tout entière est correctement mise entre parenthèses ou convertissez le type de l'expression d'adresse pour qu'il corresponde au pointeur de structure approprié.  
  
 Erreur identique à CAN0029.
