# Avertissement des outils &#201;diteur de liens LNK4076
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

fichier d'état incrémentiel non valide 'NomFichier' ; édition de liens non incrémentielle  
  
 LINK ne peut pas écrire dans le fichier d'état incrémentiel \(.ilk\).  `filename` est endommagé ou ce n'est pas une base de données de liaison incrémentielle.  Supprimez le fichier et recommencez l'édition de liens.
