# <a name="resource-compiler-error-rc2116"></a>Erreur RC2116 du compilateur de ressources 

nombre attendu pour l’ID

Nombre attendu pour le `id` champ d’une instruction de contrôle dans l’instruction **Dialog** . Vérifiez que vous disposez d’un numéro ou `#define` d’une instruction pour l’ID de contrôle.
