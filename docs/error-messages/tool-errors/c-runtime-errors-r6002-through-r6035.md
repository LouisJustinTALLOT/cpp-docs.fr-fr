# <a name="c-runtime-errors-rxxxx"></a>Erreurs Runtime C (Rxxxx)

La bibliothèque Runtime C (CRT) peut signaler une erreur d’exécution lorsque votre application est chargée ou en cours d’exécution. Même si chaque message fait référence à Microsoft Visual C++ bibliothèque d’exécution, cela ne signifie pas il existe un bogue dans la bibliothèque. Ces erreurs indiquent un bogue dans le code de votre application, ou une condition de la bibliothèque runtime ne peut pas gérer, telle que la mémoire est insuffisante. Les utilisateurs finaux de votre application peuvent recevoir ces erreurs, sauf si votre écriture votre application pour les empêcher, ou pour capturer les erreurs et de présenter une erreur conviviale du message à vos utilisateurs à la place.
