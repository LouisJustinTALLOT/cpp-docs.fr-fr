# Erreurs Runtime C R6002 &#224; R6035
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Cette section est une référence des erreurs qui peuvent être générées par la bibliothèque Runtime C \(CRT\) quand votre application est chargée ou en cours d’exécution. Ce sont des erreurs qui indiquent un bogue dans le code de votre application, ou une condition que la bibliothèque runtime ne peut pas gérer, par exemple une mémoire insuffisante. Les utilisateurs finaux de votre application peuvent recevoir ces erreurs si vous ne codez pas votre application pour empêcher les conditions qui les provoquent.
