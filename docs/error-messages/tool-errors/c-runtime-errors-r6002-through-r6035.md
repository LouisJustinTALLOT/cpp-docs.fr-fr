# <a name="c-runtime-errors-rxxxx"></a>Erreurs Runtime C (Rxxxx)

La bibliothèque Runtime C (CRT) peut signaler une erreur d’exécution lorsque votre application est chargée ou en cours d’exécution. Bien que chaque message fasse référence à la bibliothèque d’exécution Microsoft Visual C++, cela ne signifie pas qu’il existe un bogue dans la bibliothèque. Ces erreurs indiquent soit un bogue dans le code de votre application, soit une condition que la bibliothèque Runtime ne peut pas gérer, par exemple une mémoire insuffisante. Les utilisateurs finaux de votre application peuvent voir ces erreurs, sauf si vous écrivez votre application pour les empêcher, ou pour capturer les erreurs et présenter un message d’erreur convivial à vos utilisateurs à la place.
