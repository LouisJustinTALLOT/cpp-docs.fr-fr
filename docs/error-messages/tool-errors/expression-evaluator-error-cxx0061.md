# <a name="expression-evaluator-error-cxx0061"></a>Évaluateur d'expression, erreur CXX0061

profondeur d’imbrication des fonctions dépassée

L’expression contient une profondeur d’imbrication de fonctions supérieure à la limite. Modifiez l’expression pour réduire la profondeur d’imbrication.

Cette erreur est identique à CAN0061.
