# &#201;valuateur d&#39;expression, erreur CXX0061
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la limite des niveaux d'imbrication des fonctions est dépassée  
  
 L'expression contient des fonctions imbriquées à un niveau de profondeur excessif.  Modifiez l'expression pour réduire la profondeur d'imbrication.  
  
 Erreur identique à CAN0061.
