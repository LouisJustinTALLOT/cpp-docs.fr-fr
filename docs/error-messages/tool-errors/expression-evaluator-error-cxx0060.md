# <a name="expression-evaluator-error-cxx0060"></a>Évaluateur d'expression, erreur CXX0060

Registre n’est pas disponible

Une expression spécifie un Registre qui ne peut pas être utilisé. Cette erreur peut résulter en essayant d’accéder à un Registre qui n’existe pas sur l’ordinateur en cours d’exécution.

Cette erreur est identique à CAN0060.
