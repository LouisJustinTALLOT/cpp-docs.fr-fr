# &#201;valuateur d&#39;expression, erreur CXX0060
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

le Registre n'est pas disponible  
  
 Une expression spécifie un registre qui ne peut pas être utilisé.  Vous pouvez provoquer cette erreur en essayant d'accéder à un registre qui n'existe pas sur l'ordinateur.  
  
 Erreur identique à CAN0060.
