# <a name="expression-evaluator-error-cxx0060"></a>Évaluateur d'expression, erreur CXX0060

le registre n’est pas disponible

Une expression spécifie un registre qui ne peut pas être utilisé. Cette erreur peut être due à une tentative d’accès à un registre qui n’existe pas sur l’ordinateur exécutant.

Cette erreur est identique à CAN0060.
