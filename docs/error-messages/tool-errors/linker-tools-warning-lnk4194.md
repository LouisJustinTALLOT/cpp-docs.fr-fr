# <a name="linker-tools-warning-lnk4194"></a>Avertissement des outils Éditeur de liens LNK4194

/DELAYLOAD : nom de dll ignoré

L’éditeur de liens ne peut pas [différer le chargement](../../build/reference/delayload-delay-load-import.md) de la DLL demandée.
