# <a name="nmake-fatal-error-u1077"></a>Erreur irrécupérable NMAKE U1077
'programme' : code de retour 'valeur'  
  
 La commande ou programme appelé par NMAKE a échoué et retourné le code de sortie donné.  
  
 Pour supprimer cette erreur et continuer la session NMAKE, utilisez l’option /I, le **. Ignorer les** directive de point, ou que le tiret (**-**) modificateur de commande. Pour continuer la session NMAKE pour les parties non liées de l’arborescence des dépendances, utilisez l’option /K.
