# <a name="nmake-fatal-error-u1077"></a>Erreur irrécupérable NMAKE U1077

'programme' : code retour 'valeur'

La commande donnée ou un programme appelé par NMAKE a échoué et a retourné le code de sortie donné.

Pour supprimer cette erreur et continuer la session NMAKE, utilisez l’option /I, le **. Ignorer** directive de point ou le tiret (**-**) modificateur de commande. Pour continuer la session NMAKE pour les parties de l’arborescence des dépendances, utilisez l’option /K.
