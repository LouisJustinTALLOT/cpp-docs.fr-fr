# <a name="nmake-fatal-error-u1077"></a>Erreur irrécupérable NMAKE U1077

'programme' : code de retour’valeur'

La commande ou le programme donné appelé par NMAKE a échoué et a retourné le code de sortie donné.

Pour supprimer cette erreur et poursuivre la session NMAKE, utilisez l’option/I, le **. IGNORe** la directive point ou le modificateur de commande Dash ( **-** ). Pour continuer la session NMAKE pour les parties non liées de l’arborescence des dépendances, utilisez l’option/K.
