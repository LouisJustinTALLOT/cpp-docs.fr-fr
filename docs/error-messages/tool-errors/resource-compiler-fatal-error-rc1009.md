# <a name="resource-compiler-fatal-error-rc1009"></a>Erreur irrécupérable RC1009 du compilateur de ressources 

limite du compilateur : macros trop profondément imbriquées’macro'

Le fichier dépasse la limite du compilateur de ressources pour l’imbrication de macros. Modifiez le fichier source spécifié pour diminuer la profondeur d’imbrication de ses macros.
