# Erreur irr&#233;cup&#233;rable RC1009 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

limite du compilateur : macros imbriquées trop profondément 'macro'  
  
 Le fichier dépasse la limite des niveaux d'imbrication de macros du compilateur de ressources.  Modifiez le fichier source spécifié de façon à réduire les niveaux d'imbrication des macros.
