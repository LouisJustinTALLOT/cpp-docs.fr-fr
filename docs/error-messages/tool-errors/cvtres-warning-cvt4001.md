# <a name="cvtres-warning-cvt4001"></a>CVTRES, avertissement CVT4001

> type de machine non spécifié ; supposé *type*

CVTRES n’a pas trouvé une spécification de l’ordinateur. Il suppose le type d’ordinateur donné. Si la valeur par défaut n’est pas correcte, relancez CVTRES à l’aide de l’option /MACHINE.
