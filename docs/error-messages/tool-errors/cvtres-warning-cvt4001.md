# <a name="cvtres-warning-cvt4001"></a>CVTRES, avertissement CVT4001

> type d’ordinateur non spécifié ; *type* supposé

CVTRES n’a pas trouvé de spécification d’ordinateur. Il a supposé le type d’ordinateur donné. Si la valeur par défaut est incorrecte, réexécutez CVTRES avec l’option/MACHINE.
