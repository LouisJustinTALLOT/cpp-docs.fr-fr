# CVTRES, avertissement CVT4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**type d'ordinateur non spécifié ;**   
 ***type***  supposé  
  
 CVTRES n'a pas trouvé de spécification d'ordinateur.  Il utilise par défaut le type d'ordinateur donné.  Si la valeur par défaut n'est pas correcte, relancez CVTRES en utilisant l'option \/MACHINE.
