# <a name="linker-tools-warning-lnk4096"></a>Avertissement des outils Éditeur de liens LNK4096

La valeur/BASE « Number » n’est pas valide pour Windows 95 et Windows 98 ; l’image risque de ne pas s’exécuter

L’adresse de base que vous avez spécifiée n’est pas valide. Les fichiers exécutables Windows 95 et Windows 98 doivent avoir une adresse de base supérieure à 0x400000. Pour plus d’informations sur les adresses de base, consultez l’option [/base](../../build/reference/base-base-address.md) de l’éditeur de liens.
