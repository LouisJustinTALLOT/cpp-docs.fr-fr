# Avertissement des outils &#201;diteur de liens LNK4096
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la valeur \/BASE 'numéro' n'est pas valide sous Windows 95 et Windows 98 ; l'image risque de ne pas s'exécuter  
  
 L'adresse de base que vous avez spécifiée n'est pas valide.  Les fichiers exécutables Windows 95 et Windows 98 doivent avoir une adresse de base supérieure à 0x400000.  Pour plus d'informations sur les adresses de base, consultez l'option de l'éditeur de liens [\/BASE](../../build/reference/base-base-address.md).
