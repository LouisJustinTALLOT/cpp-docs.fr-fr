# <a name="linker-tools-warning-lnk4096"></a>Avertissement des outils Éditeur de liens LNK4096
La valeur /base « nombre » n’est pas valide pour Windows 95 et Windows 98 ; image ne peut pas s’exécuter  
  
 L’adresse de base que vous avez spécifié n’est pas valide. Les fichiers exécutables Windows 95 et Windows 98 doivent avoir une adresse de base supérieure à 0 x 400000. Pour plus d’informations sur les adresses de base, consultez le [/BASE](../../build/reference/base-base-address.md) option de l’éditeur de liens.
