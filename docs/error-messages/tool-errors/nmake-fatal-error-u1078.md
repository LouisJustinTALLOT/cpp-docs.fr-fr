# Erreur irr&#233;cup&#233;rable NMAKE U1078
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

dépassement de constante à 'expression'  
  
 L'expression indiquée contient une constante qui dépasse la plage – 2 147 483 648 à 2 147 483 647.  La constante apparaît dans l'une des situations suivantes :  
  
-   Une expression spécifiée avec une directive de prétraitement  
  
-   Un niveau d'erreur spécifié avec le modificateur de commande tiret \(**\-**\)
