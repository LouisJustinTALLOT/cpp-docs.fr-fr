# <a name="resource-compiler-fatal-error-rw1004"></a>Erreur irrécupérable RW1004 du compilateur de ressources 

Fin inattendue du fichier

Cette erreur peut être dû à l’absence de la dernière ligne d’un fichier texte, les caractères de retour chariot et saut de ligne.
