# <a name="resource-compiler-fatal-error-rw1004"></a>Erreur irrécupérable RW1004 du compilateur de ressources 
Fin de fichier inattendue  
  
 Cette erreur peut résulter de manquant sur la dernière ligne d’un fichier texte, les caractères de retour chariot et saut de ligne.
