# <a name="resource-compiler-fatal-error-rw1004"></a>Erreur irrécupérable RW1004 du compilateur de ressources

Fin de fichier inattendue

Cette erreur peut être due à l’absence de saut de ligne et de retour chariot sur la dernière ligne d’un fichier texte.
