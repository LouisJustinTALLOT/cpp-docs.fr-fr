# <a name="resource-compiler-fatal-error-rc1002"></a>Erreur irrécupérable RC1002 du compilateur de ressources 

espace du tas insuffisant

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Pour résoudre ce problème, appliquez les solutions possibles suivantes.

1. Augmentez l’espace de fichier d’échange Windows. Pour plus d’informations sur l’extension de l’espace de fichier d’échange, consultez mémoire virtuelle dans l’aide de Windows.

1. Fractionnez le fichier actuel en fichiers plus petits et compilez-les séparément.

1. Supprimer les autres programmes ou pilotes en cours d’exécution sur le système.
