# <a name="linker-tools-warning-lnk4010"></a>Avertissement des outils Éditeur de liens LNK4010

Numéro de version du sous-système non valide ; version de sous-système par défaut supposée

Vous pouvez spécifier une version pour le sous-système de l’image ([/Subsystem](../../build/reference/subsystem-specify-subsystem.md)). Chaque sous-système a une condition de version minimale. Si la version spécifiée est inférieure à la valeur minimale, cet avertissement se produit et l’éditeur de liens utilise simplement le sous-système par défaut.
