# Avertissement des outils &#201;diteur de liens LNK4010
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

numéro de version 'numéro' de sous\-système non valide ; version de sous\-système par défaut prise par défaut  
  
 Vous pouvez spécifier une version pour le sous\-système de l'image \([\/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)\).  Chaque sous\-système possède une exigence de version minimale.  Si la version spécifiée est inférieure à la version minimale, cet avertissement se produit et l'éditeur de liens utilise uniquement le sous\-système par défaut.
