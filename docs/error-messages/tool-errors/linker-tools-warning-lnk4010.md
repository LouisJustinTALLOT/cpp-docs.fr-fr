# <a name="linker-tools-warning-lnk4010"></a>Avertissement des outils Éditeur de liens LNK4010
nombre de numéro de version de sous-système non valide ; version de sous-système par défaut prise par défaut  
  
 Vous pouvez spécifier une version pour le sous-système de l’image ([/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)). Chaque sous-système possède une version minimale requise. Si la version spécifiée est inférieure au minimum, cet avertissement se produit et l’éditeur de liens utilisera le sous-système de la valeur par défaut.
