# &#201;valuateur d&#39;expression, erreur CXX0041
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'élément de la classe doit être un membre statique ou une fonction membre  
  
 Un membre non statique d'une classe \(ou structure ou union\) est utilisé sans spécification de l'instanciation de classe à utiliser.  
  
 Seules les données membres et les fonctions membres statiques peuvent être utilisées sans spécifier l'instanciation.  
  
 Erreur identique à CAN0041.
