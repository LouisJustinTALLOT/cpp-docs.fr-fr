# Avertissement RW4003 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

SHIFT ou CONTROL utilisé sans VIRTKEY  
  
 Dans une ressource table d'accélérateurs, SHIFT ou CONTROL requiert VIRTKEY.  SHIFT et CONTROL étant spécifiés en tant que bits indicateurs dans un accélérateur de type VIRTKEY, ils ne peuvent pas exister indépendamment de ce dernier.
