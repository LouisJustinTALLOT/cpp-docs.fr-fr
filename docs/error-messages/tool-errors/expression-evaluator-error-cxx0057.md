# &#201;valuateur d&#39;expression, erreur CXX0057
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

aucun code à la ligne  
  
 Aucun code n'a été généré pour le numéro de ligne spécifié.  Ce numéro ne peut pas être utilisé comme point d'arrêt.  
  
 Erreur identique à CAN0057.
