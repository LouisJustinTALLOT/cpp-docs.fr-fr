# <a name="expression-evaluator-error-cxx0057"></a>Évaluateur d'expression, erreur CXX0057

aucun code à la ligne numéro

Aucun code n’a été généré pour le numéro de ligne spécifié. Elle ne peut pas être utilisée comme point d’arrêt.

Cette erreur est identique à CAN0057.
