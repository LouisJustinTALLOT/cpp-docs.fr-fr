# <a name="expression-evaluator-error-cxx0062"></a>Évaluateur d'expression, erreur CXX0062

appels de constructeur non pris en charge

Une expression effectué un appel à un constructeur. Les expressions ne peuvent pas effectuer des appels explicites aux constructeurs ou effectuer des conversions qui nécessitent un appel à un constructeur.

Cette erreur est identique à CAN0062.
