# &#201;valuateur d&#39;expression, erreur CXX0062
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

les appels de constructeur ne sont pas pris en charge  
  
 Une expression appelle un constructeur.  Les expressions ne peuvent pas appeler explicitement des constructeurs, ni effectuer des conversions qui nécessitent l'appel d'un constructeur.  
  
 Erreur identique à CAN0062.
