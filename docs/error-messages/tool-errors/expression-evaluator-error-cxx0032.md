# <a name="expression-evaluator-error-cxx0032"></a>Évaluateur d'expression, erreur CXX0032
division par 0  
  
 L’expression contient un diviseur égal à zéro, ce qui n’est pas autorisé. Cette diviseur peut être le nombre littéral zéro, ou il peut être une expression qui prend la valeur zéro.  
  
 Cette erreur est identique à CAN0032.
