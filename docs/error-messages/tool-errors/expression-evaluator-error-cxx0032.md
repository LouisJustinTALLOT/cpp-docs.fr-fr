# &#201;valuateur d&#39;expression, erreur CXX0032
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

division par 0  
  
 L'expression contient un diviseur égal à zéro, ce qui n'est pas conforme.  Il peut s'agit du nombre littéral zéro ou d'une expression dont l'évaluation produit la valeur zéro.  
  
 Erreur identique à CAN0032.
