# <a name="expression-evaluator-error-cxx0054"></a>Évaluateur d'expression, erreur CXX0054
Registre dans l’expression de point d’arrêt non conforme  
  
 Un Registre ne peut pas être utilisé dans une expression de point d’arrêt.  
  
 Cette erreur est identique à CAN0054.
