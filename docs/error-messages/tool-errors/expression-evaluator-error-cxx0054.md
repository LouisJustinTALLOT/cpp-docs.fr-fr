# &#201;valuateur d&#39;expression, erreur CXX0054
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

le Registre dans l'expression du point d'arrêt est non conforme  
  
 Un registre ne peut pas être utilisé dans une expression de point d'arrêt.  
  
 Erreur identique à CAN0054.
