# <a name="expression-evaluator-error-cxx0013"></a>Évaluateur d'expression, erreur CXX0013
opérateur manquant  
  
 Un opérateur était attendu dans l’expression, mais n’a été trouvé. Vérifiez la syntaxe de l’expression.  
  
 Cette erreur est identique à CAN0013.
