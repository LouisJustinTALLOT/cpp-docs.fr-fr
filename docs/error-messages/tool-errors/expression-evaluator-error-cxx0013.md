# &#201;valuateur d&#39;expression, erreur CXX0013
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opérateur manquant  
  
 Un opérateur est attendu dans l'expression, mais il est introuvable.  Vérifiez la syntaxe de l'expression.  
  
 Erreur identique à CAN0013.
