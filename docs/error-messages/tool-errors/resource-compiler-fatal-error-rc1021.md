# Erreur irr&#233;cup&#233;rable RC1021 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

commande de préprocesseur non valide 'chaîne'  
  
 Les caractères placés après le signe numérique \(**\#**\) ne correspondent pas une directive de préprocesseur valide.  
  
 Le signe numérique ne peut pas être le caractère initial d'un identificateur.
