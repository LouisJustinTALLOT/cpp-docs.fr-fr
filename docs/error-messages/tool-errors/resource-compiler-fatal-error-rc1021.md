# <a name="resource-compiler-fatal-error-rc1021"></a>Erreur irrécupérable RC1021 du compilateur de ressources 
Commande de préprocesseur non valide 'chaîne'  
  
 Les caractères suivant le signe dièse (**#**) ne pas former une directive de préprocesseur valide.  
  
 Le signe dièse ne peut pas être utilisé comme premier caractère dans un identificateur.
