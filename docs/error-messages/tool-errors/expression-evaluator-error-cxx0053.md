# <a name="expression-evaluator-error-cxx0053"></a>Évaluateur d'expression, erreur CXX0053

correspondance des symboles non-fonction lors de la liaison de points d’arrêt

Un symbole utilisé comme point d’arrêt n’est pas une fonction. La spécification d’un membre de données comme point d’arrêt peut provoquer cette erreur.

Cette erreur est identique à CAN0053.
