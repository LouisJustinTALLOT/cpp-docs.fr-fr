# Erreur RC2111 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

type de contrôle non valide  
  
 Chaque instruction CONTROL d'une instruction **DIALOG** doit être de type 3STATE, AUTO3, AUTOCHECK, AUTORADIO, BEDIT, CHECKBOX, COMBOBOX, CONTROL, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, ICON, IEDIT, LISTBOX, LTEXT, PUSHBOX, PUSHBUTTON, RADIOBUTTON, RTEXT, SCROLLBAR ou USERBUTTON.  
  
 Assurez\-vous que ces instructions CONTROL sont correctement orthographiées.
