# <a name="resource-compiler-error-rc2111"></a>Erreur RC2111 du compilateur de ressources 

type de contrôle non valide

Chaque instruction de contrôle dans un **boîte de dialogue** instruction doit être une des opérations suivantes : TYPE 3STATE, AUTO3, VÉRIFICATION AUTOMATIQUE, AUTORADIO, BEDIT, CHECKBOX, COMBOBOX, CONTRÔLE, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, ICÔNE, IEDIT, LISTBOX, LTEXT, PUSHBOX, PUSHBUTTON, RADIOBUTTON, RTEXT, BARRE DE DÉFILEMENT, USERBUTTON.

Assurez-vous que ces instructions de contrôle sont correctement orthographiées.
