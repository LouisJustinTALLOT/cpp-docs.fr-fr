# <a name="resource-compiler-error-rc2111"></a>Erreur RC2111 du compilateur de ressources

type de contrôle non valide

Chaque instruction de contrôle dans une instruction de **boîte de dialogue** doit être l’une des suivantes : 3STATE, AUTO3, AutoCheck, autoradio, BEDIT, CheckBox, ComboBox, Control, CTEXT, DEFPUSHBUTTON, EditText, GroupBox, HEDIT, Icon, IEDIT, ListBox, LTEXT, PUSHBOX, PUSHBUTTON, RadioButton, rtext, ScrollBar, UserButton.

Assurez-vous que ces instructions de contrôle sont correctement orthographiées.
