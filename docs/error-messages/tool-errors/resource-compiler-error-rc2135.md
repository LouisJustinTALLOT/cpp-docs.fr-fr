# Erreur RC2135 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

fichier introuvable : nom de fichier  
  
 Le fichier spécifié dans la ligne de commande du compilateur de ressources est introuvable.  Vérifiez si le fichier a été déplacé dans un autre répertoire et assurez\-vous que le nom et le chemin d'accès du fichier sont corrects.  
  
 Le compilateur recherche la variable d'environnement **INCLUDE** ou le paramètre **INCLUDE** de Visual C\+\+ dans les fichiers.
