# <a name="resource-compiler-error-rc2135"></a>Erreur RC2135 du compilateur de ressources

fichier introuvable : nom de fichier

Le fichier spécifié sur la ligne de commande du compilateur de ressources est introuvable. Vérifiez si le fichier a été déplacé vers un autre répertoire et si le nom et le chemin d’accès du fichier sont correctement tapés.

Les fichiers sont recherchés à l’aide de la variable d’environnement **include** ou du paramètre Visual C++ **include** .
