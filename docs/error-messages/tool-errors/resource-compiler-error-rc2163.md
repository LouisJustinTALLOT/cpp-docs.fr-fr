# <a name="resource-compiler-error-rc2163"></a>Erreur RC2163 du compilateur de ressources 

type d’accélérateur requis [ASCII ou VIRTKEY]

Le champ `type` de l’instruction **ACCELERATORS** doit contenir la valeur ASCII ou VIRTKEY.
