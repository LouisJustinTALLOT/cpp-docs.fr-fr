# <a name="resource-compiler-fatal-error-rc1101"></a>Erreur irrécupérable RC1101 du compilateur de ressources 

> Aucun nom de fichier binaire ressource spécifié

La sortie de renommer (/ fo) option n’a pas été suivie par un nom de fichier.

Utilisez la syntaxe suivante pour l’option /fo :

> **RC /fo** *nomfichier_sortie* <em>nomfichier_entrée</em>**.rc**
