# <a name="resource-compiler-fatal-error-rc1101"></a>Erreur irrécupérable RC1101 du compilateur de ressources

> aucun nom de fichier binaire de ressource spécifié

L’option renommer la sortie (/FO) n’a pas été suivie d’un nom de fichier.

Utilisez la syntaxe suivante pour l’option/FO :

> **RC/fo** *nom-* fichier <em>inFileName</em> **. RC**
