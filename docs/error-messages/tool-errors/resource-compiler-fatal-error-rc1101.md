# Erreur irr&#233;cup&#233;rable RC1101 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

aucun nom de fichier binaire de ressource spécifié  
  
 Il manque un nom de fichier après l'option Rename Output \(\/fo\).  
  
 Utilisez la syntaxe suivante pour l'instruction \/fo  :  
  
 **RC \/fo**   
 ***outfilename infilename.* rc**
