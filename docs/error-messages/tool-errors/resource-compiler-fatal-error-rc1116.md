# <a name="resource-compiler-fatal-error-rc1116"></a>Erreur irrécupérable RC1116 du compilateur de ressources

RC se terminant après des erreurs de préprocesseur

Le compilateur de ressources s’est arrêté en raison d’autres erreurs. Corrigez les autres erreurs et recompilez.
