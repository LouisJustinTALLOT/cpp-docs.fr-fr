# Erreur irr&#233;cup&#233;rable RC1116 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Arrêt du RC après erreurs du préprocesseur  
  
 Le compilateur de ressources s'est arrêté en raison d'autres erreurs.  Rectifiez ces erreurs, puis recompilez.
