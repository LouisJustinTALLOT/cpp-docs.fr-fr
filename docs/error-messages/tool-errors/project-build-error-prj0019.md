# Erreur de g&#233;n&#233;ration de projet PRJ0019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Un outil a retourné un code d'erreur à partir de  
  
 Un niveau d'erreur était différent de zéro pour une étape de build personnalisée ou un événement de build.  
  
 Vous obtiendrez également une erreur PRJ0019 si un outil retourne un code d'erreur, mais sans message d'erreur.  Cela peut se produire, par exemple, si vous redirigez la sortie de MIDL vers NUL.  
  
 Pour plus d'informations, consultez [Résolution de problèmes liés aux étapes de build personnalisée et aux événements de build](../../ide/troubleshooting-build-customizations.md).  
  
 Cette erreur peut également se produire lorsque vous agissez en tant que membre du groupe Utilisateurs et qu'un accès administratif est requis.  Pour plus d'informations, consultez [Exécution en tant que membre du groupe Utilisateurs](../../top/running-as-a-member-of-the-users-group.md).
