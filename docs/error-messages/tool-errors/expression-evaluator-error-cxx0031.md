# &#201;valuateur d&#39;expression, erreur CXX0031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

expression non extensible  
  
 L'évaluateur d'expression a rencontré une erreur interne.  
  
 Il se peut que vous puissiez écrire une expression équivalente qui sera évaluée correctement.  
  
 Erreur identique à CAN0031.
