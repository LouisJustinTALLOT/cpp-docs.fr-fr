# <a name="expression-evaluator-error-cxx0031"></a>Évaluateur d'expression, erreur CXX0031
expression n’est pas extensible  
  
 L’évaluateur d’expression a rencontré une erreur interne.  
  
 Vous serez peut-être en mesure d’écrire une expression équivalente qui peut être évaluée correctement.  
  
 Cette erreur est identique à CAN0031.
