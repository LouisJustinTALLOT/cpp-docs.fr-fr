# <a name="project-build-error-prj0044"></a>Erreur de génération de projet PRJ0044

La propriété 'Dépendances supplémentaires' pour la règle de génération personnalisée 'règle' assignée au fichier 'fichier' n’est pas valide. Cette propriété contient 'chaîne' qui prend la valeur 'valeur'.

Le **dépendances supplémentaires** propriété évaluée à une chaîne vide, ou une chaîne contenant des caractères non valides (n’importe quel caractère qui ne peut pas être dans un nom de fichier ou répertoire). Les règles de génération personnalisées ont besoin de la sortie de l’action de génération.

Pour plus d’informations, consultez [spécifiant des outils de génération personnalisée](../../build/specifying-custom-build-tools.md).
