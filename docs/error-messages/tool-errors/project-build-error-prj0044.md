# Erreur de g&#233;n&#233;ration de projet PRJ0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La propriété 'Dépendances supplémentaires' pour la règle de génération personnalisée 'règle' assignée au fichier 'fichier' n'est pas valide.Cette propriété contient 'chaîne', qui prend la valeur 'valeur'.  
  
 La propriété **Dépendances supplémentaires** a pris la valeur d'une chaîne vide ou d'une chaîne qui contenait des caractères non valides \(des caractères qui ne pouvaient pas figurer dans un nom de fichier ou de répertoire\).  Les règles de génération personnalisées ont besoin de la sortie de l'action de génération.  
  
 Pour plus d'informations, consultez [Spécification des outils de génération personnalisée](../../ide/specifying-custom-build-tools.md).
