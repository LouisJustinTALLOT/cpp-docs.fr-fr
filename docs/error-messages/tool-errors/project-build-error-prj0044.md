# <a name="project-build-error-prj0044"></a>Erreur de génération de projet PRJ0044

La propriété’dépendances supplémentaires’de la règle de génération personnalisée’règle’assignée au fichier’fichier’n’est pas valide. Cette propriété contient 'chaîne' qui prend la valeur 'valeur'.

La propriété **dépendances supplémentaires** a été évaluée à une chaîne vide ou à une chaîne qui contenait des caractères non valides (n’importe quel caractère qui ne peut pas figurer dans un nom de fichier ou de répertoire). Les règles de génération personnalisées ont besoin de la sortie de l’action de génération.

Pour plus d’informations, consultez [spécification des outils de génération personnalisée](../../build/specifying-custom-build-tools.md).
