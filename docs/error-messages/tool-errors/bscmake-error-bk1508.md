# Erreur BSCMAKE BK1508
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossible de supprimer le fichier temporaire 'nom\_fichier' \[: raison\]  
  
 BSCMAKE ne peut pas supprimer l'un de ses fichiers temporaires.  
  
### Pour résoudre le problème en vérifiant les causes possibles suivantes  
  
1.  Le fichier est ouvert par un autre processus.  
  
2.  Une erreur matérielle.
