# <a name="expression-evaluator-error-cxx0028"></a>Évaluateur d'expression, erreur CXX0028

n’est pas un élément struct/union

Une expression de la forme’struct. Member’ou’pStruct->membre’a été spécifiée, mais \<member> n’est pas un élément de la structure.

L’expression ne peut pas être correctement placée entre parenthèses.

Cette erreur est identique à CAN0028.
