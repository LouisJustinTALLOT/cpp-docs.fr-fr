# &#201;valuateur d&#39;expression, erreur CXX0012
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

absence d'accolade ouvrante  
  
 L'expression contient une accolade non appariée.  Entrez à nouveau l'expression avec des accolades correctement appariées.  
  
 Erreur identique à CAN0012.
