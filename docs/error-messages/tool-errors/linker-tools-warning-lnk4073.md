# <a name="linker-tools-warning-lnk4073"></a>Avertissement des outils Éditeur de liens LNK4073

Impossible de créer un mappage pour le fichier. ilk ; liaison non incrémentielle

Il n’y avait pas suffisamment d’espace contigu dans la mémoire partagée pour créer le fichier d’état incrémentiel (. ilk). Le lien a effectué une génération non incrémentielle.
