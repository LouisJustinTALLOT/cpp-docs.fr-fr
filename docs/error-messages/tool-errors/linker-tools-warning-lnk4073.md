# <a name="linker-tools-warning-lnk4073"></a>Avertissement des outils Éditeur de liens LNK4073
Impossible de créer le mappage pour le fichier .ilk ; liaison non incrémentielle  
  
 Il n’était pas une grande suffisamment d’espace contigu dans la mémoire partagée pour le lien créer le fichier d’état incrémentiel (.ilk). LINK a effectué une génération non incrémentielle.
