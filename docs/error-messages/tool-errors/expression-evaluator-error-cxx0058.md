# <a name="expression-evaluator-error-cxx0058"></a>Évaluateur d'expression, erreur CXX0058

opérateur surchargé introuvable

Un type de classe a été spécifié en tant que l’opérande gauche dans une expression, mais un opérateur surchargé n’était pas défini pour la classe.

Cette erreur est identique à CAN0058.
