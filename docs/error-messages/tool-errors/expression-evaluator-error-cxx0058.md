# &#201;valuateur d&#39;expression, erreur CXX0058
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opérateur surchargé introuvable  
  
 Un type classe est spécifié comme opérande de gauche dans une expression, mais aucun opérateur surchargé n'a été défini pour cette classe.  
  
 Erreur identique à CAN0058.
