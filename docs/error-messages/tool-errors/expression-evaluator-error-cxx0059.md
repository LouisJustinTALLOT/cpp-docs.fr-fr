# &#201;valuateur d&#39;expression, erreur CXX0059
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'opérande de gauche est une classe, et non un nom de fonction  
  
 L'opérande de gauche d'un appel de fonction est un nom de classe et ne peut pas être résolu en appel de fonction.  Vous pouvez provoquer cette erreur en omettant le nom d'une fonction membre dans une expression.  
  
 Erreur identique à CAN0059.
