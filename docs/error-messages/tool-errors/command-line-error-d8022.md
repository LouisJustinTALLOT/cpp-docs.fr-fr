# Erreur de ligne de commande&#160;D8022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossible d'ouvrir 'FichierMessage'  
  
 Le fichier donné ne se trouve pas dans le répertoire actif ou dans un répertoire spécifié dans la variable d'environnement PATH.  Le fichier de messages contient un résumé rapide de la syntaxe et des options de la ligne de commande du compilateur.  
  
 Déplacez ce fichier vers le répertoire actif ou vers un répertoire présent dans le chemin d'accès en cours.  Si ce fichier reste introuvable, exécutez le programme d'installation pour le copier à partir des disques de distribution.
