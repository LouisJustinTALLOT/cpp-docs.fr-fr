# <a name="command-line-error-d8022"></a>Erreur de ligne de commande D8022

Impossible d’ouvrir’messageFile'

Le fichier donné ne figure pas dans le répertoire actif ou dans un répertoire spécifié dans la variable d’environnement PATH. Le fichier de message contient un bref résumé de la syntaxe et des options de la ligne de commande du compilateur.

Déplacez ce fichier vers le répertoire actif ou un répertoire dans le chemin d’accès actuel. Si ce fichier est introuvable, exécutez le programme d’installation pour le copier à partir des disques de distribution.
