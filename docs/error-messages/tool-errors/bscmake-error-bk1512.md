# Erreur BSCMAKE BK1512
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nom du fichier : capacité dépassée  
  
 BSCMAKE ne peut pas construire un fichier d'informations de consultation car le nombre de définitions, de références, de modules et autres informations a dépassé la limite autorisée.  
  
### Pour résoudre le problème en utilisant les solutions possibles suivantes  
  
1.  Exclusion de certaines informations à l'aide des options \/Em, \/Es et \/Ei.  
  
2.  Omission de l'option \/Iu.
