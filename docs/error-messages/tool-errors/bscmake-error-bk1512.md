# <a name="bscmake-error-bk1512"></a>Erreur BSCMAKE BK1512
nom de fichier : dépassée de capacité  
  
 BSCMAKE ne peut pas générer un fichier d’informations, car le nombre de définitions, les références, les modules ou les autres informations dépasse la limite.  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>Pour résoudre ce problème, appliquez les solutions possibles suivantes.  
  
1.  Exclure des informations à l’aide de /Em, /Es ou /Ei.  
  
2.  Omettez l’option /Iu.
