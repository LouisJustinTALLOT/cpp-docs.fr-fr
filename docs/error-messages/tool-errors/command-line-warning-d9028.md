# <a name="command-line-warning-d9028"></a>Avertissement de ligne de commande D9028
Échec de la régénération minimale, retour à la génération normale  
  
 Le fichier .idb de projet est endommagé. Supprimez le fichier et régénérez.  
  
 Pour plus d’informations sur les régénérations minimales et le fichier .idb, consultez le [option du compilateur /Gm](../../build/reference/gm-enable-minimal-rebuild.md).
