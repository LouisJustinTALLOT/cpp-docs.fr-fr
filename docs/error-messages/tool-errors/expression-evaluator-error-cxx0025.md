# <a name="expression-evaluator-error-cxx0025"></a>Évaluateur d'expression, erreur CXX0025

l’opérateur a besoin de struct/union

Un opérateur qui accepte une expression de **`struct`** **`union`** type ou a été appliqué à une expression qui n’est pas un **`struct`** ou un **`union`** .

Les composants de variables de classe, de structure ou d’Union doivent avoir un nom qualifié complet. Les composants ne peuvent pas être entrés sans spécification complète.

Cette erreur est identique à CAN0025.
