# <a name="expression-evaluator-error-cxx0007"></a>Évaluateur d'expression, erreur CXX0007
parenthèse fermante manquante  
  
 Parenthèses non équilibrées ont été trouvés dans l’expression. Tapez à nouveau l’expression entre parenthèses à charge équilibrée.  
  
 Cette erreur est identique à CAN0007.
