# &#201;valuateur d&#39;expression, erreur CXX0007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

absence de parenthèse fermante  
  
 L'expression contient des parenthèses non appariées.  Entrez à nouveau l'expression avec des parenthèses correctement appariées.  
  
 Erreur identique à CAN0007.
