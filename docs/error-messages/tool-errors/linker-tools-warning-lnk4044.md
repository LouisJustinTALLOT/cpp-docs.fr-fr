# Avertissement des outils &#201;diteur de liens LNK4044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

option 'option' non reconnue ; ignorée  
  
 L'option donnée n'est pas valide pour cet outil.  L'outil a ignoré l'option.  
  
 Pour voir une liste des options valides, exécutez l'outil sans argument ni option.
