# <a name="resource-compiler-error-rc2015"></a>Erreur RC2015 du compilateur de ressources 
trop de caractères dans la constante  
  
 Une constante caractère contient plus de deux caractères.  
  
 Les constantes de caractère sont limités à un caractère (constante de caractère standard) ou deux caractères (constante de caractère long).  
  
 Notez qu’une séquence d’échappement (par exemple, \t pour tabulation) est convertie en un seul caractère.
