# &#201;valuateur d&#39;expression, erreur CXX0055
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

symbole ambigu dans l'opérateur de contexte  
  
 Un symbole utilisé dans l'opérateur de contexte \(**{}**\) fait référence à plusieurs symboles dans le programme.  
  
 L'opérateur de résolution de portée \(`::`\) peut parfois résoudre cette ambiguïté.  
  
 Erreur identique à CAN0055.
