# Erreur des outils &#201;diteur de liens LNK1141
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

échec lors de la génération du fichier des exportations  
  
 [LINK](../../build/reference/linker-command-line-syntax.md) n'a pas pu construire le fichier des exportations \(.exp\).  
  
 Les causes de cette erreur peuvent être des problèmes sur le système, tels qu'un manque de mémoire ou des erreurs de syntaxe dans les options ou les instructions de définition de modules.  
  
 Cette erreur est précédée par une autre erreur qui donne plus d'informations.
