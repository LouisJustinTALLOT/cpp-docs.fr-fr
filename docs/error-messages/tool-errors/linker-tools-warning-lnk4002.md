# <a name="linker-tools-warning-lnk4002"></a>Avertissement des outils Éditeur de liens LNK4002

symbole défini dans l’objet

Le symbole, affiché dans sa forme décorée, a été spécifié dans sa forme non décorée dans `object`, mais Impossible de trouver une correspondance unique avec un symbole décoré. Cet avertissement est toujours précédé par l’avertissement [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) et suivie d’une erreur irrécupérable [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
