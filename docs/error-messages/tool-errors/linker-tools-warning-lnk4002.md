# Avertissement des outils &#201;diteur de liens LNK4002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'symbole' défini dans objet  
  
 Le symbole, affiché sous forme décorée, a été spécifié sous forme non décorée dans `object`, mais il a été impossible de trouver une correspondance unique avec un symbole décoré.  Cet avertissement est toujours précédé par l'avertissement [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) et suivi de l'erreur irrécupérable [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
