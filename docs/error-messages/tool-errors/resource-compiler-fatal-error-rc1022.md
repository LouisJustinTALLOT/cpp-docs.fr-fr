# <a name="resource-compiler-fatal-error-rc1022"></a>Erreur irrécupérable RC1022 du compilateur de ressources 

' #endif’attendu

Une directive `#if` , **#ifdef** ou **#ifndef** n’a pas été arrêtée avec une `#endif` directive.

Assurez-vous qu’il existe une `#if` instruction, **#ifdef** ou **#ifndef** en vigueur avant cette instruction.
