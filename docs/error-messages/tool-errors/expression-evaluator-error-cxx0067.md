# <a name="expression-evaluator-error-cxx0067"></a>Évaluateur d'expression, erreur CXX0067

évaluation de fonction non pris en charge.

L’expression contient un appel de fonction. Certains évaluateurs d’expression ne gèrent pas les appels de fonction.

Cette erreur est identique à CAN0067.
