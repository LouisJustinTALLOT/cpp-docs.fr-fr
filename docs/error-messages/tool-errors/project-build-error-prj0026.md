# <a name="project-build-error-prj0026"></a>Erreur de génération de projet PRJ0026
Le fichier réponse 'fichier' contient Unicode qui n’a pas pu être traduits à la page de codes ANSI de l’utilisateur.  
  
 ***Contenu UNICODE du fichier***  
  
 Le système de projet trouvée Unicode contenu dans un fichier de réponse qui ne peut pas être traduit correctement à la page de codes ANSI de l’utilisateur actuelle.  
  
 La résolution de cette erreur est mise à jour le contenu du fichier de réponse à utiliser ANSI ou pour installer la page de codes sur votre ordinateur et le définir en tant que la valeur par défaut du système.
