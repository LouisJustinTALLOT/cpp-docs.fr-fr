# Erreur de g&#233;n&#233;ration de projet PRJ0026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Le fichier réponse 'fichier' contient du code Unicode qui n'a pas pu être traduit dans la page de codes ANSI de l'utilisateur  
  
 ***Contenu UNICODE du fichier***  
  
 Le système de projet a trouvé dans un fichier réponse un contenu Unicode qui ne peut être traduit correctement dans la page de codes ANSI en cours de l'utilisateur.  
  
 Pour corriger cette erreur, mettez à jour le contenu du fichier réponse de façon à utiliser ANSI ou installez la page de codes sur votre ordinateur et définissez\-la en tant que valeur système par défaut.
