# <a name="project-build-error-prj0026"></a>Erreur de génération de projet PRJ0026

> Le fichier réponse'*fichier*'contient du contenu Unicode qui n’a pas pu être traduit dans la page de codes ANSI de l’utilisateur.
>
> *Contenu UNICODE du fichier*

Le système de projet a trouvé du contenu Unicode dans un fichier réponse qui ne peut pas être traduit correctement dans la page de codes ANSI actuelle de l’utilisateur.

La résolution de cette erreur consiste à mettre à jour le contenu du fichier réponse pour utiliser ANSI ou pour installer la page de codes sur votre ordinateur et la définir comme valeur système par défaut.
