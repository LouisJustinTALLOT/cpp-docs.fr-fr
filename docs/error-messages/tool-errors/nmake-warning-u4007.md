# Avertissement NMAKE U4007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nom de fichier 'nomfichier' trop long ; troncation à 8.3  
  
 Le nom de base du fichier donné contient plus de huit caractères, ou l'extension comporte plus de trois caractères.  NMAKE a tronqué le nom pour le ramener à une base de huit caractères avec une extension de trois caractères.  
  
 Si votre système de fichiers prend en charge les noms de fichiers longs, mettez le nom entre guillemets \(**"**\).
