# <a name="nmake-warning-u4007"></a>Avertissement NMAKE U4007

nom de fichier’nom de fichier’trop long ; troncation à 8,3

Le nom de base du fichier donné contient plus de huit caractères, ou l’extension contient plus de trois caractères. NMAKE a tronqué le nom à une base de huit caractères et une extension à trois caractères.

Si les noms de fichiers longs sont pris en charge par votre système de fichiers, mettez le nom entre guillemets doubles ( **"** ).
