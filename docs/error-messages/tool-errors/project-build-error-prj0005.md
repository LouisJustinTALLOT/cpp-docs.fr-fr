# <a name="project-build-error-prj0005"></a>Erreur de génération de projet PRJ0005

Impossible de créer un fichier temporaire dans le répertoire’répertoire'.

L’appel à la création d’un fichier temporaire a échoué. Les raisons de l’échec sont les suivantes :

- Les noms de fichiers temporaires sont insuffisants.

- Le répertoire temporaire est en lecture seule.

- Il n’existe pas de variable d’environnement TMP ou de répertoire Temp.

- L’espace disque disponible de votre ordinateur est insuffisant.
