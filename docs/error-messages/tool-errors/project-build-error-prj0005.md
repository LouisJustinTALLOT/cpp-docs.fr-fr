# Erreur de g&#233;n&#233;ration de projet PRJ0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossible de créer un fichier temporaire dans le répertoire 'répertoire'.  
  
 L'appel à la création d'un fichier temporaire a échoué.  Les raisons de l'échec sont les suivantes :  
  
-   Insuffisance des noms de fichiers temporaires.  
  
-   Le répertoire temporaire est en lecture seule.  
  
-   Absence de répertoire temporaire ou de variable d'environnement TMP.  
  
-   L'ordinateur ne dispose pas d'un espace disque suffisant.
