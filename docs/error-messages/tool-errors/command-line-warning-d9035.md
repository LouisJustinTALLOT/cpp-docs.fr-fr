# <a name="command-line-warning-d9035"></a>Avertissement de ligne de commande D9035

> l’option'*option*'est dépréciée et sera supprimée dans une version ultérieure

## <a name="remarks"></a>Notes

Vous avez spécifié une option de compilateur qui sera supprimée dans une prochaine version du compilateur. En cas de remplacement suggéré pour l' *option*, cet avertissement est suivi de l’avertissement [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md).

L’option spécifiée fonctionne toujours, mais vous devez mettre à jour votre configuration de build maintenant. Par conséquent, le build votre projet va probablement se poursuivre lorsque vous mettez à niveau le compilateur.
