# Avertissement de ligne de commande D9035
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

L'option 'option' est déconseillée et sera supprimée dans une version ultérieure  
  
 Vous avez spécifié une option du compilateur qui sera supprimée dans une version ultérieure du compilateur de Visual C\+\+.  Cet avertissement suit l'avertissement [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) si un remplacement est suggéré pour l' *option*.  
  
 L'option spécifiée fonctionne encore, mais vous devez maintenant mettre à jour votre configuration de build.  Par conséquent, votre projet est plus à même de continuer à générer lorsque vous mettez à niveau le compilateur.
