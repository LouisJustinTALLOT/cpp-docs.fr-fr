# <a name="command-line-warning-d9035"></a>Avertissement de ligne de commande D9035
l’option 'option' a été déconseillée et sera supprimée dans une version ultérieure  
  
 Vous avez spécifié une option du compilateur qui sera supprimée dans une prochaine version du compilateur Visual C++. Cet avertissement est suivi d’avertissement [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) s’il existe une suggestion de remplacement pour *option*.  
  
 L’option spécifiée fonctionne toujours, mais vous devez mettre à jour votre configuration de build maintenant. Par conséquent, votre projet est plus probable continuer à créer lorsque vous mettez à niveau le compilateur.
