# Erreur BSCMAKE BK1516
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

fichierbsc endommagé ; ne peut pas mettre à jour de façon incrémentielle  
  
 Le fichier .bsc a été endommagé, probablement à cause d'une panne système au cours de la génération.  Supprimez le fichier .bsc, régénérez tous les fichiers .sbr, puis le fichier .bsc.
