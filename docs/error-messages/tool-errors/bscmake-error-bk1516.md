# <a name="bscmake-error-bk1516"></a>Erreur BSCMAKE BK1516

bscfile endommagé ; Impossible de mettre à jour de façon incrémentielle

Le fichier. bsc est endommagé, probablement en raison d’une défaillance du système pendant la génération. Supprimez le fichier. bsc, régénérez tous les fichiers. SBR, puis régénérez le fichier. bsc.
