# Erreur de g&#233;n&#233;ration de projet PRJ0014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

L'objet job utilisé pour contrôler les processus générés dynamiquement a échoué.  La génération ne peut pas continuer.  
  
 Une erreur s'est produite dans l'environnement de développement.  
  
 Pour résoudre le problème, fermez et rechargez le projet.  Au besoin, quittez Visual Studio et redémarrez\-le.
