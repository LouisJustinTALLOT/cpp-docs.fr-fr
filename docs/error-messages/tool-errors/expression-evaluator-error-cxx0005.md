# &#201;valuateur d&#39;expression, erreur CXX0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opérateur non pris en charge  
  
 Une expression spécifie un opérateur C qui n'est pas pris en charge.  Écrivez une expression équivalente en utilisant les opérateurs C pris en charge.  
  
 Erreur identique à CAN0005.
