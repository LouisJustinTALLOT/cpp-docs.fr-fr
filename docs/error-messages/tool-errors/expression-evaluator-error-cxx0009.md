# <a name="expression-evaluator-error-cxx0009"></a>Évaluateur d'expression, erreur CXX0009

une constante caractère after est manquante

Le guillemet simple attendu à la fin de la constante caractère était manquant. Retapez l’expression en plaçant la constante caractère entre des guillemets simples.

Cette erreur est identique à CAN0009.
