# &#201;valuateur d&#39;expression, erreur CXX0009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

absence de ' après une constante caractère  
  
 Le guillemet simple attendu manque à la fin de la constante caractère.  Entrez à nouveau l'expression en entourant la constante caractère de guillemets simples.  
  
 Erreur identique à CAN0009.
