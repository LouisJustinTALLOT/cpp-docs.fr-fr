# <a name="expression-evaluator-error-cxx0009"></a>Évaluateur d'expression, erreur CXX0009

manquant ' après une constante caractère

Le guillemet simple attendu à la fin de la constante caractère est manquant. Tapez à nouveau l’expression en entourant la constante caractère entre guillemets simples.

Cette erreur est identique à CAN0009.
