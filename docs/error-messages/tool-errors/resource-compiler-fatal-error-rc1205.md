# Erreur irr&#233;cup&#233;rable RC1205 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

page de codes non valide  
  
 La page de codes spécifiée après l'option Specify Code Page \(\/c\) n'est pas valide.  Pour plus d'informations, consultez [Pages de codes](../../c-runtime-library/code-pages.md) dans le document *Run\-Time Library Reference*.
