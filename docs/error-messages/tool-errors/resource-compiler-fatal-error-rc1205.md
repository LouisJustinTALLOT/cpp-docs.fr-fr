# <a name="resource-compiler-fatal-error-rc1205"></a>Erreur irrécupérable RC1205 du compilateur de ressources

page de codes non valide

L’option de spécification de page de codes (/c) a été suivie d’une page de codes non valide. Pour plus d’informations, consultez [Code Pages](../../c-runtime-library/code-pages.md) dans *Run-Time Library Reference* .
