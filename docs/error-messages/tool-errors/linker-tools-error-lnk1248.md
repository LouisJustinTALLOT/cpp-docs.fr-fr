# Erreur des outils &#201;diteur de liens LNK1248
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la taille d'image \('taille'\) dépasse la taille maximale autorisée \(80 000 000\)  
  
 L'éditeur de liens a déterminé que la taille du fichier de sortie dépasserait la taille maximale possible pour une image de programme.  Vous pouvez transformer votre programme en plusieurs DLL.  Pour plus d'informations sur le fichier exécutable du programme \(PE\), consultez l'article « Peering Inside the PE: A Tour of the Win32 Portable Executable File Format » dans la [Bibliothèque MSDN](http://go.microsoft.com/fwlink/?linkid=556).
