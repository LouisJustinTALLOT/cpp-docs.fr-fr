# <a name="linker-tools-error-lnk1248"></a>Erreur des outils Éditeur de liens LNK1248

> la taille de l’image ('*taille*') dépasse la taille maximale autorisée (80 millions)

L’éditeur de liens a déterminé que la taille du fichier de sortie dépassera la plus grande taille possible pour une image de programme de 32 bits. Vous souhaiterez peut-être transformer votre programme en plusieurs dll.
