# <a name="resource-compiler-fatal-error-rc1109"></a>Erreur irrécupérable RC1109 du compilateur de ressources 
Erreur de création du nom de la ressource  
  
 N’a pas pu créer un fichier .res spécifié. Assurez-vous qu’il n’est pas créée sur un lecteur en lecture seule. Utilisez l’option /V pour déterminer si le fichier est en cours de création.
