# <a name="resource-compiler-fatal-error-rc1109"></a>Erreur irrécupérable RC1109 du compilateur de ressources 

erreur lors de la création de la ressource-nom

Impossible de créer le fichier. res spécifié. Assurez-vous qu’il n’est pas créé sur un lecteur en lecture seule. Utilisez l’option/V pour déterminer si le fichier est en cours de création.
