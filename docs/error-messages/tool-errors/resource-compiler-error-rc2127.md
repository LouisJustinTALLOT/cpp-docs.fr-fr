# Erreur RC2127 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

versions WORD séparées par des virgules attendues  
  
 Dans une ressource de version, les numéros de version doivent être de type **WORD** et être séparés par des virgules.
