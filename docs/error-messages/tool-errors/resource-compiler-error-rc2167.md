# Erreur RC2167 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

champ VERSIONINFO non reconnu ; BEGIN ou virgule attendu  
  
 Un champ non reconnu a été trouvé dans l'élément **FIXED** d'une déclaration de structure **VERSIONINFO**.  Le champ **VERSIONINFO** doit contenir des valeurs de type DWORD, séparées par une virgule.
