# <a name="resource-compiler-error-rc2167"></a>Erreur RC2167 du compilateur de ressources 

champ VERSIONINFO non reconnu ; BEGIN ou virgule ATTENDU

Un champ non reconnu a été trouvé dans la partie **fixe** d’une déclaration de structure **VERSIONINFO** . Un champ **VERSIONINFO** doit être composé de valeurs DWORD séparées par une virgule.
