# Avertissement de g&#233;n&#233;ration de projet PRJ0029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La propriété 'Sorties' de l'étape de build personnalisée au niveau du projet n'est pas définie.L'étape de build personnalisée va être ignorée.  
  
 Une étape de build personnalisée n'a pas été exécutée car aucune sortie n'était spécifiée.  
  
 Pour corriger cette erreur, effectuez l'une des opérations suivantes :  
  
-   Excluez l'étape de build personnalisée de la build.  
  
-   Ajoutez une sortie.  
  
-   Supprimez le contenu de la commande de l'étape de build personnalisée.
