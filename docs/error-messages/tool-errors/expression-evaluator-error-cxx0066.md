# <a name="expression-evaluator-error-cxx0066"></a>Évaluateur d'expression, erreur CXX0066

membre statique absent

Un membre statique d’une classe est introuvable ou n’a été défini. Cette erreur peut résulter d’une classe statique membre qui est déclaré mais pas défini, ou uniquement défini et référencé dans les modules qui ne contiennent pas les informations de débogage.

Cette erreur est identique à CAN0066.
