# &#201;valuateur d&#39;expression, erreur CXX0066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

membre statique absent  
  
 Un membre de classe statique est introuvable ou n'est pas défini.  Cette erreur peut se produire lorsqu'un membre de classe statique est déclaré mais pas défini, ou encore s'il est défini et référencé uniquement dans des modules qui ne contiennent pas d'informations de débogage.  
  
 Erreur identique à CAN0066.
