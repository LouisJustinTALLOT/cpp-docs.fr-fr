# <a name="expression-evaluator-error-cxx0066"></a>Évaluateur d'expression, erreur CXX0066

membre statique absent

Un membre statique d’une classe est introuvable ou n’a pas été défini. Cette erreur peut résulter d’un membre de classe statique qui est déclaré mais pas défini, ou qui est uniquement défini et référencé dans des modules qui ne contiennent pas d’informations de débogage.

Cette erreur est identique à CAN0066.
