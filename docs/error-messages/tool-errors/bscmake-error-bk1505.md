# Erreur BSCMAKE BK1505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossible de lire le fichier 'nom\_fichier' \[: raison\]  
  
 BSCMAKE ne peut pas lire le fichier.  
  
### Pour résoudre le problème en vérifiant les causes possibles suivantes  
  
1.  Fichier endommagé.  
  
2.  Troncation du fichier lorsque le compilateur manque d'espace disque ou lorsqu'il est interrompu lors de la création du fichier .sbr.
