# <a name="bscmake-error-bk1505"></a>Erreur BSCMAKE BK1505
Impossible de lire le fichier 'nom_fichier' [ : raison]  
  
 BSCMAKE ne peut pas lire le fichier.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes  
  
1.  Endommagement du fichier.  
  
2.  Le compilateur dispose plus de suffisamment d’espace disque ou interrompue lors de la création du fichier .sbr troncation du fichier.
