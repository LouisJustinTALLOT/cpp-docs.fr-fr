# <a name="bscmake-error-bk1505"></a>Erreur BSCMAKE BK1505

Impossible de lire à partir du fichier’nom_fichier' [ : raison]

BSCMAKE ne peut pas lire à partir du fichier.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes

1. Fichier endommagé.

1. La troncation du fichier au compilateur ne dispose plus de suffisamment d’espace disque ou est interrompue lors de la création du fichier. sbr.
