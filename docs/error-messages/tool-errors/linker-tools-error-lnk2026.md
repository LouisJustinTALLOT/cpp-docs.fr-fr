# Erreur des outils &#201;diteur de liens LNK2026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

module unsafe pour l'image SAFESEH  
  
 [\/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) a été spécifié, mais un module n'était pas compatible avec la fonction de gestion sécurisée des exceptions.  Si vous souhaitez utiliser ce module avec **\/SAFESEH**, vous devrez recompiler le module au moyen du compilateur Visual C\+\+ .NET 2003 \(ou version ultérieure\).
