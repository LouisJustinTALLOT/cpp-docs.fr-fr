# <a name="linker-tools-error-lnk2026"></a>Erreur des outils Éditeur de liens LNK2026

module unsafe pour l’image SAFESEH

[/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) a été spécifié, mais un module n’était pas compatible avec la fonctionnalité de gestion sécurisée des exceptions. Si vous souhaitez utiliser ce module avec **/SAFESEH**, vous devrez recompiler le module.
