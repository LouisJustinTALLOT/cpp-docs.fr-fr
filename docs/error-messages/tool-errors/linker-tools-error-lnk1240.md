# Erreur des outils &#201;diteur de liens LNK1240
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

échec de la compilation du contenu IDL  
  
 L'éditeur de liens a lancé MIDL pour compiler le contenu MIDL incorporé mais un problème est survenu.  Vérifiez les erreurs spécifiées par MIDL.
