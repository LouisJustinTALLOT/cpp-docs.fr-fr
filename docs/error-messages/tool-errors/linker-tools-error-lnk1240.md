# <a name="linker-tools-error-lnk1240"></a>Erreur des outils Éditeur de liens LNK1240

échec de la compilation du contenu IDL

L’éditeur de liens a généré MIDL pour compiler l’IDL incorporé, mais un problème est survenu. Vérifiez les erreurs spécifiées par MIDL.
