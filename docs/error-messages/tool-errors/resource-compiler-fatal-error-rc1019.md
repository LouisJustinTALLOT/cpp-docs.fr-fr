# <a name="resource-compiler-fatal-error-rc1019"></a>Erreur irrécupérable RC1019 du compilateur de ressources 

' #else’inattendu

La `#else` directive n’apparaît pas dans une `#if` construction, **#ifdef** ou **#ifndef** .

Assurez-vous qu’il existe une `#if` instruction, **#ifdef** ou **#ifndef** en vigueur avant cette instruction.
