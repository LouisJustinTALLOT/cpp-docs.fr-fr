# <a name="resource-compiler-fatal-error-rc1019"></a>Erreur irrécupérable RC1019 du compilateur de ressources

' #else’inattendu

La directive `#else` n’apparaissait pas dans une construction `#if`, **#ifdef**ou **#ifndef** .

Assurez-vous qu’il existe une instruction `#if`, **#ifdef**ou **#ifndef** en vigueur avant cette instruction.
