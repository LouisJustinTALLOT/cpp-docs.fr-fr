# <a name="expression-evaluator-error-cxx0002"></a>Évaluateur d'expression, erreur CXX0002

Erreur de mémoire utilisateur l’accès à

L’expression tente de référencer la mémoire qui n’est pas allouée au programme en cours de débogage.

Cette erreur est identique à CAN0002.
