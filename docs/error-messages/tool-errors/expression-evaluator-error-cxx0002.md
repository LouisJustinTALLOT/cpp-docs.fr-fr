# &#201;valuateur d&#39;expression, erreur CXX0002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erreur lors d'un accès à la mémoire utilisateur  
  
 L'expression essaie de faire référence à une mémoire qui n'est pas allouée au programme en cours de débogage.  
  
 Erreur identique à CAN0002.
