# <a name="expression-evaluator-error-cxx0002"></a>Évaluateur d'expression, erreur CXX0002

erreur lors de l’accès à la mémoire utilisateur

L’expression tente de faire référence à la mémoire qui n’est pas allouée au programme en cours de débogage.

Cette erreur est identique à CAN0002.
