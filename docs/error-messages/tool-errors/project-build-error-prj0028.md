# <a name="project-build-error-prj0028"></a>Erreur de génération de projet PRJ0028

Le fichier temporaire’fichier’contient du contenu Unicode qui n’a pas pu être traduit dans la page de codes ANSI de l’utilisateur.

Une valeur a été spécifiée avec l’option de l’éditeur de liens [/MIDL (spécifier les options de ligne de commande MIDL)](../../build/reference/midl-specify-midl-command-line-options.md) qui n’a pas pu être résolue par la page de codes système.

La page de codes utilisée lorsque vous spécifiez la commande MIDL (la page de codes d’entrée) doit être identique à la page de codes système.
