# Erreur de g&#233;n&#233;ration de projet PRJ0028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Le fichier temporaire 'fichier' contient du code Unicode qui n'a pas pu être traduit dans la page de codes ANSI de l'utilisateur.  
  
 Une valeur spécifiée à l'aide de l'option [\/MIDL \(Spécifier les options de ligne de commande MIDL\)](../../build/reference/midl-specify-midl-command-line-options.md) de l'éditeur de liens n'a pas pu être traduite par la page de codes système.  
  
 La page de codes utilisée lors de la spécification de la commande MIDL \(la page de codes d'entrée\) doit être identique à la page de codes système.
