# <a name="project-build-error-prj0028"></a>Erreur de génération de projet PRJ0028
Fichier temporaire 'fichier' contient Unicode qui n’a pas pu être traduits à la page de codes ANSI de l’utilisateur.  
  
 Une valeur a été spécifiée avec le [/MIDL (spécifier les Options de ligne de commande MIDL)](../../build/reference/midl-specify-midl-command-line-options.md) option de l’éditeur de liens n’a pas pu être résolue par la page de codes du système.  
  
 La page de codes utilisée lorsque vous spécifiez la commande MIDL (la page de codes en entrée) doit être identique à la page de codes du système.
