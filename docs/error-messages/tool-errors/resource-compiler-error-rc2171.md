# <a name="resource-compiler-error-rc2171"></a>Erreur RC2171 du compilateur de ressources 

format d’en-tête DIB inconnu

L’en-tête de la bitmap n’est pas un **structure BITMAPCOREHEADER** ou **BITMAPINFOHEADER** structure.
