# Erreur irr&#233;cup&#233;rable NMAKE U1083
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la macro cible 'cible' ne se développe vers rien  
  
 La cible donnée est un appel d'une macro qui n'a pas été définie ou dont la valeur est null.  NMAKE ne peut pas traiter une cible null.
