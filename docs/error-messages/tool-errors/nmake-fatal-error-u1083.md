# <a name="nmake-fatal-error-u1083"></a>Erreur irrécupérable NMAKE U1083

la macro cible’target’ne se développe vers rien

La cible donnée est un appel d’une macro qui n’a pas été définie ou a une valeur null. NMAKE ne peut pas traiter une cible NULL.
