# <a name="linker-tools-error-lnk1113"></a>Erreur des outils Éditeur de liens LNK1113

type de machine non valide

Le type d’ordinateur spécifié dans l’en-tête d’objet n’est pas valide.

Cette erreur peut se produire si le fichier est endommagé. Régénérez l’objet.
