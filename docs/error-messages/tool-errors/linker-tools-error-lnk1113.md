# Erreur des outils &#201;diteur de liens LNK1113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

type d'ordinateur non valide type  
  
 Le type d'ordinateur spécifié dans l'en\-tête d'objet n'est pas valide.  
  
 Cette erreur peut être provoquée si le fichier est endommagé.  Régénérez l'objet.
