# <a name="resource-compiler-fatal-error-rc1047"></a>Erreur irrécupérable RC1047 du compilateur de ressources 

« trop d’options d’option, 'chaîne' »

L’option donnée a été spécifiée trop souvent. La chaîne donnée est l’argument de l’option à l’origine de l’erreur.
