# <a name="resource-compiler-fatal-error-rc1047"></a>Erreur irrécupérable RC1047 du compilateur de ressources 

« trop d’option options 'string' »

L’option donnée a été spécifiée trop de fois. La chaîne donnée est l’argument de l’option qui a provoqué l’erreur.
