# <a name="expression-evaluator-error-cxx0004"></a>Évaluateur d'expression, erreur CXX0004

erreur de syntaxe

La syntaxe de l’expression est incorrecte.

Retapez l’expression avec la syntaxe correcte.

Cette erreur est identique à CAN0004.
