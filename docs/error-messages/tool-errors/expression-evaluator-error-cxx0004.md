# &#201;valuateur d&#39;expression, erreur CXX0004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erreur de syntaxe  
  
 La syntaxe de l'expression n'est pas correcte.  
  
 Entrez à nouveau l'expression en utilisant la syntaxe correcte.  
  
 Erreur identique à CAN0004.
