# <a name="linker-tools-error-lnk1287"></a>Erreur des outils Éditeur de liens LNK1287
fonction de point d’entrée managé non valide  
  
 Le point d’entrée n’est pas valide pour une image managée. Le type de retour d’une fonction de point d’entrée managé peut uniquement être `void` ou `int`. Le type du paramètre d’une fonction de point d’entrée managé peut uniquement être `void` ou `String []`.
