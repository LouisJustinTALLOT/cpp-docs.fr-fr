# Erreur des outils &#201;diteur de liens LNK1287
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

fonction de point d'entrée managée non valide  
  
 Ce point d'entrée n'est pas valide pour une image managée.  Le type de retour d'une fonction de point d'entrée managée ne peut être que `void` ou `int`.  Le type du paramètre d'une fonction de point d'entrée managée ne peut être que `void` ou `String []`.
