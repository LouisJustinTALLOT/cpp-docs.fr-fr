# Avertissement NMAKE U4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

le fichier de commandes ne peut être appelé qu'à partir de la ligne de commande  
  
 Un fichier de commandes, qui est appelé par le spécificateur arobas \(**@**\), ne peut pas contenir une spécification pour un autre fichier de commandes.  Une telle imbrication n'est pas autorisée.  La spécification a été ignorée.
