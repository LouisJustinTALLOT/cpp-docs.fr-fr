# <a name="nmake-warning-u4001"></a>Avertissement NMAKE U4001

> le fichier de commandes ne peut être appelé qu’à partir de la ligne de commande

Un fichier de commandes, qui est appelé par le spécificateur arobase ( **\@** ), ne peut pas contenir une spécification pour un autre fichier de commandes. Cette imbrication n’est pas autorisée. La spécification a été ignorée.
