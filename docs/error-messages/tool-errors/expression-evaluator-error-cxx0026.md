# <a name="expression-evaluator-error-cxx0026"></a>Évaluateur d'expression, erreur CXX0026
chaîne de format incorrect  
  
 Une chaîne de format a été incorrectement spécifiée. Vérifiez la syntaxe de l’expression.  
  
 Cette erreur est identique à CAN0026.  
  
 Pour plus d’informations, consultez [spécificateurs de Format en C++](/visualstudio/debugger/format-specifiers-in-cpp).
