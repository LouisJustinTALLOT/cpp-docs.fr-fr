# Erreur RC2170 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

le nom du fichier bitmap n'est pas au format 3.00  
  
 Les images bitmap au format Windows version 2.x ne peuvent pas être utilisées dans les fichiers de ressources de version 3.x.  
  
 Pour résoudre ce problème, redessinez ou convertissez l'image bitmap au format 3.x.
