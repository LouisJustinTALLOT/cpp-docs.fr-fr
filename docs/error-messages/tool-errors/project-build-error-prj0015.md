# Erreur de g&#233;n&#233;ration de projet PRJ0015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Le périphérique NULL est introuvable sur votre système.Impossible de lancer une génération.  
  
 Cette erreur peut être due à des ressources système insuffisantes.  Fermez certaines applications ou redémarrez.  
  
 Elle peut également être due à des privilèges insuffisants pour le périphérique NULL sur l'ordinateur.
