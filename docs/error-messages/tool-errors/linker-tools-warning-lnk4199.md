# <a name="linker-tools-warning-lnk4199"></a>Avertissement des outils Éditeur de liens LNK4199
/ DELAYLOAD : nomdll ignoré ; aucune importation trouvée à partir de NomDll  
  
 L’éditeur de liens ignore `dllname` , car elle ne doit pas toutes les fonctions qui `dllname` exporte.
