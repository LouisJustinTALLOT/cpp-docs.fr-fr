# <a name="linker-tools-warning-lnk4199"></a>Avertissement des outils Éditeur de liens LNK4199

/DELAYLOAD : DllName ignoré ; aucune importation trouvée à partir de DllName

L’éditeur de liens ignore `dllname`, car il n’a besoin d’aucune des fonctions que `dllname` exporte.
