# Avertissement des outils &#201;diteur de liens LNK4199
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/DELAYLOAD:NomDll ignoré ; aucune importation trouvée à partir de NomDll  
  
 L'éditeur de liens ignore `dllname` parce qu'il n'a besoin d'aucune des fonctions exportées par `dllname`.
