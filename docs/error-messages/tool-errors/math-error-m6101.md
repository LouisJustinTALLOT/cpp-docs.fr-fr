# Erreur math&#233;matique M6101
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

non valide  
  
 Opération non valide.  
  
 Cette erreur peut être provoquée lorsqu'un opérande est la valeur NaN \(n'est pas un nombre\) ou l'infini.  
  
 Le programme se termine par le code de sortie 129.
