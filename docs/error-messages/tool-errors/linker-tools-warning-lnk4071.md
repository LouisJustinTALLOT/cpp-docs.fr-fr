# Avertissement des outils &#201;diteur de liens LNK4071
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ne peut pas être lié\(e\) de façon incrémentielle sur les liens qui suivent  
  
 LINK a trouvé plusieurs définitions pour un ou plusieurs symboles, mais [\/FORCE](../../build/reference/force-force-file-output.md) ou **\/FORCE:MULTIPLE** a été utilisé pour créer un fichier de sortie malgré ces erreurs.  LINK a supprimé le fichier d'état incrémentiel \(.ilk\).
