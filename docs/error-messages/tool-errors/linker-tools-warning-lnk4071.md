# <a name="linker-tools-warning-lnk4071"></a>Avertissement des outils Éditeur de liens LNK4071

ne peut pas être lié de façon incrémentielle sur les liens qui suivent

LINK a trouvé plusieurs définitions pour un ou plusieurs symboles, mais [/FORCE](../../build/reference/force-force-file-output.md) ou **multiple** a été utilisé pour créer un fichier de sortie, quel que soit les erreurs. LIEN supprimé le fichier d’état incrémentiel (.ilk).
