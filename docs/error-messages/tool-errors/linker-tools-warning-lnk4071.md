# <a name="linker-tools-warning-lnk4071"></a>Avertissement des outils Éditeur de liens LNK4071
ne peut pas être lié de façon incrémentielle sur les liens suivants  
  
 LINK a trouvé plusieurs définitions pour un ou plusieurs symboles, mais [/force](../../build/reference/force-force-file-output.md) ou **multiple** a été utilisé pour créer un fichier de sortie malgré ces erreurs. LIEN supprimé le fichier d’état incrémentiel (.ilk).
