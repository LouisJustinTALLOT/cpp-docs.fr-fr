# <a name="linker-tools-warning-lnk4229"></a>Avertissement des outils Éditeur de liens LNK4229

/directive de directive trouvé non valide ; pas

Une directive passée via le pragma de [Commentaire](../../preprocessor/comment-c-cpp.md) n’est pas valide. L’éditeur de liens ignore `/directive`.
