# &#201;valuateur d&#39;expression, erreur CXX0016
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

constante trop grande  
  
 L'évaluateur d'expression C ne peut pas accepter une constante entière non signée plus grande que 4 294 967 295 \(0FFFFFFFF en hexadécimal\), ni une constante à virgule flottante dont la magnitude dépasse 1,8E\+308 environ.  
  
 Erreur identique à CAN0016.
