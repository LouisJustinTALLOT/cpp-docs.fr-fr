# <a name="expression-evaluator-error-cxx0016"></a>Évaluateur d'expression, erreur CXX0016

constante trop grande

L’évaluateur d’expression C ne peut pas accepter une constante entière non signée supérieure à 4 294 967 295 (0FFFFFFFF hexadécimale), ou une constante à virgule flottante dont l’amplitude est supérieure à environ 1.8 E + 308.

Cette erreur est identique à CAN0016.
