# Erreur irr&#233;cup&#233;rable RW1009 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erreur lors de la création du nom de ressource  
  
 Il n'a pas été possible de créer le fichier .res spécifié.  Assurez\-vous que celui\-ci n'est pas créé sur un lecteur en lecture seule.  Utilisez \/V pour vérifier si le fichier est en cours de création.
