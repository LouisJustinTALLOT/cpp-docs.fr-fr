# <a name="resource-compiler-fatal-error-rw1009"></a>Erreur irrécupérable RW1009 du compilateur de ressources 

Erreur lors de la création de la ressource-nom

Impossible de créer le fichier. res spécifié. Assurez-vous qu’il n’est pas créé sur un lecteur en lecture seule. Utilisez/V pour déterminer si le fichier est en cours de création.
