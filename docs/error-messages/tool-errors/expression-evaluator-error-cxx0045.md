# &#201;valuateur d&#39;expression, erreur CXX0045
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

n'est pas une fonction  
  
 Une liste d'arguments qui n'est pas le nom d'une fonction est fournie pour un symbole du programme.  
  
## Exemple  
  
```  
queue( alpha, beta )  
```  
  
 lorsque `queue` n'est pas une fonction.
