# <a name="expression-evaluator-error-cxx0006"></a>Évaluateur d'expression, erreur CXX0006
parenthèse ouvrante manquante  
  
 Parenthèses non équilibrées ont été trouvés dans l’expression. Tapez à nouveau l’expression entre parenthèses à charge équilibrée.  
  
 Cette erreur est identique à CAN0006.
