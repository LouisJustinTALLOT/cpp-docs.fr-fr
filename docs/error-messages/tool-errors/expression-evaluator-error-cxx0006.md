# <a name="expression-evaluator-error-cxx0006"></a>Évaluateur d'expression, erreur CXX0006

parenthèse gauche manquante

Non concordance des parenthèses ont été trouvés dans l’expression. Retapez l’expression entre parenthèses à charge équilibrée.

Cette erreur est identique à CAN0006.
