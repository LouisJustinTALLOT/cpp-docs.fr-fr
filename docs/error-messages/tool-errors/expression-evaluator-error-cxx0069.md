# &#201;valuateur d&#39;expression, erreur CXX0069
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la variable a besoin du frame de pile  
  
 L'évaluateur d'expression ne peut pas évaluer la variable parce qu'elle ne se trouve pas dans un frame de pile.  Cette erreur peut être causée par des variables déclarées dans le cadre d'une fonction Inline.
