# <a name="expression-evaluator-error-cxx0069"></a>Évaluateur d'expression, erreur CXX0069

la variable a besoin d’un frame de pile

L’évaluateur d’expression ne peut pas évaluer la variable, car elle ne se produit pas dans un frame de pile. Cela peut être dû à des variables déclarées dans le cadre d’une fonction Inline.
