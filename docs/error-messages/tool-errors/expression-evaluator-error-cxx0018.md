# &#201;valuateur d&#39;expression, erreur CXX0018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nom de registre incorrect  
  
 Un registre spécifié n'existe pas ou ne peut pas être affiché.  
  
 La fenêtre Espion peut afficher les registres suivants :  
  
 AX    SP    DS    IP  
  
 BX    BP    ES    FL  
  
 CX    SI    SS    GS  
  
 DX    DI    CS    SS  
  
 EAX    ESP    DS    EIP  
  
 EBX    EBP    ES    EFL  
  
 ECX    ESI    FS  
  
 EDX    EDI    CS  
  
 Cette erreur est identique à l'erreur CAN0018.
