# Avertissement RC4214 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

page de codes non valide : ignorée  
  
 La page de codes spécifiée pour l'argument CODEPAGE du fichier .rc n'est pas valide.  Pour plus d'informations, consultez [IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674).
