# <a name="resource-compiler-warning-rc4214"></a>Avertissement RC4214 du compilateur de ressources 
Page de codes non valide : ignoré  
  
 Le fichier .rc contenue un argument de la page de codes et de la page de codes spécifiée n’est pas valide. Consultez [IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674) pour plus d’informations.
