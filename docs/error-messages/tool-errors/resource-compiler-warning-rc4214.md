# <a name="resource-compiler-warning-rc4214"></a>Avertissement RC4214 du compilateur de ressources

Page de codes non valide: ignoré

Le fichier. RC contenait un argument CodePage et la page de codes spécifiée n’est pas valide. Pour plus d’informations, consultez [IsValidCodePage](/windows/win32/api/winnls/nf-winnls-isvalidcodepage) .
