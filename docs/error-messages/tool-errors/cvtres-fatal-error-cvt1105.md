# <a name="cvtres-fatal-error-cvt1105"></a>Erreur irrécupérable CVTRES CVT1105

Impossible de rechercher dans le fichier

CVTRES n’a pas pu accéder à un emplacement dans le fichier.

Cette erreur peut être due à l’endommagement d’un fichier.
