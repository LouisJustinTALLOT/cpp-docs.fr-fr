# <a name="linker-tools-warning-lnk4216"></a>Avertissement des outils Éditeur de liens LNK4216

Entrée de point d’entrée exportée

Un point d’entrée a été exporté à partir d’une DLL. Le point d’entrée d’une DLL n’a pas besoin d’être exporté.
