# Avertissement des outils &#201;diteur de liens LNK4216
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Entrée de point d'entrée exporté  
  
 Un point d'entrée a été exporté depuis une DLL.  Le point d'entrée d'une DLL n'a pas besoin d'être exporté.
