# <a name="expression-evaluator-error-cxx0023"></a>Évaluateur d'expression, erreur CXX0023

base incorrecte

L’évaluateur d’expression C ne reconnaît pas la base spécifiée. Seules les bases décimales, hexadécimales et octales sont valides.

Cette erreur est identique à CAN0023.
