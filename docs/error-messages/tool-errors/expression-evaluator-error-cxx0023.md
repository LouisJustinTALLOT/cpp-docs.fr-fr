# &#201;valuateur d&#39;expression, erreur CXX0023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

base incorrecte  
  
 L'évaluateur d'expression C ne reconnaît pas la base spécifiée.  Seules les bases décimales, hexadécimales et octales sont valides.  
  
 Erreur identique à CAN0023.
