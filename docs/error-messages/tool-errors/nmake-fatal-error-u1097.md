# <a name="nmake-fatal-error-u1097"></a>Erreur irrécupérable NMAKE U1097
syntaxe de parties de nom de fichier nécessite dependent  
  
 La dépendance actuelle n’a pas un dépendant explicite ni dépendant implicite. Syntaxe de parties de nom de fichier, qui utilise le pourcentage (`%`) spécificateur, représente des composants du premier dépendant de la cible actuelle.
