# Erreur irr&#233;cup&#233;rable RC1121 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erreur d'E\/S lors de la lecture du fichier  
  
 Le compilateur de ressources n'a pas pu lire le fichier.  Assurez\-vous que le lecteur contenant le fichier est accessible et que le fichier est valide.
