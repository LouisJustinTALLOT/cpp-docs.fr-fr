# <a name="project-build-error-prj0020"></a>Erreur de génération de projet PRJ0020

> Outil'*outil*', la propriété'*propriété*'contient le nom de fichier'*fichier*'non valide.

Le *fichier*de nom de fichier, spécifié dans la *propriété* Property de *l’outil Tool, n'* était pas valide.

Vous avez peut-être utilisé une macro inconnue ou non valide.

Pour plus d’informations sur la façon d’accéder aux propriétés de votre projet, consultez [définition des propriétés de projet visuel C++ ](../../build/working-with-project-properties.md) .
