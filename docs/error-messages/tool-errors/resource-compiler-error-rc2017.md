# <a name="resource-compiler-error-rc2017"></a>Erreur RC2017 du compilateur de ressources 

séquence d’échappement non conforme

Une séquence d’échappement est apparue alors qu’aucune n’était attendue.

Une séquence d’échappement (une barre oblique inverse ( **\\** ) suivie d’un nombre ou d’une lettre) peut se produire uniquement dans une constante de chaîne ou de caractère.
