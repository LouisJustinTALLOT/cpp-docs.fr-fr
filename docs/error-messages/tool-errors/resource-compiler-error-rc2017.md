# <a name="resource-compiler-error-rc2017"></a>Erreur RC2017 du compilateur de ressources 
séquence d’échappement non conforme  
  
 Une séquence d’échappement est apparue dans lequel une n’était pas attendue.  
  
 Une séquence d’échappement — une barre oblique inverse (  **\\**  ) suivi d’un nombre ou une lettre, peut se produire uniquement dans un caractère ou une chaîne constante.
