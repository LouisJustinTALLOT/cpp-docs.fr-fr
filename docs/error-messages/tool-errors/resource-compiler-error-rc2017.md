# Erreur RC2017 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

séquence d'échappement non conforme  
  
 Séquence d'échappement non attendue trouvée.  
  
 Une séquence d'échappement — une barre oblique inverse \(**\\**\) suivi d'un chiffre ou d'une lettre — peut uniquement être utilisée dans constante de chaîne ou une constante caractère.
