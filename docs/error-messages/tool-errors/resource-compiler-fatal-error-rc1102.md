# <a name="resource-compiler-fatal-error-rc1102"></a>Erreur irrécupérable RC1102 du compilateur de ressources 

erreur interne : arguments trop nombreux dans RCPP

Trop d’arguments ont été passés au préprocesseur du compilateur de ressources. Réduisez le nombre de symboles définis avec l’option define Symbols (/d) en les définissant dans votre source. Cette erreur peut également être due à la spécification d’un trop grand nombre de chemins d’accès de recherche de fichiers à l’aide de l’option inclure le chemin de recherche (/i).
