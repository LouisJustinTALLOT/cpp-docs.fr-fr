# Erreur des outils &#201;diteur de liens LNK1103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

informations de débogage endommagées ; recompilez le module  
  
 Cette erreur peut être provoquée parce que vous avez mis fin à la compilation avant la création d'un fichier objet valide.  
  
 Recompilez le fichier objet donné.  Si la recompilation ne corrige pas le problème, notez les circonstances de l'erreur, essayez de cerner le problème et de créer un exemple de test reproductible.  Consultez ensuite le fichier d'aide de support technique ou la section de support technique de l'un de vos manuels.
