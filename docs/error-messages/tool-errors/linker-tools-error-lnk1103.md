# <a name="linker-tools-error-lnk1103"></a>Erreur des outils Éditeur de liens LNK1103

informations de débogage endommagées ; recompiler le module

Cette erreur peut être due au fait que la compilation a été arrêtée avant la création d’un fichier objet valide.

Recompilez le fichier objet donné. Si la recompilation ne résout pas le problème, notez les circonstances de l’erreur, essayez d’isoler le problème et créez un cas de test reproductible. Consultez ensuite le fichier d’aide du support technique ou la section Support technique dans l’un de vos manuels.
