# <a name="nmake-fatal-error-u1086"></a>Erreur irrécupérable NMAKE U1086

**la règle d’inférence ne peut pas avoir de dépendants**

Le signe deux-points ( **:** ) dans une règle d’inférence doit être suivi par l’un des éléments suivants :

- Caractère de saut de ligne

- Point-virgule ( **;** ), qui peut être suivi par une commande

- Signe dièse ( **#** ), qui peut être suivi d’un commentaire
