# Avertissement BSCMAKE BK4503
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

erreur mineure dans le nom du fichier .SBR ignorée  
  
 Le fichier .sbr contenait une erreur qui n'a pas arrêté la génération, mais le fichier .bsc obtenu risque d'être incorrect.  Pour résoudre ce problème, recompilez pour créer un nouveau fichier .sbr.
