# <a name="bscmake-error-bk1504"></a>Erreur BSCMAKE BK1504

Impossible de positionner dans le fichier’nom_fichier' [ : raison]

BSCMAKE n’a pas pu se déplacer vers un emplacement dans le fichier.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes

1. Disque plein.

1. Une erreur matérielle.

1. Troncation de fichier en raison d’un espace disque insuffisant ou d’une interruption du compilateur lors de la création du fichier. sbr.
