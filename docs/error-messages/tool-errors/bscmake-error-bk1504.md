# Erreur BSCMAKE BK1504
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

positionnement impossible dans le fichier 'nom\_fichier' \[: raison\]  
  
 BSCMAKE n'a pas pu atteindre un emplacement dans le fichier.  
  
### Pour résoudre le problème en vérifiant les causes possibles suivantes  
  
1.  Un disque plein.  
  
2.  Une erreur matérielle.  
  
3.  Une troncation du fichier due au manque d'espace disque ou à une interruption du compilateur lors de la création du fichier .sbr.
