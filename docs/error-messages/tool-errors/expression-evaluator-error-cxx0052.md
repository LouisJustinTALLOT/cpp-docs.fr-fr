# &#201;valuateur d&#39;expression, erreur CXX0052
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

fonction membre absente  
  
 Une fonction membre spécifiée comme point d'arrêt est introuvable.  Vous pouvez provoquer cette erreur en définissant un point d'arrêt sur une fonction qui a été inline.  
  
 Recompilez le fichier en désactivant la fonctionnalité inline \(\/Ob0\) pour définir un point d'arrêt dans cette fonction.  
  
 Une expression a appelé une fonction qui n'était pas définie.  
  
 Erreur identique à CAN0052.
