# <a name="expression-evaluator-error-cxx0020"></a>Évaluateur d'expression, erreur CXX0020

types d’opérandes incorrects pour cette opération

Un opérateur a été appliqué à une expression dont le type n’est pas valide pour cet opérateur.

Par exemple, il n’est pas possible de prendre l’adresse d’un registre ou d’indicer un tableau avec une expression à virgule flottante.

Cette erreur est identique à CAN0020.
