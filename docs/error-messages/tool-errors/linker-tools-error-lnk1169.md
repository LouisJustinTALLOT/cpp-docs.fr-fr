# <a name="linker-tools-error-lnk1169"></a>Erreur des outils Éditeur de liens LNK1169

un ou plusieurs symboles définis par multiplication ont été trouvés

La génération a échoué en raison de plusieurs définitions d’un ou plusieurs symboles. Cette erreur est précédée de l’erreur [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md).

L’option [/force](../../build/reference/force-force-file-output.md) ou **/force : multiple** remplace cette erreur.
