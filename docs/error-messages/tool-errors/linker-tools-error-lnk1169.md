# Erreur des outils &#201;diteur de liens LNK1169
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

un ou plusieurs symboles définis à différentes reprises ont été rencontrés  
  
 La génération a échoué suite à des définitions multiples d'un ou plusieurs symboles.  Cette erreur est précédée par l'erreur [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md).  
  
 L'option [\/FORCE](../../build/reference/force-force-file-output.md) ou **\/FORCE:MULTIPLE** permet de substituer cette erreur.
