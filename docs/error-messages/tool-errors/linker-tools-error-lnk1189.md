# <a name="linker-tools-error-lnk1189"></a>Erreur des outils Éditeur de liens LNK1189

LIBTOOMANYMEMBERS:: limite du nombre d'objets dans la bibliothèque dépassée

La limite de 65 536 objets ou membres dans une bibliothèque a été dépassée.

Pour résoudre ce problème, fractionnez votre bibliothèque en au moins deux bibliothèques contenant moins d'objets ou de membres.
