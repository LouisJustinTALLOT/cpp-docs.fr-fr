# Erreur des outils &#201;diteur de liens LNK1332
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\<count\> types de Windows Runtime détectés, importés dans un module et définis dans un autre module  
  
 Lorsqu'il a généré la cible actuelle, l'éditeur de liens a détecté des types de Windows Runtime \<`count`\> , chacun étant importé dans un module et également défini dans un autre module.  
  
### Pour corriger cette erreur  
  
-   Corrigez chacune des erreurs LNK2039 dans la construction en accord avec la suggestion dans le message d'erreur.
