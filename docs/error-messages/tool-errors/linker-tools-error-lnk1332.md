# <a name="linker-tools-error-lnk1332"></a>Erreur des outils Éditeur de liens LNK1332
détecté\<count > types Windows Runtime importés dans un module et définis dans un autre module  
  
 Lorsqu’il a généré la cible actuelle, l’éditeur de liens a détecté <`count`> types Windows Runtime, chacun d’eux est importé dans un module et également défini dans un autre module.  
  
### <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Corrigez chacune des erreurs LNK2039 dans la build en fonction de la suggestion dans le message d’erreur.
