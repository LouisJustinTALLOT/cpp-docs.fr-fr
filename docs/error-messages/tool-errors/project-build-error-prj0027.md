# Erreur de g&#233;n&#233;ration de projet PRJ0027
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Le message du fichier journal Unicode 'contenu' contient du code qui n'a pas pu être traduit dans la page de codes ANSI de l'utilisateur.  
  
 Cet avertissement apparaît généralement en combinaison avec des erreurs de création de fichiers batch et\/ou réponse.  
  
 Pour corriger cette erreur, mettez à jour le contenu du journal de génération de façon à utiliser ANSI ou installez la page de codes sur votre ordinateur et définissez\-la en tant que valeur système par défaut.
