# <a name="project-build-error-prj0027"></a>Erreur de génération de projet PRJ0027

Le message du journal Unicode « contenu » contient du contenu qui n’a pas pu être traduit dans la page de codes ANSI de l’utilisateur.

En général, cet avertissement s’affiche uniquement avec des erreurs lors de la création de fichiers de commandes et/ou de réponses.

La résolution de cette erreur consiste à mettre à jour le contenu du journal de génération pour utiliser ANSI ou pour installer la page de codes sur votre ordinateur et la définir comme valeur système par défaut.
