# <a name="linker-tools-error-lnk1143"></a>Erreur des outils Éditeur de liens LNK1143

fichier non valide ou endommagé : aucun symbole pour le numéro de section COMDAT

Cette erreur peut se produire si le fichier objet est endommagé.

Régénérez le fichier.
