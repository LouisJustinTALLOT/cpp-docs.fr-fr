# Erreur irr&#233;cup&#233;rable NMAKE U1071
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

cycle dans l'arborescence des dépendances pour la cible 'nomcible'  
  
 Il existe une dépendance circulaire dans l'arborescence de dépendances de la cible donnée.  La cible donnée est un dépendant de l'un des dépendants de la cible donnée.  Les dépendances circulaires ne sont pas valides.
