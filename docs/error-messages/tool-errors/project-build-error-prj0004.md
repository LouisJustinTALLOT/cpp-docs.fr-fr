# <a name="project-build-error-prj0004"></a>Erreur de génération de projet PRJ0004

> Impossible de générer la ligne de commande pour l’outil'*Tool*'.

Une ou plusieurs propriétés ont été spécifiées de manière à rendre la syntaxe de l’appel à l' *outil* non conforme :

- Vous avez peut-être spécifié des macros incorrectes ou inconnues.

- L’espace disque disponible sur votre ordinateur est peut-être insuffisant.

Vous pouvez examiner les paramètres de l’outil en examinant la page de [Propriétés ligne de commande](../../build/reference/command-line-property-pages.md).
