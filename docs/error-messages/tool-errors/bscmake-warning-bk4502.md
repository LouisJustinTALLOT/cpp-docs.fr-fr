# Avertissement BSCMAKE BK4502
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

fichier .SBR tronqué 'nom\_fichier' n'est pas dans le nom de fichier  
  
 Un fichier .sbr de longueur nulle qui ne faisait pas originellement partie du fichier .bsc a été spécifié au cours d'une mise à jour.  
  
### Pour résoudre le problème en vérifiant les causes possibles suivantes  
  
1.  Nom de fichier spécifié incorrect.  
  
2.  Fichier supprimé. \(Résultats de l'erreur [BK1513](../../error-messages/tool-errors/bscmake-error-bk1513.md)\).  
  
3.  Fichier endommagé, BSCMAKE nécessaire pour effectuer une génération complète.
