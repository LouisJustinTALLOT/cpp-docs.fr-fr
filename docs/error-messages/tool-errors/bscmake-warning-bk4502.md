# <a name="bscmake-warning-bk4502"></a>Avertissement BSCMAKE BK4502

tronqué. Le fichier SBR’nom_fichier’n’est pas dans le nom de fichier

Un fichier. sbr de longueur nulle qui ne faisait pas partie à l’origine du fichier. bsc a été spécifié lors d’une mise à jour.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes

1. Nom de fichier spécifié incorrect.

1. Fichier supprimé. (Résultats de [BK1513](../../error-messages/tool-errors/bscmake-error-bk1513.md) d’erreur.)

1. Fichier endommagé, ce qui nécessite BSCMAKE pour effectuer une génération complète.
