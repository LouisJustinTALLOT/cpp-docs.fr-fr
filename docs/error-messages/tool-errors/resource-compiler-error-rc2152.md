# <a name="resource-compiler-error-rc2152"></a>Erreur RC2152 du compilateur de ressources

caractère de contrôle non valide

Un caractère de contrôle dans l’instruction **ACCELERATORS** n’est pas valide. Un caractère de contrôle valide se compose d’une lettre (uniquement) après un signe insertion ( **^** ).
