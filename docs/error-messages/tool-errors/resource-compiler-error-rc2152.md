# Erreur RC2152 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

caractère de contrôle non valide  
  
 Un caractère de contrôle de l'instruction **ACCELERATORS** est non valide.  Un caractère de contrôle valide ne doit comporter qu'une lettre après le signe insertion \(**^**\).
