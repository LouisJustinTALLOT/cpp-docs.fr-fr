# <a name="expression-evaluator-error-cxx0047"></a>Évaluateur d'expression, erreur CXX0047

la liste d’arguments ne correspond pas à une fonction

Une expression a appelé une fonction avec une liste de paramètres réelle qui ne correspondait pas à la liste de paramètres formels d’une fonction portant le même nom que celui défini dans le programme.

Les fonctions surchargées ne peuvent être appelées que s’il existe une correspondance de paramètre exacte ou une correspondance qui ne requiert pas la construction d’un objet.

Cette erreur est identique à CAN0047.
