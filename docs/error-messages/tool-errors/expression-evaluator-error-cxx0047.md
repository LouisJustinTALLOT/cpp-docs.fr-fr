# &#201;valuateur d&#39;expression, erreur CXX0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la liste des arguments ne correspond pas à la fonction  
  
 Une expression a appelé une fonction avec une liste de paramètres réels qui ne correspond pas à la liste de paramètres formels d'une fonction de même nom définie dans le programme.  
  
 Il n'est possible d'appeler des fonctions surchargées que si les paramètres correspondent exactement ou s'ils correspondent suffisamment pour ne pas nécessiter la construction d'un objet.  
  
 Erreur identique à CAN0047.
