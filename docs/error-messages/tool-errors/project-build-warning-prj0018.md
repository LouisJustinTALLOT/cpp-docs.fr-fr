# <a name="project-build-warning-prj0018"></a>Avertissement de génération de projet PRJ0018
Les variables d’environnement suivantes sont introuvables :  
  
 Une variable d’environnement n’est pas définie. Cette erreur répertorie les variables d’environnement qui n’étaient pas définies.  
  
 Consultez [Macros pour les commandes de génération personnalisées](../../ide/common-macros-for-build-commands-and-properties.md) pour plus d’informations sur les macros dans le processus de génération.
