# <a name="project-build-warning-prj0018"></a>Avertissement de génération de projet PRJ0018

Les variables d’environnement suivantes sont introuvables :

Une variable d’environnement n’est pas définie. Cette erreur répertorie les variables d’environnement qui n’ont pas été définies.

Pour plus d’informations sur les macros dans le processus de génération, consultez [macros pour les commandes de génération personnalisées](../../build/reference/common-macros-for-build-commands-and-properties.md) .
