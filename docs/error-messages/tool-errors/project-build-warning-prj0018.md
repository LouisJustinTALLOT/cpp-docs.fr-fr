# Avertissement de g&#233;n&#233;ration de projet PRJ0018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Les variables d'environnement suivantes sont introuvables :  
  
 Une variable d'environnement n'est pas définie.  Cette erreur répertorie les variables d'environnement qui n'étaient pas définies.  
  
 Pour plus d'informations sur les macros dans le processus de génération, consultez [Macros pour les commandes de génération personnalisées](../../ide/common-macros-for-build-commands-and-properties.md).
