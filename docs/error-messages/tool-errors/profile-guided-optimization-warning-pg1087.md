# Avertissement de l&#39;optimisation guid&#233;e par profil PG1087
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Le fichier PGC 'fichier' n'est pas valide.  
  
 Un fichier .pgc est endommagé, peut\-être en raison d'un achèvement anormal d'une session de profilage.  Réexécutez le scénario de profilage et achevez\-le normalement.
