# Erreur des outils &#201;diteur de liens LNK1120
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nombre externes non résolus  
  
 L'erreur LNK1120 donne un comptage \(`number`\) des externes non résolus pour cette édition de liens.  Les conditions pouvant être la cause d'externes non résolus sont décrites dans l'erreur [LNK2001](../../error-messages/tool-errors/linker-tools-error-lnk2001.md), qui précède ce message d'erreur \(un pour chaque externe non résolu\).  
  
 Pour plus d'informations, consultez [http:\/\/support.microsoft.com\/default.aspx?scid\=kb;en\-us;815650](http://support.microsoft.com/default.aspx?scid=kb;en-us;815650).
