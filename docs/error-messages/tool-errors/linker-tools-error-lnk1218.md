# <a name="linker-tools-error-lnk1218"></a>Erreur des outils Éditeur de liens LNK1218
Avertissement considéré comme une erreur ; Aucun fichier de sortie généré  
  
 Lorsque vous liez avec **/WX**, tous les avertissements de l’éditeur de liens sont considérés comme des erreurs, et aucun fichier de sortie ne sera créé.  
  
 Consultez [/WX (traiter les avertissements l’éditeur de liens comme des erreurs)](../../build/reference/wx-treat-linker-warnings-as-errors.md) pour plus d’informations.
