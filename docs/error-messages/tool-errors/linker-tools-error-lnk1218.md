# Erreur des outils &#201;diteur de liens LNK1218
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

avertissement traité comme erreur ; aucun fichier de sortie généré  
  
 Lorsque vous liez avec **\/WX**, tous avertissements de l'éditeur de liens sont considérés comme des erreurs, et aucun fichier de sortie n'est créé.  
  
 Pour plus d'informations, consultez [\/WX \(Traiter les avertissements de l'Éditeur de liens comme des erreurs\)](../../build/reference/wx-treat-linker-warnings-as-errors.md).
