# <a name="linker-tools-error-lnk1218"></a>Erreur des outils Éditeur de liens LNK1218

AVERTISSEMENT traité en tant qu’erreur ; aucun fichier de sortie généré

Quand vous créez un lien avec **/WX**, les avertissements de l’éditeur de liens sont traités comme des erreurs, et aucun fichier de sortie n’est créé.

Pour plus d’informations, consultez [/WX (traiter les avertissements de l’éditeur de liens comme des erreurs)](../../build/reference/wx-treat-linker-warnings-as-errors.md) .
