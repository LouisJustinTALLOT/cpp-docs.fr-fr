# <a name="resource-compiler-error-rc2165"></a>Erreur RC2165 du compilateur de ressources 

chaîne introuvable dans l’instruction DLGINCLUDE

L’instruction n’a pas spécifié de fichier include valide.

L’instruction **DLGINCLUDE** doit utiliser la syntaxe suivante :

**DLGINCLUDE "**<em>nom_fichier</em>**. h"**
