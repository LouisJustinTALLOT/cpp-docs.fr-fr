# Erreur RC2165 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

chaîne introuvable dans l'instruction DLGINCLUDE  
  
 L'instruction ne spécifie aucun fichier include valide.  
  
 L'instruction **DLGINCLUDE** doit utiliser la syntaxe suivante :  
  
 **DLGINCLUDE "*nomfichier*.h"**
