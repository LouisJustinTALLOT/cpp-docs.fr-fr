# Erreur irr&#233;cup&#233;rable RW1030 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erreur de sortie  
  
 Cette erreur peut se produire si le compilateur de ressources n'a pas pu créer le fichier de destination.  Assurez\-vous que vous avez suffisamment d'espace disque disponible et que vous disposez d'un accès en écriture pour le volume.
