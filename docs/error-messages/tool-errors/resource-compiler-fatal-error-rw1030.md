# <a name="resource-compiler-fatal-error-rw1030"></a>Erreur irrécupérable RW1030 du compilateur de ressources

Erreur de sortie

Cette erreur peut se produire si le compilateur de ressources n’a pas pu créer le fichier de destination. Assurez-vous que l’espace disque est suffisant et que vous disposez de l’autorisation en écriture sur le volume.
