# <a name="expression-evaluator-error-cxx0008"></a>Évaluateur d'expression, erreur CXX0008

**manquant «à la fin de la chaîne**

Le guillemet double attendu à la fin du littéral de chaîne est manquant. Retapez l’expression, en plaçant le littéral de chaîne entre guillemets doubles.

Cette erreur est identique à CAN0008.
