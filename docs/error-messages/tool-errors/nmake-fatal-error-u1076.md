# <a name="nmake-fatal-error-u1076"></a>Erreur irrécupérable NMAKE U1076

nom trop long

Une chaîne a dépassé l’une des limites suivantes :

- 1024 caractères pour un nom de macro.

- 256 caractères pour un chemin d’accès cible.

- 2048 caractères pour une commande.
