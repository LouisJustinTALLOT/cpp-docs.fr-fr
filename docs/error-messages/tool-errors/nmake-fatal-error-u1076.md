# Erreur irr&#233;cup&#233;rable NMAKE U1076
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nom trop long  
  
 Une chaîne dépasse l'une des limites suivantes :  
  
-   1 024 caractères pour un nom de macro.  
  
-   256 caractères pour un nom de chemin d'une cible.  
  
-   2 048 caractères pour une commande.
