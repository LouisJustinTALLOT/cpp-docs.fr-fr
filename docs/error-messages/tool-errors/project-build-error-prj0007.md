# <a name="project-build-error-prj0007"></a>Erreur de génération de projet PRJ0007

Impossible de créer le répertoire de sortie 'répertoire'. 

Visual C++ n’a pas pu créer un répertoire de sortie. Raisons possibles :

- Le répertoire dans lequel le répertoire de sortie doit être créé est en lecture seule.

- L’ordinateur ne dispose pas de suffisamment d’espace disque.

- Vous avez spécifié un chemin d’accès de répertoire non valide.

- Les autorisations de l’utilisateur ne sont pas suffisantes pour créer le répertoire.
