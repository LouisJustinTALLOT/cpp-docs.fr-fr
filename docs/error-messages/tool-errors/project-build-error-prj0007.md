# Erreur de g&#233;n&#233;ration de projet PRJ0007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossible de créer le répertoire de sortie 'répertoire'.  
  
 Visual C\+\+ n'a pas réussi à créer un répertoire de sortie.  Les raisons possibles sont les suivantes :  
  
-   Le répertoire dans lequel le répertoire de sortie devait être créé est en lecture seule.  
  
-   L'ordinateur n'offre pas un espace disque suffisant.  
  
-   Vous avez spécifié un chemin de répertoire non valide.  
  
-   Les autorisations accordées aux utilisateurs ne leur permettent pas de créer le répertoire.
