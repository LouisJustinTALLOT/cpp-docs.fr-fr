# <a name="project-build-error-prj0007"></a>Erreur de génération de projet PRJ0007

Impossible de créer la sortie répertoire 'répertoire'.

Échec de Visual C++ créer un répertoire de sortie. Raisons possibles :

- Le répertoire dans lequel le répertoire de sortie a été à créer est en lecture seule.

- Ordinateur manque d’espace disque.

- Vous avez spécifié un chemin d’accès de répertoire non valide.

- Autorisations de l’utilisateur ne sont pas suffisantes pour créer le répertoire.
