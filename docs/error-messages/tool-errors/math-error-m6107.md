# <a name="math-error-m6107"></a>Erreur mathématique M6107

désémulé

Une tentative d’exécution d’une instruction de coprocesseur non valide ou non prise en charge par l’émulateur a été effectuée.

Le programme se termine avec le code de sortie 135.
