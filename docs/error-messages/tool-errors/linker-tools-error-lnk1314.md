# Erreur des outils &#201;diteur de liens LNK1314
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

table de symboles COFF endommagée ou non valide \(symbole static ou symbole d'étiquette non défini\)  
  
 Le contenu d'une section générée par le compilateur dans l'objet donné semble endommagé.  
  
 Cette erreur peut représenter une défaillance du compilateur.
