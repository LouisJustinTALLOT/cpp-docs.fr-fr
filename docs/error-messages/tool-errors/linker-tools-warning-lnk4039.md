# Avertissement des outils &#201;diteur de liens LNK4039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la section 'nom' spécifiée avec l'option \/SECTION n'existe pas  
  
 [DUMPBIN](../../build/reference/dumpbin-reference.md) ou [EDITBIN](../../build/reference/editbin-reference.md) n'a pas pu trouver une section `name` dans le fichier d'entrée.  
  
 Pour voir les sections d'un fichier, exécutez DUMPBIN avec l'option [\/HEADERS](../../build/reference/headers.md).
