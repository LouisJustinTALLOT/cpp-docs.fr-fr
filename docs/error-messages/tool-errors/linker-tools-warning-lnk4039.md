# <a name="linker-tools-warning-lnk4039"></a>Avertissement des outils Éditeur de liens LNK4039

la section’name’spécifiée avec l’option/SECTION n’existe pas

[DUMPBIN](../../build/reference/dumpbin-reference.md) ou [EDITBIN](../../build/reference/editbin-reference.md) n’a pas pu trouver une section appelée `name` dans le fichier d’entrée.

Pour afficher les sections d’un fichier, exécutez DUMPBIN avec l’option [/headers](../../build/reference/headers.md) .
