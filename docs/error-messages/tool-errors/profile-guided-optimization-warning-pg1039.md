# <a name="profile-guided-optimization-warning-pg1039"></a>Avertissement de l'optimisation guidée par profil PG1039

Échec de l’attente une période d’inactivité balayer.

L’application n’a pas permis [pgosweep](../../build/reference/pgosweep.md) à exécuter.  Si possible, exécutez pgosweep lorsque l’application est dans un état inactif.
