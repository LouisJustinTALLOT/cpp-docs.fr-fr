# <a name="profile-guided-optimization-warning-pg1039"></a>Avertissement de l'optimisation guidée par profil PG1039

Échec de l’attente du balayage du temps silencieux.

L’application n’a pas autorisé l’exécution de [pgosweep](../../build/pgosweep.md) .  Si possible, exécutez pgosweep lorsque l’application est dans un état inactif.
