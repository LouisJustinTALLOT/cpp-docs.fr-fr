# Avertissement de l&#39;optimisation guid&#233;e par profil PG1039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Échec de l'attente d'une période d'inactivité pour procéder à un balayage.  
  
 L'application n'a pas permis l'exécution de [pgosweep](../../build/reference/pgosweep.md).  Dans la mesure du possible, exécutez pgosweep lorsque l'application est dans un état d'inactivité.
