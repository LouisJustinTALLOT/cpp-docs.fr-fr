# Erreur RC2169 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

le fichier de ressources nomfichier n'est pas au format 2.03  
  
 La ressource spécifiée a été créée dans une version antérieure à la version 2.03.  Vous devez la convertir ou la recréer en utilisant le format de la version 3.00 ou ultérieure.
