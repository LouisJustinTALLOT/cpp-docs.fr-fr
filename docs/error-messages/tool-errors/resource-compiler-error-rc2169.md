# <a name="resource-compiler-error-rc2169"></a>Erreur RC2169 du compilateur de ressources 
fichier de ressources nomfichier n’est pas au format 2.03  
  
 La ressource spécifiée utilise un format de version antérieure à la version 2.03. Le fichier de ressources doit être converti ou recréé en utilisant le format de version 3.00 ou ultérieure.
