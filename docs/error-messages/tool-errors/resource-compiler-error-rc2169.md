# <a name="resource-compiler-error-rc2169"></a>Erreur RC2169 du compilateur de ressources 

le nom du fichier de ressources n’est pas au format 2,03

La ressource spécifiée a utilisé un format antérieur à la version 2,03. Le fichier de ressources doit être converti ou recréé à l’aide du format pour la version 3,00 ou ultérieure.
