# <a name="resource-compiler-error-rc2169"></a>Erreur RC2169 du compilateur de ressources 

nom du fichier du fichier de ressources n’est pas au format 2.03

La ressource spécifiée utilisait un format antérieures à la version 2.03. Le fichier de ressources doit être converti ou recréé en utilisant le format de la version 3.00 ou ultérieure.
