# <a name="linker-tools-error-lnk1215"></a>Erreur des outils Éditeur de liens LNK1215

échec de l’opération de métadonnées (HRESULT) : erreur

L’éditeur de liens a reçu une erreur du Runtime .NET lors de la tentative de mise à jour des métadonnées par le biais du Runtime .NET.

`HRESULT` est le HRESULT de la méthode Runtime .NET. `error` est le. Texte fourni par le réseau.

Vous avez probablement un éditeur de liens incompatible et un Runtime .NET. Réinstallez C++Visual.
