# Erreur des outils &#201;diteur de liens LNK1215
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

échec de l'opération sur les métadonnées \(HRESULT\) : erreur  
  
 L'éditeur de liens a reçu une erreur du runtime .NET lors d'une tentative de mise à jour de métadonnées via le runtime .NET.  
  
 `HRESULT` est le HRESULT de la méthode du runtime .NET.  `error` est le texte fourni par .NET.  
  
 L'éditeur de liens ne correspond sans doute pas au runtime .NET ; réinstallez Visual C\+\+.
