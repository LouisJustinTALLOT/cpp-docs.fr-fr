# &#201;valuateur d&#39;expression, erreur CXX0048
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la séquence d'appel n'est pas prise en charge  
  
 Une fonction spécifiée dans l'expression utilise une séquence d'appel qui n'est pas prise en charge par l'évaluateur d'expression C.  Vous ne pouvez pas appeler cette fonction dans une expression de la fenêtre Espion.  
  
 Erreur identique à CAN0048.
