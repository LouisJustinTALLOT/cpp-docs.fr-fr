# <a name="expression-evaluator-error-cxx0048"></a>Évaluateur d'expression, erreur CXX0048

séquence d’appel non pris en charge.

Une fonction spécifiée dans l’expression utilise une séquence d’appel non pris en charge par l’évaluateur d’expression de C. Vous ne pouvez pas appeler cette fonction dans une expression de la fenêtre Espion.

Cette erreur est identique à CAN0048.
