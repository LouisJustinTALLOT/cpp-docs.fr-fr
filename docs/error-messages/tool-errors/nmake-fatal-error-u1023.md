# <a name="nmake-fatal-error-u1023"></a>Erreur irrécupérable NMAKE U1023

Erreur de syntaxe dans l’expression

Une expression n’est pas valide.

Vérifiez les opérateurs autorisés et la priorité des opérateurs.
