# <a name="linker-tools-warning-lnk4200"></a>Avertissement des outils Éditeur de liens LNK4200
informations de numéro de ligne endommagées dans le fichier objet ; ignoré  
  
 Les informations de numéro de ligne dans le fichier objet sont endommagées. Régénérez.
