# <a name="resource-compiler-fatal-error-rc1018"></a>Erreur irrécupérable RC1018 du compilateur de ressources 

' #elif’inattendu

La `#elif` directive n’apparaît pas dans une `#if` construction, **#ifdef** ou **#ifndef** .

Assurez-vous qu’il existe une `#if` instruction, **#ifdef** ou **#ifndef** en vigueur avant cette instruction.
