# Erreur des outils &#201;diteur de liens LNK1136
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

fichier non valide ou endommagé  
  
 Le fichier d'entrée a un en\-tête endommagé, une taille zéro ou anormalement petite.  
  
 L'erreur LNK1136 peut également être provoquée si le fichier n'a pas été spécifié correctement en tant que dépendance supplémentaire.
