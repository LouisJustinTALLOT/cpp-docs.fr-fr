# <a name="linker-tools-error-lnk1136"></a>Erreur des outils Éditeur de liens LNK1136

fichier non valide ou endommagé

Le fichier d’entrée a un en-tête endommagé ou est de taille zéro ou anormalement petite.

LNK1136 peut également se produire si le fichier a été incorrectement spécifié en tant que dépendance supplémentaire.
