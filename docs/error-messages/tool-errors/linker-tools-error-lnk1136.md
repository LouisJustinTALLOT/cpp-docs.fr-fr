# <a name="linker-tools-error-lnk1136"></a>Erreur des outils Éditeur de liens LNK1136

fichier non valide ou endommagé

Le fichier d’entrée contient un en-tête endommagé ou sa taille est égale à zéro ou est anormalement faible.

LNK1136 peut également être provoquée si le fichier a été spécifié de manière incorrecte en tant que dépendance supplémentaire.
