# <a name="expression-evaluator-error-cxx0072"></a>Évaluateur d'expression, erreur CXX0072

Erreur : type d’informations manquantes ou inconnues

Le fichier .pch ne pas été lié ou le code a une référence à un type qui se trouve dans un module non compilé avec/Zi.

Cette erreur est identique à CAN0067.
