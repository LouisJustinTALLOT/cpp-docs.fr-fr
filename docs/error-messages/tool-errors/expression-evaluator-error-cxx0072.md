# &#201;valuateur d&#39;expression, erreur CXX0072
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erreur : informations de type manquantes ou inconnues  
  
 Le fichier .pch n'a pas été lié ou le code comprend une référence à un type situé dans un module non compilé avec \/Zi.  
  
 Erreur identique à CAN0067.
