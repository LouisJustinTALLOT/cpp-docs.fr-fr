# <a name="project-build-error-prj0017"></a>Erreur de génération de projet PRJ0017

Le répertoire de travail actuel n’est pas valide.

Le chemin d’accès au répertoire de travail actuel est plus long que _MAXPATH. Pour résoudre cette erreur, ne créez pas votre projet à un niveau de ce type.
