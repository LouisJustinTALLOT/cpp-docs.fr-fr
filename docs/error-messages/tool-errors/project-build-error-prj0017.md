# Erreur de g&#233;n&#233;ration de projet PRJ0017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Le répertoire de travail en cours n'est pas valide.  
  
 Le chemin d'accès du répertoire de travail en cours ne se limite pas à \_MAXPATH.  Pour corriger cette erreur, évitez de créer votre projet à un tel niveau d'imbrication.
