# <a name="project-build-error-prj0017"></a>Erreur de génération de projet PRJ0017
Le répertoire de travail en cours n’est pas valide.  
  
 Le chemin d’accès au répertoire de travail actuel est supérieure à _MAXPATH. Pour résoudre cette erreur, ne créez pas de votre projet à un niveau approfondi.
