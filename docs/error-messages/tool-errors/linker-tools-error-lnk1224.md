# <a name="linker-tools-error-lnk1224"></a>Erreur des outils Éditeur de liens LNK1224

adresse de base d’image non valide

Vous avez spécifié une adresse de base non valide pour l’image. Les adresses de base doivent être alignées en 64 Ko (les quatre derniers chiffres hexadécimaux doivent être zéro) et la base de l’image doit tenir dans une valeur signée ou non signée 32 bits.
