# Erreur des outils &#201;diteur de liens LNK1224
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

adresse de base d'image non valide  
  
 Vous avez spécifié une adresse de base non valide pour l'image.  Les adresses de base doivent être alignées sur 64 Ko \(les quatre derniers chiffres hexadécimaux doivent être des zéros\) et la base d'image doit pouvoir être contenue dans une valeur 32 bits signée ou non signée.
