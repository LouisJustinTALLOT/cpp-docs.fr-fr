# <a name="resource-compiler-fatal-error-rc1105"></a>Erreur irrécupérable RC1105 du compilateur de ressources 

commutateur non valide, option : trop de commutateurs /d

Trop de symboles ont été définis à l’aide de Define Symbols (/ d) option. Définir des symboles dans votre source et la recompilation.
