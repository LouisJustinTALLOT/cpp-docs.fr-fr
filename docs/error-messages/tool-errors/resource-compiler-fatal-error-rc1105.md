# Erreur irr&#233;cup&#233;rable RC1105 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

commutateur non valide, option : trop de commutateurs \/d  
  
 Vous avez spécifié un trop grand nombre de symboles à l'aide de l'option Define Symbols \(\/d\).  Définissez quelques symboles dans votre source puis relancez la compilation.
