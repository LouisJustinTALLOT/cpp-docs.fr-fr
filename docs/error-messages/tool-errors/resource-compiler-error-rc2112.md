# <a name="resource-compiler-error-rc2112"></a>Erreur RC2112 du compilateur de ressources 
BEGIN attendu dans la boîte de dialogue  
  
 Le **commencer** mot clé doit suivre immédiatement le **boîte de dialogue** (mot clé).
