# Erreur math&#233;matique M6205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'fonction' : \_TLOSS error  
  
 Une perte complète de signification \(précision\) est survenue.  
  
 Cette erreur est peut\-être due à l'indication d'un très grand nombre comme opérande de sin, cos ou tan ; l'opérande doit être réduit à un nombre compris entre 0 et 2\*pi.
