# <a name="math-error-m6205"></a>Erreur mathématique M6205

'fonction' : erreur _TLOSS

Une perte totale de précision s’est produite.

Cette erreur peut être causée par l’attribution d’un très grand nombre d’opérandes de Sin, cos ou Tan, car l’opérande doit être réduit à un nombre compris entre 0 et 2 * pi.
