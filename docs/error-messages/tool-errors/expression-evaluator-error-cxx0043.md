# &#201;valuateur d&#39;expression, erreur CXX0043
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ce pointeur a été utilisé en dehors d'une fonction membre  
  
 Le pointeur **this** ne peut être utilisé que pour des fonctions membres non statiques.  
  
 Erreur identique à CAN0043.
