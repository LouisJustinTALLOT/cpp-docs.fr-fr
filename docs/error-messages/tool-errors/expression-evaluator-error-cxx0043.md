# <a name="expression-evaluator-error-cxx0043"></a>Évaluateur d'expression, erreur CXX0043

pointeur this utilisé en dehors de la fonction membre

Le **`this`** pointeur ne peut être utilisé que pour les fonctions membres non statiques.

Cette erreur est identique à CAN0043.
