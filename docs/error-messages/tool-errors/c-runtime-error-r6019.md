# Erreur d&#39;ex&#233;cution C R6019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossible d'ouvrir le périphérique de la console  
  
 Le programme a appelé une fonction de console déclarée dans CONIO.h, mais le système d'exploitation a refusé l'accès à cette console.
