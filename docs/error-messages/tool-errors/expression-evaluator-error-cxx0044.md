# &#201;valuateur d&#39;expression, erreur CXX0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'utilisation d'un pointeur \_based\(void\) requiert l'opérateur :\>  
  
 Un pointeur `void` ne peut pas être utilisé directement.  Vous devez constituer un pointeur complet à l'aide de l'opérateur **:\>**.  
  
 Erreur identique à CAN0044.
