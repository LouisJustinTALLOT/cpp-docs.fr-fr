# <a name="expression-evaluator-error-cxx0034"></a>Évaluateur d'expression, erreur CXX0034

**types incompatibles avec l’opérateur**

Les types d’opérandes spécifiés ne sont pas valides pour l’opération.

Par exemple, un pointeur ne peut pas être multiplié par une valeur.

Vous devrez peut-être taper le cast des opérandes en un type compatible avec l’opérateur.
