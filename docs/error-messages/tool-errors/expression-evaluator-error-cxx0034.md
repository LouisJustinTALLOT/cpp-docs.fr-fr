# &#201;valuateur d&#39;expression, erreur CXX0034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**types incompatibles avec l'opérateur**  
  
 Les types d'opérandes spécifiés ne sont pas autorisés pour l'opération.  
  
 Par exemple, un pointeur ne peut pas être multiplié par une valeur quelconque.  
  
 Il se peut que vous deviez convertir les opérandes dans un type compatible avec l'opérateur.
