# Erreur des outils &#201;diteur de liens LNK1152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossible de résoudre un ou plusieurs symboles non décorés  
  
 Cette erreur est précédée par un avertissement [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) pour chaque symbole non décoré qui n'a pas pu être résolu et par au moins deux avertissements [LNK4002](../../error-messages/tool-errors/linker-tools-warning-lnk4002.md) pour les symboles dupliqués trouvés pour le symbole non décoré.
