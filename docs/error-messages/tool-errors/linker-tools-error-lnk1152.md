# <a name="linker-tools-error-lnk1152"></a>Erreur des outils Éditeur de liens LNK1152

Impossible de résoudre un ou plusieurs symboles non décorés

Cette erreur est précédée d’un avertissement [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) pour chaque symbole non décoré qui n’a pas pu être résolu et par au moins deux avertissements [LNK4002](../../error-messages/tool-errors/linker-tools-warning-lnk4002.md) pour les symboles en double trouvés pour le symbole non décoré.
