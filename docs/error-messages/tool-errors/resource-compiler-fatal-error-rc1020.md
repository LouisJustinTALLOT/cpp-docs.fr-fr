# <a name="resource-compiler-fatal-error-rc1020"></a>Erreur irrécupérable RC1020 du compilateur de ressources 

' #endif’inattendu

Une `#endif` directive est apparue sans `#if` directive correspondante, **#ifdef** ou **#ifndef** .

Assurez-vous qu’il existe un correspondant `#endif` pour chaque `#if` instruction, **#ifdef** et **#ifndef** .
