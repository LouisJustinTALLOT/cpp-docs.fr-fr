# <a name="project-build-error-prj0040"></a>Erreur de génération de projet PRJ0040
Erreur interne lors de la build. Impossible de continuer. Rechargez le projet et réessayez.  
  
 Le système de projet n’a pas pu traiter votre projet.  
  
 Essayez de recharger le projet.
