# <a name="resource-compiler-fatal-error-rw1022"></a>Erreur irrécupérable RW1022 du compilateur de ressources 
**Fichier de l’écriture des erreurs d’e/s**  
  
 Le compilateur de ressources n’a pas pu écrire dans un fichier.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Pour corriger en vérifiant les causes possibles suivantes  
  
1.  Espace disque insuffisant. Espace libre doit être égal au moins deux fois la taille du fichier exécutable que vous créez.  
  
2.  Volume en lecture seule.  
  
3.  Secteur défectueux.  
  
4.  Violation de partage.
