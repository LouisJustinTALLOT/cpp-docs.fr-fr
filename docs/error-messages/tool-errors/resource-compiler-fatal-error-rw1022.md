# Erreur irr&#233;cup&#233;rable RW1022 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Erreur d'E\/S lors de l'écriture du fichier**  
  
 Le compilateur de ressources n'a pas pu écrire dans le fichier.  
  
### Pour résoudre le problème en vérifiant les causes possibles suivantes  
  
1.  Espace disque insuffisant.  L'espace disponible doit être au moins égal à deux fois la taille du fichier exécutable en cours de création.  
  
2.  Le volume est en lecture seule.  
  
3.  Secteur défectueux.  
  
4.  Violation de partage.
