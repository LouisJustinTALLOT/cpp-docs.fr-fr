# &#201;valuateur d&#39;expression, erreur CXX0038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

le nombre et\/ou le type d'argument\(s\) transmis à la fonction sont incompatibles  
  
 L'appel de fonction spécifié ne correspond pas au prototype de la fonction.  
  
 Entrez à nouveau l'appel de fonction avec un nombre correct d'arguments.  Si nécessaire, convertissez chaque argument dans le type correspondant au prototype.  
  
 Erreur identique à CAN0038.
