# <a name="resource-compiler-fatal-error-rw1023"></a>Erreur irrécupérable RW1023 du compilateur de ressources 

Erreur d’e/s écriture du fichier, lecteur plein

Espace libre doit être égal au moins deux fois la taille du fichier exécutable que vous créez.
