# <a name="resource-compiler-error-rc2175"></a>Erreur RC2175 du compilateur de ressources 
fichier de ressources nomfichier n’est pas au format 3.00  
  
 La ressource spécifiée utilise un format de version antérieure à la version 3.00. Le fichier de ressources doit être converti ou recréé en utilisant le format de version 3.00 ou ultérieure.
