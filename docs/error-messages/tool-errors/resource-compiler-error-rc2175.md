# <a name="resource-compiler-error-rc2175"></a>Erreur RC2175 du compilateur de ressources 

nom du fichier du fichier de ressources n’est pas au format 3.00

La ressource spécifiée utilisait un format antérieures à la version 3.00. Le fichier de ressources doit être converti ou recréé en utilisant le format de la version 3.00 ou ultérieure.
