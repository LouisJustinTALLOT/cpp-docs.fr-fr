# <a name="resource-compiler-error-rc2175"></a>Erreur RC2175 du compilateur de ressources 

le nom du fichier de ressources n’est pas au format 3,00

La ressource spécifiée a utilisé un format antérieur à la version 3,00. Le fichier de ressources doit être converti ou recréé à l’aide du format pour la version 3,00 ou ultérieure.
