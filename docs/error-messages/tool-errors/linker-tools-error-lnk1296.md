# Erreur des outils &#201;diteur de liens LNK1296
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossible de charger 'NomFichier'  
  
 La DLL donnée n'était pas disponible.  
  
 Les emplacements possibles pour le fichier DLL sont le répertoire actif, le répertoire système, le répertoire Windows et les répertoires spécifiés dans la variable d'environnement PATH.
