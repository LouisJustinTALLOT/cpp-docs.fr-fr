# Erreur irr&#233;cup&#233;rable NMAKE U1007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

guillemets doubles non autorisés dans le nom  
  
 Le nom de fichier ou le nom de cible spécifié contient des guillemets doubles \(**"**\).  
  
 Les guillemets doubles peuvent délimiter un nom de fichier mais ne peuvent pas en faire partie.
