# <a name="nmake-fatal-error-u1007"></a>Erreur irrécupérable NMAKE U1007

guillemet double non autorisé dans le nom

Le nom ou le nom de fichier cible spécifié contenait un guillemet double (**"**).

Les guillemets doubles peuvent entourer un nom de fichier, mais ne peuvent pas y figurer.
