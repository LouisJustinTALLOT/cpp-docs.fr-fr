# <a name="nmake-fatal-error-u1007"></a>Erreur irrécupérable NMAKE U1007
guillemet double non autorisé dans le nom  
  
 Le fichier ou le nom de la cible spécifiée contient un guillemet double (**»**).  
  
 Des guillemets doubles pouvez entourer un nom de fichier, mais ne peuvent pas être contenues dans celui-ci.
