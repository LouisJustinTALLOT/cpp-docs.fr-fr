# Erreur de g&#233;n&#233;ration de projet PRJ0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossible de reprendre le processus suspendu.  La génération a échoué.  
  
 Une erreur s'est produite dans l'environnement de développement.  
  
 Pour résoudre le problème, fermez et rechargez le projet.  Au besoin, quittez Visual Studio et redémarrez\-le.
