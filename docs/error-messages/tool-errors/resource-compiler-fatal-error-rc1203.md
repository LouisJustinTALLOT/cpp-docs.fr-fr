# <a name="resource-compiler-fatal-error-rc1203"></a>Erreur irrécupérable RC1203 du compilateur de ressources 

l’ID de langue hexadécimal par défaut spécifié n’est pas valide.

L’option de spécification de la langue par défaut (/l) a été suivie d’un ID de langue hexadécimal non valide. Consultez [chaînes de langue et de pays/région](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) dans la référence de la *bibliothèque Runtime* pour obtenir la liste des ID de langue valides.
