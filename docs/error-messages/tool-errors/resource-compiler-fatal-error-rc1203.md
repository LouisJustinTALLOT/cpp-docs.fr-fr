# Erreur irr&#233;cup&#233;rable RC1203 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ID de langue hexadécimal spécifié par défaut non valide.  
  
 L'option de spécification de la langue par défaut \(\/l\) est suivie d'un ID de langue hexadécimal non valide.  Consultez [Chaînes associées à la langue, au pays et à la région \(page éventuellement en anglais\)](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) dans le document *Référence de la bibliothèque Runtime \(éventuellement en anglais\)* pour obtenir une liste d'ID de langue valides.
