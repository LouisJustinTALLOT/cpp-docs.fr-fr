# <a name="resource-compiler-warning-rw4001"></a>Avertissement RW4001 du compilateur de ressources

. Options de traitement EXE (/L/M/P/T/K/E/31 ou/30)

Vous avez spécifié des options de traitement EXE, mais il n’existait aucun fichier exécutable à traiter. N’utilisez pas ces options avec un fichier. res.
