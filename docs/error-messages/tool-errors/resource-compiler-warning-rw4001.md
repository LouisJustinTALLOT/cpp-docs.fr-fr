# Avertissement RW4001 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Options de traitement .EXE \(\/L \/M \/P \/T \/K \/E \/31 ou \/30\)  
  
 Vous avez spécifié des options de traitement de fichier EXE mais aucun fichier exécutable n'est à traiter.  Vous ne devez pas utiliser ces options avec un fichier .res.
