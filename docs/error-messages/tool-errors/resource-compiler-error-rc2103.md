# Erreur RC2103 du compilateur de ressources 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

fin de fichier inattendue dans le littéral de chaîne  
  
 Fin de fichier trouvée avant la fin d'une chaîne.  Il manque probablement des guillemets fermants \(**"**\) dans celle\-ci.
