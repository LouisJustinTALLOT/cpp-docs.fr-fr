# Avertissement de ligne de commande D9040
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

option '\/analyze' ignorée ; les avertissements liés à l'analyse du code ne sont pas disponibles dans cette édition du compilateur  
  
 L'option de ligne de commande **\/analyze** n'est pas disponible dans toutes les éditions de Visual Studio.  Pour remédier à cet avertissement, basculez vers la version maintenu de Visual Studio ou supprimez l'option de ligne de commande.
