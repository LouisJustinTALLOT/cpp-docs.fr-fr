# <a name="command-line-warning-d9040"></a>Avertissement de ligne de commande D9040

> option'/Analyze’ignorée ; Les avertissements liés à l’analyse du code ne sont pas disponibles dans cette édition du compilateur

L' **`/analyze`** option de ligne de commande n’est pas disponible dans toutes les éditions de Visual Studio. Pour remédier à cet avertissement, vous pouvez basculer vers une édition prise en charge de Visual Studio ou supprimer l’option de ligne de commande.
