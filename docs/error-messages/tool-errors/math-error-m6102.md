# Erreur math&#233;matique M6102
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

« denormal »  
  
 Une opération a généré un très petit nombre à virgule flottante, qui pourrait être non valide à la suite de la perte de toute signification.  Les exceptions de virgule flottante « Denormal » sont généralement masquées, d'où leur interception et manipulation.  
  
 Le programme se termine par le code de sortie 130.
