# <a name="math-error-m6102"></a>Erreur mathématique M6102
Denormal  
  
 Une opération a généré un très petit nombre à virgule flottante, qui n’est pas valide en raison d’une perte de précision. Denormal exceptions à virgule flottante sont généralement masquées, d'où leur être interceptées et manipulé.  
  
 Programme se termine par le code de sortie 130.
