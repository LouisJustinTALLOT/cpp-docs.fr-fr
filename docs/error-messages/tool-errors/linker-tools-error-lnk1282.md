# Erreur des outils &#201;diteur de liens LNK1282
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/REBASE fichier impossible ; il a été signé  
  
 Vous tentez de changer l'adresse de base d'un assembly signé avec l'option \/REBASE pour [editbin](../../build/reference/editbin-reference.md).  Pour cela, changez d'abord l'adresse de base puis signez l'assembly.
