# Erreur d&#39;ex&#233;cution C R6024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

espace insuffisant pour la table \_onexit\/atexit  
  
 Il n'y avait pas de mémoire disponible pour la fonction `_onexit` ou `atexit`.  Cette erreur s'explique par des ressources mémoire insuffisantes.
