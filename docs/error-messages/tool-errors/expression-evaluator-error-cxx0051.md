# <a name="expression-evaluator-error-cxx0051"></a>Évaluateur d'expression, erreur CXX0051

plusieurs symboles surchargés spécifiés dans un point d’arrêt

L’évaluateur d’expression n’a pas pu déterminer lequel des symboles surchargés à utiliser comme point d’arrêt.

Cette erreur est identique à CAN0051.
