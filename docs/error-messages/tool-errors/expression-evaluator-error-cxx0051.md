# <a name="expression-evaluator-error-cxx0051"></a>Évaluateur d'expression, erreur CXX0051

plus d’un symbole surchargé spécifié au point d’arrêt

L’évaluateur d’expression n’a pas pu déterminer lequel de plus d’un symbole surchargé à utiliser comme un point d’arrêt.

Cette erreur est identique à CAN0051.
