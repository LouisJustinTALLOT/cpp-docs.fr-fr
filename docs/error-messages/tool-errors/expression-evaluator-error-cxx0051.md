# &#201;valuateur d&#39;expression, erreur CXX0051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

plusieurs symboles surchargés sont spécifiés au point d'arrêt  
  
 L'évaluateur d'expression n'a pas pu déterminer lequel de ces symboles surchargés il devait utiliser comme point d'arrêt.  
  
 Erreur identique à CAN0051.
