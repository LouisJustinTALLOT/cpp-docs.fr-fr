# Erreur de ligne de commande D8036
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\/option' non autorisé avec plusieurs fichiers source  
  
 Ces options de compilateur ne peuvent pas être utilisées avec plusieurs fichiers source :  
  
-   Nommez le listing des fichiers d'assembly \(\/Fa\)  
  
-   Renommez le fichier objet \(\/Fo\)  
  
-   Créez les informations du navigateur source sans variables locales \(\/Fr\)  
  
-   Créez les informations du navigateur source avec les variables locales \(\/FR\)
