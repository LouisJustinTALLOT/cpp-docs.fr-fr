# <a name="project-build-error-prj0046"></a>Erreur de génération de projet PRJ0046

N’a pas pu générer la ligne de commande, car celle spécifiée était vide.

Une ligne de commande vide a été spécifiée pour une configuration de makefile ; la ligne de commande est nécessaire.
