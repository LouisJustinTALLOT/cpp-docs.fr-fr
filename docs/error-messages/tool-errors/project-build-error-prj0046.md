# <a name="project-build-error-prj0046"></a>Erreur de génération de projet PRJ0046

Impossible de générer la ligne de commande, car l’une des spécifiée était vide.

Une ligne de commande vide a été spécifiée pour une configuration de Makefile ; la ligne de commande est requise.
