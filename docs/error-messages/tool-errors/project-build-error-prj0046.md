# Erreur de g&#233;n&#233;ration de projet PRJ0046
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossible de générer dynamiquement la ligne de commande spécifiée, car elle était vide.  
  
 Une ligne de commande vide a été spécifiée pour une configuration de makefile ; la ligne de commande est requise.
