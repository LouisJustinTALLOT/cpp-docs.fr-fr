# Erreur irr&#233;cup&#233;rable NMAKE U1055
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

espace d'environnement insuffisant  
  
 Le système d'exploitation ne dispose plus d'un espace suffisant pour les variables d'environnement.  
  
 Augmentez l'espace d'environnement ou réduisez le nombre de variables d'environnement définies.
