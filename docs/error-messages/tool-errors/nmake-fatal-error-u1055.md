# <a name="nmake-fatal-error-u1055"></a>Erreur irrécupérable NMAKE U1055

espace d’environnement insuffisant

Le système d’exploitation manque de place pour les variables d’environnement.

Augmentez l’espace de l’environnement ou définissez moins de variables d’environnement.
