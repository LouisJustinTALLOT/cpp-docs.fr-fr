# <a name="nmake-fatal-error-u1055"></a>Erreur irrécupérable NMAKE U1055
espace d’environnement insuffisant  
  
 Le système d’exploitation a manqué de place pour les variables d’environnement.  
  
 Augmentez l’espace d’environnement ou définir des variables d’environnement moins.
