# <a name="expression-evaluator-error-cxx0049"></a>Évaluateur d'expression, erreur CXX0049

OMF obsolète-veuillez réassocier le programme

Le programme a utilisé un ancien format OMF (format de module objet).

Reliez le programme à l’aide de la version actuelle de l’éditeur de liens.

Cette erreur est identique à CAN0049.
