# &#201;valuateur d&#39;expression, erreur CXX0049
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

OMF obsolète \- rééditez les liens du programme  
  
 Le programme utilisait un ancien format OMF \(Object Module Format\).  
  
 Rééditez le programme en utilisant la version en cours de l'éditeur de liens.  
  
 Erreur identique à CAN0049.
