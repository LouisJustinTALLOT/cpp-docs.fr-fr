# <a name="ltcwchargt"></a>&lt;cwchar&gt;

Inclut l’en-tête de la bibliothèque C Standard \<uchar.h > et ajoute les noms associés à la `std` espace de noms.

## <a name="requirements"></a>Configuration requise

**En-tête :** \<cuchar >

**Espace de noms :** std

|[c32rtomb](../standard-library/cuchar-functions.md#c32rtomb)|Aucune description.|
|[mbrtoc16](../standard-library/cuchar-functions.md#mbrtoc16)|Aucune description.|
|[mbrtoc32](../standard-library/cuchar-functions.md#mbrtoc32)|Aucune description.|

## <a name="remarks"></a>Notes

L'inclusion de cet en-tête garantit également que les noms déclarés à l'aide d'une liaison externe dans l'en-tête de la bibliothèque C standard soient déclarés dans l'espace de noms `std`.
