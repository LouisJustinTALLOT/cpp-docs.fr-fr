# <a name="ltcuchargt"></a>&lt;cuchar&gt;

Inclut l’en-tête \<UCHAR. > h de la bibliothèque standard C et ajoute les noms associés à l' `std` espace de noms.

## <a name="requirements"></a>Configuration requise

**En-tête:** \<cuchar >

**Espace de noms :** std

## <a name="remarks"></a>Notes

Cet en- `mbstate_t` tête ajoute les types de bibliothèque standard C et `size_t`, ainsi que les fonctions [c16rtomb](../c-runtime-library/reference/c16rtomb-c32rtomb1.md), [c32rtomb](../c-runtime-library/reference/c16rtomb-c32rtomb1.md), [mbrtoc16](../c-runtime-library/reference/mbrtoc16-mbrtoc323.md)et `std` [mbrtoc32](../c-runtime-library/reference/mbrtoc16-mbrtoc323.md) à l’espace de noms.
