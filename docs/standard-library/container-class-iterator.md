# <a name="container-classiterator"></a>Conteneur Class::iterator

> [!NOTE]
> Cette rubrique se trouve dans la documentation de Microsoft C++ comme un exemple non fonctionnel de conteneurs utilisés dans la bibliothèque standard C++. Pour plus d’informations, consultez [Conteneurs disponibles dans la bibliothèque standard C++](../standard-library/stl-containers.md).

Décrit un objet pouvant servir d’itérateur pour la séquence contrôlée.

## <a name="syntax"></a>Syntaxe

```cpp
typedef T5 iterator;
```

## <a name="remarks"></a>Notes

Il est décrit ici comme un synonyme du type non spécifié `T5` . Un objet de type `iterator` peut être casté en un objet de type [const_iterator](../standard-library/container-class-const-iterator.md).
