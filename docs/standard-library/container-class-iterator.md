# Conteneur Class::iterator
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  Cette rubrique est dans la documentation Visual C\+\+ comme exemple fonctionnel de conteneurs utilisés dans la bibliothèque C\+\+ standard.  Pour plus d'informations, consultez l'[Conteneurs STL de](../standard-library/stl-containers.md).  
  
 Décrit un objet pouvant servir d'itérateur situé à la séquence contrôlée.  
  
## Syntaxe  
  
```  
  
typedef T5 iterator;  
  
```  
  
## Notes  
 Il est décrit ici comme synonyme du type non spécifié **T5**.  Objet de type **iterator** peut être converti en un objet de type [const\_iterator](../standard-library/container-class-const-iterator.md).
