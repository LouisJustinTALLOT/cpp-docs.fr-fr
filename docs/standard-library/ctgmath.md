# <a name="ltctgmathgt"></a>&lt;ctgmath&gt;

Inclut les en-têtes \<ccomplex> et \<cmath> de la bibliothèque C++ Standard, qui fournissent des macros mathématiques de type générique équivalentes à \<tgmath.h>.

## <a name="syntax"></a>Syntaxe

```cpp
#include <ctgmath>
```

## <a name="remarks"></a>Notes

La fonctionnalité de l’en-tête \<tgmath.h> de la bibliothèque C Standard est fournie par des surcharges dans \<ccomplex> et \<cmath>.

L'inclusion de cet en-tête garantit également que les noms déclarés à l'aide d'une liaison externe dans l'en-tête de la bibliothèque C standard soient déclarés dans l'espace de noms `std`.
