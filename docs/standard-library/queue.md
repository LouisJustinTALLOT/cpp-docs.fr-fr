# &lt; file &gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Définit les classes de modèle priority_queue et queue et plusieurs modèles de prise en charge.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
#include <queue>  
  
```  
  
### <a name="operators"></a>Opérateurs  
  
|||  
|-|-|  
|[opérateur ! =](../Topic/%3Cqueue%3E%20operators.md#operator_neq)|Teste si l'objet de file d'attente situé à gauche de l'opérateur n'est pas égal à l'objet de file d'attente situé à droite.|  
|[opérateur <](../Topic/%3Cqueue%3E%20operators.md#operator_lt_)|Teste si l'objet de file d'attente situé à gauche de l'opérateur est inférieur à l'objet de file d'attente situé à droite.|  
|[opérateur \< =](../Topic/%3Cqueue%3E%20operators.md#operator_lt__eq)|Teste si l'objet de file d'attente situé à gauche de l'opérateur est inférieur ou égal à l'objet de file d'attente situé à droite.|  
|[opérateur ==](../Topic/%3Cqueue%3E%20operators.md#operator_eq_eq)|Teste si l'objet de file d'attente situé à gauche de l'opérateur est égal à l'objet de file d'attente situé à droite.|  
|[opérateur >](../Topic/%3Cqueue%3E%20operators.md#operator_gt_)|Teste si l'objet de file d'attente situé à gauche de l'opérateur est supérieur à l'objet de file d'attente situé à droite.|  
|[opérateur > =](../Topic/%3Cqueue%3E%20operators.md#operator_gt__eq)|Teste si l'objet de file d'attente situé à gauche de l'opérateur est supérieur ou égal à l'objet de file d'attente situé à droite.|  
  
### <a name="classes"></a>Classes  
  
|||  
|-|-|  
|[file d’attente (classe)](../standard-library/queue-class.md)|Classe d'adaptateur de conteneur de modèle qui fournit une restriction de fonctionnalité limitant l'accès aux premier et dernier éléments d'un certain type de conteneur sous-jacent.|  
|[priority_queue, classe](../standard-library/priority-queue-class.md)|Classe d'adaptateur de conteneur de modèle qui fournit une restriction de fonctionnalité limitant l'accès à l'élément supérieur d'un certain type de conteneur sous-jacent, qui est toujours le plus grand.|
