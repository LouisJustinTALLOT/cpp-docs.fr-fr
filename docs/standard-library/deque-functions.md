##  <a name="swap"></a>  swap  
 Échange les éléments de deux classes deque.  
  
```
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```  
  
### <a name="parameters"></a>Paramètres  
 `left`  
 Objet de type `deque`.  
  
 `right`  
 Objet de type `deque`.
