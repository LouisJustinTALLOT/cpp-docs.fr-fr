## <a name="swap"></a>  swap

Échange les éléments de deux classes deque.

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>Paramètres

*gauche* un objet de type `deque`.

*droit* un objet de type `deque`.
