# &lt;array&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Définit la classe de modèle de conteneur `array` et plusieurs modèles de prise en charge.  
  
## Syntaxe  
  
```  
#include <array>  
```  
  
### Classes  
  
|||  
|-|-|  
|[array](../standard-library/array-class-stl.md)|Enregistre une séquence de longueur fixe d'éléments.|  
|[tuple\_element](../standard-library/tuple-element-class-array.md)|Encapsule le type d'un élément de tableau.|  
|[tuple\_size](../standard-library/tuple-size-class-array.md)|Encapsule la taille d'un élément de tableau.|

|[.\<](../Topic/operator%3C%20%3Carray%3E.md)|Comparaison d'infériorité pour un tableau|  
|[operator\>\=](../Topic/operator%3E=%20%3Carray%3E.md)|Comparaison « supérieur ou égal » pour un tableau.|
