# R&#233;f&#233;rence de biblioth&#232;que standard C++
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Un programme C\+\+ peut faire appel à un grand nombre de fonctions depuis cette implémentation conforme de la bibliothèque C\+\+ standard.  Ces fonctions fournissent des services essentiels, tels que les entrées et sorties, ainsi que des implémentations efficaces des opérations fréquemment utilisées.  
  
 Pour plus d'informations sur les bibliothèques Runtime Visual C\+\+, voir [Fonctions de bibliothèque CRT](../c-runtime-library/crt-library-features.md).
