## <a name="istringstream"></a><a name="istringstream"></a>istringstream

Crée un `basic_istringstream` type spécialisé sur un paramètre de modèle **d’omble** chevalier.

```cpp
typedef basic_istringstream<char> istringstream;
```

### <a name="remarks"></a>Notes

Le type est un synonyme de modèle de classe [basic_istringstream](../standard-library/basic-istringstream-class.md), spécialisé pour les éléments de type **char**.
