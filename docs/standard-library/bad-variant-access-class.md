# <a name="badvariantaccess-class"></a>bad_variant_access, classe

Les objets de type bad_variant_access sont levés à accès non valide de rapport à la valeur d’un objet variant.

## <a name="syntax"></a>Syntaxe

```cpp
class bad_variant_access;
```
