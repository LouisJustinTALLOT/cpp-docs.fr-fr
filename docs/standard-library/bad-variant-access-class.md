# <a name="bad_variant_access-class"></a>Classe bad_variant_access

Les objets de type bad_variant_access sont levés pour signaler des accès non valides à la valeur d’un objet variant.

## <a name="syntax"></a>Syntaxe

```cpp
class bad_variant_access;
```
