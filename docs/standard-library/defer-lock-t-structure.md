# <a name="deferlockt-structure"></a>defer_lock_t, structure

Représente un type qui définit un objet [defer_lock](../standard-library/mutex-functions.md#defer_lock) permettant de sélectionner l’un des constructeurs surchargés de [unique_lock](../standard-library/unique-lock-class.md).

## <a name="syntax"></a>Syntaxe

```cpp
struct defer_lock_t;
```

## <a name="requirements"></a>Spécifications

**En-tête :** \<mutex >

**Espace de noms :** std
