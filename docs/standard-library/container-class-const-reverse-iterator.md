# Conteneur Class::const_reverse_iterator
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  Cette rubrique est dans la documentation Visual C\+\+ comme exemple fonctionnel de conteneurs utilisés dans la bibliothèque C\+\+ standard.  Pour plus d'informations, consultez l'[Conteneurs STL de](../standard-library/stl-containers.md).  
  
 Décrit un objet pouvant servir d'itérateur inverse constant à la séquence contrôlée.  
  
## Syntaxe  
  
```  
  
typedef T8 const_reverse_iterator;  
  
```  
  
## Notes  
 Il est décrit ici comme synonyme du type non spécifié **T8** \(généralement [reverse\_iterator](../standard-library/container-class-reverse-iterator.md) \<[const\_iterator](../standard-library/container-class-const-iterator.md)`>`\).
