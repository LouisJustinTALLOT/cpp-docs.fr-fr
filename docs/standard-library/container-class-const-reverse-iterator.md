# <a name="container-classconst_reverse_iterator"></a>Conteneur Class::const_reverse_iterator

> [!NOTE]
> Cette rubrique se trouve dans la C++ documentation de Microsoft comme un exemple non fonctionnel de conteneurs utilisés dans C++ la bibliothèque standard. Pour plus d’informations, consultez [Conteneurs de la bibliothèque standard C++](../standard-library/stl-containers.md).

Décrit un objet pouvant servir d’itérateur inverse constant pour la séquence contrôlée.

## <a name="syntax"></a>Syntaxe

```cpp
typedef T8 const_reverse_iterator;
```

## <a name="remarks"></a>Notes

Il est décrit ici comme un synonyme du type non spécifié `T8` (généralement [reverse_iterator](../standard-library/container-class-reverse-iterator.md) <[const_iterator](../standard-library/container-class-const-iterator.md)`>`).
