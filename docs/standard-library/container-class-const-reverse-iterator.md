# <a name="container-classconstreverseiterator"></a>Conteneur Class::const_reverse_iterator
> [!NOTE]
>  Cette rubrique fait partie de la documentation Visual C++ comme exemple non fonctionnel de conteneurs utilisés dans la bibliothèque C++ Standard. Pour plus d’informations, consultez [Conteneurs disponibles dans la bibliothèque C++ Standard](../standard-library/stl-containers.md).  
  
 Décrit un objet pouvant servir d’itérateur inverse constant pour la séquence contrôlée.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
 
typedef T8 const_reverse_iterator;  
```  
  
## <a name="remarks"></a>Notes  
 Il est décrit ici comme un synonyme du type non spécifié **T8** (généralement, [reverse_iterator](../standard-library/container-class-reverse-iterator.md) <[const_iterator](../standard-library/container-class-const-iterator.md)`>`).
