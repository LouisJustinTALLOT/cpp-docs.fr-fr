# &lt;typeinfo&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Incluez l'en\-tête standard `<typeinfo>` pour définir plusieurs types associés à l'opérateur [typeid, opérateur](../cpp/typeid-operator.md)de type ID, qui fournit des informations sur les types statiques et dynamiques.  
  
## Syntaxe  
  
```  
  
#include <typeinfo>  
  
```  
  
## Notes  
 Pour plus d'informations sur les classes définies dans \<le typeinfo\>, consultez les rubriques suivantes :  
  
-   [bad\_cast, exception](../cpp/bad-cast-exception.md)  
  
-   [bad\_typeid, exception](../cpp/bad-typeid-exception.md)  
  
-   [type\_info, classe](../cpp/type-info-class.md)
