# <a name="ltanygt"></a>&lt;any&gt;

Définit la classe de prise en charge de tout et plusieurs fonctions et classes.

## <a name="requirements"></a>Configuration requise

**En-tête :** \<tout >

**Espace de noms :** std

## <a name="members"></a>Membres

### <a name="functions"></a>Fonctions

|||
|-|-|
|[any_cast](../standard-library/any-functions.md#any_cast)|Rend un objet dans un n’importe quel.|
|[make_any](../standard-library/any-functions.md#make_any)|Prend les valeurs et crée un objet de n’importe quel.|
|[swap](../standard-library/any-functions.md#swap)|Échange de tous les éléments de deux objets.|

### <a name="classes"></a>Classes

|||
|-|-|
|[any](../standard-library/any-class.md)|Stocke tout type qui répond aux exigences de constructeur ou n’a aucune valeur.|
|[bad_any_cast](../standard-library/bad-any-cast-class.md)|Objets levées par un échec `any_cast`.|
