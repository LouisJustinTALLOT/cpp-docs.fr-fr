# <a name="container-classdifference_type"></a>Conteneur Class::difference_type

> [!NOTE]
> Cette rubrique se trouve dans la documentation de Microsoft C++ comme un exemple non fonctionnel de conteneurs utilisés dans la bibliothèque standard C++. Pour plus d’informations, consultez [Conteneurs disponibles dans la bibliothèque standard C++](../standard-library/stl-containers.md).

Décrit un objet qui peut représenter la différence entre les adresses de deux éléments quelconques dans la séquence contrôlée.

## <a name="syntax"></a>Syntaxe

```cpp
typedef T1 difference_type;
```

## <a name="remarks"></a>Notes

Il est décrit ici comme un synonyme du type non spécifié `T1` (généralement `Alloc::difference_type` ).
