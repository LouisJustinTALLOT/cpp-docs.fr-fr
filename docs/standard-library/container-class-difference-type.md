# <a name="container-classdifferencetype"></a>Conteneur Class::difference_type

> [!NOTE]
> Cette rubrique se trouve dans la C++ documentation de Microsoft comme un exemple non fonctionnel de conteneurs utilisés dans C++ la bibliothèque standard. Pour plus d’informations, consultez [Conteneurs de la bibliothèque standard C++](../standard-library/stl-containers.md).

Décrit un objet qui peut représenter la différence entre les adresses de deux éléments quelconques dans la séquence contrôlée.

## <a name="syntax"></a>Syntaxe

```

typedef T1 difference_type;
```

## <a name="remarks"></a>Notes

Il est décrit ici comme un synonyme du type `T1` non spécifié (généralement `Alloc::difference_type`).
