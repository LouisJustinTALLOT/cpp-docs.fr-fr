# Conteneur Class::difference_type
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  Cette rubrique est dans la documentation Visual C\+\+ comme exemple fonctionnel de conteneurs utilisés dans la bibliothèque C\+\+ standard.  Pour plus d'informations, consultez l'[Conteneurs STL de](../standard-library/stl-containers.md).  
  
 Décrit un objet qui peut représenter la différence entre les adresses de deux éléments quelconques de la séquence contrôlée.  
  
## Syntaxe  
  
```  
  
typedef T1 difference_type;  
  
```  
  
## Notes  
 Il est décrit ici comme synonyme du type non spécifié **T1** \(généralement **Alloc::difference\_type**\).
