# <a name="ltinitializer_listgt"></a>&lt;initializer_list&gt;

Définit le modèle de classe `initializer_list` de conteneur et plusieurs modèles de prise en charge.

## <a name="syntax"></a>Syntaxe

```cpp
#include <initializer_list>
```

### <a name="classes"></a>Classes

|Classe|Description|
|-|-|
|[initializer_list](../standard-library/initializer-list-class.md)|Fournit l'accès à un tableau d'éléments dans lequel chaque membre est du type spécifié.|
