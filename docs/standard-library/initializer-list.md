# <a name="ltinitializerlistgt"></a>&lt;initializer_list&gt;

Définit la classe de modèle de conteneur `initializer_list` et plusieurs modèles de prise en charge.

## <a name="syntax"></a>Syntaxe

```cpp
#include <initializer_list>
```

### <a name="classes"></a>Classes

|Classe|Description|
|-|-|
|[initializer_list](../standard-library/initializer-list-class.md)|Fournit l'accès à un tableau d'éléments dans lequel chaque membre est du type spécifié.|
