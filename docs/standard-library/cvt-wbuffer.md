# <a name="ltcvtwbuffergt"></a>&lt;cvt/wbuffer&gt;

L’en-tête `<cvt/wstring>` dans les versions précédentes de Visual Studio définissait le modèle de classe [wbuffer_convert classe](../standard-library/wbuffer-convert-class.md) dans l’espace de noms stdext :: CVT. L'en-tête est conservé à des fins de compatibilité descendante. Le nouveau code doit utiliser la version de la classe qui est définie dans [\<locale>](../standard-library/locale.md) dans l' `std` espace de noms

## <a name="syntax"></a>Syntaxe

```cpp
#include <cvt/wbuffer>
```
