# try_to_lock_t, structure
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Représente un `struct` qui définit un objet [try\_to\_lock](../Topic/try_to_lock%20Variable.md).  Permet de sélectionner l'un des constructeurs surchargés de [unique\_lock](../standard-library/unique-lock-class.md).  
  
## Syntaxe  
  
```cpp  
struct try_to_lock_t;  
```  
  
## Configuration requise  
 **En\-tête :** mutex  
  
 **Espace de noms :** std
