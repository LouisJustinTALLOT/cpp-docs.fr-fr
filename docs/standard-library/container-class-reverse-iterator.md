# <a name="container-classreverse_iterator"></a>Conteneur Class::reverse_iterator

> [!NOTE]
> Cette rubrique se trouve dans la documentation de Microsoft C++ comme un exemple non fonctionnel de conteneurs utilisés dans la bibliothèque standard C++. Pour plus d’informations, consultez [Conteneurs disponibles dans la bibliothèque standard C++](../standard-library/stl-containers.md).

Décrit un objet pouvant servir d’itérateur inverse pour la séquence contrôlée.

## <a name="syntax"></a>Syntaxe

```cpp
typedef T7 reverse_iterator;
```

## <a name="remarks"></a>Notes

Il est décrit ici comme un synonyme du type non spécifié `T7` (généralement `reverse_iterator` **\<**[iterator](../standard-library/container-class-iterator.md)**>** ).
