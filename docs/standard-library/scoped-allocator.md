# &lt;scoped_allocator&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Définit le scoped\_allocator de classe de modèle du conteneur.  
  
## Syntaxe  
  
```cpp  
#include <scoped_allocator>  
```  
  
### Opérateurs  
  
|||  
|-|-|  
|[operator\!\=](../Topic/operator!=%20Operator%20\(%3Cscoped_allocator%3E\).md)|Teste si l'objet de scoped\_allocator à gauche de l'opérateur n'est pas égal à l'objet de liste dans la partie droite.|  
|[operator\=\=](../Topic/operator==%20Operator%20\(%3Cscoped_allocator%3E\).md)|Teste si l'objet de scoped\_allocator à gauche de l'opérateur est égal à l'objet de liste dans la partie droite.|  
  
### Classes  
  
|||  
|-|-|  
|[classe de scoped\_allocator\_adaptor](../standard-library/scoped-allocator-adaptor-class.md)|Une classe de modèle qui inclut un emboîtement d'un ou plusieurs allocateurs.|
