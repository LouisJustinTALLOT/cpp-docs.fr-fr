# <a name="container-classbegin"></a>Conteneur Class::begin

> [!NOTE]
> Cette rubrique se trouve dans la documentation C++ de Visual Studio sous la forme d’un exemple non fonctionnel de C++ conteneurs utilisés dans la bibliothèque standard. Pour plus d’informations, consultez [Conteneurs de la bibliothèque standard C++](../standard-library/stl-containers.md).

Retourne un itérateur qui pointe sur le premier élément de la séquence (ou juste après la fin d’une séquence vide).

## <a name="syntax"></a>Syntaxe

```

    const_iterator begin() const;

iterator begin();
```
