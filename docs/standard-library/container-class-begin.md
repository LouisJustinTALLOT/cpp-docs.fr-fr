# <a name="container-classbegin"></a>Conteneur Class::begin

> [!NOTE]
> Cette rubrique se trouve dans la documentation de Visual Studio C++ comme un exemple non fonctionnel de conteneurs utilisés dans la bibliothèque standard C++. Pour plus d’informations, consultez [Conteneurs disponibles dans la bibliothèque standard C++](../standard-library/stl-containers.md).

Retourne un itérateur qui pointe vers le premier élément de la séquence (ou juste après la fin d’une séquence vide).

## <a name="syntax"></a>Syntaxe

```cpp
const_iterator begin() const;

iterator begin();
```
