# <a name="ltarraygt-functions"></a>&lt;array&gt;, fonctions

Le \<tableau > en-tête inclut deux fonctions non-membres, `get` et `swap`, qui opèrent sur **tableau** objets.

|||
|-|-|
|[get](#get)|[swap](#swap)|

## <a name="get"></a>  get

Retourne une référence à l’élément spécifié du tableau.

```cpp
template <int Index, class T, size_t N>
constexpr T& get(array<T, N>& arr) noexcept;

template <int Index, class T, size_t N>
constexpr const T& get(const array<T, N>& arr) noexcept;

template <int Index, class T, size_t N>
constexpr T&& get(array<T, N>&& arr) noexcept;
```

### <a name="parameters"></a>Paramètres

*Index* décalage de l’élément.

*T* le type d’un élément.

*N* le nombre d’éléments dans le tableau.

*ARR* sélectionner à partir du tableau.

### <a name="example"></a>Exemple

```cpp
#include <array>
#include <iostream>

using namespace std;

typedef array<int, 4> MyArray;

int main()
{
    MyArray c0 { 0, 1, 2, 3 };

    // display contents " 0 1 2 3"
    for (const auto& e : c0)
    {
        cout << " " << e;
    }
    cout << endl;

    // display odd elements " 1 3"
    cout << " " << get<1>(c0);
    cout << " " << get<3>(c0) << endl;
}
```

```Output
0 1 2 3
1 3
```

## <a name="swap"></a>  swap

Une spécialisation de modèle non-membre de `std::swap` qui échange deux **tableau** objets.

```cpp
template <class Ty, std::size_t N>
void swap(array<Ty, N>& left, array<Ty, N>& right);
```

### <a name="parameters"></a>Paramètres

*Ty* le type d’un élément.

*N* la taille du tableau.

*gauche* le premier tableau à échanger.

*droit* le deuxième tableau à échanger.

### <a name="remarks"></a>Notes

La fonction de modèle exécute `left.swap(right)`.

### <a name="example"></a>Exemple

```cpp
// std__array__swap.cpp
// compile with: /EHsc
#include <array>
#include <iostream>

typedef std::array<int, 4> Myarray;
int main()
{
    Myarray c0 = { 0, 1, 2, 3 };

    // display contents " 0 1 2 3"
    for (Myarray::const_iterator it = c0.begin();
        it != c0.end(); ++it)
        std::cout << " " << *it;
    std::cout << std::endl;

    Myarray c1 = { 4, 5, 6, 7 };
    c0.swap(c1);

    // display swapped contents " 4 5 6 7"
    for (Myarray::const_iterator it = c0.begin();
        it != c0.end(); ++it)
        std::cout << " " << *it;
    std::cout << std::endl;

    swap(c0, c1);

    // display swapped contents " 0 1 2 3"
    for (Myarray::const_iterator it = c0.begin();
        it != c0.end(); ++it)
        std::cout << " " << *it;
    std::cout << std::endl;

    return (0);
}
```

```Output
0 1 2 3
4 5 6 7
0 1 2 3
```
