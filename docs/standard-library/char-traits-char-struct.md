# char_traits&lt;char&gt;, struct
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Une structure qui est une spécialisation de structure **char\_traits\<CharType\>** de modèle à un élément de type `char`.  
  
## Syntaxe  
  
```  
template<> struct char_traits<char>;  
```  
  
## Notes  
 La spécialisation structure permet à d'exploiter les fonctions de la bibliothèque qui manipulent des objets de ce type `char`.  
  
## Exemple  
 Consultez les typedefs et les fonctions membres de la classe de modèle [classe de char\_traits](../standard-library/char-traits-struct.md)\<**CharType**\> pour obtenir des exemples qui impliquent des éléments de type `char`.  
  
## Configuration requise  
 **En\-tête :** \<chaîne\>  
  
 **Espace de noms :** std
