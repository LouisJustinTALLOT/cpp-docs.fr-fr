##  <a name="a-namegenericerrnoa--genericerrno"></a><a name="generic_errno"></a>  generic_errno  
 Type qui représente l’énumération qui fournit les noms symboliques de toutes les macros de code d’erreur définies par Posix dans `<errno.h>`.  
  
```
typedef errc generic_error;
```  
  
### <a name="remarks"></a>Notes  
 Le type est un synonyme de [errc](../standard-library/system-error-enums.md#errc_enumeration).
