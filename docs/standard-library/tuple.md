# &lt; tuple &gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Définit un modèle `tuple` dont les instances des objets de types différents.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
#include <tuple>  
```  
  
### <a name="classes"></a>Classes  
  
|||  
|-|-|  
|[Tuple](../standard-library/tuple-class.md)|Encapsule une séquence de longueur fixe d’éléments.|  
|[tuple_element, classe](../standard-library/tuple-element-class-tuple.md)|Encapsule le type d'un élément `tuple`.|  
|[tuple_size, classe](../standard-library/tuple-size-class-tuple.md)|Encapsule le nombre d'éléments `tuple`.|  
  
### <a name="operators"></a>Opérateurs  
  
|||  
|-|-|  
|[opérateur ==](../Topic/%3Ctuple%3E%20operators.md#operator_eq_eq)|Comparaison de `tuple` objets égales|  
|[opérateur ! =](../Topic/%3Ctuple%3E%20operators.md#operator_neq)|Comparaison de `tuple` objets, n’est pas égales|  
|[opérateur <](../Topic/%3Ctuple%3E%20operators.md#operator_lt_)|Comparaison de `tuple` objets, inférieur à|  
|[opérateur < =](../Topic/%3Ctuple%3E%20operators.md#operator_lt__eq)|Comparaison de `tuple` des objets, inférieur ou égal|  
|[opérateur >](../Topic/%3Ctuple%3E%20operators.md#operator_gt_)|Comparaison de `tuple` objets supérieurs à|  
|[opérateur > =](../Topic/%3Ctuple%3E%20operators.md#operator_gt__eq)|Comparaison de `tuple` objets, supérieurs ou égales|  
  
### <a name="functions"></a>Fonctions  
  
|||  
|-|-|  
|[Télécharger](../Topic/%3Ctuple%3E%20functions.md#get_function)|Obtient un élément auprès d'un objet `tuple`.|  
|[make_tuple](../Topic/%3Ctuple%3E%20functions.md#make_tuple_function)|Rend un `tuple` à partir des valeurs de l’élément.|  
|[lier](../Topic/%3Ctuple%3E%20functions.md#tie_function)|Rend un `tuple` de références de l’élément.|
