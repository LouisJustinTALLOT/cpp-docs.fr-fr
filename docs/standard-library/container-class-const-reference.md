# <a name="container-classconstreference"></a>Conteneur Class::const_reference
> [!NOTE]
>  Cette rubrique fait partie de la documentation Visual C++ comme exemple non fonctionnel de conteneurs utilisés dans la bibliothèque standard C++. Pour plus d’informations, consultez [Conteneurs de la bibliothèque standard C++](../standard-library/stl-containers.md).  
  
 Décrit un objet pouvant servir de référence constante à un élément de la séquence contrôlée.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
 
typedef T3 const_reference;  
```  
  
## <a name="remarks"></a>Notes  
 Il est décrit ici comme un synonyme du type non spécifié **T3** (généralement, **Alloc::const_reference**).
