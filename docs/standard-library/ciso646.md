# <a name="ltciso646gt"></a>&lt;ciso646&gt;

Inclut l’en-tête de la bibliothèque standard C \<iso646.h> et ajoute les noms associés à l' `std` espace de noms.

> [!NOTE]
> Étant donné que cet en-tête de compatibilité définit des noms qui sont des mots clés en C++, il n’a aucun effet. L' \<iso646.h> en-tête est déconseillé en C++. L' \<ciso646> en-tête est supprimé dans le brouillon c++ 20 standard.

## <a name="requirements"></a>Spécifications

**En-tête :**\<ciso646>

**Espace de noms :** std

## <a name="remarks"></a>Notes

L’inclusion de cet en-tête garantit que les noms déclarés à l’aide de la liaison externe dans l’en-tête de la bibliothèque standard C sont déclarés dans l' `std` espace de noms.
