# <a name="ltdequegt"></a>&lt;deque&gt;

Définit le modèle de classe de conteneur deque et plusieurs modèles de prise en charge.

## <a name="requirements"></a>spécifications

**En-tête** : \<deque>

> [!NOTE]
> La bibliothèque de > \<deque utilise également l’instruction `#include <initializer_list>`.

|[operator<](../standard-library/deque-operators.md#op_lt)|Teste si l'objet deque situé à gauche de l'opérateur est inférieur à l'objet deque situé à droite.|
