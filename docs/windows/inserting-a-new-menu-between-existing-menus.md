# Inserting a New Menu Between Existing Menus
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Pour insérer un nouveau menu parmi des menus existants  
  
1.  Sélectionnez un nom de menu existant, puis appuyez sur la touche **Inser**.  La zone **Nouvel élément** est insérée avant l'élément sélectionné.  
  
     ou  
  
2.  Cliquez avec le bouton droit sur la barre de menus, puis choisissez **Insérer nouveau** dans le menu contextuel.  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.*
