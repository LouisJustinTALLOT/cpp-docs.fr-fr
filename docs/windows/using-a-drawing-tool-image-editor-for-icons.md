# Using a Drawing Tool (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les outils de dessin à main levée et d'effacement de l'Éditeur d'images fonctionnent tous de la même façon : sélectionnez l'outil et, si nécessaire, [sélectionnez les couleurs de premier plan et d'arrière\-plan](../windows/selecting-foreground-or-background-colors-image-editor-for-icons.md) ainsi que les options de taille et de forme.  Vous pouvez ensuite déplacer le pointeur sur l'image et cliquer ou faire glisser le pointeur pour dessiner et pour effacer.  
  
 Lorsque vous sélectionnez l'outil **Gomme**, l'outil **Pinceau** ou l'outil **Aérographe**, le sélecteur d'options affiche les options de l'outil.  
  
> [!TIP]
>  Plutôt que d'utiliser l'outil **Gomme**, vous pouvez dessiner sur la couleur d'arrière\-plan avec l'un des outils de dessin.  
  
 Vous pouvez sélectionner les outils de dessin dans la barre d'outils **Éditeur d'images** ou le menu **Image**.  
  
### Pour sélectionner et utiliser un outil de dessin dans la barre d'outils Éditeur d'images  
  
1.  Cliquez sur un bouton dans la barre d'outils **Éditeur d'images**.  
  
    -   L'outil **Gomme** peint sur l'image avec la couleur d'arrière\-plan actuelle lorsque vous appuyez sur le bouton gauche de la souris.  
  
    -   L'outil **Crayon** dessine à main levée avec une largeur constante d'un pixel.  
  
    -   Le **sélecteur d'options détermine la forme et la taille de l'outil Pinceau**.  
  
    -   L'outil **Aérographe** distribue aléatoirement des pixels de couleur autour du centre du pinceau.  
  
        > [!TIP]
        >  Des info\-bulles s'affichent lorsque vous passez sur les boutons dans la [barre d'outils Éditeur d'images](../mfc/toolbar-image-editor-for-icons.md).  Ces info\-bulles vous aident à identifier les différents boutons mentionnés ici.  
  
2.  Si nécessaire, sélectionnez les couleurs et un pinceau :  
  
    -   Dans la [palette Couleurs](../windows/colors-window-image-editor-for-icons.md), cliquez avec le bouton gauche pour sélectionner une couleur de premier plan ou avec le bouton droit pour sélectionner une couleur d'arrière\-plan.  
  
    -   Dans le [sélecteur d'options](../mfc/toolbar-image-editor-for-icons.md), cliquez sur une forme représentant le pinceau à utiliser.  
  
3.  Pointez vers l'endroit dans l'image où vous souhaitez commencer à dessiner ou à peindre.  La forme du pointeur change en fonction de l'outil sélectionné.  
  
4.  Appuyez sur le bouton gauche \(pour la couleur de premier plan\) ou le bouton droit \(pour la couleur d'arrière\-plan\), et maintenez\-le enfoncé tout en dessinant.  
  
### Pour sélectionner et utiliser un outil de dessin dans le menu Image  
  
1.  Cliquez sur le menu **Image** et sélectionnez la commande **Outils**.  
  
2.  Dans le sous\-menu en cascade, choisissez l'outil que vous souhaitez utiliser.
