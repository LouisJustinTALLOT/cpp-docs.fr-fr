# <a name="array-attributes"></a>Attributs de tableau

Les attributs suivants s’applique aux tableaux ou `SAFEARRAY`s.

|Attribut|Description|
|---------------|-----------------|
|[library_block](library-block.md)|Place une construction à l’intérieur du bloc de bibliothèque du fichier .idl.|
|[satype](satype.md)|Spécifie le type de données de la `SAFEARRAY` structure.|
|[string](string-cpp.md)|Indique qu’unidimensionnel **char**, **wchar_t**, `byte` (ou équivalent) tableau ou pointeur vers ce type de tableau doit être traité comme une chaîne.|
