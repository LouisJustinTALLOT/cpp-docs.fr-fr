# <a name="array-attributes"></a>Attributs de tableau

Les attributs suivants s’appliquent aux tableaux ou `SAFEARRAY`s.

|Attribut|Description|
|---------------|-----------------|
|[library_block](library-block.md)|Place une construction dans le bloc de bibliothèque du fichier. idl.|
|[satype](satype.md)|Spécifie le type de données de la structure `SAFEARRAY`.|
|[string](string-cpp.md)|Indique que le tableau **char**, **wchar_t**, `byte` (ou équivalent) ou le pointeur vers un tel tableau doit être traité comme une chaîne.|
