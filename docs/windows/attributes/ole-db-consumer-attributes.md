# <a name="ole-db-consumer-attributes"></a>Attributs du consommateur OLE DB

Le consommateur OLE DB attribue le code d’injection, basé sur les [modèles de consommation OLE DB](../../data/oledb/ole-db-consumer-templates-reference.md), pour créer un consommateur OLE DB fonctionnant qui effectue des tâches telles que l’ouverture de tables, l’exécution des commandes et l’accès aux données.

|Attribut|Description|
|---------------|-----------------|
|[db_accessor](db-accessor.md)|Lie les colonnes dans un rangée et les lie aux cartes d’accesseur correspondantes.|
|[db_column](db-column.md)|Lie une colonne spécifiée à l’assaillis.|
|[db_command](db-command.md)|Exécute une commande OLE DB.|
|[db_param](db-param.md)|Associe la variable spécifiée au membre à un paramètre d’entrée ou de sortie.|
|[db_source](db-source.md)|Crée et encapsule une connexion, par l’intermédiaire d’un fournisseur, à une source de données.|
|[db_table](db-table.md)|Ouvre une table OLE DB.|
