# <a name="readonly-c"></a>readonly (C++)

Interdit l’assignation à un membre de données.

## <a name="syntax"></a>Syntaxe

```cpp
[readonly]
```

## <a name="remarks"></a>Notes

Le **readonly** attribut C++ a les mêmes fonctionnalités que le [readonly](/windows/desktop/Midl/readonly) attribut MIDL.

Si vous souhaitez interdire la modification d’un paramètre de méthode, puis utiliser le [dans](in-cpp.md) attribut.

## <a name="example"></a>Exemple

Le code suivant illustre une utilisation de l’attribut **readonly** :

```cpp
// cpp_attr_ref_readonly.cpp
// compile with: /LD
[idl_quote("midl_pragma warning(disable:2461)")];
#include "unknwn.h"
[module(name="ATLFIRELib")];

[dispinterface, uuid(11111111-1111-1111-1111-111111111111)]
__interface IFireTabCtrl
{
   [readonly, id(1)] int i();
};
```
