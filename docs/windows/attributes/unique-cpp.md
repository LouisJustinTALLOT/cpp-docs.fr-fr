# <a name="unique-c"></a>unique (C++)

Spécifie un pointeur unique.

## <a name="syntax"></a>Syntaxe

```cpp
[unique]
```

## <a name="remarks"></a>Notes

L’attribut **unique** C++ a les mêmes fonctionnalités que l’attribut MIDL [unique](/windows/win32/Midl/unique) .
