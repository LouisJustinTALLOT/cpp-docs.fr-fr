# <a name="attribute"></a>attribut

> [!NOTE]
> L’attribut **attribute** est maintenant déconseillé.  Utilisez l’attribut common language runtime `System.Attribute` pour créer directement des attributs définis par l’utilisateur. Pour plus d'informations, consultez [User-Defined Attributes](../../extensions/user-defined-attributes-cpp-component-extensions.md).
