# <a name="licensed"></a>licensed

Indique que l’objet COM auquel il s’applique est concédé sous licence et doit être instancié à l’aide de `IClassFactory2`.

## <a name="syntax"></a>Syntaxe

```cpp
[licensed]
```

## <a name="remarks"></a>Notes

L’attribut **sous licence** C++ a les mêmes fonctionnalités que l’attribut MIDL [sous licence](/windows/win32/Midl/licensed) .

## <a name="example"></a>Exemple

```cpp
// cpp_attr_ref_licensed.cpp
// compile with: /LD
#include "unknwn.h"
[object, uuid("00000000-0000-0000-0000-000000000001")]
__interface IMyI : IUnknown {
   HRESULT f();
};

[coclass, version("2.1"), uuid(12345678-1111-2222-3333-123456789012),
licensed, threading(free), progid(some.name)]
class CSample : public IMyI {
public:
   int nSize;
};

[module(name="MyLibrary", version="1.0", helpstring="My Library Block")];
```
