# <a name="nonbrowsable"></a>nonbrowsable

Indique qu’un membre d’interface ne doit pas être affiché dans un Explorateur de propriétés.

## <a name="syntax"></a>Syntaxe

```cpp
[nonbrowsable]
```

## <a name="remarks"></a>Notes

L' **nonbrowsable** C++ attribut ne pouvant être exploré a les mêmes fonctionnalités que l’attribut MIDL ne pouvant être [exploré](/windows/win32/Midl/nonbrowsable) .

## <a name="example"></a>Exemple

```cpp
// cpp_attr_ref_nonbrowsable.cpp
// compile with: /LD
#include <unknwn.h>
[module(name="MyLib")];

[object, helpstring("help string"), helpstringcontext(1),
uuid="11111111-1111-1111-1111-111111111111"]
__interface IMyI
{
   [nonbrowsable] HRESULT xx();
};
```
