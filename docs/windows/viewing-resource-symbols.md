# <a name="viewing-resource-symbols"></a>Affichage des symboles des ressources
### <a name="to-view-resource-symbols"></a>Pour afficher les symboles des ressources  
  
1.  Dans [affichage des ressources](../windows/resource-view-window.md), avec le bouton droit de votre fichier .rc.  
  
    > [!NOTE]
    >  Si votre projet ne contient pas déjà un fichier .rc, consultez [Création d'un fichier de script de ressources](../windows/how-to-create-a-resource-script-file.md).  
  
2.  Sélectionnez **symboles des ressources** dans le menu contextuel pour afficher une table de symboles des ressources dans les [boîte de dialogue Symboles des ressources](../windows/resource-symbols-dialog-box.md).  
  
    > [!NOTE]
    >  Pour afficher les symboles prédéfinis, vérifiez le **afficher les symboles en lecture seule** case à cocher.  
  

  
 **Spécifications**  
  
 Win32
