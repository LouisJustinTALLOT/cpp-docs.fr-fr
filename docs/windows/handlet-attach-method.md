# HandleT::Attach, m&#233;thode
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Associe le handle spécifié avec l'objet HandleT actuel.  
  
## Syntaxe  
  
```  
void Attach(  
   typename HandleTraits::Type h  
);  
```  
  
#### Paramètres  
 `h`  
 Handle.
