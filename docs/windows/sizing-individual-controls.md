# Sizing Individual Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Utilisez les poignées de dimensionnement pour redimensionner un contrôle.  Lorsque le pointeur est positionné sur une poignée de dimensionnement, il change de forme pour indiquer les directions en fonction desquelles le contrôle peut être redimensionné.  Les poignées de dimensionnement actives sont pleines ; si une poignée est vide, le contrôle ne peut pas être redimensionné selon cet axe.  
  
 Vous pouvez également changer la taille d'un contrôle en alignant le contrôle sur les repères ou les marges, ou en déplaçant un contrôle aligné et en l'éloignant d'un autre.  
  
### Pour dimensionner un contrôle  
  
1.  Sélectionnez le contrôle.  
  
2.  Faites glisser les poignées de dimensionnement pour changer la taille du contrôle :  
  
    -   Les poignées de dimensionnement situées en haut et sur les côtés changent la taille horizontale ou verticale.  
  
    -   Les poignées de dimensionnement situées dans les coins changent la taille horizontale et verticale en même temps.  
  
    > [!TIP]
    >  Vous pouvez redimensionner le contrôle d'une unité de boîte de dialogue \(DLU\) à la fois en maintenant la touche MAJ enfoncée et en utilisant les touches DROITE et BAS.  
  
### Pour dimensionner automatiquement un contrôle afin qu'il s'ajuste au texte qu'il contient  
  
1.  Dans le menu **Format**, cliquez sur **Ajuster au contenu**.  
  
 \- ou \-  
  
-   Cliquez avec le bouton droit sur le contrôle et choisissez **Ajuster au contenu** dans le menu contextuel.
