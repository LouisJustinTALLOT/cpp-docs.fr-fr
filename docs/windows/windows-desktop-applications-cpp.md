# Applications de bureau Windows (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Maintenant que nous avons appris à utiliser l'IDE de Visual Studio et les applications en ligne de commande, nous allons apprendre à créer des applications Windows.  Avec Visual C\+\+, vous pouvez créer des applications Windows en utilisant de nombreuses technologies différentes, telles que l'interface [API Windows](https://msdn.microsoft.com/en-us/library/cc433218.aspx) \(également appelée l'interface API Win32\) et le .NET Framework.  
  
 Dans cette section, nous allons créer deux applications Windows simples en utilisant l'interface API Win32 et le .NET Framework.  Nous créerons également un Contrôle Windows Forms à l'aide du .NET Framework, et enfin, nous créerons un jeu simple en utilisant DirectX.  
  
## Composants requis  
 Ces rubriques supposent que vous comprenez les notions de base du langage C\+\+.
