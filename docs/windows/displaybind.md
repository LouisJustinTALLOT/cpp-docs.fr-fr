# displaybind
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Indique une propriété qui doit être affichée à l'utilisateur comme pouvant être liée.  
  
## Syntaxe  
  
```  
  
[displaybind]  
  
```  
  
## Notes  
 L'attribut de **displaybind** C\+\+ a les mêmes fonctionnalités que l'attribut de [displaybind](http://msdn.microsoft.com/library/windows/desktop/aa366804) MIDL.  
  
## Exemple  
 Consultez l'exemple pour [pouvant être liée](../windows/bindable.md) pour un exemple d'utilisation **displaybind**.
