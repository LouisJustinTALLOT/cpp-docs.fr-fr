# Bo&#238;te de dialogue Param&#232;tres de la grille (&#201;diteur d&#39;images pour les ic&#244;nes)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Vous permet de spécifier les paramètres de la grille pour votre image et affiche le quadrillage sur l'image modifiée.  Les lignes sont utiles pour modifier l'image, mais ne sont pas enregistrées comme faisant partie de l'image.  
  
 **Grille de pixels**  
 Lorsque cette option est activée, affiche une grille autour de chaque pixel dans l'Éditeur d'images.  La grille s'affiche uniquement pour une résolution 4x et supérieure.  
  
 **Grille mosaïque**  
 Lorsque cette option est activée, affiche une grille autour des blocs de pixels dans l'Éditeur d'images, spécifié par les valeurs d'espacement de la grille.  
  
 **Largeur**  
 Spécifie la largeur de chaque bloc.  Cette option est utile lors du dessin de bitmaps qui contiennent plusieurs images organisées à intervalles réguliers.  
  
 **Hauteur**  
 Spécifie la hauteur de chaque bloc.  Cette option est utile lors du dessin de bitmaps qui contiennent plusieurs images organisées à intervalles réguliers.  
  
## Configuration requise  
 Aucun
