# Editing Managed Resource Files
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les fichiers de ressources managés \(.resx\) sont des fichiers XML.  Lorsque vous ajoutez un fichier de ressources managées à votre projet à partir de la boîte de dialogue **Ajouter un nouvel élément**, l'**Éditeur de ressources managées** s'ouvre par défaut.  
  
 Pour plus d'informations, consultez [Ressources dans un fichier au format .Resx](http://msdn.microsoft.com/fr-fr/0c476133-87e4-47e8-b0ef-4b88f4ef3dc5).
