# Editing Graphical Resources (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les ressources graphiques sont les images que vous définissez pour votre application.  Vous pouvez dessiner à main levée ou utiliser des formes.  Vous pouvez sélectionner des parties d'une image pour la modifier, la retourner ou la redimensionner ou vous pouvez créer un pinceau personnalisé à partir d'une partie sélectionnée d'une image et dessiner avec ce pinceau.  Vous pouvez définir les propriétés d'une image, enregistrer les images dans différents formats et convertir les images d'un format à un autre.  
  
 En plus de la création de nouvelles ressources graphiques, vous pouvez [importer des images existantes](../windows/how-to-import-and-export-resources.md) à modifier, puis à ajouter à votre projet.  Vous pouvez également ouvrir et éditer les images ne faisant pas partie d'un projet pour une [modification d'image autonome](../mfc/editing-an-image-outside-of-a-project-image-editor-for-icons.md).
