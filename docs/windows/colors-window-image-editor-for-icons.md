# Fen&#234;tre Couleurs (&#201;diteur d&#39;images pour les ic&#244;nes)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La fenêtre Couleurs comporte deux parties :  
  
1.  La palette de couleurs, un tableau d'échantillons de couleurs qui représentent les couleurs que vous pouvez utiliser.  Vous pouvez cliquer sur les échantillons pour sélectionner les couleurs de premier plan et d'arrière\-plan quand vous utilisez les outils graphiques.  
  
2.  L'indicateur de couleur, qui affiche les couleurs de premier plan et d'arrière\-plan, ainsi que les sélecteurs pour la couleur de l'écran et la couleur inversée.  
  
     ![Fenêtre Couleurs](../windows/media/vccolorswindow.png "vcColorsWindow")  
Fenêtre Couleurs  
  
> [!NOTE]
>  Les outils **Couleur de l'écran** et **Couleur inversée** ne sont disponibles que pour les icônes et les curseurs.  
  
 Vous pouvez utiliser la fenêtre Couleurs avec la [barre d'outils Éditeur d'images](../mfc/toolbar-image-editor-for-icons.md).  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.* Pour plus d'informations sur l'ajout de fichiers de ressources aux projets managés, l'accès aux ressources, l'affichage de ressources statiques et l'assignation de chaînes de ressources aux propriétés, et ce manuellement, voir [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
