# <a name="finding-binary-data"></a>Recherche de données binaires
Vous pouvez rechercher des chaînes ASCII ou des octets hexadécimaux. Par exemple, pour rechercher « Hello », vous pouvez rechercher pour soit la chaîne « Hello » ou « 48 65 6C 6C 6F » (l’équivalent hexadécimal).  
  
### <a name="to-find-binary-data"></a>Pour rechercher des données binaires  
  
1.  À partir de la **modifier** menu, cliquez sur [trouver](/visualstudio/ide/reference/find-command).  
  
2.  Dans le **rechercher** zone, sélectionnez une chaîne de recherche précédente à partir de la liste déroulante ou tapez les données que vous souhaitez rechercher.  
  
3.  Sélectionnez un de la **trouver** options.  
  
4.  Cliquez sur **Rechercher suivant**.
