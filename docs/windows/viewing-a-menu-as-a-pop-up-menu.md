# Viewing a Menu as a Pop-up Menu
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Normalement, quand vous utilisez l'Éditeur de menus, une ressource de menu s'affiche sous forme de barre de menus.  Toutefois, il est possible que des ressources de menu soient ajoutées à la barre de menus de l'application pendant l'exécution du programme.  
  
### Pour afficher une ressource de menu sous forme de menu contextuel  
  
1.  Cliquez avec le bouton droit sur le menu, puis choisissez **Afficher comme Popup** dans le menu contextuel.  
  
     Cela est uniquement une préférence d'affichage. Votre menu n'est pas modifié.  
  
    > [!NOTE]
    >  Pour revenir à l'affichage de la barre de menus, cliquez à nouveau sur **Afficher comme Popup** \(ce qui supprime la coche et restaure l'affichage de la barre de menus\).  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.* Pour plus d'informations sur l'ajout de fichiers de ressources aux projets managés, l'accès aux ressources, l'affichage de ressources statiques et l'assignation de chaînes de ressources aux propriétés, et ce manuellement, voir [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
