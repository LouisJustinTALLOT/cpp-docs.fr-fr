# Opening the Resource Editor for a Given Symbol
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Quand vous parcourez les symboles de la [boîte de dialogue Symboles des ressources](../windows/resource-symbols-dialog-box.md), vous pouvez avoir besoin de plus d'informations sur l'utilisation d'un symbole particulier.  Le bouton **Afficher l'utilisation** offre un moyen rapide d'obtenir ces informations.  
  
### Pour passer à l'éditeur de ressources où un symbole est utilisé  
  
1.  Sélectionnez un symbole dans la zone **Nom** de la boîte de dialogue **Symboles des ressources**.  
  
2.  Dans la zone **Utilisé par**, sélectionnez le type de ressource qui vous intéresse.  
  
3.  Cliquez sur le bouton **Afficher l'utilisation**.  
  
     La ressource s'affiche dans la fenêtre d'éditeur appropriée.  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.*
