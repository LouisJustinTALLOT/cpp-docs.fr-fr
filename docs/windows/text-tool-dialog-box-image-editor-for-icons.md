# Bo&#238;te de dialogue Outil texte (&#201;diteur d&#39;images pour les ic&#244;nes)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Utilisez la boîte de dialogue **Outil texte** pour ajouter du texte à une ressource curseur, bitmap ou icône.  
  
 Pour accéder à cette boîte de dialogue, ouvrez l' [Éditeur d'images](../mfc/window-panes-image-editor-for-icons.md).  Dans le menu **Image**, sélectionnez **Outils** puis la commande **Outil texte**.  
  
 **Bouton Police**  
 Ouvre la [boîte de dialogue Police de l'outil texte](../mfc/text-tool-font-dialog-box-image-editor-for-icons.md), dans laquelle vous pouvez changer la police, le style ou la taille de la police du curseur.  Les modifications sont appliquées au texte affiché dans la zone Texte.  
  
 **Zone Texte**  
 Affiche le texte qui apparaît comme partie de la ressource.  Cette zone est vide au début.  
  
> [!NOTE]
>  Si **Arrière\-plan transparent** est défini, seul le texte sera placé dans l'image.  Si **Arrière\-plan opaque** est défini, un rectangle englobant rempli avec la [couleur d'arrière\-plan](../windows/selecting-foreground-or-background-colors-image-editor-for-icons.md), sera placé derrière le texte.  Pour plus d'informations, consultez [Choix d'un arrière\-plan transparent ou opaque](../windows/choosing-a-transparent-or-opaque-background-image-editor-for-icons.md).  
  
 Vous pouvez cliquer avec le bouton droit sur la boîte de dialogue **Outil Texte** pour accéder au menu contextuel par défaut qui contient une liste des commandes standard Windows.  
  
## Configuration requise  
 Aucun
