# Picking up a Color from an Image to Use Elsewhere (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

L'outil **Sélectionner une couleur** transforme la couleur de l'image en couleur de premier plan ou d'arrière\-plan, en fonction du bouton de la souris sur lequel vous cliquez.  Pour annuler l'outil **Sélectionner une couleur**, choisissez un autre outil.  
  
> [!TIP]
>  Les info\-bulles s'affichent lorsque vous passez votre curseur sur un bouton de barre d'outils.  Ces info\-bulles peuvent vous aider à identifier la fonction de chaque bouton.  
  
### Pour choisir une couleur  
  
1.  Dans la barre d'outils **Éditeur d'images** \(ou, dans le menu **Image**, la commande **Outils**\), cliquez sur l'outil **Sélectionner une couleur**.  
  
2.  Sélectionnez la couleur souhaitée dans l'image.  
  
    > [!NOTE]
    >  Après avoir sélectionné une couleur, l'Éditeur d'images réactive le dernier outil utilisé.  
  
3.  Dessinez avec le bouton gauche pour utiliser la couleur de premier plan ou avec le bouton droit pour utiliser la couleur d'arrière\-plan.
