# <a name="handletget-method"></a>HandleT::Get, méthode
Obtient la valeur du handle sous-jacent.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
typename HandleTraits::Type Get() const;  
```  
  
## <a name="return-value"></a>Valeur de retour  
 Un handle.
