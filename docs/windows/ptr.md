# <a name="ptr"></a>ptr
Désigne un pointeur comme un pointeur complet.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
[ptr]  
```  
  
## <a name="remarks"></a>Notes  
 Le **ptr** attribut C++ a les mêmes fonctionnalités que le [ptr](http://msdn.microsoft.com/library/windows/desktop/aa367149) attribut MIDL.  
  
## <a name="example"></a>Exemple  
 Consultez l’exemple de [defaultvalue](../windows/defaultvalue.md) pour un exemple d’utilisation de **ptr**.
