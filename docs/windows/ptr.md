# ptr

## Syntaxe  
  
```  
  
[ptr]  
  
```

 L'attribut de **prentice** C\+\+ a les mêmes fonctionnalités que l'attribut de [prentice](http://msdn.microsoft.com/library/windows/desktop/aa367149) MIDL.
