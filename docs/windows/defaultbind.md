# defaultbind
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Indique la propriété unique et pouvant être liée qui représente mieux l'objet.  
  
## Syntaxe  
  
```  
  
[defaultbind]  
  
```  
  
## Notes  
 L'attribut de **defaultbind** C\+\+ a les mêmes fonctionnalités que l'attribut de [defaultbind](http://msdn.microsoft.com/library/windows/desktop/aa366790) MIDL.
