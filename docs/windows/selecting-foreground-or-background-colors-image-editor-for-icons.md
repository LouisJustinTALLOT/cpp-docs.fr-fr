# Selecting Foreground or Background Colors (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

À l'exception de la **Gomme**, les outils de la barre d'outils **Éditeur d'images** permettent de dessiner avec la couleur de premier plan ou d'arrière\-plan active quand vous appuyez sur le bouton gauche ou droit de la souris, respectivement.  
  
### Pour sélectionner une couleur de premier plan  
  
1.  Avec le bouton gauche de la souris, cliquez sur la couleur souhaitée dans la [Palette de couleurs](../windows/colors-window-image-editor-for-icons.md).  
  
### Pour sélectionner une couleur d'arrière\-plan  
  
1.  Avec le bouton droit de la souris, cliquez sur la couleur souhaitée dans la **Palette de couleurs**.  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.*
