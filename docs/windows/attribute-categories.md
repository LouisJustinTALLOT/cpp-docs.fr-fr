# Attribute Categories
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

la référence d'attributs décrit les différents types d'attributs que vous pouvez utiliser dans Visual C\+\+.  
  
## Rubriques connexes  
 [Attributs COM](../windows/com-attributes.md)  
 Injecte le code pour prendre en charge de nombreux domaines du développement COM.  
  
 [Attributs IDL](../windows/idl-attributes.md)  
 Injecte le code qui crée ou modifie le fichier associé .idl d'un fichier de code source sans Assistant et sans être familiarisé avec la structure et la syntaxe de ce fichier.  
  
 [Attributs du consommateur OLE DB](../windows/ole-db-consumer-attributes.md)  
 Injecte le code, selon [modèles du consommateur OLE DB](../data/oledb/ole-db-consumer-templates-reference.md), pour créer un consommateur OLE DB actif qui effectue des tâches telles que des tableaux d'ouverture, exécution des données de commandes, puis d'accès.  
  
 [Attributs du compilateur](../windows/compiler-attributes.md)  
 Injecte le code qui effectue diverses tâches de programmation courantes.  
  
 [programmation par attributs](../windows/attributed-programming-concepts.md)  
 Fournit des liens vers des rubriques conceptuelles décrivant la programmation par attributs.
