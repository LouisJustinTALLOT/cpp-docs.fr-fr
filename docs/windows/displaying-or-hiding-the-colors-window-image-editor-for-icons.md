# Affichage ou masquage de la fen&#234;tre Couleurs (&#201;diteur d&#39;images pour les ic&#244;nes)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Pour afficher la fenêtre Couleurs  
  
1.  Cliquez avec le bouton droit dans un volet de l'**Éditeur d'images** et choisissez **Afficher la fenêtre des couleurs** dans le menu contextuel.  
  
     \- ou \-  
  
2.  Cliquez sur **Afficher la fenêtre des couleurs** dans le [menu Image](../mfc/image-menu-image-editor-for-icons.md).  
  
### Pour masquer la fenêtre Couleurs  
  
1.  Désépinglez la fenêtre.  Cela masque automatiquement la fenêtre lorsqu'elle n'est pas utilisée.  Pour plus d'informations, consultez [Réorganisation des fenêtres](../misc/how-to-arrange-and-dock-windows.md).  
  
 \- ou \-  
  
-   Cliquez sur le bouton **Fermer**.  
  
 Configuration requise  
  
 Aucun
