# Bo&#238;te de dialogue Charger la palette de couleurs (&#201;diteur d&#39;images pour les ic&#244;nes)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Utilisez cette boîte de dialogue pour charger des palettes de couleurs particulières à utiliser dans votre projet.  Pour plus d'informations, consultez [Enregistrement et chargement de différentes palettes de couleurs](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md).  
  
 **Regarder dans**  
 Spécifie l'emplacement d'un fichier ou d'un dossier.  Cliquez sur la flèche pour sélectionner un autre emplacement ou cliquez sur l'icône représentant un dossier dans la barre d'outils pour ouvrir le dossier parent.  
  
 **Nom du fichier**  
 Vous permet de taper le nom du fichier que vous souhaitez ouvrir.  Pour trouver rapidement un fichier que vous avez déjà ouvert, cliquez sur le nom du fichier dans la liste déroulante.  
  
 Si vous recherchez un fichier, vous pouvez utiliser des astérisques \(\*\) comme caractères génériques.  Par exemple, vous pouvez taper \*.\* pour afficher une liste de tous les fichiers.  Vous pouvez également taper le chemin d'accès complet à un fichier, par exemple, C:\\Mes Documents\\MyColorPalette.pal ou \\\\NetworkServer\\MyFolder\\MyColorPalette.pal.  
  
 **Type de fichiers**  
 Répertorie les types de fichiers à afficher.  Palette \(\*.pal\) est le type de fichier par défaut pour les palettes de couleurs.  
  
## Configuration requise  
 Aucun
