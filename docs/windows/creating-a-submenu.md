# Creating a Submenu
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Pour créer un sous\-menu  
  
1.  Cliquez sur la commande de menu pour laquelle vous souhaitez créer un sous\-menu.  
  
2.  Dans la zone **Nouvel élément** qui apparaît à droite, tapez le nom de la nouvelle commande de menu. Cette nouvelle commande apparaît en premier dans le sous\-menu.  
  
3.  Ajoutez des commandes de menu supplémentaires au sous\-menu.  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.*  
  
 **Spécifications**
