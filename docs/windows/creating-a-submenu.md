# <a name="creating-a-submenu-c"></a>Création d’un sous-menu (C++)

### <a name="to-create-a-submenu"></a>Pour créer un sous-menu

1. Cliquez sur la commande de menu pour laquelle vous souhaitez créer un sous-menu.

2. Dans la zone **Nouvel élément** qui apparaît à droite, tapez le nom de la nouvelle commande de menu. Cette nouvelle commande apparaît en premier dans le sous-menu.

3. Ajoutez des commandes de menu supplémentaires au sous-menu.

Pour plus d’informations sur l’ajout de ressources aux projets managés, consultez [Resources in Desktop Apps](/dotnet/framework/resources/index) dans le *Guide du développeur .NET Framework*.

## <a name="requirements"></a>Configuration requise

Win32
