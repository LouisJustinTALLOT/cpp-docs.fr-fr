# Adding, Editing, or Deleting Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

À l'aide de l'Éditeur de boîtes de dialogue, vous pouvez ajouter, redimensionner, modifier et supprimer des contrôles dans les boîtes de dialogue.  Vous pouvez également modifier les propriétés d'un contrôle, par exemple son ID, ou définir s'il est visible au moment de l'exécution.  
  
 Vous pouvez utiliser le menu contextuel dans l'Éditeur de boîtes de dialogue pour ajouter rapidement des contrôles ActiveX inscrits à une boîte de dialogue et vous pouvez ajouter des contrôles ActiveX à la Boîte à outils pour un accès rapide \(pour plus d'informations, consultez [Personnaliser la boîte à outils, boîte de dialogue](http://msdn.microsoft.com/fr-fr/bd07835f-18a8-433e-bccc-7141f65263bb)\).  
  
-   [Ajouter un contrôle à une boîte de dialogue](../mfc/adding-a-control-to-a-dialog-box.md)  
  
-   [Dimensionner un contrôle pendant son ajout](../mfc/sizing-a-control-while-you-add-it.md)  
  
-   [Ajouter plusieurs contrôles](../mfc/adding-multiple-controls.md)  
  
-   [Ajouter un contrôle ActiveX à une boîte de dialogue](../mfc/viewing-and-adding-activex-controls-to-a-dialog-box.md)  
  
-   [Modifier les propriétés d'un contrôle](../mfc/editing-control-properties.md)  
  
-   [Supprimer un contrôle](../mfc/deleting-a-control.md)  
  
-   [Afficher ou masquer la fenêtre Boîte à outils](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md)  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.* Pour obtenir des informations sur l'ajout de fichiers de ressources aux projets managés, l'accès aux ressources, l'affichage de ressources statiques et l'assignation de chaînes de ressources aux propriétés, et ce manuellement, consultez [Procédure pas à pas : localisation des applications Windows Forms](http://msdn.microsoft.com/fr-fr/9a96220d-a19b-4de0-9f48-01e5d82679e5) et [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
