# Toolbar (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La barre d'outils **Éditeur d'images** contient les outils pour dessiner, peindre, entrer du texte, effacer et manipuler des affichages.  Elle contient également un sélecteur d'options, qui vous permet de sélectionner des options pour chaque outil.  Par exemple, vous pouvez choisir différentes largeurs de pinceau, des facteurs d'agrandissement et des styles de ligne.  
  
> [!NOTE]
>  Tous les outils disponibles dans la barre d'outils **Éditeur d'images** le sont aussi dans le menu **Image** \(sous la commande **Outils**\).  
  
 ![Barre d'outils Éditeur d'images](../mfc/media/vcimageeditortoolbar.png "vcImageEditorToolbar")  
Barre d'outils Éditeur d'images  
  
 Pour utiliser la barre d'outils **Éditeur d'images** et le sélecteur **Option**, cliquez sur l'outil ou l'option souhaité.  
  
> [!TIP]
>  Les info\-bulles s'affichent lorsque vous passez votre curseur sur un bouton de barre d'outils.  Ces info\-bulles peuvent vous aider à identifier la fonction de chaque bouton.  
  
 Avec le sélecteur **Option** vous pouvez spécifier la largeur d'une ligne, d'un trait de pinceau, etc.  L'icône sur le bouton du sélecteur **Option** change selon l'outil que vous avez sélectionné.  
  
 ![Sélecteur rectangulaire dans la barre d'outils Éditeur d'images](../mfc/media/vcimageeditortoolbaroptionselector.png "vcImageEditorToolbarOptionSelector")  
Sélecteur d'options dans la barre d'outils Éditeur d'images
