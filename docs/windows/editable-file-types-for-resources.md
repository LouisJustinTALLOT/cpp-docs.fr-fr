# Editable File Types for Resources
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Vous pouvez ouvrir les types de fichier suivants, et modifier les ressources qu'ils contiennent.  
  
|Nom de fichier|Description|  
|--------------------|-----------------|  
|.rc|Fichiers de script de ressources.|  
|.rct|Fichiers modèles de ressources.|  
|.res|Fichiers de ressources.|  
|.resx|Fichiers de ressources managées.|  
|.exe|Fichiers exécutables.  Modifiables uniquement sur les systèmes d'exploitation Windows 2000 et versions ultérieures.|  
|.dll|Fichiers bibliothèques de liens dynamiques.  Modifiables uniquement sur les systèmes d'exploitation Windows 2000 et versions ultérieures.|  
|.bmp, .ico, .dib et .cur|Fichiers bitmap, icône, barre d'outils et curseur.|  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.* Pour plus d'informations sur l'ajout de fichiers de ressources aux projets managés, l'accès aux ressources, l'affichage de ressources statiques et l'assignation de chaînes de ressources aux propriétés, et ce manuellement, voir [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
