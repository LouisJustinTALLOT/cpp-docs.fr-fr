# Module::ReleaseNotifier::Release
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Supprime l'objet Module::ReleaseNotifier actuel si l'objet a été construit avec un paramètre à `true`.  
  
## Syntaxe  
  
```cpp  
void Release() throw();  
```  
  
## Configuration requise  
 **En\-tête:** module.h  
  
 **Espace de noms:** Microsoft::WRL
