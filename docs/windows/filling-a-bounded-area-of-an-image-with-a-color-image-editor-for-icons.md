# <a name="filling-a-bounded-area-of-an-image-with-a-color-image-editor-for-icons"></a>Remplissage avec une couleur d'une zone délimitée d'une image (Éditeur d'images pour les icônes)
L’éditeur d’images fournit le **remplir** outil pour remplir les placée entre la zone d’image avec la couleur de dessin actuelle ou la couleur d’arrière-plan actuelle.  
  
> [!TIP]
>  Info-bulles s’affichent lorsque vous pointez votre curseur sur un bouton de barre d’outils. Ces conseils peuvent vous aider à identifier la fonction de chaque bouton.  
  
### <a name="to-use-the-fill-tool"></a>Pour utiliser l’outil de remplissage  
  
1.  Sur le **Éditeur d’images** barre d’outils (ou le **Image** menu, **outils** commande), cliquez sur le **remplir** outil.  
  
2.  Si nécessaire, sélectionnez les couleurs de dessin : dans le [palette de couleurs](../windows/colors-window-image-editor-for-icons.md), cliquez sur le bouton gauche de la souris pour sélectionner une couleur de premier plan ou le bouton droit de la souris pour sélectionner une couleur d’arrière-plan.  
  
3.  Déplacer le **remplissage** outil à la zone que vous souhaitez remplir.  
  
4.  Cliquez sur le bouton gauche ou droit de la souris pour remplir avec la couleur de premier plan ou la couleur d’arrière-plan, respectivement.
