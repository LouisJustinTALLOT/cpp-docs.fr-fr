# Attributed Programming Concepts
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Visual C\+\+ inclut divers matériel pour vous aider à utiliser des attributs dans vos programmes.  Les attributs, une nouvelle fonctionnalité de Visual C\+\+, sont conçus pour simplifier la programmation COM et le développement du common langage runtime du .NET Framework.  Lorsque vous incorporez des attributs dans vos fichiers sources, le compilateur avec des bibliothèques de liens dynamiques de fournisseur \(DLL\) pour insérer le code ou pour modifier le code dans les fichiers objets générés.  Il existe des attributs qui facilitent la création de fichiers .idl, des interfaces, des bibliothèques de types, et d'autres éléments de COM.  Dans l'environnement de \(IDE\) développement intégré, les attributs sont pris en charge par les assistants et par la fenêtre Propriétés.  
  
 Alors que les attributs élimine une partie de la programmation détaillée nécessaire pour écrire des objets COM, vous devez avoir une certaine expérience dans [Notions de base COM](http://msdn.microsoft.com/library/windows/desktop/ms694363) à mieux les utilisez.
