# Changing Unassigned Symbols
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Dans la [boîte de dialogue Symboles des ressources](../windows/resource-symbols-dialog-box.md), vous pouvez modifier ou supprimer les symboles existants qui ne sont pas déjà assignés à une ressource ou un objet.  
  
### Pour modifier un symbole non assigné  
  
1.  Dans la zone **Nom**, sélectionnez le symbole non assigné, puis cliquez sur **Modifier**.  
  
2.  Modifiez le nom ou la valeur du symbole dans les zones appropriées de la boîte de dialogue **Changer de symbole**.  
  
    > [!NOTE]
    >  Pour changer un symbole qui *est* assigné à une ressource ou un objet, vous devez utiliser l'éditeur de ressources ou la fenêtre Propriétés.  Pour plus d'informations, voir [Modification d'un symbole ou d'un nom de symbole](../windows/changing-a-symbol-or-symbol-name-id.md).  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.*
