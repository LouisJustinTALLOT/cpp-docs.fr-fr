# Menu Editor
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les menus permettent d’organiser les commandes de manière logique et de les trouver plus facilement. Avec l’Éditeur de menus, vous pouvez créer et modifier des menus en travaillant directement avec une barre de menus qui ressemble étroitement à celle de votre application terminée.  
  
 Pour les projets Visual C\+\+, vous pouvez utiliser l’Éditeur de menus pour :  
  
-   [Créer des menus](../windows/creating-a-menu.md)  
  
-   [Insérer un nouveau menu parmi des menus existants](../windows/inserting-a-new-menu-between-existing-menus.md)  
  
-   [Créer une commande de menu](../windows/adding-commands-to-a-menu.md)  
  
-   [Créer un menu de sous\-menu](../windows/creating-a-submenu.md)  
  
-   [Sélectionner des menus et des commandes de menu](../windows/selecting-multiple-menus-or-menu-commands.md)  
  
-   [Déplacer et copier des menus et des commandes de menu](../windows/moving-and-copying-menus-and-menu-commands.md)  
  
-   [Créer des menus contextuels](../windows/creating-pop-up-menus.md)  
  
-   [Connecter un menu contextuel à votre application](../windows/connecting-a-pop-up-menu-to-your-application.md)  
  
-   [Afficher une ressource de menu sous forme de menu contextuel](../windows/viewing-a-menu-as-a-pop-up-menu.md)  
  
-   [Associer une commande de menu à une touche accélérateur](../windows/associating-a-menu-command-with-an-accelerator-key.md)  
  
-   [Associer une commande de menu à du texte de barre d’état](../windows/associating-menu-commands-with-status-bar-text-in-mfc-applications.md)  
  
-   [Assignation de touches d'accès rapide à des commandes de menu](../windows/assigning-access-keys-to-menu-commands.md)  
  
-   [Modifier les propriétés d’une commande de menu](../windows/menu-command-properties.md)  
  
    > [!TIP]
    >  Quand vous utilisez l’Éditeur de menus, dans de nombreux cas, vous pouvez cliquer sur le bouton droit de la souris pour afficher un menu contextuel des commandes fréquemment utilisées. Les commandes disponibles varient selon la cible du pointeur.  
  
    > [!NOTE]
    >  Pour les programmes MFC \(Microsoft Foundation Class Library\) et ATL, vous pouvez utiliser les Assistants Code pour raccorder des commandes de menu au code. Pour plus d’informations, consultez [Ajout d’un événement](../ide/adding-an-event-visual-cpp.md).  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.* Pour obtenir des informations sur l’ajout de fichiers de ressources aux projets managés, l’accès aux ressources, l’affichage de ressources statiques et l’assignation de chaînes de ressources à des propriétés, et ceci manuellement, consultez [Procédure pas à pas : localisation des applications Windows Forms](http://msdn.microsoft.com/fr-fr/9a96220d-a19b-4de0-9f48-01e5d82679e5) et [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
