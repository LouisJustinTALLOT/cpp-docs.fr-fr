# <a name="menu-editor"></a>Éditeur de menus

Les menus permettent d’organiser les commandes de manière logique et de les trouver plus facilement. Avec le **Menu** éditeur, vous pouvez créer et modifier des menus en travaillant directement avec une barre de menus qui ressemble étroitement à celui de votre application terminée.

Pour les projets Visual C++, vous pouvez utiliser la **Menu** éditeur pour :

- [Créer des menus](../windows/creating-a-menu.md)

- [Insérer un nouveau menu parmi des menus existants](../windows/inserting-a-new-menu-between-existing-menus.md)

- [Créer une commande de menu](../windows/adding-commands-to-a-menu.md)

- [Créer un menu de sous-menu](../windows/creating-a-submenu.md)

- [Sélectionner des menus et des commandes de menu](../windows/selecting-multiple-menus-or-menu-commands.md)

- [Déplacer et copier des menus et des commandes de menu](../windows/moving-and-copying-menus-and-menu-commands.md)

- [Créer des menus contextuels](../windows/creating-pop-up-menus.md)

- [Connecter un menu contextuel à votre application](../windows/connecting-a-pop-up-menu-to-your-application.md)

- [Afficher une ressource de menu sous forme de menu contextuel](../windows/viewing-a-menu-as-a-pop-up-menu.md)

- [Associer une commande de menu à une touche accélérateur](../windows/associating-a-menu-command-with-an-accelerator-key.md)

- [Associer une commande de menu à du texte de barre d’état](../windows/associating-menu-commands-with-status-bar-text-in-mfc-applications.md)

- [Assignation de touches d'accès rapide à des commandes de menu](../windows/assigning-access-keys-to-menu-commands.md)

- [Modifier les propriétés d’une commande de menu](../windows/menu-command-properties.md)

   > [!TIP]
   > Lors de l’utilisation du **Menu** éditeur, dans de nombreux cas, vous pouvez cliquer sur le bouton droit de la souris pour afficher un menu contextuel des commandes fréquemment utilisées. Les commandes disponibles varient selon la cible du pointeur.

   > [!NOTE]
   > Pour les programmes MFC Microsoft Foundation Class Library () et ATL, vous pouvez utiliser **Assistants Code** pour raccorder des commandes de menu au code. Pour plus d’informations, consultez [Ajout d’un événement](../ide/adding-an-event-visual-cpp.md).

Pour plus d’informations sur l’ajout de ressources aux projets managés, consultez [Resources in Desktop Apps](/dotnet/framework/resources/index) dans le *Guide du développeur .NET Framework*. Pour plus d’informations sur l’ajout manuel de fichiers de ressources aux projets managés, l’accès aux ressources, affichage de ressources statiques et l’assignation de chaînes de ressources aux propriétés, consultez [création des fichiers de ressources pour les applications de bureau](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Pour plus d’informations sur la globalisation et localisation de ressources dans les applications gérées, consultez [globalisation et localisation d’Applications .NET Framework](/dotnet/standard/globalization-localization/index).

## <a name="requirements"></a>Configuration requise

Win32
