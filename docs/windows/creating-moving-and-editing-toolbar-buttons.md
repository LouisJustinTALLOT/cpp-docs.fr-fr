# Creating, Moving, and Editing Toolbar Buttons
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Vous pouvez facilement créer, déplacer, copier et modifier des boutons de barre d'outils.  
  
 Par défaut, un nouveau bouton vide s'affiche à l'extrémité droite de la barre d'outils.  Vous pouvez déplacer ce bouton avant de le modifier.  Lorsque vous créez un nouveau bouton, un autre bouton vide s'affiche à droite du bouton modifié.  Lorsque vous enregistrez une barre d'outils, le bouton vide n'est pas enregistré.  
  
-   [Créer un nouveau bouton de barre d'outils](../mfc/creating-a-new-toolbar-button.md)  
  
-   [Déplacer un bouton de barre d'outils](../mfc/moving-a-toolbar-button.md)  
  
-   [Copier des boutons d'une barre d'outils](../mfc/copying-buttons-from-a-toolbar.md)  
  
-   [Supprimer un bouton de barre d'outils](../mfc/deleting-a-toolbar-button.md)  
  
-   [Insérer un espace entre des boutons sur une barre d'outils](../mfc/inserting-a-space-between-buttons-on-a-toolbar.md)  
  
-   [Supprimer un espace entre les boutons sur une barre d'outils](../mfc/removing-space-between-buttons-on-a-toolbar.md)  
  
-   [Modifier la page de propriétés d'une barre d'outils](../mfc/changing-the-properties-of-a-toolbar-button.md)  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.* Pour obtenir des informations sur l'ajout de fichiers de ressources aux projets managés, l'accès aux ressources, l'affichage de ressources statiques et l'assignation de chaînes de ressources aux propriétés, et ce manuellement, consultez [Procédure pas à pas : localisation des applications Windows Forms](http://msdn.microsoft.com/fr-fr/9a96220d-a19b-4de0-9f48-01e5d82679e5) et [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
