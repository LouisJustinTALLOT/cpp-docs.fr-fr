# <a name="creating-moving-and-editing-toolbar-buttons"></a>Création, déplacement et modification de boutons de barre d'outils

Vous pouvez facilement créer, déplacer, copier et modifier des boutons de barre d’outils.

Par défaut, un nouveau bouton vide s’affiche à l’extrémité droite de la barre d’outils. Vous pouvez déplacer ce bouton avant de le modifier. Lorsque vous créez un nouveau bouton, un autre bouton vide s’affiche à droite du bouton modifié. Lorsque vous enregistrez une barre d’outils, le bouton vide n’est pas enregistré.

- [Créer un nouveau bouton de barre d’outils](creating-a-new-toolbar-button.md)

- [Déplacement d’un bouton de barre d’outils](moving-a-toolbar-button.md)

- [Boutons de copie à partir d’une ressource de barre d’outils](../windows/copying-buttons-from-a-toolbar.md)

- [Supprimer un bouton de barre d’outils](deleting-a-toolbar-button.md)

- [Insérer un espace entre les boutons sur une ressource de barre d’outils](../windows/inserting-a-space-between-buttons-on-a-toolbar.md)

- [Fermer un espace entre les boutons sur une barre d’outils](../windows/removing-space-between-buttons-on-a-toolbar.md)

- [Modifier la page de propriétés d’une barre d’outils](../windows/changing-the-properties-of-a-toolbar-button.md)

Pour plus d’informations sur l’ajout de ressources aux projets managés, consultez [Resources in Desktop Apps](/dotnet/framework/resources/index) dans le *Guide du développeur .NET Framework*. Pour plus d’informations sur l’ajout manuel de fichiers de ressources aux projets managés, l’accès aux ressources, affichage de ressources statiques et l’assignation de chaînes de ressources aux propriétés, consultez [création des fichiers de ressources pour les applications de bureau](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Pour plus d’informations sur la globalisation et localisation de ressources dans les applications gérées, consultez [globalisation et localisation d’Applications .NET Framework](/dotnet/standard/globalization-localization/index).

## <a name="requirements"></a>Configuration requise

MFC ou ATL
