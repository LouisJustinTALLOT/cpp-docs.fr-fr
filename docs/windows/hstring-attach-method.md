# <a name="hstringattach-method"></a>HString::Attach, méthode

Associe les **HString** objet actuelle **HString** objet.

## <a name="syntax"></a>Syntaxe

```cpp
void Attach(
       HSTRING hstr
       ) throw()  
```

### <a name="parameters"></a>Paramètres

*HSTR*  
Un existant **HString** objet.
