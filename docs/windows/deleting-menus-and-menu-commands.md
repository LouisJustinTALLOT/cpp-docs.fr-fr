# Deleting Menus and Menu Commands
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Pour supprimer un menu ou une commande de menu  
  
1.  Cliquez avec le bouton droit sur le nom ou la commande de menu.  
  
2.  Choisissez **Supprimer** dans le menu contextuel.  
  
    > [!NOTE]
    >  De même, vous pouvez utiliser le menu contextuel pour effectuer d'autres actions telles que Copier, Couper, Coller, Insérer nouveau, Insérer un séparateur, Modifier ID, Afficher comme Popup, Vérifier les mnémoniques, etc.  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.*
