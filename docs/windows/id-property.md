# ID Property
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La propriété ID référence chaque entrée de la table d'accélérateurs dans le code du programme.  Il s'agit de la valeur de la commande que le programme reçoit lorsqu'un utilisateur appuie sur la touche accélérateur ou une combinaison de touches.  Pour qu'un accélérateur soit identique à un élément de menu, leurs ID doivent être identiques \(tant que l'ID de la table d'accélérateurs est identique à l'ID de la ressource de menu\).  
  
## Configuration requise  
 Win32
