# Attributes by Group
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les attributs C\+\+ sont organisés dans des groupes fonctionnels suivants.  
  
|Attribut|Description|  
|--------------|-----------------|  
|[COM Attributes](../windows/com-attributes.md)|Injecter du code pour prendre en charge de nombreux domaines de développement COM et le développement du common langage runtime du.NET Framework.|  
|[IDL Attributes](../windows/idl-attributes.md)|Activez la modification du fichier .idl à partir d'un fichier de code source sans utiliser d'un assistant et sans connaissance de la structure et la syntaxe de ce fichier.|  
|[Attributs du consommateur OLE DB](../windows/ole-db-consumer-attributes.md)|Inject code est basé sur [modèles du consommateur OLE DB](../data/oledb/ole-db-consumer-templates-reference.md)— pour créer un consommateur OLE DB travaillant qui effectue des tâches telles que des tableaux d'ouverture, l'exécution des données de commandes, puis d'accès.|  
|[Attributs du compilateur](../windows/compiler-attributes.md)|Fourni par le compilateur Visual C\+\+.|  
  
 Pour plus d'informations sur la définition des attributs personnalisés pour votre application. NET Framework, consultez l' [User\-Defined Attributes](../windows/user-defined-attributes-cpp-component-extensions.md).
