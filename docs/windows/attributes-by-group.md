# <a name="attributes-by-group"></a>Attributs par groupe
Les attributs C++ sont organisés en groupes fonctionnels suivants.  
  
|Attribut|Description|  
|---------------|-----------------|  
|[Attributs COM](../windows/com-attributes.md)|Injecter du code pour prendre en charge plusieurs domaines de développement COM et le développement du common language runtime du .NET Framework.|  
|[Attributs IDL](../windows/idl-attributes.md)|Activer la modification du fichier .idl à partir d’un fichier de code source sans l’utilisation d’un Assistant et sans connaissance de la structure et la syntaxe de ce fichier.|  
|[Attributs du consommateur OLE DB](../windows/ole-db-consumer-attributes.md)|Injecter du code, selon la [modèles du consommateur OLE DB](../data/oledb/ole-db-consumer-templates-reference.md): pour créer un travail consommateur OLE DB qui effectue des tâches telles que les tables de l’ouverture, l’exécution de commandes et l’accès aux données.|  
|[Attributs de compilateur](../windows/compiler-attributes.md)|Fourni par le compilateur Visual C++.|  
  
 Pour plus d’informations sur la façon de définir des attributs personnalisés pour votre application .NET Framework, consultez [les attributs définis par l’utilisateur](../windows/user-defined-attributes-cpp-component-extensions.md).
