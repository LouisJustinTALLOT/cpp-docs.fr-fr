# Cr&#233;ation d’une application de bureau Windows vide
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Pour créer une application de bureau Windows vide  
  
1.  Dans le menu **Fichier**, sélectionnez **Nouveau, Projet**.  
  
2.  Dans le volet gauche de la boîte de dialogue **Nouveau projet**, cliquez sur **Win32** puis, dans le volet central, sélectionnez **Application console Win32**.  
  
3.  Entrez un nom pour votre nouveau projet, un chemin d’accès au répertoire du projet, puis cliquez **OK**.  
  
4.  Dans l’[Assistant Application Win32](../windows/win32-application-wizard.md), cliquez sur la page **Paramètres de l’application**. Sélectionnez le **Type d’application** que vous souhaitez créer avec votre fichier de code source, puis cochez la case **Projet vide** sous **Options supplémentaires**.  
  
5.  Cliquez sur **OK**.  
  
 Le projet s’affiche dans l’**Explorateur de solutions** avec trois répertoires destinés à contenir les fichiers sources, les fichiers d’en\-têtes et les fichiers de ressources.  
  
 Ensuite, vous pouvez [ajouter des fichiers à votre projet Visual C\+\+ vide](../windows/adding-files-to-an-empty-win32-applications.md).
