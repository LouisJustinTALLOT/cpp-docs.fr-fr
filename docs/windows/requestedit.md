# requestedit
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

indique que la propriété prend en charge la notification d' **OnRequestEdit** .  
  
## Syntaxe  
  
```  
  
[requestedit]  
  
```  
  
## Notes  
 L'attribut de **requestedit** C\+\+ a les mêmes fonctionnalités que l'attribut de [requestedit](http://msdn.microsoft.com/library/windows/desktop/aa367155) MIDL.  
  
## Exemple  
 Consultez l'exemple pour [pouvant être liée](../windows/bindable.md) pour un usage d'exemple de **requestedit**.
