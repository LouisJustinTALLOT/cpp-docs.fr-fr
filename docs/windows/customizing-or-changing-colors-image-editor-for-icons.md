# Customizing or Changing Colors (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La [palette de couleurs](../windows/colors-window-image-editor-for-icons.md) de l'éditeur d'images affiche initialement 16 couleurs standard.  En plus des couleurs affichées, vous pouvez créer vos couleurs personnalisées.  Vous pouvez ensuite [enregistrer et charger une palette de couleurs personnalisées](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md).  
  
### Pour changer les couleurs de la palette de couleurs  
  
1.  Dans le menu **Image**, choisissez **Affiner les couleurs**.  
  
2.  Dans la [boîte de dialogue Affiner les couleurs](../windows/custom-color-selector-dialog-box-image-editor-for-icons.md), définissez la couleur en tapant les valeurs RVB ou TSL dans les zones de texte appropriées, ou choisissez une couleur dans la zone **Affichage des dégradés de couleur**.  
  
3.  Définissez la luminosité en déplaçant le curseur sur la barre **Luminosité**.  
  
4.  De nombreuses couleurs personnalisées sont tramées.  Si vous souhaitez la couleur unie la plus proche de la couleur tramée, double\-cliquez sur la zone **Couleur**.  
  
     Si vous optez plus tard pour la couleur tramée, déplacez le curseur sur la barre **Luminosité**, ou déplacez le réticule dans la zone **Affichage des dégradés de couleur** pour rétablir le tramage.  
  
5.  Cliquez sur **OK** pour ajouter la nouvelle couleur.  
  
 Pour plus d'informations sur l'ajout de ressources aux projets managés, consultez [Ressources dans les applications](../Topic/Resources%20in%20Desktop%20Apps.md) dans le *Guide du développeur .NET Framework.* Pour plus d'informations sur l'ajout de fichiers de ressources aux projets managés, l'accès aux ressources, l'affichage de ressources statiques et l'assignation de chaînes de ressources aux propriétés, et ce manuellement, voir [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
