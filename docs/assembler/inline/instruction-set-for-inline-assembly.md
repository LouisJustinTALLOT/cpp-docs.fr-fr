# Instruction set pour un assembly inline
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

## Section spécifique à Microsoft  
 Le compilateur Visual C\+\+ prend en charge tous les opcodes via les processeurs Pentium 4 et AMD Athlon.  Il est possible de créer des instructions supplémentaires prises en charge par le processeur cible au moyen de la [pseudo\-instruction \_emit](../../assembler/inline/emit-pseudoinstruction.md).  
  
 **FIN de la section spécifique à Microsoft**
