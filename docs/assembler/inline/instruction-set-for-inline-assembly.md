# <a name="instruction-set-for-inline-assembly"></a>Jeu d'instructions pour l'assembly inline

**Section spécifique à Microsoft**

Le compilateur Visual C++ prend en charge tous les opcodes via les processeurs Pentium 4 et AMD Athlon. Instructions supplémentaires prises en charge par le processeur cible peuvent être créées avec le [pseudo-instruction _emit](../../assembler/inline/emit-pseudoinstruction.md).

**FIN de la section spécifique à Microsoft**
