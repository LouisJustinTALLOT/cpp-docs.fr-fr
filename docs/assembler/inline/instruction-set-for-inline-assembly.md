# <a name="instruction-set-for-inline-assembly"></a>Jeu d'instructions pour l'assembly inline

**Spécifique à Microsoft**

Le compilateur Microsoft C++ prend en charge tous les OpCodes via les processeurs Pentium 4 et AMD Athlon. Des instructions supplémentaires prises en charge par le processeur cible peuvent être créées avec le [_Emit Emit](../../assembler/inline/emit-pseudoinstruction.md).
