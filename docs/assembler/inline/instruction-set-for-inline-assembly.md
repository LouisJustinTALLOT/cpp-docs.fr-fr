# <a name="instruction-set-for-inline-assembly"></a>Jeu d'instructions pour l'assembly inline
## <a name="microsoft-specific"></a>Section spécifique à Microsoft  
 Le compilateur Visual C++ prend en charge tous les opcodes via les processeurs Pentium 4 et AMD Athlon. Des instructions supplémentaires prises en charge par le processeur cible peuvent être créées avec la [pseudo-instruction _emit](../../assembler/inline/emit-pseudoinstruction.md).
