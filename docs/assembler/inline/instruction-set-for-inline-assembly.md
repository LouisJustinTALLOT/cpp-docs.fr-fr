# <a name="instruction-set-for-inline-assembly"></a>Jeu d'instructions pour l'assembly inline

**Section spécifique de Microsoft**

Le compilateur C++ Microsoft prend en charge tous les OpCodes via les processeurs Pentium 4 et AMD Athlon. Des instructions supplémentaires prises en charge par le processeur cible peuvent être créées avec le [_Emit Emit](../../assembler/inline/emit-pseudoinstruction.md).

**Fin de la section spécifique de Microsoft**
