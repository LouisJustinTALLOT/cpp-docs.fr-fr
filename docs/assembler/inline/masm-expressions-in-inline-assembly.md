# <a name="masm-expressions-in-inline-assembly"></a>Expressions MASM dans l'assembly inline

**Section spécifique de Microsoft**

Le code assembleur inline peut utiliser n'importe quelle expression MASM, qui est une combinaison d'opérandes et d'opérateurs qui correspond à une valeur ou à une adresse unique.

**Fin de la section spécifique de Microsoft**
