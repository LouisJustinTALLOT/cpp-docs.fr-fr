# <a name="while-32-bit-masm"></a>. WHILe (MASM 32 bits)

Génère du code qui exécute le bloc d' *instructions* lorsque la *condition* reste true. (uniquement MASM 32 bits.)

## <a name="syntax"></a>Syntaxe

> *instructions*\
