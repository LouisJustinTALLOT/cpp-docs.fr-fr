# <a name="while"></a>.WHILE
Génère du code qui exécute le bloc de `statements` tandis que `condition` conserve la valeur true.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
   .WHILE condition  
statements  
.ENDW  
```
