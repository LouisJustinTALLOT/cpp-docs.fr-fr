# <a name="while-32-bit-masm"></a>. WHILe (MASM 32 bits)

Génère du code qui exécute le bloc de `statements` alors que `condition` reste true. (uniquement MASM 32 bits.)

## <a name="syntax"></a>Syntaxe

> . Condition WHILe<br/>
> instructions<br/>
> .ENDW
