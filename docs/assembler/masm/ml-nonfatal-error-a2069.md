# ML Nonfatal Error A2069
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**aucun opérandes autorisée pour cette instruction**  
  
 Un ou plusieurs opérandes ont été spécifiés avec une instruction qui ne prend pas d'opérandes.
