# <a name="ml-nonfatal-error-a2097"></a>Erreur ML non fatale A2097
**segment attendu : identificateur**  
  
 Le [groupe](../../assembler/masm/group.md) directive a été attribuée à un identificateur qui n’était pas un segment défini.
