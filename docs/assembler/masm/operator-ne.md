# operator NE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retourne la valeur true \(\- 1\) si *expression1* n'a pas *expression2*, ou retourne la valeur false \(0\) cas échéant.  
  
## Syntaxe  
  
```  
  
expression1 NE expression2  
```
