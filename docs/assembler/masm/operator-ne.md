# <a name="operator-ne"></a>NE, opérateur

Retourne true (-1) si *expression1* n’est pas égal à *Expression2*, ou retourne false (0) dans le cas contraire.

## <a name="syntax"></a>Syntaxe

> *expression1* ne *Expression2*
