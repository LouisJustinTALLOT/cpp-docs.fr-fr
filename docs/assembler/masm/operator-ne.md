# <a name="operator-ne"></a>NE, opérateur

Retourne la valeur true (-1) si *expression1* n’est pas égal *expression2*, ou retourne la valeur false (0) si c’est le cas.

## <a name="syntax"></a>Syntaxe

> expression2 NE expression1
