# <a name="operator-lowword"></a>LOWWORD, opérateur

Retourne les 16 bits de poids faibles de *expression*. Expressions MASM sont des valeurs 64 bits.

## <a name="syntax"></a>Syntaxe

> Expression de LOWWORD
