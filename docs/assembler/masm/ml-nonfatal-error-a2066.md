# <a name="ml-nonfatal-error-a2066"></a>Erreur ML non fatale A2066
**incompatible taille de mode et le segment de l’UC**  
  
 Une tentative a été effectuée pour ouvrir un segment avec un **USE16**, **USE32**, ou **plat** attribut qui n’est pas compatible avec l’UC spécifié, ou pour modifier un processeur de 16 bits en 32 bits segment.  
  
 Le **USE32** et **plat** attributs doivent être précédés par le.386 ou directive de processeur supérieure.
