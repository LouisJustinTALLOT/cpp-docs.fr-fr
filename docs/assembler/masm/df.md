# <a name="df"></a>DF

Alloue et initialise éventuellement 6 octets de stockage pour chaque *initialiseur*. **DF** est un synonyme de [FWORD](fword.md).

## <a name="syntax"></a>Syntaxe

> ⟦*nom*de l' *initialiseur* ⟧ **DF** ⟦ __,__ *initialiseur* ... ⟧

## <a name="remarks"></a>Notes

Peut également être utilisé comme spécificateur de type partout où un type est légal.
