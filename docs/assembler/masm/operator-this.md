# <a name="operator-this"></a>THIS, opérateur
Retourne un opérande de spécifié `type` dont les valeurs offset et du segment sont égales à la valeur de compteur d’emplacement actuel.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
THIS type  
```
