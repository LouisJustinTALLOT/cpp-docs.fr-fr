# <a name="operator-this"></a>THIS, opérateur

Retourne un opérande de *type* spécifié dont les valeurs de décalage et de segment sont égales à la valeur de compteur d’emplacement actuelle.

## <a name="syntax"></a>Syntaxe

> **Ce** *type*
