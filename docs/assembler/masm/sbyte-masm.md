# <a name="sbyte-masm"></a>SBYTE (MASM)
Alloue et éventuellement Initialise un octet signé de stockage pour chaque `initializer`. Peut également être utilisé comme un spécificateur de type n’importe où un type est autorisé.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name SBYTEinitializer [[, initializer]]...  
```
