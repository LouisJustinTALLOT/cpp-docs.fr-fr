# SBYTE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

alloue et initialise éventuellement un octet signé de stockage pour chaque `initializer`.  Peut également être utilisé comme un spécificateur de type partout où un type est conforme.  
  
## Syntaxe  
  
```  
  
name SBYTE initializer [[, initializer]]...  
```
