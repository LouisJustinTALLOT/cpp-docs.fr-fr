# <a name="ml-warning-a4012"></a>Avertissement ML A4012

**informations sur le numéro de ligne pour le segment sans classe’CODE'**

Des instructions s’affichent dans un segment qui n’a pas de nom de classe qui se termine par « CODE ». L’assembleur n’a pas généré d’informations CodeView pour ces instructions.

CodeView ne peut pas traiter les modules contenant du code dans des segments avec des noms de classe qui ne se terminent pas par « CODE ».
