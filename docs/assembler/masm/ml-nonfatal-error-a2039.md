# ML Nonfatal Error A2039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**ligne trop long**  
  
 Une ligne de fichier source a dépassé la limite de 512 caractères.  
  
 Si plusieurs Lignes physiques sont concaténées avec le caractère de continuation de ligne \(\\\), la Ligne logique résultant est encore limitée à 512 caractères.
