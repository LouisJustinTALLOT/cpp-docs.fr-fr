# <a name="operator-"></a>opérateur :
Remplace le segment par défaut de *expression* avec `segment`. Le `segment` peut être un Registre de segment, nom du groupe, nom de segment ou expression de segment. Le *expression* doit être une constante.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
segment  
: expression  
```
