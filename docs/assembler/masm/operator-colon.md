# operator :
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

substitue le segment par défaut de l' *expression* avec `segment`.  `segment` peut être un registre de segment, un nom de groupe, un nom de segment, ou une expression de segment.  *l'expression* doit être une constante.  
  
## Syntaxe  
  
```  
  
segment  
: expression  
```
