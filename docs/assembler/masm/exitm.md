# EXITM
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Termine l'expansion de la répétition actuelle ou de macro bloc et commence l'assembly de l'instruction suivante en dehors de le bloc.  
  
## Syntaxe  
  
```  
  
EXITM [[textitem]]   
```  
  
## Notes  
 dans une fonction macro, le *textitem* est la valeur retournée.
