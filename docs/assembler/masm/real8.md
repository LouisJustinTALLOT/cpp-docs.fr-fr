# <a name="real8"></a>REAL8
Alloue et éventuellement Initialise un nombre à virgule flottante de double précision (8 octets) pour chaque *initialiseur*.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name REAL8 initializer [[, initializer]]...  
```
