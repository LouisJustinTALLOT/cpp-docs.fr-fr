# <a name="real8"></a>REAL8

Alloue et initialise éventuellement un nombre à virgule flottante de double précision (8 octets) pour chaque *initialiseur*.

## <a name="syntax"></a>Syntaxe

> initialiseur de REAL8 nom [[, initialiseur]]...
