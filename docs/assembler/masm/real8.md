# <a name="real8"></a>REAL8

Alloue et initialise éventuellement un nombre à virgule flottante double précision (8 octets) pour chaque *initialiseur*.

## <a name="syntax"></a>Syntaxe

> *nom* **REAL8** *initializer* ⟦ __,__ *initializer* ... ⟧
