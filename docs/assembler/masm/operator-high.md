# <a name="operator-high"></a>HIGH, opérateur

Retourne les 8 bits de poids fort des 16 bits de poids faible de l' *expression*. Les expressions MASM sont des valeurs 64 bits.

## <a name="syntax"></a>Syntaxe

> *Expression* haute
