# <a name="operator-high"></a>HIGH, opérateur

Retourne les 8 bits de poids fort de 16 bits de poids faibles de *expression*. Expressions MASM sont des valeurs 64 bits.

## <a name="syntax"></a>Syntaxe

> Expression élevée
