# <a name="struct-masm"></a>STRUCT (MASM)
Déclare un type de structure ayant spécifié *fielddeclarations*. Chaque champ doit être une définition de données valide. Identique à [STRUC](../../assembler/masm/struc.md).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
   name STRUCT [[alignment]] [[, NONUNIQUE]]  
fielddeclarations  
name ENDS  
```
