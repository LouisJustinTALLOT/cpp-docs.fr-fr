# .CONTINUE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Génère du code pour accéder au début d'un bloc de [.WHILE](../../assembler/masm/dot-while.md) ou de [.REPEAT](../../assembler/masm/dot-repeat.md) si `condition` est true.  
  
## Syntaxe  
  
```  
  
.CONTINUE [[.IF condition]]  
```
