# <a name="real10"></a>REAL10
Alloue et éventuellement Initialise un nombre à virgule flottante de 10 octets pour chaque *initialiseur*.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name REAL10 initializer [[, initializer]]...  
```
