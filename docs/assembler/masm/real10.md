# <a name="real10"></a>REAL10

Alloue et initialise éventuellement un nombre à virgule flottante de 10 octets pour chaque *initialiseur*.

## <a name="syntax"></a>Syntaxe

> *nom* **REAL10** *initializer* ⟦ __,__ *initializer* ... ⟧
