# <a name="stack-32-bit-masm"></a>. STACK (MASM 32 bits)

En cas d’utilisation avec [. MODEL](../../assembler/masm/dot-model.md), définit un segment de pile (avec la pile de noms de segment). Le `size` facultatif spécifie le nombre d’octets pour la pile (valeur par défaut 1 024). La directive `.STACK` ferme automatiquement l’instruction Stack. (uniquement MASM 32 bits.)

## <a name="syntax"></a>Syntaxe

> . PILE [[Taille]]
