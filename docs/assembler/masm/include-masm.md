# INCLUDE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Insère le code source du fichier source donné par *le nom de fichier* dans le fichier source actuel pendant l'assembly.  
  
## Syntaxe  
  
```  
  
INCLUDE filename  
```  
  
## Notes  
 *Le nom de fichier* doit être placé entre crochets pointus s'il comprend une barre oblique inverse, point\-virgule, supériorité le symbole, infériorité le symbole, le guillemet simple, ou le guillemet double.
