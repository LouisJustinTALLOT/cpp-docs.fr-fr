# operator SHORT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

définit le type de *nom au* short.  Tous les passe *à les étiqueter* doivent être courts \(dans la plage \- 128 à \+127 de l'instruction de saut *de étiqueter*\).  
  
## Syntaxe  
  
```  
  
SHORT label  
```
