# <a name="operator-short"></a>opérateur SHORT

Définit le type de *étiquette* en type short. Permet d’atteindre tous les *étiquette* doit être court (comprise entre -128 à + 127 octets à partir de l’instruction de saut et *étiquette*).

## <a name="syntax"></a>Syntaxe

> Étiquette courts
