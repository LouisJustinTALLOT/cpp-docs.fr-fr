# <a name="ml-nonfatal-error-a2189"></a>Erreur ML non fatale A2189

**combinaison non valide avec alignement de segment**

L’alignement spécifié par la directive **align** ou `EVEN` était supérieur à l’alignement du segment actuel tel que spécifié par la directive **segment** .
