# <a name="ml-nonfatal-error-a2189"></a>Erreur ML non fatale A2189

**combinaison non valide avec l’alignement de segment**

L’alignement spécifié par le **ALIGN** ou `EVEN` directive était supérieure à l’alignement du segment actuel tel que spécifié par le **SEGMENT** directive.
