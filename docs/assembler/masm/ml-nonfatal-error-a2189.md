# <a name="ml-nonfatal-error-a2189"></a>Erreur ML non fatale A2189

**combinaison non valide avec alignement de segment**

L’alignement spécifié par l' **alignement ou la** `EVEN` directive était supérieur à l’alignement du segment actuel tel que spécifié par la directive du **segment** .
