# <a name="ml-nonfatal-error-a2119"></a>Erreur ML non fatale A2119

**le type de langage doit être spécifié**

Aucun type de langage n’a été attribué à une définition de procédure ou un prototype.

Un type de langage doit être déclaré dans chaque définition de procédure ou prototype si aucun type de langue par défaut n’est spécifié. Un type de langue par défaut est défini à l’aide de [. ](dot-model.md)Directive de modèle, **option lang** **ou options de** ligne de commande de l’ml **.**
