# ML Nonfatal Error A2119
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**le type de langage doit être spécifié**  
  
 Une définition ou un prototype de la procédure n'a pas été attribué à un type de langage.  
  
 un type de langage doit être déclaré dans chaque définition ou prototype de procédure si un type de langue par défaut n'est pas spécifié.  Un type de langue par défaut est défini à l'aide de la directive de [.MODEL](../../assembler/masm/dot-model.md) , **LANG D'OPTION**, ou les options de ligne de commande **\/Gc** ou **\/Gd**de ML.
