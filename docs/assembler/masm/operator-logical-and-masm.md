# <a name="operator-ampamp-masm"></a>opérateur &amp; &amp; (MASM)
Remplace `parameter` avec sa valeur de l’argument correspondant.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
&  
parameter  
&  
  
```
