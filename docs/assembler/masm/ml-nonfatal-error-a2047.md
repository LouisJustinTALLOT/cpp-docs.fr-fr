# ML Nonfatal Error A2047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**chaîne \(null\) vide**  
  
 Une chaîne qui se compose d'une paire la délimitation de guillemets et d'aucun caractères dans.  
  
 Pour qu'une chaîne soit valide, elle doit contenir 1\-255 caractères.
