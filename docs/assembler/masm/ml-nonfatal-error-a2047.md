# <a name="ml-nonfatal-error-a2047"></a>Erreur ML non fatale A2047

**chaîne vide (null)**

Chaîne constituée d’une paire délimitant les guillemets et sans caractère dans.

Pour qu’une chaîne soit valide, elle doit contenir 1-255 caractères.
