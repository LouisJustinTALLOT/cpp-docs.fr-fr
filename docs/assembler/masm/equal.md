# <a name=""></a>=

Affecte la valeur numérique de l' *expression* à *Name*.

## <a name="syntax"></a>Syntaxe

> *nom* **=** *expression*

## <a name="remarks"></a>Notes

Le symbole peut être redéfini ultérieurement.
