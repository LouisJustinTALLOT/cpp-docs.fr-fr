# <a name="echo"></a>ECHO

Affiche le *message* sur le périphérique de sortie standard (par défaut, l’écran).

## <a name="syntax"></a>Syntaxe

>  *Message* d’écho

## <a name="remarks"></a>Notes

Identique à [% out](percent-out.md).
