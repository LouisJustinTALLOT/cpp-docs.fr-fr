# <a name="echo"></a>ECHO

Affiche *message* au périphérique de sortie standard (par défaut, l’écran).

## <a name="syntax"></a>Syntaxe

> ECHO *message*

## <a name="remarks"></a>Notes

Identique à [%out](../../assembler/masm/percent-out.md).
