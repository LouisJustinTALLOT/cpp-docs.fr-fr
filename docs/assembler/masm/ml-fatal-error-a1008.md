# <a name="ml-fatal-error-a1008"></a>Erreur ML irrécupérable A1008

**imbrication des macros non appariées**

Soit une macro n’a pas été arrêtée avant la fin du fichier, soit la directive de fin [ENDM](endm.md) a été trouvée en dehors d’un bloc macro.

L’une des causes de cette erreur est l’omission du point avant [. RÉPÉTez](dot-repeat.md) ou [. WHILe](dot-while.md).
