# <a name="operator-highword"></a>HIGHWORD, opérateur

Retourne les 16 bits de poids fort des 32 bits de poids faible de l' *expression*. Les expressions MASM sont des valeurs 64 bits.

## <a name="syntax"></a>Syntaxe

> *Expression* highword,
