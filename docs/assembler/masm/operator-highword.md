# operator HIGHWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retourne les bits de grande 16 des bits du bas 32 *de l'expression*.  Les expressions MASM sont des valeurs 64 bits.  
  
## Syntaxe  
  
```  
  
HIGHWORD expression  
```
