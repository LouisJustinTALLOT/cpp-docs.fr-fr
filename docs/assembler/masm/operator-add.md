# <a name="operator-"></a>opérateur +
Le premier opérateur retourne *expression1* plus *expression2*. Le deuxième opérateur inverse le signe de *expression*.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
      expression1 + expression2  
-expression  
```
