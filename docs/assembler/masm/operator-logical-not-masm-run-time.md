# <a name="operator--masm-run-time"></a>opérateur ! (Durée d’exécution MASM)

Négation logique. Utilisé uniquement dans [. IF](../../assembler/masm/dot-if.md), [. Bien que](../../assembler/masm/dot-while.md), ou [. RÉPÉTEZ](../../assembler/masm/dot-repeat.md) bloque et évaluée au moment de l’exécution, pas au moment de l’assembly.

## <a name="syntax"></a>Syntaxe

> ! *Expression*
