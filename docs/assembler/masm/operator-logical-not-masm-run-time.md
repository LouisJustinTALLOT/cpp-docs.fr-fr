# <a name="operator--masm-run-time"></a>opérateur ! (MASM Run Time)

Négation logique. Utilisé uniquement dans [. Si](dot-if.md), [. WHILe](dot-while.md), ou [. RÉPÈTE](dot-repeat.md) les blocs et les évalue au moment de l’exécution, et non au moment de l’assembly.

## <a name="syntax"></a>Syntaxe

> **!** *expression*
