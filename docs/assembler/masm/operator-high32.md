# <a name="operator-high32"></a>HIGH32, opérateur

Retourne les 32 bits de poids fort de l' *expression*. Les expressions MASM sont des valeurs 64 bits.

## <a name="syntax"></a>Syntaxe

>  *Expression* HIGH32,
