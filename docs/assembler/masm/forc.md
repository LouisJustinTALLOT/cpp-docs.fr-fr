# FORC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Marque un bloc qui sera répété une fois pour chaque caractère dans `string`, avec le caractère actuel en remplaçant `parameter` sur chaque répétition.  
  
## Syntaxe  
  
```  
  
   FORC  
parameter, <string> statements  
ENDM  
```  
  
## Notes  
 De la même manière qu' [IRPC](../../assembler/masm/irpc.md).
