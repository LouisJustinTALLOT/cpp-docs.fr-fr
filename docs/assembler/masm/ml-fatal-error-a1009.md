# <a name="ml-fatal-error-a1009"></a>Erreur ML irrécupérable A1009

**ligne trop longue**

Une ligne d’un fichier source a dépassé la limite de 512 caractères.

Si plusieurs lignes physiques sont concaténées avec le caractère de continuation de ligne (\), la ligne logique résultante est toujours limitée à 512 caractères.
