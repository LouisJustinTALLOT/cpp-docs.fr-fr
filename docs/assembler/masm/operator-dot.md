# <a name="operator-"></a>., opérateur

Le premier opérateur retourne *expression* ainsi que le décalage de *champ* dans sa structure ou une union. Le deuxième opérateur retourne la valeur à l’emplacement vers lequel pointé *inscrire* ainsi que le décalage de *champ* dans sa structure ou une union.

## <a name="syntax"></a>Syntaxe

> *Expression*<br/> . *champ* [[. *champ*]]...

> [*inscrire*]. *champ* [[. *champ*]]...
