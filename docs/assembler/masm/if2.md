# <a name="if2"></a>IF2
**IF** bloc est évalué à chaque passage de l’assembly si **OPTION : SETIF2** est **TRUE**.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
IF2   
expression  
  
```  
  
## <a name="remarks"></a>Notes  
 Consultez [IF](../../assembler/masm/if-masm.md) pour connaître la syntaxe complète.
