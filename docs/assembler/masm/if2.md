# IF2
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Le bloc d'**IF** est évalué à chaque passe d'assembly si **OPTION : SETIF2** est **TRUE**.  
  
## Syntaxe  
  
```  
  
IF2   
expression  
  
```  
  
## Notes  
 Consultez [SI](../../assembler/masm/if-masm.md) pour la syntaxe complète.
