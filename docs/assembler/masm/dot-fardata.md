# <a name="fardata"></a>.FARDATA
Lorsqu’il est utilisé avec [. MODÈLE](../../assembler/masm/dot-model.md), démarre un segment de données éloigné pour les données initialisées (nom de segment FAR_DATA ou *nom*).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
.FARDATA [[name]]  
```
