# .FARDATA
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

En cas de utilisation avec [.MODEL](../../assembler/masm/dot-model.md), démarre un segment de données lointain pour des données initialisées \(nom de segment FAR\_DATA ou *nom*\).  
  
## Syntaxe  
  
```  
  
.FARDATA [[name]]  
```
