# INCLUDELIB (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Signale à l'éditeur de liens le module actuel doit être lié avec *le libraryname*.  
  
## Syntaxe  
  
```  
  
INCLUDELIB libraryname  
```  
  
## Notes  
 *Le libraryname* doit être placé entre crochets pointus s'il comprend une barre oblique inverse, point\-virgule, supériorité le symbole, infériorité le symbole, le guillemet simple, ou le guillemet double.
