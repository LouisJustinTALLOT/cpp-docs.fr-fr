# <a name="operator-shr"></a>SHR, opérateur

Retourne le résultat de la décaler les bits du *nombre de caractères de droite de* l' *expression* .

## <a name="syntax"></a>Syntaxe

> *nombre* d' *expressions* de **SHR**
