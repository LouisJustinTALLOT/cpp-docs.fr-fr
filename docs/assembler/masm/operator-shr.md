# operator SHR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retourne le résultat de déplacer les bits du numéro *de nombre* de droite *d'expression* de bits.  
  
## Syntaxe  
  
```  
  
expression SHR count  
```
