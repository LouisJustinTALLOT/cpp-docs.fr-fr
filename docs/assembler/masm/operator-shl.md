# <a name="operator-shl"></a>SHL, opérateur

Retourne le résultat de la décaler les bits de *nombre de* bits de la partie gauche de l' *expression* .

## <a name="syntax"></a>Syntaxe

> *nombre* d' *expressions* **SHL**
