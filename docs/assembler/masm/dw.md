# <a name="dw"></a>DW

Alloue et initialise éventuellement un mot (2 octets) de stockage pour chaque *initialiseur*. Peut également être utilisé comme spécificateur de type partout où un type est légal. **DW** est un synonyme de [Word](word.md).

## <a name="syntax"></a>Syntaxe

> ⟦*nom* de l' *initialiseur* ⟧ **DW** ⟦__,__ *initialiseur* ... ⟧
