# DW
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

alloue et initialise éventuellement un mot \(2 octets\) du stockage pour chaque `initializer`.  Peut également être utilisé comme un spécificateur de type partout où un type est conforme.  `DW` est un synonyme de [WORD](../../assembler/masm/word.md).  
  
## Syntaxe  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
