# <a name=""></a>@@:

Définit une étiquette de code reconnaissable uniquement entre *label1* et *label2*, où *label1* est le début du code ou la précédente `@@:` étiquette et *label2* est fin du code ou la prochaine `@@:` étiquette. Consultez [ @B ](../../assembler/masm/at-b.md) et [ @F ](../../assembler/masm/at-f.md).

## <a name="syntax"></a>Syntaxe

> @@:
