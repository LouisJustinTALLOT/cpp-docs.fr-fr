# ML Nonfatal Error A2044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**caractère non valide dans le fichier**  
  
 Le fichier source contient un caractère en dehors d'un commentaire, d'une chaîne, ou d'un littéral qui n'a pas été marqué comme un opérateur ou un autre caractère conforme.
