# <a name="ml-nonfatal-error-a2044"></a>Erreur ML non fatale A2044

**caractère non valide dans le fichier**

Le fichier source contenait un caractère en dehors d’un commentaire, d’une chaîne ou d’un littéral qui n’a pas été reconnu comme opérateur ou autre caractère légal.
