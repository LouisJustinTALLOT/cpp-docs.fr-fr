# <a name="ml-nonfatal-error-a2044"></a>Erreur ML non fatale A2044

**caractère non valide dans le fichier**

Le fichier source contenue un caractère en dehors d’un commentaire, chaîne, ou littéral qui n’est pas reconnue comme un opérateur ou un autre caractère juridique.
