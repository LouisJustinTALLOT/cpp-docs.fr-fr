# <a name="tbyte"></a>TBYTE

Alloue et initialise éventuellement 10 octets de stockage pour chaque `initializer`. Peut également être utilisé comme un spécificateur de type n’importe où un type est autorisé.

## <a name="syntax"></a>Syntaxe

> [[nom]] TBYTEinitializer [[, initialiseur]]...
