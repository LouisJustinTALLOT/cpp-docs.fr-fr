# <a name="data"></a>.DATA

 (uniquement MASM 32 bits.) En cas d’utilisation avec [. MODÈLE](dot-model.md), démarre un segment de données near pour les données initialisées (nom de segment _data).

## <a name="syntax"></a>Syntaxe

> **. MÉTADONNÉE**\
> ⟦ *segmentItem* ⟧...
