# <a name="errdef"></a>.ERRDEF
Génère une erreur si *nom* est une étiquette définie précédemment, une variable ou un symbole.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
.ERRDEF name [[, message]]  
```
