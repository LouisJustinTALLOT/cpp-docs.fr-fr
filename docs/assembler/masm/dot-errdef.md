# <a name="errdef"></a>.ERRDEF

Génère une erreur si le *nom* est une étiquette, une variable ou un symbole précédemment défini.

## <a name="syntax"></a>Syntaxe

> **. ERRDEF** *nom* __de ⟦,__ *message*⟧
