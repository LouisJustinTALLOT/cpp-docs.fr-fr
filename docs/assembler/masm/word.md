# <a name="word"></a>WORD

Alloue et initialise éventuellement un mot (2 octets) de stockage pour chaque *initialiseur*. Peut également être utilisé comme spécificateur de type partout où un type est légal.

## <a name="syntax"></a>Syntaxe

> ⟦*nom*de l' *initialiseur* de **mot** ⟧ ⟦ __,__ *initialiseur* ... ⟧
