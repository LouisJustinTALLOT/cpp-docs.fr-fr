# <a name="ifidn"></a>IFIDN]

Assembly d’allocations si les éléments de texte sont identiques.

## <a name="syntax"></a>Syntaxe

> IFIDN [[I]] textitem1, textitem2

## <a name="remarks"></a>Notes

Si **je** est donné, la comparaison respecte la casse. Consultez [IF](../../assembler/masm/if-masm.md) pour la syntaxe complète.
