# <a name="ml-fatal-error-a1005"></a>Erreur ML irrécupérable A1005

**limite de l’assembleur : table de nom de paramètre de macro complète**

Un trop grand nombre de paramètres, de variables locales ou d’étiquettes de macro ont été définis pour une macro. La table des noms de macro n’a plus d’espace.

Définissez des noms plus courts ou moins nombreux, ou supprimez les macros inutiles.
