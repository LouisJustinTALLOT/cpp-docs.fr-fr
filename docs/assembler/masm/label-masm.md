# <a name="label-masm"></a>LABEL (MASM)
Crée une nouvelle étiquette en affectant la valeur de compteur à l’emplacement actuelle et la donnée `type` à *nom*.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
