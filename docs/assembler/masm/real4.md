# <a name="real4"></a>REAL4

Alloue et initialise éventuellement un nombre à virgule flottante de simple précision (4 octets) pour chaque *initialiseur*.

## <a name="syntax"></a>Syntaxe

> initialiseur de REAL4 nom [[, initialiseur]]...
