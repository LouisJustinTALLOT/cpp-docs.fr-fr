# <a name="real4"></a>REAL4

Alloue et initialise éventuellement un nombre à virgule flottante simple précision (4 octets) pour chaque *initialiseur*.

## <a name="syntax"></a>Syntaxe

> *nom*  **REAL4** *initializer* ⟦__,__ *initializer* ... ⟧
