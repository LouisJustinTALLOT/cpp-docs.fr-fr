# <a name="datasize"></a>@DataSize

0 pour **TINY**, **petit**, **support**, et **plat** des modèles, 1 pour **COMPACT** et  **GRANDES** modèles et 2 pour **énorme** modèle (numérique sont équivalentes).

## <a name="syntax"></a>Syntaxe

> @DataSize
