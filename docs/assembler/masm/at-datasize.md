# <a name="datasize"></a>@DataSize
0 pour **minuscule**, **petit**, **support**, et **plat** modèles, 1 pour **COMPACT** et  **GRAND** modèles et 2 pour **énorme** modèle (numérique sont équivalentes).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
@DataSize  
  
```
