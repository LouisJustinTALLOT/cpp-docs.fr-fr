# @DataSize
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

0 pour les modèles de **MINUSCULE**, de **PETIT**, de **MOYEN**, et d' **APPARTEMENT** , 1 pour les modèles de **CONTRAT** et de **GRAND** , et 2 du modèle d' **ÉNORME** \(numérique égalisez\).  
  
## Syntaxe  
  
```  
  
@DataSize  
  
```
