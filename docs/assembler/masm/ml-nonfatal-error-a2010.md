# <a name="ml-nonfatal-error-a2010"></a>Erreur ML non fatale A2010

**expression de type non valide**

L’opérande de [ce](../../assembler/masm/operator-this.md) ou [ptr](../../assembler/masm/operator-ptr.md) n’est pas une expression de type valide.
