# <a name="ml-nonfatal-error-a2010"></a>Erreur ML non fatale A2010

**expression de type non valide**

L’opérande à [THIS](../../assembler/masm/operator-this.md) ou [PTR](../../assembler/masm/operator-ptr.md) n’était pas une expression de type valide.
