# <a name="db"></a>DB

Alloue et initialise éventuellement un octet de stockage pour chaque *initialiseur*. **DB** est un synonyme de [Byte](byte-masm.md).

## <a name="syntax"></a>Syntaxe

> ⟦*nom*de l' *initialiseur* ⟧ **DB** ⟦ __,__ *initialiseur* ... ⟧

## <a name="remarks"></a>Notes

Peut également être utilisé comme spécificateur de type partout où un type est légal.
