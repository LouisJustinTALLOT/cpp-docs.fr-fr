# DB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

alloue et initialise éventuellement un octet de stockage pour chaque `initializer`.  `DB` est un synonyme d' [OCTET](../../assembler/masm/byte-masm.md)  
  
## Syntaxe  
  
```  
[[name]] DB initializer [[, initializer]] ...  
```  
  
## Notes  
 Peut également être utilisé comme un spécificateur de type partout où un type est conforme.
