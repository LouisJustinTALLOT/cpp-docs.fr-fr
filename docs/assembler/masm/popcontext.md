# <a name="popcontext"></a>POPCONTEXT

Restaure tout ou partie des cours `context` (enregistré par le [PUSHCONTEXT](../../assembler/masm/pushcontext.md) directive). Le `context` peut être **suppose que**, `RADIX`, **RÉPERTORIANT**, **processeur**, ou **tous les**.

## <a name="syntax"></a>Syntaxe

> Contexte POPCONTEXT
