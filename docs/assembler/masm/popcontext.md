# <a name="popcontext"></a>POPCONTEXT
Restaure tout ou partie des actuel `context` (enregistrés par le [PUSHCONTEXT](../../assembler/masm/pushcontext.md) directive). Le `context` peut être **suppose que**, `RADIX`, **liste**, **processeur**, ou **tous les**.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
POPCONTEXT context  
```
