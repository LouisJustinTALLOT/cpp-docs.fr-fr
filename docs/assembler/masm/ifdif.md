# <a name="ifdif"></a>IFDIF]

Assembly d’allocations si les éléments de texte sont différentes.

## <a name="syntax"></a>Syntaxe

> IFDIF [[I]] textitem1, textitem2

## <a name="remarks"></a>Notes

Si **je** est donné, la comparaison respecte la casse. Consultez [IF](../../assembler/masm/if-masm.md) pour la syntaxe complète.
