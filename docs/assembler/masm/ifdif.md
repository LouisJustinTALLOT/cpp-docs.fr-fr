# IFDIF]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Accorde l'assembly si les éléments de texte sont différents.  
  
## Syntaxe  
  
```  
  
IFDIF[[I]] textitem1, textitem2  
```  
  
## Notes  
 Si **i** est fourni, la comparaison ne respecte pas la casse.  Consultez [SI](../../assembler/masm/if-masm.md) pour la syntaxe complète.
