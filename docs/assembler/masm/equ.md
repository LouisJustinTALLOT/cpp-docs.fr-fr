# EQU
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la première directive assigne la valeur numérique de l' *expression au* *nom*.  
  
## Syntaxe  
  
```  
  
      name EQU expression  
name EQU <text>  
```  
  
## Notes  
 *Le nom* ne peut pas être redéfini ultérieurement.  
  
 La deuxième directive assigne *le texte* spécifié avec le *nom*.  *Le nom* peut être assigné *un texte* différent ultérieurement.  Consultez [TEXTEQU](../../assembler/masm/textequ.md).
