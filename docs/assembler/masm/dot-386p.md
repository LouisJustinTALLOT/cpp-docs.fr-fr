# .386P
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Active l'assembly de l'instruction \(privilégié\) pour le processeur 80386 ; désactive l'assembly de l'instruction présenté avec les processeurs ultérieurs.  
  
## Syntaxe  
  
```  
  
.386P  
  
```  
  
## Notes  
 Permet également à 80387 instructions.
