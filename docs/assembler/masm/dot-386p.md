# <a name="386p"></a>.386P

Permet à l’assembly de toutes les instructions (y compris les privilèges) pour le processeur 80386 ; désactive l’assembly d’instructions introduite avec des processeurs plus loin.

## <a name="syntax"></a>Syntaxe

> .386P

## <a name="remarks"></a>Notes

Permet également d’obtenir des 80387 instructions.
