# <a name="dd"></a>DD

Alloue et initialise éventuellement un mot double (4 octets) de stockage pour chaque *initialiseur*. **DD** est un synonyme de [DWORD](dword.md).

## <a name="syntax"></a>Syntaxe

> ⟦*⟧* **DD** *initializer* ⟦__,__ *initializer* ... ⟧

## <a name="remarks"></a>Notes

Peut également être utilisé comme spécificateur de type partout où un type est légal.
