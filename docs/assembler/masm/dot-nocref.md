# .NOCREF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Supprime la liste des symboles dans la table de symboles et le fichier browser.  
  
## Syntaxe  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## Notes  
 Si les noms sont spécifiés, seuls les noms fournis sont supprimés.  mêmes que [.XCREF](../../assembler/masm/dot-xcref.md).
