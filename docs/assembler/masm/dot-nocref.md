# <a name="nocref"></a>.NOCREF

Supprime la liste des symboles dans le fichier de table et le navigateur de symbole.

## <a name="syntax"></a>Syntaxe

> . NOCREF [[nom [[, name]]...]]

## <a name="remarks"></a>Notes

Si les noms sont spécifiés, seuls les noms donnés sont supprimés. Identique à [. XCREF](../../assembler/masm/dot-xcref.md).
