# DT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

alloue et initialise éventuellement 10 octets de stockage pour chaque `initializer`.  Peut également être utilisé comme un spécificateur de type partout où un type est conforme.  `DT` est un synonyme de [TBYTE](../../assembler/masm/tbyte.md).  
  
## Syntaxe  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
