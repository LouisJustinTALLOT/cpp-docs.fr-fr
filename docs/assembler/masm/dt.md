# <a name="dt"></a>DT
Alloue et initialise éventuellement de 10 octets de stockage pour chaque `initializer`. Peut également être utilisé comme un spécificateur de type n’importe où un type est autorisé. `DT` est un synonyme de [to](../../assembler/masm/tbyte.md).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
