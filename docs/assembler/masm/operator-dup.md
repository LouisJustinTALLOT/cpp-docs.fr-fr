# <a name="operator-dup"></a>DUP, opérateur

*Spécifie le nombre de* déclarations de *InitialValue*.

## <a name="syntax"></a>Syntaxe

> **comduplication** __de nombre (__*InitialValue* ⟦__,__ *InitialValue* ...  ⟧__)__
