# <a name="public-masm"></a>PUBLIC (MASM)

Rend chaque variable, une étiquette ou un symbole absolu spécifié en tant que *nom* disponible à tous les autres modules dans le programme.

## <a name="syntax"></a>Syntaxe

> Nom PUBLIC [[langtype]] [[, [[langtype]] nom]]...
