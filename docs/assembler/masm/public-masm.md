# PUBLIC (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fait chaque variable, nom, ou absolu symbole spécifié en tant que *nom* disponible à tous les autres modules dans le programme.  
  
## Syntaxe  
  
```  
  
PUBLIC [[langtype]] name [[, [[langtype]] name]]...  
```
