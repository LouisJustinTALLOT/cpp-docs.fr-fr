# operator LOW32
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retourne les bits du bas 32 *de l'expression*.  Les expressions MASM sont des valeurs 64 bits.  
  
## Syntaxe  
  
```  
  
LOW32 expression  
```
