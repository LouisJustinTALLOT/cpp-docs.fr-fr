# <a name="operator-low32"></a>LOW32, opérateur

Retourne les 32 bits de poids faible de l' *expression*. Les expressions MASM sont des valeurs 64 bits.

## <a name="syntax"></a>Syntaxe

> *Expression* LOW32,
