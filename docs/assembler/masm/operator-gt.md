# <a name="operator-gt"></a>GT, opérateur

Retourne true (-1) si *expression1* est supérieur à *Expression2*, ou retourne false (0) dans le cas contraire.

## <a name="syntax"></a>Syntaxe

> *expression1* **gt** *Expression2*
