# <a name="operator-gt"></a>GT, opérateur
Retourne la valeur true (-1) si *expression1* est supérieur à *expression2*, ou la valeur false (0) si elle n’est pas.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
expression1 GT expression2  
```
