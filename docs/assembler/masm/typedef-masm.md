# <a name="typedef"></a>TYPEDEF

Définit un nouveau type ou **proto** appelé *Name*, qui est équivalent au *type* ou *protoDefinition*.

## <a name="syntax"></a>Syntaxe

> *nom* *type* typedef | **proto** *protoDefinition*
