# TYPEDEF (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Définit un nouveau type appelé *le nom*, qui équivaut à `type`.  
  
## Syntaxe  
  
```  
  
name TYPEDEF type  
```
