# <a name="ends-masm"></a>ENDS (MASM)
Marque la fin du segment, structure ou union *nom* précédemment commencé avec [SEGMENT](../../assembler/masm/segment.md), [STRUCT](../../assembler/masm/struct-masm.md), [UNION](../../assembler/masm/union.md), ou un segment simplifié directive.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name ENDS  
```
