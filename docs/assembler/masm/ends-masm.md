# ENDS (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Marque la fin du segment, la structure, ou *le nom* de l'union précédemment démarré avec [SEGMENT](../../assembler/masm/segment.md), [STRUCT](../../assembler/masm/struct-masm.md), [UNION](../../assembler/masm/union.md), ou une directive simplifiée de segment.  
  
## Syntaxe  
  
```  
  
name ENDS  
```
