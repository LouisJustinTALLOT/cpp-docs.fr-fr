# <a name="ends"></a>ENDS

Marque la fin d’un segment, d’une structure ou d’un *nom* d’Union précédemment commencé avec un [segment](segment.md), un [struct](struct-masm.md), une [Union](union.md)ou une directive de segment simplifiée.

## <a name="syntax"></a>Syntaxe

>  **fin** du nom
