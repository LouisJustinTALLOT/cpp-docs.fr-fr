# IFDEF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Accorde l'assembly si *le nom* est un nom, une variable, ou un symbole défini précédemment.  
  
## Syntaxe  
  
```  
  
IFDEF name  
```  
  
## Notes  
 Consultez [SI](../../assembler/masm/if-masm.md) pour la syntaxe complète.
