# <a name="ifdef"></a>IFDEF

Accorde un assembly si *Name* est une étiquette, une variable ou un symbole précédemment défini.

## <a name="syntax"></a>Syntaxe

> *Nom* ifdef

Vérifiez [si](if-masm.md) la syntaxe est complète.
