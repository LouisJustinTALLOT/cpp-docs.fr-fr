# <a name="ifdef"></a>IFDEF
Accorde l’assembly si *nom* est une étiquette définie précédemment, une variable ou un symbole.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
IFDEF name  
```  
  
## <a name="remarks"></a>Notes  
 Consultez [IF](../../assembler/masm/if-masm.md) pour connaître la syntaxe complète.
