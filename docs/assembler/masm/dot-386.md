# .386
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

active l'assembly de l'instruction nonprivileged pour le processeur 80386 ; désactive l'assembly de l'instruction présenté avec les processeurs ultérieurs.  
  
## Syntaxe  
  
```  
  
.386  
  
```  
  
## Notes  
 Permet également à 80387 instructions.
