# <a name="386"></a>.386

Permet à l’assembly d’instructions non privilégiées du processeur 80386 ; désactive l’assembly d’instructions introduite avec des processeurs plus loin.

## <a name="syntax"></a>Syntaxe

> .386

## <a name="remarks"></a>Notes

Permet également d’obtenir des 80387 instructions.
