# <a name="386-32-bit-masm"></a>.386 (MASM 32 bits)

Active l’assembly d’instructions non privilégiées pour le processeur 80386 ; désactive l’assemblage des instructions introduites avec des processeurs ultérieurs. (uniquement MASM 32 bits.)

## <a name="syntax"></a>Syntaxe

> **.386**

## <a name="remarks"></a>Notes

Active également les instructions 80387.
