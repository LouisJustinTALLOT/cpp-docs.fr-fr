# <a name="end"></a>Fin

Marque la fin d’un module et, éventuellement, définit le point d’entrée du programme sur *procId*.

## <a name="syntax"></a>Syntaxe

> **End** ⟦*procId*⟧

## <a name="remarks"></a>Notes

L’argument *procId* est valide uniquement dans MASM 32 bits.
