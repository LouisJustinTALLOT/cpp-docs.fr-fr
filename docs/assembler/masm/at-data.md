# <a name="data"></a>@data
Le nom du groupe de données par défaut. Prend la valeur DGROUP pour tous les modèles sauf **plat**. A la valeur **plat** sous le **plat** le modèle de mémoire (macro de texte).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
@data  
  
```
