# <a name="listall"></a>.LISTALL

Démarre la liste de toutes les instructions.

## <a name="syntax"></a>Syntaxe

> .LISTALL

## <a name="remarks"></a>Notes

Équivaut à la combinaison de [. LISTE](../../assembler/masm/dot-list.md), [. LISTESI](../../assembler/masm/dot-listif.md), et [. LISTMACROALL](../../assembler/masm/dot-listmacroall.md).
