# .LISTALL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Commence la liste de toutes les instructions.  
  
## Syntaxe  
  
```  
  
.LISTALL  
  
```  
  
## Notes  
 équivalent à la combinaison de [.LIST](../../assembler/masm/dot-list.md), de [.LISTIF](../../assembler/masm/dot-listif.md), et de [.LISTMACROALL](../../assembler/masm/dot-listmacroall.md).
