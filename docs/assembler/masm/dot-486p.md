# <a name="486p-32-bit-masm"></a>.486P (MASM 32 bits)

Active l’assembly de toutes les instructions (y compris les privilèges) pour le processeur 80486. (uniquement MASM 32 bits.)

## <a name="syntax"></a>Syntaxe

> **.486P**
