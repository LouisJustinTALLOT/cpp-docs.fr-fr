# <a name="while-masm"></a>WHILE (MASM)

Répète l’assembly du bloc *instructions* tant que *expression* conserve la valeur true.

## <a name="syntax"></a>Syntaxe

> Bien que *expression*<br/>
> *Instructions*<br/>
> ENDM
