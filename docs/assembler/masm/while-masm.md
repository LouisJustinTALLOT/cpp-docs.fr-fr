# <a name="while"></a>WHILE

Répète l’assembly des *instructions* de bloc tant que *constantExpression* conserve la valeur true.

## <a name="syntax"></a>Syntaxe

> **WHILE** *ConstantExpression*\
> *instructions*\
> **ENDM**
