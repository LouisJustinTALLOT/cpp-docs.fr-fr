# <a name="operator--masm"></a>!=, opérateur (MASM)

N’est pas égal à. Utilisé uniquement dans [. Si](dot-if.md), [. WHILe](dot-while.md), ou [. RÉPÈTE](dot-repeat.md) les blocs et les évalue au moment de l’exécution, et non au moment de l’assembly.

## <a name="syntax"></a>Syntaxe

> *expression1* **! =** *Expression2*
