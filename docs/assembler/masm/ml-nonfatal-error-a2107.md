# <a name="ml-nonfatal-error-a2107"></a>Erreur ML non fatale A2107

**Impossible d’avoir un saut Far implicite ou un appel à Near label**

Une tentative a été effectuée pour effectuer un aller-but implicite ou un appel à une étiquette near dans un autre segment.
