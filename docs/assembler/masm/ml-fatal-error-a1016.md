# <a name="ml-fatal-error-a1016"></a>Erreur ML irrécupérable A1016

**Erreur interne d’assembleur**

Le pilote MASM, appelé ML.exe, a généré une erreur système.

Notez les circonstances de l’erreur et signalez-la à Microsoft Corporation. Les services de support technique sont disponibles à l’adresse [https://support.microsoft.com/](https://support.microsoft.com/) .
