# ML Fatal Error A1016
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Erreur interne d'assembly**  
  
 Le pilote MASM, ML.exe appelé, a généré une erreur système.  
  
 Notez les circonstances de l'erreur et avertissez Microsoft Corporation.  Les services de Support Technique de Microsoft sont disponibles dans [http:\/\/support.microsoft.com\/](http://support.microsoft.com/).
