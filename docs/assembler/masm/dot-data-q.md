# <a name="data"></a>.DATA?

Lorsqu’il est utilisé avec [. MODÈLE](../../assembler/masm/dot-model.md), démarre un segment de données proche pour les données non initialisées (_BSS de nom de segment).

## <a name="syntax"></a>Syntaxe

> .DATA?
