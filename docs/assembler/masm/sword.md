# <a name="sword"></a>SWORD

Alloue et initialise éventuellement un mot signé (2 octets) de stockage pour chaque `initializer`. Peut également être utilisé comme un spécificateur de type n’importe où un type est autorisé.

## <a name="syntax"></a>Syntaxe

> initialiseur de mot de passe de nom [[, initialiseur]]...
