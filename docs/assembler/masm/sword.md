# SWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

alloue et initialise éventuellement un mot signé \(2 octets\) du stockage pour chaque `initializer`.  Peut également être utilisé comme un spécificateur de type partout où un type est conforme.  
  
## Syntaxe  
  
```  
  
name SWORD initializer [[, initializer]]...  
```
