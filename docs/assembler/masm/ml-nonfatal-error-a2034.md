# ML Nonfatal Error A2034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**doit être dans le bloc de segment**  
  
 L'une des opérations suivantes a été trouvé en dehors d'un bloc de segment :  
  
-   une instruction  
  
-   une définition de nom  
  
-   un opérateur de [CECI](../../assembler/masm/operator-this.md)  
  
-   un opérateur de [$](../../assembler/masm/dollar.md)  
  
-   une définition de procédure  
  
-   une directive d' [ALIGNEZ](../../assembler/masm/align-masm.md)  
  
-   une directive d' [ORG](../../assembler/masm/org.md)
