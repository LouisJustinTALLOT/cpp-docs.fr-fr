# <a name="ml-nonfatal-error-a2074"></a>Erreur ML non fatale A2074
**Impossible d’accéder à étiquette via les registres de segment**  
  
 Une tentative a été effectuée pour accéder à une étiquette via un Registre de segment qui a été pas supposé que son segment ou un groupe.
