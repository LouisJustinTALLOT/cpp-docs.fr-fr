# ML Nonfatal Error A2074
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**impossible d'accéder au nom via des registres de segment**  
  
 Une tentative d'accéder à un nom via un registre de segment qui n'a pas été considéré à son segment ou groupe.
