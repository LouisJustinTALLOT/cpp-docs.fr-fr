# <a name="ml-nonfatal-error-a2074"></a>Erreur ML non fatale A2074

**Impossible d’accéder à l’étiquette via les registres de segment**

Une tentative a été effectuée pour accéder à une étiquette via un registre de segment qui n’a pas été considéré comme segment ou groupe.
