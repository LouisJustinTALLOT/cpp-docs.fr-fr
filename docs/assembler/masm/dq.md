# <a name="dq"></a>DQ

Alloue et initialise éventuellement 8 octets de stockage pour chaque *initialiseur*. Peut également être utilisé comme spécificateur de type partout où un type est légal. **DQ** est un synonyme de [QWord](qword.md).

## <a name="syntax"></a>Syntaxe

> ⟦ *⟧* **DQ** *initializer* ⟦ __,__ *initializer* ... ⟧
