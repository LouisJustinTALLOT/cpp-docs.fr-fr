# DQ
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

alloue et initialise éventuellement 8 octets de stockage pour chaque `initializer`.  Peut également être utilisé comme un spécificateur de type n'importe où qu'un type est conforme.  `DQ` est un synonyme de [QWORD](../../assembler/masm/qword.md).  
  
## Syntaxe  
  
```  
[[name]] DQ initializer [[, initializer]]...  
```
