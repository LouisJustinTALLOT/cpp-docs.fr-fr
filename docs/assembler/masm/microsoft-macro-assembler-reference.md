# <a name="microsoft-macro-assembler-reference"></a>Référence de Microsoft Macro Assembler
Microsoft Macro Assembler (MASM) offre plusieurs avantages par rapport à l’assembly inline. MASM contient une macro qui dispose de fonctionnalités telles que le traitement de chaîne de bouclage arithmétique et texte. MASM également offre un meilleur contrôle sur le matériel, car il prend en charge les jeux d’instructions de la 386, 486 et Pentium. À l’aide de MASM, vous pouvez également réduire surcharge de temps et de mémoire.
