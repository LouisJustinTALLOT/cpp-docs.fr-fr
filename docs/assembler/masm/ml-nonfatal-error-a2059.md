# ML Nonfatal Error A2059
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**erreur de liaison : les chaînes sont égales**  
  
 La conditionnel\-erreur [.ERRIDN](../../assembler/masm/dot-erridn.md) directive ou **.ERRIDNI** a été utilisée pour générer cette erreur.
