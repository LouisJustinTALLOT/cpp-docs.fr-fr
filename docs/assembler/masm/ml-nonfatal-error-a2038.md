# <a name="ml-nonfatal-error-a2038"></a>Erreur ML non fatale A2038

**opérande manquant pour l’opérateur de macro**

L’assembleur trouvé à la fin de la liste des paramètres d’une macro immédiatement après le [!](../../assembler/masm/operator-logical-not-masm-run-time.md) ou [ % ](../../assembler/masm/operator-percent.md) opérateur.
