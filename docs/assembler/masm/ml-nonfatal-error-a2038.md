# <a name="ml-nonfatal-error-a2038"></a>Erreur ML non fatale A2038

**opérande manquant pour l’opérateur de macro**

L’assembleur a trouvé la fin de la liste de paramètres d’une macro juste après le [!](../../assembler/masm/operator-logical-not-masm-run-time.md) ou [%](../../assembler/masm/operator-percent.md) opérateur.
