# <a name="operator-length"></a>LENGTH, opérateur
Retourne le nombre d’éléments de données dans *variable* créé par l’initialiseur de première.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
LENGTH variable  
```
