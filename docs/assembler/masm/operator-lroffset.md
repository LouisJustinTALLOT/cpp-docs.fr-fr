# operator LROFFSET
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retourne l'offset *de l'expression*.  Même qu' **OFFSET**, mais il génère un offset résolu par chargeur, qui permet aux fenêtres pour déplacer des segments de code.  
  
## Syntaxe  
  
```  
  
LROFFSET expression  
```
