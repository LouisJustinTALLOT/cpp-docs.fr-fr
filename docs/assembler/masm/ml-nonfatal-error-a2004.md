# ML Nonfatal Error A2004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**conflit de type de symbole : identificateur**  
  
 La directive d' [EXTERNDEF](../../assembler/masm/externdef.md) ou de [NOM](../../assembler/masm/label-masm.md) a été utilisée sur une variable, un symbole, une structure de données, ou un nom défini dans le même module mais avec un autre type.
