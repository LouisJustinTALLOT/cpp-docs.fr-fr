# <a name="ml-nonfatal-error-a2004"></a>Erreur ML non fatale A2004

**conflit de type de symbole : identificateur**

La directive [EXTERNDEF](../../assembler/masm/externdef.md) ou [label](../../assembler/masm/label-masm.md) a été utilisée sur une variable, un symbole, une structure de données ou une étiquette qui a été définie dans le même module, mais avec un type différent.
