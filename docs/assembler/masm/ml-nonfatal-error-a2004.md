# <a name="ml-nonfatal-error-a2004"></a>Erreur ML non fatale A2004

**conflit de type de symbole : identificateur**

Le [EXTERNDEF](../../assembler/masm/externdef.md) ou [étiquette](../../assembler/masm/label-masm.md) directive a été utilisée sur une variable, un symbole, une structure de données ou une étiquette qui a été défini dans le même module, mais avec un type différent.
