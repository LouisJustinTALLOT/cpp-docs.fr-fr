# <a name="ifndef"></a>IFNDEF

Accorde un assembly si le *nom* n’a pas été défini.

## <a name="syntax"></a>Syntaxe

> **Ifndef** *nom*

## <a name="remarks"></a>Notes

Vérifiez [si](if-masm.md) la syntaxe est complète.
