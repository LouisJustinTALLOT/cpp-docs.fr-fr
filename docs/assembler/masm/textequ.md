# <a name="textequ"></a>TEXTEQU

Assigne *textitem* à *nom*. Le *textitem* peut être une chaîne littérale, une constante précédée par un `%`, ou la chaîne retournée par une fonction de la macro.

## <a name="syntax"></a>Syntaxe

> nom TEXTEQU [[textitem]]
