# TEXTEQU
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Assigne *le textitem* *le nom*.   *le textitem* peut être une chaîne littéral, une constante précédée par `%`, ou la chaîne retournée par une fonction macro.  
  
## Syntaxe  
  
```  
  
name TEXTEQU [[textitem]]  
```
