# <a name="operator-ptr"></a>PTR, opérateur
La première force opérateur la *expression* pour être considérée comme ayant spécifié `type`. Le deuxième opérateur spécifie un pointeur vers `type`.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
