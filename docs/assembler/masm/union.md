# UNION
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

déclare une union d'un ou plusieurs types de données.  *les fielddeclarations* doivent être des définitions de données valides.  Omettez le nom *de nom* d' [EXTRÉMITÉS](../../assembler/masm/ends-masm.md)sur les définitions imbriquées d' **UNION** .  
  
## Syntaxe  
  
```  
  
      name   
      UNION [[alignment]] [[, NONUNIQUE]]  
   fielddeclarations  
[[name]] ENDS  
```
