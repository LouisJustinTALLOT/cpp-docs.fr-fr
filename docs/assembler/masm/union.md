# <a name="union"></a>UNION

Déclare une Union d’un ou plusieurs types de données. Les *déclarations Field* doivent être des définitions de données valides. Omettez l’étiquette de *nom* de [fin](ends-masm.md) sur les définitions d' **Union** imbriquées.

## <a name="syntax"></a>Syntaxe

> *nom* **Union** ⟦*alignement*⟧ ⟦ __,__ ⟧ non **unique**\
> *Field-declarations*\
> le*nom*⟦ ⟧ **se termine**
