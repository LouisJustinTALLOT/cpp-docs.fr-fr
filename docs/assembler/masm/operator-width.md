# <a name="operator-width"></a>WIDTH, opérateur

Retourne la largeur en bits de l' *recordfieldname* ou de l' *enregistrement*actuel.

## <a name="syntax"></a>Syntaxe

> **Largeur** {*recordfieldname* | *enregistrement*}
