# <a name="operator-width"></a>WIDTH, opérateur
Retourne la largeur en bits d’actuel *recordfieldname* ou *enregistrement*.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
WIDTH {recordfieldname | record}  
```
