# <a name="ml-nonfatal-error-a2060"></a>Erreur ML non fatale A2060
**forcé erreur : différent de chaînes**  
  
 La directive conditionnelle-erreur [. ERRDIF](../../assembler/masm/dot-errdif.md) ou **. ERRDIFI** a été utilisé pour générer cette erreur.
