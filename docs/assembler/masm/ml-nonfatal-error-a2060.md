# <a name="ml-nonfatal-error-a2060"></a>Erreur ML non fatale A2060

**erreur forcée : les chaînes ne sont pas égales**

Directive d’erreur conditionnelle [. ERRDIF](../../assembler/masm/dot-errdif.md) ou **. ERRDIFI** a été utilisé pour générer cette erreur.
