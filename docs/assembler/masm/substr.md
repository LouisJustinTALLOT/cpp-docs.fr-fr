# SUBSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retourne une sous\-chaîne *de textitem*, *à partir de la position*.  *le textitem* peut être une chaîne littéral, une constante précédée par `%`, ou la chaîne retournée par une fonction macro.  
  
## Syntaxe  
  
```  
  
name SUBSTR textitem, position [[, length]]  
```
