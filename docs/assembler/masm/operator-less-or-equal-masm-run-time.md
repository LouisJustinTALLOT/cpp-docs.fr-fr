# <a name="operator-lt-masm-run-time"></a>opérateur &lt;= (MASM Run Time)

Est inférieur ou égal à. Utilisé uniquement dans [. IF](../../assembler/masm/dot-if.md), [. Bien que](../../assembler/masm/dot-while.md), ou [. RÉPÉTEZ](../../assembler/masm/dot-repeat.md) bloque et évaluée au moment de l’exécution, pas au moment de l’assembly.

## <a name="syntax"></a>Syntaxe

> expression1 < = expression2
