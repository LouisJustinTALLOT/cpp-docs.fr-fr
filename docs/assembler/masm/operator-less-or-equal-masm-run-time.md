# <a name="operator-lt-masm-run-time"></a>&lt;opérateur = (MASM Run Time)

Est inférieur ou égal à. Utilisé uniquement dans [. Si](../../assembler/masm/dot-if.md), [. WHILe](../../assembler/masm/dot-while.md), ou [. RÉPÈTE](../../assembler/masm/dot-repeat.md) les blocs et les évalue au moment de l’exécution, et non au moment de l’assembly.
