# <a name="operator-lt-masm-run-time"></a>opérateur &lt; = (MASM Run Time)

Est inférieur ou égal à. Utilisé uniquement dans [. Si](dot-if.md), [. WHILe](dot-while.md), ou [. RÉPÈTE](dot-repeat.md) les blocs et les évalue au moment de l’exécution, et non au moment de l’assembly.
