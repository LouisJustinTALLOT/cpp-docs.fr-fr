# <a name="codesize"></a>@CodeSize

0 pour **TINY**, **petit**, **COMPACT**, et **plat** modèles et 1 pour **support**,  **GRANDES**, et **énorme** modèles (numérique sont équivalentes).

## <a name="syntax"></a>Syntaxe

> @CodeSize
