# @CodeSize
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

0 pour les modèles de **MINUSCULE**, de **PETIT**, de **CONTRAT**, et d' **APPARTEMENT** , et 1 pour les modèles de **MOYEN**, de **GRAND**, et d' **ÉNORME** \(numériques égalisez\).  
  
## Syntaxe  
  
```  
  
@CodeSize  
  
```
