# <a name="codesize"></a>@CodeSize
0 pour **minuscule**, **petit**, **COMPACT**, et **plat** modèles et 1 pour **support**,  **GRAND**, et **énorme** modèles (numérique sont équivalentes).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
@CodeSize  
  
```
