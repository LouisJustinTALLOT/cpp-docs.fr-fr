# <a name="codesize"></a>\@Code

0 pour les modèles **minuscules**, **petits**, **compacts** et **plats** , et 1 pour les modèles de **moyenne**, **grande taille** et **énorme** (équivalent numérique).

## <a name="syntax"></a>Syntaxe

> **\@Code**
