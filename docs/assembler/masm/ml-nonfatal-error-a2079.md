# <a name="ml-nonfatal-error-a2079"></a>Erreur ML non fatale A2079
**instruction n’autorise pas l’adressage présent direct**  
  
 Une boucle ou renvoi conditionnel ne peut pas être à un groupe ou un segment différent.
