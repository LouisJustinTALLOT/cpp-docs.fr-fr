# ML Fatal Error A1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**impossible d'ouvrir le fichier : nom de fichier**  
  
 L'assembly n'a pas pu ouvrir une fenêtre source, inclut, ou fichier de sortie.  
  
 L'une des opérations suivantes peut être une cause :  
  
-   Le fichier n'existe pas.  
  
-   Le fichier est utilisé par un autre processus.  
  
-   Le nom de fichier est pas valide.  
  
-   un fichier en lecture seule avec le nom de fichier de sortie existe déjà.  
  
-   Le lecteur actif est plein.  
  
-   Le répertoire actif est la racine et est plein.  
  
-   Le périphérique ne peut pas être écrits dans.  
  
-   Le joueur n'est pas prêt.
