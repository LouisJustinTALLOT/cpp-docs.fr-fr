# <a name="ml-fatal-error-a1000"></a>Erreur ML irrécupérable A1000
**Impossible d’ouvrir le fichier : nom de fichier**  
  
 L’assembleur n’a pas pu ouvrir une source, inclus ou fichier de sortie.  
  
 Un des éléments suivants peut être la cause :  
  
-   Le fichier n'existe pas.  
  
-   Le fichier est en cours d’utilisation par un autre processus.  
  
-   Le nom de fichier n’est pas valide.  
  
-   Il existe déjà un fichier en lecture seule avec le nom de fichier de sortie.  
  
-   Le lecteur actuel est complet.  
  
-   Le répertoire actif est la racine et est plein.  
  
-   L’appareil ne peut pas être écrit pour.  
  
-   Le lecteur n’est pas prêt.
