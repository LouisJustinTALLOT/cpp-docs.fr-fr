# DWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

alloue et initialise éventuellement un double mot \(4 octets\) du stockage pour chaque `initializer`.  
  
## Syntaxe  
  
```  
[[name]] DWORD initializer [[, initializer]]...  
```  
  
## Notes  
 Peut également être utilisé comme un spécificateur de type partout où un type est conforme.
