# PAGE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La première directive définit la longueur de la ligne et la largeur des caractères de la liste de programme.  Si aucun argument n'est fourni, génère un saut de page.  La deuxième directive incrémente le nombre de sections et réinitialise le numéro de page à 1.  
  
## Syntaxe  
  
```  
  
      PAGE [[[[length]], width]]  
PAGE +  
```
