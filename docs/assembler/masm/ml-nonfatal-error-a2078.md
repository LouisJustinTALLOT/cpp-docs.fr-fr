# <a name="ml-nonfatal-error-a2078"></a>Erreur ML non fatale A2078

**instruction n’autorise pas loin adressage indirect**

Une boucle ou renvoi conditionnel ne peut pas prendre un opérande de mémoire. Vous devez lui donner une adresse relative ou une étiquette.
