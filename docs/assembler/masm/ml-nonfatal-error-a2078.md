# ML Nonfatal Error A2078
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**l'instruction n'autorise pas l'indirection LOINTAIN**  
  
 Un arrêt conditionnel ou une boucle ne peut pas prendre un opérande de mémoire.  Il doit comporter une adresse connexe ou un nom.
