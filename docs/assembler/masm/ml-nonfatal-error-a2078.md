# <a name="ml-nonfatal-error-a2078"></a>Erreur ML non fatale A2078

**l’instruction n’autorise pas un adressage indirect lointain**

Un saut ou une boucle conditionnel ne peut pas accepter un opérande de mémoire. Elle doit recevoir une adresse ou une étiquette relative.
