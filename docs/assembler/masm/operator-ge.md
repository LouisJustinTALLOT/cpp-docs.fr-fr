# <a name="operator-ge"></a>GE, opérateur

Retourne true (-1) si *expression1* est supérieur ou égal à *Expression2*, ou retourne false (0) dans le cas contraire.

## <a name="syntax"></a>Syntaxe

> *expression1* **GE** *Expression2*
