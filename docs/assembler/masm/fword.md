# <a name="fword"></a>FWORD

Alloue et initialise éventuellement 6 octets de stockage pour chaque `initializer`.

## <a name="syntax"></a>Syntaxe

> [[nom]] Initialiseur FWORD [[, initialiseur]]...

## <a name="remarks"></a>Notes

Peut également être utilisé comme un spécificateur de type n’importe où un type est autorisé.
