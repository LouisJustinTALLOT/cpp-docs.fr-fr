# FWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

alloue et initialise éventuellement 6 octets de stockage pour chaque `initializer`.  
  
## Syntaxe  
  
```  
[[name]] FWORD initializer [[, initializer]]...  
```  
  
## Notes  
 Peut également être utilisé comme un spécificateur de type n'importe où qu'un type est conforme.
