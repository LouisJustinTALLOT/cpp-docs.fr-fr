# <a name="group-32-bit-masm"></a>GROUP (MASM 32 bits)

Ajoutez les *segments* spécifiés au groupe appelé *Name*. (uniquement MASM 32 bits.)

## <a name="syntax"></a>Syntaxe

> *nom* du *segment* de **groupe** ⟦ __,__ *segment* ... ⟧
