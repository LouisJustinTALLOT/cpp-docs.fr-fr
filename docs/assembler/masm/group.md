# <a name="group"></a>GROUP

(uniquement MASM 32 bits.) Ajoutez les *segments* spécifiés au groupe appelé *Name*.

## <a name="syntax"></a>Syntaxe

> *nom* du *segment* de **groupe** ⟦__,__ *segment* ... ⟧
