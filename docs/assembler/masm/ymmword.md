# YMMWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Utilisé pour opérandes multimédias 256 bits avec Intel a évolué l'instruction vectorielle des extensions \(AVX\).  
  
## Syntaxe  
  
```  
YMMWORD  
```  
  
## Notes  
 `YMMWORD` est conçu pour représenter le même type que `__m256` pour les éléments intrinsèques d'AVX.
