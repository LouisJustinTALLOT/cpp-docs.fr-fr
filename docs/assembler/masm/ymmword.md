# <a name="ymmword"></a>YMMWORD
Utilisé pour les opérandes multimédias 256 bits avec des instructions Intel Advanced Vector Extensions (AVX).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
YMMWORD  
```  
  
## <a name="remarks"></a>Notes  
 `YMMWORD` est destiné à représenter le même type que `__m256` pour les fonctions intrinsèques AVX.
