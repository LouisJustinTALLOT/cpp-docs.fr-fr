# <a name="endprolog"></a>.ENDPROLOG

Signale la fin des déclarations de prologue.

## <a name="syntax"></a>Syntaxe

> **.ENDPROLOG**

## <a name="remarks"></a>Notes

L’utilisation de l’une des déclarations de prologue en dehors de la région entre la [](proc.md) **trame** proc et la valeur est une erreur **. ENDPROLOG**.

Pour plus d’informations, consultez [MASM pour x64 (ml64.exe)](masm-for-x64-ml64-exe.md).
