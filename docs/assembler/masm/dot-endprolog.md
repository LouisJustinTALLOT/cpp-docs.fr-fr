# <a name="endprolog"></a>.ENDPROLOG

Signale la fin des déclarations de prologue.

## <a name="syntax"></a>Syntaxe

> **.ENDPROLOG**

## <a name="remarks"></a>Notes

C’est une erreur d’utiliser l’une des déclarations de prologue en dehors de la région entre [PROC](proc.md) **FRAME** et **.ENDPROLOG**.

Pour plus d’informations, consultez [MASM pour x64 (ml64. exe)](masm-for-x64-ml64-exe.md).
