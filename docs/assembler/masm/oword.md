# <a name="oword"></a>OWORD

Utilisé comme spécificateur de type lorsqu’un type de données de 16 octets est requis.

## <a name="syntax"></a>Syntaxe

> ⟦*⟧* **OWORD**  *initializer* ⟦__,__ *initializer* ... ⟧
