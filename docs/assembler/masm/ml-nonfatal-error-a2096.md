# ML Nonfatal Error A2096
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**segment, groupe, ou registre de segment attendu**  
  
 Un segment ou un groupe était attendu est introuvable.  
  
 L'une des opérations suivantes produit :  
  
-   L'opérande de gauche spécifié avec l'opérateur de substitution de segments \(**Numéro de téléphone :**\) n'est pas un registre de segment \(CS, DS, ss, es, FS, ou GS\), nom de groupe, nom de segment, ou expression de segment.  
  
-   La directive de [SUPPOSEZ](../../assembler/masm/assume.md) a été attribué un registre de segment sans adresse valide de segment, le registre de segment, le groupe, ou le groupe particulier d' **APPARTEMENT** .
