# <a name="586p-32-bit-masm"></a>.586P (MASM 32 bits)

Active l’assemblage de toutes les instructions (y compris les privilèges) pour le processeur Pentium. (uniquement MASM 32 bits.)

## <a name="syntax"></a>Syntaxe

> **.586P**
