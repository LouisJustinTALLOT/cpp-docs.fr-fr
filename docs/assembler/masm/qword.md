# <a name="qword"></a>QWORD

Alloue et initialise éventuellement 8 octets de stockage pour chaque `initializer`. Peut également être utilisé comme un spécificateur de type n’importe où un type est autorisé.

## <a name="syntax"></a>Syntaxe

> [[nom]] Initialiseur QWORD [[, initialiseur]]...
