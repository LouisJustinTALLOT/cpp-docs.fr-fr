# <a name="qword"></a>QWORD

Alloue et initialise éventuellement 8 octets de stockage pour chaque *initialiseur*. Peut également être utilisé comme spécificateur de type partout où un type est légal. **QWord** est un synonyme de [DQ](dq.md).

## <a name="syntax"></a>Syntaxe

> ⟦*nom* de l' *initialiseur* ⟧ **QWord** ⟦__,__ *initializer* ... ⟧
