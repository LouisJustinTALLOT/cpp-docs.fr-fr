# operator SECTIONREL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retourne le décalage relative de section de l'expression par rapport à la section contenant la cible dans le fichier exécutable final.  
  
## Syntaxe  
  
```  
SECTIONREL expression  
```  
  
## Notes  
 SECTIONREL est uniquement disponible avec l'émission d'objet COFF.
