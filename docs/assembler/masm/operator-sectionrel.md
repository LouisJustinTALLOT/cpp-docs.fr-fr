# <a name="operator-sectionrel"></a>SECTIONREL, opérateur

Retourne le décalage relatif de la section d’expression par rapport à la section qui contient la cible dans le fichier exécutable final.

## <a name="syntax"></a>Syntaxe

> Expression de SECTIONREL

## <a name="remarks"></a>Notes

SECTIONREL est disponible uniquement avec l’émission d’objet COFF.
