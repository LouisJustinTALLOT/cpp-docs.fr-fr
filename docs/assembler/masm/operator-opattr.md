# <a name="operator-opattr"></a>OPATTR, opérateur
Retourne un mot définissant le mode et l’étendue de *expression*. L’octet de poids faible est identique à l’octet retourné par [. TYPE](../../assembler/masm/operator-dot-type.md). L’octet de poids fort contient des informations supplémentaires.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
OPATTR expression  
```
