# <a name="operator-opattr"></a>OPATTR, opérateur

Retourne un mot définissant le mode et la portée de *expression*. L’octet de poids faible est identique à l’octet retourné par [. TYPE](../../assembler/masm/operator-dot-type.md). L’octet fort contient des informations supplémentaires.

## <a name="syntax"></a>Syntaxe

> Expression de OPATTR
