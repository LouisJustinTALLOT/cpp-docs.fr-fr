# operator OPATTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retourne un mot affectant la valeur au mode et la portée *de l'expression*.  Le bas octet identique à l'octet retourné par [.TYPE](../../assembler/masm/operator-dot-type.md).  L'octet le plus élevé contient des informations supplémentaires.  
  
## Syntaxe  
  
```  
  
OPATTR expression  
```
