# <a name="operator-le"></a>LE, opérateur
Retourne la valeur true (-1) si *expression1* est inférieur ou égal à *expression2*, ou la valeur false (0) si elle n’est pas.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
expression1 LE expression2  
```
