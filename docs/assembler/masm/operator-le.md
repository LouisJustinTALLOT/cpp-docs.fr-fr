# <a name="operator-le"></a>LE, opérateur

Retourne true (-1) si *expression1* est inférieur ou égal à *Expression2*, ou retourne false (0) dans le cas contraire.

## <a name="syntax"></a>Syntaxe

> *expression1* **le** *Expression2*
