# <a name="686p-32-bit-masm"></a>.686P (MASM 32 bits)

Active l’assemblage de toutes les instructions (y compris les privilèges) pour le processeur Pentium Pro. (uniquement MASM 32 bits.)

## <a name="syntax"></a>Syntaxe

> **.686P**
