# SDWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

alloue et initialise éventuellement un double mot signé \(4 octets\) du stockage pour chaque `initializer`.  Peut également être utilisé comme un spécificateur de type n'importe où qu'un type est conforme.  
  
## Syntaxe  
  
```  
  
name SDWORD initializer [[, initializer]]...  
```
