# operator CARRY?
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

État de l'indicateur de report.  Utilisé uniquement dans [.IF](../../assembler/masm/dot-if.md), [.WHILE](../../assembler/masm/dot-while.md), ou de blocs de [.REPEAT](../../assembler/masm/dot-repeat.md) et évalué au moment de l'exécution, pas au moment de l'assembly.  
  
## Syntaxe  
  
```  
  
CARRY?  
  
```
