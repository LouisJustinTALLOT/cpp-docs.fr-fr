# <a name="operator-eq"></a>EQ, opérateur

Retourne la valeur true (-1) si *expression1* est égal à *expression2*, ou retourne la valeur false (0) si elle n’est pas le cas.

## <a name="syntax"></a>Syntaxe

> expression1 EQ expression2
