# <a name="operator-eq"></a>EQ, opérateur

Retourne true (-1) si *expression1* est égal à *Expression2*, ou retourne false (0) dans le cas contraire.

## <a name="syntax"></a>Syntaxe

> *expression1* **EQ** *Expression2*
