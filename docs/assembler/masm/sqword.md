# <a name="sqword"></a>SQWORD

Alloue et initialise éventuellement 8 octets signés de stockage pour chaque `initializer`. Peut également être utilisé comme un spécificateur de type n’importe où un type est autorisé.

## <a name="syntax"></a>Syntaxe

> initialiseur de SQDWORD nom [[, initialiseur]]...
