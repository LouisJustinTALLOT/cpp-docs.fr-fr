# <a name="sqword"></a>SQWORD

Alloue et initialise éventuellement 8 octets signés de stockage pour chaque *initialiseur*. Peut également être utilisé comme spécificateur de type partout où un type est légal.

## <a name="syntax"></a>Syntaxe

> *nom* **SQWORD** *initializer* ⟦ __,__ *initializer* ... ⟧
