# ELSEIF2
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Bloc d'[ELSEIF](../../assembler/masm/elseif-masm.md) évalué à chaque passe d'assembly si **OPTION : SETIF2** est **TRUE**.  
  
## Syntaxe  
  
```  
  
ELSEIF2  
  
```
