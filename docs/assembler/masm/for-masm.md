# <a name="for-masm"></a>FOR (MASM)

Marque un bloc qui est répété une fois pour chaque *argument*, avec en cours *argument* en remplaçant *paramètre* à chaque répétition.

## <a name="syntax"></a>Syntaxe

> POUR *paramètre* [[ : no | : =*par défaut*]], \< *argument* [[, *argument*]]... ><br/>
> *Instructions*<br/>
> ENDM

## <a name="remarks"></a>Notes

Identique à [IRP](../../assembler/masm/irp.md).
