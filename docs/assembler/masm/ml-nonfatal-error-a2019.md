# <a name="ml-nonfatal-error-a2019"></a>Erreur ML non fatale A2019
**opérande doit être de type d’enregistrement ou champ**  
  
 Les éléments suivants d’opérande le [largeur](../../assembler/masm/operator-width.md) ou [masque](../../assembler/masm/operator-mask.md) opérateur n’était pas valide.  
  
 Le [largeur](../../assembler/masm/operator-width.md) l’opérateur accepte un opérande qui est le nom d’un champ ou un enregistrement. Le [masque](../../assembler/masm/operator-mask.md) l’opérateur accepte un opérande qui est le nom d’un champ ou un type d’enregistrement.
