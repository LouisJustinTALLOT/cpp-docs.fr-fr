# ML Nonfatal Error A2019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**l'opérande doit être type d'enregistrement ou champ**  
  
 L'opérande suivant l'opérateur de [SPOOFING](../../assembler/masm/operator-width.md) ou de [MASQUE](../../assembler/masm/operator-mask.md) n'était pas valide.  
  
 L'opérateur de [SPOOFING](../../assembler/masm/operator-width.md) accepte un opérande qui est le nom d'un champ ou d'un enregistrement.  L'opérateur de [MASQUE](../../assembler/masm/operator-mask.md) accepte un opérande qui est le nom d'un champ ou un type d'enregistrement.
