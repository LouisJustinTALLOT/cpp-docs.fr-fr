# <a name="ml-nonfatal-error-a2019"></a>Erreur ML non fatale A2019

**l’opérande doit être un type d’enregistrement ou un champ**

L’opérande qui suit l’opérateur de [largeur](operator-width.md) ou de [masque](operator-mask.md) n’est pas valide.

L’opérateur [Width](operator-width.md) accepte un opérande qui est le nom d’un champ ou d’un enregistrement. L’opérateur [Mask](operator-mask.md) accepte un opérande qui est le nom d’un champ ou d’un type d’enregistrement.
