# <a name="ifnb"></a>IFNB

Accorde un assembly si *textItem* n’est pas vide.

## <a name="syntax"></a>Syntaxe

> **IFNB** *textItem*

## <a name="remarks"></a>Notes

Vérifiez [si](../../assembler/masm/if-masm.md) la syntaxe est complète.
