# <a name="ifnb"></a>IFNB
Accorde l’assembly si *textitem* n’est pas vide.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
IFNB textitem  
```  
  
## <a name="remarks"></a>Notes  
 Consultez [IF](../../assembler/masm/if-masm.md) pour connaître la syntaxe complète.
