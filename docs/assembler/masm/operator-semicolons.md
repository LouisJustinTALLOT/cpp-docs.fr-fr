# operator ;;
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Les traite *le texte* de commentaire dans une macro qui s'affiche uniquement dans la définition de macro.  La liste n'affiche pas *de texte* où le est développée.  
  
## Syntaxe  
  
```  
  
;;  
text  
  
```
