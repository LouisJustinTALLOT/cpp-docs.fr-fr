# <a name="repeat"></a>REPEAT

Marque un bloc qui doit être répétée *expression* fois. Identique à [REPT](../../assembler/masm/rept.md).

## <a name="syntax"></a>Syntaxe

> RÉPÉTEZ *expression*<br/>
> *Instructions*<br/>
> ENDM
