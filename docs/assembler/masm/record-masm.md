# <a name="record-masm"></a>RECORD (MASM)

Déclare un type d’enregistrement comprenant les champs spécifiés. *FieldName* désigne le champ, *largeur* Spécifie le nombre de bits, et *expression* donne sa valeur initiale.

## <a name="syntax"></a>Syntaxe

> *Nom_Enregistrement* enregistrement *fieldname*:*largeur* [[= *expression*]] [[, *fieldname*:*delalargeur* [[= *expression*]]]]...
