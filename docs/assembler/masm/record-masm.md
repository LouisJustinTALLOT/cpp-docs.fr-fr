# <a name="record-masm"></a>RECORD (MASM)
Déclare un type d’enregistrement comprenant les champs spécifiés. *FieldName* le champ, les noms *largeur* Spécifie le nombre de bits, et *expression* donne sa valeur initiale.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
   recordname RECORD fieldname:width [[= expression]]   
[[, fieldname:width [[= expression]]]]...  
```
