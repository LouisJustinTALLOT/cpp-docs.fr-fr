# ML Warning A4014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

instruction et données initialisées non gérées dans les segments de BSS  
  
 Une tentative de définition des données initialisées dans une section de BSS.  Une section de BSS est définie comme une classe dont le nom est BSS.  Cela inclut le segment simplifié `.data?`.
