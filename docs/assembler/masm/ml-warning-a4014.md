# <a name="ml-warning-a4014"></a>Avertissement ML A4014

instructions et données initialisées non prises en charge dans les segments BSS

Une tentative de définition de données initialisées dans une section BSS a été effectuée.  Une section BSS est définie en tant que classe dont le nom est BSS.  Cela comprend le segment simplifié `.data?`.
