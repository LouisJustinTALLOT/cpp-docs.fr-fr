# <a name="ml-warning-a4014"></a>Avertissement ML A4014

des instructions et des données initialisées ne pas pris en charge dans les segments BSS

Une tentative a été effectuée pour définir les données initialisées dans une section BSS.  Une section BSS est définie en tant que classe dont le nom est BSS.  Cela inclut le segment simplifié `.data?`.
