# <a name="errdifi"></a>.ERRDIF[[I]]

Génère une erreur si les éléments de texte sont différentes.

## <a name="syntax"></a>Syntaxe

> .ERRDIF[[I]] *textitem1*, *textitem2* [[, *message*]]

## <a name="remarks"></a>Notes

Si **je** est donné, la comparaison respecte la casse.
