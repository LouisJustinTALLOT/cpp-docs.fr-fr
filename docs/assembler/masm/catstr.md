# <a name="catstr"></a>CATSTR

Concatène les éléments de texte.

## <a name="syntax"></a>Syntaxe

> nom CATSTR [[textitem1 [[, textitem2]]...]]

## <a name="remarks"></a>Notes

Chaque élément de texte peut être une chaîne littérale, une constante précédée par un `%`, ou la chaîne retournée par une fonction de la macro.
