# <a name="operator-xor"></a>XOR, opérateur

Retourne le résultat d’une opération **Xor** au niveau du bit pour *expression1* et *Expression2*.

## <a name="syntax"></a>Syntaxe

> *expression1* **Xor** *Expression2*
