# <a name="operator--"></a>opérateur-

Retourne *expression1* moins *expression2*. Le deuxième opérateur inverse le signe de *expression*.

## <a name="syntax"></a>Syntaxe

> *expression1* - *expression2*
>
> -*Expression*
