# <a name="operator--"></a>-, opérateur

Retourne *expression1* moins *Expression2*. Le deuxième opérateur inverse le signe de l' *expression*.

## <a name="syntax"></a>Syntaxe

> *expression1* **-** *Expression2*
>
> *expression*-
