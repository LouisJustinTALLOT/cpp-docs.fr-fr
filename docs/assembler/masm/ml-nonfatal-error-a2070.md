# ML Nonfatal Error A2070
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**opérandes valides d'instruction**  
  
 Un ou plusieurs opérandes sont pas valides pour l'instruction avec laquelle ils ont été spécifiés.
