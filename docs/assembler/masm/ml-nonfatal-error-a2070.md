# <a name="ml-nonfatal-error-a2070"></a>Erreur ML non fatale A2070
**opérandes de l’instruction non valide**  
  
 Un ou plusieurs opérandes ne sont pas valides pour l’instruction avec lequel ils ont été spécifiés.
