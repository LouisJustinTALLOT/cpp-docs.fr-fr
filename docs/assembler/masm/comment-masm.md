# <a name="comment"></a>COMMENT

Traite tout le *texte* entre ou sur la même ligne que les délimiteurs comme un commentaire.

## <a name="syntax"></a>Syntaxe

>  *Délimiteur* de commentaire ⟦*texte*⟧ \
> ⟦*texte*⟧ \
> ⟦*Text*⟧ *Delimiter* ⟦*Text*⟧
