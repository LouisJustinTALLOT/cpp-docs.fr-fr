# <a name="comment-masm"></a>COMMENT (MASM)

Traite tous les *texte* entre ou sur la même ligne que les délimiteurs sous forme de commentaire.

## <a name="syntax"></a>Syntaxe

> Délimiteur de commentaire [[texte]]<br/>
> [[text]]<br/>
> [[text]] delimiter [[text]]
