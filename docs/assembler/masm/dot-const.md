# <a name="const-32-bit-masm"></a>. CONSt (MASM 32 bits)

En cas d’utilisation avec [. MODÈLE](dot-model.md), démarre un segment de données constant (avec le nom de segment **const**).

## <a name="syntax"></a>Syntaxe

> **. CONST**\
> ⟦ *segmentItem* ⟧...

## <a name="remarks"></a>Notes

Ce segment a l’attribut de lecture seule.
