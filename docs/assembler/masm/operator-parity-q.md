# <a name="operator-parity"></a>PARITY?, opérateur

État de l’indicateur de parité. Utilisé uniquement dans [. Si](../../assembler/masm/dot-if.md), [. WHILe](../../assembler/masm/dot-while.md), ou [. RÉPÈTE](../../assembler/masm/dot-repeat.md) les blocs et les évalue au moment de l’exécution, et non au moment de l’assembly.

## <a name="syntax"></a>Syntaxe

> **PARITÉ?**
