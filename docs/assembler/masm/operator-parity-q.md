# <a name="operator-parity"></a>PARITY?, opérateur
État de l’indicateur de parité. Utilisé uniquement dans [. IF](../../assembler/masm/dot-if.md), [. Alors que](../../assembler/masm/dot-while.md), ou [. RÉPÉTEZ](../../assembler/masm/dot-repeat.md) bloque et évaluées au moment de l’exécution, pas au moment de l’assembly.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
PARITY?  
  
```
