# <a name="elseif"></a>ELSEIF

Combine [else](else-masm.md) et [If](if-masm.md) dans une instruction.

## <a name="syntax"></a>Syntaxe

> **ElseIf** *constantExpression*\
> *publication*\
> ⟦**Else**\
> *else-States*⟧ \
> **ENDIF**
