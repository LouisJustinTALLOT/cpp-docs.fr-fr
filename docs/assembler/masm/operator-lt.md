# <a name="operator-lt"></a>LT, opérateur
Retourne la valeur true (-1) si *expression1* est inférieure à *expression2*, ou la valeur false (0) si elle n’est pas.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
expression1 LT expression2  
```
