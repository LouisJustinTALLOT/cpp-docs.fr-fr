# <a name="operator-lt"></a>LT, opérateur

Retourne true (-1) si *expression1* est inférieur à *Expression2*, ou retourne false (0) dans le cas contraire.

## <a name="syntax"></a>Syntaxe

> *expression1* **lt** *Expression2*
