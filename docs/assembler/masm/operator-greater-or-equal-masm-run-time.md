# <a name="operator-gt-masm-run-time"></a>opérateur &gt; = (MASM Run Time)

Est supérieur ou égal à. Utilisé uniquement dans [. Si](dot-if.md), [. WHILe](dot-while.md), ou [. RÉPÈTE](dot-repeat.md) les blocs et les évalue au moment de l’exécution, et non au moment de l’assembly.
