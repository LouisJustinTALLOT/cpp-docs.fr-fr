# operator &gt;= (MASM Run Time)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Est supérieure ou égale à.  Utilisé uniquement dans [.IF](../../assembler/masm/dot-if.md), [.WHILE](../../assembler/masm/dot-while.md), ou de blocs de [.REPEAT](../../assembler/masm/dot-repeat.md) et évalué au moment de l'exécution, pas au moment de l'assembly.  
  
## Syntaxe  
  
```  
  
expression1 >= expression2  
```
