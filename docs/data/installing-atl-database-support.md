# Installation de la prise en charge des bases de donn&#233;es ATL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le programme d'installation installe tous les composants nécessaires pour la prise en charge de la programmation OLE DB dans le cadre de l'installation de base.  Les interfaces OLE DB sont installées en même temps que le kit de développement MDAC SDK.  Les modèles OLE DB, qui prennent en charge la programmation OLE DB dans Visual C\+\+ .NET, sont installés avec la bibliothèque ATL.  Le programme d'installation de Visual C\+\+ .NET installe à la fois le kit de développement MDAC SDK \(version 2.8\) et la bibliothèque ATL.
