# O&#249; puis-je trouver des articles de la Base de connaissances Microsoft traitant des bases de donn&#233;es&#160;?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Pour trouver des articles de Base de connaissances sur les rubriques ODBC MFC, effectuez une recherche dans la Base de connaissances à partir du mot clé MFCDATABASE.  
  
 Pour trouver des articles de Base de connaissances sur les rubriques DAO MFC, effectuez une recherche dans la Base de connaissances à partir du mot clé MFCDAO.
