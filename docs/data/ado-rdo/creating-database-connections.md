# Cr&#233;ation de connexions de base de donn&#233;es
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Pour utiliser la liaison de données, vous devez configurer une source de données.  Lorsque vous utilisez le contrôle de données ADO, vous devez configurer une connexion OLE DB.  Lors de l'utilisation du contrôle RemoteData RDO, vous devez créer une connexion ODBC \(Open Database Connectivity\).  Dans la mesure où Visual C\+\+ est livré avec un fournisseur OLE DB pour les sources de données ODBC, le contrôle de données ADO peut également tirer profit des connexions ODBC.
