# Insertion du contr&#244;le dans une application Visual&#160;C++
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Vous pouvez insérer un contrôle ActiveX dans une boîte de dialogue à partir de la barre d'outils ou en ayant recours à la boîte de dialogue [Insérer un contrôle ActiveX](../../mfc/insert-activex-control-dialog-box.md).  
  
### Pour insérer un contrôle ActiveX à partir de la boîte à outils  
  
1.  Cliquez avec le bouton droit sur une section vide de la boîte à outils.  
  
2.  Dans le menu contextuel, cliquez sur **Personnaliser la boîte à outils**, puis sélectionnez les contrôles qui vous intéressent.  
  
3.  Faites glisser les contrôles jusqu'à la boîte de dialogue dans l'Éditeur de boîtes de dialogue.  
  
### Pour insérer un contrôle ActiveX à partir de l'Éditeur de boîtes de dialogue  
  
1.  Cliquez avec le bouton droit sur la boîte de dialogue.  
  
2.  Dans le menu contextuel, cliquez sur [Insérer un contrôle ActiveX](../../mfc/insert-activex-control-dialog-box.md).  
  
    > [!NOTE]
    >  Lorsque vous insérez un contrôle ActiveX dans un projet à partir de la boîte de dialogue **Insérer un contrôle ActiveX**, les [Classes wrapper](../../data/ado-rdo/wrapper-classes.md) ne sont pas incluses dans le projet.  Il vous appartient de créer une classe wrapper pour personnaliser le fonctionnement du contrôle.
