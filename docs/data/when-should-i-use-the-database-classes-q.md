# Quand dois-je utiliser les classes de base de donn&#233;es&#160;?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Un grand nombre d'applications exigent le stockage de données dans une base de données, alors que de nombreuses autres applications pourraient tirer parti de l'utilisation d'une base de données.  Une base de données permet de disposer d'un référentiel de données souple et accessible, dans bien des cas, par de multiples utilisateurs et applications.  Les bases de données peuvent stocker de grandes quantités de données et permettent un accès rapide aux données pour les requêtes et les mises à jour.
