# Source de donn&#233;es&#160;: d&#233;termination du sch&#233;ma de la source de donn&#233;es (ODBC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Cette rubrique s'applique aux classes ODBC MFC.  
  
 Pour installer les membres de données dans vos objets `CRecordset`, vous devez connaître le schéma de la source de données à laquelle vous êtes connecté.  Pour déterminer ce schéma, vous devez obtenir une liste des tables dans la source de données, une liste des colonnes de chaque table, le type de données de chaque colonne et l'existence de tous les index.
