# <a name="data-source-determining-the-schema-of-the-data-source-odbc"></a>Source de données : détermination du schéma de la source de données (ODBC)

Cette rubrique s’applique aux classes ODBC MFC.

Pour configurer des membres de données dans vos objets `CRecordset`, vous devez connaître le schéma de la source de données à laquelle vous vous connectez. Pour déterminer le schéma d’une source de données, vous devez obtenir une liste des tables de la source de données, une liste des colonnes de chaque table, le type de données de chaque colonne et l’existence de tous les index.
