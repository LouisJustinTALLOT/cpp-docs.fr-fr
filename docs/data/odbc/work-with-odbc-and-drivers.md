# <a name="work-with-odbc-and-drivers"></a>Utilisation d'ODBC et de pilotes

ODBC vous permet d’écrire des applications qui ne sont pas dépendantes sur un seul système de gestion de base de données (SGBD). Vous pouvez écrire une application qui peut fonctionner sur plusieurs systèmes SGBD dont vos utilisateurs ont le pilote ODBC de SGBD spécifiques.
