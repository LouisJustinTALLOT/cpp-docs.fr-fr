# Utilisation d&#39;ODBC et de pilotes
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ODBC vous permet d'écrire des applications qui ne sont pas dépendantes d'un seul système de gestion de base de données \(SGBD\).  Vous pouvez écrire une application capable d'opérer sur plusieurs systèmes SGBD, pour lesquels vos utilisateurs possèdent le pilote ODBC spécifique adéquat.
