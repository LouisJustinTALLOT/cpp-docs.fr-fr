# <a name="work-with-odbc-and-drivers"></a>Utilisation d'ODBC et de pilotes

ODBC vous permet d’écrire des applications qui ne dépendent pas d’un seul système de gestion de base de données (SGBD). Vous pouvez écrire une application qui peut fonctionner sur plusieurs SGBD pour lesquels vos utilisateurs disposent du pilote ODBC correct propre au SGBD.
