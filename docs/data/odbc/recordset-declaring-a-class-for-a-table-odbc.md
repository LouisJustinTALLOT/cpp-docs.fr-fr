# Recordset&#160;: d&#233;claration de la classe d&#39;une table (ODBC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Cette rubrique s'applique aux classes ODBC MFC.  
  
 La classe de recordset la plus courante ouvre une seule table.  Pour déclarer la classe de recordset d'une seule table, utilisez l'[Assistant Consommateur ODBC MFC](../../mfc/reference/adding-an-mfc-odbc-consumer.md) à partir de l'Assistant **Ajouter une classe** et choisissez chaque colonne souhaitée en attribuant un nom au membre correspondant de données de type champ de recordset.  
  
 Parmi les autres utilisations des recordsets figurent :  
  
-   la jointure de deux ou de plusieurs tables ;  
  
-   le stockage des résultats d'une requête prédéfinie.
