# <a name="odbc-driver-list"></a>Liste de pilotes ODBC

Visual C++ fournit des pilotes ODBC pour les bases de données suivantes :

- SQL Server

- Microsoft Access

- Microsoft Excel

- dBASE

- Paradox

- Microsoft Oracle ODBC

- Fichiers texte

Pour plus d’informations sur les pilotes ODBC disponibles auprès de Microsoft et d’autres sociétés, y compris ODBC Driver Pack, contactez les services de support technique Microsoft.
