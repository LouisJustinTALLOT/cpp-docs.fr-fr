# IDBInitializeImpl::Uninitialize
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Place l'objet source de données dans un état non initialisé en libérant des ressources internes telles que la prise en charge de la propriété.  
  
## Syntaxe  
  
```  
  
      STDMETHOD(Uninitialize)(   
   void    
);  
```  
  
## Notes  
 Consultez [IDBInitialize::Uninitialize](https://msdn.microsoft.com/en-us/library/ms719648.aspx) dans le *Guide de référence du programmeur OLE DB*.
