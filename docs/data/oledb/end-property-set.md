# <a name="endpropertyset"></a>END_PROPERTY_SET
Marque la fin d’un jeu de propriétés.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp
END_PROPERTY_SET(guid)  
  
```  
  
#### <a name="parameters"></a>Paramètres  
 `guid`  
 [in] La propriété GUID.
