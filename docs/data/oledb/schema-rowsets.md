# Jeux de lignes du sch&#233;ma
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Le schéma \(ou métadonnées\) représente toutes les informations relatives à la structure ou à l'organisation d'une base de données, telles que les informations sur le fournisseur, un jeu de lignes, une table, des colonnes ou d'autres informations en dehors du contenu réel de la base de données.  OLE DB permet d'obtenir ces informations sans ouvrir le jeu de lignes, en utilisant des jeux de lignes du schéma.  
  
 Pour plus d'informations sur l'utilisation des jeux de lignes du schéma, consultez [Récupération de métadonnées à l'aide de jeux de lignes du schéma](../../data/oledb/obtaining-metadata-with-schema-rowsets.md).
