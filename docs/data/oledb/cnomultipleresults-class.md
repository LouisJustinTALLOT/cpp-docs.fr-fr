# <a name="cnomultipleresults-class"></a>CNoMultipleResults, classe

Utilisé comme argument de modèle (*TMultiple*) sur [CCommand](../../data/oledb/ccommand-class.md) pour créer une commande optimisée qui gère un seul jeu de résultats.

## <a name="syntax"></a>Syntaxe

```cpp
class CNoMultipleResults
```

## <a name="remarks"></a>Notes

Si vous souhaitez qu’une commande gère plusieurs jeux de résultats, utilisez [CMultipleResults](../../data/oledb/cmultipleresults-class.md) à la place.

## <a name="requirements"></a>Spécifications

**En-tête :** atldbcli.h
