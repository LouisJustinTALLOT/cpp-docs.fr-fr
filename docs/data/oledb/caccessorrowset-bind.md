# CAccessorRowset::Bind
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Crée des liaisons si vous spécifiez **bBind** comme false dans [CCommand::Open](../../data/oledb/ccommand-open.md).  
  
## Syntaxe  
  
```  
  
HRESULT Bind( );  
  
```  
  
## Valeur de retour  
 Un `HRESULT` standard.  
  
## Configuration requise  
 **En\-tête :** atldbcli.h
