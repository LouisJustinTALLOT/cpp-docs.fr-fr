# <a name="caccessorrowsetbind"></a>CAccessorRowset::Bind

## <a name="syntax"></a>Syntaxe  
  
```cpp
HRESULT Bind();  
  
```  
  
## <a name="return-value"></a>Valeur de retour  
 `HRESULT` standard.
