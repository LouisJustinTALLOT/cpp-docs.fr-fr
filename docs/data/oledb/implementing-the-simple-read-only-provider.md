# Impl&#233;mentation d&#39;un fournisseur simple accessible en lecture seule
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

L'exemple étendu de cette rubrique montre comment modifier les fichiers créés par l'Assistant pour créer un fournisseur simple accessible en lecture seule qui lit un ensemble de deux chaînes à partir d'un fichier texte.  Pour créer ce fournisseur à partir des fichiers de l'Assistant, ajoutez du code pour exécuter les tâches suivantes :  
  
-   [Lecture de chaînes dans le fournisseur OLE DB](../../data/oledb/reading-strings-into-the-ole-db-provider.md)  
  
-   [Stockage de chaînes dans le fournisseur OLE DB](../../data/oledb/storing-strings-in-the-ole-db-provider.md)
