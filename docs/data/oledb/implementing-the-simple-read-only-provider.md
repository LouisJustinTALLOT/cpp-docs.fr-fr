# <a name="implementing-the-simple-read-only-provider"></a>Implémentation d'un fournisseur simple accessible en lecture seule

L’exemple étendu de cette rubrique montre comment modifier les fichiers créés par l’Assistant pour créer un fournisseur simple en lecture seule qui lit un ensemble de deux chaînes à partir d’un fichier texte. Pour créer ce fournisseur à partir des fichiers de l’Assistant, ajoutez le code permettant d’effectuer les tâches suivantes :  
  
- [Lecture de chaînes dans le fournisseur](../../data/oledb/reading-strings-into-the-ole-db-provider.md)  
  
- [Store les chaînes dans le fournisseur](../../data/oledb/storing-strings-in-the-ole-db-provider.md)
