# <a name="cdynamicaccessorclose"></a>CDynamicAccessor::Close
Annule toutes les colonnes, libère la mémoire allouée et libère le [IAccessor](https://msdn.microsoft.com/en-us/library/ms719672.aspx) pointeur d’interface dans la classe.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp
void Close() throw();  
  
```  
  
## <a name="requirements"></a>Configuration requise  
 **En-tête :** atldbcli.h
