# END_PARAM_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Marque la fin des entrées de mappage de paramètres.  
  
## Syntaxe  
  
```  
  
END_PARAM_MAP( )  
  
```  
  
## Exemple  
 Consultez l'exemple de la macro [BEGIN\_PARAM\_MAP](../../data/oledb/begin-param-map.md).  
  
## Configuration requise  
 **En\-tête :** atldbcli.h
