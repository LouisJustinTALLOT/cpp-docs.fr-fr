# <a name="cenumeratoraccessormntype"></a>CEnumeratorAccessor::m_nType
Une variable qui indique si la ligne décrit une source de données ou d’un énumérateur.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp
USHORT m_nType;  
  
```  
  
## <a name="remarks"></a>Notes  
 Consultez [ISourcesRowset::GetSourcesRowset](https://msdn.microsoft.com/en-us/library/ms711200.aspx) dans les *de référence du programmeur OLE DB* pour plus d’informations.
