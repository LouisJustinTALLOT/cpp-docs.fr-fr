# CEnumeratorAccessor::m_nType
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Une variable qui indique si la ligne décrit une source de données ou un énumérateur.  
  
## Syntaxe  
  
```  
  
USHORT m_nType;  
  
```  
  
## Notes  
 Voir [ISourcesRowset::GetSourcesRowset](https://msdn.microsoft.com/en-us/library/ms711200.aspx) dans *La référence du programmeur de DB OLE* pour plus d'informations.  
  
## Configuration requise  
 **En\-tête :** atldbcli.h
