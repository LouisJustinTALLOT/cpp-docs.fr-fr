# <a name="cmultipleresults-class"></a>CMultipleResults, classe

Si vous souhaitez qu’une commande gère plusieurs jeux de résultats, utilisez `CMultipleResults` pour l' `CCommand` argument de modèle *TMultiple*.

## <a name="syntax"></a>Syntaxe

```cpp
class CMultipleResults
```

## <a name="remarks"></a>Notes

Pour gérer plusieurs jeux de résultats, [CCommand](../../data/oledb/ccommand-class.md) doit hériter de cette classe.

## <a name="requirements"></a>Spécifications

**En-tête :** Atldbcli
