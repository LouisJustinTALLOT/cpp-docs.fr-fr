# CEnumeratorAccessor::m_bIsParent
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Une variable qui indique si l'énumérateur est un énumérateur parent, si la ligne est un énumérateur.  
  
## Syntaxe  
  
```  
  
VARIANT_BOOL m_bIsParent;  
  
```  
  
## Notes  
 Voir [ISourcesRowset::GetSourcesRowset](https://msdn.microsoft.com/en-us/library/ms711200.aspx) dans *La référence du programmeur de DB OLE* pour plus d'informations.  
  
## Configuration requise  
 **En\-tête :** atldbcli.h
