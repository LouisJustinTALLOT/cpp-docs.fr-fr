# CDBPropIDSet::AddPropertyID
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ajoute un ID de propriété à l'ensemble des ID de propriété.  
  
## Syntaxe  
  
```  
  
      bool AddPropertyID(   
   DBPROPID propid    
) throw( );  
```  
  
#### Paramètres  
 `propid`  
 \[in\] ID de propriété à ajouter à l'ensemble des ID de propriété.  
  
## Configuration requise  
 **En\-tête :** atldbcli.h
