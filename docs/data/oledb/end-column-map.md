# END_COLUMN_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Marque la fin des entrées de mappage de colonnes.  
  
## Syntaxe  
  
```  
  
END_COLUMN_MAP( )  
  
```  
  
## Notes  
 Il est utilisé avec un seul accesseur sur un ensemble de lignes.  La macro `BEGIN_COLUMN_MAP` est terminée avec la macro `END_COLUMN_MAP`.  
  
## Exemple  
 Voir [BEGIN\_COLUMN\_MAP](../../data/oledb/begin-column-map.md).  
  
## Configuration requise  
 **En\-tête :** atldbcli.h
