Pour plus d'informations, voir les rubriques suivantes :

- [Historique des pilotes pour Microsoft SQL Server](/sql/connect/connect-history)

- [ODBC Driver 13.1 pour SQL Server libéré](https://blogs.technet.microsoft.com/dataplatforminsider/2016/08/03/odbc-driver-13-1-for-sql-server-released/) Microsoft ODBC Driver 13 pour SQL Server devrait être utilisé pour créer de nouvelles applications ou améliorer les applications existantes qui doivent tirer parti des nouvelles fonctionnalités SQL Server 2016.
