> [!NOTE]
>  Microsoft recommande d’utiliser ODBC par le biais de Microsoft ODBC Driver 13 (ou version ultérieure) pour SQL Server pour les nouveaux projets. DAO n’est plus pris en charge.

Pour plus d’informations, consultez les rubriques suivantes :

- [Historique des pilotes pour Microsoft SQL Server](/sql/connect/connect-history)

- [Pilote ODBC 13,1 pour SQL Server publié](https://blogs.technet.microsoft.com/dataplatforminsider/2016/08/03/odbc-driver-13-1-for-sql-server-released/) Microsoft ODBC Driver 13 for SQL Server doit être utilisé pour créer de nouvelles applications ou pour améliorer les applications existantes qui doivent tirer parti des nouvelles fonctionnalités SQL Server 2016.
