# rename_search_namespace
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Section spécifique à C\+\+**  
  
 Possède les mêmes fonctionnalités que l'attribut [rename\_namespace](../preprocessor/rename-namespace.md) mais est utilisé sur les bibliothèques de types dans lesquelles vous utilisez la directive \#import avec l'attribut [auto\_search](../preprocessor/auto-search.md).  
  
## Syntaxe  
  
```  
rename_search_namespace("NewName")  
```  
  
#### Paramètres  
 `NewName`  
 Nom du nouvel espace de noms.
