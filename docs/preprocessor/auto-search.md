# auto_search
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Section spécifique à C\+\+**  
  
 Spécifie que, lorsqu'une bibliothèque de types est référencée avec \#import et fait elle\-même référence à une autre bibliothèque de types, le compilateur peut effectuer une opération \#import implicite pour l'autre bibliothèque de types.  
  
## Syntaxe  
  
```  
auto_search  
```
