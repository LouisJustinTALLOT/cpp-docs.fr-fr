# named_guids
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Section spécifique à C\+\+**  
  
 Indique au compilateur de définir et initialiser des variables GUID dans le style ancien, de la forme **LIBID\_MyLib**, **CLSID\_MyCoClass**, **IID\_MyInterface** et **DIID\_MyDispInterface**.  
  
## Syntaxe  
  
```  
named_guids  
```
