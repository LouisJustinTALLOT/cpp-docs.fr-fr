# <a name="push_macro-pragma"></a>push_macro, pragma

Enregistre la valeur de la macro de *nom de macro* en haut de la pile pour cette macro.

## <a name="syntax"></a>Syntaxe

> **#pragma push_macro ("** _macro-Name_ **")**

## <a name="remarks"></a>Notes

Vous pouvez récupérer la valeur de *macro-Name* avec `pop_macro`.

Pour obtenir un exemple, consultez [pop_macro](../preprocessor/pop-macro.md) .
