# <a name="nosearchnamespace"></a>no_search_namespace
**Spécifique à C++**  
  
 A les mêmes fonctionnalités que le [no_namespace](../preprocessor/no-namespace.md) attribut mais est utilisé sur les bibliothèques de types que vous utilisez la directive #import avec la [auto_search](../preprocessor/auto-search.md) attribut.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
no_search_namespace  
```
