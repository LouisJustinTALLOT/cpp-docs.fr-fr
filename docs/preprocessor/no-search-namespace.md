# no_search_namespace
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Section spécifique à C\+\+**  
  
 Possède les mêmes fonctionnalités que l'attribut [no\_namespace](../preprocessor/no-namespace.md) mais est utilisé sur les bibliothèques de types dans lesquelles vous utilisez la directive \#import avec l'attribut [auto\_search](../preprocessor/auto-search.md).  
  
## Syntaxe  
  
```  
no_search_namespace  
```
