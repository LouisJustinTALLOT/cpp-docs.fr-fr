# <a name="no_search_namespace-import-attribute"></a>no_search_namespace l’attribut d’importation

**Section spécifique à C++**

A les mêmes fonctionnalités que l’attribut [no_namespace](../preprocessor/no-namespace.md) , mais est utilisé sur les bibliothèques de types où vous utilisez la `#import` directive avec l’attribut [auto_search](../preprocessor/auto-search.md) .

## <a name="syntax"></a>Syntaxe

>  *bibliothèque de types* #import **no_search_namespace**

**FIN de la section spécifique à C++**
