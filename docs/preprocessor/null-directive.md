# Directive null
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La directive de préprocesseur null est un signe dièse \(**\#**\) seul sur une ligne.  Elle n'a aucun effet.  
  
## Syntaxe  
  
```  
  
#  
  
```
