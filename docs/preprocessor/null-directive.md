# <a name="null-directive"></a>Directive null
La directive de préprocesseur null est un signe dièse unique (**#**) seul sur une ligne. Elle n'a aucun effet.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
#  
  
```
