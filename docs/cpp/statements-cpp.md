# <a name="statements-c"></a>Instructions (C++)

Les relevés de CMD sont les éléments du programme qui contrôlent comment et dans quel ordre les objets sont manipulés. Cette section comprend :

- [Vue d’ensemble](../cpp/overview-of-cpp-statements.md)

- [Déclarations étiquetées](../cpp/labeled-statements.md)

- Catégories d’énoncés

  - [Déclarations d’expression](../cpp/expression-statement.md). Ces énoncés évaluent une expression pour ses effets secondaires ou pour sa valeur de retour.

  - [Déclarations nulles](../cpp/null-statement.md). Ces déclarations peuvent être fournies lorsqu’une déclaration est exigée par la syntaxe C, mais où aucune mesure n’est à prendre.

  - [Déclarations composées](../cpp/compound-statements-blocks.md). Ces déclarations sont des groupes de déclarations enfermées dans des accolades bouclées. Ils peuvent être utilisés partout où une seule instruction peut être utilisée.

  - [Déclarations de sélection](../cpp/selection-statements-cpp.md). Ces énoncés effectuent un test; ils exécutent ensuite une section de code si le test évalue à vrai (nonzero). Ils peuvent exécuter une autre section de code si le test s’évalue à faux.

  - [Déclarations d’itération](../cpp/iteration-statements-cpp.md). Ces déclarations prévoient l’exécution répétée d’un bloc de code jusqu’à ce qu’un critère de résiliation spécifiée soit respecté.

  - [Énoncés de saut](../cpp/jump-statements-cpp.md). Ces déclarations transfèrent immédiatement le contrôle à un autre endroit de la fonction, soit retournent le contrôle de la fonction.

  - [Déclarations](declarations-and-definitions-cpp.md). Les déclarations introduisent un nom dans un programme.

Pour plus d’informations sur les déclarations de traitement des exceptions, consultez [Exception Handling](../cpp/exception-handling-in-visual-cpp.md).
