# <a name="statements-c"></a>Instructions (C++)

C++les instructions sont les éléments de programme qui contrôlent comment et dans quel ordre les objets sont manipulés. Cette section inclut les éléments suivants :

- [Vue d'ensemble](../cpp/overview-of-cpp-statements.md)

- [Instructions étiquetées](../cpp/labeled-statements.md)

- Catégories d’instructions

   - [Instructions d’expression](../cpp/expression-statement.md). Ces instructions évaluent une expression pour ses effets secondaires ou pour sa valeur de retour.

   - [Instructions null](../cpp/null-statement.md). Ces instructions peuvent être fournies quand une instruction est requise par la C++ syntaxe, mais où aucune action ne doit être effectuée.

   - [Instructions composées](../cpp/compound-statements-blocks.md). Ces instructions sont des groupes d’instructions entre accolades ({}). Elles peuvent être utilisées partout où une seule instruction peut être utilisée.

   - [Instructions de sélection](../cpp/selection-statements-cpp.md). Ces instructions effectuent un test. elles exécutent ensuite une section de code si le test prend la valeur true (différente de zéro). Ils peuvent exécuter une autre section de code si le test prend la valeur false.

   - [Instructions d’itération](../cpp/iteration-statements-cpp.md). Ces instructions fournissent une exécution répétée d’un bloc de code jusqu’à ce qu’un critère de terminaison spécifié soit respecté.

   - [Instructions de saut](../cpp/jump-statements-cpp.md). Ces instructions transfèrent le contrôle immédiatement vers un autre emplacement dans la fonction ou retournent le contrôle à partir de la fonction.

   - [Instructions de déclaration](declarations-and-definitions-cpp.md). Les déclarations introduisent un nom dans un programme.

Pour plus d’informations sur les instructions de gestion des exceptions, consultez [gestion des exceptions](../cpp/exception-handling-in-visual-cpp.md).
