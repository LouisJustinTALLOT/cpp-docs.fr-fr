# Instructions (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les instructions C\+\+ sont des éléments de programme qui contrôlent comment et dans quel ordre les objets sont manipulés.  Cette section inclut :  
  
-   [Vue d'ensemble](../cpp/overview-of-cpp-statements.md)  
  
-   [Instructions étiquetées](../cpp/labeled-statements.md)  
  
-   Catégories d'instructions  
  
    -   [Instructions d'expression](../cpp/expression-statement.md).  Ces instructions évaluent une expression afin de déterminer ses effets secondaires ou sa valeur de retour.  
  
    -   [Instructions null](../cpp/null-statement.md).  Ces instructions peuvent être fournies à l'endroit où une instruction est requise par la syntaxe C\+\+ mais où aucune action ne doit être prise.  
  
    -   [Instructions composées](../cpp/compound-statements-blocks.md).  Ces instructions sont des groupes d'instructions placées entre accolades \({}\).  Elles peuvent être utilisées partout où une seule instruction peut être utilisée.  
  
    -   [Instructions de sélection](../cpp/selection-statements-cpp.md).  Ces instructions réalisent un test ; ils exécutent ensuite une section de code si le test a la valeur true \(une valeur différente de zéro\).  Elles peuvent exécuter une autre section de code si le test a la valeur false.  
  
    -   [Instructions d'itération](../cpp/iteration-statements-cpp.md).  Ces instructions permettent l'exécution répétée d'un bloc de code jusqu'à ce qu'un critère spécifié d'arrêt soit rempli.  
  
    -   [Instructions de saut](../cpp/jump-statements-cpp.md).  Ces instructions transfèrent le contrôle instantanément vers un autre emplacement dans la fonction ou retournent le contrôle depuis la fonction.  
  
    -   [Instructions de déclaration](http://msdn.microsoft.com/fr-fr/14538558-356f-450e-9e1e-3cd62ba952b9).  Les déclarations introduisent un nom dans un programme. \([Déclarations](../misc/declarations.md) fournit davantage d'informations détaillées sur les déclarations.\)  
  
 Pour plus d'informations sur les instructions de gestion des exceptions consultez [Gestion des exceptions](../cpp/exception-handling-in-visual-cpp.md).
