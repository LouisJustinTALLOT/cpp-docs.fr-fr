# <a name="__m64"></a>__m64

**Section spécifique de Microsoft**

Le type de données **__m64** est destiné à être utilisé avec les instructions MMX et 3DNow ! intrinsèques, et est défini dans \<> xmmintrin. h.

```cpp
// data_types__m64.cpp
#include <xmmintrin.h>
int main()
{
   __m64 x;
}
```

## <a name="remarks"></a>Notes

Vous ne devez pas accéder directement aux champs **__m64** . Vous pouvez toutefois visualiser ces types dans le débogueur. Une variable de type **__m64** est mappée aux registres mm [0-7].

Les variables de type **_m64** sont automatiquement alignées sur des limites de 8 octets.

Le type de données **__m64** n’est pas pris en charge sur les processeurs x64. Les applications qui utilisent __m64 dans le cadre des intrinsèques MMX doivent être réécrites pour utiliser les intrinsèques SSE et SSE2 équivalentes.

**Fin de la section spécifique de Microsoft**
