# <a name="numerical-limits-c"></a>Limites numériques (C++)

Les deux fichiers Include standard, \<limits.h> et \<float.h> , définissent les limites numériques, ou les valeurs minimales et maximales qu’une variable d’un type donné peut contenir. Ces valeurs minimales et maximales sont garanties comme étant portables à n’importe quel compilateur C++ qui utilise la même représentation de données que le langage C ANSI. Le \<limits.h> fichier include définit les [limites numériques pour les types intégraux](../cpp/integer-limits.md)et \<float.h> définit les [limites numériques pour les types flottants](../cpp/floating-limits.md).
