# Limites num&#233;riques (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les deux fichiers include standard, LIMITS.H et FLOAT.H, définissent les limites numériques ou les valeurs minimales et maximales qu'une variable d'un type donné peut contenir.  Ces valeurs minimales et maximales sont garanties comme portables pour tout compilateur C\+\+ qui utilise la même représentation des données qu'ANSI C.  Le fichier include LIMITS.H définit les limites numériques [pour les types intégraux](../cpp/integer-limits.md) et FLOAT.H définit les limites numériques [pour les types flottants](../cpp/floating-limits.md).
