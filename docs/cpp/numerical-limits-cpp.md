# <a name="numerical-limits-c"></a>Limites numériques (C++)

Les deux fichiers Include standard, \<Limits. h > et \<float. h >, définissent les limites numériques, ou les valeurs minimales et maximales qu’une variable d’un type donné peut contenir. Ces valeurs minimales et maximales sont garanties portables pour tout C++ compilateur qui utilise la même représentation de données que le C ANSI. \<le fichier include Limits. h > définit les [limites numériques pour les types intégraux](../cpp/integer-limits.md)et \<float. h > définit les [limites numériques pour les types flottants](../cpp/floating-limits.md).
