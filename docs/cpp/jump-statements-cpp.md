# Instructions de saut (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Une instruction de saut C\+\+ exécute un transfert de contrôle local immédiat.  
  
## Syntaxe  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
