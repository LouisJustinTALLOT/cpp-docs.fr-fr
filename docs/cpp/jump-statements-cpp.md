# <a name="jump-statements-c"></a>Instructions de saut (C++)
Une instruction de saut C++ exécute un transfert de contrôle local immédiat.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
