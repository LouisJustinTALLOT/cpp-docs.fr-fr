# Indirection sur les types de tableau
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

L'utilisation de l'opérateur d'indirection \(**\***\) sur un type de tableau dimensionnel *n*génère un tableau unidimensionnel *n*.  Si *n* correspond à 1, une variable scalaire \(ou un élément de tableau\) est générée.
