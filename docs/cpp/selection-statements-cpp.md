# Instructions de s&#233;lection (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Les instructions de sélection C\+\+, [if](../cpp/if-else-statement-cpp.md) et [switch](../cpp/switch-statement-cpp.md), permettent d'exécuter de manière conditionnelle des sections de code.  
  
 Les instructions [\_\_if\_exists](../cpp/if-exists-statement.md) et [\_\_if\_not\_exists](../cpp/if-not-exists-statement.md) vous permettent d'inclure de manière conditionnelle du code en fonction de l'existence d'un symbole.  
  
 Consultez les différentes rubriques concernant la syntaxe de ces instructions.
