# <a name="__m128"></a>__m128

**Section spécifique à Microsoft**

Le type de **données __m128** , à utiliser avec les extensions streaming SIMD et les instructions streaming SIMD Extensions 2, est défini dans \<xmmintrin. h >.

```cpp
// data_types__m128.cpp
#include <xmmintrin.h>
int main() {
   __m128 x;
}
```

## <a name="remarks"></a>Notes

Vous ne devez pas accéder directement aux champs **__m128** . Vous pouvez toutefois visualiser ces types dans le débogueur. Une variable de type **__m128** est mappée aux registres XMM [0-7].

Les variables de type **__m128** sont alignées automatiquement sur des limites de 16 octets.

Le type de données **__m128** n’est pas pris en charge sur les processeurs ARM.

**Fin de la section spécifique à Microsoft**
