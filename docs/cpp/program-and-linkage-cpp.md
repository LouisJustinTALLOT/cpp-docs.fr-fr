# Programme et liaison (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Un programme se compose d'un ou de plusieurs unités de traduction liées ensemble.  L'exécution \(conceptuellement\) commence dans l'unité de traduction qui contient la fonction **main**.  \(Pour plus d'informations sur les unités de traduction, consultez [Phases de traduction](../preprocessor/phases-of-translation.md) dans *Référence du préprocesseur*.\) Pour plus d'informations sur la fonction **main**, consultez [Démarrage du programme : la fonction main](../cpp/main-program-startup.md).
