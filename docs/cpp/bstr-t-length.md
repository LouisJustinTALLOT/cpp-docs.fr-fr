# _bstr_t::length
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Section spécifique à Microsoft**

## Syntaxe  
  
```  
  
unsigned int length ( ) const throw( );  
  
```
