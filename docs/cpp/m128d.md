# <a name="__m128d"></a>__m128d

**Section spécifique de Microsoft**

Le type de données **__m128d** , pour une utilisation avec les intrinsèques d’extensions streaming SIMD 2, est défini dans \<emmintrin. h >.

```cpp
// data_types__m128d.cpp
#include <emmintrin.h>
int main() {
   __m128d x;
}
```

## <a name="remarks"></a>Notes

Vous ne devez pas accéder directement aux champs **__m128d** . Vous pouvez toutefois visualiser ces types dans le débogueur. Une variable de type **__m128** est mappée aux registres XMM [0-7].

Les variables de type **_m128d** sont alignées automatiquement sur des limites de 16 octets.

Le type de données **__m128d** n’est pas pris en charge sur les processeurs ARM.

**Fin de la section spécifique de Microsoft**
