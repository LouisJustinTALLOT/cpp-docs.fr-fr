# <a name="comerroroperator-"></a>_com_error::operator =
**Section spécifique à Microsoft**  
  
 Assigne un objet `_com_error` existant à un autre.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
      _com_error& operator = (  
   const _com_error& that   
) throw ( );  
```  
  
#### <a name="parameters"></a>Paramètres  
 `that`  
 Objet `_com_error`.
