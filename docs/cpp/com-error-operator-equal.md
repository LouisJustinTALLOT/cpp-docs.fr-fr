# <a name="_com_erroroperator-"></a>_com_error::operator =

**Spécifique à Microsoft**

Assigne un objet `_com_error` existant à un autre.

## <a name="syntax"></a>Syntaxe

```
_com_error& operator = (
   const _com_error& that
) throw ( );
```

#### <a name="parameters"></a>Paramètres

*que*<br/>
Objet `_com_error`.
