# <a name="_com_erroroperator-"></a>_com_error::operator =

**Section spécifique de Microsoft**

Assigne un objet `_com_error` existant à un autre.

## <a name="syntax"></a>Syntaxe

```
_com_error& operator = (
   const _com_error& that
) throw ( );
```

#### <a name="parameters"></a>Paramètres

*comprenant*<br/>
Objet `_com_error` .
