# _com_error::operator =
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Section spécifique à Microsoft**  
  
 Assigne un objet `_com_error` existant à un autre.  
  
## Syntaxe  
  
```  
  
      _com_error& operator = (  
   const _com_error& that   
) throw ( );  
```  
  
#### Paramètres  
 `that`  
 Objet `_com_error`.
