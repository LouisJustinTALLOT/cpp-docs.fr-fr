# find (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Recherche la position de la première occurrence d'un élément d'une plage ayant une valeur spécifiée.  
  
## Syntaxe  
  
```  
template<class _InIt, class _Ty> inline  
    _InIt find(_InIt _First, _InIt _Last, const _Ty% _Val);  
```  
  
## Notes  
 Cette fonction se comporte comme la fonction STL `find`.  Pour plus d'informations, consultez [find](../Topic/find%20\(STL\).md).  
  
## Configuration requise  
 **En\-tête :** \<cliext\/algorithm\>  
  
 **Espace de nom :** cliext
