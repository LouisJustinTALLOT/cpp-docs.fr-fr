# <a name="standard-conversions-and-implicit-boxing"></a>Conversions standard et boxing implicite

Une conversion standard sera choisie par le compilateur via une conversion qui n√©cessite une conversion boxing.

## <a name="example"></a>Exemple

```
// clr_implicit_boxing_Std_conversion.cpp
// compile with: /clr
int f3(int ^ i) {   // requires boxing
   return 1;
}

int f3(char c) {   // no boxing required, standard conversion
   return 2;
}

int main() {
   int i = 5;
   System::Console::WriteLine(f3(i));
}
```

```Output
2
```
