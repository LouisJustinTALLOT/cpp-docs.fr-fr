# Comment&#160;: cr&#233;er des projets vides CLR
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

To create a CLR empty project, use the **CLR Empty Project** template, which is available from the **New Project** dialog box.  
  
> [!NOTE]
>  L'apparence des fonctionnalités dans l'environnement IDE peut dépendre de votre édition ou de vos paramètres actifs, et peut différer de celle décrite dans l'aide.  Pour modifier vos paramètres, choisissez **Importation et exportation de paramètres** dans le menu **Outils**.  Pour plus d'informations, consultez [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/fr-fr/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
### Pour créer un projet vide CLR  
  
1.  Dans le menu **Fichier**, cliquez sur **Nouveau**, puis sur **Projet**.  
  
     La boîte de dialogue **Nouveau projet** s'affiche.  
  
2.  Sous **Modèles installés**, cliquez sur le nœud **Visual C\+\+**, puis sur le nœud **CLR**.  Choisissez l'icône **Projet vide CLR**.  
  
3.  Dans la zone **Nom**, entrez un nom unique pour votre application.  
  
    > [!NOTE]
    >  Vous pouvez également spécifier d'autres paramètres de projet et de solution dans la boîte de dialogue **Nouveau projet**, mais ces paramètres ne sont pas obligatoires.  
  
4.  Cliquez sur **OK**.
