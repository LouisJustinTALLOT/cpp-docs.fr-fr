# cliext, espace de noms
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

L'espace de noms `cliext` contient tous les types de la bibliothèque STL\/CLR.  Pour obtenir la liste de tous ces types, et des liens vers des informations sur les types de STL\/CLR, consultez [STL\/CLR, bibliothèque](../dotnet/stl-clr-library-reference.md).
