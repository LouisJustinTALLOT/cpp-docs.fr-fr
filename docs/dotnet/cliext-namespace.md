# <a name="cliext-namespace"></a>cliext, espace de noms

L' `cliext` espace de noms contient tous les types de la bibliothèque STL/CLR. Pour obtenir la liste de tous ces types et des liens vers des informations supplémentaires sur les types STL/CLR, consultez Référence de la [bibliothèque STL/CLR](../dotnet/stl-clr-library-reference.md).
