# <a name="utility-stlclr"></a>utility (STL/CLR)
Incluez l’en-tête STL/CLR `<cliext/utility>` pour définir la classe de modèle `pair` et plusieurs fonctions de modèle de prise en charge.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
#include <utility>  
```  
  
## <a name="declarations"></a>Déclarations  
  
|Classe|Description|  
|-----------|-----------------|  
|[pair (STL/CLR)](../dotnet/pair-stl-clr.md)|Encapsuler une paire d’éléments.|  
  
|Opérateur|Description|  
|--------------|-----------------|  
|[operator== (pair) (STL/CLR)](../dotnet/operator-equality-pair-stl-clr.md)|Comparaison égale de la paire.|  
|[operator!= (pair) (STL/CLR)](../dotnet/operator-inequality-pair-stl-clr.md)|Paires de comparaison n’est pas égal.|  
|[operator< (pair) (STL/CLR)](../dotnet/operator-less-than-pair-stl-clr.md)|Paire de comparaison inférieur.|  
|[opérateur\<= (paire) (STL/CLR)](../dotnet/operator-less-or-equal-pair-stl-clr.md)|Inférieur ou égal paire comparaison.|  
|[operator> (pair) (STL/CLR)](../dotnet/operator-greater-than-pair-stl-clr.md)|Supérieure à la comparaison de la paire.|  
|[operator>= (pair) (STL/CLR)](../dotnet/operator-greater-or-equal-pair-stl-clr.md)|Paire supérieur ou égal comparaison.|  
  
|Fonction|Description|  
|--------------|-----------------|  
|[make_pair (STL/CLR)](../dotnet/make-pair-stl-clr.md)|Rendre une paire d’une paire de valeurs.|  
  
## <a name="requirements"></a>Spécifications  
 **En-tête :** \<cliext/utilitaire >  
  
 **Namespace :** cliext
