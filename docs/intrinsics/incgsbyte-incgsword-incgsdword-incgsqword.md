# <a name="incgsbyte-incgsword-incgsdword-incgsqword"></a>__incgsbyte, __incgsword, __incgsdword, __incgsqword

**Section spécifique à Microsoft**

Ajouter un à la valeur à un emplacement de mémoire spécifié par un offset par rapport au début de la `GS` segment.

## <a name="syntax"></a>Syntaxe

```
void __incgsbyte(
   unsigned long Offset
);
void __incgsword(
   unsigned long Offset
);
void __incgsdword(
   unsigned long Offset
);
void __incgsqword(
   unsigned long Offset
);
```

#### <a name="parameters"></a>Paramètres

*Offset*<br/>
[in] Le décalage à partir du début de `GS`.

## <a name="requirements"></a>Configuration requise

|Intrinsèque|Architecture|
|---------------|------------------|
|`__incgsbyte`|X64|
|`__incgsword`|X64|
|`__incgsdword`|X64|
|`__incgsqword`|X64|

## <a name="remarks"></a>Notes

Ces routines sont uniquement disponibles comme intrinsèque.
