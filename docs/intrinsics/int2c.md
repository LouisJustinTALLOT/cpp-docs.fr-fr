# <a name="int2c"></a>__int2c
**Section spécifique à Microsoft**  
  
 Génère le `int 2c` instruction, ce qui déclenche le `2c` interruption.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
void __int2c(void);  
```
