# <a name="__int2c"></a>__int2c

**Spécifique à Microsoft**

Génère l' `int 2c` instruction, qui déclenche l' `2c` interruption.

## <a name="syntax"></a>Syntaxe

```C
void __int2c(void);
```

## <a name="requirements"></a>Spécifications

|Intrinsic|Architecture|
|---------------|------------------|
|`__int2c`|x86, x64|
