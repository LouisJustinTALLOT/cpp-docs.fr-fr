# C++ REST SDK (Codename &quot;Casablanca&quot;)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

À l'aide du Kit de développement logiciel \(SDK\) C\+\+ REST \(nom de code « Casablanca »\), vous pouvez plus facilement écrire du code C\+\+ moderne et asynchrone qui se connecte aux services cloud.  
  
 Le projet Casablanca n'est plus livré avec Visual Studio.  La documentation et les sources sont hébergées et occasionnellement mises à jour sur CodePlex.  Pour plus d'informations sur ce projet, consultez [Kit de développement logiciel \(SDK\) C\+\+ REST \(nom de code « Casablanca »\)](http://casablanca.codeplex.com/).
